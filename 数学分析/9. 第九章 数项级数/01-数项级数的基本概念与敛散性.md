## 2. 背景与应用场景

在第二章中，我们深入研究了数列 $\{a_n\}$ 及其极限。然而，数学和工程中经常遇到的是另一种形式的"无穷和"问题：将一列数依次相加，问总和是多少？

**问题示例**：
- 将单位线段不断对半取长再求和：$1 + \frac12 + \frac14 + \frac18 + \cdots$——这个和是 $2$ 吗？
- 在银行中每年存入 1 元，年利率 5%，按复利计算，最终财富是多少？
- 将一个函数展开为多项式之和（如 $\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$）——这无穷多项相加有意义吗？

所有这些问题的核心都是同一个数学对象：**级数**（series）——将无穷多个数相加的表达式。

与有限和不同，无穷和不能直接"加完"——我们永远加不到最后一项。因此需要一种间接的方式来定义无穷和：**考虑部分和数列的极限**。

从直觉上看，如果前 $n$ 项的和 $S_n$ 随着 $n$ 的增大越来越接近某个有限值 $S$，那么我们就称这个无穷级数的和为 $S$。这正是级数理论的基本出发点。

**本章内容概览**：本章研究**数项级数**——每一项都是常数的级数。我们将系统学习：
- 级数的定义与收敛性判断
- 几类重要级数（等比级数、$p$-级数）的敛散性
- 正项级数的各种判别法（比较判别法、比值判别法、根值判别法）
- 任意项级数（交错级数、绝对收敛与条件收敛）

本文件作为第一章，重点建立级数的基本概念框架，并介绍两种最基本的方法：**裂项相消法**和**等比级数**。

## 3. 核心概念与符号约定

### 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\sum_{n=1}^{\infty} a_n$ | 数项级数（$a_n$ 为通项） | $\sum_{n=1}^{\infty} \frac{1}{n}$ |
| $S_n$ | 前 $n$ 项部分和，$S_n = \sum_{k=1}^{n} a_k$ | $S_n = 1 + \frac12 + \cdots + \frac1n$ |
| $\{S_n\}$ | 部分和数列 | $S_1 = a_1, S_2 = a_1+a_2, S_3 = a_1+a_2+a_3, \dots$ |
| $S$ | 级数的和（若收敛），$S = \lim S_n$ | $\sum_{n=1}^{\infty} \frac{1}{2^{n-1}} = 2$ |
| $R_n$ | 余项，$R_n = S - S_n = \sum_{k=n+1}^{\infty} a_k$ | 截断误差 |
| $a_n \to 0$ | $\lim a_n = 0$（通项趋于零） | $\frac{1}{n} \to 0$，$\frac{1}{\sqrt{n}} \to 0$ |
| $|r| < 1$ | 等比级数的收敛条件 | $r = \frac12$ 收敛，$r = 2$ 发散 |

### 关键概念的精确定义

#### 3.1 数项级数的定义

**定义 9.1（数项级数）**：设 $\{a_n\}$ 是一个无穷数列（$n = 1, 2, 3, \dots$）。形式表达式
$$\sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + \cdots + a_n + \cdots$$
称为**数项级数**（简称**级数**），其中 $a_n$ 称为级数的**通项**（一般项）。

**注意**：$\sum_{n=1}^{\infty} a_n$ 在未被赋予收敛性意义之前，仅仅是一个形式记号——它表示"把所有 $a_n$ 加起来"这个操作，但不代表一个具体的数值。要使它有意义，必须引入收敛的概念。

#### 3.2 部分和与收敛的定义

**定义 9.2（部分和与收敛性）**：对级数 $\sum_{n=1}^{\infty} a_n$，定义其**前 $n$ 项部分和**（简称**部分和**）为
$$S_n = \sum_{k=1}^{n} a_k = a_1 + a_2 + \cdots + a_n$$

于是得到部分和数列 $\{S_n\}$：
$$S_1 = a_1,\quad S_2 = a_1 + a_2,\quad S_3 = a_1 + a_2 + a_3,\quad \dots$$

1. 若部分和数列 $\{S_n\}$ **收敛**（即极限 $\lim_{n\to\infty} S_n$ 存在且有限），则称级数 $\sum_{n=1}^{\infty} a_n$ **收敛**（converges），并记
   $$\sum_{n=1}^{\infty} a_n = \lim_{n\to\infty} S_n = S$$
   称 $S$ 为级数的**和**。

2. 若部分和数列 $\{S_n\}$ **发散**（极限不存在或趋于无穷），则称级数 $\sum_{n=1}^{\infty} a_n$ **发散**（diverges）。

**级数收敛的核心思想**：级数的收敛性完全等价于部分和数列 $\{S_n\}$ 的收敛性。因此，第二章中所有关于数列极限的定理（四则运算法则、夹逼准则、单调有界定理、Cauchy 收敛准则等）都可通过 $\{S_n\}$ 迁移到级数理论中。

#### 3.3 余项

**定义 9.3（余项）**：若级数 $\sum_{n=1}^{\infty} a_n$ 收敛于 $S$，则称
$$R_n = S - S_n = \sum_{k=n+1}^{\infty} a_k$$
为级数的 **$n$ 阶余项**（或**截断误差**）。

由于 $\lim S_n = S$，对任意收敛级数有 $\lim_{n\to\infty} R_n = 0$。

**例 1**：考虑级数 $\sum_{n=1}^{\infty} \frac{1}{2^{n-1}} = 1 + \frac12 + \frac14 + \frac18 + \cdots$。

部分和为 $S_n = 1 + \frac12 + \frac14 + \cdots + \frac1{2^{n-1}}$。

由文件 11 定理 11.3（等比数列求和公式）：
$$S_n = \frac{1 - (1/2)^n}{1 - 1/2} = 2\left(1 - \frac1{2^n}\right)$$

因此 $\lim_{n\to\infty} S_n = 2$，级数收敛，和为 $2$。余项 $R_n = 2 - S_n = \frac{1}{2^{n-1}} \to 0$。

## 4. 原理与方法

### 4.1 级数收敛的必要条件

在级数理论中，第一个要回答的问题是：通项 $a_n$ 满足什么条件时，级数 $\sum a_n$ 才 **可能** 收敛？

**定理 9.1（级数收敛的必要条件）**：若级数 $\sum_{n=1}^{\infty} a_n$ 收敛，则其通项必趋于零，即
$$\lim_{n\to\infty} a_n = 0$$

**证明**：

设级数 $\sum a_n$ 收敛于 $S$，即 $\lim_{n\to\infty} S_n = S$，其中 $S_n = \sum_{k=1}^{n} a_k$。

注意到通项 $a_n$ 可以表示为相邻部分和的差：
$$a_n = S_n - S_{n-1} \quad (n \geq 2)$$

这是因为 $S_n = a_1 + a_2 + \cdots + a_{n-1} + a_n$，而 $S_{n-1} = a_1 + a_2 + \cdots + a_{n-1}$，相减即得 $a_n$。

由于 $\lim S_n = S$，去掉首项后的子列 $\{S_{n-1}\}$ 也收敛到同一极限 $S$（数列极限的性质——去掉有限多项不影响极限）。因此由极限的差法则（文件 09 定理 9.1(1)）：
$$\lim_{n\to\infty} a_n = \lim_{n\to\infty} (S_n - S_{n-1}) = \lim_{n\to\infty} S_n - \lim_{n\to\infty} S_{n-1} = S - S = 0$$

证毕。

**逆否命题**：定理 9.1 的逆否命题提供了一个判断级数发散的有力工具：

$$\boxed{\text{若 } \lim_{n\to\infty} a_n \neq 0 \text{（或不存在），则级数 } \sum a_n \text{ 必发散}}$$

**注意**：$\lim a_n \neq 0$ 包含两种情况：
- $\lim a_n$ 存在但不等于 $0$（如 $a_n \to \frac12$）
- $\lim a_n$ 不存在（如 $a_n = (-1)^n$）

只要满足其中之一，级数就必然发散。

**例 2**：判断级数 $\sum_{n=1}^{\infty} \frac{n+1}{2n-1}$ 的敛散性。

**解**：计算通项的极限：
$$\lim_{n\to\infty} \frac{n+1}{2n-1} = \lim_{n\to\infty} \frac{1 + \frac1n}{2 - \frac1n} = \frac12 \neq 0$$
（分子分母同除以 $n$，由文件 09 有理函数极限的基本模式。）

由于通项极限不为零，由定理 9.1 的逆否命题，该级数发散。

**进一步说明发散的"强度"**：实际上，由于 $a_n \to \frac12$，当 $n$ 充分大时 $a_n > \frac14$。因此：
$$S_n = \sum_{k=1}^{n} a_k > \sum_{k=1}^{N} a_k + \frac14 (n-N) \to +\infty \quad (n \to \infty)$$
部分和数列发散到 $+\infty$。

### 4.2 通项趋于零的非充分性

定理 9.1 告诉我们的结论是单向的：
$$\text{级数收敛} \;\Longrightarrow\; \lim a_n = 0$$

初学者极易犯的错误是认为反向也成立——即"若 $\lim a_n = 0$，则级数收敛"。这个推断是**错误的**。下面用两个反例说明。

#### 反例 1：调和级数 $\displaystyle\sum_{n=1}^{\infty} \frac{1}{n}$

调和级数的通项 $a_n = \frac{1}{n}$ 满足 $\lim a_n = 0$（文件 08 例 5）。

但其部分和 $H_n = \sum_{k=1}^{n} \frac{1}{k}$ 在文件 10 例 4 中已被证明发散。简要回顾该证明：

取 $\varepsilon_0 = \frac12$。对任意 $N$，取 $n = N$，$m = 2N$，则：
$$|H_{2N} - H_N| = \sum_{k=N+1}^{2N} \frac{1}{k} \geq \sum_{k=N+1}^{2N} \frac{1}{2N} = N \cdot \frac{1}{2N} = \frac12 = \varepsilon_0$$

因此 $\{H_n\}$ 不是 Cauchy 列，由 Cauchy 收敛准则（定理 10.3）知 $\{H_n\}$ 发散。事实上 $H_n \to +\infty$（单调递增且无上界）。

**即**：通项 $1/n \to 0$，但级数 $\sum 1/n$ 发散。

#### 反例 2：级数 $\displaystyle\sum_{n=1}^{\infty} \frac{1}{\sqrt{n}}$

通项 $a_n = \frac{1}{\sqrt{n}}$ 满足 $\lim a_n = 0$（文件 08 例 6）。

现在考察部分和 $S_n = \sum_{k=1}^{n} \frac{1}{\sqrt{k}}$。对每个 $k = 1, 2, \dots, n$，由于 $\sqrt{k} \leq \sqrt{n}$，有：
$$\frac{1}{\sqrt{k}} \geq \frac{1}{\sqrt{n}}$$

因此：
$$S_n = \sum_{k=1}^{n} \frac{1}{\sqrt{k}} \geq \sum_{k=1}^{n} \frac{1}{\sqrt{n}} = n \cdot \frac{1}{\sqrt{n}} = \sqrt{n}$$

由于 $\lim_{n\to\infty} \sqrt{n} = +\infty$，由夹逼准则的推广（下界发散到无穷则被夹数列也发散到无穷），$\{S_n\}$ 发散到 $+\infty$。

**即**：通项 $1/\sqrt{n} \to 0$，但级数 $\sum 1/\sqrt{n}$ 发散。

#### 结论

综合定理 9.1 和以上两个反例，我们得到级数理论中一个最基本的认识：

$$\boxed{\lim_{n\to\infty} a_n = 0 \text{ 是级数 } \sum a_n \text{ 收敛的**必要条件**，但**不是充分条件**}}$$

形象地说：$a_n \to 0$ 是级数收敛的"门槛"——不满足则一定发散，满足则未必收敛。两个著名反例（调和级数和 $\sum 1/\sqrt{n}$）说明，即使通项趋于零，级数仍然可能发散。这迫使我们必须寻找更精细的判别准则（将在后续文件中学习）。

### 4.3 裂项相消法（Telescoping Method）

当我们面对的级数通项是有理分式（如 $\frac{1}{n(n+1)}$、$\frac{1}{n(n+2)}$）时，可以通过**裂项相消法**来求部分和的闭式，从而判断敛散性。

#### 4.3.1 部分分式分解

部分分式分解是将一个复杂分式拆成多个简单分式之和的过程。

**基本模式**：对形如 $\frac{1}{(n+a)(n+b)}$（其中 $a, b$ 为常数）的表达式，可以设：
$$\frac{1}{(n+a)(n+b)} = \frac{c}{n+a} + \frac{d}{n+b}$$

通过通分并比较分子来确定常数 $c, d$。

**例 3**：将 $\frac{1}{n(n+1)}$ 分解为部分分式。

**解**：设 $\frac{1}{n(n+1)} = \frac{c}{n} + \frac{d}{n+1}$。通分右端：
$$\frac{c}{n} + \frac{d}{n+1} = \frac{c(n+1) + dn}{n(n+1)} = \frac{(c+d)n + c}{n(n+1)}$$

比较分子，得 $(c+d)n + c = 1$ 对所有 $n$ 成立，因此：
$$\begin{cases}
c + d = 0 \\
c = 1
\end{cases}
\implies c = 1,\ d = -1$$

因此：
$$\boxed{\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}}$$

#### 4.3.2 裂项求和的一般步骤

对于级数 $\sum_{n=1}^{\infty} f(n)$，若 $f(n)$ 可以写成 $g(n) - g(n+1)$（或类似形式），则称为**可裂项**的级数。求部分和的步骤为：

1. **裂项**：将通项 $a_k$ 分解为两个（或多个）简单项的差，如 $a_k = g(k) - g(k+1)$
2. **展开**：将 $S_n = \sum_{k=1}^{n} a_k$ 逐项写出
3. **消去**：观察抵消模式，删除互为相反数的项
4. **确定幸存项**：识别未被抵消的项
5. **写出闭式**：得到 $S_n$ 的显式表达式

**例 4**（文件 10 练习题 4 回顾）：求级数 $\sum_{n=1}^{\infty} \frac{1}{n(n+1)}$ 的部分和 $S_n$，并判断敛散性。

**解**：

**第 1 步：裂项**。由例 3：
$$a_k = \frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$$

**第 2 步：展开求和**。
$$S_n = \sum_{k=1}^{n} \left(\frac{1}{k} - \frac{1}{k+1}\right)$$

逐项写出（前几项和最后几项）：
$$
\begin{aligned}
S_n = &\left(\frac{1}{1} - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \left(\frac{1}{3} - \frac{1}{4}\right) \\
&+ \cdots + \left(\frac{1}{n-1} - \frac{1}{n}\right) + \left(\frac{1}{n} - \frac{1}{n+1}\right)
\end{aligned}
$$

**第 3 步：观察抵消**。$-\frac12$ 和 $+\frac12$ 抵消，$-\frac13$ 和 $+\frac13$ 抵消，$\dots$，$-\frac1n$ 和 $+\frac1n$ 抵消。

**第 4 步：确定幸存项**。只剩第一项 $\frac11$ 和最后一项 $-\frac{1}{n+1}$。

**第 5 步：写出闭式**。
$$\boxed{S_n = 1 - \frac{1}{n+1}}$$

**第 6 步：判断敛散性**。$\lim_{n\to\infty} S_n = \lim_{n\to\infty} \left(1 - \frac{1}{n+1}\right) = 1$。因此级数收敛，和为 $1$。

#### 4.3.3 系数不为 1 的裂项

当分母中两项的差不是 $1$ 时，裂项后会出现系数。

**例 5**：求 $\sum_{n=1}^{\infty} \frac{1}{n(n+2)}$ 的部分和 $S_n$，并判断敛散性。

**解**：

**第 1 步：部分分式分解**。设：
$$\frac{1}{n(n+2)} = \frac{c}{n} + \frac{d}{n+2}$$

通分右端：
$$\frac{c}{n} + \frac{d}{n+2} = \frac{c(n+2) + dn}{n(n+2)} = \frac{(c+d)n + 2c}{n(n+2)}$$

比较分子得：
$$\begin{cases}
c + d = 0 \\
2c = 1
\end{cases}
\implies c = \frac12,\ d = -\frac12$$

因此：
$$\boxed{\frac{1}{n(n+2)} = \frac12\left(\frac{1}{n} - \frac{1}{n+2}\right)}$$

**第 2 步：展开求和**。
$$S_n = \sum_{k=1}^{n} \frac12\left(\frac{1}{k} - \frac{1}{k+2}\right) = \frac12 \sum_{k=1}^{n} \left(\frac{1}{k} - \frac{1}{k+2}\right)$$

展开求和符号内部的表达式：
$$
\begin{aligned}
\sum_{k=1}^{n} \left(\frac{1}{k} - \frac{1}{k+2}\right) =& \left(\frac{1}{1} - \frac{1}{3}\right) + \left(\frac{1}{2} - \frac{1}{4}\right) + \left(\frac{1}{3} - \frac{1}{5}\right) \\
&+ \left(\frac{1}{4} - \frac{1}{6}\right) + \cdots + \left(\frac{1}{n-2} - \frac{1}{n}\right) \\
&+ \left(\frac{1}{n-1} - \frac{1}{n+1}\right) + \left(\frac{1}{n} - \frac{1}{n+2}\right)
\end{aligned}
$$

**第 3 步：观察抵消**。
- $-\frac13$ 与 $+\frac13$ 抵消
- $-\frac14$ 与 $+\frac14$ 抵消
- 依此类推，$-\frac{1}{n}$ 与 $+\frac{1}{n}$ 抵消
- 但 $-\frac{1}{n+1}$ 和 $-\frac{1}{n+2}$ 是负项，没有正项与之抵消

**第 4 步：确定幸存项**。未抵消的项为：
- 正项：$\frac11$ 和 $\frac12$
- 负项：$-\frac{1}{n+1}$ 和 $-\frac{1}{n+2}$

因此：
$$\sum_{k=1}^{n} \left(\frac{1}{k} - \frac{1}{k+2}\right) = 1 + \frac12 - \frac{1}{n+1} - \frac{1}{n+2}$$

**第 5 步：写出闭式**。
$$\boxed{S_n = \frac12\left(\frac32 - \frac{1}{n+1} - \frac{1}{n+2}\right) = \frac34 - \frac12\left(\frac{1}{n+1} + \frac{1}{n+2}\right)}$$

**第 6 步：判断敛散性**。
$$\lim_{n\to\infty} S_n = \frac34 - \frac12(0 + 0) = \frac34$$

因此级数收敛，和为 $\dfrac34$。

**与例 4 的对比**：在例 4 中 $\frac{1}{n(n+1)}$ 裂项后系数为 $1$，幸存项只有 $2$ 项（$1$ 和 $-\frac{1}{n+1}$）；而在本例 $\frac{1}{n(n+2)}$ 中，由于分母中两项相差 $2$，裂项多出系数 $\frac12$，且幸存项变为 $4$ 项（正项 $1, \frac12$，负项 $-\frac{1}{n+1}, -\frac{1}{n+2}$）。

### 4.4 等比级数（Geometric Series）

**定义 9.4（等比级数）**：形如
$$\sum_{n=1}^{\infty} ar^{n-1} = a + ar + ar^2 + ar^3 + \cdots$$
的级数称为**等比级数**（或**几何级数**），其中 $a \neq 0$ 为首项，$r$ 为公比。

**等比级数的部分和**：由文件 11 定理 11.3（等比数列前 $n$ 项和公式）：
$$S_n = a + ar + ar^2 + \cdots + ar^{n-1} = a \cdot \frac{1 - r^n}{1 - r} \quad (r \neq 1)$$

**定理 9.2（等比级数的敛散性）**：等比级数 $\sum_{n=1}^{\infty} ar^{n-1}$（$a \neq 0$）的敛散性完全由公比 $r$ 决定：

1. 当 $|r| < 1$ 时，级数**收敛**，且
   $$\sum_{n=1}^{\infty} ar^{n-1} = \frac{a}{1 - r}$$

2. 当 $|r| \geq 1$ 时，级数**发散**（包括 $r = 1$ 时部分和 $S_n = na \to \infty$；$r = -1$ 时 $S_n$ 在 $a$ 和 $0$ 间振荡；$|r| > 1$ 时 $S_n$ 无界振荡或趋于无穷）。

**证明**：

**情形 1：$|r| < 1$**。由文件 11 定理 11.4（$\lim q^n = 0$ 当 $|q| < 1$），$\lim_{n\to\infty} r^n = 0$。因此：
$$\lim_{n\to\infty} S_n = \lim_{n\to\infty} a \cdot \frac{1 - r^n}{1 - r} = \frac{a}{1 - r} \cdot \lim_{n\to\infty} (1 - r^n) = \frac{a}{1 - r} \cdot (1 - 0) = \frac{a}{1 - r}$$

故级数收敛，和为 $\dfrac{a}{1 - r}$。

**情形 2：$|r| \geq 1$**。分三种子情形：
- 若 $r = 1$：$S_n = a + a + \cdots + a = na$，当 $n \to \infty$ 时 $S_n \to \pm\infty$（取决于 $a$ 的符号），故发散。
- 若 $r = -1$：$S_n = a - a + a - a + \cdots$，$S_n = \begin{cases} a, & n \text{ 为奇数} \\ 0, & n \text{ 为偶数} \end{cases}$，$\{S_n\}$ 振荡不收敛，故发散。
- 若 $|r| > 1$：此时 $\lim |r|^n = \infty$，$\lim r^n$ 不存在（当 $r < -1$ 时振荡）或不等于 $0$，由定理 9.1 的逆否命题，通项 $ar^{n-1}$ 不趋于 $0$，级数发散。

综合即得定理结论。证毕。

**例 6**：判断级数 $\sum_{n=1}^{\infty} \frac{2^n}{3^{n-1}}$ 的敛散性，若收敛求其和。

**解**：先将通项化为等比级数的标准形式：
$$a_n = \frac{2^n}{3^{n-1}} = 3 \cdot \frac{2^n}{3^n} = 3 \cdot \left(\frac{2}{3}\right)^n$$

因此：
$$\sum_{n=1}^{\infty} a_n = \sum_{n=1}^{\infty} 3 \cdot \left(\frac{2}{3}\right)^n = 3 \sum_{n=1}^{\infty} \left(\frac{2}{3}\right)^n$$

这是一个等比级数，首项 $a = 3 \cdot \frac{2}{3} = 2$，公比 $r = \frac{2}{3}$。注意求和从 $n=1$ 开始，所以首项对应的指数为 $1$。

直接应用定理 9.2：$|r| = \frac{2}{3} < 1$，级数收敛。和为：
$$\sum_{n=1}^{\infty} 3 \cdot \left(\frac{2}{3}\right)^n = 3 \cdot \frac{\frac{2}{3}}{1 - \frac{2}{3}} = 3 \cdot \frac{\frac{2}{3}}{\frac13} = 3 \cdot 2 = 6$$

**验证**：部分和 $S_n = 6\left(1 - \left(\frac{2}{3}\right)^n\right)$（由等比数列求和公式），当 $n=5$ 时 $S_5 = 6(1 - (32/243)) = 6 \times (211/243) \approx 5.2099$，$n=10$ 时 $S_10 \approx 5.939$，逐渐逼近 $6$。

## 5. 例题

### 例题 1：裂项相消法求部分和与和

给定数列 $a_n = \dfrac{1}{(2n-1)(2n+1)}$（$n = 1, 2, 3, \dots$）。记 $S_n = \sum_{k=1}^{n} a_k$。
(1) 将 $a_n$ 裂项，写成 $\dfrac{c}{2n-1} + \dfrac{d}{2n+1}$ 的形式。
(2) 求 $S_n$ 的闭式。
(3) 判断级数 $\sum_{n=1}^{\infty} a_n$ 的敛散性，若收敛求其和。

**解**：

**(1) 部分分式分解**：

设 $\dfrac{1}{(2n-1)(2n+1)} = \dfrac{c}{2n-1} + \dfrac{d}{2n+1}$。

通分右端：
$$\frac{c}{2n-1} + \frac{d}{2n+1} = \frac{c(2n+1) + d(2n-1)}{(2n-1)(2n+1)} = \frac{(2c+2d)n + (c-d)}{(2n-1)(2n+1)}$$

比较分子得：
$$\begin{cases}
2c + 2d = 0 \\
c - d = 1
\end{cases}
\implies \begin{cases}
c + d = 0 \\
c - d = 1
\end{cases}
\implies c = \frac12,\ d = -\frac12$$

因此：
$$\boxed{\frac{1}{(2n-1)(2n+1)} = \frac12\left(\frac{1}{2n-1} - \frac{1}{2n+1}\right)}$$

**(2) 求 $S_n$ 的闭式**：

$$S_n = \sum_{k=1}^{n} \frac12\left(\frac{1}{2k-1} - \frac{1}{2k+1}\right) = \frac12\sum_{k=1}^{n} \left(\frac{1}{2k-1} - \frac{1}{2k+1}\right)$$

展开：
$$
\begin{aligned}
\sum_{k=1}^{n} \left(\frac{1}{2k-1} - \frac{1}{2k+1}\right) =& \left(\frac{1}{1} - \frac{1}{3}\right) + \left(\frac{1}{3} - \frac{1}{5}\right) \\
&+ \left(\frac{1}{5} - \frac{1}{7}\right) + \cdots + \left(\frac{1}{2n-1} - \frac{1}{2n+1}\right)
\end{aligned}
$$

$-\frac13$ 与 $+\frac13$ 抵消，$-\frac15$ 与 $+\frac15$ 抵消，依此类推。幸存项为 $\frac11$ 和 $-\frac{1}{2n+1}$。

因此：
$$\sum_{k=1}^{n} \left(\frac{1}{2k-1} - \frac{1}{2k+1}\right) = 1 - \frac{1}{2n+1}$$

$$\boxed{S_n = \frac12\left(1 - \frac{1}{2n+1}\right) = \frac12 - \frac{1}{2(2n+1)}}$$

**(3) 判断敛散性**：

$$\lim_{n\to\infty} S_n = \frac12 - 0 = \frac12$$

因此级数收敛，和为 $\dfrac12$。

### 例题 2：用必要条件判断发散

判断级数 $\sum_{n=1}^{\infty} \frac{3n^2 + 1}{2n^2 - n + 3}$ 的敛散性。

**解**：

计算通项的极限：
$$\lim_{n\to\infty} \frac{3n^2 + 1}{2n^2 - n + 3} = \lim_{n\to\infty} \frac{3 + \frac{1}{n^2}}{2 - \frac{1}{n} + \frac{3}{n^2}} = \frac{3}{2} \neq 0$$

（分子分母同除以 $n^2$，由文件 09 有理函数极限的基本模式。）

由定理 9.1 的逆否命题：若 $\lim a_n \neq 0$，则级数 $\sum a_n$ 必发散。因此该级数发散。

**深入分析**：不仅通项趋于 $\frac32$，而且对充分大的 $n$，$a_n > 1$，因此部分和 $S_n > n$ 无界，发散到 $+\infty$。

### 例题 3：等比级数敛散性判别

判断下列级数的敛散性，若收敛求其和。

(1) $\displaystyle\sum_{n=1}^{\infty} \frac{3^{n+1}}{2^{n+2}}$

(2) $\displaystyle\sum_{n=0}^{\infty} \left(-\frac{2}{3}\right)^n$

**解**：

**(1)** $\displaystyle\sum_{n=1}^{\infty} \frac{3^{n+1}}{2^{n+2}}$

先化简通项：
$$a_n = \frac{3^{n+1}}{2^{n+2}} = \frac{3 \cdot 3^n}{4 \cdot 2^n} = \frac34 \cdot \left(\frac{3}{2}\right)^n$$

公比 $r = \frac{3}{2} > 1$，由定理 9.2，等比级数发散。

（也可由定理 9.1 的逆否命题：$\lim a_n = \lim \frac34 (3/2)^n = \infty \neq 0$，发散。）

**(2)** $\displaystyle\sum_{n=0}^{\infty} \left(-\frac{2}{3}\right)^n$

这是标准的等比级数形式，首项 $a = 1$（$n=0$ 时 $(-2/3)^0 = 1$），公比 $r = -\frac{2}{3}$。

$|r| = \frac{2}{3} < 1$，由定理 9.2（注意 $n$ 从 $0$ 开始）：
$$\sum_{n=0}^{\infty} \left(-\frac{2}{3}\right)^n = \frac{1}{1 - (-\frac{2}{3})} = \frac{1}{\frac{5}{3}} = \frac{3}{5}$$

级数收敛，和为 $\dfrac35$。

**验证**：部分和 $S_5 = 1 - \frac23 + \frac49 - \frac{8}{27} + \frac{16}{81} \approx 0.6049$，$S_{10} \approx 0.598$，趋于 $0.6$。

### 例题 4：通项趋于零的非充分性——综合判断

判断以下说法是否正确，并说明理由。

(1) "若 $\lim a_n = 0$，则级数 $\sum a_n$ 收敛。"

(2) "若 $\lim a_n = 0$，则级数 $\sum a_n$ 可能收敛也可能发散。"

(3) 对于级数 $\sum_{n=1}^{\infty} \frac{1}{\sqrt{n}}$，有 $\lim a_n = 0$，但该级数发散。这说明什么？

**解**：

**(1) 错误**。$\lim a_n = 0$ 是级数收敛的必要条件（定理 9.1），但不是充分条件。反例：调和级数 $\sum 1/n$ 通项趋于 $0$ 但发散。

**(2) 正确**。$\lim a_n = 0$ 只是"门槛"条件——满足它只是级数可能收敛的必要前提，是否真正收敛需要进一步判断。

**(3)** 级数 $\sum_{n=1}^{\infty} \frac{1}{\sqrt{n}}$ 的通项 $a_n = \frac{1}{\sqrt{n}} \to 0$，但部分和 $S_n \geq \sqrt{n} \to \infty$，因此发散。这说明即使通项趋于 $0$，如果它"衰减得不够快"，部分和仍然可能发散到无穷。这引出了后续章节的一个重要问题：通项需要衰减得多快才能保证级数收敛？

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 级数 $\sum a_n$ 收敛当且仅当通项 $a_n \to 0$ | $\lim a_n = 0$ 只是必要条件，不是充分条件。$a_n \to 0$ 时级数可能收敛也可能发散 |
| 调和级数 $\sum 1/n$ 发散是因为 $\lim 1/n \neq 0$（"太小了所以发散"这种直觉） | 恰恰相反，$\lim 1/n = 0$（它趋近于 0），但级数仍然发散。发散的原因是通项衰减得不够快——$1/n$ 的衰减速度不足以使无穷和有限 |
| 级数的和 $\sum a_n$ 就是简单地"把所有项加起来" | 无穷和不能直接"加完"。级数的和是通过部分和数列的极限来定义的：$\sum a_n = \lim S_n$。无限项相加不是有限项相加的简单推广 |
| 裂项 $\frac{1}{n(n+2)} = \frac{1}{n} - \frac{1}{n+2}$ 时忘记乘以 $\frac12$ | 正确的部分分式分解结果应是 $\frac12(\frac1n - \frac1{n+2})$。因为 $\frac1n - \frac1{n+2} = \frac{2}{n(n+2)}$，与原式相差因子 $2$ |
| 等比级数 $\sum ar^{n-1}$ 收敛 $\iff$ $|r| \leq 1$（认为 $r = 1$ 时也收敛） | $r = 1$ 时 $S_n = na \to \infty$，发散。收敛条件是严格 $|r| < 1$ |
| 等比级数的和 $\frac{a}{1-r}$ 可以直接用于任意 $r$ | 公式 $\sum ar^{n-1} = \frac{a}{1-r}$ 只有在 $|r| < 1$ 时才成立。当 $|r| \geq 1$ 时级数发散，该表达式无意义 |
| 部分和 $S_n$ 的极限存在且有限 $\iff$ 级数收敛，因此 $S_n \to \infty$ 也是"一种收敛" | $\infty$ 不是实数，$S_n \to \infty$ 时极限不存在（在实数意义下），因此级数发散。分析学中"收敛"特指极限为有限实数 |
| 若 $\lim a_n = 0$，则 $\lim S_n$ 存在 | 这是错误的三段论。$S_n$ 是累积和，即使每项都很小，累积效果可能无限大。调和级数就是典型例子 |

**1.** 写出下列级数的前 3 项部分和 $S_1, S_2, S_3$，并猜测其敛散性。

(1) $\displaystyle\sum_{n=1}^{\infty} \frac{1}{n^2}$

(2) $\displaystyle\sum_{n=1}^{\infty} (-1)^{n+1}$

(3) $\displaystyle\sum_{n=1}^{\infty} \frac{1}{2^n}$

(4) $\displaystyle\sum_{n=1}^{\infty} \frac{n}{n+1}$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\sum_{n=1}^{\infty} \frac{1}{n^2}$

$S_1 = 1$，$S_2 = 1 + \frac14 = \frac54 = 1.25$，$S_3 = 1 + \frac14 + \frac19 = \frac{49}{36} \approx 1.361$。

猜测：收敛。因为 $\sum 1/n$ 发散（衰减太慢），而 $\sum 1/n^2$ 的每项比 $\sum 1/n$ 衰减更快（$1/n^2$ 比 $1/n$ 小），文件 10 例 3 已用 Cauchy 准则证明了 $\sum 1/k^2$ 的部分和收敛。

**(2)** $\displaystyle\sum_{n=1}^{\infty} (-1)^{n+1}$

$S_1 = 1$，$S_2 = 1 - 1 = 0$，$S_3 = 1 - 1 + 1 = 1$。

猜测：发散。部分和在 $0$ 和 $1$ 之间振荡，不收敛。通项 $(-1)^{n+1}$ 不趋于 $0$（它在 $\pm 1$ 间振荡），由定理 9.1 逆否命题知发散。

**(3)** $\displaystyle\sum_{n=1}^{\infty} \frac{1}{2^n}$

$S_1 = \frac12$，$S_2 = \frac12 + \frac14 = \frac34$，$S_3 = \frac12 + \frac14 + \frac18 = \frac78$。

猜测：收敛，和为 $1$。这是公比 $r = \frac12$ 的等比级数，$\sum_{n=1}^{\infty} \frac{1}{2^n} = \frac{1/2}{1-1/2} = 1$。

**(4)** $\displaystyle\sum_{n=1}^{\infty} \frac{n}{n+1}$

$S_1 = \frac12$，$S_2 = \frac12 + \frac23 = \frac76 \approx 1.167$，$S_3 = \frac12 + \frac23 + \frac34 = \frac{23}{12} \approx 1.917$。

猜测：发散。通项 $\frac{n}{n+1} \to 1 \neq 0$，由定理 9.1 逆否命题知发散。

**2.** 判断下列级数是否收敛（利用必要条件或裂项求和）：

(1) $\displaystyle\sum_{n=2}^{\infty} \frac{2n+3}{n-1}$

(2) $\displaystyle\sum_{n=1}^{\infty} \frac{1}{(n+1)(n+3)}$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\sum_{n=2}^{\infty} \frac{2n+3}{n-1}$

通项极限：$\lim_{n\to\infty} \frac{2n+3}{n-1} = \lim_{n\to\infty} \frac{2 + \frac{3}{n}}{1 - \frac{1}{n}} = 2 \neq 0$。

由必要条件逆否命题：通项不趋于 $0$，级数发散。

**注意**：虽然 $n=1$ 时分母为 $0$（第一项无定义），但级数的收敛性只关心 $n$ 充分大之后的行为。我们可以从 $n=2$ 开始求和，不影响发散结论。

**(2)** $\displaystyle\sum_{n=1}^{\infty} \frac{1}{(n+1)(n+3)}$

先裂项：设 $\frac{1}{(n+1)(n+3)} = \frac{c}{n+1} + \frac{d}{n+3}$。

通分：$\frac{c(n+3) + d(n+1)}{(n+1)(n+3)} = \frac{(c+d)n + (3c+d)}{(n+1)(n+3)}$。

比较分子得：
$$\begin{cases}
c + d = 0 \\
3c + d = 1
\end{cases}
\implies \begin{cases}
c + d = 0 \\
2c = 1
\end{cases}
\implies c = \frac12,\ d = -\frac12$$

因此 $\frac{1}{(n+1)(n+3)} = \frac12\left(\frac{1}{n+1} - \frac{1}{n+3}\right)$。

部分和：
$$
\begin{aligned}
S_n &= \frac12\sum_{k=1}^{n} \left(\frac{1}{k+1} - \frac{1}{k+3}\right) \\
&= \frac12\Big[\left(\frac12 - \frac14\right) + \left(\frac13 - \frac15\right) + \left(\frac14 - \frac16\right) + \cdots \\
&\qquad\quad + \left(\frac{1}{n} - \frac{1}{n+2}\right) + \left(\frac{1}{n+1} - \frac{1}{n+3}\right)\Big]
\end{aligned}
$$

幸存项为：$\frac12 + \frac13 - \frac{1}{n+2} - \frac{1}{n+3}$。

因此：
$$S_n = \frac12\left(\frac12 + \frac13 - \frac{1}{n+2} - \frac{1}{n+3}\right) = \frac{5}{12} - \frac12\left(\frac{1}{n+2} + \frac{1}{n+3}\right)$$

$\lim S_n = \frac{5}{12}$，级数收敛于 $\frac{5}{12}$。

</details>

**3.** 将下列等比级数化为标准形式，判断敛散性；若收敛，求其和。

(1) $\displaystyle\sum_{n=1}^{\infty} \frac{2^{n-1}}{3^n}$

(2) $\displaystyle\sum_{n=0}^{\infty} \frac{(-3)^n}{4^{n+1}}$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\sum_{n=1}^{\infty} \frac{2^{n-1}}{3^n}$

化简通项：
$$a_n = \frac{2^{n-1}}{3^n} = \frac{1}{3} \cdot \frac{2^{n-1}}{3^{n-1}} = \frac13 \cdot \left(\frac{2}{3}\right)^{n-1}$$

这是首项 $a = \frac13$、公比 $r = \frac23$ 的等比级数。$|r| = \frac23 < 1$，收敛。

和：
$$\sum_{n=1}^{\infty} \frac{2^{n-1}}{3^n} = \frac{\frac13}{1 - \frac23} = \frac{\frac13}{\frac13} = 1$$

**(2)** $\displaystyle\sum_{n=0}^{\infty} \frac{(-3)^n}{4^{n+1}}$

化简通项：
$$a_n = \frac{(-3)^n}{4^{n+1}} = \frac14 \cdot \frac{(-3)^n}{4^n} = \frac14 \cdot \left(-\frac34\right)^n$$

这是首项 $a = \frac14$、公比 $r = -\frac34$ 的等比级数。$|r| = \frac34 < 1$，收敛。

和：
$$\sum_{n=0}^{\infty} \frac{(-3)^n}{4^{n+1}} = \frac{\frac14}{1 - (-\frac34)} = \frac{\frac14}{\frac74} = \frac17$$

</details>

### 迁移应用

**4.** 已知 $\displaystyle\sum_{n=1}^{\infty} \frac{1}{(n+a)(n+b)}$（$a, b$ 为常数，$a < b$ 且均为正整数）是收敛的。试求其和（用 $a, b$ 表示）。

<details><summary>参考答案</summary>

**解**：

先裂项：
$$\frac{1}{(n+a)(n+b)} = \frac{1}{b-a}\left(\frac{1}{n+a} - \frac{1}{n+b}\right)$$

验证：右端通分 $\frac{1}{b-a} \cdot \frac{(n+b) - (n+a)}{(n+a)(n+b)} = \frac{1}{b-a} \cdot \frac{b-a}{(n+a)(n+b)} = \frac{1}{(n+a)(n+b)}$。

部分和：
$$S_n = \frac{1}{b-a} \sum_{k=1}^{n} \left(\frac{1}{k+a} - \frac{1}{k+b}\right)$$

展开，观察到从 $k=1$ 到 $k=n$ 的求和中，$-\frac{1}{a+1}$ 与 $+\frac{1}{a+1}$ 等项会抵消。详细展开：
$$
\begin{aligned}
\sum_{k=1}^{n} \left(\frac{1}{k+a} - \frac{1}{k+b}\right) =& \left(\frac{1}{1+a} - \frac{1}{1+b}\right) + \left(\frac{1}{2+a} - \frac{1}{2+b}\right) \\
&+ \left(\frac{1}{3+a} - \frac{1}{3+b}\right) + \cdots + \left(\frac{1}{n+a} - \frac{1}{n+b}\right)
\end{aligned}
$$

观察抵消模式：对于 $k = 1, 2, \dots, n-b$，负项会被后面的正项抵消（假设 $n$ 足够大）。最终幸存项为：
- 正项：$\frac{1}{1+a} + \frac{1}{2+a} + \cdots + \frac{1}{b}$（前 $b-a$ 个正项）
- 负项：$-\frac{1}{n+a+1} - \frac{1}{n+a+2} - \cdots - \frac{1}{n+b}$（最后 $b-a$ 个负项）

更简洁地：
$$\sum_{k=1}^{n} \left(\frac{1}{k+a} - \frac{1}{k+b}\right) = \sum_{j=1}^{b-a} \frac{1}{j+a} - \sum_{j=1}^{b-a} \frac{1}{n + a + j}$$

取极限 $n \to \infty$，第二项和 $\to 0$，因此：
$$\lim_{n\to\infty} S_n = \frac{1}{b-a} \sum_{j=1}^{b-a} \frac{1}{j+a}$$

特别地：
- 若 $a=0, b=1$：和为 $\frac{1}{1-0} \cdot \frac{1}{0+1} = 1$（即例 4）
- 若 $a=0, b=2$：和为 $\frac12\left(\frac11 + \frac12\right) = \frac34$（即例 5）
- 若 $a=1, b=3$：和为 $\frac12\left(\frac12 + \frac13\right) = \frac{5}{12}$（练习题 2(2)）

</details>

**5.** （综合）设数列 $\{a_n\}$ 的通项为 $a_n = \sqrt{n+1} - \sqrt{n}$。求级数 $\sum_{n=1}^{\infty} a_n$ 的部分和 $S_n$，并判断其敛散性。

<details><summary>参考答案</summary>

**解**：

**第 1 步：观察裂项的可能性**。通项本身已经是"差"的形式——这正是裂项求和的理想情形。

$$a_n = \sqrt{n+1} - \sqrt{n}$$

**第 2 步：求部分和**。
$$S_n = \sum_{k=1}^{n} (\sqrt{k+1} - \sqrt{k})$$

展开：
$$
\begin{aligned}
S_n = &(\sqrt{2} - \sqrt{1}) + (\sqrt{3} - \sqrt{2}) + (\sqrt{4} - \sqrt{3}) \\
&+ \cdots + (\sqrt{n} - \sqrt{n-1}) + (\sqrt{n+1} - \sqrt{n})
\end{aligned}
$$

**第 3 步：抵消**。$\sqrt{2}$ 与 $-\sqrt{2}$ 抵消，$\sqrt{3}$ 与 $-\sqrt{3}$ 抵消，$\dots$，$\sqrt{n}$ 与 $-\sqrt{n}$ 抵消。

**第 4 步：幸存项**。只剩最开始的 $-\sqrt{1}$ 和最后的 $\sqrt{n+1}$：
$$S_n = \sqrt{n+1} - 1$$

**第 5 步：判断敛散性**。$\lim_{n\to\infty} S_n = \lim_{n\to\infty} (\sqrt{n+1} - 1) = +\infty$。因此级数发散。

**验证必要条件**：通项 $a_n = \sqrt{n+1} - \sqrt{n} = \frac{1}{\sqrt{n+1} + \sqrt{n}} \to 0$（分子有理化后易见）。但级数仍发散，这又是一个 $a_n \to 0$ 但级数发散的例子。

</details>

**6.** （挑战）证明：若 $\sum_{n=1}^{\infty} a_n$ 收敛，则 $\sum_{n=1}^{\infty} (a_n + a_{n+1})$ 也收敛。反之是否成立？

<details><summary>参考答案</summary>

**证明**：

设 $\sum a_n$ 收敛于 $S$，其部分和为 $S_n = \sum_{k=1}^{n} a_k$，$\lim S_n = S$。

考虑新级数 $\sum_{n=1}^{\infty} (a_n + a_{n+1})$ 的部分和 $T_n$：
$$
\begin{aligned}
T_n &= \sum_{k=1}^{n} (a_k + a_{k+1}) \\
&= (a_1 + a_2) + (a_2 + a_3) + \cdots + (a_n + a_{n+1}) \\
&= a_1 + 2(a_2 + a_3 + \cdots + a_n) + a_{n+1}
\end{aligned}
$$

这个表达式可以写成 $T_n = a_1 + 2(S_n - a_1) + a_{n+1} = 2S_n - a_1 + a_{n+1}$。

由 $\sum a_n$ 收敛知 $\lim S_n = S$ 且 $\lim a_{n+1} = 0$（定理 9.1），因此：
$$\lim T_n = 2S - a_1 + 0 = 2S - a_1$$

极限存在且有限，所以 $\sum_{n=1}^{\infty} (a_n + a_{n+1})$ 收敛。

**反之不成立**。反例：令 $a_n = (-1)^{n+1}$，则 $a_n + a_{n+1} = (-1)^{n+1} + (-1)^{n+2} = 0$，因此 $\sum (a_n + a_{n+1})$ 收敛于 $0$。但 $\sum a_n$ 的部分和在 $0$ 和 $1$ 之间振荡，发散（且 $\lim a_n \neq 0$ 不满足必要条件）。

**更本质的反例**：令 $a_n = (-1)^{n+1}$。则 $a_n + a_{n+1} = 0$ 恒成立，新级数显然收敛。但原级数 $\sum (-1)^{n+1}$ 发散。

因此 $\sum a_n$ 收敛 $\Rightarrow$ $\sum (a_n + a_{n+1})$ 收敛，但逆命题不成立。
</details>
