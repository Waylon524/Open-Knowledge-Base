## 2. 背景与应用场景

### 2.1 从质量到质心：一阶矩的物理含义

在文件06中，我们学习了用三重积分计算物体的总质量 $M = \iiint \rho\,dV$（定理13.17）。但总质量只告诉我们"有多少"，而没有告诉我们"分布在哪里"。

考虑一个实验：用一根手指托起一块形状不规则的木板。只有当手指放在某个特定点时，木板才能保持平衡——这个点就是**质心**（center of mass），也称**重心**。质心是"质量分布的平均位置"，是物体在重力场中的等效受力点。

在工程与科学中，质心具有重要的实际意义：
- **航空航天**：飞行器的质心决定了其飞行稳定性——质心必须位于空气动力中心之前才能保持静稳定
- **机械设计**：旋转机械（如飞轮、涡轮）的质心必须在转轴上，否则会产生剧烈振动
- **生物力学**：人体质心随姿态变化，分析运动时必须追踪质心位置
- **天体物理**：双星系统绕共同的质心旋转

### 2.2 从一阶矩到二阶矩的过渡

将文件06（转动惯量）与本节（质心）进行对比，可以清晰地看到"矩"的层次：

| 矩的阶数 | 名称 | 积分形式 | 物理含义 |
|:---|:---|:---|:---|
| 零阶矩 | 质量 | $\displaystyle \iiint \rho\,dV$ | 总量的度量 |
| **一阶矩** | **静矩** | $\displaystyle \iiint x\rho\,dV$ | **质心位置** |
| 二阶矩 | 转动惯量 | $\displaystyle \iiint r^2\rho\,dV$ | 旋转惯性 |

零阶矩（质量）只涉及密度本身。一阶矩（静矩）是密度乘以坐标的一次方，刻画了"质量在哪个方向上更多"。二阶矩（转动惯量）是密度乘以坐标的平方，放大了远离转轴的质量的贡献。这三个层次的矩构成了完整描述质量分布的数学工具。

### 2.3 连续质量分布引力的科学意义

牛顿万有引力定律描述了两个质点之间的引力，但宇宙中的天体（地球、太阳、星系）都是连续质量分布。要计算一个天体对远处质点的引力，不能简单将天体视为全部质量集中于某点的质点——这个近似仅在球对称分布时才严格成立。对于一般形状的物体，必须通过积分来精确计算引力。

连续质量分布的引力计算在以下领域有重要应用：
- **地球物理勘探**：通过测量重力异常推断地下密度分布
- **卫星轨道力学**：地球非球形引力摄动（由于地球并非完美球体）影响卫星轨道
- **天体物理学**：计算不均匀天体对周围物体的引力

## 3. 核心概念与符号约定

### 3.1 平面薄片的静矩与质心

**物理直觉**：考虑一个平面薄片（位于 $xy$ 平面内），其质量分布不均匀。当我们想用一个点来代表整个薄片的位置时，这个点应该是"质量加权平均位置"，即：

$$\bar{x} = \frac{\text{各质点 } x\text{ 坐标按质量加权平均}}{\text{总质量}},\qquad \bar{y} = \frac{\text{各质点 } y\text{ 坐标按质量加权平均}}{\text{总质量}}$$

对于离散质点系统 $\{m_i\}$ 在点 $(x_i, y_i)$，质心坐标为：

$$\bar{x} = \frac{\sum m_i x_i}{\sum m_i},\qquad \bar{y} = \frac{\sum m_i y_i}{\sum m_i}$$

对于连续分布，求和变为积分。

**定义 13.19（平面薄片的静矩与质心 / First Moment and Centroid of Planar Lamina）**：设平面薄片占据区域 $D \subset \mathbb{R}^2$，面密度为 $\rho(x,y)$（在 $D$ 上连续）。则：

- 薄片对 $y$ 轴的静矩（一阶矩）为：
  $$\boxed{M_y = \iint_D x\,\rho(x,y)\,dA}$$

- 薄片对 $x$ 轴的静矩（一阶矩）为：
  $$\boxed{M_x = \iint_D y\,\rho(x,y)\,dA}$$

- 薄片的总质量为 $M = \iint_D \rho(x,y)\,dA$（定理13.17的二维版本）。

- 薄片的质心坐标 $(\bar{x}, \bar{y})$ 为：
  $$\boxed{\bar{x} = \frac{M_y}{M} = \frac{\displaystyle\iint_D x\rho(x,y)\,dA}{\displaystyle\iint_D \rho(x,y)\,dA},\qquad 
    \bar{y} = \frac{M_x}{M} = \frac{\displaystyle\iint_D y\rho(x,y)\,dA}{\displaystyle\iint_D \rho(x,y)\,dA}}$$

**静矩的物理含义**：静矩中的"矩"（moment）一词来自拉丁语 momentum，意为"推动的力"。$M_y$ 度量的是"质量在 $x$ 方向上的分布不平衡程度"——如果 $M_y$ 很大，说明大部分质量集中在 $x$ 较大的区域，因此质心的 $x$ 坐标也较大。

**均匀密度情形（形心）**：当密度为常数 $\rho(x,y) = \rho_0$ 时，$\rho_0$ 可以从分子和分母中同时约去：

$$\boxed{\bar{x} = \frac{\displaystyle\iint_D x\,dA}{\displaystyle\iint_D 1\,dA} = \frac{\displaystyle\iint_D x\,dA}{A(D)},\qquad
  \bar{y} = \frac{\displaystyle\iint_D y\,dA}{\displaystyle\iint_D 1\,dA} = \frac{\displaystyle\iint_D y\,dA}{A(D)}}$$

其中 $A(D) = \iint_D 1\,dA$ 是区域 $D$ 的面积。此时质心完全由区域的几何形状决定，称为**形心**（centroid）。换句话说，均匀薄片的质心就是其所在区域的形心。

### 3.2 三维物体的静矩与质心

对三维物体的质心，概念完全类似：质心是"质量在三维空间中的加权平均位置"。

**定义 13.20（三维物体的静矩与质心 / First Moment and Centroid of 3D Body）**：设三维物体 $\Omega \subset \mathbb{R}^3$ 的体密度为 $\rho(x,y,z)$（在 $\Omega$ 上连续）。则：

- 物体对 $yz$ 平面（即 $x=0$ 平面）的静矩为：
  $$\boxed{M_{yz} = \iiint_\Omega x\,\rho(x,y,z)\,dV}$$

- 物体对 $xz$ 平面（即 $y=0$ 平面）的静矩为：
  $$\boxed{M_{xz} = \iiint_\Omega y\,\rho(x,y,z)\,dV}$$

- 物体对 $xy$ 平面（即 $z=0$ 平面）的静矩为：
  $$\boxed{M_{xy} = \iiint_\Omega z\,\rho(x,y,z)\,dV}$$

- 物体的质心坐标 $(\bar{x}, \bar{y}, \bar{z})$ 为：
  $$\boxed{\bar{x} = \frac{M_{yz}}{M} = \frac{\displaystyle\iiint_\Omega x\rho\,dV}{\displaystyle\iiint_\Omega \rho\,dV},\qquad
    \bar{y} = \frac{M_{xz}}{M} = \frac{\displaystyle\iiint_\Omega y\rho\,dV}{\displaystyle\iiint_\Omega \rho\,dV},\qquad
    \bar{z} = \frac{M_{xy}}{M} = \frac{\displaystyle\iiint_\Omega z\rho\,dV}{\displaystyle\iiint_\Omega \rho\,dV}}$$

其中 $M = \iiint_\Omega \rho\,dV$ 是物体的总质量（定理13.17）。

**均匀密度情形**：当 $\rho(x,y,z) = \rho_0$（常数）时：

$$\boxed{\bar{x} = \frac{\displaystyle\iiint_\Omega x\,dV}{V(\Omega)},\qquad
  \bar{y} = \frac{\displaystyle\iiint_\Omega y\,dV}{V(\Omega)},\qquad
  \bar{z} = \frac{\displaystyle\iiint_\Omega z\,dV}{V(\Omega)}}$$

其中 $V(\Omega) = \iiint_\Omega 1\,dV$ 是 $\Omega$ 的体积。此时质心即为形心。

**质心的向量形式**：有时将三个分量统一写成向量形式更方便：

$$\boxed{\bar{\mathbf{r}} = \frac{1}{M} \iiint_\Omega \mathbf{r}\,\rho(\mathbf{r})\,dV}$$

其中 $\mathbf{r} = (x,y,z)$，$\bar{\mathbf{r}} = (\bar{x},\bar{y},\bar{z})$。这个形式强调了质心是位置向量的加权平均。

### 3.3 连续质量分布对质点的引力

**万有引力定律复习**：两个质点 $m_1$ 和 $m_2$，相距 $r$，之间的引力大小为：

$$F = G\,\frac{m_1 m_2}{r^2}$$

引力方向沿两质点的连线，相互吸引。其中 $G \approx 6.674 \times 10^{-11} \,\text{N}\cdot\text{m}^2/\text{kg}^2$ 为万有引力常数。

**从离散到连续**：对于连续质量分布（物体 $\Omega$），先将物体分割成许多小体积元 $dV$，每个体积元的质量为 $dm = \rho\,dV$。该质量元与外部质点 $m$（位于点 $P_0$）之间的引力大小为：

$$dF = G\,\frac{m \cdot dm}{r^2} = G\,\frac{m\rho\,dV}{r^2}$$

其中 $r$ 是 $dV$ 到 $P_0$ 的距离。引力方向沿 $dV$ 与 $P_0$ 的连线。

**定义 13.21（连续质量分布的引力积分公式 / Gravitational Force of Continuous Mass Distribution）**：设物体 $\Omega \subset \mathbb{R}^3$ 的密度为 $\rho(\mathbf{r})$，外部质点 $m$ 位于 $\mathbf{r}_0$。则物体对质点的引力为向量积分：

$$\boxed{\vec{F} = \iiint_\Omega G\,\frac{m\rho(\mathbf{r})}{|\mathbf{r} - \mathbf{r}_0|^3} (\mathbf{r}_0 - \mathbf{r})\,dV}$$

**推导**：质量元 $dm = \rho\,dV$ 位于 $\mathbf{r}$ 处，对质点 $m$（位于 $\mathbf{r}_0$）的引力大小为 $dF = G\,m\,dm / |\mathbf{r} - \mathbf{r}_0|^2$，方向为从质量元指向质点的单位向量 $(\mathbf{r}_0 - \mathbf{r}) / |\mathbf{r}_0 - \mathbf{r}|$。因此：

$$d\vec{F} = G\,\frac{m\,\rho\,dV}{|\mathbf{r} - \mathbf{r}_0|^2} \cdot \frac{\mathbf{r}_0 - \mathbf{r}}{|\mathbf{r}_0 - \mathbf{r}|}
    = G\,\frac{m\rho(\mathbf{r})}{|\mathbf{r} - \mathbf{r}_0|^3} (\mathbf{r}_0 - \mathbf{r})\,dV$$

**实际计算中**，通常将 $\vec{F}$ 分解为 $x$、$y$、$z$ 方向的分量分别积分。若物体和质点具有对称性，可大大简化。

**二维平面中的引力积分**：当物体是平面薄片 $D$（位于 $xy$ 平面内），面密度为 $\sigma(x,y)$，质点 $m$ 位于 $(0,0,b)$（$b>0$）时，薄片对质点的引力大小为：

$$dF = G\,\frac{m\sigma(x,y)}{x^2 + y^2 + b^2}\,dA$$

$z$ 方向分量为 $dF_z = dF \cdot \dfrac{b}{\sqrt{x^2+y^2+b^2}}$，因此总 $z$ 方向引力为：

$$F_z = \iint_D G\,\frac{m\sigma(x,y)\,b}{(x^2+y^2+b^2)^{3/2}}\,dA$$

由对称性，$x$ 和 $y$ 方向的分量在圆盘情形下相互抵消，只剩 $z$ 方向分量。

## 4. 原理与方法

### 4.1 平面薄片质心计算的完整流程

**第 1 步：确定密度函数 $\rho(x,y)$**
- 若为"均匀薄片"或"密度为常数 $\rho_0$"，则 $\rho(x,y) = \rho_0$
- 若密度为变量，则保留为 $\rho(x,y)$ 的函数形式

**第 2 步：描述区域 $D$**
- 判断是 X-型区域还是 Y-型区域，或需要分割为多个子区域
- 对 X-型区域：$D = \{(x,y) \mid a \leq x \leq b,\; y_1(x) \leq y \leq y_2(x)\}$
- 对 Y-型区域：$D = \{(x,y) \mid c \leq y \leq d,\; x_1(y) \leq x \leq x_2(y)\}$

**第 3 步：计算总质量 $M$**
$$M = \iint_D \rho(x,y)\,dA$$

**第 4 步：计算静矩 $M_y$ 和 $M_x$**
$$M_y = \iint_D x\,\rho(x,y)\,dA,\qquad M_x = \iint_D y\,\rho(x,y)\,dA$$

**第 5 步：计算质心坐标**
$$\bar{x} = \frac{M_y}{M},\qquad \bar{y} = \frac{M_x}{M}$$

**第 6 步：验证**
- 检查 $\bar{x}$ 和 $\bar{y}$ 是否位于区域 $D$ 内部（物理上质心应在物体内部或边界上）
- 对均匀薄片，质心即形心，应具有与区域几何形状相符的对称性

### 4.2 三维物体质心计算的完整流程

**第 1 步：确定密度函数 $\rho(x,y,z)$**

**第 2 步：描述区域 $\Omega$**
- 选择合适的坐标系（直角坐标、柱坐标、球坐标）
- 用投影法或截面法描述区域

**第 3 步：计算总质量 $M$**
$$M = \iiint_\Omega \rho(x,y,z)\,dV$$

**第 4 步：计算三个方向的静矩**
$$M_{yz} = \iiint_\Omega x\rho\,dV,\quad M_{xz} = \iiint_\Omega y\rho\,dV,\quad M_{xy} = \iiint_\Omega z\rho\,dV$$

**第 5 步：计算质心坐标**
$$\bar{x} = \frac{M_{yz}}{M},\quad \bar{y} = \frac{M_{xz}}{M},\quad \bar{z} = \frac{M_{xy}}{M}$$

**第 6 步：验证**
- 利用对称性检查：若区域关于 $xz$ 平面对称且密度对称，则 $\bar{y}=0$；类似地，对称性可以立即得出某些坐标分量为零
- 检查量纲：静矩的量纲为 $[\text{质量}] \times [\text{长度}]$，质心的量纲为 $[\text{长度}]$

### 4.3 利用对称性简化计算

对称性是质心计算中最强大的简化工具。

**平面情形**：
- 若区域 $D$ 关于 $y$ 轴对称且密度对称，则 $\bar{x} = 0$
- 若区域 $D$ 关于 $x$ 轴对称且密度对称，则 $\bar{y} = 0$

**三维情形**：
- 若区域 $\Omega$ 关于 $yz$ 平面对称且密度对称，则 $\bar{x} = 0$
- 若区域 $\Omega$ 关于 $xz$ 平面对称且密度对称，则 $\bar{y} = 0$
- 若区域 $\Omega$ 关于 $xy$ 平面对称且密度对称，则 $\bar{z} = 0$

**原因**：对称性使得被积函数是奇函数（在对称区域上积分为零）。例如，若 $D$ 关于 $y$ 轴对称，$x$ 在对称区域上积分为零（$x$ 是奇函数），因此 $M_y = 0$。但注意：这仅在密度也是对称的情况下成立——若密度不对称（如 $\rho = x$），即使区域对称，$\bar{x}$ 也未必为零。

### 4.4 引力计算的基本方法

**第 1 步：建立坐标系**
- 将质点放在方便的位置（通常放在坐标轴或原点处）
- 将物体放在方便计算的坐标位置

**第 2 步：写出质量元 $dm$ 到质点的距离 $r$ 的表达式**

**第 3 步：写出 $dF$ 的标量大小，并确定需要哪些分量**

**第 4 步：利用对称性判断哪些分量为零**
- 例如，圆盘对其轴上质点的引力，$x$ 和 $y$ 方向分量抵消

**第 5 步：对各非零分量分别积分**

**第 6 步：验证**
- 检查量纲：$G Mm / r^2$ 的量纲为力
- 检查极限情况：$b \to \infty$ 时引力应趋于零；圆盘半径 $R \to \infty$ 时引力应趋近于无限大均匀平面对点质量的引力（一种特殊情况）

### 4.5 解题策略总结

| 情形 | 推荐方法 | 说明 |
|:---|:---|:---|
| 均匀密度平面薄片的质心 | 形心公式 $(\bar{x},\bar{y}) = (\iint x\,dA / A,\; \iint y\,dA / A)$ | 只需计算面积和一阶矩 |
| 可变密度平面薄片的质心 | 先算 $M$，再算 $M_x, M_y$，最后相除 | 密度不能约去 |
| 对称区域的质心 | 利用对称性直接得出某些坐标为零 | 大幅减少积分计算量 |
| 均匀半球体的质心 | 由对称性得 $\bar{x}=\bar{y}=0$，只需算 $\bar{z}$ | 用截面法或球坐标 |
| 一维细杆引力 | 一维定积分 | 最简单情形，建立 $dF$ 后积分 |
| 圆盘对其轴上质点的引力 | 极坐标变换 + 仅剩 $z$ 分量 | 利用对称性消去 $x,y$ 分量 |

## 5. 例题

### 例 1（对应考试题 Q1）：抛物线区域上均匀薄片的质心

设 $D$ 是由抛物线 $y = x^2$ 与直线 $y = 1$ 所围成的区域（$x \geq 0$）。一平面薄片占据区域 $D$，其面密度为常数 $\rho_0 = 1$（质量单位/面积单位）。求该薄片的质心坐标 $(\bar{x}, \bar{y})$。

**解**：

**第 1 步：描述区域**

抛物线 $y = x^2$ 与直线 $y = 1$ 的交点：$x^2 = 1 \Rightarrow x = 1$（$x \geq 0$）。区域 $D$ 描述为 X-型区域：

$$D = \big\{ (x,y) \mid 0 \leq x \leq 1,\; x^2 \leq y \leq 1 \big\}$$

**第 2 步：计算总质量 $M$**

密度为常数 $\rho_0 = 1$，总质量等于区域面积：

$$
\begin{aligned}
M &= \iint_D 1\,dA = \int_{0}^{1} \int_{x^2}^{1} 1\,dy\,dx \\
&= \int_{0}^{1} (1 - x^2)\,dx \\
&= \left[ x - \frac{x^3}{3} \right]_{0}^{1} = 1 - \frac{1}{3} = \frac{2}{3}
\end{aligned}
$$

故 $M = \dfrac{2}{3}$。

**第 3 步：计算静矩 $M_y$ 和 $M_x$**

对 $y$ 轴的静矩 $M_y = \iint_D x\,dA$（因为 $\rho_0=1$）：

$$
\begin{aligned}
M_y &= \int_{0}^{1} \int_{x^2}^{1} x\,dy\,dx = \int_{0}^{1} x(1 - x^2)\,dx \\
&= \int_{0}^{1} (x - x^3)\,dx = \left[ \frac{x^2}{2} - \frac{x^4}{4} \right]_{0}^{1} = \frac{1}{2} - \frac{1}{4} = \frac{1}{4}
\end{aligned}
$$

对 $x$ 轴的静矩 $M_x = \iint_D y\,dA$：

$$
\begin{aligned}
M_x &= \int_{0}^{1} \int_{x^2}^{1} y\,dy\,dx = \int_{0}^{1} \left[ \frac{y^2}{2} \right]_{x^2}^{1} dx \\
&= \int_{0}^{1} \left( \frac{1}{2} - \frac{x^4}{2} \right) dx = \frac{1}{2} \int_{0}^{1} (1 - x^4)\,dx \\
&= \frac{1}{2} \left[ x - \frac{x^5}{5} \right]_{0}^{1} = \frac{1}{2} \left(1 - \frac{1}{5}\right) = \frac{1}{2} \cdot \frac{4}{5} = \frac{2}{5}
\end{aligned}
$$

故 $M_y = \dfrac{1}{4}$，$M_x = \dfrac{2}{5}$。

**第 4 步：计算质心坐标**

由定义13.19（平面薄片的质心公式）：

$$\bar{x} = \frac{M_y}{M} = \frac{1/4}{2/3} = \frac{1}{4} \cdot \frac{3}{2} = \frac{3}{8}$$

$$\bar{y} = \frac{M_x}{M} = \frac{2/5}{2/3} = \frac{2}{5} \cdot \frac{3}{2} = \frac{3}{5}$$

故该薄片的质心坐标为 $\displaystyle (\bar{x}, \bar{y}) = \left( \frac{3}{8},\; \frac{3}{5} \right)$。

**验证**：通过观察区域形状可以判断质心是否大致合理。该区域在 $[0,1] \times [0,1]$ 内，抛物线上方的部分质量集中于 $(0.5, 0.7)$ 附近。计算得 $\bar{x}=0.375$、$\bar{y}=0.6$，符合直观预期。$\checkmark$

### 例 2（对应考试题 Q2）：均匀实心半球体的质心

设 $\Omega$ 是均匀实心半球体 $x^2 + y^2 + z^2 \leq R^2$、$z \geq 0$，体密度为常数 $\rho_0 = 1$。求该半球体的质心坐标 $(\bar{x}, \bar{y}, \bar{z})$。

**解**：

**第 1 步：利用对称性确定 $\bar{x}$ 和 $\bar{y}$**

半球体关于 $yz$ 平面对称（即 $x \to -x$ 对称），且密度均匀。因此被积函数 $x$ 在对称区域上积分为零：

$$\iiint_\Omega x\,dV = 0 \Rightarrow \bar{x} = \frac{0}{M} = 0$$

同理，关于 $xz$ 平面对称可得 $\bar{y} = 0$。

因此只需计算 $\bar{z}$。

**第 2 步：计算体积 $V$（即质量 $M$）**

密度 $\rho_0 = 1$，所以 $M = V$。半球体体积为：

$$V = \frac{2}{3}\pi R^3$$

（球坐标变换验证：$V = \int_0^{2\pi} \int_0^{\pi/2} \int_0^R \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta = 2\pi \cdot 1 \cdot \frac{R^3}{3} = \frac{2}{3}\pi R^3$。）

故 $M = \dfrac{2}{3}\pi R^3$。

**第 3 步：计算 $M_{xy} = \iiint_\Omega z\,dV$**

**方法一（截面法）**：固定 $z \in [0, R]$，截面 $D(z)$ 是半径为 $\sqrt{R^2 - z^2}$ 的圆盘，面积为 $\pi(R^2 - z^2)$。由截面法（定理13.12）：

$$
\begin{aligned}
\iiint_\Omega z\,dV &= \int_{0}^{R} z \left( \iint_{D(z)} 1\,dxdy \right) dz \\
&= \int_{0}^{R} z \cdot \pi(R^2 - z^2)\,dz \\
&= \pi \int_{0}^{R} (R^2 z - z^3)\,dz \\
&= \pi \left[ \frac{R^2 z^2}{2} - \frac{z^4}{4} \right]_{0}^{R} \\
&= \pi \left( \frac{R^4}{2} - \frac{R^4}{4} \right) = \frac{\pi R^4}{4}
\end{aligned}
$$

**方法二（球坐标变换）**：
在球坐标（定义13.16）下：
$$x = \rho\sin\varphi\cos\theta,\quad y = \rho\sin\varphi\sin\theta,\quad z = \rho\cos\varphi,\quad dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$$
$$\Omega' = \big\{ (\rho,\varphi,\theta) \mid 0 \leq \rho \leq R,\; 0 \leq \varphi \leq \pi/2,\; 0 \leq \theta \leq 2\pi \big\}$$

$$
\begin{aligned}
\iiint_\Omega z\,dV &= \int_{0}^{2\pi} \int_{0}^{\pi/2} \int_{0}^{R} (\rho\cos\varphi) \cdot \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta \\
&= \left( \int_{0}^{2\pi} d\theta \right) \cdot \left( \int_{0}^{\pi/2} \cos\varphi\sin\varphi\,d\varphi \right) \cdot \left( \int_{0}^{R} \rho^3\,d\rho \right)
\end{aligned}
$$

计算各因子：

$$\int_{0}^{2\pi} d\theta = 2\pi$$

$$\int_{0}^{\pi/2} \cos\varphi\sin\varphi\,d\varphi = \frac{1}{2} \int_{0}^{\pi/2} \sin 2\varphi\,d\varphi = \frac{1}{2} \left[ -\frac{1}{2}\cos 2\varphi \right]_{0}^{\pi/2} = \frac{1}{2} \left( \frac{1}{2} + \frac{1}{2} \right) = \frac{1}{2}$$

（或直接用 $u = \sin\varphi$，$du = \cos\varphi\,d\varphi$，得 $\int_{0}^{\pi/2} \cos\varphi\sin\varphi\,d\varphi = \int_{0}^{1} u\,du = \frac{1}{2}$。）

$$\int_{0}^{R} \rho^3\,d\rho = \left[ \frac{\rho^4}{4} \right]_{0}^{R} = \frac{R^4}{4}$$

因此：
$$\iiint_\Omega z\,dV = 2\pi \cdot \frac{1}{2} \cdot \frac{R^4}{4} = \frac{\pi R^4}{4}$$

两种方法结果一致，验证了计算正确性。$\checkmark$

**第 4 步：计算 $\bar{z}$**

由定义13.20（三维物体的质心公式）：

$$\bar{z} = \frac{M_{xy}}{M} = \frac{\displaystyle\iiint_\Omega z\,dV}{V} = \frac{\pi R^4 / 4}{2\pi R^3 / 3} = \frac{\pi R^4}{4} \cdot \frac{3}{2\pi R^3} = \frac{3R}{8}$$

故该均匀实心半球体的质心坐标为 $(0, 0, \dfrac{3R}{8})$。

**验证**：从物理直觉判断，半球体的质心在对称轴上（$z$ 轴），且应位于底面 $z=0$ 和顶部 $z=R$ 之间靠近底部的位置（因为底部质量更大）。$3R/8 \approx 0.375R$，处于底面和顶部的中点 $0.5R$ 以下，符合物理直觉。$\checkmark$

### 例 3（对应考试题 Q3）：连续质量分布的引力

**问题 A（细杆引力）**：一均匀细直杆位于 $x$ 轴上，从 $x = a$ 到 $x = a + L$（$a > 0$），线密度为常数 $\lambda$（单位：$\text{kg}/\text{m}$）。原点处有一质点 $m$（$\text{kg}$）。求该细杆对质点的引力大小和方向。

**解 A**：

**第 1 步：建立微元**

位于 $x$ 处、长度为 $dx$ 的一小段的质量为 $dm = \lambda\,dx$。该小段与原点处质点 $m$ 的距离为 $x$。

**第 2 步：写出微元引力大小**

由万有引力定律，该小段对 $m$ 的引力大小为：

$$dF = G\,\frac{m \cdot dm}{x^2} = G\,\frac{m\lambda}{x^2}\,dx$$

方向沿 $x$ 轴正向（因为引力是吸引力，作用在质点 $m$ 上的方向指向杆，即 $+x$ 方向）。

**第 3 步：积分求总引力**

$$F = \int_{a}^{a+L} G\,\frac{m\lambda}{x^2}\,dx = Gm\lambda \int_{a}^{a+L} x^{-2}\,dx$$

计算积分：
$$\int_{a}^{a+L} x^{-2}\,dx = \left[ -x^{-1} \right]_{a}^{a+L} = -\frac{1}{a+L} + \frac{1}{a} = \frac{1}{a} - \frac{1}{a+L}$$

因此：
$$F = Gm\lambda \left( \frac{1}{a} - \frac{1}{a+L} \right) = \frac{Gm\lambda L}{a(a+L)}$$

故细杆对质点 $m$ 的引力大小为 $\displaystyle F = Gm\lambda \left( \frac{1}{a} - \frac{1}{a+L} \right)$，方向沿 $x$ 轴正向（从质点指向杆）。

**问题 B（圆盘引力）**：将细杆换成均匀圆盘 $x^2 + y^2 \leq R^2$（位于 $z = 0$ 平面上），面密度为常数 $\sigma$（单位：$\text{kg}/\text{m}^2$）。质点 $m$ 放在 $(0,0,b)$（$b > 0$）。求圆盘对质点的引力在 $z$ 方向的分量 $F_z$ 的二重积分表达式（用极坐标化为累次积分）。

**解 B**：

**第 1 步：分析对称性**

圆盘关于 $z$ 轴旋转对称。对于圆盘上的任意面积元 $dA$ 位于 $(x,y,0)$，存在对称的面积元位于 $(-x,y,0)$，二者的引力 $x$ 分量相互抵消。同理 $y$ 分量也相互抵消。因此只有 $z$ 方向分量非零。

**第 2 步：写出面积元对质点的引力大小**

面积元 $dA = dx\,dy$ 的质量为 $dm = \sigma\,dA$，到质点 $(0,0,b)$ 的距离为 $\sqrt{x^2 + y^2 + b^2}$。由万有引力定律：

$$dF = G\,\frac{m \cdot \sigma\,dA}{x^2 + y^2 + b^2}$$

方向从质点指向面积元位置 $(x,y,0)$。

**第 3 步：写出 $z$ 方向分量**

如图所示，引力向量与 $z$ 轴的夹角 $\alpha$ 满足 $\cos\alpha = \dfrac{b}{\sqrt{x^2 + y^2 + b^2}}$，因此：

$$dF_z = dF \cdot \cos\alpha = G\,\frac{m\sigma\,b}{(x^2 + y^2 + b^2)^{3/2}}\,dA$$

**第 4 步：建立二重积分**

$$F_z = \iint_{x^2 + y^2 \leq R^2} G\,\frac{m\sigma b}{(x^2 + y^2 + b^2)^{3/2}}\,dx\,dy$$

**第 5 步：用极坐标变换化为累次积分**

作极坐标变换（定理13.14）：$x = r\cos\theta$，$y = r\sin\theta$，$dx\,dy = r\,dr\,d\theta$，$x^2 + y^2 = r^2$。圆盘在极坐标下描述为 $0 \leq r \leq R$，$0 \leq \theta \leq 2\pi$。

$$
\begin{aligned}
F_z &= \int_{0}^{2\pi} \int_{0}^{R} G\,\frac{m\sigma b}{(r^2 + b^2)^{3/2}} \cdot r\,dr\,d\theta \\
&= G m\sigma b \int_{0}^{2\pi} d\theta \int_{0}^{R} \frac{r}{(r^2 + b^2)^{3/2}}\,dr \\
&= 2\pi G m\sigma b \int_{0}^{R} \frac{r}{(r^2 + b^2)^{3/2}}\,dr
\end{aligned}
$$

此即化为极坐标后的累次积分形式。如果进一步计算（超出本题要求），作换元 $u = r^2 + b^2$，$du = 2r\,dr$，可得：

$$\int_{0}^{R} \frac{r}{(r^2 + b^2)^{3/2}}\,dr = \frac{1}{2} \int_{b^2}^{R^2+b^2} u^{-3/2}\,du = \left[ -u^{-1/2} \right]_{b^2}^{R^2+b^2} = \frac{1}{b} - \frac{1}{\sqrt{R^2 + b^2}}$$

从而 $F_z = 2\pi G m\sigma b \left( \dfrac{1}{b} - \dfrac{1}{\sqrt{R^2 + b^2}} \right) = 2\pi G m\sigma \left( 1 - \dfrac{b}{\sqrt{R^2 + b^2}} \right)$。

**验证**：当 $b \to \infty$ 时，$\dfrac{b}{\sqrt{R^2 + b^2}} \to 1$，$F_z \to 0$，符合预期（质点离圆盘无限远时引力趋于零）。当 $R \to \infty$ 时（无限大均匀平面），$\dfrac{b}{\sqrt{R^2 + b^2}} \to 0$，$F_z \to 2\pi G m\sigma$，即无限大均匀平面对质点的引力是一个与距离 $b$ 无关的常数——这是均匀引力场的经典结论。$\checkmark$

### 例 4：非均匀密度物体的质心

设 $D$ 是由 $x = 0$、$y = 0$ 和 $x + 2y = 2$ 所围成的三角形区域。一平面薄片占据区域 $D$，面密度为 $\rho(x,y) = x$（从 $x=0$ 向左到 $x=2$ 密度线性增大）。求该薄片的质量和质心坐标。

**解**：

**第 1 步：描述区域**

直线 $x + 2y = 2$ 可写为 $y = 1 - \frac{x}{2}$。区域 $D$ 描述为 X-型区域：

$$D = \big\{ (x,y) \mid 0 \leq x \leq 2,\; 0 \leq y \leq 1 - x/2 \big\}$$

**第 2 步：计算总质量 $M$**

$$M = \iint_D \rho(x,y)\,dA = \int_{0}^{2} \int_{0}^{1 - x/2} x\,dy\,dx$$

先对 $y$ 积分（被积函数 $x$ 与 $y$ 无关）：

$$\int_{0}^{1 - x/2} x\,dy = x \int_{0}^{1 - x/2} 1\,dy = x\left(1 - \frac{x}{2}\right) = x - \frac{x^2}{2}$$

再对 $x$ 积分：

$$M = \int_{0}^{2} \left( x - \frac{x^2}{2} \right) dx = \left[ \frac{x^2}{2} - \frac{x^3}{6} \right]_{0}^{2} = \left( \frac{4}{2} - \frac{8}{6} \right) = 2 - \frac{4}{3} = \frac{2}{3}$$

故 $M = \dfrac{2}{3}$（质量单位）。

**第 3 步：计算静矩**

对 $y$ 轴的静矩：
$$M_y = \iint_D x\rho(x,y)\,dA = \iint_D x \cdot x\,dA = \iint_D x^2\,dA$$

$$
\begin{aligned}
M_y &= \int_{0}^{2} \int_{0}^{1 - x/2} x^2\,dy\,dx = \int_{0}^{2} x^2 \left(1 - \frac{x}{2}\right) dx \\
&= \int_{0}^{2} \left( x^2 - \frac{x^3}{2} \right) dx = \left[ \frac{x^3}{3} - \frac{x^4}{8} \right]_{0}^{2} = \frac{8}{3} - \frac{16}{8} = \frac{8}{3} - 2 = \frac{2}{3}
\end{aligned}
$$

对 $x$ 轴的静矩：
$$M_x = \iint_D y\rho(x,y)\,dA = \iint_D y \cdot x\,dA$$

$$
\begin{aligned}
M_x &= \int_{0}^{2} \int_{0}^{1 - x/2} xy\,dy\,dx = \int_{0}^{2} x \left[ \frac{y^2}{2} \right]_{0}^{1 - x/2} dx \\
&= \int_{0}^{2} \frac{x}{2} \left(1 - \frac{x}{2}\right)^2 dx = \frac{1}{2} \int_{0}^{2} x \left(1 - x + \frac{x^2}{4}\right) dx \\
&= \frac{1}{2} \int_{0}^{2} \left( x - x^2 + \frac{x^3}{4} \right) dx \\
&= \frac{1}{2} \left[ \frac{x^2}{2} - \frac{x^3}{3} + \frac{x^4}{16} \right]_{0}^{2} \\
&= \frac{1}{2} \left( \frac{4}{2} - \frac{8}{3} + \frac{16}{16} \right) = \frac{1}{2} \left( 2 - \frac{8}{3} + 1 \right) = \frac{1}{2} \cdot \frac{9 - 8}{3} = \frac{1}{2} \cdot \frac{1}{3} = \frac{1}{6}
\end{aligned}
$$

故 $M_y = \dfrac{2}{3}$，$M_x = \dfrac{1}{6}$。

**第 4 步：计算质心坐标**

$$\bar{x} = \frac{M_y}{M} = \frac{2/3}{2/3} = 1,\qquad \bar{y} = \frac{M_x}{M} = \frac{1/6}{2/3} = \frac{1}{6} \cdot \frac{3}{2} = \frac{1}{4}$$

故该薄片的质心坐标为 $\displaystyle (\bar{x}, \bar{y}) = \left(1, \frac{1}{4}\right)$。

**物理直观**：密度 $\rho = x$ 在 $x=0$ 处为 0，在 $x=2$ 处为 2，因此质量集中在右侧。与均匀密度时的形心相比（若密度均匀，形心的 $\bar{x}$ 应在 $2/3$ 左右），$\bar{x}=1$ 向右偏移，符合"右端密度更大"的预期。$\checkmark$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|:---|:---|
| 静矩 $M_y = \iint x\rho\,dA$ 和转动惯量 $I_z = \iint (x^2+y^2)\rho\,dA$ 是差不多的概念 | 两者完全不同：静矩是**一阶矩**（$\propto x$），用于求质心；转动惯量是**二阶矩**（$\propto x^2$），用于描述旋转惯性。量纲也不同：静矩为 $[\text{质量}][\text{长度}]$，转动惯量为 $[\text{质量}][\text{长度}]^2$ |
| 质心 $\bar{x}$ 的计算公式是 $\bar{x} = \frac{\iint x\,dA}{\iint 1\,dA}$（忘记密度因子） | 这个公式仅在**均匀密度**时成立。对非均匀密度，必须包含密度因子：$\bar{x} = \frac{\iint x\rho\,dA}{\iint \rho\,dA}$ |
| 引力积分中直接对标量 $dF$ 积分就得到了总引力向量 | 引力是**向量**，$d\vec{F}$ 有方向。必须先确定感兴趣的**分量**（如 $z$ 分量），对各分量的微元分别积分。对标量 $dF$ 积分得到的是所有方向的大小之和，这没有物理意义 |
| 质心一定在物体内部 | 质心不一定在物体内部！例如圆环（甜甜圈）的质心在环心处（不在材料内部）；L形物体的质心可能在物体外部。质心是"质量分布的平均位置"，不要求该点属于物体 |
| 均匀密度时质心就是几何中心 | "几何中心"一词不够精确。均匀密度时质心是**形心**（centroid），即区域的几何中心，但不一定等于"对称中心"。如果区域有对称性，形心在对称中心上；否则需要通过积分计算 |
| 半球体的质心在 $z = R/2$（高度的一半处） | 均匀实心半球体的质心在 $z = 3R/8$，不是 $R/2$。原因是半球体底部质量更大（截面积随 $z$ 增大而减小），所以质心偏下 |

### 检查点

- [ ] 能否区分静矩（一阶矩）和转动惯量（二阶矩）？能否写出各自的积分表达式？
- [ ] 对平面薄片，能否写出质心公式 $\bar{x} = M_y / M$，$\bar{y} = M_x / M$？
- [ ] 对三维物体，能否写出三个方向的质心公式？
- [ ] 在什么条件下质心公式可以简化为形心公式？
- [ ] 给定一个具有对称性的区域，能否立即判断出哪些质心坐标分量为零？
- [ ] 能否写出细杆对端部质点的引力积分表达式？
- [ ] 能否写出圆盘对其轴上质点的引力 $z$ 分量积分表达式？

- [ ] 能否正确计算半球体的 $\bar{z} = 3R/8$？
- [ ] 均匀密度与非均匀密度的质心计算有何异同？

**1.**（静矩与质心基础）设区域 $D$ 是由 $x$ 轴、$y$ 轴和直线 $x + y = 1$ 所围成的三角形。一均匀薄片（密度 $\rho_0 = 1$）占据该区域。求该薄片的质心坐标。

区域描述为 X-型区域：$D = \{(x,y) \mid 0 \leq x \leq 1,\; 0 \leq y \leq 1 - x\}$。

均匀密度，用形心公式：
$$\bar{x} = \frac{\iint_D x\,dA}{A(D)},\qquad \bar{y} = \frac{\iint_D y\,dA}{A(D)}$$

先求面积：
$$A(D) = \iint_D 1\,dA = \int_{0}^{1} \int_{0}^{1-x} 1\,dy\,dx = \int_{0}^{1} (1-x)\,dx = \left[ x - \frac{x^2}{2} \right]_{0}^{1} = \frac{1}{2}$$

求 $\iint_D x\,dA$：
$$\iint_D x\,dA = \int_{0}^{1} \int_{0}^{1-x} x\,dy\,dx = \int_{0}^{1} x(1-x)\,dx = \int_{0}^{1} (x - x^2)\,dx = \left[ \frac{x^2}{2} - \frac{x^3}{3} \right]_{0}^{1} = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}$$

求 $\iint_D y\,dA$：
$$\iint_D y\,dA = \int_{0}^{1} \int_{0}^{1-x} y\,dy\,dx = \int_{0}^{1} \frac{(1-x)^2}{2}\,dx = \frac{1}{2} \int_{0}^{1} (1 - 2x + x^2)\,dx = \frac{1}{2} \left[ x - x^2 + \frac{x^3}{3} \right]_{0}^{1} = \frac{1}{2} \cdot \frac{1}{3} = \frac{1}{6}$$

因此：
$$\bar{x} = \frac{1/6}{1/2} = \frac{1}{3},\qquad \bar{y} = \frac{1/6}{1/2} = \frac{1}{3}$$

故该均匀三角形薄片的质心坐标为 $\displaystyle \left(\frac{1}{3}, \frac{1}{3}\right)$，这正是三角形三条中线的交点（重心）。

</details>

**2.**（三维质心基础）设 $\Omega$ 是底面半径为 $R$、高为 $H$ 的均匀实心正圆柱体（密度为常数 $\rho_0$），轴线沿 $z$ 轴方向，底面位于 $z=0$ 平面，顶面位于 $z=H$ 平面。求该圆柱体的质心坐标。

<details><summary>参考答案</summary>

由对称性，$\bar{x} = \bar{y} = 0$（圆柱体关于 $x$ 和 $y$ 方向对称，密度均匀）。只需计算 $\bar{z}$。

质量 $M = \rho_0 V = \rho_0 \pi R^2 H$。

计算 $M_{xy} = \iiint_\Omega z\rho_0\,dV = \rho_0 \iiint_\Omega z\,dV$。

用截面法：固定 $z$，截面是半径为 $R$ 的圆盘，面积为 $\pi R^2$。

$$\iiint_\Omega z\,dV = \int_{0}^{H} z \cdot \pi R^2\,dz = \pi R^2 \int_{0}^{H} z\,dz = \pi R^2 \cdot \frac{H^2}{2} = \frac{\pi R^2 H^2}{2}$$

因此：
$$\bar{z} = \frac{\rho_0 \cdot \pi R^2 H^2 / 2}{\rho_0 \cdot \pi R^2 H} = \frac{H}{2}$$

故均匀实心圆柱体的质心在其几何中心 $(0,0,H/2)$ 处。这正是由均匀密度和圆柱体的对称性所预期的结果。

</details>

### 迁移应用

**3.**（非均匀密度质心）设 $\Omega$ 是位于第一卦限的四面体，由 $x=0$、$y=0$、$z=0$ 和 $x + y + z = 1$ 所围成。体密度为 $\rho(x,y,z) = z$。求该四面体的总质量 $M$ 和质心坐标 $\bar{z}$。

<details><summary>参考答案</summary>

**求总质量 $M$**：
$$M = \iiint_\Omega \rho\,dV = \iiint_\Omega z\,dV$$

区域描述：$0 \leq x \leq 1$，$0 \leq y \leq 1-x$，$0 \leq z \leq 1-x-y$。

$$
\begin{aligned}
M &= \int_{0}^{1} \int_{0}^{1-x} \int_{0}^{1-x-y} z\,dz\,dy\,dx \\
&= \int_{0}^{1} \int_{0}^{1-x} \frac{(1-x-y)^2}{2}\,dy\,dx
\end{aligned}
$$

对 $y$ 积分：令 $u = 1-x-y$，$du = -dy$，当 $y=0$ 时 $u=1-x$，$y=1-x$ 时 $u=0$。

$$\int_{0}^{1-x} \frac{(1-x-y)^2}{2}\,dy = \frac{1}{2} \int_{0}^{1-x} u^2\,(-du) = \frac{1}{2} \int_{0}^{1-x} u^2\,du = \frac{1}{2} \cdot \frac{(1-x)^3}{3} = \frac{(1-x)^3}{6}$$

再对 $x$ 积分：
$$M = \int_{0}^{1} \frac{(1-x)^3}{6}\,dx = \frac{1}{6} \int_{0}^{1} (1-x)^3\,dx = \frac{1}{6} \left[ -\frac{(1-x)^4}{4} \right]_{0}^{1} = \frac{1}{6} \cdot \frac{1}{4} = \frac{1}{24}$$

故 $M = \dfrac{1}{24}$。

**求 $\bar{z}$**：
由 $\rho = z$，$\bar{z} = \dfrac{\iiint_\Omega z\rho\,dV}{M} = \dfrac{\iiint_\Omega z^2\,dV}{M}$。

$$\iiint_\Omega z^2\,dV = \int_{0}^{1} \int_{0}^{1-x} \int_{0}^{1-x-y} z^2\,dz\,dy\,dx = \int_{0}^{1} \int_{0}^{1-x} \frac{(1-x-y)^3}{3}\,dy\,dx$$

对 $y$ 积分（同样用 $u = 1-x-y$）：
$$\int_{0}^{1-x} \frac{(1-x-y)^3}{3}\,dy = \frac{1}{3} \int_{0}^{1-x} u^3\,du = \frac{1}{3} \cdot \frac{(1-x)^4}{4} = \frac{(1-x)^4}{12}$$

再对 $x$ 积分：
$$\iiint_\Omega z^2\,dV = \int_{0}^{1} \frac{(1-x)^4}{12}\,dx = \frac{1}{12} \left[ -\frac{(1-x)^5}{5} \right]_{0}^{1} = \frac{1}{12} \cdot \frac{1}{5} = \frac{1}{60}$$

因此：
$$\bar{z} = \frac{1/60}{1/24} = \frac{1}{60} \cdot \frac{24}{1} = \frac{2}{5}$$

故 $\bar{z} = \dfrac{2}{5}$。由于密度 $\rho = z$ 在 $z$ 方向线性增大（顶部密度更大），质心 $\bar{z} = 0.4$ 高于均匀密度时的形心 $\bar{z} = 0.25$（均匀四面体的形心位于 $z=1/4$），符合直观预期。

</details>

**4.**（引力积分）一均匀细杆位于 $x$ 轴上从 $x=0$ 到 $x=L$，线密度为常数 $\lambda$。现将一个质量为 $m$ 的质点放在 $(0, b)$ 处（$b > 0$，即质点位于 $y$ 轴上）。求该细杆对质点的引力在 $x$ 方向和 $y$ 方向的分量。

<details><summary>参考答案</summary>

**分析**：杆位于 $x$ 轴 $[0, L]$，质点位于 $(0,b)$。取位于 $x$ 处、长度为 $dx$ 的小段，质量为 $dm = \lambda\,dx$，到质点的距离为 $r = \sqrt{x^2 + b^2}$。

**引力大小**：
$$dF = G\,\frac{m \cdot \lambda\,dx}{x^2 + b^2}$$

**方向分析**：该小段对质点的引力沿从质点 $(0,b)$ 指向小段 $(x,0)$ 的方向。设该方向与 $x$ 轴夹角为 $\theta$，则：
- $x$ 方向分量：$dF_x = dF \cdot \frac{x}{\sqrt{x^2 + b^2}}$（质点所受引力 $x$ 分量，负号表示指向正 $x$ 方向）
- $y$ 方向分量：$dF_y = dF \cdot \frac{b}{\sqrt{x^2 + b^2}}$（质点所受引力 $y$ 分量，负号表示指向负 $y$ 方向，因为质点被向下拉）

更精确地，质点受的力指向杆，所以：
- $x$ 分量：从质点指向杆的 $x$ 方向为正，即 $+x$ 方向。$dF_x = dF \cdot \frac{x}{\sqrt{x^2 + b^2}}$
- $y$ 分量：从质点指向杆的方向有向下的分量，即 $-y$ 方向。$dF_y = -dF \cdot \frac{b}{\sqrt{x^2 + b^2}}$

**$x$ 方向分量**：
$$F_x = \int_{0}^{L} G\,\frac{m\lambda}{x^2 + b^2} \cdot \frac{x}{\sqrt{x^2 + b^2}}\,dx = Gm\lambda \int_{0}^{L} \frac{x}{(x^2 + b^2)^{3/2}}\,dx$$

作换元 $u = x^2 + b^2$，$du = 2x\,dx$：
$$\int_{0}^{L} \frac{x}{(x^2 + b^2)^{3/2}}\,dx = \frac{1}{2} \int_{b^2}^{L^2+b^2} u^{-3/2}\,du = \frac{1}{2} \left[ -2u^{-1/2} \right]_{b^2}^{L^2+b^2} = \frac{1}{b} - \frac{1}{\sqrt{L^2 + b^2}}$$

因此：
$$F_x = Gm\lambda \left( \frac{1}{b} - \frac{1}{\sqrt{L^2 + b^2}} \right)$$

**$y$ 方向分量**（取正数表示大小，方向为 $-y$）：
$$|F_y| = \int_{0}^{L} G\,\frac{m\lambda}{x^2 + b^2} \cdot \frac{b}{\sqrt{x^2 + b^2}}\,dx = Gm\lambda b \int_{0}^{L} \frac{1}{(x^2 + b^2)^{3/2}}\,dx$$

对 $\displaystyle \int \frac{dx}{(x^2 + b^2)^{3/2}}$，用换元 $x = b\tan\theta$，$dx = b\sec^2\theta\,d\theta$，$x^2 + b^2 = b^2\sec^2\theta$：

当 $x=0$ 时 $\theta=0$，$x=L$ 时 $\theta = \arctan(L/b)$。

$$\begin{aligned}
\int_{0}^{L} \frac{dx}{(x^2 + b^2)^{3/2}} &= \int_{0}^{\arctan(L/b)} \frac{b\sec^2\theta\,d\theta}{(b^2\sec^2\theta)^{3/2}} = \int_{0}^{\arctan(L/b)} \frac{b\sec^2\theta}{b^3\sec^3\theta}\,d\theta \\
&= \frac{1}{b^2} \int_{0}^{\arctan(L/b)} \cos\theta\,d\theta = \frac{1}{b^2} \left[ \sin\theta \right]_{0}^{\arctan(L/b)} \\
&= \frac{1}{b^2} \cdot \frac{L}{\sqrt{L^2 + b^2}}
\end{aligned}$$

因此：
$$|F_y| = Gm\lambda b \cdot \frac{1}{b^2} \cdot \frac{L}{\sqrt{L^2 + b^2}} = \frac{Gm\lambda L}{b\sqrt{L^2 + b^2}}$$

方向沿 $-y$ 方向（向下，指向杆）。

故细杆对质点的引力分量为：
$$F_x = Gm\lambda \left( \frac{1}{b} - \frac{1}{\sqrt{L^2 + b^2}} \right),\qquad F_y = -\frac{Gm\lambda L}{b\sqrt{L^2 + b^2}}$$

**验证**：当 $L \to 0$（杆缩为点）时，$F_x \to Gm\lambda L / b^2 = GmM / b^2$，$F_y \to -GmM / b^2$（这里 $M = \lambda L$）。这是两个质点之间的引力公式，正确。当 $b \to \infty$ 时，$F_x \to 0$，$F_y \to 0$，合理。

</details>
