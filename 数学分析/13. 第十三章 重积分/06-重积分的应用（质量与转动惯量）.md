## 2. 背景与应用场景

### 2.1 从三重积分到物理应用

在文件01中，我们学习了三重积分的定义（定义13.2）：
$$I = \iiint_\Omega f(x,y,z)\,dV = \lim_{\lambda\to0} \sum_{i=1}^n f(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$$

这个数学结构有一个极为重要的物理解释：若将 $f$ 解释为某种**密度**（单位体积所含的物理量），则三重积分就是该物理量在区域 $\Omega$ 上的**总量**。

具体来说，如果有一个三维物体 $\Omega$，其体密度函数为 $\rho(x,y,z)$（单位：$\text{kg}/\text{m}^3$），那么：
- 在 $\Omega$ 中取一个很小的体积元 $\Delta V_i$
- 该体积元的质量近似为 $\Delta m_i \approx \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$
- 将所有体积元的质量相加再取极限，就得到总质量

这正是三重积分定义中 Riemann 和 $\sum \rho(\xi_i,\eta_i,\zeta_i)\Delta V_i$ 的物理含义。

### 2.2 质量计算的工程与科学意义

质量计算是工程设计的基石：
- 航空航天：计算飞行器各部件的质量分布
- 机械设计：确定零件质量以评估载荷
- 材料科学：对非均匀材料（如复合材料、梯度功能材料）计算总质量

当密度为常数时，质量简化为密度乘以体积，此时三重积分退化为体积计算。但当密度随位置变化时（如地球内部密度随深度变化），必须通过三重积分才能准确计算总质量。

### 2.3 转动惯量的物理直觉

转动惯量（moment of inertia）是刚体绕轴旋转时**抵抗角加速度**的度量——它是"旋转的惯性"，与质量在"抵抗线加速度"中的角色类似。

考虑一个简单的实验：旋转一根长杆。同样质量的杆，如果质量分布在离转轴更远的地方（如长杆 vs 短杆），旋转起来更困难。这说明：
- 转动惯量不仅取决于总质量，还取决于**质量相对于转轴的分布**
- 距离转轴越远的质量元，对转动惯量的贡献越大——具体地，贡献与距离的平方成正比

对于离散质点系，转动惯量的定义为 $I = \sum m_i r_i^2$，其中 $r_i$ 是第 $i$ 个质点到转轴的垂直距离。对于连续体，求和变为积分。

转动惯量在工程中有广泛应用：
- 飞轮设计：通过将质量集中在轮缘来增大转动惯量，储存更多旋转动能
- 陀螺仪：利用高转动惯量维持方向稳定
- 机械系统：计算旋转部件的角加速度 $\alpha = \tau / I$（$\tau$ 为扭矩）

## 3. 核心概念与符号约定

### 3.1 三维物体的质量公式

**定理 13.17（三维物体的质量公式 / Mass via Triple Integral）**：设三维物体 $\Omega \subset \mathbb{R}^3$ 占据有界闭区域，其体密度函数为 $\rho(x,y,z)$（在 $\Omega$ 上连续）。则该物体的总质量为：
$$\boxed{M = \iiint_\Omega \rho(x,y,z)\,dV}$$

**从 Riemann 和到质量公式的推导**：

按照定义13.2（三重积分的 Riemann 和定义）的流程：

1. **分割**：将 $\Omega$ 分割成 $n$ 个小区域 $\Delta V_1, \Delta V_2, \dots, \Delta V_n$，每个体积为 $\Delta V_i$
2. **取样**：在第 $i$ 个小区域上任取一点 $(\xi_i,\eta_i,\zeta_i)$，该点的密度为 $\rho(\xi_i,\eta_i,\zeta_i)$
3. **近似**：第 $i$ 个小区域的质量近似为 $\Delta m_i \approx \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$
4. **求和**：总质量的近似为 $\displaystyle M_n = \sum_{i=1}^n \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$
5. **取极限**：当细度 $\lambda \to 0$（所有小区域收缩为点）时，近似值趋于精确值：
   $$M = \lim_{\lambda \to 0} \sum_{i=1}^n \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i = \iiint_\Omega \rho(x,y,z)\,dV$$

因此，三重积分的 Riemann 和 $\sum \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$ 被赋予了物理含义——它是总质量的近似值，而三重积分本身给出了总质量的精确值。

**常数密度情形**：若物体的密度为常数 $\rho(x,y,z) = \rho_0$（均匀物体），则：
$$M = \iiint_\Omega \rho_0\,dV = \rho_0 \iiint_\Omega 1\,dV = \rho_0 \cdot V(\Omega)$$
其中 $V(\Omega) = \iiint_\Omega 1\,dV$ 是区域 $\Omega$ 的体积（这是文件01中三重积分几何意义的直接应用——常数函数 $1$ 的积分等于区域体积）。

### 3.2 转动惯量的一般定义

**物理直觉**：质量度量物体抵抗**线加速度**的惯性（$F = ma$），转动惯量度量物体抵抗**角加速度**的惯性（$\tau = I\alpha$，$\tau$ 为扭矩，$\alpha$ 为角加速度）。

**从离散到连续**：

对于 $n$ 个质点组成的系统，各质点质量为 $m_i$，到转轴的垂直距离为 $r_i$，系统绕该轴的转动惯量为：
$$I = \sum_{i=1}^n m_i r_i^2$$

对于连续体 $\Omega$，将物体分割成许多小体积元 $\Delta V_i$，每个体积元的质量为 $\Delta m_i \approx \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$。该体积元对转动惯量的贡献近似为 $r_i^2 \Delta m_i$，其中 $r_i$ 是该体积元到转轴的垂直距离。求和并取极限：

$$I = \lim_{\lambda\to0} \sum_{i=1}^n r_i^2 \rho(\xi_i,\eta_i,\zeta_i)\,\Delta V_i = \iiint_\Omega r^2 \rho(x,y,z)\,dV$$

其中 $r = r(x,y,z)$ 是点 $(x,y,z)$ 到转轴的垂直距离。

**定理 13.18（转动惯量的三重积分公式 / Moment of Inertia via Triple Integral）**：设三维物体 $\Omega$ 的体密度为 $\rho(x,y,z)$，则物体绕给定转轴的转动惯量为：
$$\boxed{I = \iiint_\Omega r^2 \rho(x,y,z)\,dV}$$
其中 $r = r(x,y,z)$ 是点 $(x,y,z)$ 到转轴的垂直距离。

### 3.3 绕坐标轴和原点的转动惯量公式

**绕 $z$ 轴的转动惯量 $I_z$**：

点 $(x,y,z)$ 到 $z$ 轴的垂直距离为 $\sqrt{x^2 + y^2}$（即该点在 $xy$ 平面上投影到原点的距离），因此 $r^2 = x^2 + y^2$：
$$\boxed{I_z = \iiint_\Omega (x^2 + y^2)\,\rho(x,y,z)\,dV}$$

**绕 $x$ 轴的转动惯量 $I_x$**：

点 $(x,y,z)$ 到 $x$ 轴的垂直距离为 $\sqrt{y^2 + z^2}$，因此 $r^2 = y^2 + z^2$：
$$\boxed{I_x = \iiint_\Omega (y^2 + z^2)\,\rho(x,y,z)\,dV}$$

**绕 $y$ 轴的转动惯量 $I_y$**：

点 $(x,y,z)$ 到 $y$ 轴的垂直距离为 $\sqrt{x^2 + z^2}$，因此 $r^2 = x^2 + z^2$：
$$\boxed{I_y = \iiint_\Omega (x^2 + z^2)\,\rho(x,y,z)\,dV}$$

**关于原点的转动惯量 $I_O$**（有时称为"极转动惯量"）：

点 $(x,y,z)$ 到原点的距离为 $\sqrt{x^2 + y^2 + z^2}$，因此 $r^2 = x^2 + y^2 + z^2$：
$$\boxed{I_O = \iiint_\Omega (x^2 + y^2 + z^2)\,\rho(x,y,z)\,dV}$$

**迹恒等式（Trace Identity）**：将 $I_x$、$I_y$、$I_z$ 的表达式相加：
$$I_x + I_y + I_z = \iiint_\Omega [(y^2+z^2) + (x^2+z^2) + (x^2+y^2)]\,\rho\,dV = \iiint_\Omega 2(x^2+y^2+z^2)\,\rho\,dV = 2I_O$$

因此：
$$\boxed{I_x + I_y + I_z = 2I_O}$$

这个恒等式可用于验证计算结果的正确性——三个坐标轴的转动惯量之和必须等于 $2$ 倍的原点转动惯量。

**距离平方速查表**：

| 转轴/点 | 垂直距离 $r$ | $r^2$（被积函数中的表达式） |
|:---|:---|:---|
| $z$ 轴 | $\sqrt{x^2 + y^2}$ | $x^2 + y^2$ |
| $x$ 轴 | $\sqrt{y^2 + z^2}$ | $y^2 + z^2$ |
| $y$ 轴 | $\sqrt{x^2 + z^2}$ | $x^2 + z^2$ |
| 原点 | $\sqrt{x^2 + y^2 + z^2}$ | $x^2 + y^2 + z^2$ |

**记忆方法**：绕哪个轴转动，被积函数中就**缺失哪个坐标的平方**。例如：
- 绕 $z$ 轴 → 被积函数缺 $z^2$，出现 $x^2+y^2$
- 绕 $x$ 轴 → 被积函数缺 $x^2$，出现 $y^2+z^2$
- 绕 $y$ 轴 → 被积函数缺 $y^2$，出现 $x^2+z^2$

### 3.4 符号表

| 符号 | 含义 | 单位 |
|:---|:---|:---|
| $\rho(x,y,z)$ | 体密度函数（单位体积所含质量） | $\text{kg}/\text{m}^3$ |
| $M$ | 总质量 | $\text{kg}$ |
| $I$ | 转动惯量（绕给定转轴） | $\text{kg}\cdot\text{m}^2$ |
| $I_x, I_y, I_z$ | 绕 $x$、$y$、$z$ 轴的转动惯量 | $\text{kg}\cdot\text{m}^2$ |
| $I_O$ | 关于原点的转动惯量 | $\text{kg}\cdot\text{m}^2$ |
| $r$ | 点到转轴（或原点）的垂直距离 | $\text{m}$ |
| $V(\Omega)$ | 区域 $\Omega$ 的体积 | $\text{m}^3$ |
| $dV$ | 体积元素 | $\text{m}^3$ |

## 4. 原理与方法

### 4.1 质量计算的完整流程

使用三重积分计算三维物体质量的步骤如下：

**第 1 步：确定密度函数 $\rho(x,y,z)$**
- 检查密度是常数还是变量。若题目说"均匀物体"或"密度为常数 $\rho_0$"，则 $\rho(x,y,z) = \rho_0$
- 若密度是变量（如 $\rho = x^2 + y^2$），则保留为函数形式

**第 2 步：选择坐标系，描述区域 $\Omega$**
- 直角坐标：适合长方体、四面体等区域
- 柱坐标（$dV = r\,dr\,d\theta\,dz$）：适合绕 $z$ 轴旋转对称的区域（圆柱体、圆锥体等）
- 球坐标（$dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$）：适合球对称区域

**第 3 步：写出三重积分并计算**
$$M = \iiint_\Omega \rho(x,y,z)\,dV$$

使用投影法（定理13.11）或截面法（定理13.12）将三重积分化为累次积分，必要时结合柱坐标或球坐标变换（定理13.14）。

**第 4 步：验证**
- 常数密度情形下，验证 $M = \rho_0 \cdot V(\Omega)$
- 检查量纲：密度 $\times$ 体积 = 质量

### 4.2 转动惯量计算的完整流程

**第 1 步：确定转轴**
- 明确要求绕哪条轴计算转动惯量。对于坐标轴，直接使用定理13.18中的公式
- 对于任意直线（如 $z = h$ 平面上的某条线），需要计算点到该直线的距离

**第 2 步：确定距离平方 $r^2$ 的表达式**
- 绕 $z$ 轴：$r^2 = x^2 + y^2$
- 绕 $x$ 轴：$r^2 = y^2 + z^2$
- 绕 $y$ 轴：$r^2 = x^2 + z^2$
- 关于原点：$r^2 = x^2 + y^2 + z^2$

**第 3 步：写出转动惯量的三重积分表达式**
$$I = \iiint_\Omega r^2 \rho(x,y,z)\,dV$$

**第 4 步：选择合适的坐标系和积分方法**
- 若被积函数含 $x^2+y^2$ 且区域绕 $z$ 轴旋转对称 → 柱坐标（此时 $x^2+y^2 = r^2$，$dV = r\,dr\,d\theta\,dz$）
- 若被积函数含 $x^2+y^2+z^2$ 且区域球对称 → 球坐标（此时 $x^2+y^2+z^2 = \rho^2$，$dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$）
- 若区域形状简单但被积函数复杂 → 选择截面法或投影法

**第 5 步：计算三重积分**

**第 6 步：验证**
- 对均匀密度物体，可用迹恒等式 $I_x + I_y + I_z = 2I_O$ 验证
- 检查量纲：$\text{kg} \cdot \text{m}^2$（质量 $\times$ 距离平方）

### 4.3 柱坐标和球坐标下转动惯量公式的简化

当被积函数和区域同时适合柱坐标或球坐标时，转动惯量公式可以大幅简化。

**柱坐标下的简化**：

在柱坐标变换（定义13.15）下：
$$x = r\cos\theta,\quad y = r\sin\theta,\quad z = z,\quad dV = r\,dr\,d\theta\,dz$$

各转动惯量表达式变为：

- $I_z = \iiint_\Omega (x^2+y^2)\rho\,dV = \iiint_{\Omega'} r^2 \cdot \rho(r\cos\theta, r\sin\theta, z) \cdot r\,dr\,d\theta\,dz = \iiint_{\Omega'} r^3 \rho\,dr\,d\theta\,dz$
- $I_x = \iiint_\Omega (y^2+z^2)\rho\,dV = \iiint_{\Omega'} (r^2\sin^2\theta + z^2) \cdot \rho \cdot r\,dr\,d\theta\,dz$
- $I_O = \iiint_\Omega (x^2+y^2+z^2)\rho\,dV = \iiint_{\Omega'} (r^2+z^2) \cdot \rho \cdot r\,dr\,d\theta\,dz$

当 $\rho$ 和区域都绕 $z$ 轴旋转对称时（$\rho$ 与 $\theta$ 无关），$I_z$ 的计算简化为 $r$ 和 $z$ 的二重积分（$\theta$ 方向积分给出因子 $2\pi$）。

**球坐标下的简化**：

在球坐标变换（定义13.16）下：
$$x = \rho\sin\varphi\cos\theta,\quad y = \rho\sin\varphi\sin\theta,\quad z = \rho\cos\varphi,\quad dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$$

- $I_O = \iiint_\Omega (x^2+y^2+z^2)\rho\,dV = \iiint_{\Omega'} \rho^2 \cdot \rho \cdot \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta = \iiint_{\Omega'} \rho^4\sin\varphi \cdot \rho\,d\rho\,d\varphi\,d\theta$

当 $\rho$ 为常数且区域为球体时，$I_O$ 的计算可以分离变量，极为简洁。

### 4.4 解题策略总结

| 情形 | 推荐方法 | 说明 |
|:---|:---|:---|
| 密度为常数，只需总质量 | $M = \rho_0 \cdot V(\Omega)$ | 无需积分，直接计算体积 |
| 密度为变量，区域为长方体 | 直角坐标 + Fubini定理 | 直接化为三次累次积分 |
| 密度为变量，区域绕 $z$ 轴对称 | 柱坐标 | 被积函数中 $x^2+y^2$ 变为 $r^2$ |
| 区域球对称 | 球坐标 | 被积函数中 $x^2+y^2+z^2$ 变为 $\rho^2$ |
| 只依赖于 $z$ 的密度 | 截面法 | $f = f(z)$ 时截面法特别简便 |
| 验证计算结果 | 迹恒等式 $I_x+I_y+I_z = 2I_O$ | 检查 $I_x$、$I_y$、$I_z$ 是否满足该关系 |

## 5. 例题

### 例 1：圆柱体的质量

设 $\Omega$ 是圆柱体 $x^2 + y^2 \leq 4$、$0 \leq z \leq 3$，体密度为 $\rho(x,y,z) = x^2 + y^2$。求该圆柱体的总质量 $M$。

**解**：

**第 1 步：确定密度函数和区域**
$$\rho(x,y,z) = x^2 + y^2$$
$$\Omega = \big\{ (x,y,z) \mid x^2 + y^2 \leq 4,\; 0 \leq z \leq 3 \big\}$$

**第 2 步：选择坐标系**

区域是绕 $z$ 轴旋转对称的圆柱体，且被积函数 $x^2 + y^2$ 适合柱坐标变换。选用柱坐标（定义13.15）：
$$x = r\cos\theta,\quad y = r\sin\theta,\quad z = z,\quad dV = r\,dr\,d\theta\,dz$$

**第 3 步：用柱坐标描述区域**

$$\Omega' = \big\{ (r,\theta,z) \mid 0 \leq r \leq 2,\; 0 \leq \theta \leq 2\pi,\; 0 \leq z \leq 3 \big\}$$

这是一个长方体区域。

**第 4 步：转换被积函数**

$$\rho(x,y,z) = x^2 + y^2 = r^2$$

**第 5 步：写出三重积分并计算**

由定理13.17（质量公式）：
$$M = \iiint_\Omega (x^2+y^2)\,dV = \iiint_{\Omega'} r^2 \cdot r\,dr\,d\theta\,dz = \iiint_{\Omega'} r^3\,dr\,d\theta\,dz$$

由于 $\Omega'$ 是长方体且被积函数 $r^3$ 可分离，三重积分化为三个一元积分的乘积：
$$M = \left( \int_{0}^{3} dz \right) \cdot \left( \int_{0}^{2\pi} d\theta \right) \cdot \left( \int_{0}^{2} r^3\,dr \right)$$

逐层计算：
$$\int_{0}^{3} dz = 3,\qquad \int_{0}^{2\pi} d\theta = 2\pi,\qquad \int_{0}^{2} r^3\,dr = \left[ \frac{r^4}{4} \right]_{0}^{2} = \frac{16}{4} = 4$$

因此：
$$M = 3 \cdot 2\pi \cdot 4 = 24\pi$$

故该圆柱体的总质量为 $M = 24\pi$（质量单位）。

**第 6 步：验证**

由例2（文件05例1）的结果，对 $\iiint (x^2+y^2)\,dV$ 在同样圆柱体上（$R=2$、$h=3$），结果为 $\frac{\pi h R^4}{2} = \frac{\pi \cdot 3 \cdot 16}{2} = 24\pi$。这正是我们的 $M$，因为 $\rho = x^2+y^2$，所以 $M = \iiint \rho\,dV = \iiint (x^2+y^2)\,dV = 24\pi$。$\checkmark$

### 例 2：圆柱体的转动惯量

设 $\Omega$ 是圆柱体 $x^2 + y^2 \leq 4$、$0 \leq z \leq 3$，体密度为常数 $\rho_0 = 2$（均匀圆柱体）。求：

(1) 绕 $z$ 轴的转动惯量 $I_z$

(2) 绕 $x$ 轴的转动惯量 $I_x$

**解 (1)：$I_z$ 的计算**

**第 1 步：确定转轴和距离平方**

绕 $z$ 轴：$r^2 = x^2 + y^2$

**第 2 步：写出转动惯量公式**

由定理13.18：
$$I_z = \iiint_\Omega (x^2 + y^2) \cdot \rho_0 \,dV = \rho_0 \iiint_\Omega (x^2 + y^2)\,dV$$

因为 $\rho_0$ 是常数，可以提到积分号外。

**第 3 步：选择柱坐标变换**

与例1相同，$x^2 + y^2 = r^2$，$dV = r\,dr\,d\theta\,dz$，区域 $0 \leq r \leq 2$、$0 \leq \theta \leq 2\pi$、$0 \leq z \leq 3$。

**第 4 步：计算三重积分**

$$I_z = \rho_0 \iiint_{\Omega'} r^2 \cdot r\,dr\,d\theta\,dz = \rho_0 \iiint_{\Omega'} r^3\,dr\,d\theta\,dz$$

这个积分与例1中的积分完全一致（除了多了一个 $\rho_0$ 因子）：
$$I_z = \rho_0 \cdot \left( \int_{0}^{3} dz \right) \cdot \left( \int_{0}^{2\pi} d\theta \right) \cdot \left( \int_{0}^{2} r^3\,dr \right) = \rho_0 \cdot 24\pi$$

代入 $\rho_0 = 2$：
$$I_z = 2 \cdot 24\pi = 48\pi$$

因此 $I_z = 48\pi$（单位：$\text{kg}\cdot\text{m}^2$）。

**解 (2)：$I_x$ 的计算**

**第 1 步：确定转轴和距离平方**

绕 $x$ 轴：$r^2 = y^2 + z^2$。因此：
$$I_x = \rho_0 \iiint_\Omega (y^2 + z^2)\,dV$$

**第 2 步：拆分积分**
$$I_x = \rho_0 \iiint_\Omega y^2\,dV + \rho_0 \iiint_\Omega z^2\,dV$$

分别计算两项。

**第 3 步：计算 $\iiint_\Omega y^2\,dV$**

在圆柱体 $\Omega: x^2+y^2 \leq 4,\; 0 \leq z \leq 3$ 上。由对称性，$\iiint_\Omega x^2\,dV = \iiint_\Omega y^2\,dV$。

用柱坐标变换：
$$y = r\sin\theta,\quad y^2 = r^2\sin^2\theta,\quad dV = r\,dr\,d\theta\,dz$$

$$\begin{aligned}
\iiint_\Omega y^2\,dV &= \int_{0}^{3} \int_{0}^{2\pi} \int_{0}^{2} r^2\sin^2\theta \cdot r\,dr\,d\theta\,dz \\
&= \left( \int_{0}^{3} dz \right) \cdot \left( \int_{0}^{2\pi} \sin^2\theta\,d\theta \right) \cdot \left( \int_{0}^{2} r^3\,dr \right)
\end{aligned}$$

计算各个因子：
$$\int_{0}^{3} dz = 3$$
$$\int_{0}^{2} r^3\,dr = \frac{16}{4} = 4$$
$$\int_{0}^{2\pi} \sin^2\theta\,d\theta = \int_{0}^{2\pi} \frac{1 - \cos 2\theta}{2}\,d\theta = \frac{1}{2} \cdot 2\pi - 0 = \pi$$

因此：
$$\iiint_\Omega y^2\,dV = 3 \cdot \pi \cdot 4 = 12\pi$$

**第 4 步：计算 $\iiint_\Omega z^2\,dV$**

用柱坐标变换，$z$ 在柱坐标下保持不变：
$$\begin{aligned}
\iiint_\Omega z^2\,dV &= \int_{0}^{3} \int_{0}^{2\pi} \int_{0}^{2} z^2 \cdot r\,dr\,d\theta\,dz \\
&= \left( \int_{0}^{3} z^2\,dz \right) \cdot \left( \int_{0}^{2\pi} d\theta \right) \cdot \left( \int_{0}^{2} r\,dr \right)
\end{aligned}$$

计算各个因子：
$$\int_{0}^{3} z^2\,dz = \left[ \frac{z^3}{3} \right]_{0}^{3} = \frac{27}{3} = 9$$
$$\int_{0}^{2\pi} d\theta = 2\pi$$
$$\int_{0}^{2} r\,dr = \left[ \frac{r^2}{2} \right]_{0}^{2} = \frac{4}{2} = 2$$

因此：
$$\iiint_\Omega z^2\,dV = 9 \cdot 2\pi \cdot 2 = 36\pi$$

**第 5 步：合并结果**
$$I_x = \rho_0 \cdot (12\pi + 36\pi) = \rho_0 \cdot 48\pi = 2 \cdot 48\pi = 96\pi$$

因此 $I_x = 96\pi$。

**第 6 步：验证（用迹恒等式）**

由对称性，$I_y = I_x = 96\pi$（对均匀圆柱体，绕 $x$ 轴和 $y$ 轴的转动惯量相等）。已知 $I_z = 48\pi$。

计算 $I_x + I_y + I_z = 96\pi + 96\pi + 48\pi = 240\pi$。

计算 $I_O = \iiint_\Omega (x^2+y^2+z^2)\rho_0\,dV = \rho_0 \left( \iiint_\Omega (x^2+y^2)\,dV + \iiint_\Omega z^2\,dV \right)$。

由例1，$\iiint_\Omega (x^2+y^2)\,dV = 24\pi$，且 $\iiint_\Omega z^2\,dV = 36\pi$，所以：
$$I_O = \rho_0 (24\pi + 36\pi) = 2 \cdot 60\pi = 120\pi$$

验证迹恒等式：$I_x + I_y + I_z = 240\pi = 2 \times 120\pi = 2I_O$。$\checkmark$

### 例 3：球体的转动惯量

设 $\Omega$ 是半径为 $R$ 的球体 $x^2 + y^2 + z^2 \leq R^2$，体密度为常数 $\rho_0$（均匀球体）。求：

(1) 关于原点的转动惯量 $I_O$

(2) 绕 $z$ 轴的转动惯量 $I_z$

**解 (1)：$I_O$ 的计算**

**第 1 步：确定被积函数**

关于原点：$r^2 = x^2 + y^2 + z^2$。因此：
$$I_O = \rho_0 \iiint_\Omega (x^2 + y^2 + z^2)\,dV$$

**第 2 步：选择球坐标变换**

球体适合球坐标变换（定义13.16）：
$$x = \rho\sin\varphi\cos\theta,\quad y = \rho\sin\varphi\sin\theta,\quad z = \rho\cos\varphi,\quad dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$$

**第 3 步：用球坐标描述区域**

$$\Omega' = \big\{ (\rho,\varphi,\theta) \mid 0 \leq \rho \leq R,\; 0 \leq \varphi \leq \pi,\; 0 \leq \theta \leq 2\pi \big\}$$

**第 4 步：转换被积函数**

$$x^2 + y^2 + z^2 = \rho^2$$

**第 5 步：写出三重积分并计算**

$$I_O = \rho_0 \iiint_{\Omega'} \rho^2 \cdot \rho^2\sin\varphi \,d\rho\,d\varphi\,d\theta = \rho_0 \iiint_{\Omega'} \rho^4\sin\varphi \,d\rho\,d\varphi\,d\theta$$

由于 $\Omega'$ 是长方体且被积函数可分离：
$$I_O = \rho_0 \cdot \left( \int_{0}^{R} \rho^4\,d\rho \right) \cdot \left( \int_{0}^{\pi} \sin\varphi\,d\varphi \right) \cdot \left( \int_{0}^{2\pi} d\theta \right)$$

逐层计算：
$$\int_{0}^{R} \rho^4\,d\rho = \left[ \frac{\rho^5}{5} \right]_{0}^{R} = \frac{R^5}{5}$$
$$\int_{0}^{\pi} \sin\varphi\,d\varphi = \left[ -\cos\varphi \right]_{0}^{\pi} = -\cos\pi + \cos 0 = 1 + 1 = 2$$
$$\int_{0}^{2\pi} d\theta = 2\pi$$

因此：
$$I_O = \rho_0 \cdot \frac{R^5}{5} \cdot 2 \cdot 2\pi = \frac{4\pi\rho_0 R^5}{5}$$

故均匀球体关于原点的转动惯量为 $I_O = \dfrac{4\pi\rho_0 R^5}{5}$。

**验证**：球体体积 $V(\Omega) = \dfrac{4}{3}\pi R^3$，质量为 $M = \rho_0 \cdot \dfrac{4}{3}\pi R^3$。$I_O = \dfrac{3}{5}MR^2$，与物理学的经典结果一致（均匀球体关于球心的转动惯量为 $\frac{3}{5}MR^2$）。$\checkmark$

**解 (2)：$I_z$ 的计算**

**第 1 步：写出转动惯量公式**

绕 $z$ 轴：$r^2 = x^2 + y^2$。因此：
$$I_z = \rho_0 \iiint_\Omega (x^2 + y^2)\,dV$$

**第 2 步：用球坐标变换**

$$x^2 + y^2 = \rho^2\sin^2\varphi$$

$$I_z = \rho_0 \iiint_{\Omega'} (\rho^2\sin^2\varphi) \cdot \rho^2\sin\varphi \,d\rho\,d\varphi\,d\theta = \rho_0 \iiint_{\Omega'} \rho^4\sin^3\varphi \,d\rho\,d\varphi\,d\theta$$

**第 3 步：分离变量**

$$I_z = \rho_0 \cdot \left( \int_{0}^{R} \rho^4\,d\rho \right) \cdot \left( \int_{0}^{\pi} \sin^3\varphi \,d\varphi \right) \cdot \left( \int_{0}^{2\pi} d\theta \right)$$

对 $\rho$ 和 $\theta$ 的积分与 $I_O$ 中相同：
$$\int_{0}^{R} \rho^4\,d\rho = \frac{R^5}{5},\qquad \int_{0}^{2\pi} d\theta = 2\pi$$

**第 4 步：计算 $\varphi$ 的积分**

$$\begin{aligned}
\int_{0}^{\pi} \sin^3\varphi \,d\varphi &= \int_{0}^{\pi} \sin\varphi\,(1 - \cos^2\varphi)\,d\varphi \\
&= \int_{0}^{\pi} \sin\varphi\,d\varphi - \int_{0}^{\pi} \sin\varphi\cos^2\varphi\,d\varphi
\end{aligned}$$

第一项：$\displaystyle \int_{0}^{\pi} \sin\varphi\,d\varphi = 2$

第二项：作换元 $u = \cos\varphi$，$du = -\sin\varphi\,d\varphi$。当 $\varphi = 0$ 时 $u = 1$，$\varphi = \pi$ 时 $u = -1$。
$$\int_{0}^{\pi} \sin\varphi\cos^2\varphi\,d\varphi = \int_{1}^{-1} u^2 \cdot (-du) = \int_{-1}^{1} u^2\,du = \left[ \frac{u^3}{3} \right]_{-1}^{1} = \frac{1}{3} - \left(-\frac{1}{3}\right) = \frac{2}{3}$$

因此：
$$\int_{0}^{\pi} \sin^3\varphi \,d\varphi = 2 - \frac{2}{3} = \frac{4}{3}$$

**第 5 步：合并结果**

$$I_z = \rho_0 \cdot \frac{R^5}{5} \cdot \frac{4}{3} \cdot 2\pi = \frac{8\pi\rho_0 R^5}{15}$$

故均匀球体绕 $z$ 轴的转动惯量为 $I_z = \dfrac{8\pi\rho_0 R^5}{15}$。

**验证**：由球对称性，$I_x = I_y = I_z = \dfrac{8\pi\rho_0 R^5}{15}$。由迹恒等式：
$$I_x + I_y + I_z = 3 \cdot \frac{8\pi\rho_0 R^5}{15} = \frac{24\pi\rho_0 R^5}{15} = \frac{8\pi\rho_0 R^5}{5}$$

而 $2I_O = 2 \cdot \dfrac{4\pi\rho_0 R^5}{5} = \dfrac{8\pi\rho_0 R^5}{5}$。

因此 $I_x + I_y + I_z = 2I_O$ 成立。$\checkmark$

**对比**：$I_z = \dfrac{8\pi\rho_0 R^5}{15}$，而 $I_O = \dfrac{4\pi\rho_0 R^5}{5} = \dfrac{12\pi\rho_0 R^5}{15}$。$I_O > I_z$，这是合理的——关于原点的转动惯量相当于"绕三个轴都有贡献"，因此大于单独绕某个轴的转动惯量。

用质量 $M = \dfrac{4}{3}\pi\rho_0 R^3$ 表示：
$$I_z = \frac{8\pi\rho_0 R^5}{15} = \frac{2}{5}MR^2,\qquad I_O = \frac{4\pi\rho_0 R^5}{5} = \frac{3}{5}MR^2$$

这就是物理学中均匀球体绕直径转动惯量为 $\frac{2}{5}MR^2$、关于球心转动惯量为 $\frac{3}{5}MR^2$ 的经典结论。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|:---|:---|
| 绕 $x$ 轴的转动惯量 $I_x = \iiint (x^2+y^2)\rho\,dV$（被积函数中出现了转轴对应的坐标） | 绕 $x$ 轴时被积函数应为 $y^2+z^2$，**不包含转轴方向的坐标**。记忆法：绕哪个轴转，被积函数中就缺哪个坐标的平方 |
| 转动惯量 $I$ 中距离 $r$ 是点到转轴的直线距离（不是点到转轴上某点的距离） | $r$ 是点到转轴的**垂直距离**（垂线长度），不是点到转轴上某点的斜线距离。对于 $z$ 轴，垂足是 $(x,y,0)$，距离为 $\sqrt{x^2+y^2}$ |
| 转动惯量是向量，有方向 | 转动惯量是**标量**，只有大小（非负），没有方向。它描述的是物体绕给定转轴的转动惯性 |
| 密度为常数时，$M = \iiint \rho_0\,dV$ 可直接计算，但 $I = \iiint r^2\rho_0\,dV$ 中 $\rho_0$ 不能提出 | $\rho_0$ 是常数，可以**提到积分号外**：$I = \rho_0 \iiint r^2\,dV$ |
| 关于原点的转动惯量 $I_O$ 是绕某条通过原点的轴的转动惯量 | $I_O$ 是关于**点**的转动惯量（点到原点的距离平方 $\times$ 质量元的积分），不是绕轴的。它与绕轴的转动惯量通过迹恒等式 $I_x+I_y+I_z=2I_O$ 关联 |
| 柱坐标下 $I_z$ 的被积函数为 $r^3$ 时，可以直接用 $I_z = \rho_0 \cdot \int r^3\,dr \cdot \int d\theta \cdot \int dz$ | 正确，但前提是积分区域是长方体（即 $r,\theta,z$ 的积分限都是常数），且被积函数可分离为各变量的乘积 |

### 检查点

- [ ] 能否写出绕 $z$ 轴、$x$ 轴、$y$ 轴和原点的转动惯量公式（定理13.18）？
- [ ] 能否解释为什么绕 $z$ 轴的转动惯量公式中被积函数是 $x^2+y^2$ 而非 $z^2$？

## 练习题

### 基础巩固

**1.**（质量计算）设 $\Omega$ 是圆锥体 $z = \sqrt{x^2+y^2}$、$0 \leq z \leq 2$，体密度为常数 $\rho_0 = 3$。求该圆锥体的总质量 $M$。

<details><summary>参考答案</summary>

由定理13.17：$M = \iiint_\Omega \rho_0\,dV = \rho_0 \cdot V(\Omega)$。

圆锥体的体积为 $V = \frac{1}{3}\pi R^2 h$，其中底面半径 $R = 2$（因为 $z=2$ 时 $\sqrt{x^2+y^2}=2$），高 $h=2$。

$$V(\Omega) = \frac{1}{3}\pi \cdot 2^2 \cdot 2 = \frac{8\pi}{3}$$

因此：
$$M = \rho_0 \cdot \frac{8\pi}{3} = 3 \cdot \frac{8\pi}{3} = 8\pi$$

故圆锥体的总质量为 $8\pi$（质量单位）。

**也可用柱坐标直接验证**：

柱坐标下，$\Omega': 0 \leq \theta \leq 2\pi,\; 0 \leq r \leq 2,\; r \leq z \leq 2$。

$$M = \rho_0 \int_{0}^{2\pi} \int_{0}^{2} \int_{r}^{2} r\,dz\,dr\,d\theta
= 3 \cdot 2\pi \int_{0}^{2} r(2-r)\,dr
= 6\pi \left[ r^2 - \frac{r^3}{3} \right]_{0}^{2}
= 6\pi \left(4 - \frac{8}{3}\right)
= 6\pi \cdot \frac{4}{3} = 8\pi \quad \checkmark$$

</details>

---

**2.**（转动惯量基础）设 $\Omega$ 是长方体 $0 \leq x \leq a$、$0 \leq y \leq b$、$0 \leq z \leq c$，体密度为常数 $\rho_0$。求：

**(a)** $I_z$（绕 $z$ 轴的转动惯量）

**(b)** $I_O$（关于原点的转动惯量）

<details><summary>参考答案</summary>

**(a) $I_z$ 的计算**

绕 $z$ 轴：$r^2 = x^2 + y^2$。
$$I_z = \rho_0 \iiint_\Omega (x^2 + y^2)\,dV = \rho_0 \left( \iiint_\Omega x^2\,dV + \iiint_\Omega y^2\,dV \right)$$

先计算 $\iiint_\Omega x^2\,dV$（用 Fubini 定理）：
$$\iiint_\Omega x^2\,dV = \int_{0}^{a} \int_{0}^{b} \int_{0}^{c} x^2\,dz\,dy\,dx = \left( \int_{0}^{a} x^2\,dx \right) \left( \int_{0}^{b} dy \right) \left( \int_{0}^{c} dz \right) = \frac{a^3}{3} \cdot b \cdot c = \frac{a^3 bc}{3}$$

同理：
$$\iiint_\Omega y^2\,dV = \int_{0}^{a} \int_{0}^{b} \int_{0}^{c} y^2\,dz\,dy\,dx = \left( \int_{0}^{a} dx \right) \left( \int_{0}^{b} y^2\,dy \right) \left( \int_{0}^{c} dz \right) = a \cdot \frac{b^3}{3} \cdot c = \frac{ab^3 c}{3}$$

因此：
$$I_z = \rho_0 \left( \frac{a^3 bc}{3} + \frac{ab^3 c}{3} \right) = \frac{\rho_0 abc}{3}(a^2 + b^2)$$

用总质量 $M = \rho_0 abc$ 表示：$I_z = \dfrac{M}{3}(a^2 + b^2)$。

**(b) $I_O$ 的计算**

关于原点：$r^2 = x^2 + y^2 + z^2$。
$$I_O = \rho_0 \iiint_\Omega (x^2 + y^2 + z^2)\,dV = \rho_0 \left( \iiint_\Omega x^2\,dV + \iiint_\Omega y^2\,dV + \iiint_\Omega z^2\,dV \right)$$

前两项已在 (a) 中算出。计算 $\iiint_\Omega z^2\,dV$：
$$\iiint_\Omega z^2\,dV = \left( \int_{0}^{a} dx \right) \left( \int_{0}^{b} dy \right) \left( \int_{0}^{c} z^2\,dz \right) = a \cdot b \cdot \frac{c^3}{3} = \frac{abc^3}{3}$$

因此：
$$I_O = \rho_0 \left( \frac{a^3 bc}{3} + \frac{ab^3 c}{3} + \frac{abc^3}{3} \right) = \frac{\rho_0 abc}{3}(a^2 + b^2 + c^2) = \frac{M}{3}(a^2 + b^2 + c^2)$$

**验证迹恒等式**：由对称性，$I_x = \dfrac{M}{3}(b^2 + c^2)$，$I_y = \dfrac{M}{3}(a^2 + c^2)$，$I_z = \dfrac{M}{3}(a^2 + b^2)$。

$$I_x + I_y + I_z = \frac{M}{3}(b^2+c^2 + a^2+c^2 + a^2+b^2) = \frac{2M}{3}(a^2+b^2+c^2) = 2I_O \quad \checkmark$$

</details>

---

### 迁移应用

**3.**（柱坐标下的转动惯量）设 $\Omega$ 是圆柱体 $x^2 + y^2 \leq R^2$、$0 \leq z \leq H$，体密度为 $\rho(x,y,z) = x^2 + y^2$（与例1相同，但用符号参数 $R,H$ 代替具体数值）。求：

**(a)** 总质量 $M$

**(b)** 绕 $z$ 轴的转动惯量 $I_z$

<details><summary>参考答案</summary>

**(a) 总质量 $M$**

由定理13.17：$M = \iiint_\Omega (x^2+y^2)\,dV = \iiint_\Omega r^2\,dV$（柱坐标下）。

柱坐标下 $\Omega' = \{ (r,\theta,z) \mid 0 \leq r \leq R,\; 0 \leq \theta \leq 2\pi,\; 0 \leq z \leq H \}$，$dV = r\,dr\,d\theta\,dz$：
$$M = \int_{0}^{H} \int_{0}^{2\pi} \int_{0}^{R} r^2 \cdot r\,dr\,d\theta\,dz = H \cdot 2\pi \cdot \int_{0}^{R} r^3\,dr = 2\pi H \cdot \frac{R^4}{4} = \frac{\pi HR^4}{2}$$

**(b) 绕 $z$ 轴的转动惯量 $I_z$**

由定理13.18，绕 $z$ 轴 $r^2 = x^2+y^2$。注意这里"$r$"是点到 $z$ 轴的距离，在柱坐标下正好等于柱坐标的径向坐标 $r$。为避免混淆，我们用 $d^2 = x^2+y^2 = r^2$（柱坐标中的 $r$）表示距离平方。

$$I_z = \iiint_\Omega (x^2+y^2) \cdot \rho(x,y,z)\,dV = \iiint_\Omega (x^2+y^2)^2\,dV$$

因为在本题中 $\rho = x^2+y^2$，所以 $(x^2+y^2) \cdot \rho = (x^2+y^2)^2 = r^4$。

柱坐标下：
$$I_z = \int_{0}^{H} \int_{0}^{2\pi} \int_{0}^{R} r^4 \cdot r\,dr\,d\theta\,dz = H \cdot 2\pi \cdot \int_{0}^{R} r^5\,dr = 2\pi H \cdot \frac{R^6}{6} = \frac{\pi HR^6}{3}$$

因此总质量为 $\displaystyle M = \frac{\pi HR^4}{2}$，绕 $z$ 轴的转动惯量为 $\displaystyle I_z = \frac{\pi HR^6}{3}$。

</details>

---

**4.**（转动惯量的对称性应用）利用对称性和前面例题的结果，快速计算以下问题：

设 $\Omega$ 是半径为 $R$ 的球体 $x^2+y^2+z^2 \leq R^2$，体密度为常数 $\rho_0$。求 $\displaystyle \iiint_\Omega x^2\,dV$。

<details><summary>参考答案</summary>

由对称性，$\iiint_\Omega x^2\,dV = \iiint_\Omega y^2\,dV = \iiint_\Omega z^2\,dV$。

由例3的第(2)问，均匀球体绕 $z$ 轴的转动惯量为：
$$I_z = \rho_0 \iiint_\Omega (x^2+y^2)\,dV = \frac{8\pi\rho_0 R^5}{15}$$

另一方面：
$$\iiint_\Omega (x^2+y^2)\,dV = \iiint_\Omega x^2\,dV + \iiint_\Omega y^2\,dV = 2\iiint_\Omega x^2\,dV$$

因此：
$$\iiint_\Omega x^2\,dV = \frac{1}{2} \iiint_\Omega (x^2+y^2)\,dV = \frac{1}{2} \cdot \frac{8\pi R^5}{15} = \frac{4\pi R^5}{15}$$

**验证**：由例3第(1)问，$I_O = \rho_0 \iiint_\Omega (x^2+y^2+z^2)\,dV = \frac{4\pi\rho_0 R^5}{5}$，所以：
$$\iiint_\Omega (x^2+y^2+z^2)\,dV = \frac{4\pi R^5}{5}$$

而由对称性，$\iiint_\Omega x^2\,dV = \frac{1}{3} \iiint_\Omega (x^2+y^2+z^2)\,dV = \frac{1}{3} \cdot \frac{4\pi R^5}{5} = \frac{4\pi R^5}{15}$。

两种方法结果一致。$\checkmark$

由此可见，已知转动惯量后，可以利用对称性反过来求某些三重积分的值——这是转动惯量公式的另一种应用。

</details>
