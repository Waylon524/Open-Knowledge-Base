## 2. 背景与应用场景

### 2.1 从长方体到一般空间区域

在文件01（定理13.6）中，我们学习了长方体上的三重积分Fubini定理：

$$\iiint_{[a,b]\times[c,d]\times[e,f]} f(x,y,z)\,dV = \int_a^b \int_c^d \int_e^f f(x,y,z)\,dz\,dy\,dx$$

但它只适用于长方体区域——三个方向的积分限都是常数。然而实际中遇到的区域（如四面体、圆锥、球体）往往不是长方体。例如：

- **四面体**：由三个坐标面和平面 $x+y+z = 1$ 围成的区域
- **圆锥体**：$z = \sqrt{x^2 + y^2}$ 和 $z = h$ 围成的区域
- **球体**：$x^2 + y^2 + z^2 \leq R^2$

在这些区域上，至少有一个方向的边界是曲面而非平面，积分限不再是常数，而是变量的函数。我们需要将Fubini定理从长方体推广到一般区域。

### 2.2 两种思路：投影法与截面法

从二重积分在一般区域上的计算经验（文件02）我们知道，关键在于**用不等式组描述区域**，然后确定合适的积分次序。

对于三重积分，有两种自然的推广思路：

1. **投影法（先一后二）**：先在一个方向上积分（如 $z$ 方向），再在剩下的二维投影区域上做二重积分。这类似于二重积分中"先在一个方向上积分，再在另一个方向上积分"的思路的三维类比。

2. **截面法（先二后一）**：先固定一个方向（如 $z$），在该方向的截面上做二重积分，再对 $z$ 方向积分。这类似于"切片"——把三维物体切成薄片，每片上的量用二重积分计算，然后沿垂直方向累加。

两种方法本质上是**不同的积分次序**，适用于不同形状的区域。

## 3. 核心概念与符号约定

### 3.1 一般空间区域的不等式组描述

与二重积分类似，计算三重积分的第一步是将三维区域 $\Omega$ 用不等式组精确描述。

**定义（一般空间区域的描述）**：一个有界闭区域 $\Omega \subset \mathbb{R}^3$ 如果可以表示为：

$$\Omega = \big\{ (x,y,z) \mid (x,y) \in D,\; z_1(x,y) \leq z \leq z_2(x,y) \big\}$$

其中 $D \subset \mathbb{R}^2$ 是 $\Omega$ 在 $xy$ 平面上的投影区域，$z_1(x,y)$ 和 $z_2(x,y)$ 是定义在 $D$ 上的连续函数，且 $z_1(x,y) \leq z_2(x,y)$，则称 $\Omega$ 在 $z$ 方向上是**简单区域**（$z$-simple region）。

类似地可以定义在 $y$ 方向或 $x$ 方向上的简单区域。

**例 1（四面体）**：由 $x \geq 0$、$y \geq 0$、$z \geq 0$、$x+y+z \leq 1$ 围成的四面体区域 $\Omega$：

- $z$ 方向：$0 \leq z \leq 1-x-y$
- 投影到 $xy$ 平面：$D = \{(x,y) \mid x \geq 0,\ y \geq 0,\ x+y \leq 1\}$
- $\Omega = \{(x,y,z) \mid 0 \leq x \leq 1,\ 0 \leq y \leq 1-x,\ 0 \leq z \leq 1-x-y\}$

**例 2（圆锥体）**：由 $z = \sqrt{x^2 + y^2}$ 和 $z = h$ 围成的圆锥体 $\Omega$：

- $z$ 方向：$\sqrt{x^2 + y^2} \leq z \leq h$
- 投影到 $xy$ 平面：$D = \{(x,y) \mid x^2 + y^2 \leq h^2\}$（半径为 $h$ 的圆盘）
- $\Omega = \{(x,y,z) \mid x^2 + y^2 \leq h^2,\ \sqrt{x^2 + y^2} \leq z \leq h\}$

**例 3（球体）**：$x^2 + y^2 + z^2 \leq R^2$ 的球体 $\Omega$：

- $z$ 方向：$-\sqrt{R^2 - x^2 - y^2} \leq z \leq \sqrt{R^2 - x^2 - y^2}$
- 投影到 $xy$ 平面：$D = \{(x,y) \mid x^2 + y^2 \leq R^2\}$

### 3.2 投影法（先一后二）

**定理 13.11（投影法 / Projection Method / "先一后二"）**：设 $\Omega \subset \mathbb{R}^3$ 是一个有界闭区域，且可以表示为：

$$\Omega = \big\{ (x,y,z) \mid (x,y) \in D,\; z_1(x,y) \leq z \leq z_2(x,y) \big\}$$

其中 $D$ 是 $\Omega$ 在 $xy$ 平面上的投影区域（零边界有界闭区域），$z_1(x,y)$ 和 $z_2(x,y)$ 在 $D$ 上连续。则对在 $\Omega$ 上连续的函数 $f(x,y,z)$，有：

$$\boxed{\iiint_\Omega f(x,y,z)\,dV = \iint_D \left( \int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z)\,dz \right) dxdy}$$

**说明**：

1. **计算次序**：先对 $z$ 积分（将 $x,y$ 视为常数），再在 $D$ 上做二重积分——这就是"先一后二"的含义（先做一维积分，再做二维积分）。
2. **投影方向的灵活性**：也可以投影到 $yz$ 平面或 $xz$ 平面，相应地先对 $x$ 积分或先对 $y$ 积分。
3. **理论基础**：将 $f$ 在 $\Omega$ 外部延拓为 $0$，然后应用长方体上的Fubini定理（定理13.6），与文件02中二重积分从矩形到一般区域的推广思路完全一致。
4. **对 $D$ 的要求**：$D$ 是一个有界闭区域，可以用X-型或Y-型的不等式组描述（为最终化为累次积分做准备）。

**"先一后二"的命名含义**：先做一维定积分（对 $z$），再做二重积分（在 $D$ 上）——"一"指一维，"二"指二维，合起来正好是三重积分的三维。

### 3.3 截面法（先二后一）

**定理 13.12（截面法 / Cross-section Method / "先二后一"）**：设 $\Omega \subset \mathbb{R}^3$ 是一个有界闭区域，且对每个 $z \in [z_{\min}, z_{\max}]$，截面

$$D(z) = \big\{ (x,y) \mid (x,y,z) \in \Omega \big\}$$

是非空的零边界有界闭区域，且 $z \mapsto D(z)$ 连续变化（即 $D(z)$ 的边界连续依赖于 $z$）。则对在 $\Omega$ 上连续的函数 $f(x,y,z)$，有：

$$\boxed{\iiint_\Omega f(x,y,z)\,dV = \int_{z_{\min}}^{z_{\max}} \left( \iint_{D(z)} f(x,y,z)\,dxdy \right) dz}$$

**说明**：

1. **计算次序**：先对 $x,y$ 在截面 $D(z)$ 上做二重积分（将 $z$ 视为常数），再对 $z$ 积分——"先二后一"。
2. **$D(z)$ 的描述**：截面 $D(z)$ 可以用X-型或Y-型描述，其边界一般依赖于 $z$。
3. **适用条件**：当截面 $D(z)$ 的形状简单（圆盘、三角形、矩形等）时，截面法非常有效。
4. **截面法的几何直观**：将三维物体切成平行于 $xy$ 平面的薄片，每片的"贡献"是 $\big(\iint_{D(z)} f\,dxdy\big) dz$，然后沿 $z$ 方向累加。

**"先二后一"的命名含义**：先做二重积分（在截面 $D(z)$ 上），再做一维积分（对 $z$）——次序与投影法相反。

### 3.4 符号表

| 符号 | 含义 | 说明 |
|:---|:---|:---|
| $\Omega$ | 三重积分的积分区域（三维有界闭区域） | $\Omega \subset \mathbb{R}^3$ |
| $D$ | $\Omega$ 在某坐标平面上的投影区域 | 投影法使用 |
| $z_1(x,y)$ | $z$ 方向的下边界函数 | 投影法：$z$ 积分下限 |
| $z_2(x,y)$ | $z$ 方向的上边界函数 | 投影法：$z$ 积分上限 |
| $D(z)$ | 高度 $z$ 处的截面区域 | 截面法使用 |
| $z_{\min}, z_{\max}$ | $z$ 的最小值和最大值 | 截面法：$z$ 的积分范围 |
| $\iiint_\Omega f\,dV$ | $f$ 在 $\Omega$ 上的三重积分 | $dV = dxdydz$ |
| $\iint_{D(z)} f\,dxdy$ | 截面 $D(z)$ 上的二重积分 | 将 $z$ 视为常数 |

## 4. 原理与方法

### 4.1 投影法（先一后二）的完整流程

**第 1 步：确定投影区域**
确定 $\Omega$ 在 $xy$ 平面上的投影区域 $D$。方法是消去 $z$：将 $\Omega$ 的边界中所有 $z$ 的限制去掉，只保留 $x,y$ 的约束。例如：
- 四面体 $x \geq 0, y \geq 0, z \geq 0, x+y+z \leq 1$ → 消去 $z$ 得 $x \geq 0, y \geq 0, x+y \leq 1$
- 圆锥 $0 \leq z \leq h, \sqrt{x^2+y^2} \leq z$ → 消去 $z$ 得 $x^2+y^2 \leq h^2$

**第 2 步：确定 $z$ 的积分限**
对每个 $(x,y) \in D$，确定 $z$ 的下边界 $z = z_1(x,y)$ 和上边界 $z = z_2(x,y)$。用不等式描述：
$$\Omega = \big\{ (x,y,z) \mid (x,y) \in D,\; z_1(x,y) \leq z \leq z_2(x,y) \big\}$$

**第 3 步：将投影区域 $D$ 用X-型或Y-型描述（调用文件02的定理13.8）**
- 若 $D$ 是X-型：$D = \{(x,y) \mid a \leq x \leq b,\ \varphi_1(x) \leq y \leq \varphi_2(x)\}$
- 若 $D$ 是Y-型：$D = \{(x,y) \mid c \leq y \leq d,\ \psi_1(y) \leq x \leq \psi_2(y)\}$

**第 4 步：写出累次积分**
X-型描述下的累次积分（先对 $z$，再对 $y$，再对 $x$）：
$$\iiint_\Omega f\,dV = \int_a^b \left( \int_{\varphi_1(x)}^{\varphi_2(x)} \left( \int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z)\,dz \right) dy \right) dx$$

Y-型描述下的累次积分（先对 $z$，再对 $x$，再对 $y$）：
$$\iiint_\Omega f\,dV = \int_c^d \left( \int_{\psi_1(y)}^{\psi_2(y)} \left( \int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z)\,dz \right) dx \right) dy$$

**第 5 步：逐层计算**
从最内层积分开始，逐层向外计算。每计算一层，就消去一个变量，最终得到一个数值。

**关键要点**：
- 内层积分（对 $z$）计算时，$x,y$ 视为常数
- 中层积分（对 $y$ 或 $x$）计算时，外层变量视为常数
- 代入上下限时，下层积分限可能是外层变量的函数
- 每一步都是一元定积分

**投影到其他坐标平面**：如果投影到 $yz$ 平面更方便，则：
$$\iiint_\Omega f\,dV = \iint_{D_{yz}} \left( \int_{x_1(y,z)}^{x_2(y,z)} f(x,y,z)\,dx \right) dydz$$

类似地可以投影到 $xz$ 平面。选择投影到哪个平面，取决于哪个方向上的边界函数最简单、投影区域的形状最规则。

### 4.2 截面法（先二后一）的完整流程

**第 1 步：确定 $z$ 的范围**
找到 $\Omega$ 在 $z$ 方向上的最小值 $z_{\min}$ 和最大值 $z_{\max}$。例如：
- 四面体 $x \geq 0, y \geq 0, z \geq 0, x+y+z \leq 1$：$z \in [0, 1]$
- 圆锥 $0 \leq z \leq h, \sqrt{x^2+y^2} \leq z$：$z \in [0, h]$
- 球体 $x^2+y^2+z^2 \leq R^2$：$z \in [-R, R]$

**第 2 步：确定截面 $D(z)$**
对每个 $z \in [z_{\min}, z_{\max}]$，确定该高度处截面区域 $D(z)$ 的描述。方法是将 $z$ 视为常数，找出 $x,y$ 满足的约束。

例如，对圆锥体 $\sqrt{x^2+y^2} \leq z$：固定 $z$，截面 $D(z)$ 是半径为 $z$ 的圆盘：
$$D(z) = \big\{ (x,y) \mid x^2+y^2 \leq z^2 \big\}$$

对四面体 $x+y+z \leq 1$：固定 $z$，截面 $D(z)$ 是三角形：
$$D(z) = \big\{ (x,y) \mid x \geq 0,\ y \geq 0,\ x+y \leq 1-z \big\}$$

**第 3 步：在截面 $D(z)$ 上做二重积分**
对每个 $z$，将 $z$ 视为常数，计算截面 $D(z)$ 上的二重积分：
$$F(z) = \iint_{D(z)} f(x,y,z)\,dxdy$$

这一步中 $D(z)$ 可以用X-型或Y-型描述，必要时使用极坐标变换。

**第 4 步：对 $z$ 积分**
将 $F(z)$ 关于 $z$ 在 $[z_{\min}, z_{\max}]$ 上积分：
$$\iiint_\Omega f\,dV = \int_{z_{\min}}^{z_{\max}} F(z)\,dz$$

**截面法的关键优势**：当 $f(x,y,z)$ 只依赖于 $z$（即 $f = g(z)$）时：
$$\iint_{D(z)} g(z)\,dxdy = g(z) \cdot \text{Area}(D(z))$$

这时截面法的内层二重积分简化为 $g(z)$ 乘以截面面积，计算极大简化。

**当截面形状简单时的常用技巧**：
- 截面为圆盘：用极坐标变换（$dxdy = r\,dr\,d\theta$），$r$ 的上限可能依赖于 $z$
- 截面为三角形：用X-型或Y-型描述，按二重积分的方法计算

### 4.3 截面法与极坐标变换的组合

当截面 $D(z)$ 是圆盘（或圆环、扇形等）时，在截面上的二重积分可以借助极坐标变换来计算。这是极为常见的技巧组合。

**形式化描述**：若对每个 $z \in [z_{\min}, z_{\max}]$，截面 $D(z)$ 在极坐标下可描述为：
$$D(z) = \big\{ (r,\theta) \mid 0 \leq r \leq r_{\max}(z),\ \alpha(z) \leq \theta \leq \beta(z) \big\}$$

则：
$$\iiint_\Omega f(x,y,z)\,dV = \int_{z_{\min}}^{z_{\max}} \left( \int_{\alpha(z)}^{\beta(z)} \left( \int_0^{r_{\max}(z)} f(r\cos\theta, r\sin\theta, z) \cdot r\,dr \right) d\theta \right) dz$$

**关键注意**：
- 极坐标变换只对 $x,y$ 进行（即在 $D(z)$ 上使用），$z$ 仍然是一个参数
- $r$ 的积分上限 $r_{\max}(z)$ 可能依赖于 $z$，因为截面的大小随 $z$ 变化
- 面积元素 $dxdy = r\,dr\,d\theta$（与文件03的结论一致）
- 当 $f$ 与 $\theta$ 无关时（如 $f = f(r,z)$），$\theta$ 的积分可以独立进行，得到 $2\pi$（或相应的角度跨度）

**典型应用场景**：
- 圆锥体：$D(z)$ 为半径 $r = \frac{R}{h}z$ 的圆盘（$R$ 为底面半径，$h$ 为高）
- 球体：$D(z)$ 为半径 $r = \sqrt{R^2 - z^2}$ 的圆盘
- 旋转抛物面：$D(z)$ 为半径 $r = \sqrt{z}$ 的圆盘

### 4.4 积分次序的选择策略

投影法（先一后二）和截面法（先二后一）给出了两种不同的积分次序。**对同一个区域，选择不同的方法会导致不同的累次积分形式**，计算的难易程度可能有显著差异。

| 对比维度 | 投影法（先一后二） | 截面法（先二后一） |
|:---|:---|:---|
| 积分次序 | 先对 $z$ 积分 → 再在 $D$ 上做二重积分 | 先在 $D(z)$ 上做二重积分 → 再对 $z$ 积分 |
| 适用条件 | $z$ 的边界由显式函数 $z = z(x,y)$ 给出，投影区域 $D$ 形状简单 | 各截面 $D(z)$ 的形状简单（圆、三角形、矩形等） |
| 第一步 | 确定 $z$ 的范围（关于 $x,y$ 的函数） | 确定 $z$ 的范围（常数） |
| 被积函数依赖 | 适合 $f$ 与 $z$ 的关系简单（如 $f$ 是 $z$ 的多项式） | 适合 $f$ 只依赖于 $z$ 或 $f$ 在截面上容易积分 |
| 几何直觉 | "穿针引线"——竖直线穿过区域 | "切片"——将物体切成薄片再累加 |

**选择建议**：

1. **若 $z$ 方向边界由显式函数给出，且投影区域 $D$ 是规则的 X-型/Y-型区域** → 用投影法
   - 如四面体、由 $z = f(x,y)$ 和 $z = g(x,y)$ 围成的区域

2. **若截面 $D(z)$ 的形状简单（圆盘、三角形、矩形等）** → 用截面法
   - 如圆锥体、球体、旋转体

3. **若被积函数 $f$ 只依赖于 $z$（即 $f = g(z)$）** → 强烈推荐截面法
   - 因为 $\iint_{D(z)} g(z)\,dxdy = g(z) \cdot \text{Area}(D(z))$，内层积分直接化为面积计算

4. **若截面为圆盘，且 $f$ 适合极坐标** → 截面法 + 极坐标变换
   - 这是圆锥体、球体等旋转对称区域的标准处理方法

5. **投影平面的灵活选择**：投影法不一定必须投影到 $xy$ 平面。如果 $x$ 方向的边界更简单，可以投影到 $yz$ 平面，先对 $x$ 积分。同理可以投影到 $xz$ 平面，先对 $y$ 积分。

$$\iiint_\Omega f\,dV = \iint_{D_{yz}} \left( \int_{x_1(y,z)}^{x_2(y,z)} f\,dx \right) dydz
= \iint_{D_{xz}} \left( \int_{y_1(x,z)}^{y_2(x,z)} f\,dy \right) dxdz$$

**一般原则**：选择使得**内层积分限最简单**的次序和方法。当区域在某个方向上的边界描述复杂时，避免在该方向上先积分。

## 5. 例题

### 例 1：投影法（先一后二）——四面体上的三重积分

计算 $\displaystyle I = \iiint_\Omega x\,dV$，其中 $\Omega$ 是由 $x=0$、$y=0$、$z=0$、$x+2y+3z=6$ 所围成的四面体区域。

**第 1 步：确定 $z$ 的积分限**

从 $x+2y+3z=6$ 解出 $z$：
$$3z = 6 - x - 2y \quad\Longrightarrow\quad z = \frac{6 - x - 2y}{3}$$

四面体的下边界是 $z=0$（$xy$ 平面），上边界是斜面 $z = (6-x-2y)/3$。注意在上边界中，需要 $6-x-2y \geq 0$ 才能保证 $z \geq 0$。

因此：$0 \leq z \leq \dfrac{6 - x - 2y}{3}$。

**第 2 步：确定投影区域 $D$（投影到 $xy$ 平面）**

投影到 $xy$ 平面时，$z$ 的限制被移除，保留 $x,y$ 的约束：
- $x \geq 0$（来自 $x=0$ 的边界）
- $y \geq 0$（来自 $y=0$ 的边界）
- $x + 2y \leq 6$（来自 $z \geq 0$ 的要求，即 $6 - x - 2y \geq 0$）

因此 $D$ 是一个三角形区域：
$$D = \big\{ (x,y) \mid x \geq 0,\ y \geq 0,\ x + 2y \leq 6 \big\}$$

**第 3 步：用X-型描述投影区域**

将 $D$ 用X-型描述：
- $x$ 的范围：从 $0$ 到 $6$（当 $y=0$ 时 $x = 6$）
- 对每个 $x$，$y$ 的范围：从 $0$ 到 $(6-x)/2$

$$D = \big\{ (x,y) \mid 0 \leq x \leq 6,\ 0 \leq y \leq (6-x)/2 \big\}$$

**第 4 步：写出累次积分**

由定理13.11（投影法）：
$$I = \iiint_\Omega x\,dV = \iint_D \left( \int_0^{(6-x-2y)/3} x\,dz \right) dxdy$$

将 $D$ 的X-型描述代入：
$$I = \int_{x=0}^{6} \int_{y=0}^{(6-x)/2} \left( \int_{z=0}^{(6-x-2y)/3} x\,dz \right) dy\,dx$$

**第 5 步：逐层计算**

先对 $z$ 积分（将 $x,y$ 视为常数）：
$$\int_{0}^{(6-x-2y)/3} x\,dz = x \cdot \left[ z \right]_{0}^{(6-x-2y)/3} = x \cdot \frac{6-x-2y}{3}$$

再对 $y$ 积分（将 $x$ 视为常数）：
$$
\begin{aligned}
\int_{0}^{(6-x)/2} \frac{x(6-x-2y)}{3}\,dy
&= \frac{x}{3} \int_{0}^{(6-x)/2} (6-x-2y)\,dy \\
&= \frac{x}{3} \left[ (6-x)y - y^2 \right]_{0}^{(6-x)/2} \\
&= \frac{x}{3} \left[ (6-x)\cdot\frac{6-x}{2} - \frac{(6-x)^2}{4} \right] \\
&= \frac{x}{3} \left[ \frac{(6-x)^2}{2} - \frac{(6-x)^2}{4} \right] \\
&= \frac{x}{3} \cdot \frac{(6-x)^2}{4} = \frac{x(6-x)^2}{12}
\end{aligned}
$$

最后对 $x$ 积分：
$$
\begin{aligned}
I &= \int_{0}^{6} \frac{x(6-x)^2}{12}\,dx \\
&= \frac{1}{12} \int_{0}^{6} x(36 - 12x + x^2)\,dx \\
&= \frac{1}{12} \int_{0}^{6} (36x - 12x^2 + x^3)\,dx \\
&= \frac{1}{12} \left[ 18x^2 - 4x^3 + \frac{x^4}{4} \right]_{0}^{6} \\
&= \frac{1}{12} \left( 18 \cdot 36 - 4 \cdot 216 + \frac{1296}{4} \right) \\
&= \frac{1}{12} (648 - 864 + 324) \\
&= \frac{1}{12} \cdot 108 = 9
\end{aligned}
$$

因此 $\displaystyle \iiint_\Omega x\,dV = 9$。

**验证**：四面体的顶点为 $(0,0,0)$、$(6,0,0)$、$(0,3,0)$、$(0,0,2)$，体积为 $\frac{1}{6} \times 6 \times 3 \times 2 = 6$。积分值 $9$ 大于体积，合理（因为被积函数 $x$ 在区域上非负且平均值为 $1.5$）。

### 例 2：截面法（先二后一）——圆锥体上的三重积分

计算 $\displaystyle I = \iiint_\Omega (x^2 + y^2)\,dV$，其中 $\Omega$ 是由 $z = \sqrt{x^2 + y^2}$ 和 $z = 2$ 所围成的圆锥体（顶点在原点，底面在 $z=2$）。

**第 1 步：确定 $z$ 的范围**

圆锥的底部在 $z=0$（顶点），顶部在 $z=2$（底面），因此：
$$z \in [0, 2]$$

**第 2 步：确定截面 $D(z)$**

在高度 $z$ 处，截面由 $z = \sqrt{x^2 + y^2}$ 决定：$x^2 + y^2 \leq z^2$。因此截面 $D(z)$ 是半径为 $z$ 的圆盘：
$$D(z) = \big\{ (x,y) \mid x^2 + y^2 \leq z^2 \big\}$$

**第 3 步：在截面 $D(z)$ 上用极坐标变换做二重积分**

由于 $D(z)$ 是圆盘且被积函数 $x^2+y^2$ 适合极坐标变换，我们在截面上使用极坐标：
$$x = r\cos\theta,\quad y = r\sin\theta,\quad dxdy = r\,dr\,d\theta$$

在固定 $z$ 的条件下，$D(z)$ 在极坐标下的描述为：
$$0 \leq r \leq z,\quad 0 \leq \theta \leq 2\pi$$

注意：这里 $r$ 的上限 $z$ 依赖于外层积分变量 $z$——这是截面法与极坐标组合的关键特征。

被积函数 $(x^2+y^2)$ 在极坐标下变为 $r^2$。

因此内层二重积分计算如下：
$$
\begin{aligned}
F(z) &= \iint_{D(z)} (x^2 + y^2)\,dxdy \\
&= \int_{\theta=0}^{2\pi} \int_{r=0}^{z} r^2 \cdot r\,dr\,d\theta \\
&= \int_{0}^{2\pi} d\theta \cdot \int_{0}^{z} r^3\,dr \\
&= 2\pi \cdot \frac{z^4}{4} = \frac{\pi z^4}{2}
\end{aligned}
$$

**第 4 步：对 $z$ 积分**

$$
\begin{aligned}
I &= \int_{z=0}^{2} F(z)\,dz = \int_{0}^{2} \frac{\pi z^4}{2}\,dz \\
&= \frac{\pi}{2} \left[ \frac{z^5}{5} \right]_{0}^{2} = \frac{\pi}{2} \cdot \frac{32}{5} = \frac{16\pi}{5}
\end{aligned}
$$

因此 $\displaystyle \iiint_\Omega (x^2 + y^2)\,dV = \frac{16\pi}{5}$。

**验证（用投影法对比）**：

用投影法重新计算，以验证结果。将 $\Omega$ 投影到 $xy$ 平面：$x^2 + y^2 \leq 4$（半径 $2$ 的圆盘），$z$ 从 $\sqrt{x^2+y^2}$ 到 $2$。

$$I = \iint_{x^2+y^2 \leq 4} \left( \int_{\sqrt{x^2+y^2}}^{2} (x^2 + y^2)\,dz \right) dxdy$$

先用极坐标：
$$I = \int_{0}^{2\pi} \int_{0}^{2} \left( \int_{r}^{2} r^2\,dz \right) r\,dr\,d\theta$$

内层对 $z$ 积分：$\int_{r}^{2} r^2\,dz = r^2(2 - r)$。

$$I = \int_{0}^{2\pi} \int_{0}^{2} r^2(2 - r) \cdot r\,dr\,d\theta = 2\pi \int_{0}^{2} (2r^3 - r^4)\,dr = 2\pi \left[ \frac{r^4}{2} - \frac{r^5}{5} \right]_{0}^{2} = 2\pi \left( 8 - \frac{32}{5} \right) = 2\pi \cdot \frac{8}{5} = \frac{16\pi}{5}$$

结果一致。$\checkmark$

**方法对比**：本题中截面法比投影法更自然，因为在截面 $D(z)$ 上做二重积分时，$r$ 的上限 $z$ 简洁明了；而投影法需要处理 $z$ 的下限 $\sqrt{x^2+y^2}$，计算稍复杂。

### 例 3：截面法与极坐标组合——球体上的三重积分

计算 $\displaystyle I = \iiint_\Omega z^2\,dV$，其中 $\Omega$ 是半径为 $R$ 的球体 $x^2 + y^2 + z^2 \leq R^2$。

**第 1 步：确定 $z$ 的范围**

球体在 $z$ 方向的范围：$z \in [-R, R]$。

**第 2 步：确定截面 $D(z)$**

在高度 $z$ 处（$-R \leq z \leq R$），截面由球面方程决定：
$$x^2 + y^2 \leq R^2 - z^2$$

因此 $D(z)$ 是半径为 $\sqrt{R^2 - z^2}$ 的圆盘。当 $z = 0$ 时，截面半径最大为 $R$；当 $z = \pm R$ 时，截面半径为零。

**第 3 步：在截面 $D(z)$ 上用极坐标变换做二重积分**

被积函数 $z^2$ 在截面 $D(z)$ 上是一个常数（因为 $z$ 固定），所以：
$$\iint_{D(z)} z^2\,dxdy = z^2 \cdot \text{Area}(D(z)) = z^2 \cdot \pi (R^2 - z^2)$$

为了完整展示截面法与极坐标变换的结合，我们显式计算这个二重积分：

在极坐标下，$D(z)$ 的描述为：
$$0 \leq r \leq \sqrt{R^2 - z^2},\quad 0 \leq \theta \leq 2\pi$$

注意：这里 $r$ 的上限 $\sqrt{R^2 - z^2}$ 依赖于 $z$——被积函数 $z^2$ 在截面上是常数，但 $r$ 界限随 $z$ 变化。

$$
\begin{aligned}
\iint_{D(z)} z^2\,dxdy &= z^2 \iint_{D(z)} 1\,dxdy \\
&= z^2 \int_{0}^{2\pi} \int_{0}^{\sqrt{R^2 - z^2}} r\,dr\,d\theta \\
&= z^2 \cdot \int_{0}^{2\pi} d\theta \cdot \int_{0}^{\sqrt{R^2 - z^2}} r\,dr \\
&= z^2 \cdot 2\pi \cdot \frac{R^2 - z^2}{2} \\
&= \pi z^2 (R^2 - z^2)
\end{aligned}
$$

**第 4 步：对 $z$ 积分**

$$
\begin{aligned}
I &= \int_{z=-R}^{R} \pi z^2 (R^2 - z^2)\,dz \\
&= \pi \int_{-R}^{R} (R^2 z^2 - z^4)\,dz \\
&= 2\pi \int_{0}^{R} (R^2 z^2 - z^4)\,dz \quad (\text{利用被积函数的偶性}) \\
&= 2\pi \left[ R^2 \cdot \frac{z^3}{3} - \frac{z^5}{5} \right]_{0}^{R} \\
&= 2\pi \left( \frac{R^5}{3} - \frac{R^5}{5} \right) \\
&= 2\pi \cdot \frac{2R^5}{15} = \frac{4\pi R^5}{15}
\end{aligned}
$$

因此 $\displaystyle \iiint_\Omega z^2\,dV = \frac{4\pi R^5}{15}$。

**讨论**：由对称性，$\iiint_\Omega x^2\,dV = \iiint_\Omega y^2\,dV = \iiint_\Omega z^2\,dV = \frac{4\pi R^5}{15}$。由此可得球体的转动惯量等物理量。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|:---|:---|
| 投影法（先一后二）必须投影到 $xy$ 平面、先对 $z$ 积分 | 可以根据区域形状选择投影到 $xy$、$yz$ 或 $xz$ 平面，相应地对 $z$、$x$ 或 $y$ 先积分 |
| 截面法（先二后一）只适用于旋转对称区域 | 截面法适用于任何 $D(z)$ 形状简单的区域，不限于旋转对称。$D(z)$ 可以是三角形、矩形等其他形状 |
| 截面法中，截面 $D(z)$ 的形状和大小与 $z$ 无关 | $D(z)$ 一般依赖于 $z$（如圆锥截面的半径、四面体截面的边长都随 $z$ 变化） |
| 截面法中的极坐标变换，$r$ 的积分限是常数 | 截面法中使用极坐标时，$r$ 的上限可以依赖于 $z$（因为 $z$ 是外层积分变量，在内部二重积分中被视为常数） |
| 对 $z$ 积分时下限一定在 $xy$ 平面（$z=0$） | $z$ 的上下界由区域边界曲面决定，不一定从 $0$ 开始（如球体的 $z$ 从 $-R$ 到 $R$） |
| 截面法 $\iint_{D(z)} f\,dxdy$ 中的 $z$ 是积分变量，所以不能当作常数处理 | 在截面法内部二重积分中，$z$ 是外层积分变量，内部积分时 $z$ 被视为常数——这正是"先二后一"的含义 |

### 检查点

- [ ] 能否用不等式组描述一个四面体/圆锥体/球体区域？
- [ ] 能否写出投影法（先一后二）的公式 $\displaystyle \iiint_\Omega f\,dV = \iint_D \left( \int_{z_1(x,y)}^{z_2(x,y)} f\,dz \right) dxdy$ 并说明每一步的含义？
- [ ] 能否写出截面法（先二后一）的公式 $\displaystyle \iiint_\Omega f\,dV = \int_{z_{\min}}^{z_{\max}} \left( \iint_{D(z)} f\,dxdy \right) dz$ 并说明每一步的含义？
- [ ] 给定一个区域，能否判断应该用投影法还是截面法？
- [ ] 能否在截面 $D(z)$ 上使用极坐标变换，正确处理 $r$ 的上限与 $z$ 的关系？
- [ ] 能否将投影法自然地推广到投影到 $yz$ 平面或 $xz$ 平面？
- [ ] 同样的区域用投影法和截面法会得到不同的累次积分形式，如何验证两者结果一致？

## 练习题

### 基础巩固

**1.**（投影法）计算 $\displaystyle I = \iiint_\Omega z\,dV$，其中 $\Omega$ 是由 $x=0$、$y=0$、$z=0$、$x+y+z=1$ 所围成的四面体区域。

<details><summary>参考答案</summary>

**第 1 步：确定区域的不等式描述**
$$\Omega = \big\{ (x,y,z) \mid 0 \leq x \leq 1,\ 0 \leq y \leq 1-x,\ 0 \leq z \leq 1-x-y \big\}$$

投影到 $xy$ 平面：$D = \{(x,y) \mid 0 \leq x \leq 1,\ 0 \leq y \leq 1-x\}$。

**第 2 步：写出累次积分**
$$I = \int_{0}^{1} \int_{0}^{1-x} \left( \int_{0}^{1-x-y} z\,dz \right) dy\,dx$$

**第 3 步：逐层计算**

先对 $z$ 积分：
$$\int_{0}^{1-x-y} z\,dz = \left[ \frac{z^2}{2} \right]_{0}^{1-x-y} = \frac{(1-x-y)^2}{2}$$

再对 $y$ 积分：
$$
\begin{aligned}
\int_{0}^{1-x} \frac{(1-x-y)^2}{2}\,dy
&= \frac{1}{2} \int_{0}^{1-x} (1-x-y)^2\,dy \\
&\text{令 } u = 1-x-y,\ du = -dy,\ y=0 \to u=1-x,\ y=1-x \to u=0 \\
&= \frac{1}{2} \int_{1-x}^{0} u^2 (-du) = \frac{1}{2} \int_{0}^{1-x} u^2\,du \\
&= \frac{1}{2} \cdot \frac{(1-x)^3}{3} = \frac{(1-x)^3}{6}
\end{aligned}
$$

最后对 $x$ 积分：
$$I = \int_{0}^{1} \frac{(1-x)^3}{6}\,dx = \frac{1}{6} \left[ -\frac{(1-x)^4}{4} \right]_{0}^{1} = \frac{1}{6} \cdot \frac{1}{4} = \frac{1}{24}$$

因此 $\displaystyle \iiint_\Omega z\,dV = \frac{1}{24}$。

**验证**：由对称性，$\iiint_\Omega x\,dV = \iiint_\Omega y\,dV = \iiint_\Omega z\,dV = \frac{1}{24}$。四面体体积为 $\frac{1}{6}$，$x+y+z$ 的平均值应为 $\frac{1}{4}$，与 $\frac{1/24}{1/6} = \frac{1}{4}$ 一致。$\checkmark$

</details>

**2.**（截面法）用截面法计算 $\displaystyle I = \iiint_\Omega 1\,dV$（即 $\Omega$ 的体积），其中 $\Omega$ 是由 $z = x^2 + y^2$ 和 $z = 4$ 所围成的区域（旋转抛物面截出的立体）。

<details><summary>参考答案</summary>

**第 1 步：确定 $z$ 的范围**
抛物面 $z = x^2 + y^2$ 在 $z=0$ 处为顶点，被 $z=4$ 截断，所以 $z \in [0, 4]$。

**第 2 步：确定截面 $D(z)$**
在高度 $z$ 处，由 $z = x^2 + y^2$ 得：
$$x^2 + y^2 \leq z$$
因此 $D(z)$ 是半径为 $\sqrt{z}$ 的圆盘。
$$\text{Area}(D(z)) = \pi (\sqrt{z})^2 = \pi z$$

**第 3 步：对被积函数 $f=1$ 使用截面法**
由于 $f=1$ 在截面上与 $x,y$ 无关：
$$\iint_{D(z)} 1\,dxdy = \text{Area}(D(z)) = \pi z$$

**第 4 步：对 $z$ 积分**
$$I = \int_{0}^{4} \pi z\,dz = \pi \left[ \frac{z^2}{2} \right]_{0}^{4} = \pi \cdot \frac{16}{2} = 8\pi$$

因此该旋转抛物面立体的体积为 $8\pi$。

**验证（用投影法）**：投影到 $xy$ 平面得 $D: x^2+y^2 \leq 4$（半径 $2$ 的圆盘），$z$ 从 $x^2+y^2$ 到 $4$。用极坐标：
$$I = \int_{0}^{2\pi} \int_{0}^{2} \left( \int_{r^2}^{4} 1\,dz \right) r\,dr\,d\theta = \int_{0}^{2\pi} \int_{0}^{2} (4 - r^2) r\,dr\,d\theta = 2\pi \left[ 2r^2 - \frac{r^4}{4} \right]_{0}^{2} = 2\pi (8 - 4) = 8\pi \quad \checkmark$$

</details>

### 迁移应用

**3.**（截面法与极坐标组合）计算 $\displaystyle I = \iiint_\Omega (x^2 + y^2)\,dV$，其中 $\Omega$ 是上半球体 $x^2 + y^2 + z^2 \leq R^2$、$z \geq 0$。

<details><summary>参考答案</summary>

**第 1 步：确定 $z$ 的范围**
上半球体的 $z \in [0, R]$。

**第 2 步：确定截面 $D(z)$**
在高度 $z \in [0, R]$ 处：
$$x^2 + y^2 \leq R^2 - z^2$$
$D(z)$ 是半径为 $\sqrt{R^2 - z^2}$ 的圆盘。

**第 3 步：在截面 $D(z)$ 上用极坐标变换**
在固定 $z$ 的条件下，对 $D(z)$ 上的二重积分使用极坐标变换，$r$ 的上限为 $\sqrt{R^2 - z^2}$：
$$
\begin{aligned}
F(z) &= \iint_{D(z)} (x^2 + y^2)\,dxdy \\
&= \int_{0}^{2\pi} \int_{0}^{\sqrt{R^2 - z^2}} r^2 \cdot r\,dr\,d\theta \\
&= 2\pi \int_{0}^{\sqrt{R^2 - z^2}} r^3\,dr \\
&= 2\pi \cdot \frac{(R^2 - z^2)^2}{4} = \frac{\pi}{2} (R^2 - z^2)^2
\end{aligned}
$$

**第 4 步：对 $z$ 积分**
$$
\begin{aligned}
I &= \int_{0}^{R} \frac{\pi}{2} (R^2 - z^2)^2\,dz \\
&= \frac{\pi}{2} \int_{0}^{R} (R^4 - 2R^2z^2 + z^4)\,dz \\
&= \frac{\pi}{2} \left[ R^4 z - \frac{2R^2 z^3}{3} + \frac{z^5}{5} \right]_{0}^{R} \\
&= \frac{\pi}{2} \left( R^5 - \frac{2R^5}{3} + \frac{R^5}{5} \right) \\
&= \frac{\pi}{2} \cdot \frac{15R^5 - 10R^5 + 3R^5}{15} \\
&= \frac{\pi}{2} \cdot \frac{8R^5}{15} = \frac{4\pi R^5}{15}
\end{aligned}
$$

因此 $\displaystyle \iiint_\Omega (x^2 + y^2)\,dV = \frac{4\pi R^5}{15}$。

**注**：结合例3的结果，对整球 $\iiint_{\text{球}} z^2\,dV = \frac{4\pi R^5}{15}$，由对称性 $\iiint_{\text{球}} x^2\,dV = \iiint_{\text{球}} y^2\,dV = \frac{4\pi R^5}{15}$。因此上半球 $\iiint (x^2+y^2)\,dV = \frac{1}{2} \times \frac{4\pi R^5}{15} \times 2 = \frac{4\pi R^5}{15}$（上半球的 $x^2+y^2$ 积分是整球的一半，而整球中 $x^2+y^2$ 的积分等于 $2 \times \frac{4\pi R^5}{15} = \frac{8\pi R^5}{15}$）。结果一致。$\checkmark$

</details>

**4.**（方法选择）考虑由 $x=0$、$y=0$、$z=0$、$x+y+z=1$ 围成的四面体区域 $\Omega$。分别用投影法和截面法写出 $\displaystyle \iiint_\Omega f(x,y,z)\,dV$ 的累次积分表达式（不需要具体计算）。比较两种方法的差异。

<details><summary>参考答案</summary>

**投影法（先一后二）——投影到 $xy$ 平面**：

$\Omega$ 的不等式组描述：
$$0 \leq x \leq 1,\quad 0 \leq y \leq 1-x,\quad 0 \leq z \leq 1-x-y$$

累次积分：
$$\iiint_\Omega f\,dV = \int_{0}^{1} \int_{0}^{1-x} \left( \int_{0}^{1-x-y} f(x,y,z)\,dz \right) dy\,dx$$

**截面法（先二后一）**：

$z$ 的范围：$0 \leq z \leq 1$。

在高度 $z$ 处的截面 $D(z)$：
$$D(z) = \big\{ (x,y) \mid x \geq 0,\ y \geq 0,\ x+y \leq 1-z \big\}$$

$D(z)$ 在X-型描述下：$0 \leq x \leq 1-z,\ 0 \leq y \leq 1-z-x$。

累次积分：
$$\iiint_\Omega f\,dV = \int_{0}^{1} \left( \int_{0}^{1-z} \int_{0}^{1-z-x} f(x,y,z)\,dy\,dx \right) dz$$

**方法对比**：
- 投影法得到的是 $\int dx \int dy \int dz$ 的次序（先 $z$ 再 $y$ 再 $x$）
- 截面法得到的是 $\int dz \int dx \int dy$ 的次序（先 $y$ 再 $x$ 再 $z$）
- 两者是同一个三重积分的**不同积分次序**，结果必然相等
- 若 $f$ 与 $z$ 无关（如 $f = g(x,y)$），截面法中内部二重积分的被积函数不含 $z$，计算更简便；若 $f$ 与 $x,y$ 无关（如 $f = h(z)$），截面法可直接利用截面面积
- 若 $f$ 是 $x$ 的函数（如 $f = x$），投影法中 $z$ 积分得到 $x\cdot(1-x-y)$，然后对 $y$ 积分再对 $x$ 积分，较为直接
