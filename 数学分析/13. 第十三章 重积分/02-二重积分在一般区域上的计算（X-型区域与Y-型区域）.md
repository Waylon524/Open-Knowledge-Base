## 2. 背景与应用场景

### 2.1 从矩形到一般区域

在 ch13-01 中，我们学习了 Fubini 定理（定理 13.6），它允许我们将矩形区域上的二重积分转化为累次积分：

$$\iint_{[a,b]\times[c,d]} f(x,y)\,dxdy = \int_a^b \left( \int_c^d f(x,y)\,dy \right) dx$$

但实际应用中，积分区域很少是完美的矩形。平面薄板可能是三角形、圆形或者由两条曲线围成的月牙形。例如：

- 计算由抛物线 $y = x^2$ 和直线 $y = 2x$ 围成的区域上的质量分布
- 计算半圆盘上的电荷总量
- 计算三角形薄板的转动惯量

这些区域的共同特征是：**在某个坐标方向上，边界是曲线而非直线**，内层积分限不再是常数，而是变量的函数。

### 2.2 直观理解

考虑由曲线 $y = \varphi_1(x)$（下边界）和 $y = \varphi_2(x)$（上边界）以及竖直线 $x = a$、$x = b$ 围成的区域。对每个固定的 $x$，被积函数在 $y$ 方向上的"累积"范围从 $\varphi_1(x)$ 到 $\varphi_2(x)$——这是一个随 $x$ 变化的区间。Fubini 定理经过适当推广后，仍然适用：先对 $y$ 积分（将 $x$ 视为常数），再对 $x$ 积分。

## 3. 核心概念与符号约定

### 3.1 X-型区域

**定义（X-型区域）**：设 $D \subset \mathbb{R}^2$ 是一个有界闭区域。若存在区间 $[a,b]$ 以及定义在 $[a,b]$ 上的连续函数 $\varphi_1(x)$、$\varphi_2(x)$（满足 $\varphi_1(x) \leq \varphi_2(x)$ 对一切 $x \in [a,b]$ 成立），使得：

$$D = \big\{ (x,y) \mid a \leq x \leq b,\; \varphi_1(x) \leq y \leq \varphi_2(x) \big\}$$

则称 $D$ 为 **X-型区域**（$x$-simple region）。

**理解要点**：
- 外层变量是 $x$，变化范围是常数区间 $[a,b]$
- 内层变量是 $y$，**下限和上限都是 $x$ 的函数**
- 几何含义：用垂直于 $x$ 轴的直线去截区域，截线是**一条连续的竖直线段**（从下边界到上边界）
- **必要条件**：对每个 $x \in (a,b)$，直线 $x = \text{常数}$ 与 $D$ 的交集要么是空的，要么是一条线段

**例**：由 $y = x$、$y = 2x$ 和 $x = 1$ 围成的三角形区域。
- $x$ 的范围：$0 \leq x \leq 1$
- 对每个 $x$：$y$ 从下边界的 $y = x$ 到上边界的 $y = 2x$
- 因此：$D = \{(x,y) \mid 0 \leq x \leq 1,\; x \leq y \leq 2x\}$

### 3.2 Y-型区域

**定义（Y-型区域）**：设 $D \subset \mathbb{R}^2$ 是一个有界闭区域。若存在区间 $[c,d]$ 以及定义在 $[c,d]$ 上的连续函数 $\psi_1(y)$、$\psi_2(y)$（满足 $\psi_1(y) \leq \psi_2(y)$ 对一切 $y \in [c,d]$ 成立），使得：

$$D = \big\{ (x,y) \mid c \leq y \leq d,\; \psi_1(y) \leq x \leq \psi_2(y) \big\}$$

则称 $D$ 为 **Y-型区域**（$y$-simple region）。

**理解要点**：
- 外层变量是 $y$，变化范围是常数区间 $[c,d]$
- 内层变量是 $x$，**下限和上限都是 $y$ 的函数**
- 几何含义：用垂直于 $y$ 轴的直线去截区域，截线是**一条连续的水平线段**（从左边到右边）
- **必要条件**：对每个 $y \in (c,d)$，直线 $y = \text{常数}$ 与 $D$ 的交集要么是空的，要么是一条线段

**例**：同一个三角形区域 $y = x$、$y = 2x$ 和 $x = 1$ 围成的区域，用 Y-型描述。
- 从 $y = x$ 解得 $x = y$，从 $y = 2x$ 解得 $x = y/2$
- $y$ 的范围：$0 \leq y \leq 2$
- 但对 $y \in (1,2]$，$x = y$ 可能超过 $1$（因为 $x = 1$ 有右边界）。因此需要分段！
- 正确做法：后面再详细讨论

### 3.3 X-型与Y-型的关系

- 同一区域可以同时是 X-型和 Y-型（例如矩形、三角形、圆盘）
- 有些区域只能是其中一种（例如月牙形区域可能既不是 X-型也不是 Y-型，需要分割成若干子区域）
- 选择哪种描述方式取决于：计算简便性（哪种积分次序能先积出内层积分）

### 3.4 符号表

| 符号 | 含义 | 说明 |
|------|------|------|
| $\varphi_1(x)$ | X-型区域的下边界函数 | 对每个 $x$，$y$ 的下限 |
| $\varphi_2(x)$ | X-型区域的上边界函数 | 对每个 $x$，$y$ 的上限 |
| $\psi_1(y)$ | Y-型区域的左边界函数 | 对每个 $y$，$x$ 的下限 |
| $\psi_2(y)$ | Y-型区域的右边界函数 | 对每个 $y$，$x$ 的上限 |
| $\int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy$ | X-型区域上的累次积分 | 先对 $y$ 后对 $x$ |
| $\int_c^d dy \int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,dx$ | Y-型区域上的累次积分 | 先对 $x$ 后对 $y$ |

## 4. 原理与方法

### 4.1 Fubini 定理在一般区域上的推广

**定理 13.8（一般区域上的 Fubini 定理）**：设 $f(x,y)$ 在有界闭区域 $D$ 上连续（或可积）。

**(1) 若 $D$ 是 X-型区域**：$D = \{(x,y) \mid a \leq x \leq b,\; \varphi_1(x) \leq y \leq \varphi_2(x)\}$，则：

$$\iint_D f(x,y)\,dxdy = \int_a^b \left( \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy \right) dx$$

**(2) 若 $D$ 是 Y-型区域**：$D = \{(x,y) \mid c \leq y \leq d,\; \psi_1(y) \leq x \leq \psi_2(y)\}$，则：

$$\iint_D f(x,y)\,dxdy = \int_c^d \left( \int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,dx \right) dy$$

**核心思想**：将一般区域"嵌入"一个矩形 $[a,b] \times [c,d]$ 中，令 $f$ 在 $D$ 外部延拓为 $0$，然后应用矩形区域上的 Fubini 定理。由于 $D$ 的边界是零面积集，延拓后的函数仍然可积，且积分值不变。

**计算步骤**（以 X-型为例）：
1. **确定 $x$ 的范围**：找到区域在 $x$ 轴上的投影区间 $[a,b]$
2. **确定 $y$ 的下边界**：对每个 $x$，找到下方的边界曲线 $y = \varphi_1(x)$
3. **确定 $y$ 的上边界**：对每个 $x$，找到上方的边界曲线 $y = \varphi_2(x)$
4. **先对 $y$ 积分**：将 $x$ 视为常数，计算 $\int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy$
5. **再对 $x$ 积分**：将第 4 步的结果关于 $x$ 积分

### 4.2 如何将区域描述为 X-型或 Y-型

#### 4.2.1 确定积分区域的通用方法（对应 Type A-4）

**步骤 1：画出区域草图**
- 画出所有边界曲线（直线和曲线）
- 标记交点坐标
- 确定围成的有界闭区域

**步骤 2：求交点**
- 联立边界曲线方程
- 解出交点坐标

**步骤 3：确定外层变量的范围**
- X-型：找出 $x$ 的最小值和最大值（区域在 $x$ 轴上的投影）
- Y-型：找出 $y$ 的最小值和最大值（区域在 $y$ 轴上的投影）

**步骤 4：确定内层变量的上下边界**
- X-型：固定 $x$，找到下方曲线 $\varphi_1(x)$ 和上方曲线 $\varphi_2(x)$
- Y-型：固定 $y$，找到左边曲线 $\psi_1(y)$ 和右边曲线 $\psi_2(y)$

**步骤 5：判断哪条是上/下（或左/右）边界**
- 在外层变量的整个范围内比较函数值大小
- 较大的函数值是上边界（X-型）或右边界（Y-型）
- 较小的函数值是下边界（X-型）或左边界（Y-型）

#### 4.2.2 示例：由 $y = \sqrt{x}$ 和 $y = x$ 围成的区域

**第 1 步：求交点**
解 $\sqrt{x} = x$，两边平方得 $x = x^2$，即 $x(x-1) = 0$，解得 $x = 0$ 或 $x = 1$。交点为 $(0,0)$ 和 $(1,1)$。

**第 2 步：判断上下边界**
在 $(0,1)$ 区间内比较：取 $x = 1/4$，$\sqrt{1/4} = 0.5$，$x = 0.25$，所以 $\sqrt{x} > x$。因此在 $[0,1]$ 上，$y = \sqrt{x}$ 是上边界，$y = x$ 是下边界。

**第 3a 步：X-型描述**
- $x$ 的范围：$0 \leq x \leq 1$
- 对每个 $x$：$y$ 从 $x$（下边界）到 $\sqrt{x}$（上边界）
- $D = \{(x,y) \mid 0 \leq x \leq 1,\; x \leq y \leq \sqrt{x}\}$

**第 3b 步：Y-型描述**
- 将曲线改写为 $x$ 关于 $y$ 的表达式：从 $y = \sqrt{x}$ 得 $x = y^2$；从 $y = x$ 得 $x = y$
- $y$ 的范围：$0 \leq y \leq 1$
- 比较 $y^2$ 和 $y$：在 $[0,1]$ 上 $y^2 \leq y$，因此 $x$ 从 $y^2$（左边界）到 $y$（右边界）
- $D = \{(x,y) \mid 0 \leq y \leq 1,\; y^2 \leq x \leq y\}$

### 4.3 X-型区域上的二重积分计算

对于 X-型区域 $D = \{(x,y) \mid a \leq x \leq b,\; \varphi_1(x) \leq y \leq \varphi_2(x)\}$：

$$\iint_D f(x,y)\,dxdy = \int_a^b \left( \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy \right) dx$$

**计算流程**：
1. 将 $x$ 视为常数，对 $y$ 积分（使用 Newton-Leibniz 公式）
2. 得到一个关于 $x$ 的函数 $F(x) = \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy$
3. 对 $F(x)$ 在 $[a,b]$ 上积分

**关键细节**：
- 内层积分限 $\varphi_1(x)$ 和 $\varphi_2(x)$ 必须代入 $y$ 的上下限，而不是 $x$ 的
- 代入上下限后，得到的是关于 $x$ 的表达式（可能还有 $x$ 的项）
- 外层积分是关于 $x$ 的一元定积分

### 4.4 Y-型区域上的二重积分计算

对于 Y-型区域 $D = \{(x,y) \mid c \leq y \leq d,\; \psi_1(y) \leq x \leq \psi_2(y)\}$：

$$\iint_D f(x,y)\,dxdy = \int_c^d \left( \int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,dx \right) dy$$

**计算流程**与 X-型完全对称，只是 $x$ 和 $y$ 的角色互换。

### 4.5 交换积分次序（对应 Type A-3）

交换积分次序是指：将一个累次积分（例如先 $y$ 后 $x$ 的 X-型形式）改写为另一种次序（先 $x$ 后 $y$ 的 Y-型形式），反之亦然。**核心在于识别两种描述对应的是同一个区域**。

#### 4.5.1 交换次序的标准流程

**第 1 步：从累次积分反推区域**

给定累次积分 $I = \displaystyle\int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy$，由外层 $x \in [a,b]$ 和内层 $y \in [\varphi_1(x), \varphi_2(x)]$ 确定区域：
$$D = \{ (x,y) \mid a \leq x \leq b,\; \varphi_1(x) \leq y \leq \varphi_2(x) \}$$

**第 2 步：将区域重新描述为另一种类型**

将 $y$ 作为外层变量，确定 $y$ 的范围 $[c,d]$，然后将边界曲线改写为 $x = \psi(y)$ 的形式：
- 从 $y = \varphi_1(x)$ 解出 $x = \psi_1(y)$（左边界）
- 从 $y = \varphi_2(x)$ 解出 $x = \psi_2(y)$（右边界）

**第 3 步：判断是否需要分段**

检查对每个 $y \in [c,d]$，左边界和右边界的表达式是否在整个 $[c,d]$ 上一致有效。若边界表达式在 $y$ 的不同区间上不同，则需要分段描述。

#### 4.5.2 简单情形（无需分段）

考虑 $I = \displaystyle\int_0^1 dx \int_0^{x^2} f(x,y)\,dy$。

**第 1 步：反推区域**
- 外层：$x \in [0,1]$
- 内层：$y \in [0, x^2]$
- $D = \{(x,y) \mid 0 \leq x \leq 1,\; 0 \leq y \leq x^2\}$

**第 2 步：重新描述为 Y-型**
- $y$ 的范围：当 $x \in [0,1]$ 时，$y$ 从 $0$ 到 $x^2$，所以 $y \in [0,1]$
- 从 $y = x^2$ 解得 $x = \sqrt{y}$（在 $x \geq 0$ 条件下）
- 对每个 $y \in [0,1]$：$x$ 从 $x = \sqrt{y}$（左边界，因为 $x \geq \sqrt{y}$）到 $x = 1$（右边界，原区域的右边界）

**第 3 步：检查一致性**
- 在 $y \in [0,1]$ 上，$\sqrt{y}$ 始终有定义且小于等于 $1$
- 不需要分段

$$I = \int_0^1 dy \int_{\sqrt{y}}^{1} f(x,y)\,dx$$

#### 4.5.3 复杂情形（需要分段）

有些区域在一种描述下很简单，但在另一种描述下需要用两个（或多个）Y-型子区域拼接。

考虑 $I = \displaystyle\int_0^3 dx \int_{x/3}^{x} f(x,y)\,dy$。

**第 1 步：反推区域**
- $x \in [0,3]$，$y \in [x/3, x]$
- 下边界：$y = x/3$（从原点出发，斜率为 $1/3$ 的直线）
- 上边界：$y = x$（从原点出发，斜率为 $1$ 的直线）
- 右边界：$x = 3$（竖直线）
- 该区域是一个三角形，顶点为 $(0,0)$、$(3,1)$、$(3,3)$

**第 2 步：改写为 Y-型描述**
- $y$ 的范围：$0 \leq y \leq 3$
- 从 $y = x$ 得 $x = y$；从 $y = x/3$ 得 $x = 3y$
- 若不对 $y$ 分段，则表述为：$y \leq x \leq 3y$

**第 3 步：检查是否需要分段**
- 当 $y = 2$ 时，$3y = 6 > 3$（超过了 $x$ 的最大值 $3$）
- 这说明在 $y > 1$ 的区域，$x$ 不能到达 $3y$，最多只能到 $x = 3$（右边界）
- 因此按 $y = 1$ 为界分段：

**当 $0 \leq y \leq 1$ 时**，$3y \leq 3$，$x$ 从 $x = y$ 到 $x = 3y$。
**当 $1 \leq y \leq 3$ 时**，$3y > 3$，$x$ 从 $x = y$ 到 $x = 3$（受右边界限制）。

因此交换次序后：

$$I = \int_0^1 dy \int_{y}^{3y} f(x,y)\,dx \;+\; \int_1^3 dy \int_{y}^{3} f(x,y)\,dx$$

**何时需要分段？**

根本原因：原区域在 X-型描述下为 $y \in [x/3, x]$，$x \in [0,3]$。当转换为 Y-型时，我们需要将 $x$ 的边界表示为 $y$ 的函数。但原区域中 $x$ 的上限由两条共同决定：$y \leq x$（即 $x \geq y$）和 $x \leq 3$（区域右边界）。在 Y-型描述下，$x$ 的上限应该是 $x = \min(3y, 3)$——这就是分段产生的根源。

**一般规则**：若原区域的一条边界是多段曲线组成的（或由不等式 $x \leq \text{常数}$ 和 $x \leq \text{函数}$ 共同约束），则在新描述中可能会产生分段。

## 5. 例题

### 例 1：X-型与 Y-型区域的描述

设 $D$ 是由抛物线 $y = 4 - x^2$ 和直线 $y = x + 2$ 所围成的有界闭区域。

**(a)** 求交点坐标，画出区域草图。
**(b)** 将 $D$ 表示为 X-型区域。
**(c)** 将 $D$ 表示为 Y-型区域。

**解 (a)**：

联立方程：
$$4 - x^2 = x + 2 \quad\Longrightarrow\quad x^2 + x - 2 = 0 \quad\Longrightarrow\quad (x+2)(x-1) = 0$$

解得 $x = -2$ 或 $x = 1$。对应地 $y = 0$ 或 $y = 3$。交点坐标为 $(-2,0)$ 和 $(1,3)$。

区域草图：抛物线 $y = 4 - x^2$ 开口向下，直线 $y = x + 2$ 斜率为 $1$。在 $x \in (-2,1)$ 上，比较两函数：
$$(4 - x^2) - (x + 2) = 2 - x - x^2 = (2 + x)(1 - x)$$

当 $x \in (-2,1)$ 时，$(2 + x) > 0$，$(1 - x) > 0$，所以 $4 - x^2 > x + 2$。因此抛物线是上边界，直线是下边界。

**解 (b)：X-型区域描述**

- $x$ 的变化范围：从 $-2$ 到 $1$
- 对每个 $x \in [-2, 1]$：$y$ 从下边界 $y = x + 2$ 到上边界 $y = 4 - x^2$

$$D = \big\{ (x,y) \mid -2 \leq x \leq 1,\; x + 2 \leq y \leq 4 - x^2 \big\}$$

**解 (c)：Y-型区域描述**

将曲线改写为 $x$ 关于 $y$ 的形式：
- 从 $y = x + 2$ 得 $x = y - 2$
- 从 $y = 4 - x^2$ 得 $x^2 = 4 - y$，在 $[-2,1]$ 区间上 $x \leq 0$？不对，$x$ 在 $[-2,1]$ 上跨越了正负。需要仔细处理。

观察抛物线 $y = 4 - x^2$：对同一个 $y$，$x = \pm\sqrt{4 - y}$。但在区域 $D$ 中，$x$ 从 $-2$ 到 $1$，既有负值也有正值。对于 Y-型描述，需要区分左半和右半。

$y$ 的范围：$y \in [0, 4]$（因为抛物线最高点 $y = 4$，最低点 $y = 0$）。

**当 $0 \leq y \leq 3$ 时**：区域由抛物线左支 $x = -\sqrt{4-y}$、直线 $x = y-2$ 以及右边界 $x = 1$ 共同围成。不对——需要更仔细。

实际上，这个区域在 Y-型描述下需要分段处理：
- $y \in [0,3]$：$x$ 从左边的 $x = -\sqrt{4-y}$（抛物线左支）到右边的 $x = y-2$（直线）——但 $y-2$ 可能小于 $-\sqrt{4-y}$？验证：在 $y=0$ 时 $-\sqrt{4} = -2$，$y-2 = -2$，相等。在 $y=3$ 时 $-\sqrt{1} = -1$，$y-2 = 1$。所以对于 $y \in [0,3]$，$x \in [-\sqrt{4-y}, y-2]$。
- $y \in [3,4]$：$x$ 从 $-\sqrt{4-y}$ 到 $\sqrt{4-y}$（整个抛物线宽）

因此 Y-型描述为：
$$D = \big\{ (x,y) \mid 0 \leq y \leq 3,\; -\sqrt{4-y} \leq x \leq y-2 \big\}
\;\cup\; \big\{ (x,y) \mid 3 \leq y \leq 4,\; -\sqrt{4-y} \leq x \leq \sqrt{4-y} \big\}$$

这个例子说明：**同一个区域在一种描述下可能很简单（X-型只需一段），在另一种描述下可能需要分段**。选择恰当的积分次序可以大大简化计算。

### 例 2：X-型区域上计算二重积分

计算 $\displaystyle \iint_D (2x - y)\,dxdy$，其中 $D$ 是由 $y = x^2$ 和 $y = 4$ 所围成的区域（$x \geq 0$ 部分）。

**第 1 步：确定区域的描述**

曲线 $y = x^2$ 和 $y = 4$ 的交点：$x^2 = 4$，在 $x \geq 0$ 下得 $x = 2$，交点 $(2,4)$。另一交点 $(0,0)$ 不在 $y=4$ 上？实际上 $y=0$ 时 $x=0$ 是抛物线顶点。

区域 $D$ 在 $x \geq 0$ 部分：$x$ 从 $0$ 到 $2$，对每个 $x$，$y$ 从 $x^2$ 到 $4$。

用 X-型描述：
$$D = \big\{ (x,y) \mid 0 \leq x \leq 2,\; x^2 \leq y \leq 4 \big\}$$

**第 2 步：化为累次积分**

$$\iint_D (2x - y)\,dxdy = \int_0^2 dx \int_{x^2}^{4} (2x - y)\,dy$$

**第 3 步：先对 $y$ 积分（将 $x$ 视为常数）**

$$
\begin{aligned}
\int_{x^2}^{4} (2x - y)\,dy
&= \left[ 2xy - \frac{y^2}{2} \right]_{y=x^2}^{y=4} \\
&= \left( 2x \cdot 4 - \frac{4^2}{2} \right) - \left( 2x \cdot x^2 - \frac{(x^2)^2}{2} \right) \\
&= (8x - 8) - \left( 2x^3 - \frac{x^4}{2} \right) \\
&= 8x - 8 - 2x^3 + \frac{x^4}{2}
\end{aligned}
$$

**第 4 步：再对 $x$ 积分**

$$
\begin{aligned}
\iint_D (2x - y)\,dxdy
&= \int_0^2 \left( 8x - 8 - 2x^3 + \frac{x^4}{2} \right) dx \\
&= \left[ 4x^2 - 8x - \frac{x^4}{2} + \frac{x^5}{10} \right]_0^2 \\
&= (16 - 16 - 8 + \frac{32}{10}) - 0 \\
&= -8 + \frac{16}{5} = -\frac{40}{5} + \frac{16}{5} = -\frac{24}{5}
\end{aligned}
$$

因此：$\displaystyle \iint_D (2x - y)\,dxdy = -\frac{24}{5}$。

### 例 3：交换积分次序（简单情形）

交换累次积分 $I = \displaystyle \int_0^1 dx \int_{e^x}^{e} f(x,y)\,dy$ 的积分次序。

**第 1 步：反推区域**

外层 $x \in [0,1]$，内层 $y \in [e^x, e]$。因此：
$$D = \big\{ (x,y) \mid 0 \leq x \leq 1,\; e^x \leq y \leq e \big\}$$

下边界：$y = e^x$（指数曲线）
上边界：$y = e$（水平线）
左边界：$x = 0$（$y$ 轴）
右边界：$x = 1$（竖直线）

**第 2 步：重新描述为 Y-型**

$y$ 的范围：当 $x \in [0,1]$ 时 $y \in [e^x, e]$，所以 $y$ 从 $1$（$e^0$）到 $e$，即 $y \in [1, e]$。

从 $y = e^x$ 解得 $x = \ln y$。

对每个 $y \in [1, e]$：$x$ 从 $x = \ln y$（左边界，因为 $x \geq \ln y$）到 $x = 1$（右边界）。

**第 3 步：检查是否需要分段**

在 $y \in [1, e]$ 上，$\ln y$ 单调递增从 $0$ 到 $1$，始终有定义且 $\ln y \leq 1$。不需要分段。

因此：
$$I = \int_1^e dy \int_{\ln y}^{1} f(x,y)\,dx$$

### 例 4：交换积分次序（需要分段）

交换累次积分 $I = \displaystyle \int_0^2 dx \int_{x}^{2x} f(x,y)\,dy$ 的积分次序。

**第 1 步：反推区域**

$$D = \big\{ (x,y) \mid 0 \leq x \leq 2,\; x \leq y \leq 2x \big\}$$

下边界：$y = x$，上边界：$y = 2x$，左边界：$x = 0$，右边界：$x = 2$。

顶点：$(0,0)$、$(2,2)$（从 $y=x$ 和 $x=2$ 得）、$(2,4)$（从 $y=2x$ 和 $x=2$ 得）。

**第 2 步：重新描述为 Y-型**

$y$ 的范围：$0 \leq y \leq 4$。

从 $y = x$ 得 $x = y$；从 $y = 2x$ 得 $x = y/2$。

若直接写：$y/2 \leq x \leq y$，检查是否在整个 $[0,4]$ 上成立：

- 当 $y = 3$ 时：$y/2 = 1.5$，$y = 3$，但 $x \leq 2$（右边界），而 $y = 3 > 2$，所以 $x$ 不能到 $3$。
- 当 $y = 1$ 时：$y/2 = 0.5$，$y = 1$，均在 $[0,2]$ 范围内。

**第 3 步：分段**

按 $y$ 的范围分为两段：

**当 $0 \leq y \leq 2$ 时**：$y \leq 2$，因此 $y/2 \leq y \leq 2$，$x$ 从 $y/2$ 到 $y$。
**当 $2 \leq y \leq 4$ 时**：$y > 2$，$x$ 的上限不能超过 $2$，即 $x$ 从 $y/2$ 到 $2$。

因此：
$$I = \int_0^2 dy \int_{y/2}^{y} f(x,y)\,dx \;+\; \int_2^4 dy \int_{y/2}^{2} f(x,y)\,dx$$

**验证**：取 $f(x,y) = 1$（计算区域面积）。

原次序：
$$I = \int_0^2 dx \int_x^{2x} 1\,dy = \int_0^2 (2x - x)\,dx = \int_0^2 x\,dx = \left[ \frac{x^2}{2} \right]_0^2 = 2$$

新次序：
$$
\begin{aligned}
I &= \int_0^2 dy \int_{y/2}^{y} 1\,dx + \int_2^4 dy \int_{y/2}^{2} 1\,dx \\
&= \int_0^2 (y - y/2)\,dy + \int_2^4 (2 - y/2)\,dy \\
&= \int_0^2 \frac{y}{2}\,dy + \int_2^4 \left(2 - \frac{y}{2}\right) dy \\
&= \left[ \frac{y^2}{4} \right]_0^2 + \left[ 2y - \frac{y^2}{4} \right]_2^4 \\
&= 1 + \left[ (8 - 4) - (4 - 1) \right] = 1 + [4 - 3] = 2
\end{aligned}
$$

两种次序结果一致。$\checkmark$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| X-型区域就是左右对称的区域，Y-型区域是上下对称的区域 | X-型的核心是"用竖直线截区域得连续线段"，与是否对称无关。一个左右不对称的区域也可以是 X-型区域 |
| 交换积分次序就是简单地把 $dx$ 和 $dy$ 的顺序调换 | 交换积分次序需要完整的三步流程：反推区域 → 重新描述 → 写出新积分。仅调换 $dx$ 和 $dy$ 而不改变积分限是错误的 |
| 在 X-型区域的累次积分中，外层积分限可以是函数 | 外层积分限必须是常数。X-型中 $x$ 的范围是常数区间 $[a,b]$，$y$ 的积分限才是函数 $\varphi_1(x)$、$\varphi_2(x)$ |
| 先积分时，上下限代入后得到的结果不再含原来的变量 | 在 X-型中先对 $y$ 积分，代入上下限 $\varphi_1(x)$ 和 $\varphi_2(x)$ 后得到的是关于 $x$ 的函数（$\varphi_1$、$\varphi_2$ 本身就是 $x$ 的函数） |
| 区域边界由 $y = f(x)$ 给出，在 Y-型描述中直接写 $x = f(y)$ | 从 $y = f(x)$ 解出 $x = g(y)$ 需要保证反函数存在（即 $f$ 在该区间上是单射）。若 $f$ 不是单射（如 $y = x^2$ 在 $[-1,1]$ 上），则需要分段处理 |

### 检查点

- [ ] 能否用自己的语言给出 X-型区域和 Y-型区域的定义？
- [ ] 能否判断区域 $D = \{(x,y) \mid x^2 + y^2 \leq 1\}$ 既是 X-型也是 Y-型？（提示：$x \in [-1,1]$，$y \in [-\sqrt{1-x^2}, \sqrt{1-x^2}]$；$y \in [-1,1]$，$x \in [-\sqrt{1-y^2}, \sqrt{1-y^2}]$）
- [ ] 能否完成"从累次积分反推区域 → 重新描述 → 写出新积分"的交换次序流程？
- [ ] 能否识别何时需要分段描述区域？
- [ ] 能否说出交换次序时外层积分限为什么必须是常数？
- [ ] 给定两条曲线 $y = f(x)$ 和 $y = g(x)$，能否通过求交点和比较函数值确定积分域的描述？

**2.** （X-型区域积分）计算 $\displaystyle \iint_D (x^2 + y)\,dxdy$，其中 $D$ 是由 $y = x^2$ 和 $y = x$ 在 $0 \leq x \leq 1$ 部分所围成的区域。

<details><summary>参考答案</summary>

**第 1 步：确定区域描述**
在 $[0,1]$ 上 $x^2 \leq x$，所以 $y = x^2$ 是下边界，$y = x$ 是上边界。
$$D = \big\{ (x,y) \mid 0 \leq x \leq 1,\; x^2 \leq y \leq x \big\}$$

**第 2 步：化为累次积分**
$$\iint_D (x^2 + y)\,dxdy = \int_0^1 dx \int_{x^2}^{x} (x^2 + y)\,dy$$

**第 3 步：先对 $y$ 积分**
$$
\begin{aligned}
\int_{x^2}^{x} (x^2 + y)\,dy
&= \left[ x^2y + \frac{y^2}{2} \right]_{y=x^2}^{y=x} \\
&= \left( x^2 \cdot x + \frac{x^2}{2} \right) - \left( x^2 \cdot x^2 + \frac{(x^2)^2}{2} \right) \\
&= \left( x^3 + \frac{x^2}{2} \right) - \left( x^4 + \frac{x^4}{2} \right) \\
&= x^3 + \frac{x^2}{2} - \frac{3x^4}{2}
\end{aligned}
$$

**第 4 步：再对 $x$ 积分**
$$
\begin{aligned}
\iint_D (x^2 + y)\,dxdy
&= \int_0^1 \left( x^3 + \frac{x^2}{2} - \frac{3x^4}{2} \right) dx \\
&= \left[ \frac{x^4}{4} + \frac{x^3}{6} - \frac{3x^5}{10} \right]_0^1 \\
&= \frac{1}{4} + \frac{1}{6} - \frac{3}{10} \\
&= \frac{15}{60} + \frac{10}{60} - \frac{18}{60} = \frac{7}{60}
\end{aligned}
$$

因此 $\displaystyle \iint_D (x^2 + y)\,dxdy = \frac{7}{60}$。

</details>

<details><summary>参考答案</summary>

**第 3 步：检查分段**
在 $y \in [0,1]$ 上 $y^2 \leq 1$，无需分段。

$$I = \int_0^1 dy \int_{y^2}^{1} f(x,y)\,dx$$

**4.** （交换积分次序——需要分段）交换累次积分 $I = \displaystyle \int_0^4 dx \int_{x/4}^{\sqrt{x}} f(x,y)\,dy$ 的积分次序，并在 $f(x,y) = 1$ 时计算 $I$ 的值验证结果。

<details><summary>参考答案</summary>

**第 1 步：反推区域**
$$D = \big\{ (x,y) \mid 0 \leq x \leq 4,\; x/4 \leq y \leq \sqrt{x} \big\}$$

下边界 $y = x/4$（直线），上边界 $y = \sqrt{x}$（抛物线）。
在 $x=0$ 处两者均为 $0$；在 $x=4$ 处 $4/4 = 1 = \sqrt{4}$。交点 $(0,0)$ 和 $(4,1)$。
当 $x \in (0,4)$ 时 $\sqrt{x} > x/4$（验证：在 $x=1$ 时 $1 > 0.25$），所以上边界大于下边界。

**第 2 步：重新描述为 Y-型**
$y$ 的范围：$0 \leq y \leq 2$（因为 $\sqrt{4} = 2$）。
从 $y = \sqrt{x}$ 得 $x = y^2$；从 $y = x/4$ 得 $x = 4y$。

**第 3 步：检查分段**
当 $y = 1$ 时 $4y = 4$（刚好等于 $x$ 的最大值 $4$）。
当 $y > 1$ 时 $4y > 4$，超过了 $x$ 的上限。
分段点为 $y = 1$：

**当 $0 \leq y \leq 1$ 时**：$x$ 从 $y^2$ 到 $4y$（因为 $4y \leq 4$）。
**当 $1 \leq y \leq 2$ 时**：$x$ 从 $y^2$ 到 $4$（右边界）。

$$I = \int_0^1 dy \int_{y^2}^{4y} f(x,y)\,dx \;+\; \int_1^2 dy \int_{y^2}^{4} f(x,y)\,dx$$

**验证**：取 $f(x,y) = 1$（计算面积）。

原次序：
$$I = \int_0^4 dx \int_{x/4}^{\sqrt{x}} 1\,dy = \int_0^4 (\sqrt{x} - x/4)\,dx = \left[ \frac{2}{3}x^{3/2} - \frac{x^2}{8} \right]_0^4 = \frac{2}{3} \cdot 8 - \frac{16}{8} = \frac{16}{3} - 2 = \frac{10}{3}$$

新次序：
$$
\begin{aligned}
I &= \int_0^1 dy \int_{y^2}^{4y} 1\,dx + \int_1^2 dy \int_{y^2}^{4} 1\,dx \\
&= \int_0^1 (4y - y^2)\,dy + \int_1^2 (4 - y^2)\,dy \\
&= \left[ 2y^2 - \frac{y^3}{3} \right]_0^1 + \left[ 4y - \frac{y^3}{3} \right]_1^2 \\
&= \left( 2 - \frac{1}{3} \right) + \left[ \left( 8 - \frac{8}{3} \right) - \left( 4 - \frac{1}{3} \right) \right] \\
&= \frac{5}{3} + \left( \frac{16}{3} - \frac{11}{3} \right) = \frac{5}{3} + \frac{5}{3} = \frac{10}{3}
\end{aligned}
$$

两种次序结果一致。$\checkmark$

</details>

**5.** （综合题）设 $D$ 是由 $y = \ln x$、$y = 0$ 和 $x = e$ 所围成的区域。

**(a)** 将 $D$ 分别描述为 X-型和 Y-型区域。
**(b)** 选择较简便的次序计算 $\displaystyle \iint_D x\,dxdy$。

<details><summary>参考答案</summary>

**(a) 区域描述**

求交点：$y = \ln x$ 与 $y = 0$ 的交点：$\ln x = 0 \Rightarrow x = 1$，即 $(1,0)$。
$y = \ln x$ 与 $x = e$ 的交点：$x = e$ 时 $y = \ln e = 1$，即 $(e,1)$。

**X-型描述**：
$x$ 的范围：$1 \leq x \leq e$。
对每个 $x$：$y$ 从 $0$ 到 $\ln x$。
$$D = \big\{ (x,y) \mid 1 \leq x \leq e,\; 0 \leq y \leq \ln x \big\}$$

**Y-型描述**：
$y$ 的范围：$0 \leq y \leq 1$。
从 $y = \ln x$ 得 $x = e^y$。
对每个 $y$：$x$ 从 $e^y$ 到 $e$。
$$D = \big\{ (x,y) \mid 0 \leq y \leq 1,\; e^y \leq x \leq e \big\}$$

**(b) 计算积分**

选择 Y-型（先对 $x$ 积分，因为 $\int e^y\,dy$ 很简单）：

$$
\begin{aligned}
\iint_D x\,dxdy &= \int_0^1 dy \int_{e^y}^{e} x\,dx \\
&= \int_0^1 \left[ \frac{x^2}{2} \right]_{x=e^y}^{x=e} dy \\
&= \int_0^1 \left( \frac{e^2}{2} - \frac{e^{2y}}{2} \right) dy \\
&= \frac{1}{2} \int_0^1 (e^2 - e^{2y})\,dy \\
&= \frac{1}{2} \left[ e^2y - \frac{e^{2y}}{2} \right]_0^1 \\
&= \frac{1}{2} \left[ \left( e^2 - \frac{e^2}{2} \right) - \left( 0 - \frac{1}{2} \right) \right] \\
&= \frac{1}{2} \left( \frac{e^2}{2} + \frac{1}{2} \right) = \frac{e^2 + 1}{4}
\end{aligned}
$$

因此 $\displaystyle \iint_D x\,dxdy = \frac{e^2 + 1}{4}$。

（如果用 X-型：$\int_1^e dx \int_0^{\ln x} x\,dy = \int_1^e x\ln x\,dx$，需要分部积分，计算量稍大但不失为另一种方法。）

</details>
