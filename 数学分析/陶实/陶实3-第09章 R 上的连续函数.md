# 第9章 R 上的连续函数

在前几章中，我们主要研究了序列。序列  $ (a_{n})_{n=0}^{\infty} $ 可以看作是一个从 N 到 R 的函数，即对每一个自然数 n 都指定了一个实数  $ a_{n} $ 的映射。然后我们对这些从 N 到 R 的函数做了各种各样的事情，例如在无穷大处取它们的极限（当函数收敛时），或者构造上确界和下确界等，又或者计算一个序列的全体元素之和（同样要假设级数是收敛的）。

现在我们要研究的不是定义在“离散的”自然数集 N 上的函数，而是定义在一个连续统 $ ^{①} $上的函数，例如定义在实直线 R 上的函数，或者定义在像  $ \{x \in \mathbb{R} : a \leqslant x \leqslant b\} $ 这样区间上的函数。最终，我们将对这些函数进行大量运算，包括取极限、求导以及计算积分值。在本章中，我们主要研究函数的极限以及与之密切相关的连续函数的概念。

在讨论函数之前，我们必须先给出关于实直线子集的一些记号。

## 9.1 实直线的子集

在分析学中，我们通常不会对整个实直线 R 进行研究，而是研究实直线的某些特定子集，比如正实轴  $ \{x \in \mathbf{R} : x > 0\} $。另外，我们偶尔也与 6.2 节中定义的广义实直线  $ R^* $ 打交道，或者在它的子集中展开研究。

实直线显然有无穷多个子集。事实上，康托尔定理（定理 8.3.1，参见习题 8.3.4）表明了实直线子集的个数甚至比实数的还要多。但是实直线（和广义实直线）的一些特殊子集才是我们经常遇到的，其中区间就是一种这样的集合。

定义 9.1.1（区间）设  $ a, b \in \mathbb{R}^* $ 是广义实数，我们定义闭区间  $ [a, b] $ 为

 $$ [a,b]:=\{x\in\mathbf{R}^{*}:a\leqslant x\leqslant b\} $$ 

半开区间  $ [a,b) $ 和  $ (a,b] $ 为

 $$ [a,b):=\{x\in\mathbf{R}^{*}:a\leqslant x<b\};(a,b]:=\{x\in\mathbf{R}^{*}:a<x\leqslant b\} $$ 

开区间 $ (a,b) $为

 $$ (a,b):=\{x\in\mathbf{R}^{*}:a<x<b\} $$ 

我们称 a 为这些区间的左端点，b 为这些区间的右端点。

注 9.1.2 这里我们再一次使用了圆括号。例如，现在我们既可以用  $ (2,3) $ 表示一个从 2 到 3 的开区间，也可以用它来表示笛卡儿平面  $ \mathbf{R}^{2} := \mathbf{R} \times \mathbf{R} $ 上的一个有序对。这或许会导致歧义，但你应该能够从上下文中了解圆括号所表达的意思。在某些教材中，使用反向的中括号代替圆括号来解决这个问题，例如  $ [a,b) $ 就变成了  $ [a,b] $， $ (a,b] $ 就是  $ [a,b] $，而  $ (a,b) $ 就记作  $ [a,b] $。

例 9.1.3 如果 a 和 b 是实数（即它们既不等于 +∞，也不等于 -∞），那么所有上述区间都是实直线的子集，例如  $ [2,3) = \{x \in \mathbf{R} : 2 \leqslant x < 3\} $。正实轴  $ \{x \in \mathbf{R} : x > 0\} $ 就是开区间  $ (0,+\infty) $，而非负实轴  $ \{x \in \mathbf{R} : x \geqslant 0\} $ 就是一个半开区间  $ [0,+\infty) $。类似地，负实轴  $ \{x \in \mathbf{R} : x < 0\} $ 是  $ (-∞,0) $，并且非正实轴  $ \{x \in \mathbf{R} : x \leqslant 0\} $ 是  $ (-∞,0] $。最后，实直线  $ \mathbf{R} $ 本身是开区间  $ (-∞,+∞) $，而广义实直线  $ \mathbf{R}^{*} $ 是闭区间  $ [-∞,+∞] $。有时候我们称其中一个端点是无限  $ (-∞,-∞) $ 的区间为半无限区间，称两个端点都是无限的区间为双无限区间；而其余所有的区间都被称为有界区间。所以  $ [2,3) $ 是有界区间，正实轴和负实轴都是半无限区间，而  $ \mathbf{R} $ 和  $ \mathbf{R}^{*} $ 都是双无限区间。

例 9.1.4 如果 a > b，那么四个区间  $ [a, b] $、 $ [a, b) $、 $ (a, b] $ 和  $ (a, b) $ 都是空集（为什么？）。如果 a = b，那么三个区间  $ [a, b) $、 $ (a, b] $ 和  $ (a, b) $ 都是空集，而  $ [a, b] $ 就是一个单元素集  $ \{a\} $。（为什么？）因此，我们称这些区间是退化的。我们所学习的分析理论的绝大部分（但并非全部）内容都限制在非退化的区间上。

当然，区间并不是实直线唯一一个有意义的子集。其他重要的例子包括自然数集 N、整数集 Z 以及有理数集 Q。通过使用并集和交集（参见 3.1 节）这样的运算，我们还可以构造其他集合，例如我们可以求出两个区间不连通的并集，如  $ (1,2) \cup [3,4] $，或者考虑 -1 和 1 之间（包含端点在内）的有理数集合  $ [-1,1] \cap Q $。利用这些运算，我们显然可以构造出无限多个集合。

就像实数序列存在极限点一样，实数集也有附着点，现在我们来定义附着点。

定义 9.1.5（ $ \varepsilon $-附着点）设 X 是 R 的子集， $ \varepsilon > 0 $，并且设  $ x \in \mathbb{R} $。我们称 x 是  $ \varepsilon $-附着于 X 的，当且仅当存在一个  $ y \in X $ 是  $ \varepsilon $-接近于 x 的（即  $ |x - y| \leqslant \varepsilon $）。

注 9.1.6 术语 “ε-附着于” 不是文献中的标准术语，但是我们简单地用它来定义附着点的概念，其中附着点是一个标准术语。

例 9.1.7 点 1.1 是 0.5-附着于开区间  $ (0,1) $ 的，但不是 0.1-附着于这个区间的。（为什么？）点 1.1 是 0.5-附着于有限集  $ \{1,2,3\} $ 的。点 1 是 0.5-附着于  $ \{1,2,3\} $ 的。（为什么？）

定义 9.1.8（附着点）设 X 是 R 的一个子集，并且设  $ x \in \mathbb{R} $。我们称 x 是 X 的一个附着点，当且仅当对于任意的  $ \varepsilon > 0 $，x 都是  $ \varepsilon $-附着于 X 的。

例 9.1.9 对于任意的  $ \varepsilon > 0 $，数字1都是  $ \varepsilon $-附着于开区间  $ (0,1) $ 的，（为什么？）从而 1 是  $ (0,1) $ 的一个附着点。类似地，点 0.5 也是  $ (0,1) $ 的一个附着点。但数字 2 不是（例如）0.5-附着于  $ (0,1) $ 的，因此 2 不是  $ (0,1) $ 的附着点。

定义 9.1.10（闭包）设 X 是 R 的一个子集。X 的闭包，有时记作  $ \overline{X} $，是指由 X 的全体附着点所构成的集合。

引理 9.1.11（闭包的初等性质） 设 X 和 Y 是  $ \mathbb{R} $ 的任意两个子集，那么  $ X \subseteq \overline{X} $， $ \overline{X} \cup Y = \overline{X} \cup \overline{Y} $，且  $ \overline{X} \cap Y \subseteq \overline{X} \cap \overline{Y} $。如果  $ X \subseteq Y $，那么  $ \overline{X} \subseteq \overline{Y} $。

证明：参见习题 9.1.2。

现在我们来计算一些闭包。

引理 9.1.12（区间的闭包）设 a < b 是实数，并且设 I 是四个区间  $ (a, b) $、 $ (a, b] $、 $ [a, b) $ 和  $ [a, b] $ 中的任意一个。那么 I 的闭包是  $ [a, b] $。类似地， $ (a, +\infty) $ 或  $ [a, +\infty) $ 的闭包是  $ [a, +\infty) $，而  $ (-\infty, a) $ 或  $ (-\infty, a] $ 的闭包是  $ (-\infty, a] $。最后， $ (-\infty, +\infty) $ 的闭包是  $ (-\infty, +\infty) $。

证明：我们只证明这些结论中的一个，即  $ (a,b) $ 的闭包是  $ [a,b] $。其他的结论可以类似地证明（或者利用习题 9.1.1）。

我们首先证明  $ [a,b] $ 的每一个元素都是附着于  $ (a,b) $ 的。设  $ x \in [a,b] $，如果  $ x \in (a,b) $，那么它肯定是附着于  $ (a,b) $ 的。如果 x = b，那么 x 也是附着于  $ (a,b) $ 的。（为什么？）当 x = a 时也有类似的结论。因此  $ [a,b] $ 中的每一点都是附着于  $ (a,b) $ 的。

现在我们证明附着于  $ (a,b) $ 的每一个点 x 都属于  $ [a,b] $。利用反证法，假设 x 不属于  $ [a,b] $，于是要么 x > b，要么 x < a。如果 x > b，那么 x 不是  $ (x - b) $-附着于  $ (a,b) $ 的（为什么？），从而 x 不是  $ (a,b) $ 的附着点。类似地，如果 x < a，那么 x 不是  $ (a - x) $-附着于  $ (a,b) $ 的，从而 x 不是  $ (a,b) $ 的附着点。这个矛盾说明 x 实际上属于  $ [a,b] $，结论得证。

引理 9.1.13 N 的闭包是 N，Z 的闭包是 Z，Q 的闭包是 R，R 的闭包是 R。空集  $ \varnothing $ 的闭包是  $ \varnothing $。

证明：参见习题 9.1.3。

下面的引理证明了集合 X 的附着点可以作为 X 中元素的极限而得到。

引理 9.1.14 设 X 是 R 的子集，并且设  $ x \in \mathbb{R} $。那么 x 是 X 的一个附着点，当且仅当存在一个完全由 X 中元素构成的序列  $ (a_{n})_{n=0}^{\infty} $ 收敛于 x。

证明：参见习题 9.1.5。

定义 9.1.15 如果  $ \overline{E} = E $，那么子集  $ E \subseteq R $ 是闭的，即 E 的所有附着点都包含在 E 中。

例 9.1.16 根据引理 9.1.12 可知，如果 a < b 是实数，那么  $ [a, b] $、 $ [a, +\infty) $、 $ (-\infty, a] $ 和  $ (-\infty, +\infty) $ 都是闭的，而  $ (a, b) $、 $ (a, b] $、 $ [a, b) $、 $ (a, +\infty) $ 和  $ (-\infty, a) $ 都不是闭的。根据引理 9.1.13 可知，N、Z、R 和  $ \varnothing $ 都是闭的，但 Q 不是闭的。

根据引理 9.1.14，我们可以用序列来定义闭包。

推论 9.1.17 设 X 是 R 的子集，如果 X 是闭的，并且  $ (a_{n})_{n=0}^{\infty} $ 是一个由 X 中的元素构成的收敛序列，那么  $ \lim_{n\to\infty}a_{n} $ 也属于 X。反过来，如果每一个由 X 中元素构成的收敛序列  $ (a_{n})_{n=0}^{\infty} $ 的极限也都属于 X，那么 X 一定是闭的。

在下一章学习微分时，我们需要把“附着点”的概念替换成一个与它有着密切联系的概念，即极限点。

定义 9.1.18（极限点）设 X 是实直线的一个子集，我们称 x 是 X 的极限点（或聚点），当且仅当 x 是  $ X \setminus \{x\} $ 的一个附着点。如果  $ x \in X $，并且存在某个  $ \varepsilon > 0 $ 使得  $ |x - y| > \varepsilon $ 对所有的  $ y \in X \setminus \{x\} $ 均成立，那么我们称 x 是 X 的孤立点。

例 9.1.19 设 X 是集合  $ X = (1, 2) \cup \{3\} $，那么 3 是 X 的附着点，但不是 X 的极限点，因为 3 不附着于  $ X - \{3\} = (1, 2) $，但 3 是 X 的一个孤立点。另外，2 是 X 的一个极限点，因为 2 是附着于  $ X - \{2\} = X $ 的，但 2 不是孤立点。（为什么？）

注 9.1.20 根据引理 9.1.14 可知，x 是 X 的极限点，当且仅当存在一个完全由 X 中不同于 x 的元素构成的序列  $ (a_{n})_{n=0}^{\infty} $，并且该序列收敛于 x。这表明附着点组成的集合可以分成两部分，一部分是由极限点组成的集合，另一部分是由孤立点组成的集合（习题 9.1.9）。

引理 9.1.21 设 I 是一个区间（可以是无限的），即 I 是一个形如  $ (a, b) $、 $ (a, b] $、 $ [a, b) $、 $ [a, b] $、 $ (a, +\infty) $、 $ [a, +\infty) $、 $ (- \infty, a) $ 或  $ (- \infty, a] $ 的集合，其中前四种情形均满足 a < b。那么 I 的每一个元素都是 I 的极限点。

他方面的论证与前面相同。

接下来我们定义有界集合的概念。

定义 9.1.22（有界集合）如果存在某个实数 M > 0 使得  $ X \subset [-M, M] $，那么实直线的子集 X 是有界的。

例 9.1.23 对于任意的实数 a、b，区间  $ [a, b] $ 都是有界的，因为它被包含在  $ [-M, M] $ 中，其中  $ M := \max\{|a|, |b|\} $。然而，半无限区间  $ [0, +\infty) $ 是无界的。（为什么？）事实上，所有的半无限区间和双无限区间都是无界的。集合 N、Z、Q 和 R 都是无界的。（为什么？）

关于闭的有界集合的一个基本性质如下。

定理 9.1.24（直线上的海涅-博雷尔定理）设 X 是 R 的一个子集，那么下面两个命题是等价的：

(a) X 是闭的且有界的。

(b) 给定任意一个在 X 中取值（即对所有的 n 均有  $ a_{n} \in X $）的实数序列  $ (a_{n})_{n=0}^{\infty} $，存在它的一个子序列  $ (a_{n_{j}})_{j=0}^{\infty} $ 收敛于 X 中的某个数 L。

证明：参见习题 9.1.13。

注 9.1.25 这个定理在本章的后面几节中将起到关键作用。用距离空间拓扑学的语言来说，该定理断定了实直线的任意一个闭的且有界的子集都是紧的，参见11.7节。这个定理更一般的形式是由爱德华 海涅（1821—1881）和埃米尔 博雷尔（1871—1956）给出的，参见定理 11.7.7。

## 9.2 实值函数的代数

你熟悉很多从实直线到实直线的函数  $ f: \mathbf{R} \rightarrow \mathbf{R} $，例如： $ f(x) := x^{2} + 3x + 5 $、 $ f(x) := 2^{x}/(x^{2} + 1) $ 和  $ f(x) := \sin(x) \exp(x) $（我们将在第15章正式定义  $ \sin $ 和  $ \exp $）。这些都是从  $ \mathbf{R} $ 到  $ \mathbf{R} $ 的函数，因为它们对任意的实数 x 都指定了单独一个实数  $ f(x) $。我们还可以考察一些更独特的函数，如

 $$ f(x):=\begin{cases}1& 若 \;x\in\mathbf{Q}\\0& 若 \;x\notin\mathbf{Q}\end{cases} $$ 

这个函数不是代数的（也就是说，它不能单纯地用 +、-、×、/、√ 等这些关于 x 的标准代数运算来表达，在本书中我们不需要这个概念），但它依然是从 R 到 R 的函数，因为它对每一个  $ x \in \mathbb{R} $ 都指定了一个实数  $ f(x) $。

我们可以从上述定义在  $ \mathbf{R} $ 上的函数  $ f: \mathbf{R} \to \mathbf{R} $ 中任取一个，并把定义域限制在一个更小的集合  $ X \subseteq \mathbf{R} $ 上，从而构造了一个从  $ X $ 到  $ \mathbf{R} $ 的新函数，有时记作  $ f|_{X} $。这个新函数与原函数是一样的，只不过新函数被定义在一个更小的集合上。（所以当  $ x \in X $ 时， $ f|_{X}(x) := f(x) $，而当  $ x \notin X $ 时， $ f|_{X}(x) $ 无定义。）例如，我们可以把最初定义在  $ \mathbf{R} $ 到  $ \mathbf{R} $ 上的函数  $ f(x) := x^2 $ 限制在区间  $ [1, 2] $ 上，于是我们构造了一个新的函数  $ f|_{[1,2]} : [1, 2] \to \mathbf{R} $，它的定义是当  $ x \in [1, 2] $ 时  $ f|_{[1,2]}(x) = x^2 $，

而在其他情况下则没有定义。

当然，如果  $ f(x) $ 的取值都落在 R 的某个更小的子集 Y 内，那么我们也可以把值域从 R 限制到 Y 上。例如，定义为  $ f(x) := x^{2} $ 的函数  $ f: \mathbf{R} \to \mathbf{R} $ 也可以被看作是从 R 到  $ [0, +\infty) $ 的函数，从而取代了 R 到 R 的函数。正式地说，这两个函数是不同的函数，但由于它们之间的区别太小，我们在讨论中通常不太关注函数的值域。

严格地说，一个函数  $ f $ 与该函数在点  $ x $ 处的函数值  $ f(x) $ 是有区别的。 $ f $ 是一个函数，但是  $ f(x) $ 是一个数值（它取决于自变量  $ x $）。两者之间的区别相当微妙，而我们对此不做过多强调，但有时我们必须对两者进行区分。例如，如果  $ f: \mathbf{R} \to \mathbf{R} $ 是函数  $ f(x) := x^2 $，而  $ g := f_{[1,2]} $ 是通过把  $ f $ 限制在区间  $ [1,2] $ 上得到的，那么  $ f $ 和  $ g $ 都是求平方的运算，即  $ f(x) = x^2 $ 且  $ g(x) = x^2 $，但  $ f $ 和  $ g $ 不能看作同一个函数，即  $ f \neq g $，因为它们的定义域不同。尽管存在这样的区别，但我们常常会忽略这一点，并用“考虑函数  $ x^2 + 2x + 3 $”来代替正式的说法：“考虑定义为  $ f(x) := x^2 + 2x + 3 $ 的函数  $ f: \mathbf{R} \to \mathbf{R} $”。（当我们开始涉及微分之类的内容时，这种区别将变得更为明显。例如，如果  $ f: \mathbf{R} \to \mathbf{R} $ 是函数  $ f(x) = x^2 $，那么显然有  $ f(3) = 9 $。但是  $ f $ 在 3 处的导数是 6，而 9 在 3 处的导数却显然是 0。因此我们不能简单地对  $ f(3) = 9 $ 的“两端同时微分”而断言  $ 6 = 0 $。）

如果 X 是 R 的一个子集，并且  $ f: X \to \mathbb{R} $ 是一个函数，那么我们可以构造出函数 f 的图  $ \{(x, f(x)) : x \in X\} $，它是  $ X \times R $ 的一个子集，从而是欧几里得平面  $ R^2 = R \times R $ 的一个子集。利用平面  $ R^2 $ 的几何（例如，使用切线、面积等之类的概念），我们当然可以通过图像来研究函数。但我们将更多地采用“分析的”方法，其中包括利用实数的性质去分析这些函数。这两种方法是互补的，几何方法提供更多的视觉直观，而分析方法则更加严格准确。当把单变量函数的分析推广到多变量（甚至是无限多个变量）函数时，几何直观与解析形式都将变得很有用。

就像数可以进行算术运算一样，函数也可以进行算术运算：两个函数之和是一个函数，两个函数的乘积也是一个函数，等等。

定义 9.2.1（函数的算术运算） 给定两个函数  $ f: X \to \mathbb{R} $ 和  $ g: X \to \mathbb{R} $，定义它们的和  $ f + g: X \to \mathbb{R} $ 为

 $$ (f+g)(x):=f(x)+g(x) $$ 

它们的差  $ f - g: X \to \mathbb{R} $ 为

 $$ (f-g)(x):=f(x)-g(x) $$ 

它们的最大值  $ \max(f, g): X \to \mathbb{R} $ 为

 $$ \max(f,g)(x):=\max(f(x),g(x)) $$ 

它们的最小值  $ \min(f,g):X\to\mathbf{R} $ 为

 $$ \min(f,g)(x):=\min(f(x),g(x)) $$ 

它们的乘积  $  f_g : X \to \mathbf{R}  $（或  $  f \cdot g : X \to \mathbf{R}  $）为

 $$ (fg)(x):=f(x)g(x) $$ 

以及（若对所有的  $ x \in X $ 均有  $ g(x) \neq 0 $）商  $ f/g: X \to \mathbb{R} $ 为

 $$ (f/g)(x):=f(x)/g(x) $$ 

最后，如果 c 是一个实数，我们定义函数  $ cf: X \to \mathbb{R} $（或  $ c \cdot f: X \to \mathbb{R} $）为

 $$ (cf)(x):=c\times f(x) $$ 

例 9.2.2 如果  $ f: \mathbf{R} \to \mathbf{R} $ 是函数  $ f(x): = x^2 $，并且  $ g: \mathbf{R} \to \mathbf{R} $ 是函数  $ g(x): = 2x $，那么  $ f + g: \mathbf{R} \to \mathbf{R} $ 是函数  $ (f + g)(x): = x^2 + 2x $，而  $ fg: \mathbf{R} \to \mathbf{R} $ 是函数  $ (fg)(x): = 2x^3 $。类似地， $ f - g: \mathbf{R} \to \mathbf{R} $ 是函数  $ (f - g)(x): = x^2 - 2x $，而  $ 6f: \mathbf{R} \to \mathbf{R} $ 是函数  $ (6f)(x): = 6x^2 $。观察可知， $ fg $ 与  $ f \circ g $ 不是同一个函数，其中  $ f \circ g $ 是映射  $ x \mapsto 4x^2 $。而且  $ fg $ 与  $ g \circ f $ 也不是同一个函数，其中  $ g \circ f $ 是映射  $ x \mapsto 2x^2 $（为什么？）。因此，函数的乘法与函数的复合是两种不同的运算。

## 9.3 函数的极限值

在第 6 章中，我们定义了一个序列  $ (a_{n})_{n=0}^{\infty} $ 收敛于极限 L 是什么意思。现在我们定义一个类似的概念，即定义在实直线上或者定义在实直线的某个子集上的函数 f 在一点处收敛于某个值是什么意思。就像我们利用  $ \varepsilon $-接近性和最终  $ \varepsilon $-接近性来处理序列的极限那样，我们需要用  $ \varepsilon $-接近性和局部  $ \varepsilon $-接近性的概念来处理函数的极限。

定义 9.3.1（ε-接近性）设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 是一个函数，L 是一个实数，并且设  $ \varepsilon > 0 $ 也是一个实数。我们称函数 f 是  $ \varepsilon $-接近于 L 的，当且仅当对于任意的  $ x \in X $， $ f(x) $ 都是  $ \varepsilon $-接近于 L 的。

例 9.3.2 当函数  $ f(x) := x^{2} $ 被限制在区间  $ [1, 3] $ 上时，f 是 5-接近于 4 的，因为当  $ x \in [1, 3] $ 时  $ 1 \leqslant f(x) \leqslant 9 $，从而  $ |f(x) - 4| \leqslant 5 $。而当 f 被限制在更小的区间  $ [1.9, 2.1] $ 上时，f 是 0.41-接近于 4 的。因为若  $ x \in [1.9, 2.1] $，则有  $ 3.61 \leqslant f(x) \leqslant 4.41 $，从而  $ |f(x) - 4| \leqslant 0.41 $。

定义 9.3.3（局部  $ \varepsilon $-接近性） 设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 是一个函数，L 是一个实数， $ x_0 $ 是 X 的一个附着点，并且设  $ \varepsilon > 0 $ 是一个实数。我们称 f 在  $ x_0 $ 附近是  $ \varepsilon $-接近于 L 的，当且仅当存在一个  $ \delta > 0 $ 使得当 f 被限制在集合  $ \{x \in X: |x - x_0| < \delta\} $ 上时，f 是  $ \varepsilon $-接近于 L 的。

例 9.3.4 设  $ f: [1,3] \rightarrow \mathbb{R} $ 是限制在区间  $ [1,3] $ 上的函数  $ f(x): = x^{2} $，该函数不是 0.1-接近于 4 的，因为比如  $ f(1) $ 就不是 0.1-接近于 4 的。但 f 在 2 附近是 0.1-接近于 4 的，因为当 f 被限制在集合  $ \{x \in [1,3]: |x - 2| < 0.01\} $ 上时，函数 f 的确是 0.1-接近于 4 的。因为当  $ |x - 2| < 0.01 $ 时，我们有  $ 1.99 < x < 2.01 $，从而  $ 3.9601 < f(x) < 4.0401 $。特别地， $ f(x) $ 是 0.1-接近于 4 的。

例 9.3.5 继续考察上例中的函数 f，我们观察到 f 不是 0.1-接近于 9 的，因为比如 f(1) 就不是 0.1-接近于 9 的。但 f 在 3 附近是 0.1-接近于 9 的，因为当 f 被限制在集合  $ \{x \in [1,3] : |x - 3| < 0.01\} $ —— 即半开区间  $ (2.99,3] $ （为什么？）上时，函数 f 是 0.1-接近于 9 的（因为若  $ 2.99 < x \leqslant 3 $，则有  $ 8.9401 < f(x) \leqslant 9 $，从而  $ f(x) $ 是 0.1-接近于 9 的）。

定义 9.3.6（函数在一点处收敛）设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 是一个函数，E 是 X 的一个子集， $ x_0 $ 是 E 的一个附着点，并且设 L 是一个实数。我们称 f 在点  $ x_0 $ 处沿着 E 收敛于 L，并记作  $ \lim_{x \to x_0: x \in E} f(x) = L $，当且仅当对于任意的  $ \varepsilon > 0 $，被限制在 E 上的函数 f 在  $ x_0 $ 附近都是  $ \varepsilon $-接近于 L 的。如果 f 在  $ x_0 $ 处不收敛于任何数 L，那么我们称 f 在  $ x_0 $ 处发散，并且  $ \lim_{x \to x_0: x \in E} f(x) $ 是无定义的。

换言之， $ \lim_{x\to x_0;x\in E}f(x)=L $ 当且仅当对于任意的  $ \varepsilon>0 $，存在一个  $ \delta>0 $ 使得  $ \left|f(x)-L\right|\leqslant\varepsilon $ 对所有满足  $ \left|x-x_0\right|<\delta $ 的  $ x\in E $ 均成立。（为什么这个定义与上面给出的定义是等价的？）

注 9.3.7 在很多情况下，我们都会把上面记号里的集合 E 略去（即我们只说 f 在  $ x_0 $ 处收敛于 L，或者  $ \lim_{x \to x_0} f(x) = L $），尽管这样做的确存在一定的风险。例如，有时 E 是不是真的包含  $ x_0 $ 是有差别的。比如，如果  $ f: \mathbf{R} \to \mathbf{R} $ 是定义如下的函数：当 x = 0 时，令  $ f(x) = 1 $；当 x = 0 时，令  $ f(x) = 0 $，那么  $ \lim_{x \to 0; x \in \mathbf{R} \setminus \{0\}} f(x) = 0 $，而  $ \lim_{x \to 0; x \in \mathbf{R}} f(x) $ 是无定义的。有些作者只定义 E 不包含  $ x_0 $ 时（那么此时  $ x_0 $ 是 E 的极限点而不是附着点）的极限  $ \lim_{x \to x_0; x \in E} f(x) $，或者用  $ \lim_{x \to x_0; x \in E} f(x) $ 来表示我们所说的  $ \lim_{x \to x_0; x \in E \setminus \{x_0\}} f(x) $。但是我们会选取一个更一般的记号，它允许 E 包含  $ x_0 $ 的情况出现。

例 9.3.8 设  $ f: [1,3] \rightarrow \mathbb{R} $ 是函数  $ f(x): = x^{2} $，之前我们已经看到 f 在 2 附近是 0.1-接近于 4 的。同理可以证明 f 在 2 附近是 0.01-接近于 4 的（我们只需要让  $ \delta $ 取一个更小的值就可以了）。

定义 9.3.6 相当不实用，我们可以用更熟悉的语言来改写这个定义，比如序列的极限。

命题 9.3.14（函数的极限定律）设 X 是 R 的一个子集，E 是 X 的一个子集， $ x_{0} $ 是 E 的一个附着点，并且设  $ f: X \to \mathbb{R} $ 和  $ g: X \to \mathbb{R} $ 都是函数。假设 f 在  $ x_{0} $ 处沿着 E 有极限 L，并且 g 在  $ x_{0} $ 处沿着 E 有极限 M，那么  $ f + g $ 在  $ x_{0} $ 处沿着 E 有极限  $ L + M $， $ f - g $ 在  $ x_{0} $ 处沿着 E 有极限 L - M， $ \max(f, g) $ 在  $ x_{0} $ 处沿着 E 有极限  $ \max(L, M) $， $ \min(f, g) $ 在  $ x_{0} $ 处沿着 E 有极限  $ \min(L, M) $，而且  $ fg $ 在  $ x_{0} $ 处沿着 E 有极限 LM。如果 c 是一个实数，那么 cf 在  $ x_{0} $ 处沿着 E 有极限 cL。最后，如果 g 在 E 上不为零（即对一切  $ x \in E $ 均有  $ g(x) \neq 0 $）并且 M 不等于 0，那么  $ f / g $ 在  $ x_{0} $ 处沿着 E 有极限 L/M。

证明：我们只证明第一个结论（即  $ f + g $ 有极限  $ L + M $），其余结论的证明非常类似，并把这部分留作习题 9.3.2。因为  $ x_{0} $ 是 E 的一个附着点，所以根据引理 9.1.14 可知，存在一个由 E 中的元素构成的序列  $ (a_{n})_{n=0}^{\infty} $，它是收敛于  $ x_{0} $ 的。由于 f 在  $ x_{0} $ 处沿着 E 有极限 L，根据命题 9.3.9 可知， $ (f(a_{n}))_{n=0}^{\infty} $ 收敛于 L。类似地， $ (g(a_{n}))_{n=0}^{\infty} $ 收敛于 M。根据序列的极限定律（定理 6.1.19），我们推导出  $ (f + g)(a_{n}))_{n=0}^{\infty} $ 收敛于  $ L + M $。再次由命题 9.3.9 可知，这蕴涵了  $ f + g $ 在  $ x_{0} $ 处沿着 E 有极限  $ L + M $（因为  $ (a_{n})_{n=0}^{\infty} $ 是 E 中任意一个收敛于  $ x_{0} $ 的序列），这就是要证明的结论。

例 9.3.17 设  $ f(x) $ 是函数

 $$ f(x):=\begin{cases}1&\quad 若 x=0\\0&\quad 若 x\neq0\end{cases} $$ 

于是  $ \lim_{x\to0;x\in\mathbf{R}-\{0\}}f(x)=0 $ ，（为什么？）但  $ \lim_{x\to0;x\in\mathbf{R}}f(x) $ 无定义。（为什么？）（在这种情况下，我们说 f 在 0 处有“可去奇点”或“可去间断点”。由于这种奇点的存在，我们有时约定写  $ \lim_{x\to x_{0}}f(x) $ 时就默认把  $ x_{0} $ 排除在集合之外。例如，本书中  $ \lim_{x\to x_{0}}f(x) $ 是  $ \lim_{x\to x_{0};x\in X-\{x_{0}\}}f(x) $ 的简写形式。）

另外，在  $ x_{0} $ 处的极限只与  $ x_{0} $ 附近的函数值有关，远离  $ x_{0} $ 的函数值与该极限无关。下面的命题就体现了这种直观。

命题 9.3.18（极限是局部的）设 X 是 R 的一个子集，E 是 X 的一个子集， $ x_{0} $ 是 E 的一个附着点， $ f: X \to \mathbb{R} $ 是一个函数，并且设 L 是一个实数。设  $ \delta > 0 $，那么我们有

 $$ \lim_{x\to x_{0};x\in E}f(x)=L $$ 

当且仅当

 $$ \lim_{x\to x_{0};x\in E\cap(x_{0}-\delta,x_{0}+\delta)}f(x)=L $$ 

证明：参见习题 9.3.3。

通俗地说，上面的命题断定了

 $$ \lim_{x\to x_{0};x\in E}f(x)=\lim_{x\to x_{0};x\in E\cap(x_{0}-\delta,x_{0}+\delta)}f(x) $$ 

于是，如果函数在  $ x_{0} $ 处的极限存在，那么它只取决于  $ x_{0} $ 附近函数 f 的值，较远处的值实际上对该极限没有任何影响。

现在我们再给出几个极限的例子。

例 9.3.19 考虑函数  $ f: \mathbf{R} \rightarrow \mathbf{R} $ 和  $ g: \mathbf{R} \rightarrow \mathbf{R} $，它们分别被定义为  $ f(x) := x + 2 $ 和  $ g(x) := x + 1 $，那么  $ \lim_{x \to -2; x \in \mathbf{R}} f(x) = 4 $ 且  $ \lim_{x \to -2; x \in \mathbf{R}} g(x) = 3 $。我们希望利用极限定律推导出  $ \lim_{x \to -2; x \in \mathbf{R}} f(x)/g(x) = 4/3 $，即  $ \lim_{x \to -2; x \in \mathbf{R}} \frac{x + 2}{x + 1} = \frac{4}{3} $。严格地说，我们无法使用命题 9.3.14 来保证这一点，这是因为  $ x + 1 $ 在  $ x = -1 $ 处为 0，所以  $ f(x)/g(x) $ 是无定义的。但很容易解决这个问题，只需要把  $ f $ 和  $ g $ 的定义域从  $ \mathbf{R} $ 限制到更小的区域上就可以了，如  $ \mathbf{R} - \{-1\} $。所以根据命题 9.3.14，我们可以得到  $ \lim_{x \to -2; x \in \mathbf{R} - \{-1\}} \frac{x + 2}{x + 1} = \frac{4}{3} $。

例 9.3.20 考虑定义为  $ f(x) = (x^{2} - 1)/(x - 1) $ 的函数  $ f: \mathbf{R} - \{1\} \to \mathbf{R} $。对于任意一个不等于 1 的实数，这个函数都是有意义的，而  $ f(1) $ 是无定义的。但 1 仍是  $ \mathbf{R} - \{1\} $ 的一个附着点（为什么？），并且极限  $ \lim_{x \to 1; x \in \mathbf{R} - \{1\}} f(x) $ 是有定义的。这是因为在定义域  $ \mathbf{R} - \{1\} $ 上我们有恒等式  $ (x^{2} - 1)/(x - 1) = (x + 1)(x - 1)/(x - 1) = x + 1 $，而且  $ \lim_{x \to 1; x \in \mathbf{R} - \{1\}} x + 1 = 2 $。

例 9.3.21 设  $ f: \mathbf{R} \to \mathbf{R} $ 为函数

 $$ f(x):=\begin{cases}1& 若 x\in\mathbf{Q}\\0& 若 x\notin\mathbf{Q}\end{cases} $$ 

我们要证明  $ f(x) $ 在 0 处沿着 R 没有极限。利用反证法，假设  $ f(x) $ 在 0 处沿着 R 有某个极限 L。那么只要  $ (a_{n})_{n=1}^{\infty} $ 是由不为零的数构成的一个收敛于 0 的序列，我们就有  $ \lim_{n\to\infty}f(a_{n})=L $。因为  $ (1/n)_{n=1}^{\infty} $ 就是这样的序列，因此

 $$ L=\lim_{n\to\infty}f(1/n)=\lim_{n\to\infty}1=1 $$ 

另外，因为  $ (\sqrt{2}/n)_{n=1}^{\infty} $ 是另一个由不为零的数构成的收敛于 0 的序列，但这些数都是无理数而非有理数，所以

 $$ L=\lim_{n\to\infty}f(\sqrt{2}/n)=\lim_{n\to\infty}0=0 $$ 

又因为 1 = 0，这就出现了矛盾。因此这个函数在 0 处没有极限。

## 9.4 连续函数

现在我们引入函数理论中最基本的概念之一，即连续的概念。

定义 9.4.1（连续）设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 是一个函数，并且设  $ x_{0} $ 是 X 中的一个元素，我们称 f 在  $ x_{0} $ 处是连续的，当且仅当

 $$ \lim_{x\to x_{0};x\in X}\lim f(x)=f(x_{0}) $$ 

换言之，当 x 沿着 X 收敛于  $ x_{0} $ 时， $ f(x) $ 的极限存在并且等于  $ f(x_{0}) $。我们称 f 在 X 上是连续的（或简单地说连续的），当且仅当对于任意的  $ x_{0} \in X $，f 在  $ x_{0} $ 处都是连续的。我们称 f 在  $ x_{0} $ 处是间断的，当且仅当 f 在  $ x_{0} $ 处是不连续的。

例 9.4.2 设 c 是一个实数，并且设  $ f: \mathbf{R} \to \mathbf{R} $ 是常数函数  $ f(x): = c $，那么对任意实数  $ x_0 \in \mathbf{R} $，我们有

 $$ \lim_{x\to x_{0};x\in\mathbf{R}}f(x)=\lim_{x\to x_{0};x\in\mathbf{R}}c=c=f(x_{0}) $$ 

于是  $ f $ 在每一个点  $ x_0 \in \mathbb{R} $ 处都连续，或者说  $ f $ 在  $ \mathbb{R} $ 上是连续的。

例 9.4.3 设  $ f: \mathbf{R} \to \mathbf{R} $ 是恒等函数  $ f(x) := x $，那么对于任意的实数  $ x_0 \in \mathbf{R} $，我们有

 $$ \lim_{x\to x_{0};x\in\mathbf{R}}f(x)=\lim_{x\to x_{0};x\in\mathbf{R}}x=x_{0}=f(x_{0}) $$ 

于是  $ f $ 在每一个点  $ x_0 \in \mathbb{R} $ 处都连续，或者说  $ f $ 在  $ \mathbb{R} $ 上是连续的。

例 9.4.4 设  $ sgn: \mathbf{R} \rightarrow \mathbf{R} $ 是例 9.3.16 中定义的符号函数，那么  $ sgn $ 在每一个非零的  $ x $ 处都是连续的。例如，在 1 处，我们有（利用命题 9.3.18）

 $$ \begin{aligned}\lim_{x\rightarrow1;x\in\mathbf{R}}sgn(x)&=\lim_{x\rightarrow1;x\in(0.9,1.1)}sgn(x)\\&=\lim_{x\rightarrow1;x\in(0.9,1.1)}1\\&=1\\&=sgn(1)\end{aligned} $$ 

另外，sgn 在 0 处不连续，因为极限  $ \lim_{x\to0:x\in\mathbf{R}}\operatorname{sgn}(x) $ 不存在。

例 9.4.5 设  $ f: \mathbf{R} \to \mathbf{R} $ 是函数

 $$ f(x):=\begin{cases}1& 若 x\in\mathbf{Q}\\0& 若 x\notin\mathbf{Q}\end{cases} $$ 

那么根据上一节的讨论可知，f 在 0 处不连续。事实上，f 在任意实数  $ x_{0} $ 处都不连续（你能看出为什么吗？）。

例 9.4.6 设  $ f: \mathbf{R} \to \mathbf{R} $ 是函数

 $$ f(x):=\begin{cases}1&\quad 若 x\geqslant0\\0&\quad 若 x<0\end{cases} $$ 

那么 f 在任意的非零实数处都是连续的（为什么？），但 f 在 0 处不连续。然而，如果我们把 f 限制在右半直线  $ [0, +\infty) $ 上，那么得到的函数  $ f|_{[0, +\infty)} $ 在它定义域内的每一点（包含 0）处都是连续的。于是，通过对函数的定义域进行限制可以把一个间断的函数重新变成连续的。

有多种方式可以表述“f 在  $ x_{0} $ 处是连续的”。

命题 9.4.7（连续性的等价表述）设 X 是 R 的一个子集， $ f: X \to R $ 是一个函数，并且设  $ x_{0} $ 是 X 的元素。那么下面四个命题在逻辑上是等价的：

(a) f 在  $ x_{0} $ 处是连续的。

(b) 对于任意一个由 X 中元素构成的且满足  $ \lim_{n\to\infty}a_{n}=x_{0} $ 的序列  $ (a_{n})_{n=0}^{\infty} $，都有  $ \lim_{n\to\infty}f(a_{n})=f(x_{0}) $。

(c) 对于任意的  $ \varepsilon > 0 $，都存在一个  $ \delta > 0 $ 使得  $ |f(x) - f(x_0)| < \varepsilon $ 对所有满足  $ |x - x_0| < \delta $ 的  $ x \in X $ 均成立。

(d) 对于任意的  $ \varepsilon > 0 $，都存在一个  $ \delta > 0 $ 使得  $ \left|f(x) - f(x_{0})\right| \leqslant \varepsilon $ 对所有满足  $ \left|x - x_{0}\right| \leqslant \delta $ 的  $ x \in X $ 均成立。

证明：参见习题 9.4.1。

注 9.4.8 命题 9.4.7 的一个特别实用的结论如下：如果 f 在  $ x_{0} $ 处是连续的，并且当  $ n \to \infty $ 时  $ a_{n} \to x_{0} $，那么当  $ n \to \infty $ 时  $ f(a_{n}) \to f(x_{0}) $（当然，序列  $ (a_{n})_{n=0}^{\infty} $ 的所有元素都在 f 的定义域中）。因此，连续函数在计算极限时是非常有用的。

命题 9.3.14 中的极限定律与定义 9.4.1 中的连续性概念结合在一起，就可以马上推出下述命题。

命题 9.4.9（算术运算保持连续性）设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 和  $ g: X \to \mathbb{R} $ 都是函数，并且设  $ x_0 \in X $。如果 f 和 g 在  $ x_0 $ 处都是连续的，那么函数  $ f + g $、 $ f - g $、 $ \max(f, g) $、 $ \min(f, g) $ 和  $ fg $ 也都在  $ x_0 $ 处连续。如果 g 在 X 上不为零，那么  $ f/g $ 在  $ x_0 $ 处也连续。

特别地，连续函数的和、差、最大值、最小值以及乘积都是连续的，而且只要分母不为零，两个连续函数的商也是连续的。

我们可以利用命题 9.4.9 来证明大量函数的连续性。例如，只利用常数函数是连续的和恒等函数  $ f(x)=x $ 是连续的（习题 9.4.2），我们就可以证明函数  $ g(x):=\max(x^{3}+4x^{2}+x+5,x^{4}-x^{3})/(x^{2}-4) $ 在 R 中除了 x=2 和 x=-2（在这两点处分母为零）以外的每一点处都是连续的。

## 9.5 左极限和右极限

我们引入左极限和右极限的概念，它们可以看作是一个完整极限  $ \lim_{x\to x_0;x\in X}f(x) $ 的两个分离的“半边”。

定义 9.5.1（左极限和右极限）设 X 是 R 的一个子集， $ f:X \to R $ 是一个函数，并且设  $ x_{0} $ 是一个实数。如果  $ x_{0} $ 是  $ X \cap (x_{0}, +\infty) $ 的一个附着点，那么我们定义 f 在  $ x_{0} $ 处的右极限  $ f(x_{0}+ $ 为

 $$ f(x_{0}+):=\lim_{x\to x_{0};x\in X\cap(x_{0},+\infty)}f(x) $$ 

当然前提是这个极限是存在的。类似地，如果  $ x_{0} $ 是  $ X \cap (-\infty, x_{0}) $ 的一个附着点，那么我们定义 f 在  $ x_{0} $ 处的左极限  $ f(x_{0}- $ 为

 $$ f(x_{0}-):=\lim_{x\to x_{0};x\in X\cap(-\infty,x_{0})}f(x) $$ 

当然前提也是这个极限是存在的。(因此，在很多情况下， $ f(x_{0}+) $ 和  $ f(x_{0}-) $ 是无定义的。)

我们有时会采用下面这些简化的记号：

 $$ \lim_{x\to x_{0}+}f(x):=\lim_{x\to x_{0};x\in X\cap(x_{0},+\infty)}f(x) $$ 

 $$ \lim_{x\to x_{0}-}f(x):=\lim_{x\to x_{0};x\in X\cap(-\infty,x_{0})}f(x) $$

由命题 9.4.7 可知，如果右极限  $ f(x_{0}+) $ 存在，并且  $ (a_{n})_{n=0}^{\infty} $ 是 X 中的一个从右侧收敛于  $ x_{0} $ 的序列（即对于所有的  $ n \in \mathbf{N} $ 均有  $ a_{n} > x_{0} $），那么  $ \lim_{n \to \infty} f(a_{n}) = f(x_{0}+) $。类似地，如果  $ (b_{n})_{n=0}^{\infty} $ 是一个从左侧收敛于  $ x_{0} $ 的序列（即对所有的  $ n \in \mathbf{N} $ 均有  $ b_{n} < x_{0} $），那么  $ \lim_{n \to \infty} f(b_{n}) = f(x_{0}-) $。

设  $ x_{0} $ 同时是  $ X \cap (x_{0}, +\infty) $ 和  $ X \cap (-\infty, x_{0}) $ 的附着点，如果 f 在  $ x_{0} $ 处连续，那么根据命题 9.4.7 可知， $ f(x_{0}+) $ 和  $ f(x_{0}-) $ 都存在且都等于  $ f(x_{0}) $。（你能看出为什么吗？）反过来也成立（将此与命题 6.4.12(f) 进行对比）。

命题 9.5.3 设 X 是 R 的一个包含实数  $ x_{0} $ 的子集，并且设  $ x_{0} $ 同时是  $ X \cap (x_{0}, +\infty) $ 和  $ X \cap (-\infty, x_{0}) $ 的附着点，设  $ f: X \to \mathbb{R} $ 是一个函数。如果  $ f(x_{0}+) $ 和  $ f(x_{0}-) $ 都存在且都等于  $ f(x_{0}) $，那么 f 在  $ x_{0} $ 处是连续的。

证明：我们记  $ L := f(x_{0}) $，根据假设，我们有

 $$ \lim_{x\to x_{0};x\in X\cap(x_{0},+\infty)}f(x)=L $$ 

以及

 $$ \lim_{x\to x_{0};x\in X\cap(-\infty,x_{0})}f(x)=L $$ 

设  $ \varepsilon > 0 $ 是给定的，根据式(9.1) 和命题 9.4.7（应用于  $ f $ 在  $ X \cap (x_0, +\infty) $ 上的限制函数）可知，存在一个  $ \delta_+ > 0 $ 使得  $ |f(x) - L| < \varepsilon $ 对所有满足  $ |x - x_0| < \delta_+ $ 的  $ x \in X \cap (x_0, +\infty) $ 均成立。类似地，根据式(9.2) 可知，存在一个  $ \delta_- > 0 $ 使得  $ |f(x) - L| < \varepsilon $ 对所有满足  $ |x - x_0| < \delta_- $ 的  $ x \in X \cap (-\infty, x_0) $ 均成立。现在设  $ \delta := \min(\delta_-, \delta_+ $)，那么  $ \delta > 0 $（为什么？）。假设  $ x \in X $ 满足  $ |x - x_0| < \delta $，于是存在三种情形： $ x > x_0 $、 $ x = x_0 $ 和  $ x < x_0 $，但在这三种情形下都有  $ |f(x) - L| < \varepsilon $（为什么？每种情形的原因是不同的）。于是根据命题 9.4.7 可知， $ f $ 在  $ x_0 $ 处是连续的，结论得证。

正如我们在例 9.3.16 中看到的符号函数那样，函数 f 在点  $ x_{0} $ 的左极限  $ f(x_{0}-) $ 和右极限  $ f(x_{0}+) $ 有可能同时存在但不相等。此时，我们称 f 在  $ x_{0} $ 处有一个跳跃间断点。例如符号函数在 0 处有一个跳跃间断点。另外，左极限  $ f(x_{0}-) $ 和右极限  $ f(x_{0}+) $ 也有可能同时存在且相等，但都不等于  $ f(x_{0}) $。此时，我们称 f 在  $ x_{0} $ 处有一个可去间断点（或可去奇点）。例如，若设  $ f: \mathbf{R} \to \mathbf{R} $ 为函数

 $$ f(x):=\begin{cases}1&\quad 若 x=0\\0&\quad 若 x\neq0\end{cases} $$ 

那么  $ f(0+ $ 和  $ f(0- $ 都存在且都等于 0（为什么？），但是  $ f(0) $ 等于 1。所以 f 在 0 处有一个可去间断点。

注 9.5.4 函数间断点并不是只有跳跃间断点和可去间断点这两种类型。另一种类型是函数在间断点处趋向于无穷大的情形。例如，定义为  $ f(x) := 1/x $ 的函数  $ f : \mathbf{R} - \{0\} \to \mathbf{R} $ 在 0 处有一个间断点，但它既不是跳跃间断点也不是可去奇点。通俗地说，当 x 从右侧趋向于 0 时， $ f(x) $ 收敛于  $ +\infty $，而当 x 从左侧趋向于 0 时， $ f(x) $ 收敛于  $ -\infty $。这种类型的奇点有时被称作渐近间断点。还有一种间断点叫作振荡间断点，其中函数 f 在  $ x_0 $ 附近有界但没有极限。例如，定义为

 $$ f(x):=\begin{cases}1& 若 x\in\mathbf{Q}\\0& 若 x\notin\mathbf{Q}\end{cases} $$ 

的函数  $ f: \mathbf{R} \to \mathbf{R} $ 在 0 处有一个振荡间断点（事实上，在任意其他的实数处都有震荡间断点），这是因为函数在 0 处既没有左极限也没有右极限，尽管该函数是有界的。

间断性（也叫作奇异性）的研究不断发展，但这超出了本书的范围。例如，奇异性在复分析中就有着关键作用。

## 9.6 最大值原理

虽然从前两节中我们看到很多函数都是连续的，但并非所有的函数都是连续的。现在我们证明连续函数具有许多其他有用的性质，尤其当它们的定义域为闭区间时。现在，我们开始利用海涅-博雷尔定理（定理 9.1.24）所具有的强大力量。

定义 9.6.1 设 X 是 R 的一个子集，并且设  $ f: X \to \mathbb{R} $ 是一个函数。如果存在一个实数 M 使得对所有的  $ x \in X $ 均有  $ f(x) \leqslant M $，那么我们称 f 是有上界的。如果存在一个实数 M 使得对所有的  $ x \in X $ 均有  $ f(x) \geqslant -M $，那么我们称 f 是有下界的。如果存在一个实数 M 使得对所有的  $ x \in X $ 均有  $ |f(x)| \leqslant M $，那么我们称 f 是有界的。

注 9.6.2 一个函数是有界的，当且仅当它既有上界又有下界。(为什么？注意，“当且仅当”其中一部分的技巧性比另一部分要稍强一些)另外，函数  $ f: X \to \mathbb{R} $

是有界的，当且仅当它的象  $ f(X) $ 在定义 9.1.22 的意义下是一个有界集合。(为什么？)

并非所有的连续函数都是有界的。例如，定义域为 R 的函数  $ f(x) := x $ 是连续的，但是无界的，（为什么？）尽管它在某个更小的定义域（如  $ [1,2] $）上是有界的。函数  $ f(x) := 1/x $ 在  $ (0,1) $ 上是连续的，但是无界的，（为什么？）虽然它在  $ [1,2] $ 上既是连续的又是有界的。（为什么？）然而如果连续函数的定义域是一个有界的闭区间，那么它必定是有界的。

引理 9.6.3 设 a < b 都是实数，并且设  $ f: [a, b] \to \mathbb{R} $ 是  $ [a, b] $ 上的连续函数，那么 f 是一个有界函数。

证明：利用反证法，假设  $ f $ 是无界的，那么对任意的实数  $ M $ 都存在一个元素  $ x \in [a, b] $ 使得  $ |f(x)| \geq M $。

特别地，对于每一个自然数 n，集合  $ \{x \in [a, b] : |f(x)| \geq n\} $ 都是非空的。所以我们可以选取  $ {}^{①}[a, b] $ 中的一个序列  $ (x_{n})_{n=0}^{\infty} $ 使得  $ |f(x_{n})| \geq n $ 对所有的 n 均成立。由于这个序列属于  $ [a, b] $，从而根据定理 9.1.24 可知，存在一个收敛于某个极限  $ L \in [a, b] $ 的子序列  $ (x_{n_{j}})_{j=0}^{\infty} $，其中  $ n_{0} < n_{1} < n_{2} < \cdots $ 是一个递增的自然数序列。特别地，对于所有的  $ j \in \mathbf{N} $，均有  $ n_{j} \geq j $。（为什么？利用归纳法。）

因为 f 在  $ [a,b] $ 上连续，所以它在 L 处是连续的，并且我们有

 $$ \lim_{j\to\infty}f(x_{n_{j}})=f(L) $$ 

所以序列  $ (f(x_{n_j}))_{j=0}^{\infty} $ 是收敛的，从而是有界的。另外，我们从序列的构造过程中看出  $ |f(x_{n_j})| \geqslant n_j \geqslant j $ 对所有的  $ j $ 均成立，从而序列  $ (f(x_{n_j}))_{j=0}^{\infty} $ 是无界的，这是一个矛盾。

注 9.6.4 关于这个证明，有两点值得我们注意。第一，它表明了海涅-博雷尔定理（定理 9.1.24）多么有用。第二，它是一个间接的证明，没有说明该如何找到 f 的界，而是证明了函数 f 无界将导致矛盾。

现在我们加强引理 9.6.3 来谈论更多的事情。

定义 9.6.5（最大值和最小值）设  $ f: X \to \mathbb{R} $ 是一个函数，并且设  $ x_0 \in X $。如果对于所有的  $ x \in X $ 均有  $ f(x_0) \geq f(x) $（即  $ f $ 在  $ x_0 $ 处的值大于或等于  $ f $ 在  $ X $ 中其他任意一点处的值），那么我们说  $ f $ 在  $ x_0 $ 处达到它的最大值。如果  $ f(x_0) \leq f(x) $，那么我们说  $ f $ 在  $ x_0 $ 处达到它的最小值。

注 9.6.6 如果一个函数在某一点处达到它的最大值，那么该函数一定有上界。（为什么？）类似地，如果它在某一点处达到最小值，那么它一定有下界。最大值和最小值这些概念都是全局性的，局部性的形式将在定义 10.2.1 中给出。

命题 9.6.7（最大值原理）设  $ a < b $ 都是实数，并且设  $ f: [a, b] \to \mathbb{R} $ 是  $ [a, b] $ 上的连续函数。那么  $ f $ 在某一点  $ x_{\max} \in [a, b] $ 处达到最大值，并且在某一点  $ x_{\min} \in [a, b] $ 处达到最小值。

注 9.6.8 严格地说，“最大值原理”是一个不恰当的说法，因为这个原理中还涉及了最小值。一个更加准确的名称或许应该是“最值原理”，“最值”一词同时表示“最大值”和“最小值”。

证明：我们只证明 f 在某点处达到最大值，关于 f 在某点处达到最小值的证明是类似的，这部分内容留给读者。

根据引理 9.6.3 可知，f 是有界的，所以存在一个 M 使得  $ -M \leqslant f(x) \leqslant M $ 对任意的  $ x \in [a, b] $ 均成立。现在设 E 表示集合

 $$ E:=\{f(x):x\in[a,b]\} $$ 

（即  $ E := f([a, b]) $）。根据上述内容可知，这个集合是  $ [-M, M] $ 的子集。而且 E 还是非空集合，因为它包含点  $ f(a) $。根据最小上界原理可知，它有一个实数上确界  $ \sup(E) $。

记  $ m := \sup(E) $，根据上确界的定义可知，对所有的  $ y \in E $ 均有  $ y \leqslant m $。而根据 E 的定义可知，这意味着  $ f(x) \leqslant m $ 对所有的  $ x \in [a, b] $ 均成立。因此，为了证明 f 在某点处达到最大值，我们只需要找到一个  $ x_{\max} \in [a, b] $ 使得  $ f(x_{\max}) = m $ 即可。（为什么只要这样就可以了？）

设  $ n \geqslant 1 $ 是任意一个整数，那么  $ m - \frac{1}{n} < m = \sup(E) $。因为  $ \sup(E) $ 是 E 的最小上界，所以  $ m - \frac{1}{n} $ 不可能是 E 的上界，从而存在一个  $ y \in E $ 使得  $ m - \frac{1}{n} < y $。又由 E 的定义可知，这蕴涵着存在一个  $ x \in [a, b] $ 使得  $ m - \frac{1}{n} < f(x) $。

现在我们按照下面的方法选取一个序列  $ (x_n)_n=1 $：对于每一个  $ n $，选取  $ x_n $ 为  $ [a,b] $ 中使得  $ m-\frac{1}{n}<f(x_n) $ 的元素。（同样，这需要用到选择公理，然而不利用选择公理也可以证明这个原理。例如，在命题 11.10.2 中你将看到一个更好的证明，它使用了紧致性概念证明这个命题。）这是  $ [a,b] $ 中的一个序列，于是根据海涅-博雷尔定理（定理 9.1.24），我们可以找到一个收敛于某极限  $ x_{\max}\in[a,b] $ 的子序列  $ (x_{n_j})_{j=1}^{\infty} $，其中  $ n_1<n_2<\cdots $。因为  $ (x_{n_j})_{j=1}^{\infty} $ 收敛于  $ x_{\max} $ 并且  $ f $ 在  $ x_{\max} $ 处连续，所以我们像前面那样得到：

 $$ \lim_{j\to\infty}f(x_{n_{j}})=f(x_{\max}) $$ 

另外，根据该序列的构造过程可知，

 $$ f(x_{n_{j}})>m-\frac{1}{n_{j}}\geqslant m-\frac{1}{j} $$ 

从而对上式两端同时取极限可得，

 $$ f(x_{\max})=\lim_{j\to\infty}f(x_{n_{j}})\geqslant\lim_{j\to\infty}m-\frac{1}{j}=m $$ 

另外， $ f(x) \leqslant m $ 对所有的  $ x \in [a, b] $ 均成立，从而  $ f(x_{\max}) \leqslant m $。联合这两个不等式就得到  $ f(x_{\max}) = m $，结论得证。

注意，最大值原理无法阻止函数在多个点处达到最大值或者最小值。例如，定义在区间  $ [-2,2] $ 上的函数  $ f(x)=x^{2} $ 在两个不同的点 -2 和 2 处都达到最大值。

我们把  $ \sup\{f(x):x\in[a,b]\} $ 简写成  $ \sup_{x\in[a,b]}f(x) $，并类似地定义  $ \inf_{x\in[a,b]}f(x) $。于是最大值原理断定了  $ m:=\sup_{x\in[a,b]}f(x) $ 是一个实数，并且它是 f 在  $ [a,b] $ 上的最大值，即至少存在一个点  $ x_{\max}\in[a,b] $ 使得  $ f(x_{\max})=m $，而且对任意其他的  $ x\in[a,b] $， $ f(x) $ 都是小于或等于 m 的。类似地， $ \inf_{x\in[a,b]}f(x) $ 是 f 在  $ [a,b] $ 上的最小值。

现在我们知道在闭区间上，每一个连续的函数都是有界的并且至少有一次达到它的最大值，也至少有一次达到它的最小值。但对于开区间和无限区间而言，上述结论就不成立了，参见习题 9.6.1。

注 9.6.9 在复分析或偏微分方程中，你可能会遇到相当不同的“最大值原理”，其中连续函数在复分析和偏微分方程中将分别被替换成解析函数和调和函数。那些最大值原理与这里的并没有直接的关联（尽管它们也涉及最大值是否存在以及在哪里达到最大值）。

## 9.7 介值定理

我们刚刚证明了一个连续函数达到了它的最大值和最小值，现在来证明函数

f 还可以达到最大值和最小值之间的每一个值。因此，我们首先证明一个非常直观的定理。

定理 9.7.1（介值定理）设 a < b， $ f: [a, b] \to \mathbb{R} $ 是  $ [a, b] $ 上的连续函数，并且设 y 是  $ f(a) $ 和  $ f(b) $ 之间的一个实数，即要么  $ f(a) \leqslant y \leqslant f(b) $，要么  $ f(a) \geqslant y \geqslant f(b) $，那么存在  $ c \in [a, b] $ 使得  $ f(c) = y $。

证明：存在两种情形： $ f(a) \leqslant y \leqslant f(b) $ 和  $ f(a) \geqslant y \geqslant f(b) $。我们不妨假设前一种情形，即  $ f(a) \leqslant y \leqslant f(b) $。后一种情形的证明类似，留给读者。

如果  $ y = f(a) $ 或  $ y = f(b) $，那么容易得出结论，因为我们只需要简单地令 c = a 或 c = b 就可以了。所以我们考虑  $ f(a) < y < f(b) $ 的情形，令 E 表示集合

 $$ E:=\{x\in[a,b]:f(x)<y\} $$ 

显然 E 是  $ [a,b] $ 的子集，从而 E 是有界的。另外，因为  $ f(a) < y $，所以 a 是 E 中的元素，因此 E 是非空的。根据最小上界原理可知，上确界

 $$ c:=\sup(E) $$ 

是有限的。因为 E 以 b 为上界，所以  $ c \leqslant b $。又因为 E 中包含 a，所以  $ c \geqslant a $。因此  $ c \in [a, b] $。为了完成证明，现在证明  $ f(c) = y $。思路是从 c 的左侧去证明  $ f(c) \leqslant y $，然后从 c 的右侧去证明  $ f(c) \geqslant y $。

设  $ n \geqslant 1 $ 是整数。数  $ c - \frac{1}{n} $ 小于  $ c = \sup(E) $，所以它不可能是 E 的上界。于是存在一个点，记作  $ x_{n} $，属于 E 并且大于  $ c - \frac{1}{n} $。而且根据 c 是 E 的上界可知， $ x_{n} \leqslant c $。于是

 $$ c-\frac{1}{n}\leqslant x_{n}\leqslant c $$ 

根据夹逼定理（推论 6.4.14）可得， $ \lim_{n\to\infty}x_n=c $。又根据 f 在 c 处连续可知，这蕴涵了  $ \lim_{n\to\infty}f(x_n)=f(c) $。因为对每一个 n 都有  $ x_n\in E $，所以  $ f(x_n)<y $ 对所有的 n 均成立。根据比较原理（引理 6.4.13）可知， $ f(c)\leq y $。因为  $ f(b)>f(c) $，所以  $ c\neq b $。

由于  $ c \neq b $ 且  $ c \in [a, b] $，那么必定有  $ c < b $。特别地，存在一个 N > 0 使得对所有的 n > N 都有  $ c + \frac{1}{n} < b $（因为当  $ n \to \infty $ 时， $ c + \frac{1}{n} $ 收敛于 c）。因为 c 是 E 的上确界且  $ c + \frac{1}{n} > c $，所以  $ c + \frac{1}{n} \notin E $ 对所有的 n > N 均成立。又因为  $ c + \frac{1}{n} \in [a, b] $，于是对所有的  $ n \geq N $ 都有  $ f(c + \frac{1}{n}) \geq y $。而  $ c + \frac{1}{n} $ 收敛于 c，并且 f 在 c 处连续，于是  $ f(c) \geq y $。但是我们已经知道  $ f(c) \leq y $，于是  $ f(c) = y $，这就是要证明的结论。

介值定理表明如果 f 的值能够取到  $ f(a) $ 和  $ f(b) $，那么它也一定能够取到介于  $ f(a) $ 和  $ f(b) $ 之间的所有值。注意，如果没有假设 f 是连续的，那么就不能使用介值定理。例如，如果  $ f: [-1,1] \rightarrow \mathbf{R} $ 是函数

 $$ f(x):=\begin{cases}-1&\quad 若 x\leqslant0\\1&\quad 若 x>0\end{cases} $$ 

那么  $  f(-1) = -1  $ 且  $  f(1) = 1  $，但是不存在  $  c \in [-1, 1]  $ 使得  $  f(c) = 0  $。因此，如果函数是间断的，那么它可能会“跳过”中间值，但连续函数就不会发生这样的状况。

注 9.7.2 一个连续函数有可能会多次取到同一个中间值。例如，若  $ f: [-2,2] \rightarrow \mathbb{R} $ 是函数  $ f(x): = x^{3} - x $，那么  $ f(-2) = -6 $ 且  $ f(2) = 6 $，所以存在  $ c \in [-2,2] $ 使得  $ f(c) = 0 $。事实上，此时存在三个这样的  $ c: f(-1) = f(0) = f(1) = 0 $。

注 9.7.3 介值定理给出了证明可以取到一个数的 n 次方根的另一种方法。例如，为了构造 2 的平方根，考虑定义为  $ f(x) := x^{2} $ 的函数  $ f : [0, 2] \to \mathbf{R} $。这是一个连续函数并且  $ f(0) = 0 $， $ f(2) = 4 $。于是存在一个  $ c \in [0, 2] $ 使得  $ f(c) = 2 $，即  $ c^{2} = 2 $（这种论述没有证明 2 恰有一个平方根，而是证明了 2 至少有一个平方根）。

推论 9.7.4（连续函数的象）设  $ a < b $， $ f: [a, b] \to \mathbb{R} $ 是  $ [a, b] $ 上的连续函数，设  $ M := \sup_{x \in [a, b]} f(x) $ 是  $ f $ 的最大值， $ m := \inf_{x \in [a, b]} f(x) $ 是  $ f $ 的最小值，并且设  $ y $ 是介于  $ m $ 和  $ M $ 之间的一个实数（即  $ m \leqslant y \leqslant M $），那么存在一个  $ c \in [a, b] $ 使得  $ f(c) = y $。更进一步地， $ f([a, b]) = [m, M] $。

证明：参见习题 9.7.1。

习题

9.7.1 证明推论 9.7.4。(提示：除了介值定理外，你可能还会用到习题 9.4.6。)

9.7.2 设  $ f: [0,1] \rightarrow [0,1] $ 是一个连续函数。证明：在  $ [0,1] $ 中存在一个实数 x 使得  $ f(x) = x $。(提示：对函数  $ f(x) - x $ 使用中值定理。) 这个点 x 被称作 f 的不动点，这个结果是不动点定理的一个基本例子，它在一定类型的分析理论中有着重要作用。

## 9.8 单调函数

现在我们讨论一类函数，它与连续函数不同，但又具有某些与连续函数类似的性质，即单调函数。

定义 9.8.1（单调函数）设 X 是  $ \mathbf{R} $ 的一个子集，并且设  $ f: X \to \mathbf{R} $ 是一个函数。我们称 f 是单调递增的，当且仅当只要  $ x, y \in X $ 且  $ y > x $，就有  $ f(y) \geqslant f(x) $。我们称 f 是严格单调递增的，当且仅当只要  $ x, y \in X $ 且  $ y > x $，就有  $ f(y) > f(x) $。类似地，我们称 f 是单调递减的，当且仅当只要  $ x, y \in X $ 且  $ y > x $，就有  $ f(y) \leqslant f(x) $，并且称 f 是严格单调递减的，当且仅当只要  $ x, y \in X $ 且  $ y > x $，就有  $ f(y) < f(x) $。

如果 f 是单调递增的或单调递减的，那么我们称 f 是单调的。如果 f 是严格单调递增的或严格单调递减的，那么我们称 f 是严格单调的。

例 9.8.2 如果把函数  $ f(x) := x^2 $ 限制到区域  $ [0, +\infty) $ 上，那么此时  $ f $ 就是严格单调递增的。（为什么？）但当把  $ f $ 限制到区域  $ (-\infty, 0] $ 上时，它就是严格单调递减的。（为什么？）所以该函数在  $ (-\infty, 0] $ 和  $ [0, +\infty) $ 上都是严格单调的，但它在整个实直线  $ (-\infty, +\infty) $ 上却不是严格单调的（或者不是单调的）。注意，如果一个函数在某个区域  $ X $ 上是严格单调的，那么它在同一个区域  $ X $ 上也是单调的。如果把常数函数  $ f(x) := 6 $ 限制到任何一个区域  $ X \subseteq \mathbb{R} $ 上，那么它既是单调递增的又是单调递减的，但它不是严格单调的。（除非  $ X $ 由至多一个点构成，为什么？）

连续函数不一定是单调的（例如，考虑 R 上的函数  $ f(x)=x^{2} $），而且单调函数也不一定是连续的。例如，考虑前面定义的函数  $ f:[-1,1]\to\mathbf{R} $

 $$ f(x):=\begin{cases}-1&\quad 若 x\leqslant0\\1&\quad 若 x>0\end{cases} $$ 

单调函数遵守最大值原理（习题 9.8.1），但不遵守中值定理（习题 9.8.2）。另外，一个单调函数可能存在非常多的间断点（习题 9.8.5）。

如果一个函数既是严格单调的，又是连续的，那么它就有许多好的性质。特别地，它是可逆的。

命题 9.8.3 设 a < b 都是实数，并且设  $ f: [a, b] \to \mathbb{R} $ 是一个既连续又严格单调递增的函数，那么 f 是从  $ [a, b] $ 到  $ [f(a), f(b)] $ 的双射，并且它的反函数  $ f^{-1}: [f(a), f(b)] \to [a, b] $ 也是既连续又严格单调递增的。

证明：参见习题 9.8.4。

关于严格单调递减的函数也有类似的命题，参见习题 9.8.4。

例 9.8.4 设 n 是一个正整数且 R > 0，因为函数  $ f(x) := x^{n} $ 在区间  $ [0, R] $ 上是严格递增的，所以根据命题 9.8.3 可知，该函数是从  $ [0, R] $ 到  $ [0, R^{n}] $ 的双射，从而存在一个从  $ [0, R^{n}] $ 到  $ [0, R] $ 的反函数。这可以给出构造数  $ x \in [0, R^{n}] $ 的 n 次方根  $ x^{1/n} $ 的另一种方法，从而代替了我们在引理 5.6.5 中所做的工作。

9.8.5 本题中我们给出这样一个函数的例子：它在每一个有理数点处都是间断的，但在每个无理数点处都是连续的。因为有理数集是可数的，所以记  $ Q = \{q(0), q(1), q(2), \cdots\} $，其中  $ q: \mathbf{N} \to \mathbf{Q} $ 是从  $ \mathbf{N} $ 到  $ Q $ 的双射。现在定义函数  $ g: \mathbf{Q} \to \mathbf{R} $ 为  $ g(q(n)) := 2^{-n} $，其中  $ n $ 是任意的自然数。于是  $ g $ 把  $ q(0) $ 映射到 1，把  $ q(1) $ 映射到  $ 2^{-1} $，等等。因为  $ \sum_{n=0}^{\infty} 2^{-n} $ 是绝对收敛的，所以  $ \sum_{r \in \mathbf{Q}} g(r) $ 也是绝对收敛的。现在定义函数  $ f: \mathbf{R} \to \mathbf{R} $ 为

## 9.9 一致连续性

我们知道，在闭区间  $ [a, b] $ 上的连续函数是有界的（根据最大值原理可知，该函数实际上达到了它的最大值和最小值）。但是，如果我们把闭区间换成开区间，那么连续函数就不一定是有界的。例如，定义为  $ f(x) := 1/x $ 的函数  $ f: (0, 2) \to \mathbf{R} $。这个函数在  $ (0, 2) $ 的每一点处都是连续的，从而它在  $ (0, 2) $ 上是连续的，但它在  $ (0, 2) $ 上是无界的。通俗地说，这里的问题在于，尽管函数的确在开区间  $ (0, 2) $ 的每一点处都连续，但当自变量不断靠近端点 0 时，它就变得“越来越不”连续。

根据连续性的“ $ \varepsilon-\delta $”定义，即命题 9.4.7(c)，我们来进一步分析这种现象。我们知道，如果  $ f: X \to \mathbb{R} $ 在点  $ x_0 $ 处是连续的，那么对于任意的  $ \varepsilon > 0 $ 总存在一个  $ \delta $ 使得只要  $ x \in X $ 是  $ \delta $-接近于  $ x_0 $ 的， $ f(x) $ 就是  $ \varepsilon $-接近于  $ f(x_0) $ 的。换言之，如果

我们能够保证 x 是充分接近于  $ x_{0} $ 的，那么我们就可以迫使  $ f(x) \varepsilon $-接近于  $ f(x_{0}) $。在每一个点  $ x_{0} $ 附近都存在一个“稳定岛”  $ (x_{0}-\delta, x_{0}+\delta) $ 使得函数  $ f(x) $ 偏离  $ f(x_{0}) $ 的范围不超过  $ \varepsilon $，这是思考这一点的方法之一。

例 9.9.1 考虑上文提到的函数  $ f(x) = 1/x $，并且令  $ x_{0} = 1 $。为了保证  $ f(x) $ 是 0.1-接近于  $ f(x_{0}) $ 的，我们只需要令 x 是 1/11-接近于  $ x_{0} $ 的，因为如果 x 是 1/11-接近于  $ x_{0} $ 的，那么 10/11 < x < 12/11，从而 11/12 < f(x) < 11/10，所以  $ f(x) $ 是 0.1-接近于  $ f(x_{0}) $ 的。因此在点  $ x_{0} = 1 $ 处，使得  $ f(x) $ 是 0.1-接近于  $ f(x_{0}) $ 的 “ $ \delta $” 大概取值在 1/11 附近。

现在我们考虑  $ x_{0}=0.1 $ 的情形。函数  $ f(x)=1/x $ 在该点处仍是连续的，但是我们应看到此处的连续性变弱了。为了保证  $ f(x) $ 是 0.1-接近于  $ f(x_{0}) $ 的，我们需要让 x 是 1/1010-接近于  $ x_{0} $ 的。事实上，如果 x 是 1/1010-接近于  $ x_{0} $ 的，那么 10/101 < x < 102/1010，从而 9.901 < f(x) < 10.1，于是  $ f(x) $ 是 0.1-接近于  $ f(x_{0}) $ 的。因此，对于同一个  $ \varepsilon $，我们需要一个更小的 “ $ \delta $”，即  $ f(x) $ 在 0.1 附近比在 1 附近更 “不稳定”，也就是说在 0.1 附近的 “稳定岛” 比在 1 附近的 “稳定岛” 要更小（如果我们要保持  $ f(x) $ 是 0.1-稳定的话）。

另外，还存在其他的连续函数并不具备上述的性质。考虑定义为  $ g(x) := 2x $ 的函数  $ g: (0, 2) \to \mathbb{R} $。就像前面那样，我们固定  $ \varepsilon = 0.1 $ 并考察  $ x_0 = 1 $ 附近的稳定岛。显然，如果  $ x $ 是 0.05-接近于  $ x_0 $ 的，那么  $ g(x) $ 就是 0.1-接近于  $ g(x_0) $ 的。此时，在  $ x_0 = 1 $ 处我们可以取  $ \delta $ 为 0.05。如果我们变动  $ x_0 $，即如果令  $ x_0 $ 为 0.1，那么  $ \delta $ 的取值不发生变动。尽管  $ x_0 $ 由 1 变为 0.1，但只要  $ x $ 是 0.05-接近于  $ x_0 $ 的， $ g(x) $ 就保持 0.1-接近于  $ g(x_0) $。事实上，对于任意的  $ x_0 $，上述  $ \delta $ 的取值均适用。在这种情况下，我们说函数  $ g $ 是一致连续的。更准确的说法如下。

定义 9.9.2（一致连续）设 X 是 R 的一个子集，并且设  $ f: X \to \mathbb{R} $ 是一个函数。我们称 f 是一致连续的，如果对于任意的  $ \varepsilon > 0 $，都存在一个  $ \delta > 0 $ 使得只要  $ x, x_0 \in X $ 是 X 中  $ \delta $-接近的两个点， $ f(x) $ 和  $ f(x_0) $ 就是  $ \varepsilon $-接近的。

注 9.9.3 应该把这个定义与连续的概念进行对比。从命题9.4.7(c)中可知，一个函数 f 是连续的，指的是对于任意的  $ \varepsilon > 0 $ 和任意的  $ x_0 \in X $，存在一个  $ \delta > 0 $ 使得只要  $ x \in X $ 是  $ \delta $-接近于  $ x_0 $ 的， $ f(x) $ 和  $ f(x_0) $ 就是  $ \varepsilon $-接近的。一致连续和连续之间的区别在于，在一致连续中我们可以取到单独一个  $ \delta $ 使得这个  $ \delta $ 对所有的  $ x_0 \in X $ 均适用；而对于一般的连续，不同的  $ x_0 \in X $ 可能使用了不同的  $ \delta $。因此，每一个一致连续的函数都是连续的，反之不成立。

例 9.9.4（非正式的）定义为  $ f(x) = 1/x $ 的函数  $ f: (0,2) \to \mathbb{R} $ 在  $ (0,2) $ 上是连续的，但不是一致连续的，因为当  $ x \to 0 $ 时，连续性（或者更准确地说， $ \delta $ 对  $ \varepsilon $ 的依赖性）变得越来越弱（在例 9.9.10 中，我们将更准确地阐述这一点）。

回忆一下，附着点的概念和连续函数的概念都有若干种等价的描述，而且两者都存在“ $ \varepsilon-\delta $”类型的描述（其中包含了 $ \varepsilon $-接近性的概念）以及“序列的”描述（其中包含了序列的收敛性），参见引理9.1.14和命题9.3.9。类似地，一致连续的概念也可以用序列来描述，这里要用到等价序列的概念（参见定义5.2.6，但现在我们把有理数序列推广到实数序列，而且不再要求序列是柯西序列）。

定义 9.9.5（等价序列）设 m 是一个整数， $ (a_{n})_{\substack{n=m}} $ 和  $ (b_{n})_{\substack{n=m}}^{\infty} $ 是两个实数序列，并且设  $ \varepsilon > 0 $ 是给定的。 $ (a_{n})_{\substack{n=m}}^{\infty} $ 是  $ \varepsilon $-接近于  $ (b_{n})_{\substack{n=m}}^{\infty} $ 的，当且仅当对于任意的  $ n \geqslant m $， $ a_{n} $ 都是  $ \varepsilon $-接近于  $ b_{n} $ 的。我们称  $ (a_{n})_{\substack{n=m}}^{\infty} $ 是最终  $ \varepsilon $-接近于  $ (b_{n})_{\substack{n=m}}^{\infty} $ 的，当且仅当存在一个  $ N \geqslant m $ 使得序列  $ (a_{n})_{\substack{n=N}}^{\infty} $ 和  $ (b_{n})_{\substack{n=N}}^{\infty} $ 是  $ \varepsilon $-接近的。两个序列  $ (a_{n})_{\substack{n=m}}^{\infty} $ 和  $ (b_{n})_{\substack{n=m}}^{\infty} $ 是等价的，当且仅当对于任意的  $ \varepsilon > 0 $，序列  $ (a_{n})_{\substack{n=m}}^{\infty} $ 和  $ (b_{n})_{\substack{n=m}}^{\infty} $ 都是最终  $ \varepsilon $-接近的。

注 9.9.6 或许有人会问，是否应该假设  $ \varepsilon $ 是有理数或实数，但对命题6.1.4稍作修改就可以证明附加上这种假设后的定义与上述定义没有什么不同。

利用极限的语言可以更加简洁地表述等价的概念。

引理 9.9.7 设  $ (a_{n})_{n=1}^{\infty} $ 和  $ (b_{n})_{n=1}^{\infty} $ 都是实数序列（不必是有界的或收敛的），那么  $ (a_{n})_{n=1}^{\infty} $ 和  $ (b_{n})_{n=1}^{\infty} $ 是等价的当且仅当  $ \lim_{n\to\infty}(a_{n}-b_{n})=0 $。

证明：参见习题 9.9.1。

同时，一致连续的概念可以用等价序列来描述。

命题 9.9.8 设 X 是 R 的一个子集，并且设  $ f: X \to R $ 是一个函数，那么下述两个命题在逻辑上是等价的：

(a) f 在 X 上是一致连续的。

(b) 如果  $ (x_{n})_{n=0}^{\infty} $ 和  $ (y_{n})_{n=0}^{\infty} $ 是由 X 中元素构成的两个等价序列，那么序列  $ (f(x_{n}))_{n=0}^{\infty} $ 和  $ (f(y_{n}))_{n=0}^{\infty} $ 也是等价的。

证明：参见习题 9.9.2。

注 9.9.9 读者应该把这个结论与命题 9.3.9 进行对比。命题 9.3.9 断定了如果 f 是连续的，那么 f 就把收敛序列映射成收敛序列。相比之下，命题 9.9.8 断定了如果 f 是一致连续的，那么 f 就把等价的序列映射成等价的序列。为了找出两个命题之间是如何联系的，根据引理 9.9.7 可知， $ (x_{n})_{n=0}^{\infty} $ 收敛于  $ x_{*} $，当且仅当序列  $ (x_{n})_{n=0}^{\infty} $ 和  $ (x_{*})_{n=0}^{\infty} $ 是等价的。

例 9.9.10 考虑定义为  $ f(x) := 1/x $ 的函数  $ f:(0,2) \to \mathbb{R} $。根据引理 9.9.7 可知，序列  $ (1/n)_{n=1}^{\infty} $ 和  $ (1/2n)_{n=1}^{\infty} $ 是  $ (0,2) $ 中的等价序列。但是序列  $ f(1/n)_{n=1}^{\infty} $ 和  $ (f(1/2n))_{n=1}^{\infty} $ 不是等价的。（为什么？再次使用引理 9.9.7。）所以根据命题 9.9.8 可

知，f 不是一致连续的。(这些序列从 1 开始而不是从 0 开始，但读者能够容易看出这并不影响上述讨论。)

例 9.9.11 考虑定义为  $ f(x) := x^{2} $ 的函数  $ f: \mathbf{R} \to \mathbf{R} $。这是  $ \mathbf{R} $ 上的连续函数，但它不是一致连续的；当自变量趋向于无穷大时，连续性在某种意义上变得“越来越弱”。可以通过命题 9.9.8 来定量分析它。考虑序列  $ (n)_{n=1}^{\infty} $ 和  $ (n + \frac{1}{n})_{n=1}^{\infty} $，根据引理 9.9.7 可知，这两个序列是等价的。但序列  $ (f(n))_{n=1}^{\infty} $ 和  $ (f(n + \frac{1}{n}))_{n=1}^{\infty} $ 不等价，因为  $ f(n + \frac{1}{n}) = n^{2} + 2 + \frac{1}{n^{2}} = f(n) + 2 + \frac{1}{n^{2}} $ 不是最终 2-接近于  $ f(n) $ 的。所以根据命题 9.9.8 可得， $ f $ 不是一致连续的。

一致连续函数的另一个性质是把柯西序列映射成柯西序列。

命题 9.9.12 设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 是一致连续的函数，并且设  $ (x_{n})_{n=0}^{\infty} $ 是完全由 X 中的元素构成的柯西序列，那么  $ (f(x_{n}))_{n=0}^{\infty} $ 也是一个柯西序列。

证明：参见习题 9.9.3。

例 9.9.13 我们再次证明定义为  $ f(x) := 1/x $ 的函数  $ f: (0, 2) \to \mathbb{R} $ 不是一致连续的。序列  $ (1/n)_{n=1}^{\infty} $ 是  $ (0, 2) $ 中的柯西序列，但是序列  $ (f(1/n))_{n=1}^{\infty} $ 不是柯西序列。（为什么？）所以由命题 9.9.12 可知， $ f $ 不是一致连续的。

推论 9.9.14 设 X 是 R 的一个子集， $ f: X \to \mathbb{R} $ 是一致连续的函数，并且设  $ x_0 $ 是 X 的附着点，那么极限  $ \lim_{x \to x_0 \to x} f(x) $ 存在（特别地，它是一个实数）。

证明：参见习题 9.9.4。

现在我们证明一致连续的函数把有界集映射成有界集。

命题 9.9.15 设 X 是 R 的一个子集，并且设  $ f: X \to R $ 是一致连续的函数。如果 E 是 X 的一个有界子集，那么  $ f(E) $ 也是有界的。

证明：参见习题 9.9.5。

就像我们刚才反复看到的那样，并非所有的连续函数都是一致连续的。但是，如果函数的定义域是一个闭区间，那么连续函数实际上就是一致连续函数。

定理 9.9.16 设  $ a < b $ 都是实数，并且设  $ f: [a, b] \to \mathbb{R} $ 是  $ [a, b] $ 上的连续函数，那么  $ f $ 也是一致连续的。

证明：利用反证法，假设 f 不是一致连续的，根据命题 9.9.8 可知，在  $ [a,b] $ 中一定存在两个等价的序列  $ (x_{n})_{n=0}^{\infty} $ 和  $ (y_{n})_{n=0}^{\infty} $ 使得序列  $ (f(x_{n}))_{n=0}^{\infty} $ 和  $ (f(y_{n}))_{n=0}^{\infty} $ 不等价。特别地，我们可以找到一个  $ \varepsilon>0 $ 使得  $ (f(x_{n}))_{n=0}^{\infty} $ 和  $ (f(y_{n}))_{n=0}^{\infty} $ 不是最终  $ \varepsilon $-接近的。

将  $ \varepsilon $ 固定不变，并且设 E 是集合

 $$ E:=\{n\in\mathbf{N}:f(x_{n}) 和 f(y_{n}) 不是 \varepsilon- 接近的 \} $$ 

E 一定是无限的，因为如果 E 是有限的，那么  $ (f(x_{n}))_{n=0}^{\infty} $ 和  $ (f(y_{n}))_{n=0}^{\infty} $ 将是最终  $ \varepsilon $-接近的。（为什么？）根据命题 8.1.5 可知，E 是可数集。事实上，从命题 8.1.5 的证明过程中可以看出，我们能够找到一个完全由 E 中的元素构成的无限序列

 $$ n_{0}<n_{1}<n_{2}<\cdots $$ 

特别地，

 $$ \left|f(x_{n_{j}})-f(y_{n_{j}})\right|>\varepsilon 对所有的 j\in\mathbf{N} 均成立 $$ 

另外，序列  $ (x_{n_{j}})_{j=0}^{\infty} $ 是  $ [a,b] $ 中的序列，从而由海涅-博雷尔定理（定理 9.1.24）可知，它必定有一个收敛于某个极限  $ L\in[a,b] $ 的子序列  $ (x_{n_{j_{k}}})_{k=0}^{\infty} $。特别地，f 在 L 处是连续的，从而根据命题 9.4.7 可知

 $$ \lim_{k\to\infty}f(x_{n_{j_{k}}})=f(L) $$ 

注意，由引理 6.6.4 可知， $ (x_{n_{j_{k}}})_{k=0}^{\infty} $ 是  $ (x_{n})_{n=0}^{\infty} $ 的一个子序列，并且  $ (y_{n_{j_{k}}})_{k=0}^{\infty} $ 是  $ (y_{n})_{n=0}^{\infty} $ 的一个子序列。另外，根据引理 9.9.7 可得，

 $$ \lim_{n\to\infty}\left(x_{n}-y_{n}\right)=0 $$ 

所以利用命题 6.6.5 可得，

 $$ \lim_{k\to\infty}(x_{n_{j_{k}}}-y_{n_{j_{k}}})=0 $$ 

因为当  $ k \to \infty $ 时， $ x_{n_{jk}} $ 收敛于 L，所以根据极限定律，有

 $$ \lim_{k\to\infty}y_{n_{j_{k}}}=L $$ 

从而根据 f 在 L 处的连续性可得，

 $$ \lim_{k\to\infty}f(y_{n_{j_{k}}})=f(L) $$ 

利用极限定律，从式 $  (9.4)  $中减去上面这个式子就得到

 $$ \lim_{k\to\infty}(f(x_{n_{j_{k}}})-f(y_{n_{j_{k}}}))=0 $$ 

但这与式 $  (9.3)  $矛盾。(为什么?)由这个矛盾可以得出f实际上是一致连续的。☐

## 9.10 在无限处的极限

到目前为止，我们已经讨论了如果  $ x_{0} $ 是实数，当  $ x \to x_{0} $ 时，函数  $ f: X \to \mathbb{R} $ 有极限是什么意思。现在我们简单地讨论一下，当  $ x_{0} $ 等于  $ +\infty $ 或  $ -\infty $ 时取极限的意思（这是拓扑空间上关于连续函数的更一般理论中的一部分，参见 13.5 节）。

首先，我们需要一个概念，它描述了  $ +\infty $ 或  $ -\infty $ 附着于一个集合是什么意思。

定义 9.10.1（无限附着点）设 X 是 R 的一个子集，+ $ \infty $ 是际附着于 X 的，当且仅当对于任意的  $ M \in \mathbb{R} $ 都存在一个  $ x \in X $ 使得  $ x > M $。- $ \infty $ 是附着于 X 的，当且仅当对于任意的  $ M \in \mathbb{R} $ 都存在一个  $ x \in X $ 使得 x < M。

换言之，+ $ \infty $ 是附着于 X 的，当且仅当 X 没有上界，即当且仅当  $ \sup(X)=+\infty $。类似地，- $ \infty $ 是附着于 X 的，当且仅当 X 没有下界，或者当且仅当  $ \inf(X)=-\infty $。所以，一个集合是有界的，当且仅当 + $ \infty $ 和 - $ \infty $ 都不是它的附着点。

注 9.10.2 这个定义看起来与定义 9.1.8 非常不同，但是利用广义实直线  $ R^{*} $ 的拓扑结构可以把它们统一起来，对此我们在这里不展开讨论。

定义 9.10.3（在无限处的极限）设 X 是 R 的一个子集，并且 +∞ 是 X 的一个附着点，设  $ f: X \to \mathbb{R} $ 是一个函数。我们称当  $ x \to +\infty $ 时， $ f(x) $ 沿着 X 收敛于 L，并记作  $ \lim_{x \to +\infty; x \in X} f(x) = L $，当且仅当对于任意的  $ \varepsilon > 0 $ 都存在一个 M 使得 f 在  $ X \cap (M, +\infty) $ 上是  $ \varepsilon $-接近于 L 的（即对所有满足  $ x > M $ 的  $ x \in X $，均有  $ |f(x) - L| \leqslant \varepsilon $。类似地，我们称当  $ x \to -\infty $ 时， $ f(x) $ 收敛于 L，当且仅当对于任意的  $ \varepsilon > 0 $ 都存在一个 M 使得 f 在  $ X \cap (-\infty, M) $ 上是  $ \varepsilon $-接近于 L 的。

例 9.10.4 设  $ f: (0, +\infty) \to \mathbb{R} $ 是函数  $ f(x) := 1/x $，则  $ \lim_{x \to +\infty} \frac{1}{x \in (0, +\infty)} = 0 $。(你能根据定义看出为什么吗？)

我们可以使用这些在无限处的极限做很多事情，就像我们使用  $ x_{0} $ 处的极限所做的那样。例如，所有的极限定律仍然成立。然而，由于在本书中不常使用这些

极限，所以我们不会把过多的精力放在这上面。但要注意，这个定义与序列极限  $ \lim_{n\to\infty}a_{n} $ 的概念是一致的（习题 9.10.1）。

 $$ \lim_{n\to+\infty;n\in\mathbf{N}}a_{n}=\lim_{n\to\infty}a_{n} $$
