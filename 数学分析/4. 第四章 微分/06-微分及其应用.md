## 2. 背景与应用场景

在前五份文件中，我们建立了完整的求导体系——从定义出发计算导数，再到四则运算法则、链式法则、隐函数与参数方程求导，以及高阶导数。导数 $f'(x_0)$ 告诉我们函数在 $x_0$ 处的**变化率**——但变化率本身并不直接回答一个更实际的工程问题：

**当自变量改变一个微小量时，函数值大约改变了多少？**

考虑以下场景：

- **工程中的误差估计**：用卡尺测量一个球的直径 $D$，得到 $D = 10\text{ cm}$，测量误差为 $\pm 0.1\text{ cm}$。球的体积 $V = \frac{1}{6}\pi D^3$。那么体积的误差是多少？我们不需要精确计算 $V(10.1) - V(10)$，而是通过**微分**快速估算：$dV = V'(10)\cdot dD = \frac{1}{2}\pi D^2\cdot dD = 50\pi \cdot 0.1 = 5\pi\text{ cm}^3$。

- **物理中的线性近似**：单摆的周期公式 $T = 2\pi\sqrt{L/g}$。若摆长 $L$ 有微小变化 $dL$，周期的变化 $dT = T'(L)\,dL = \frac{\pi}{\sqrt{gL}}\,dL$。

- **数值计算中的近似**：计算 $\sqrt{4.1}$ 时，不必开平方，而是利用 $\sqrt{4}=2$ 和导数 $(\sqrt{x})'|_{x=4} = 1/4$ 得到近似值 $2.025$。

所有这些问题的数学核心是：**将函数增量 $\Delta y$ 近似为 $f'(x_0)\Delta x$——即函数的微分 $dy$**。$dy$ 是 $\Delta y$ 的**线性主部**，二者的差是高阶无穷小。这个看似简单的思想，是整个微分学的基石：它将非线性问题（$f(x)$ 的真实变化）在局部近似为线性问题（$dy$ 的变化），而线性问题是可解的。

从知识体系看，微分 $dy = f'(x)dx$ 不仅统一了 Leibniz 记号 $\frac{dy}{dx}$ 的真正含义（微分之商），更在后续的积分换元法、常微分方程中扮演核心角色。本节的内容连接了"求导"和"积分"两大分支。

## 3. 核心概念与符号约定

### 3.1 符号表

| 符号 | 含义 | 示例（$f(x)=x^2$ 在 $x=2$ 处，$\Delta x=0.1$） |
|------|------|------|
| $\Delta x$ | 自变量的增量 | $\Delta x=0.1$ |
| $\Delta y$ | 函数的增量 $f(x_0+\Delta x)-f(x_0)$ | $\Delta y = (2.1)^2-4=0.41$ |
| $dy$ | 函数在 $x_0$ 处的微分 $f'(x_0)\Delta x$ | $dy=4\times 0.1 = 0.4$ |
| $dx$ | 自变量的微分，约定 $dx=\Delta x$ | $dx=0.1$ |
| $o(\Delta x)$ | 比 $\Delta x$ 高阶的无穷小 | $\Delta y - dy = 0.01 = o(\Delta x)$ |
| $d^2y$ | 二阶微分 $d(dy)=f''(x)dx^2$ | $d^2(x^2)=2\,dx^2$ |
| $d^ny$ | $n$ 阶微分 $f^{(n)}(x)dx^n$ | $d^n(x^m)=m(m-1)\cdots(m-n+1)x^{m-n}dx^n$ |

### 3.2 微分概念的引入——函数增量的分解

设 $y = f(x)$ 在 $x_0$ 处可导。考虑 $x$ 从 $x_0$ 变化到 $x_0+\Delta x$ 时函数的增量：

$$\Delta y = f(x_0+\Delta x) - f(x_0)$$

由可导的定义（文件 01 定义 4.1）：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x) - f(x_0)}{\Delta x} = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x}$$

由极限与无穷小的关系（第三章），这等价于存在一个当 $\Delta x\to0$ 时趋于 $0$ 的量 $\alpha(\Delta x)$，使得：

$$\frac{\Delta y}{\Delta x} = f'(x_0) + \alpha(\Delta x), \quad \text{其中 } \lim_{\Delta x\to0} \alpha(\Delta x) = 0$$

两边乘以 $\Delta x$：

$$\Delta y = f'(x_0)\Delta x + \alpha(\Delta x)\Delta x$$

令 $\beta(\Delta x) = \alpha(\Delta x)\Delta x$，则 $\beta(\Delta x)$ 满足：

$$\lim_{\Delta x \to 0} \frac{\beta(\Delta x)}{\Delta x} = \lim_{\Delta x \to 0} \alpha(\Delta x) = 0$$

这正是 $o(\Delta x)$ 的定义——$\beta(\Delta x)$ 是比 $\Delta x$ 更高阶的无穷小。因此：

$$\boxed{\Delta y = f'(x_0)\Delta x + o(\Delta x)}$$

**核心解读**：这个分解式是微分概念的基石。

- **$f'(x_0)\Delta x$**：称为 $\Delta y$ 的**线性主部**（linear main part）。它是 $\Delta x$ 的线性函数，当 $\Delta x$ 很小时，这一项是 $\Delta y$ 中起主导作用的部分。
- **$o(\Delta x)$**：称为**高阶无穷小**。当 $\Delta x \to 0$ 时，这项比 $\Delta x$ 更快地趋于零。在近似计算中可以忽略。
- 所谓"线性"：指 $f'(x_0)\Delta x$ 是 $\Delta x$ 的一次函数（形如 $k\Delta x$）。
- 所谓"主部"：因为 $f'(x_0) \neq 0$ 时，$\lim_{\Delta x\to0} \frac{f'(x_0)\Delta x}{\Delta y} = 1$，即线性主部在 $\Delta y$ 中占主导。

### 3.3 微分的严格定义

**定义 4.18（微分）**：设函数 $y = f(x)$ 在点 $x_0$ 的某邻域内有定义。若存在常数 $A$（与 $x_0$ 有关但与 $\Delta x$ 无关），使得当 $\Delta x \to 0$ 时：

$$\Delta y = A \cdot \Delta x + o(\Delta x)$$

则称 $f$ 在 $x_0$ 处**可微**（differentiable），并称 $A\cdot\Delta x$ 为 $f$ 在 $x_0$ 处的**微分**（differential），记作：

$$dy\big|_{x=x_0} \quad \text{或} \quad df(x_0)$$

即 $dy = A\cdot\Delta x$。

**定义的核心**：可微性要求 $\Delta y$ 可以分解为一个"$\Delta x$ 的线性函数"加上一个"比 $\Delta x$ 高阶的无穷小"。这是"用线性逼近非线性"思想的严格表述。

### 3.4 可导与可微的等价性

**定理 4.19（可导 ⇔ 可微）**：函数 $y = f(x)$ 在 $x_0$ 处可微当且仅当 $f$ 在 $x_0$ 处可导。且当可微时，分解式中的常数 $A$ 恰为 $f'(x_0)$，即：

$$dy = f'(x_0)\,\Delta x$$

**证明**：需要证明两个方向。

**($\Rightarrow$) 可微 ⇒ 可导**：

设 $f$ 在 $x_0$ 处可微，则存在常数 $A$ 使得 $\Delta y = A\Delta x + o(\Delta x)$。

当 $\Delta x \neq 0$ 时，两边除以 $\Delta x$：

$$\frac{\Delta y}{\Delta x} = A + \frac{o(\Delta x)}{\Delta x}$$

取极限 $\Delta x \to 0$。由于 $\lim_{\Delta x\to0} \frac{o(\Delta x)}{\Delta x} = 0$（高阶无穷小的定义），由极限加法法则：

$$\lim_{\Delta x\to0} \frac{\Delta y}{\Delta x} = A + 0 = A$$

因此 $\lim_{\Delta x\to0} \frac{\Delta y}{\Delta x}$ 存在且等于 $A$。按导数定义，$f$ 在 $x_0$ 处可导，且 $f'(x_0) = A$。

**($\Leftarrow$) 可导 ⇒ 可微**：

设 $f$ 在 $x_0$ 处可导，即 $f'(x_0) = \lim_{\Delta x\to0} \frac{\Delta y}{\Delta x}$ 存在。

由极限与无穷小的关系（第三章结论），存在函数 $\alpha(\Delta x)$ 满足 $\lim_{\Delta x\to0} \alpha(\Delta x) = 0$，使得：

$$\frac{\Delta y}{\Delta x} = f'(x_0) + \alpha(\Delta x)$$

两边乘以 $\Delta x$：

$$\Delta y = f'(x_0)\Delta x + \alpha(\Delta x)\Delta x$$

令 $\beta(\Delta x) = \alpha(\Delta x)\Delta x$，则：

$$\lim_{\Delta x\to0} \frac{\beta(\Delta x)}{\Delta x} = \lim_{\Delta x\to0} \alpha(\Delta x) = 0$$

即 $\beta(\Delta x) = o(\Delta x)$。因此 $\Delta y = f'(x_0)\Delta x + o(\Delta x)$，满足可微定义，$A = f'(x_0)$。证毕。

**推论**：由定理 4.19，可微与可导是同一概念的两个等价刻画——可导强调**变化率**的存在，可微强调**线性逼近**的存在。

**记号约定**：定义自变量 $x$ 的微分 $dx$ 等于其增量：$dx = \Delta x$。于是函数 $y = f(x)$ 的微分可简洁写作：

$$\boxed{dy = f'(x)\,dx}$$

这立刻导出了导数的一种新的理解方式：

$$\frac{dy}{dx} = f'(x)$$

**从整体记号到分数**：在文件 01 §3.2 中，我们特别强调 $\frac{dy}{dx}$ 是一个整体记号，不是分数 $dy \div dx$。现在，在引入微分之后，这个记号获得了新的含义——**$dy$ 与 $dx$ 之商**。这不是对之前说法的否定，而是微分概念成熟之后的新认识。Leibniz 的天才之处正在于，他选择了这个最终可以被"兑现"为分数的记号。

### 3.5 微分的几何意义

设曲线 $y = f(x)$ 在点 $M_0(x_0, f(x_0))$ 处的切线为 $MT$，切线斜率为 $f'(x_0)$。考虑 $x$ 从 $x_0$ 变化到 $x_0+\Delta x$：

- **$\Delta y$** = $f(x_0+\Delta x) - f(x_0)$：曲线上纵坐标的实际改变量。
- **$dy$** = $f'(x_0)\Delta x$：切线上纵坐标的改变量。

几何图示（文字描述）：

```
y
^
|         / 
|        /  曲线 y = f(x)
|       /   
|  f(x0+Δx) ----• Q (曲线上)
|      |       / 
|  f(x0) ----• M0 ———• T (切线上)
|      |  dy  |
|      |←dx→|
|      |←—— Δy ——→|
+----------------------> x
     x0  x0+Δx
```

其中：
- $M_0 = (x_0, f(x_0))$ 是切点
- $Q = (x_0+\Delta x, f(x_0+\Delta x))$ 是曲线上的邻近点
- $T = (x_0+\Delta x, f(x_0) + f'(x_0)\Delta x)$ 是切线上对应 $x_0+\Delta x$ 的点

**几何关系**：
- $\Delta y$ = 线段 $M_0Q$ 的纵坐标变化 = $f(x_0+\Delta x) - f(x_0)$
- $dy$ = 线段 $M_0T$ 的纵坐标变化 = $f'(x_0)\Delta x$
- $\Delta y - dy$ = 线段 $TQ$ 的长度 = $o(\Delta x)$（高阶无穷小）

**$dy \neq \Delta y$ 的原因**：曲线有弯曲（曲率非零），而切线是直的。只有当 $f$ 是线性函数（$f(x) = ax+b$，图像为直线）时，$dy = \Delta y$ 严格成立。

**几何直觉的含义**：当 $\Delta x$ 很小时，切线在切点附近"紧贴"曲线，因此切线上纵坐标的变化 $dy$ 可以近似代替曲线上纵坐标的变化 $\Delta y$。这就是线性逼近的几何基础。

## 4. 原理与方法

### 4.1 微分的四则运算法则

由 $dy = f'(x)dx$ 可知，求微分本质上就是**先求导，再乘以 $dx$**。因此，求导的四则运算法则可以立即转化为微分运算法则。

**定理 4.20（微分运算法则）**：设 $u = u(x)$ 和 $v = v(x)$ 均为 $x$ 的可导函数，$c$ 为常数，则：

1. **常数法则**：$d(c) = 0$
2. **和差法则**：$d(u \pm v) = du \pm dv$
3. **乘积法则**：$d(uv) = v\,du + u\,dv$
4. **商法则**：$d\!\left(\dfrac{u}{v}\right) = \dfrac{v\,du - u\,dv}{v^2}$（$v \neq 0$）

**证明**：每条法则从导数对应法则出发，两边乘以 $dx$ 即得。

**法则 1**（常数法则）：

$$\frac{d}{dx}(c) = 0 \quad \Rightarrow \quad d(c) = 0 \cdot dx = 0$$

常数的微分为零——常数不随自变量变化。

**法则 2**（和差法则）：

由求导的和差法则 $(u \pm v)' = u' \pm v'$，两边乘以 $dx$：

$$d(u \pm v) = (u' \pm v')dx = u'\,dx \pm v'\,dx = du \pm dv$$

**法则 3**（乘积法则）：

由求导的乘积法则 $(uv)' = u'v + uv'$，两边乘以 $dx$：

$$d(uv) = (u'v + uv')dx = v \cdot u'\,dx + u \cdot v'\,dx = v\,du + u\,dv$$

**与导数乘积法则的结构一致性**：导数形式是"轮流求导再相加"；微分形式是"轮流求微分再与另一函数交叉相乘"。

**法则 4**（商法则）：

由求导的商法则 $\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}$，两边乘以 $dx$：

$$
\begin{aligned}
d\!\left(\frac{u}{v}\right) &= \frac{u'v - uv'}{v^2}\,dx \\
&= \frac{v \cdot u'\,dx - u \cdot v'\,dx}{v^2} \\
&= \frac{v\,du - u\,dv}{v^2}
\end{aligned}
$$

**对应关系表**：

| 导数法则 | 微分法则 | 联系 |
|----------|----------|------|
| $\frac{d}{dx}(c)=0$ | $d(c)=0$ | $d(c)=c'dx=0\cdot dx=0$ |
| $\frac{d}{dx}(u\pm v)=u'\pm v'$ | $d(u\pm v)=du\pm dv$ | $d(u\pm v)=(u'\pm v')dx$ |
| $\frac{d}{dx}(uv)=u'v+uv'$ | $d(uv)=v\,du+u\,dv$ | $d(uv)=(u'v+uv')dx$ |
| $\frac{d}{dx}\!\left(\frac{u}{v}\right)=\frac{u'v-uv'}{v^2}$ | $d\!\left(\frac{u}{v}\right)=\frac{v\,du-u\,dv}{v^2}$ | $d(u/v)=(u'v-uv')dx/v^2$ |

**何时用微分形式更便捷**：当表达式中含有复合函数，且希望利用一阶微分形式不变性（§4.2）时，微分形式往往比导数形式更直接。

### 4.2 一阶微分形式不变性

**定理 4.21（一阶微分形式不变性）**：设 $y = f(u)$ 可微，则无论 $u$ 是自变量还是关于另一个变量 $x$ 的可微函数（即 $u = g(x)$ 为中间变量），一阶微分 $dy$ 都具有相同的形式：

$$\boxed{dy = f'(u)\,du}$$

**证明**：分两种情形验证。

**情形 1：$u$ 是自变量**。

由微分定义直接得到：$dy = f'(u)\,du$。这是微分的定义式，显然成立。

**情形 2：$u = g(x)$ 是中间变量**（即 $y = f(g(x))$ 是 $x$ 的复合函数）。

由链式法则（文件 03 定理 4.9）：

$$\frac{dy}{dx} = f'(g(x)) \cdot g'(x) = f'(u) \cdot \frac{du}{dx}$$

两边乘以 $dx$：

$$dy = f'(u) \cdot \frac{du}{dx} \cdot dx = f'(u)\,du$$

因此 $dy = f'(u)\,du$ 在 $u$ 为中间变量时仍然成立。✓

**定理的含义**：无论 $u$ 是自变量还是中间变量，一阶微分 $dy$ 总可以写作 $f'(u)$ 乘以 $du$ 的形式。这就是说，**微分形式 $dy = f'(u)du$ 不依赖于 $u$ 的身份**。

**应用策略**：对于复合函数 $y = f(g(x))$，引入中间变量 $u = g(x)$，则：

$$dy = f'(u)\,du = f'(g(x)) \cdot g'(x)\,dx$$

这实际上就是链式法则的微分版本。但形式不变性的优势在于可以**逐层微分**——先对 $y = f(u)$ 求微分得 $dy = f'(u)du$，再对 $u = g(x)$ 求微分得 $du = g'(x)dx$，然后代入。每一步只涉及简单函数的微分，无需一次性写出完整的复合结构。

**"一阶"的含义**：此性质**只对一阶微分成立**。为什么？当 $u$ 是中间变量时，$du = g'(x)dx$，而 $dx$ 是自变量 $x$ 的微分。在计算二阶微分 $d^2y$ 时，我们需要对 $dy$ 再求微分，此时 $du$ 不再被视为"自变量"的微分——$d^2u$ 未必为零。高阶微分的形式不变性不成立（详见 §4.4）。

### 4.3 微分在近似计算中的应用

**线性逼近公式**：由 $\Delta y = f(x_0 + \Delta x) - f(x_0) = f'(x_0)\Delta x + o(\Delta x)$，当 $|\Delta x|$ 充分小时，高阶无穷小 $o(\Delta x)$ 可以忽略，得到：

$$\boxed{f(x_0 + \Delta x) \approx f(x_0) + f'(x_0)\Delta x}$$

等价地，令 $x = x_0 + \Delta x$：

$$f(x) \approx f(x_0) + f'(x_0)(x - x_0)$$

这个公式称为函数在 $x_0$ 处的**线性逼近**（linear approximation）或**一次近似**（first-order approximation）。它的几何意义是：用切线上的函数值近似曲线上的函数值。

**标准化操作流程**：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 第 1 步（选函数与基点） | 构造 $f(x)$ 和基点 $x_0$ | $f(x_0)$ 和 $f'(x_0)$ 应容易计算；$\Delta x = x - x_0$ 应充分小 |
| 第 2 步（求导数值） | 计算 $f'(x_0)$ | 使用已建立的求导法则 |
| 第 3 步（构造近似） | 代入 $f(x_0+\Delta x) \approx f(x_0) + f'(x_0)\Delta x$ | 注意 $\Delta x$ 有正负之分 |

**误差的定性分析**：近似误差为：

$$|f(x_0+\Delta x) - [f(x_0) + f'(x_0)\Delta x]| = |o(\Delta x)|$$

这意味着当 $\Delta x \to 0$ 时，误差趋于零的速度比 $|\Delta x|$ 更快。更具体地，若 $f$ 二阶可导，误差约为 $\frac{1}{2}f''(\xi)(\Delta x)^2$（$\xi$ 在 $x_0$ 与 $x_0+\Delta x$ 之间）——这来自泰勒定理，将在后续章节严格讨论。

**常用的线性近似公式**（在 $x_0 = 0$ 附近，$|x|$ 很小）：

| 函数 | 线性近似 | 推导 |
|------|----------|------|
| $\sqrt{1+x}$ | $1 + \dfrac{1}{2}x$ | $f(x)=\sqrt{1+x}$, $f(0)=1$, $f'(0)=\frac{1}{2}$ |
| $\dfrac{1}{1+x}$ | $1 - x$ | $f(x)=\frac{1}{1+x}$, $f(0)=1$, $f'(0)=-1$ |
| $(1+x)^\alpha$ | $1 + \alpha x$ | $f(x)=(1+x)^\alpha$, $f(0)=1$, $f'(0)=\alpha$ |

这些公式将复杂函数的计算近似为**一次多项式**的计算，是用线性（简单）逼近非线性（复杂）的典型范例。

### 4.4 高阶微分

**定义 4.22（高阶微分）**：设 $y = f(x)$ 在区间上 $n$ 阶可导。$y$ 的**二阶微分**定义为微分 $dy$ 的微分：

$$d^2y = d(dy)$$

一般地，$n$ 阶微分递归定义为：

$$d^ny = d(d^{n-1}y) \quad (n \geq 2)$$

同时约定 $d^1y = dy$。

**定理 4.23（高阶微分公式）**：当 $x$ 是自变量时，对 $n$ 阶可导函数 $y = f(x)$：

$$\boxed{d^ny = f^{(n)}(x)\,dx^n}$$

其中 $dx^n$ 是 $(dx)^n$ 的简写，即自变量微分的 $n$ 次幂。

**证明**（对 $n$ 进行归纳）：

**归纳基础（$n=1$）**：由微分定义，$dy = f'(x)dx$，即为 $d^1y = f^{(1)}(x)dx^1$。✓

**归纳假设**：设 $d^{k}y = f^{(k)}(x)dx^k$ 对 $k \geq 1$ 成立。

**归纳递推**：对 $d^{k}y$ 再求一次微分。由于 $x$ 是自变量，$dx$ 是与 $x$ 无关的常数（$dx = \Delta x$ 是固定的增量，不随 $x$ 变化）。因此：

$$
\begin{aligned}
d^{k+1}y &= d(d^{k}y) \\
&= d\left[f^{(k)}(x)\,dx^k\right] \\
&= d\left[f^{(k)}(x)\right] \cdot dx^k \quad \text{（$dx^k$ 是常数，可提到微分算子外面）} \\
&= \left[f^{(k+1)}(x)\,dx\right] \cdot dx^k \\
&= f^{(k+1)}(x)\,dx^{k+1}
\end{aligned}
$$

由数学归纳法，$d^ny = f^{(n)}(x)dx^n$ 对一切正整数 $n$ 成立。✓

**与高阶导数的关系**：由定理 4.23 立即得到：

$$\frac{d^ny}{dx^n} = f^{(n)}(x)$$

Leibniz 记号 $\frac{d^ny}{dx^n}$ 现在获得了自然的"分数"解释——分子是 $n$ 阶微分，分母是 $dx$ 的 $n$ 次幂。这正是 Leibniz 记号设计的高明之处。

**重要区别——高阶微分无形式不变性**：

当 $x$ 是中间变量（即 $x = g(t)$ 是 $t$ 的函数）时，情况完全不同。此时 $dx = g'(t)dt$ 不再是常数，$d^2x = g''(t)dt^2 \neq 0$。于是二阶微分的计算变为：

$$
\begin{aligned}
d^2y &= d(dy) = d[f'(x)dx] \\
&= d[f'(x)] \cdot dx + f'(x) \cdot d(dx) \quad \text{（微分乘积法则）} \\
&= f''(x)dx \cdot dx + f'(x) \cdot d^2x \\
&= f''(x)dx^2 + f'(x)d^2x
\end{aligned}
$$

当 $x$ 是自变量时，$d^2x = 0$，退化为 $d^2y = f''(x)dx^2$。但当 $x$ 是中间变量时，$d^2x \neq 0$，多出了一项 $f'(x)d^2x$。因此，**高阶微分不具有形式不变性**——这正是 §4.2 中强调"一阶"的原因。

## 5. 例题

### 例 1：可导与可微的等价性验证

设 $f(x) = x^3 - 2x$，求 $f$ 在 $x=1$ 处的微分 $dy$，并验证 $\Delta y - dy = o(\Delta x)$。

**解**：

**第 1 步：求 $f'(1)$ 和 $dy$**。

$$f'(x) = 3x^2 - 2, \qquad f'(1) = 3 - 2 = 1$$

因此 $dy = f'(1)\,dx = 1 \cdot \Delta x = \Delta x$（由约定 $dx = \Delta x$）。

**第 2 步：计算 $\Delta y$**。

$$
\begin{aligned}
\Delta y &= f(1+\Delta x) - f(1) \\
&= \left[(1+\Delta x)^3 - 2(1+\Delta x)\right] - \left[1^3 - 2\cdot 1\right] \\
&= \left[1 + 3\Delta x + 3(\Delta x)^2 + (\Delta x)^3 - 2 - 2\Delta x\right] - [-1] \\
&= \left[\Delta x + 3(\Delta x)^2 + (\Delta x)^3 - 1\right] + 1 \\
&= \Delta x + 3(\Delta x)^2 + (\Delta x)^3
\end{aligned}
$$

**第 3 步：计算差值并验证 $o(\Delta x)$ 性质**。

$$\Delta y - dy = \left[\Delta x + 3(\Delta x)^2 + (\Delta x)^3\right] - \Delta x = 3(\Delta x)^2 + (\Delta x)^3$$

验证高阶无穷小条件：

$$\lim_{\Delta x \to 0} \frac{\Delta y - dy}{\Delta x} = \lim_{\Delta x \to 0} \frac{3(\Delta x)^2 + (\Delta x)^3}{\Delta x} = \lim_{\Delta x \to 0} \left[3\Delta x + (\Delta x)^2\right] = 0$$

因此 $\Delta y - dy = o(\Delta x)$，即 $\Delta y = dy + o(\Delta x)$——验证了分解式成立。✓

**数值检验**：取 $\Delta x = 0.01$：
- $dy = 0.01$
- $\Delta y = 0.01 + 3(0.0001) + (0.000001) = 0.010301$
- $\Delta y - dy = 0.000301$
- $\frac{\Delta y - dy}{\Delta x} = 0.0301$（很小，且随 $\Delta x$ 减小而减小，趋于 0）

### 例 2：微分运算法则

(1) 设 $y = (x^2 + 2x)(x^3 - 1)$，求 $dy$。

(2) 设 $y = \dfrac{x^2}{x+1}$（$x \neq -1$），求 $dy$。

(3) 设 $y = 3x^5 - 2x^3 + x$，求 $dy$。

**解 (1)**：

令 $u = x^2 + 2x$，$v = x^3 - 1$。则 $du = (2x + 2)dx$，$dv = 3x^2 dx$。

由乘积法则 $d(uv) = v\,du + u\,dv$：

$$
\begin{aligned}
dy &= v\,du + u\,dv \\
&= (x^3 - 1)(2x + 2)dx + (x^2 + 2x)(3x^2)dx \\
&= \left[(x^3 - 1)(2x + 2) + (x^2 + 2x)(3x^2)\right]dx \\
&= \left[2x^4 + 2x^3 - 2x - 2 + 3x^4 + 6x^3\right]dx \\
&= (5x^4 + 8x^3 - 2x - 2)dx
\end{aligned}
$$

**验证**：先展开 $y = (x^2+2x)(x^3-1) = x^5 + 2x^4 - x^2 - 2x$，求导得 $y' = 5x^4 + 8x^3 - 2x - 2$，$dy = y'dx$ 一致。✓

**解 (2)**：

令 $u = x^2$，$v = x+1$。$du = 2x\,dx$，$dv = dx$。

由商法则 $d\!\left(\frac{u}{v}\right) = \frac{v\,du - u\,dv}{v^2}$：

$$dy = \frac{(x+1)(2x\,dx) - x^2\cdot dx}{(x+1)^2} = \frac{2x(x+1) - x^2}{(x+1)^2}dx = \frac{x^2 + 2x}{(x+1)^2}dx$$

**验证**：$y' = \frac{2x(x+1) - x^2}{(x+1)^2} = \frac{x^2+2x}{(x+1)^2}$，$dy = y'dx$ 一致。✓

**解 (3)**：

直接用线性性：$d(3x^5 - 2x^3 + x) = 3\,d(x^5) - 2\,d(x^3) + d(x)$

由幂法则的微分形式：$d(x^n) = n x^{n-1}dx$

$$dy = 3(5x^4)dx - 2(3x^2)dx + (1)dx = (15x^4 - 6x^2 + 1)dx$$

### 例 3：一阶微分形式不变性

设 $y = (x^2 + 1)^3$，用一阶微分形式不变性求 $dy$。

**解法一（直接法——先求导再乘 $dx$）**：

$$y' = 3(x^2 + 1)^2 \cdot 2x = 6x(x^2 + 1)^2$$

$$dy = 6x(x^2 + 1)^2\,dx$$

**解法二（形式不变性——引入中间变量）**：

令 $u = x^2 + 1$，则 $y = u^3$。由形式不变性：

$$dy = 3u^2\,du$$

再求 $du$：$du = d(x^2 + 1) = 2x\,dx$。

代入：

$$dy = 3(x^2 + 1)^2 \cdot 2x\,dx = 6x(x^2 + 1)^2\,dx$$

两种方法结果一致。✓

**解法二的优势**：将复合函数的微分分解为两个简单步骤——先对外层 $u^3$ 求微分，再对内层 $x^2+1$ 求微分。每一步都只处理简单的函数形式，降低了在单一步骤中同时处理多层复合的出错概率。

**推广到多层复合**：设 $y = \sqrt{(x^2+1)^3} = [(x^2+1)^3]^{1/2}$。令 $v = x^2+1$，$u = v^3$，$y = u^{1/2}$。逐层微分：

$$dy = \frac{1}{2}u^{-1/2}du = \frac{1}{2}(v^3)^{-1/2} \cdot 3v^2 dv = \frac{3}{2}v^{3/2} \cdot 2x\,dx = 3x(x^2+1)^{1/2}dx$$

**验证**：直接法 $y = (x^2+1)^{3/2}$，$y' = \frac{3}{2}(x^2+1)^{1/2} \cdot 2x = 3x\sqrt{x^2+1}$，$dy = 3x\sqrt{x^2+1}\,dx$ 一致。✓

### 例 4：微分在近似计算中的综合应用

(1) 用微分近似计算 $\sqrt[3]{26}$ 的值。

(2) 用微分近似计算 $(1.02)^5$ 的值。

**解 (1)**：

**第 1 步（选函数与基点）**：令 $f(x) = \sqrt[3]{x}$，取 $x_0 = 27$（因为 $\sqrt[3]{27}=3$ 容易计算），$\Delta x = -1$（因为 $26 = 27-1$）。

**第 2 步（求导数值）**：$f'(x) = \frac{1}{3}x^{-2/3} = \frac{1}{3\sqrt[3]{x^2}}$。

$$f'(27) = \frac{1}{3\sqrt[3]{27^2}} = \frac{1}{3\sqrt[3]{729}} = \frac{1}{3 \times 9} = \frac{1}{27}$$

**第 3 步（构造近似）**：

$$\sqrt[3]{26} = f(27-1) \approx f(27) + f'(27) \cdot (-1) = 3 + \frac{1}{27} \times (-1) = 3 - \frac{1}{27} \approx 2.96296$$

**实际值**：$\sqrt[3]{26} \approx 2.96250$，近似误差约为 $4.6 \times 10^{-4}$。

**基点选择的分析**：为什么选 $x_0=27$ 而非 $x_0=8$？因为 $26$ 离 $27$ 更近（$|\Delta x| = 1$），也离 $8$ 更近确实更近一些，但 $27$ 处的导数值 $\frac{1}{27}$ 更小，同样 $\Delta x$ 下误差更小。实际上，用 $x_0=8$ 时 $\Delta x = 18$ 过大，线性近似不再有效。因此基点的选择应同时考虑：**$x_0$ 处的函数值易算，且 $|\Delta x|$ 充分小**。

**解 (2)**：

**第 1 步（选函数与基点）**：令 $f(x) = x^5$，取 $x_0 = 1$，$\Delta x = 0.02$（因为 $1.02 = 1 + 0.02$）。

**第 2 步（求导数值）**：$f'(x) = 5x^4$，$f'(1) = 5$。

**第 3 步（构造近似）**：

$$(1.02)^5 \approx 1^5 + 5 \times 0.02 = 1 + 0.1 = 1.1$$

**实际值**：$(1.02)^5 = 1.10408...$，近似误差约为 $0.00408$。

**验证**：也可用常用近似公式 $(1+x)^\alpha \approx 1 + \alpha x$（$x$ 很小时），令 $x = 0.02$，$\alpha = 5$，得 $1 + 5 \times 0.02 = 1.1$，结果一致。✓

### 例 5：高阶微分

设 $f(x) = x^5 - 3x^2$，求 $dy$，$d^2y$，$d^3y$，$d^4y$，$d^5y$，$d^6y$。

**解**：

**一阶微分**：$f'(x) = 5x^4 - 6x$，$dy = (5x^4 - 6x)dx$。

**二阶微分**：$f''(x) = 20x^3 - 6$，$d^2y = (20x^3 - 6)dx^2$。

**三阶微分**：$f'''(x) = 60x^2$，$d^3y = 60x^2\,dx^3$。

**四阶微分**：$f^{(4)}(x) = 120x$，$d^4y = 120x\,dx^4$。

**五阶微分**：$f^{(5)}(x) = 120$，$d^5y = 120\,dx^5$。

**六阶微分**：$f^{(6)}(x) = 0$，$d^6y = 0$。

**验证高阶微分与高阶导数的关系**：

$$\frac{d^2y}{dx^2} = \frac{(20x^3 - 6)dx^2}{dx^2} = 20x^3 - 6 = f''(x)$$

$$\frac{d^3y}{dx^3} = \frac{60x^2\,dx^3}{dx^3} = 60x^2 = f'''(x)$$

结果完全吻合。✓

**观察**：五次多项式从六阶微分开始全部为零——这与"$n$ 次多项式的 $n+1$ 阶以上导数为零"一致。微分运算本质上与求导运算同步，区别仅在于每次运算后额外乘上一个 $dx$。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 微分 $dy$ 等于 $\Delta y$，二者没有区别 | $dy$ 是 $\Delta y$ 的线性主部：$\Delta y = dy + o(\Delta x)$。只有当 $f$ 是线性函数（图像为直线）时才有 $dy = \Delta y$ |
| $\frac{dy}{dx}$ 从来就是分数 $dy \div dx$ | 在未定义微分之前，$\frac{dy}{dx}$ 是作为整体记号引入的（文件 01）。在定义了微分之后，它才获得了"微分之商"的意义——这是概念发展的结果而非起点 |
| $d(uv) = du \cdot dv$ | $d(uv) = v\,du + u\,dv$。微分乘积法则仍然是两项之和，与导数乘积法则 $(uv)' = u'v+uv'$ 对应。$du\cdot dv$ 是二阶小量，不是一阶微分 |
| 一阶微分形式不变性对任意阶微分都成立 | 只有一阶微分具有形式不变性。当 $x$ 是中间变量时，$d^2y = f''(x)dx^2 + f'(x)d^2x$，与 $x$ 为自变量时的 $d^2y = f''(x)dx^2$ 不同 |
| 近似公式 $f(x_0+\Delta x) \approx f(x_0) + f'(x_0)\Delta x$ 中 $\Delta x$ 可以任意大 | 该近似的误差为 $o(\Delta x)$，只有在 $|\Delta x|$ 充分小时才有效。$\Delta x$ 越大，误差通常越大且不可控 |
| $d^2y$ 是 $d(dy)$，$dx^2$ 是 $d(x^2)$ | $dx^2$ 是 $(dx)^2$ 的简写，即自变量微分的平方。$d(x^2) = 2x\,dx$ 是完全不同的量 |
| 在微分近似计算中，基点选哪里都可以 | 基点应同时满足：(1) $f(x_0)$ 容易计算；(2) $f'(x_0)$ 容易计算；(3) $|\Delta x|$ 充分小。三个条件缺一不可 |

**(3)** $\dfrac{1}{1.03}$

**选函数与基点**：$f(x) = \frac{1}{x}$，$x_0 = 1$，$\Delta x = 0.03$。

**求导数值**：$f'(x) = -\frac{1}{x^2}$，$f'(1) = -1$。

**构造近似**：

$$\frac{1}{1.03} \approx \frac{1}{1} + (-1) \times 0.03 = 1 - 0.03 = 0.97$$

实际值 $\frac{1}{1.03} \approx 0.970874$，误差约 $8.7 \times 10^{-4}$。

也可直接用常用公式 $\frac{1}{1+x} \approx 1 - x$（$|x|$ 很小），代入 $x=0.03$ 得 $0.97$。✓

**5.** 设 $f(x) = x^4 - 2x^2$，求 $dy$，$d^2y$，$d^3y$，$d^4y$，并验证 $\dfrac{d^2y}{dx^2} = f''(x)$，$\dfrac{d^3y}{dx^3} = f'''(x)$。

<details><summary>参考答案</summary>

**一阶微分**：$f'(x) = 4x^3 - 4x$，$dy = (4x^3 - 4x)dx$。

**二阶微分**：$f''(x) = 12x^2 - 4$，$d^2y = (12x^2 - 4)dx^2$。

**验证**：$\dfrac{d^2y}{dx^2} = \dfrac{(12x^2 - 4)dx^2}{dx^2} = 12x^2 - 4 = f''(x)$。✓

**三阶微分**：$f'''(x) = 24x$，$d^3y = 24x\,dx^3$。

**验证**：$\dfrac{d^3y}{dx^3} = \dfrac{24x\,dx^3}{dx^3} = 24x = f'''(x)$。✓

**四阶微分**：$f^{(4)}(x) = 24$，$d^4y = 24\,dx^4$。

此后 $n \geq 5$ 时 $f^{(n)}(x) = 0$，$d^ny = 0$。

</details>

**6.** 一金属立方体的棱长 $L = 10\text{ cm}$，测量误差为 $\pm 0.05\text{ cm}$。用微分估算立方体体积 $V = L^3$ 的最大可能误差。

<details><summary>参考答案</summary>

**建模**：体积 $V(L) = L^3$，$L_0 = 10$，$|dL| \leq 0.05$。

**求微分**：$dV = V'(L_0)\,dL = 3L_0^2\,dL = 3 \times 100 \times dL = 300\,dL$。

**估计误差**：$|dV| = 300 \times |dL| \leq 300 \times 0.05 = 15\text{ cm}^3$。

因此体积的最大可能误差约为 $\pm 15\text{ cm}^3$。

**相对误差**：$\left|\frac{dV}{V}\right| = \frac{300|dL|}{1000} = 0.3|dL| = 0.3 \times 0.05 = 0.015 = 1.5\%$。

即体积的相对误差约为 $1.5\%$，而棱长的相对误差为 $0.5\%$——体积误差被放大了 $3$ 倍，这是因为 $V \propto L^3$，指数 $3$ 从导数中"传递"到了误差中。

</details>

**7.** 设 $y = (x^2+1)(x^3-2x)$，用微分运算法则求 $dy$ 及在 $x=1$ 处 $\Delta x = 0.1$ 时 $dy$ 的数值。

<details><summary>参考答案</summary>

令 $u = x^2+1$，$v = x^3-2x$。$du = 2x\,dx$，$dv = (3x^2-2)dx$。

由乘积法则：

$$
\begin{aligned}
dy &= v\,du + u\,dv \\
&= (x^3-2x)(2x\,dx) + (x^2+1)(3x^2-2)dx \\
&= [2x(x^3-2x) + (x^2+1)(3x^2-2)]dx \\
&= [2x^4 - 4x^2 + 3x^4 - 2x^2 + 3x^2 - 2]dx \\
&= (5x^4 - 3x^2 - 2)dx
\end{aligned}
$$

在 $x=1$ 处：$dy = (5-3-2)dx = 0$。

当 $\Delta x = 0.1$ 时：$dy = 0 \times 0.1 = 0$。

**几何含义**：曲线 $y = (x^2+1)(x^3-2x)$ 在 $x=1$ 处有水平切线（导数为零），因此在 $x=1$ 附近，函数的线性主部为零，$\Delta y$ 的变化完全由高阶无穷小贡献。

**验证**：$y(1) = (1+1)(1-2) = 2 \times (-1) = -2$。
$y(1.1) = (1.21+1)(1.331-2.2) = 2.21 \times (-0.869) \approx -1.920$。
$\Delta y = -1.920 - (-2) = 0.080$，而 $dy = 0$。
$\frac{\Delta y}{\Delta x} = 0.80$——这正好是一阶导数附近差商的值，但因其趋于 0，所以线性主部为 0。

</details>

**8.** 证明：若 $f$ 在 $x_0$ 处可微，则 $f$ 在 $x_0$ 处连续。（提示：利用可微 ⇒ 可导 ⇒ 连续，或直接从可微定义出发）

<details><summary>参考答案</summary>

**证法一（利用已知定理）**：由定理 4.19，可微 ⇒ 可导。再由文件 01 定理 4.3（可导必连续），$f$ 在 $x_0$ 处连续。证毕。✓

**证法二（从可微定义直接证）**：设 $f$ 在 $x_0$ 处可微，则存在常数 $A$ 使得：

$$\Delta y = A\Delta x + o(\Delta x)$$

取极限 $\Delta x \to 0$：

$$\lim_{\Delta x\to0} \Delta y = \lim_{\Delta x\to0} [A\Delta x + o(\Delta x)] = A \cdot 0 + 0 = 0$$

因此 $\lim_{x\to x_0} f(x) = f(x_0)$，即 $f$ 在 $x_0$ 处连续。证毕。✓

**两种证法的对比**：证法一利用了"可导"这个中间桥梁；证法二直接使用可微定义，更简洁地展示了可微与连续的关系。

</details>
