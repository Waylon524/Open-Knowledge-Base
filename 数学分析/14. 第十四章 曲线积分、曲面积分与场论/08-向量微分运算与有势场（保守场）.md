## 2. 向量微分算子 ∇ 的系统框架 (Type A-3)

### 2.1 Nabla 算子的定义

**定义 14.17（Nabla 算子 / Del Operator）**：Nabla 算子 ∇ 是一个形式向量微分算子：
$$\boxed{\nabla = \left( \frac{\partial}{\partial x},\; \frac{\partial}{\partial y},\; \frac{\partial}{\partial z} \right)}$$

∇ 不是普通的向量——它的"分量"是偏导数运算符，必须作用于函数才产生意义。Nabla 算子有三种基本运算方式，对应于它和标量/向量场的三种"乘法"。

### 2.2 梯度：∇ 作用于标量场

**定义 14.18（梯度 / Gradient）**：设 f(x,y,z) 是具有一阶连续偏导数的标量函数。f 的**梯度**定义为：
$$\boxed{\nabla f = \left( \frac{\partial f}{\partial x},\; \frac{\partial f}{\partial y},\; \frac{\partial f}{\partial z} \right)}$$

梯度是将 Nabla 算子与标量函数进行**形式数乘**（类比：向量 × 标量）。

- **输入**：标量函数 f(x,y,z)（每个点一个数值）
- **输出**：向量场 ∇f(x,y,z)（每个点一个向量）
- **物理意义**：∇f 的方向是 f 增长最快的方向，大小是 f 在该方向的变化率

**例 1**：f(x,y,z) = x²y + yz² + e^y sin x
$$\nabla f = (2xy + e^y\cos x,\; x² + z² + e^y\sin x,\; 2yz)$$

### 2.3 散度：∇ 点积向量场

**定义 14.19（散度 / Divergence）**：设 F(x,y,z) = (P,Q,R) 是具有一阶连续偏导数的向量场。F 的**散度**定义为：
$$\boxed{\nabla\cdot\mathbf{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}}$$

散度是将 Nabla 算子与向量场进行**形式点积**（类比：向量 · 向量）。

- **输入**：向量场 F(x,y,z)（每个点一个向量）
- **输出**：标量场 ∇·F(x,y,z)（每个点一个数值）
- **物理意义**：散度度量"通量体密度"——该点处单位体积的净通量。∇·F > 0 表示源，∇·F < 0 表示汇，∇·F = 0 表示无散

**例 2**：F(x,y,z) = (x², y², z²)
∇·F = 2x + 2y + 2z

**例 3**：F(x,y,z) = (y, z, x)
∇·F = ∂y/∂x + ∂z/∂y + ∂x/∂z = 0 + 0 + 0 = 0

### 2.4 旋度：∇ 叉积向量场

**定义 14.20（旋度 / Curl）**：设 F(x,y,z) = (P,Q,R) 具有一阶连续偏导数。F 的**旋度**定义为：
$$\boxed{\nabla\times\mathbf{F} = \left( \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z},\; \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x},\; \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right)}$$

等价的行列式记忆形式：
$$\nabla\times\mathbf{F} = \begin{vmatrix}
\mathbf{i} & \mathbf{j} & \mathbf{k} \\[2pt]
\dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\[6pt]
P & Q & R
\end{vmatrix}$$

旋度是将 Nabla 算子与向量场进行**形式叉积**（类比：向量 × 向量）。

- **输入**：向量场 F(x,y,z)（每个点一个向量）
- **输出**：向量场 ∇×F(x,y,z)（每个点一个向量）
- **物理意义**：旋度度量"环量体密度"——该点处局部旋转的轴和强度。∇×F = 0 表示无旋，∇×F ≠ 0 表示有旋

**例 4**：F(x,y,z) = (y, z, x)
$$\nabla\times\mathbf{F} = (1-1,\; 1-1,\; 1-1) = (0,0,0)$$

**例 5**：F(x,y,z) = (y, -x, 0)
$$\nabla\times\mathbf{F} = (0-0,\; 0-0,\; -1-1) = (0,0,-2)$$

### 2.5 三种运算的统一视角

| 运算 | 记号 | 类比 | 输入 | 输出 | 零的含义 |
|:-----|:-----|:-----|:-----|:-----|:---------|
| 梯度 | ∇f | 标量 × 向量 | 标量 f | 向量场 | 常数函数 |
| 散度 | ∇·F | 向量 · 向量 | 向量场 F | 标量场 | 无散（管量场） |
| 旋度 | ∇×F | 向量 × 向量 | 向量场 F | 向量场 | 无旋（保守场） |

**核心思想**：三种运算都源于同一个算子 ∇，只是"乘法"方式不同（数乘、点积、叉积）。理解这一统一视角有助于记忆公式和区分不同运算的输入输出类型。

## 3. 向量微分恒等式 (Type A-4)

以下两个恒等式是向量分析中最基本的微分恒等式，分别称为"旋度的散度恒为零"和"梯度的旋度恒为零"。

### 3.1 恒等式一：∇·(∇×F) ≡ 0（旋度的散度恒为零）

**定理 14.21**：设 F(x,y,z) = (P,Q,R) 具有二阶连续偏导数，则：
$$\boxed{\nabla\cdot(\nabla\times\mathbf{F}) \equiv 0}$$

**证明**：由旋度定义（定义 14.20）和散度定义（定义 14.19）：
$$\begin{aligned}
\nabla\cdot(\nabla\times\mathbf{F}) &= \frac{\partial}{\partial x}\left( \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} \right) + \frac{\partial}{\partial y}\left( \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} \right) + \frac{\partial}{\partial z}\left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) \\[4pt]
&= \frac{\partial^2 R}{\partial x\partial y} - \frac{\partial^2 Q}{\partial x\partial z} + \frac{\partial^2 P}{\partial y\partial z} - \frac{\partial^2 R}{\partial y\partial x} + \frac{\partial^2 Q}{\partial z\partial x} - \frac{\partial^2 P}{\partial z\partial y}
\end{aligned}$$

重新组合，将含有同一种函数的项放在一起：
$$= \left( \frac{\partial^2 R}{\partial x\partial y} - \frac{\partial^2 R}{\partial y\partial x} \right) + \left( \frac{\partial^2 Q}{\partial z\partial x} - \frac{\partial^2 Q}{\partial x\partial z} \right) + \left( \frac{\partial^2 P}{\partial y\partial z} - \frac{\partial^2 P}{\partial z\partial y} \right)$$

由混合偏导数的对称性（Clairaut 定理，需 F 具有二阶连续偏导数）：
$$\frac{\partial^2 R}{\partial x\partial y} = \frac{\partial^2 R}{\partial y\partial x},\quad
\frac{\partial^2 Q}{\partial z\partial x} = \frac{\partial^2 Q}{\partial x\partial z},\quad
\frac{\partial^2 P}{\partial y\partial z} = \frac{\partial^2 P}{\partial z\partial y}$$

因此括号内各项均相互抵消：
$$\nabla\cdot(\nabla\times\mathbf{F}) = 0 + 0 + 0 = 0 \quad \blacksquare$$

**物理意义**：任何向量场的旋度都是**无散场**（divergence-free field，亦称**管量场**）。这意味着：
- 在流体力学中，涡量场（速度场的旋度）是不可压缩的——涡线既不能产生也不能消灭，必须形成闭合的涡环或延伸到边界
- 在电磁学中，磁场满足 ∇·B = 0，这一方程暗示**磁场可以表示为某个向量势的旋度**：B = ∇×A。磁感线总是闭合的，不存在磁单极

### 3.2 恒等式二：∇×(∇f) ≡ 0（梯度的旋度恒为零）

**定理 14.22**：设 f(x,y,z) 具有二阶连续偏导数，则：
$$\boxed{\nabla\times(\nabla f) \equiv \mathbf{0}}$$

**证明**：由梯度定义（定义 14.18），∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z)。代入旋度定义（定义 14.20）：

第一分量：
$$\frac{\partial}{\partial y}\left( \frac{\partial f}{\partial z} \right) - \frac{\partial}{\partial z}\left( \frac{\partial f}{\partial y} \right) = \frac{\partial^2 f}{\partial y\partial z} - \frac{\partial^2 f}{\partial z\partial y} = 0$$

第二分量：
$$\frac{\partial}{\partial z}\left( \frac{\partial f}{\partial x} \right) - \frac{\partial}{\partial x}\left( \frac{\partial f}{\partial z} \right) = \frac{\partial^2 f}{\partial z\partial x} - \frac{\partial^2 f}{\partial x\partial z} = 0$$

第三分量：
$$\frac{\partial}{\partial x}\left( \frac{\partial f}{\partial y} \right) - \frac{\partial}{\partial y}\left( \frac{\partial f}{\partial x} \right) = \frac{\partial^2 f}{\partial x\partial y} - \frac{\partial^2 f}{\partial y\partial x} = 0$$

三个分量全部为零，因此 ∇×(∇f) = (0,0,0) = **0**。$\blacksquare$

**物理意义**：任何梯度场都是**无旋场**（irrotational field）。这意味着：
- 梯度场的旋度恒为零，因此梯度场一定是保守场（在下一节中将详细讨论）
- 静电场 E 由电势 V 的负梯度给出：E = -∇V，因此 ∇×E = 0，即静电场是无旋场
- 重力场 F = -∇(GMm/r) 也是无旋场

### 3.3 两个恒等式的对比

| 恒等式 | 记法 | 含义 | 物理应用 |
|:-------|:-----|:-----|:---------|
| 旋度的散度 | ∇·(∇×F) ≡ 0 | 旋度场无散 | B = ∇×A, ∇·B = 0 |
| 梯度的旋度 | ∇×(∇f) ≡ 0 | 梯度场无旋 | E = -∇V, ∇×E = 0 |

### 3.4 拉普拉斯算子

作为 Nabla 算子的一个重要复合运算，定义**拉普拉斯算子**：
$$\boxed{\nabla^2 f = \nabla\cdot(\nabla f) = \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} + \frac{\partial^2 f}{\partial z^2}}$$

∇²f 是标量函数 f 的**梯度的散度**，是一个标量。它出现在 Poisson 方程 ∇²V = -ρ/ε₀（电势与电荷密度的关系）、热传导方程、波动方程等众多物理方程中。

## 4. 有势场与保守场 (Type A-5)

### 4.1 有势场的定义

**定义 14.23（有势场 / Potential Field）**：设 F 是定义在区域 Ω ⊂ ℝ³ 上的向量场。若存在标量函数 U(x,y,z)，使得：
$$\boxed{\mathbf{F} = \nabla U}$$

则称 F 为**有势场**（potential field），U 称为 F 的**势函数**（potential function）。

**物理起源**：在物理学中，保守力 F（如重力、静电力）可以表示为势能函数的负梯度：F = -∇V。本教材使用正号约定 F = ∇U，相当于 U = -V。

**例 6**：F = (2x, 2y, -2z) 是有势场，因为存在 U = x² + y² - z² 满足 ∇U = (2x, 2y, -2z) = F。

### 4.2 保守场与路径无关

**定义 14.24（保守场 / Conservative Field）**：设 F 是区域 Ω 上的向量场。若曲线积分 ∫_L F·dr 的值只依赖于起点 A 和终点 B（而与连接 A、B 的具体路径无关），则称 F 为**保守场**。等价地，若对 Ω 中任意分段光滑闭曲线 C 有 ∮_C F·dr = 0，则 F 为保守场。

### 4.3 等价关系链

以下四个条件在**单连通区域**（定义见 4.4 节）上是相互等价的：

**定理 14.25（保守场等价条件）**：设 Ω ⊂ ℝ³ 是单连通开区域，F 在 Ω 上具有连续一阶偏导数。则以下四个命题等价：
1. **有势场**（存在性）：存在标量函数 U 使 F = ∇U
2. **无旋场**（旋度为零）：∇×F = **0** 在 Ω 上处处成立
3. **路径无关**：曲线积分 ∫_L F·dr 只取决于起点和终点，与路径无关
4. **闭曲线积分为零**：对 Ω 中任意分段光滑闭曲线 C，有 ∮_C F·dr = 0

**证明的等价链条**：

**(1) ⇒ (3)**：若 F = ∇U，则对任意路径 L 从 A 到 B，由链式法则（将在 6.1 节正式证明）：
$$\int_L \mathbf{F}\cdot d\mathbf{r} = \int_L \nabla U\cdot d\mathbf{r} = U(B) - U(A)$$
结果只依赖于端点，与路径无关。

**(3) ⇒ (4)**：若积分与路径无关，取闭曲线 C 上任意两点 A、B 将 C 分为两段 L₁ 和 L₂，则：
$$\oint_C \mathbf{F}\cdot d\mathbf{r} = \int_{L_1} + \int_{L_2} = \int_{A\to B}^{L_1} - \int_{A\to B}^{L_2} = 0$$
（因为两段路径的积分由路径无关性必然相等）

**(4) ⇒ (1)**：若所有闭曲线积分为零，可定义势函数：
$$U(x,y,z) = \int_{M_0}^{(x,y,z)} \mathbf{F}\cdot d\mathbf{r}$$
其中 M₀ 是 Ω 中某固定点。路径无关性保证了 U 是良定义的，且可直接验证 ∇U = F。

**(2) ⇔ (4)**：由 Stokes 公式，对任意以 C 为边界的曲面 Σ ⊂ Ω：
$$\oint_C \mathbf{F}\cdot d\mathbf{r} = \iint_{\Sigma} (\nabla\times\mathbf{F})\cdot\mathbf{n}\,dS \quad \text{(定理 14.16)}$$

若 ∇×F = **0**，则右端为零，因此所有闭曲线积分为零。反过来，若所有闭曲线积分为零，取 C 为微小环路，由 Stokes 公式可推出 ∇×F = **0**。

注意：(2) ⇒ (4) 的推导不需要单连通条件（Stokes 公式只需存在以 C 为边界的曲面 Σ ⊂ Ω）。但 (4) ⇒ (2) 需要"微小环路可以收缩到一点"的局部性质，这在任何开区域上都成立。**关键在于 (4) 或 (3) 定义的势函数必须全局成立**，这就要求区域是**单连通的**（否则可能存在闭曲线不能收缩到一点，使得从 (4) 定义的势函数不是良定义的）。

$\blacksquare$

### 4.4 单连通区域的条件讨论

**定义 14.26（单连通区域 / Simply Connected Domain）**：区域 Ω ⊂ ℝ³ 称为**单连通的**，如果 Ω 中任何一条闭曲线都可以连续地收缩到 Ω 中的一点而不离开 Ω。

**直观理解**：
- ℝ³（全空间）是单连通的 ✓
- ℝ³ 中挖去一个点，仍是单连通的（三维空间中点不阻挡曲线收缩）✓
- ℝ³ 中挖去一条直线段（如 z 轴），**不是**单连通的（曲线可以绕 z 轴一圈，无法收缩到一点）✗
- 二维平面 ℝ² 挖去一个点不是单连通的 ✗
- 实心球、立方体是单连通的 ✓
- 环面（甜甜圈形状）的内部不是单连通的 ✗

**单连通条件的重要性**：当区域不是单连通时，即使 ∇×F = 0 处处成立，也可能不存在全局定义的势函数。经典反例：在 ℝ² \ {(0,0)}（二维平面挖去原点）上，F = (-y/(x²+y²), x/(x²+y²)) 满足旋度为零（在定义域内），但沿绕原点一周的闭曲线积分为 2π ≠ 0，因此不是保守场。

### 4.5 三维情形下的完整判定流程

**判定工作流**：给定向量场 F = (P,Q,R)，判断其是否为有势场。

**Step 1**：检查定义域 Ω 是否为单连通区域。
- 若 Ω 非单连通，需额外谨慎——旋度为零是保守的必要条件但不充分。

**Step 2**：计算旋度 ∇×F。
- 若 ∇×F ≠ **0**（某点处旋度非零）→ **不是有势场**。停止。
- 若 ∇×F = **0** 处处成立，进入下一步。

**Step 3**（单连通区域）：若 Ω 是单连通的，则 ∇×F = **0** 等价于 F 是有势场。
- 直接进入势函数求法（第 5 节）。

**Step 3'**（非单连通区域）：若 Ω 非单连通，还需验证沿"关键闭曲线"的积分为零（这些闭曲线是不能收缩到零的环路）。
- 例如：若 Ω = ℝ³ 除去 z 轴，需验证绕 z 轴一圈的闭曲线积分为零。
- 若所有关键闭曲线上积分为零，则是有势场；否则不是。

## 5. 势函数的求法 (Type A-1)

### 5.1 直接积分法（逐次偏积分 + 待定函数法）

若已知 F = (P,Q,R) 是有势场，求 U 使 ∇U = F，即解偏微分方程组：
$$\frac{\partial U}{\partial x} = P,\quad \frac{\partial U}{\partial y} = Q,\quad \frac{\partial U}{\partial z} = R$$

**直接积分法标准步骤**：

1. **对 x 偏积分**：由 ∂U/∂x = P，得：
   $$U(x,y,z) = \int P\,dx + \varphi(y,z)$$
   其中 φ(y,z) 是待定的**二元函数**（积分"常数"与 x 无关，但可能依赖 y,z）。

2. **对 y 偏导并与 Q 比较**：对第 1 步的结果求 ∂/∂y：
   $$\frac{\partial U}{\partial y} = \frac{\partial}{\partial y}\int P\,dx + \frac{\partial\varphi}{\partial y} = Q$$
   解得 ∂φ/∂y = Q - ∂(∫P\,dx)/∂y。
   对 y 积分：φ(y,z) = ∫(Q - ∂(∫P\,dx)/∂y) dy + ψ(z)，其中 ψ(z) 是待定的一元函数。

3. **对 z 偏导并与 R 比较**：代入第 2 步结果，求 ∂/∂z：
   $$\frac{\partial U}{\partial z} = \frac{\partial}{\partial z}(\cdots) + \frac{\partial\psi}{\partial z} = R$$
   解得 ψ'(z) = R - ∂(⋯)/∂z。

4. **积分得到 ψ(z)**，代入回代即得 U(x,y,z)。

5. **添加任意常数**：最终结果 U = U₀ + C，其中 C 是任意常数（势函数的"零点"自由选择）。

### 5.2 直接积分法例题

**例 7**：判断 F = (2x, 2y, -2z) 是否为有势场。若是，求其势函数。

**解**：

**Step 1（旋度检验）**：
∇×F = (∂(-2z)/∂y - ∂(2y)/∂z, ∂(2x)/∂z - ∂(-2z)/∂x, ∂(2y)/∂x - ∂(2x)/∂y)
= (0 - 0, 0 - 0, 0 - 0) = (0,0,0)。旋度为零，且定义域 ℝ³ 单连通，故是有势场。

**Step 2（直接积分法）**：

第 1 步：∂U/∂x = 2x → U = x² + φ(y,z)

第 2 步：∂U/∂y = ∂φ/∂y = 2y → φ = y² + ψ(z)

第 3 步：代入得 U = x² + y² + ψ(z)
∂U/∂z = ψ'(z) = -2z → ψ(z) = -z² + C

第 4 步：U(x,y,z) = x² + y² - z² + C。取 C = 0，得一个势函数 U = x² + y² - z²。

**验证**：∇U = (2x, 2y, -2z) = F。✓

---

**例 8**：判断 F = (y²\cos x + z,\; 2y\sin x + 1,\; x + 3z²) 是否为有势场。若是，求势函数。

**解**：

**Step 1（旋度检验）**：
P = y²\cos x + z, Q = 2y\sin x + 1, R = x + 3z²

$$\begin{aligned}
\nabla\times\mathbf{F} &= \left( \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z},\; \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x},\; \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) \\
&= (0 - 0,\; 1 - 1,\; 2y\cos x - 2y\cos x) = (0,0,0)
\end{aligned}$$

旋度为零，ℝ³ 单连通，是有势场。

**Step 2（直接积分法）**：

第 1 步：由 ∂U/∂x = y²\cos x + z：
$$U = \int (y^2\cos x + z)\,dx = y^2\sin x + xz + \varphi(y,z)$$

第 2 步：∂U/∂y = 2y\sin x + ∂φ/∂y = 2y\sin x + 1
因此 ∂φ/∂y = 1 → φ = y + ψ(z)

第 3 步：代入得 U = y²\sin x + xz + y + ψ(z)
∂U/∂z = x + ψ'(z) = x + 3z² → ψ'(z) = 3z² → ψ(z) = z³ + C

第 4 步：U(x,y,z) = y²\sin x + xz + y + z³ + C

**验证**：∇U = (y²\cos x + z, 2y\sin x + 1, x + 3z²) = F。✓

### 5.3 线积分法

**原理**：在保守场中，积分与路径无关。选择从某固定点（通常取原点）到点 (x,y,z) 的方便路径，计算曲线积分即得势函数：
$$U(x,y,z) = \int_{(0,0,0)}^{(x,y,z)} \mathbf{F}\cdot d\mathbf{r} + C$$

**标准折线路径**：从原点沿坐标轴分三段前进：
$$(0,0,0) \xrightarrow{L_1} (x,0,0) \xrightarrow{L_2} (x,y,0) \xrightarrow{L_3} (x,y,z)$$

各段的参数化：
- L₁：从 (0,0,0) 到 (x,0,0)，参数 t: 0→x，r(t) = (t,0,0)，dr = (dt,0,0)
- L₂：从 (x,0,0) 到 (x,y,0)，参数 t: 0→y，r(t) = (x,t,0)，dr = (0,dt,0)
- L₃：从 (x,y,0) 到 (x,y,z)，参数 t: 0→z，r(t) = (x,y,t)，dr = (0,0,dt)

**线积分法公式**：
$$\boxed{U(x,y,z) = \int_0^x P(t,0,0)\,dt + \int_0^y Q(x,t,0)\,dt + \int_0^z R(x,y,t)\,dt + C}$$

### 5.4 线积分法例题

**例 9**：用线积分法求 F = (2x, 2y, -2z) 的势函数。

**解**：P = 2x, Q = 2y, R = -2z。

沿折线路径：
$$\begin{aligned}
U &= \int_0^x 2t\,dt + \int_0^y 2t\,dt + \int_0^z (-2t)\,dt + C \\
&= [t^2]_0^x + [t^2]_0^y + [-t^2]_0^z + C \\
&= x^2 + y^2 - z^2 + C
\end{aligned}$$

与直接积分法的结果一致。✓

---

**例 10**：用线积分法求 F = (y²\cos x + z,\; 2y\sin x + 1,\; x + 3z²) 的势函数。

**解**：P = y²\cos x + z, Q = 2y\sin x + 1, R = x + 3z²。

沿折线路径：
- L₁ (t,0,0)：P(t,0,0) = 0²·cos t + 0 = 0
  $$\int_0^x 0\,dt = 0$$

- L₂ (x,t,0)：Q(x,t,0) = 2t·sin x + 1
  $$\int_0^y (2t\sin x + 1)\,dt = [t^2\sin x + t]_0^y = y^2\sin x + y$$

- L₃ (x,y,t)：R(x,y,t) = x + 3t²
  $$\int_0^z (x + 3t^2)\,dt = [xt + t^3]_0^z = xz + z^3$$

求和：U = 0 + (y²\sin x + y) + (xz + z³) + C = y²\sin x + xz + y + z³ + C

与直接积分法结果一致。✓

**两种方法的对比**：

| 方法 | 优势 | 劣势 | 适用场景 |
|:-----|:-----|:-----|:---------|
| 直接积分法 | 纯代数操作，不需要参数化 | 待定函数易出错，需要仔细求偏导 | 分量表达式较整齐的场 |
| 线积分法 | 步骤程式化，不易遗漏 | 需要分段计算三个一维积分 | 各类场通用，特别是复杂场 |

## 6. 有势场中线积分等于势函数差值的定理 (Type A-2)

### 6.1 定理陈述与证明

**定理 14.27（势函数基本定理 / Potential Function Theorem）**：设 F 是有势场，U 是 F 的一个势函数（即 F = ∇U）。L 是区域中从点 A 到点 B 的分段光滑有向曲线，则：
$$\boxed{\int_L \mathbf{F}\cdot d\mathbf{r} = U(B) - U(A)}$$

**证明**：设曲线 L 的参数化表示为 r(t) = (x(t), y(t), z(t))，t ∈ [a,b]，其中 r(a) = A，r(b) = B。由第二型曲线积分的参数化计算公式和链式法则：

$$\begin{aligned}
\int_L \mathbf{F}\cdot d\mathbf{r} &= \int_a^b \mathbf{F}(\mathbf{r}(t))\cdot \mathbf{r}'(t)\,dt \quad \text{(参数化公式)} \\
&= \int_a^b \nabla U(\mathbf{r}(t))\cdot \mathbf{r}'(t)\,dt \quad (\because \mathbf{F} = \nabla U) \\
&= \int_a^b \left( \frac{\partial U}{\partial x}\frac{dx}{dt} + \frac{\partial U}{\partial y}\frac{dy}{dt} + \frac{\partial U}{\partial z}\frac{dz}{dt} \right) dt \\
&= \int_a^b \frac{d}{dt}\big[U(\mathbf{r}(t))\big]\,dt \quad \text{(多元链式法则)} \\
&= U(\mathbf{r}(b)) - U(\mathbf{r}(a)) \quad \text{(微积分基本定理)} \\
&= U(B) - U(A) \quad \blacksquare
\end{aligned}$$

**核心意义**：该定理将曲线积分（一个复杂的极限和）简化为势函数在端点的函数值之差——这是**微积分基本定理在向量场中的推广**：
- 一维：∫_a^b f'(x) dx = f(b) - f(a)
- 向量场：∫_L ∇U·dr = U(B) - U(A)

### 6.2 定理的应用示例

**例 11**：设 F = (y²\cos x + z,\; 2y\sin x + 1,\; x + 3z²)，L 是从 A(0,0,0) 到 B(π/2, 1, 1) 的任意光滑曲线。计算：
$$I = \int_L (y^2\cos x + z)\,dx + (2y\sin x + 1)\,dy + (x + 3z^2)\,dz$$

**解**：由例 8，F 是有势场，势函数为 U = y²\sin x + xz + y + z³。
由定理 14.27：
$$\begin{aligned}
I &= U(B) - U(A) \\
&= U(\pi/2, 1, 1) - U(0, 0, 0) \\
&= (1^2\cdot\sin(\pi/2) + (\pi/2)(1) + 1 + 1^3) - (0 + 0 + 0 + 0) \\
&= (1 + \pi/2 + 1 + 1) - 0 = 3 + \frac{\pi}{2}
\end{aligned}$$

**关键看点**：不需要做任何积分计算！只需知道势函数在端点的值。相比直接参数化曲线积分，这极大地简化了计算。而且由于定理保证结果与路径无关，我们甚至不需要知道 L 的具体形状。

---

**例 12**：计算 ∫_L 2x\,dx + 2y\,dy - 2z\,dz，其中 L 是从 A(1,1,1) 到 B(2,3,0) 的任意分段光滑曲线。

**解**：F = (2x, 2y, -2z)。由例 7 知势函数 U = x² + y² - z²。
由定理 14.27：
$$\begin{aligned}
\int_L 2x\,dx + 2y\,dy - 2z\,dz &= U(2,3,0) - U(1,1,1) \\
&= (4 + 9 - 0) - (1 + 1 - 1) \\
&= 13 - 1 = 12
\end{aligned}$$

**验证**（用线积分法直接计算，选折线路径 (1,1,1)→(2,1,1)→(2,3,1)→(2,3,0)）：
- L₁: (1,1,1)→(2,1,1): ∫₁² 2t dt = [t²]₁² = 4-1 = 3
- L₂: (2,1,1)→(2,3,1): ∫₁³ 2t dt = [t²]₁³ = 9-1 = 8
- L₃: (2,3,1)→(2,3,0): ∫₁⁰ -2t dt = [-t²]₁⁰ = 0-(-1) = 1
- 总和 = 3+8+1 = 12 ✓

## 7. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|:---------|:---------|
| 梯度 ∇f、散度 ∇·F、旋度 ∇×F 只是记号不同，没有本质区别 | 三种运算的输入输出完全不同：∇f 标量→向量，∇·F 向量→标量，∇×F 向量→向量。物理意义也完全不同：梯度指向最大变化方向，散度度量源汇，旋度度量旋转 |
| ∇·(∇×F) = 0 是显而易见的，不需要检查条件 | 该恒等式的成立依赖于 F 具有**二阶连续偏导数**（Clairaut 定理的前提），否则混合偏导数不一定相等，恒等式可能不成立 |
| ∇×(∇f) ≡ 0 说明 f 的梯度场没有旋度，这总是对的 | 正确，但同样需要 f 具有**二阶连续偏导数**。如果 f 二阶偏导不连续，恒等式可能失效 |
| ∇×F = 0 一定等价于 F 是保守场 | 这只在**单连通区域**上成立。在非单连通区域（如 ℝ³ 挖去 z 轴），即使旋度处处为零，也可能有非零的闭曲线积分 |
| 势函数是唯一的 | 势函数相差一个任意常数：若 F = ∇U₁ = ∇U₂，则 U₁ - U₂ = 常数。物理中通常通过设定参考点（如 U(0,0,0) = 0）来固定势函数 |
| 只有能用直接积分法求出的场才是保守场 | 直接积分法和线积分法都能求保守场的势函数。线积分法更为程式化，适合复杂场；直接积分法需要一些代数技巧 |
| 定理 ∫_L F·dr = U(B) - U(A) 只适用于直线路径 | 该定理适用于**任何**分段光滑曲线 L（从 A 到 B），与路径形状无关。这正是保守场的本质特征 |

**9.**（有势场的等价关系）设 F = (f(x), g(y), h(z))，其中 f,g,h 均为连续可微的一元函数。证明 F 总是有势场，并求其势函数。

<details><summary>参考答案</summary>

F = (f(x), g(y), h(z))，即 P = f(x)，Q = g(y)，R = h(z)。

计算旋度：
∇×F = \left( \dfrac{\partial h(z)}{\partial y} - \dfrac{\partial g(y)}{\partial z},\; \dfrac{\partial f(x)}{\partial z} - \dfrac{\partial h(z)}{\partial x},\; \dfrac{\partial g(y)}{\partial x} - \dfrac{\partial f(x)}{\partial y} \right)
= (0 - 0,\; 0 - 0,\; 0 - 0) = (0,0,0)

因此 F 在任何单连通区域上都是有势场。

求势函数：∂U/∂x = f(x) → U = ∫f(x) dx + φ(y,z)
∂U/∂y = ∂φ/∂y = g(y) → φ = ∫g(y) dy + ψ(z)
∂U/∂z = ψ'(z) = h(z) → ψ(z) = ∫h(z) dz + C

U(x,y,z) = ∫f(x) dx + ∫g(y) dy + ∫h(z) dz + C

即**三个独立的一元函数的原函数之和**。

</details>

**10.**（思考题——非单连通区域的保守性）考虑平面向量场（视角退化到二维）：
$$\mathbf{F} = \left( \frac{-y}{x^2 + y^2},\; \frac{x}{x^2 + y^2} \right)$$
定义域为 Ω = ℝ² \ {(0,0)}（二维平面挖去原点）。

**(1)** 计算 ∇×F（在二维意义下，即 ∂Q/∂x - ∂P/∂y）。
**(2)** F 在 Ω 上是否是有势场？解释原因。
**(3)** 若将定义域限制为 Ω' = {(x,y) | x > 0}（右半平面），F 是否有势场？

<details><summary>参考答案</summary>

**(1)** P = -y/(x²+y²), Q = x/(x²+y²)
∂Q/∂x = (x²+y² - x·2x)/(x²+y²)² = (y²-x²)/(x²+y²)²
∂P/∂y = -(x²+y² - y·2y)/(x²+y²)² = -(x²-y²)/(x²+y²)² = (y²-x²)/(x²+y²)²
∂Q/∂x - ∂P/∂y = (y²-x²)/(x²+y²)² - (y²-x²)/(x²+y²)² = 0

因此在定义域内旋度处处为零。

**(2)** 不是有势场。虽然旋度为零，但 Ω = ℝ²\{(0,0)} **不是单连通区域**（平面去掉一个点，绕该点的闭曲线不能连续收缩到一点）。沿绕原点一周的单位圆 C: x²+y²=1（逆时针）计算：
$$\oint_C \mathbf{F}\cdot d\mathbf{r} = \int_0^{2\pi} (-\sin t, \cos t)\cdot(-\sin t, \cos t)\,dt = \int_0^{2\pi} (\sin^2 t + \cos^2 t)\,dt = 2\pi \neq 0$$

因此 F 在 Ω 上不是保守场。

**(3)** **是**有势场。Ω' = {x > 0} 是单连通区域（凸集，显然是单连通的），且旋度为零。势函数为 U(x,y) = \arctan(y/x)（主值），满足 ∇U = F。

**重要教训**：旋度为零只是保守的必要条件，充分性需要区域单连通。

</details>
