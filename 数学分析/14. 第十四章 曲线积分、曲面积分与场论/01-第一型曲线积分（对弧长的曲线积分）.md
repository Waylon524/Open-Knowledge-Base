## 2. 背景与应用场景

### 2.1 为什么需要"曲线上的积分"？

在第七章中，我们学习了在一维区间 $[a,b]$ 上的定积分 $\int_a^b f(x)\,dx$，在第十三章中推广到了二维区域 $D$ 上的二重积分 $\iint_D f(x,y)\,d\sigma$ 和三维区域 $\Omega$ 上的三重积分 $\iiint_\Omega f(x,y,z)\,dV$。这些积分都是在"区域"上进行的——一维的区间、二维的平面区域、三维的空间区域。

但在自然科学和工程中，我们经常需要计算定义在**一条曲线**上的累积量：

- **曲线形物体的质量**：一根细金属丝（可视为一条空间曲线）的线密度为 $\rho(x,y,z)$，如何计算其总质量？
- **电荷分布**：电荷均匀或非均匀分布在一段导线上，如何计算总电荷？
- **曲线的质心**：一根不均匀的细线，其质心位置在哪里？
- **引力计算**：一段曲线形物体对某点的引力是多少？

这些问题都不能用定积分或重积分直接解决，因为积分对象是**一维曲线**而非一维区间或二维区域。但"分割—近似—求和—取极限"的思想可以再次推广：将曲线分割成许多小弧段，在每个小弧段上用密度 $\times$ 弧长近似该小段的质量，求和后取极限。

这就是**第一型曲线积分**（line integral of the first type，也称**对弧长的曲线积分**）的核心思想。

### 2.2 从重积分到曲线积分的类比

| 积分类型 | 积分对象 | 微元含义 | 物理模型 |
|:---------|:---------|:---------|:---------|
| 定积分 $\int_a^b f\,dx$ | 一维区间 $[a,b]$ | 长度微元 $dx$ | 细杆的质量（线密度 $\times$ 长度） |
| 二重积分 $\iint_D f\,d\sigma$ | 二维区域 $D$ | 面积微元 $d\sigma$ | 薄板的质量（面密度 $\times$ 面积） |
| 三重积分 $\iiint_\Omega f\,dV$ | 三维区域 $\Omega$ | 体积微元 $dV$ | 物体的质量（体密度 $\times$ 体积） |
| **第一型曲线积分** $\int_L f\,ds$ | **一维曲线 $L$** | **弧长微元 $ds$** | **细线的质量（线密度 $\times$ 弧长）** |

### 2.3 几何意义

当 $f(x,y) \equiv 1$ 时，第一型曲线积分退化为曲线的弧长：
$$\int_L 1\,ds = \text{曲线 } L \text{ 的弧长}$$

这类似于 $\int_a^b 1\,dx = b-a$（区间长度）和 $\iint_D 1\,d\sigma = \text{Area}(D)$（区域面积）。

## 3. 核心概念与符号约定

### 3.1 第一型曲线积分的 Riemann 和定义

我们按照"分割—近似—求和—取极限"的流程来定义曲线上的积分。

**问题设定**：设 $L \subset \mathbb{R}^2$ 是一条可求长（rectifiable）的连续曲线，函数 $f(x,y)$ 在 $L$ 上有定义。我们要定义 $f$ 在 $L$ 上的累积量。

**第 1 步：分割曲线**

将曲线 $L$ 任意分割成 $n$ 个小弧段，记第 $i$ 个小弧段的长度为 $\Delta s_i$（$i = 1, 2, \dots, n$）。这里 $\Delta s_i > 0$ 表示第 $i$ 段弧的实际长度（不是投影）。

分割的**细度**（mesh）定义为所有小弧段长度的最大值：
$$\lambda = \max_{1 \leq i \leq n} \Delta s_i$$

细度 $\lambda$ 刻画了分割的精细程度——$\lambda \to 0$ 意味着每个小弧段都趋于无穷短。

**第 2 步：选取取样点**

在第 $i$ 个小弧段上任取一点 $(\xi_i, \eta_i)$ 作为**取样点**（sample point），满足 $(\xi_i, \eta_i)$ 在该小弧段上。

**第 3 步：构造 Riemann 和**

用 $f(\xi_i, \eta_i)$ 近似函数 $f$ 在第 $i$ 个小弧段上的平均值，乘以该弧段的弧长 $\Delta s_i$，然后求和：
$$S_n = \sum_{i=1}^{n} f(\xi_i, \eta_i)\,\Delta s_i$$

这个和式称为函数 $f$ 在曲线 $L$ 上的 **Riemann 和**（Riemann sum），也称为**积分和**。

**物理含义**：若 $f(x,y)$ 表示曲线在点 $(x,y)$ 处的线密度，则 $f(\xi_i, \eta_i)\,\Delta s_i$ 近似为第 $i$ 个小弧段的质量，Riemann 和近似为整条曲线的总质量。

**第 4 步：取极限**

**定义 14.1（第一型曲线积分 / Line Integral of the First Type）**：设曲线 $L \subset \mathbb{R}^2$ 是可求长曲线，函数 $f(x,y)$ 在 $L$ 上有定义。若存在实数 $I$，使得对任意 $\varepsilon > 0$，存在 $\delta > 0$，对 $L$ 的任意分割（满足细度 $\lambda < \delta$）和任意一组取样点 $(\xi_i, \eta_i)$，都有：
$$\left| \sum_{i=1}^{n} f(\xi_i, \eta_i)\,\Delta s_i - I \right| < \varepsilon$$

则称 $f$ 在曲线 $L$ 上 **Riemann 可积**（简称**可积**），$I$ 称为 $f$ 在 $L$ 上的**第一型曲线积分**（对弧长的曲线积分），记作：
$$I = \int_L f(x,y)\,ds$$

其中：
- $L$ — **积分曲线**（integration curve）
- $f(x,y)$ — **被积函数**（integrand）
- $ds$ — **弧长微元**（arc length element）

**可积性条件**：若 $L$ 是光滑或分段光滑的曲线（即可求长），且 $f$ 在 $L$ 上连续，则 $f$ 在 $L$ 上可积。在后续讨论中，我们默认曲线 $L$ 为光滑或分段光滑曲线，$f$ 在 $L$ 上连续，从而积分总是存在的。

**三维情形的定义**：对于空间曲线 $L \subset \mathbb{R}^3$，定义完全类似——只需将分割的小弧段视为三维空间中的弧段，取样点 $(\xi_i, \eta_i, \zeta_i)$，Riemann 和为 $\sum f(\xi_i, \eta_i, \zeta_i)\,\Delta s_i$，极限即为 $\int_L f(x,y,z)\,ds$。

### 3.2 第一型曲线积分与一维定积分的区别

尽管第一型曲线积分在形式上与一维定积分 $\int_a^b f(x)\,dx$ 相似（都是一维积分），但存在本质区别：

| 比较项 | 定积分 $\int_a^b f(x)\,dx$ | 第一型曲线积分 $\int_L f\,ds$ |
|:-------|:---------------------------|:------------------------------|
| 积分区域 | 数轴上的区间 $[a,b]$ | 平面或空间中的曲线 $L$ |
| 分割对象 | 将区间分成小区间 | 将曲线分成小弧段 |
| 微元 | 区间长度 $dx$ | 弧长 $ds$（依赖于曲线的形状） |
| 方向依赖性 | 积分限从小到大 | **与曲线方向无关**（因为 $ds$ 是标量长度） |
| 维数 | 被积函数 $f(x)$ 是一元的 | 被积函数 $f(x,y)$ 是二元的（或 $f(x,y,z)$ 是三元） |

**特别重要的性质**：第一型曲线积分 $\int_L f\,ds$ 的**值与曲线的定向无关**。无论是从 $A$ 到 $B$ 还是从 $B$ 到 $A$，积分值相同。这是因为 $ds$ 始终表示弧长的微小增量（正数），不依赖于方向。这一点与后续将要学习的第二型曲线积分（对坐标的曲线积分）有本质区别。

### 3.3 弧长微元 $ds$ 的参数化表达

定义 14.1 给出了曲线积分的严格框架，但按定义计算几乎不可行——我们需要一个实用的计算方法，将曲线积分转化为可计算的定积分。

关键一步是建立**弧长微元 $ds$ 与参数微分 $dt$** 之间的关系。

**第 1 步：曲线的参数化**

设光滑曲线 $L$ 由参数方程给出：
$$\begin{cases}
x = x(t) \\
y = y(t)
\end{cases},\quad a \leq t \leq b$$

其中 $x(t), y(t)$ 在 $[a,b]$ 上连续可微，且 $x'(t), y'(t)$ 不同时为零（这保证了曲线是光滑的，即处处有切向量）。

**第 2 步：切向量与弧长微元的几何关系**

曲线在参数 $t$ 处的**切向量**（tangent vector）为：
$$\mathbf{r}'(t) = (x'(t),\; y'(t))$$

当参数 $t$ 变化一个微小的量 $dt$ 时，曲线上对应的位移为：
$$d\mathbf{r} = \mathbf{r}'(t)\,dt = (x'(t)\,dt,\; y'(t)\,dt)$$

根据勾股定理，该微小位移的弧长为：
$$ds = \|d\mathbf{r}\| = \sqrt{[x'(t)\,dt]^2 + [y'(t)\,dt]^2} = \sqrt{x'(t)^2 + y'(t)^2}\;|dt|$$

由于我们通常取 $t$ 从 $a$ 到 $b$ 单调递增（$a < b$），$dt > 0$，因此绝对值可去掉，得到：

$$\boxed{ds = \sqrt{x'(t)^2 + y'(t)^2}\;dt}$$

这就是**弧长微元的参数化公式**。

**几何直观**：$\sqrt{x'(t)^2 + y'(t)^2}$ 正是切向量的模长 $\|\mathbf{r}'(t)\|$，它衡量了曲线在参数 $t$ 处的"瞬时速率"——参数每变化一个单位，曲线实际走过多长的弧。$ds = \|\mathbf{r}'(t)\|\,dt$ 的含义是：弧长微元 = 瞬时速率 $\times$ 参数微分。

**特例（函数图像的弧长微元）**：若曲线可以写成 $y = \varphi(x)$（$x$ 作为参数，$a \leq x \leq b$），则参数化取 $x(t) = t$，$y(t) = \varphi(t)$，于是 $x'(t) = 1$，$y'(t) = \varphi'(t)$，得到：
$$ds = \sqrt{1 + [\varphi'(x)]^2}\,dx$$

这正是第七章中曲线 $y = \varphi(x)$ 的弧长微元公式。

### 3.4 第一型曲线积分化为定积分的公式

利用弧长微元的参数化表达式，我们可以将第一型曲线积分转化为关于参数 $t$ 的定积分。

**定理 14.2（第一型曲线积分化定积分公式）**：设光滑曲线 $L$ 由参数方程
$$\begin{cases}
x = x(t) \\
y = y(t)
\end{cases},\quad a \leq t \leq b$$

给出，$x(t), y(t)$ 在 $[a,b]$ 上连续可微且 $x'(t), y'(t)$ 不同时为零。若 $f(x,y)$ 在 $L$ 上连续，则：
$$\boxed{\int_L f(x,y)\,ds = \int_a^b f\big(x(t),\,y(t)\big)\;\sqrt{x'(t)^2 + y'(t)^2}\;dt}$$

**推导思路**：将 Riemann 和 $\sum f(\xi_i,\eta_i)\,\Delta s_i$ 中的 $\Delta s_i$ 用 $\sqrt{x'(\tau_i)^2 + y'(\tau_i)^2}\,\Delta t_i$ 近似（其中 $\tau_i$ 是参数区间上对应第 $i$ 段的取样点），当 $\lambda \to 0$ 时，Riemann 和的极限就化为上述定积分。严格证明依赖于定积分的换元法和弧长微元的定义。

**计算模板（三步法）**：

1. **参数化**：将曲线 $L$ 用参数方程 $x = x(t),\; y = y(t)$ 表示，确定参数范围 $a \leq t \leq b$
2. **求弧长微元**：计算 $x'(t), y'(t)$，代入 $ds = \sqrt{x'(t)^2 + y'(t)^2}\;dt$
3. **代入积分**：$f(x,y)$ 中的 $x,y$ 用参数方程替换，$ds$ 用上一步的表达式替换，化为定积分并计算

### 3.5 三维空间曲线的弧长微元与曲线积分

将平面曲线的结果自然推广到三维空间。

设空间光滑曲线 $L$ 由参数方程：
$$\begin{cases}
x = x(t) \\
y = y(t) \\
z = z(t)
\end{cases},\quad a \leq t \leq b$$

其中 $x(t), y(t), z(t)$ 在 $[a,b]$ 上连续可微，且 $x'(t), y'(t), z'(t)$ 不同时为零。

**弧长微元（三维）**：由空间中的勾股定理 $ds^2 = dx^2 + dy^2 + dz^2$，代入 $dx = x'(t)dt$，$dy = y'(t)dt$，$dz = z'(t)dt$，得到曲面空间曲线的弧长微元公式：
$$\boxed{ds = \sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}\;dt}$$

**物理意义**：$\sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2} = \|\mathbf{r}'(t)\|$ 是三维切向量的模长，代表空间中运动的瞬时速率。

**三维第一型曲线积分化定积分公式**：
$$\boxed{\int_L f(x,y,z)\,ds = \int_a^b f\big(x(t),\,y(t),\,z(t)\big)\;\sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}\;dt}$$

### 3.6 第一型曲线积分的基本性质

与定积分和重积分类似，第一型曲线积分满足以下性质（由定义 14.1 直接可得）：

**性质 1（线性性质）**：
$$\int_L [\alpha f(x,y) + \beta g(x,y)]\,ds = \alpha \int_L f\,ds + \beta \int_L g\,ds$$

其中 $\alpha, \beta$ 为任意常数。

**性质 2（曲线可加性）**：若曲线 $L$ 由两段光滑曲线 $L_1$ 和 $L_2$ 首尾相接而成（$L = L_1 \cup L_2$），则：
$$\int_L f\,ds = \int_{L_1} f\,ds + \int_{L_2} f\,ds$$

**性质 3（保序性）**：若在 $L$ 上 $f(x,y) \leq g(x,y)$，则：
$$\int_L f\,ds \leq \int_L g\,ds$$

**性质 4（与曲线定向无关）**：第一型曲线积分的值与曲线的方向无关，即：
$$\int_{L_{AB}} f\,ds = \int_{L_{BA}} f\,ds$$

其中 $L_{AB}$ 和 $L_{BA}$ 表示同一条曲线但方向相反。

**性质 5（常数函数的积分）**：当 $f \equiv 1$ 时，
$$\int_L 1\,ds = \text{曲线 } L \text{ 的弧长}$$

| 符号 | 含义 | 说明 |

## 4. 原理与方法

### 4.1 标准计算流程

第一型曲线积分的计算全部遵循定理 14.2 的转化策略。以下是完整的五步流程：

**第 1 步：参数化曲线 $L$**
- 根据曲线的几何描述（直线、圆周、椭圆、一般曲线），选择适当的参数 $t$
- 写出参数方程 $x = x(t),\; y = y(t)$（平面）或 $x = x(t),\; y = y(t),\; z = z(t)$（空间）
- 确定参数范围 $a \leq t \leq b$（确保参数化覆盖整条曲线且不重复）

**第 2 步：求各分量的导数**
- 计算 $x'(t)$、$y'(t)$（三维还需 $z'(t)$）
- 检查导数是否连续且不同时为零（验证曲线的光滑性）

**第 3 步：计算弧长微元**
- $ds = \sqrt{x'(t)^2 + y'(t)^2}\;dt$（平面）或 $ds = \sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}\;dt$（空间）
- 注意：根号内是平方和，不是简单的和

**第 4 步：将被积函数用参数表示**
- 将 $f(x,y)$ 中的 $x,y$ 替换为 $x(t), y(t)$
- 得到 $f(x(t), y(t))$

**第 5 步：代入公式并计算定积分**
- $$\int_L f\,ds = \int_a^b f(x(t), y(t))\,\sqrt{x'(t)^2 + y'(t)^2}\,dt$$
- 计算定积分（使用第七章的方法）

### 4.2 参数化的常用技巧

选择合适的参数化方式是计算成功的关键。以下是一些常用参数的化方案：

| 曲线类型 | 常用参数化 | 参数范围 |
|:---------|:-----------|:---------|
| 连接 $(x_1,y_1)$ 到 $(x_2,y_2)$ 的直线段 | $\begin{cases}x = x_1 + (x_2-x_1)t \\ y = y_1 + (y_2-y_1)t\end{cases}$ | $0 \leq t \leq 1$ |
| 以 $(0,0)$ 为圆心、$R$ 为半径的圆 | $\begin{cases}x = R\cos t \\ y = R\sin t\end{cases}$ | $0 \leq t \leq 2\pi$ |
| 上半圆（$y \geq 0$） | $\begin{cases}x = R\cos t \\ y = R\sin t\end{cases}$ | $0 \leq t \leq \pi$ |
| 椭圆 $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$ | $\begin{cases}x = a\cos t \\ y = b\sin t\end{cases}$ | $0 \leq t \leq 2\pi$ |
| 函数图像 $y = \varphi(x)$ | $\begin{cases}x = t \\ y = \varphi(t)\end{cases}$ | $x_1 \leq t \leq x_2$ |
| 空间螺旋线 | $\begin{cases}x = R\cos t \\ y = R\sin t \\ z = ct\end{cases}$ | $0 \leq t \leq 2\pi$ 等 |

### 4.3 重要技巧：简化被积函数后再积分

在计算第一型曲线积分时，有时可以在代入参数化之前利用曲线的几何方程简化被积函数，使计算更加简便。

**原理**：由于曲线积分中变量 $(x,y)$ 被限制在曲线 $L$ 上，因此曲线 $L$ 的方程可以用来简化 $f(x,y)$ 的表达式。例如，若曲线 $L$ 满足 $x^2 + y^2 = R^2$，则对于 $L$ 上的点，$x^2 + y^2$ 恒等于 $R^2$，这一简化可以在代入参数化之前完成。

**示例**：
$$I = \int_L (x^2 + y^2)\,ds,\quad L: x^2 + y^2 = R^2$$

由于 $L$ 上的点都满足 $x^2 + y^2 = R^2$，我们直接得到：
$$I = \int_L R^2\,ds = R^2 \cdot (\text{圆的周长}) = R^2 \cdot 2\pi R = 2\pi R^3$$

这比分步参数化计算要简单得多。但在使用这一技巧时，必须在曲线上成立等式，不能将仅在部分曲线上成立的等式用到整条曲线。

## 5. 例题

### 例 1：直线段上的曲线积分（对应考试题 Q2）

计算曲线积分
$$I = \int_L (x+y)\,ds$$
其中 $L$ 是平面上连接点 $A(1,0)$ 和 $B(0,1)$ 的直线段。

**解**：

**第 1 步：参数化曲线**

连接 $A(1,0)$ 到 $B(0,1)$ 的直线段，取参数 $t$ 从 $0$ 到 $1$，令 $x$ 从 $1$ 线性减少到 $0$，$y$ 从 $0$ 线性增加到 $1$：
$$\begin{cases}
x = 1 - t \\
y = t
\end{cases},\quad 0 \leq t \leq 1$$

验证：$t=0$ 时 $(x,y) = (1,0) = A$，$t=1$ 时 $(x,y) = (0,1) = B$。

**第 2 步：求导数**
$$x'(t) = -1,\qquad y'(t) = 1$$

$x'(t)$ 和 $y'(t)$ 在 $[0,1]$ 上连续且不同时为零（$(-1)^2 + 1^2 = 2 > 0$），满足光滑条件。

**第 3 步：计算弧长微元**
$$ds = \sqrt{x'(t)^2 + y'(t)^2}\;dt = \sqrt{(-1)^2 + 1^2}\;dt = \sqrt{2}\;dt$$

弧长微元 $ds = \sqrt{2}\,dt$ 在整个曲线上为常数（因为直线段的切向量方向恒定，模长不变）。

**第 4 步：在曲线上写出被积函数**
$$x + y = (1 - t) + t = 1$$

被积函数在整条直线段上恒等于 $1$。这是一个重要的简化——因为 $x+y$ 在连接 $(1,0)$ 和 $(0,1)$ 的直线上为常数。

**第 5 步：代入公式并计算定积分**
$$\begin{aligned}
I &= \int_L (x+y)\,ds \\
  &= \int_0^1 \big((1-t) + t\big) \cdot \sqrt{2}\;dt \quad (\text{定理 14.2})\\
  &= \sqrt{2} \int_0^1 1\;dt \\
  &= \sqrt{2} \cdot [t]_{0}^{1} \\
  &= \sqrt{2}
\end{aligned}$$

因此 $\displaystyle I = \sqrt{2}$。

**验证（几何直觉）**：直线段 $L$ 的长度为 $\sqrt{(1-0)^2 + (0-1)^2} = \sqrt{2}$。被积函数 $x+y$ 在 $L$ 上恒为 $1$，所以积分值等于 $1 \times$ 弧长 $= \sqrt{2}$。$\checkmark$

### 例 2：上半圆周上的曲线积分（对应考试题 Q3）

计算曲线积分
$$I = \int_L y\,ds$$
其中 $L$ 是上半圆周 $x^2 + y^2 = R^2$（$y \geq 0$），起点为 $(R,0)$，终点为 $(-R,0)$。

**解**：

**第 1 步：参数化曲线**

上半圆周采用三角函数参数化（这是最自然的选择，因为圆的标准参数方程就是三角函数）：
$$\begin{cases}
x = R\cos t \\
y = R\sin t
\end{cases},\quad 0 \leq t \leq \pi$$

验证：$t=0$ 时 $(x,y) = (R,0)$（起点），$t=\pi$ 时 $(x,y) = (-R,0)$（终点）。$y = R\sin t \geq 0$ 在 $t \in [0,\pi]$ 上成立，满足上半圆周的条件。

**第 2 步：求导数**
$$x'(t) = -R\sin t,\qquad y'(t) = R\cos t$$

$x'(t)$ 和 $y'(t)$ 在 $[0,\pi]$ 上连续且不同时为零（$x'(t)^2 + y'(t)^2 = R^2(\sin^2 t + \cos^2 t) = R^2 > 0$）。

**第 3 步：计算弧长微元**
$$\begin{aligned}
ds &= \sqrt{x'(t)^2 + y'(t)^2}\;dt \\
   &= \sqrt{(-R\sin t)^2 + (R\cos t)^2}\;dt \\
   &= \sqrt{R^2(\sin^2 t + \cos^2 t)}\;dt \\
   &= \sqrt{R^2 \cdot 1}\;dt \\
   &= R\;dt
\end{aligned}$$

**关键观察**：上半圆周的弧长微元 $ds = R\,dt$ 恰好为常数。这是因为半径为 $R$ 的圆的参数化中，参数 $t$ 的几何意义是圆心角，弧长 $= R \times$ 圆心角。

**第 4 步：在曲线上写出被积函数**
$$y = R\sin t$$

**第 5 步：代入公式并计算定积分**
$$\begin{aligned}
I &= \int_L y\,ds \\
  &= \int_0^\pi (R\sin t) \cdot R\;dt \quad (\text{定理 14.2})\\
  &= R^2 \int_0^\pi \sin t\;dt
\end{aligned}$$

计算定积分 $\int_0^\pi \sin t\,dt$：
$$\int_0^\pi \sin t\,dt = \big[-\cos t\big]_{0}^{\pi} = (-\cos\pi) - (-\cos 0) = (-(-1)) - (-1) = 1 + 1 = 2$$

因此：
$$I = R^2 \cdot 2 = 2R^2$$

所以 $\displaystyle \int_L y\,ds = 2R^2$。

**验证（对称性分析）**：将上半圆周水平翻转（$x \to -x$，$y$ 不变），曲线不变，被积函数 $y$ 不变，因此积分值与翻转前相同。也可以从几何直觉理解：$y$ 在上半圆周上的平均值是 $2R/\pi$（因为 $\frac{1}{\pi}\int_0^\pi R\sin t\,dt = \frac{2R}{\pi}$），弧长为 $\pi R$，故 $I = \frac{2R}{\pi} \cdot \pi R = 2R^2$。$\checkmark$

### 例 3：空间螺旋线上的曲线积分（对应考试题 Q4）

计算曲线积分
$$I = \int_L z\,ds$$
其中 $L$ 是空间螺旋线
$$\mathbf{r}(t) = (\cos t,\ \sin t,\ t),\quad 0 \leq t \leq 2\pi$$

**解**：

**第 1 步：参数化曲线**

题目已给出螺旋线的参数方程，直接写出各分量：
$$\begin{cases}
x(t) = \cos t \\
y(t) = \sin t \\
z(t) = t
\end{cases},\quad 0 \leq t \leq 2\pi$$

**第 2 步：求导数**

对各分量分别求导：
$$x'(t) = -\sin t,\qquad y'(t) = \cos t,\qquad z'(t) = 1$$

三个导数在 $[0,2\pi]$ 上连续，且 $x'(t)^2 + y'(t)^2 + z'(t)^2 = \sin^2 t + \cos^2 t + 1 = 2 > 0$，满足光滑条件。

**第 3 步：计算弧长微元（三维）**

使用三维弧长微元公式 $ds = \sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}\;dt$：
$$\begin{aligned}
ds &= \sqrt{(-\sin t)^2 + (\cos t)^2 + 1^2}\;dt \\
   &= \sqrt{\sin^2 t + \cos^2 t + 1}\;dt \\
   &= \sqrt{1 + 1}\;dt \quad (\text{因为 } \sin^2 t + \cos^2 t = 1) \\
   &= \sqrt{2}\;dt
\end{aligned}$$

**关键观察**：螺旋线的弧长微元 $ds = \sqrt{2}\,dt$ 恰好为常数！这是因为螺旋线在 $xy$ 平面上的投影是匀速圆周运动（$x'(t)^2 + y'(t)^2 = 1$），$z$ 方向匀速上升（$z'(t) = 1$），合速度 $\sqrt{1^2 + 1^2} = \sqrt{2}$ 恒定。

**第 4 步：在曲线上写出被积函数**
$$z = t$$

**第 5 步：代入公式并计算定积分**
$$\begin{aligned}
I &= \int_L z\,ds \\
  &= \int_0^{2\pi} t \cdot \sqrt{2}\;dt \quad (\text{定理 14.2 的三维版本})\\
  &= \sqrt{2} \int_0^{2\pi} t\,dt
\end{aligned}$$

计算定积分：
$$\int_0^{2\pi} t\,dt = \left[\frac{t^2}{2}\right]_{0}^{2\pi} = \frac{(2\pi)^2}{2} - 0 = \frac{4\pi^2}{2} = 2\pi^2$$

因此：
$$I = \sqrt{2} \cdot 2\pi^2 = 2\sqrt{2}\,\pi^2$$

所以 $\displaystyle \int_L z\,ds = 2\sqrt{2}\,\pi^2$。

**验证（几何直觉）**：螺旋线 $L$ 绕了完整一圈（$t: 0 \to 2\pi$），弧长为 $\int_0^{2\pi} \sqrt{2}\,dt = 2\sqrt{2}\pi$。$z$ 从 $0$ 线性增加到 $2\pi$，平均值为 $\frac{1}{2\pi}\int_0^{2\pi} t\,dt = \pi$。因此积分值 $= \pi \cdot 2\sqrt{2}\pi = 2\sqrt{2}\pi^2$。$\checkmark$

### 例 4（补充例）：利用曲线方程简化被积函数

计算曲线积分
$$I = \int_L \frac{ds}{x^2 + y^2}$$
其中 $L$ 是圆周 $x^2 + y^2 = a^2$（$a > 0$）上从 $(a,0)$ 到 $(0,a)$ 的一段（第一象限的四分之一圆弧）。

**解法一（利用曲线方程简化后计算）**：

**第 1 步：利用曲线方程简化**

在 $L$ 上，$x^2 + y^2 = a^2$ 恒成立，因此：
$$\frac{1}{x^2 + y^2} = \frac{1}{a^2} \quad (\text{在 } L \text{ 上})$$

于是：
$$I = \int_L \frac{1}{a^2}\,ds = \frac{1}{a^2} \int_L 1\,ds = \frac{1}{a^2} \times (\text{四分之一圆弧长})$$

**第 2 步：计算弧长**

半径为 $a$ 的圆周长 $= 2\pi a$，四分之一弧长 $= \dfrac{\pi a}{2}$。

因此：
$$I = \frac{1}{a^2} \cdot \frac{\pi a}{2} = \frac{\pi}{2a}$$

**解法二（参数化解法，验证用）**：

参数化四分之一圆弧：$x = a\cos t$，$y = a\sin t$，$0 \leq t \leq \dfrac{\pi}{2}$。

$x'(t) = -a\sin t$，$y'(t) = a\cos t$，$ds = \sqrt{a^2\sin^2 t + a^2\cos^2 t}\,dt = a\,dt$。

$$I = \int_0^{\pi/2} \frac{1}{a^2\cos^2 t + a^2\sin^2 t} \cdot a\,dt = \int_0^{\pi/2} \frac{a}{a^2}\,dt = \frac{1}{a} \cdot \frac{\pi}{2} = \frac{\pi}{2a}$$

两种方法结果一致。$\checkmark$

**关键洞察**：方法一利用了"在曲线上"这个关键条件——被积函数中的 $x^2 + y^2$ 并非自由的二维变量，而是被限制在曲线 $L$ 上。这种方法在 $f(x,y)$ 可以简化为常数时尤其有效。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|:---------|:---------|
| 弧长微元 $ds$ 就是 $dt$，忽略了根号因子 | $ds = \sqrt{x'(t)^2 + y'(t)^2}\,dt$，根号因子$\sqrt{x'(t)^2 + y'(t)^2}$ 是切向量的模长，必须计算 |
| 第一型曲线积分与曲线方向有关（像定积分那样积分限从大到小会改变符号） | 第一型曲线积分与方向无关，因为 $ds$ 始终为正。从 $B$ 到 $A$ 的积分与从 $A$ 到 $B$ 的积分值相同 |
| 参数化时直接用 $x$ 作为参数而忘记写 $y'(x)$ | 若用 $x$ 作为参数（$y = \varphi(x), a \leq x \leq b$），$ds = \sqrt{1 + [\varphi'(x)]^2}\,dx$，根号内是 $1 + (\varphi')^2$ 而非 $1 + \varphi'$ |
| 将 $ds$ 误写为 $ds = \sqrt{x'(t)^2}\,dt = |x'(t)|\,dt$ | $ds$ 需要所有分量导数的平方和开方，不是单一分量。$ds = \sqrt{x'(t)^2 + y'(t)^2}\,dt$ |
| 在三维问题中仍然用平面公式 $ds = \sqrt{x'^2+y'^2}\,dt$，漏掉 $z'^2$ | 三维弧长微元为 $ds = \sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}\,dt$，$z'(t)$ 项不可遗漏 |
| $ds$ 是负的（因为 $dt$ 可以为负） | 在标准参数化中取 $t$ 递增方向，$dt > 0$；若参数化方向相反，$ds$ 公式中的 $dt$ 前应取绝对值，但第一型曲线积分与方向无关，因此无需关心方向 |
| 认为第一型曲线积分与定积分 $\int_a^b f(x)\,dx$ 一样是"区域上的积分" | 第一型曲线积分是"曲线上的积分"——积分区域是一维曲线嵌入二维/三维空间中，而非数轴上的区间 |

**1.**（定义理解）用 $\varepsilon$-$\delta$ 语言叙述函数 $f(x,y)$ 在可求长曲线 $L \subset \mathbb{R}^2$ 上第一型曲线积分可积的确切含义。

<details><summary>参考答案</summary>

存在实数 $I$，使得对任意 $\varepsilon > 0$，存在 $\delta > 0$，对曲线 $L$ 的任意分割 $\{\Delta s_i\}_{i=1}^n$（满足细度 $\lambda = \max_i \Delta s_i < \delta$）以及任意取样点 $(\xi_i, \eta_i)$ 在第 $i$ 个小弧段上，都有：
$$\left| \sum_{i=1}^{n} f(\xi_i, \eta_i)\,\Delta s_i - I \right| < \varepsilon$$

这时称 $I$ 为 $f$ 在 $L$ 上的第一型曲线积分，记作 $I = \int_L f(x,y)\,ds$。

**2.**（参数化与弧长微元）对下列曲线，写出其一种参数方程，并计算弧长微元 $ds$：

**(1)** 连接点 $(0,0)$ 和 $(3,4)$ 的直线段

**(2)** 下半圆周 $x^2 + y^2 = 1$（$y \leq 0$），从 $(-1,0)$ 到 $(1,0)$

**(3)** 空间螺旋线 $\mathbf{r}(t) = (2\cos t,\ 2\sin t,\ 3t)$，$0 \leq t \leq 4\pi$

<details><summary>参考答案</summary>

**(1)** 直线段参数化：
$$\begin{cases}
x = 3t \\
y = 4t
\end{cases},\quad 0 \leq t \leq 1$$

导数：$x'(t) = 3$，$y'(t) = 4$

弧长微元：$ds = \sqrt{3^2 + 4^2}\,dt = \sqrt{9 + 16}\,dt = 5\,dt$

**(2)** 下半圆周参数化（注意 $y \leq 0$，所以 $\sin t$ 取负号）：
$$\begin{cases}
x = \cos t \\
y = -\sin t
\end{cases},\quad 0 \leq t \leq \pi$$

验证：$t=0$ 时 $(1,0)$，$t=\pi$ 时 $(-1,0)$，$t=\pi/2$ 时 $(0,-1)$，正确覆盖下半圆周。

导数：$x'(t) = -\sin t$，$y'(t) = -\cos t$

弧长微元：$ds = \sqrt{(-\sin t)^2 + (-\cos t)^2}\,dt = \sqrt{\sin^2 t + \cos^2 t}\,dt = 1\,dt$

**(3)** 螺旋线参数方程已给出：
$$x(t) = 2\cos t,\quad y(t) = 2\sin t,\quad z(t) = 3t$$

导数：$x'(t) = -2\sin t$，$y'(t) = 2\cos t$，$z'(t) = 3$

弧长微元：
$$\begin{aligned}
ds &= \sqrt{(-2\sin t)^2 + (2\cos t)^2 + 3^2}\;dt \\
   &= \sqrt{4\sin^2 t + 4\cos^2 t + 9}\;dt \\
   &= \sqrt{4(\sin^2 t + \cos^2 t) + 9}\;dt \\
   &= \sqrt{4 + 9}\;dt = \sqrt{13}\;dt
\end{aligned}$$

**3.**（第一型曲线积分计算）计算下列曲线积分：

**(1)** $\displaystyle \int_L (x^2 + y^2)\,ds$，其中 $L$ 是连接 $(0,0)$ 到 $(2,0)$ 的直线段

**(2)** $\displaystyle \int_L (x + y)\,ds$，其中 $L$ 是连接 $(0,0)$ 到 $(1,2)$ 的直线段

**(3)** $\displaystyle \int_L \frac{ds}{x^2 + y^2 + z^2}$，其中 $L$ 是空间螺旋线 $\mathbf{r}(t) = (\cos t,\ \sin t,\ t)$，$0 \leq t \leq 2\pi$

<details><summary>参考答案</summary>

**(1)** 参数化：$x = t$，$y = 0$，$0 \leq t \leq 2$。$x'(t) = 1$，$y'(t) = 0$，$ds = \sqrt{1^2 + 0^2}\,dt = dt$。

被积函数在曲线上：$x^2 + y^2 = t^2$。

$$\int_L (x^2 + y^2)\,ds = \int_0^2 t^2 \cdot 1\,dt = \left[\frac{t^3}{3}\right]_0^2 = \frac{8}{3}$$

**(2)** 参数化：$x = t$，$y = 2t$，$0 \leq t \leq 1$。$x'(t) = 1$，$y'(t) = 2$，$ds = \sqrt{1^2 + 2^2}\,dt = \sqrt{5}\,dt$。

被积函数在曲线上：$x + y = t + 2t = 3t$。

$$\int_L (x + y)\,ds = \int_0^1 3t \cdot \sqrt{5}\,dt = 3\sqrt{5} \int_0^1 t\,dt = 3\sqrt{5} \cdot \frac{1}{2} = \frac{3\sqrt{5}}{2}$$

**(3)** 已知 $x(t) = \cos t$，$y(t) = \sin t$，$z(t) = t$，$x'(t) = -\sin t$，$y'(t) = \cos t$，$z'(t) = 1$，$ds = \sqrt{(-\sin t)^2 + (\cos t)^2 + 1^2}\,dt = \sqrt{2}\,dt$。

被积函数在曲线上：$x^2 + y^2 + z^2 = \cos^2 t + \sin^2 t + t^2 = 1 + t^2$。

$$\begin{aligned}
\int_L \frac{ds}{x^2 + y^2 + z^2} &= \int_0^{2\pi} \frac{1}{1 + t^2} \cdot \sqrt{2}\,dt \\
&= \sqrt{2} \int_0^{2\pi} \frac{dt}{1 + t^2} \\
&= \sqrt{2} \big[\arctan t\big]_{0}^{2\pi} \\
&= \sqrt{2} (\arctan 2\pi - \arctan 0) \\
&= \sqrt{2} \arctan 2\pi
\end{aligned}$$

</details>

### 迁移应用

**4.**（利用曲线方程简化计算）计算 $\displaystyle \int_L (x^2 + y^2)\,ds$，其中 $L$ 是圆周 $x^2 + y^2 = 4$ 上从 $(2,0)$ 逆时针回到 $(2,0)$ 的整个圆周。

<details><summary>参考答案</summary>

**方法一（利用曲线方程简化）**：

在 $L$ 上，$x^2 + y^2 = 4$ 恒成立。因此：
$$\int_L (x^2 + y^2)\,ds = \int_L 4\,ds = 4 \times (\text{圆周长}) = 4 \times (2\pi \times 2) = 4 \times 4\pi = 16\pi$$

**方法二（参数化解法验证）**：

参数化：$x = 2\cos t$，$y = 2\sin t$，$0 \leq t \leq 2\pi$。

$x'(t) = -2\sin t$，$y'(t) = 2\cos t$，$ds = \sqrt{4\sin^2 t + 4\cos^2 t}\,dt = 2\,dt$。

$$x^2 + y^2 = 4\cos^2 t + 4\sin^2 t = 4$$

$$\int_0^{2\pi} 4 \cdot 2\,dt = 8 \cdot 2\pi = 16\pi \quad \checkmark$$

注意：方法一利用曲线方程 $x^2 + y^2 = R^2$ 直接将被积函数替换为常数 $R^2$，大大简化了计算。这种技巧在 $f(x,y)$ 可以用曲线方程化简时非常有效。

</details>

**5.**（分段曲线）计算 $\displaystyle \int_L (x+y)\,ds$，其中 $L$ 是由点 $A(0,0)$ 沿直线到 $B(1,0)$，再沿直线到 $C(1,1)$ 的折线路径。

<details><summary>参考答案</summary>

**第 1 步：分解曲线**

折线 $L = L_1 \cup L_2$，其中：
- $L_1$：从 $A(0,0)$ 到 $B(1,0)$ 的水平线段
- $L_2$：从 $B(1,0)$ 到 $C(1,1)$ 的竖直线段

由曲线可加性：
$$\int_L (x+y)\,ds = \int_{L_1} (x+y)\,ds + \int_{L_2} (x+y)\,ds$$

**第 2 步：计算 $L_1$ 上的积分**

参数化 $L_1$：$x = t$，$y = 0$，$0 \leq t \leq 1$。

$x'(t) = 1$，$y'(t) = 0$，$ds = \sqrt{1^2 + 0^2}\,dt = dt$。

被积函数在 $L_1$ 上：$x + y = t + 0 = t$。

$$\int_{L_1} (x+y)\,ds = \int_0^1 t \cdot 1\,dt = \left[\frac{t^2}{2}\right]_0^1 = \frac{1}{2}$$

**第 3 步：计算 $L_2$ 上的积分**

参数化 $L_2$：$x = 1$，$y = t$，$0 \leq t \leq 1$。

$x'(t) = 0$，$y'(t) = 1$，$ds = \sqrt{0^2 + 1^2}\,dt = dt$。

被积函数在 $L_2$ 上：$x + y = 1 + t$。

$$\int_{L_2} (x+y)\,ds = \int_0^1 (1 + t) \cdot 1\,dt = \left[t + \frac{t^2}{2}\right]_0^1 = 1 + \frac{1}{2} = \frac{3}{2}$$

**第 4 步：求和**

$$\int_L (x+y)\,ds = \frac{1}{2} + \frac{3}{2} = 2$$

因此 $\displaystyle \int_L (x+y)\,ds = 2$。

**重要提示**：分段光滑曲线的曲线积分等于各段光滑曲线上的曲线积分之和。在计算中，每一段都需要独立进行参数化，不能试图用统一的参数方程描述整条折线（因为折线的角点处导数不连续）。

</details>

**6.**（思考题）设曲线 $L$ 是圆 $x^2 + y^2 = R^2$ 上的一段弧，$R > 0$。证明：若 $f$ 是 $L$ 上的连续函数，则第一型曲线积分 $\int_L f(x,y)\,ds$ 的值与 $L$ 的起点和终点的选择无关（仅与曲线本身有关）。

<details><summary>参考答案</summary>

**证明**：

设曲线 $L$ 是圆上的某一段固定弧（不含整圆），对其给定两种不同的方向：
- $L_1$：从 $A$ 到 $B$（取参数 $t$ 从 $a$ 到 $b$，$a < b$）
- $L_2$：从 $B$ 到 $A$（取参数 $t$ 从 $b$ 到 $a$）

对 $L_1$：参数化 $(x(t), y(t))$，$a \leq t \leq b$，$ds = \sqrt{x'(t)^2 + y'(t)^2}\,dt$。

对 $L_2$：用同一参数化但参数方向相反。令 $t = a + b - \tau$（$\tau$ 从 $a$ 到 $b$），则：
$$x(\tau) = x(a+b-\tau),\quad y(\tau) = y(a+b-\tau)$$

由链式法则：
$$\frac{dx}{d\tau} = -x'(a+b-\tau),\quad \frac{dy}{d\tau} = -y'(a+b-\tau)$$

弧长微元：
$$ds_{L_2} = \sqrt{(-x')^2 + (-y')^2}\;d\tau = \sqrt{x'^2 + y'^2}\;d\tau$$

因此 $ds$ 的表达式在两种方向下完全相同（因为平方项消除了符号）。于是：
$$\int_{L_2} f\,ds = \int_a^b f(x(\tau), y(\tau))\,\sqrt{x'^2 + y'^2}\;d\tau = \int_a^b f(x(t), y(t))\,\sqrt{x'^2 + y'^2}\;dt = \int_{L_1} f\,ds$$

这就证明了第一型曲线积分的值与曲线方向无关。这个结论对任意光滑曲线（不仅是圆）都成立。

**直观理解**：$ds$ 是弧长微元，表示"一小段曲线的长度"。不论你从哪个方向走，这段曲线的长度是一样的。因此，基于弧长的累积量（总质量、总电荷等）不依赖于测量方向。

</details>
