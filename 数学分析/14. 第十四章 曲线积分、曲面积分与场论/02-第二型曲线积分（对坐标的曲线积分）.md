## 2. 背景与应用场景

### 2.1 物理背景：变力沿曲线所做的功

在自然界和工程中，我们经常需要计算**变力沿曲线所做的功**。

考虑一个质点在平面力场 $\mathbf{F}(x,y) = (P(x,y), Q(x,y))$ 中沿一条有向曲线 $L$ 从起点 $A$ 运动到终点 $B$。力场 $\mathbf{F}$ 的大小和方向随位置 $(x,y)$ 变化（例如重力场、电场、磁场）。如何计算这个力对质点所做的总功？

**微元法的思路**：在曲线 $L$ 上取一个微小的有向弧段 $d\mathbf{r} = (dx, dy)$。在该微小弧段上，力 $\mathbf{F}$ 近似为常量，所做的微功为力与位移的点积：
$$dW = \mathbf{F} \cdot d\mathbf{r} = P\,dx + Q\,dy$$

**几何含义**：只有力的切线方向分量做功。法线方向分量与位移垂直，不做功。$P\,dx + Q\,dy$ 恰好表达了力在切线方向上的贡献——$P$ 与 $x$ 方向位移 $dx$ 的乘积加上 $Q$ 与 $y$ 方向位移 $dy$ 的乘积。

**总功**：将所有微小弧段上的微功累积起来，得到：
$$W = \int_L \mathbf{F} \cdot d\mathbf{r} = \int_L P\,dx + Q\,dy$$

这就是第二型曲线积分最核心的物理模型。与第一型曲线积分（质量、电荷等标量累积）的本质区别在于：这里累积的是**有向的**微元贡献——位移的方向至关重要，方向不同功可能为正、为负或为零。

**更广泛的应用**：除了变力做功，第二型曲线积分还出现在：
- **环量**（circulation）：向量场沿闭合曲线的积分 $\oint_L \mathbf{F} \cdot d\mathbf{r}$，描述场的旋转强度
- **电动势**：电场对电荷所做的功
- **通量**（flux）：向量场穿过曲线的流量（后续曲面积分中进一步讨论）

### 2.2 第一型与第二型的对比

| 比较项 | 第一型 $\int_L f\,ds$ | 第二型 $\int_L P\,dx + Q\,dy$ |
|:-------|:----------------------|:-----------------------------|
| 微元 | $ds$（弧长，始终为正的标量） | $dx, dy$（有向投影，可正可负） |
| 被积函数 | 单个标量函数 $f(x,y)$ | 两个函数 $P(x,y), Q(x,y)$ 分别与 $dx, dy$ 配对 |
| 方向依赖性 | **与方向无关**（$ds$ 是标量） | **方向反转时变号**（$dx,dy$ 均变号） |
| 参数化公式 | $\int f\sqrt{x'^2+y'^2}\,dt$（有根号） | $\int (P\,x' + Q\,y')\,dt$（无根号，线性因子） |
| 物理模型 | 质量、电荷、弧长等标量累积 | 变力做功、环量等向量场累积 |
| 几何意义 | 以曲线为底、$f$ 为高的"曲线帷幕"面积 | 向量场沿有向曲线的"总效果" |

### 2.3 从第一型的承诺到第二型的实现

在文件 01（第十四章）的 3.2 节中，我们在讨论第一型曲线积分的方向无关性时，特别指出"这与后续将要学习的第二型曲线积分有本质区别"。现在，这个区别正式登场。

**核心差异一句话**：第一型曲线积分累积的是标量（与方向无关），第二型曲线积分累积的是有向量的投影（与方向密切有关）。

## 3. 核心概念与符号约定

### 3.1 第二型曲线积分的 Riemann 和定义

我们沿用"分割—近似—求和—取极限"的基本框架，但微元的含义发生根本变化——从弧长变为有向投影。

**问题设定**：设 $L \subset \mathbb{R}^2$ 是一条有向光滑曲线（指定了从起点到终点的方向），函数 $P(x,y)$ 和 $Q(x,y)$ 在 $L$ 上有定义。定义曲线积分 $\int_L P\,dx + Q\,dy$。

**第 1 步：分割曲线并确定有向投影**

将曲线 $L$ 按指定方向依次分割为 $n$ 个小弧段，分点依次为
$$A = M_0(x_0,y_0),\; M_1(x_1,y_1),\; \dots,\; M_n(x_n,y_n) = B$$

**关键概念——有向投影**：第 $i$ 个小弧段 $M_{i-1}M_i$ 在 $x$ 轴和 $y$ 轴上的有向投影（带符号的坐标变化量）定义为：
$$\Delta x_i = x_i - x_{i-1},\qquad \Delta y_i = y_i - y_{i-1}$$

与第一型曲线积分中的弧长 $\Delta s_i > 0$ 不同：
- $\Delta x_i$ **可正可负**——若沿 $x$ 正向走则 $\Delta x_i > 0$，沿 $x$ 负向走则 $\Delta x_i < 0$
- $\Delta y_i$ **可正可负**——若沿 $y$ 正向走则 $\Delta y_i > 0$，沿 $y$ 负向走则 $\Delta y_i < 0$
- 只有当曲线段在 $x$ 方向上没有变化时 $\Delta x_i = 0$（同理 $\Delta y_i = 0$）

分割的**细度**（mesh）定义为所有小弧段弦长的最大值：
$$\lambda = \max_{1 \leq i \leq n} \sqrt{\Delta x_i^2 + \Delta y_i^2}$$

细度 $\lambda$ 描述分割的精细程度——$\lambda \to 0$ 意味着每个小弧段都趋于一点（弦长趋于零）。

**与第一型的细度对比**：第一型用 $\lambda = \max \Delta s_i$（最大弧长），第二型用 $\lambda = \max \sqrt{\Delta x_i^2 + \Delta y_i^2}$（最大弦长）。两者在光滑曲线上的极限行为等价（弧长和弦长在细分时趋于一致），但第二型细度更自然地与有向微元 $\Delta x_i, \Delta y_i$ 对应。

**第 2 步：选取取样点**

在第 $i$ 个小弧段 $M_{i-1}M_i$ 上任取一点 $(\xi_i, \eta_i)$ 作为**取样点**。

**第 3 步：构造 Riemann 和**

在第 $i$ 个小弧段上，用 $P(\xi_i,\eta_i)$ 近似该段上的 $P$ 函数值，用 $Q(\xi_i,\eta_i)$ 近似 $Q$ 函数值，然后用 $P(\xi_i,\eta_i)\Delta x_i + Q(\xi_i,\eta_i)\Delta y_i$ 近似该段上的贡献（例如微小功），求和得到：
$$S_n = \sum_{i=1}^{n} \big[P(\xi_i,\eta_i)\,\Delta x_i + Q(\xi_i,\eta_i)\,\Delta y_i\big]$$

这个和式称为函数 $P,Q$ 在曲线 $L$ 上的 **Riemann 和**（积分和）。

**物理含义**：若 $\mathbf{F} = (P,Q)$ 表示力场，则 $P(\xi_i,\eta_i)\Delta x_i + Q(\xi_i,\eta_i)\Delta y_i$ 近似为第 $i$ 个小弧段上力所做的微小功，Riemann 和近似为整个路径上的总功。

**第 4 步：取极限**

**定义 14.3（第二型曲线积分 / Line Integral of the Second Type）**：设有向光滑曲线 $L \subset \mathbb{R}^2$，函数 $P(x,y), Q(x,y)$ 在 $L$ 上有定义。若存在实数 $I$，使得对任意 $\varepsilon > 0$，存在 $\delta > 0$，对 $L$ 的任意分割（满足细度 $\lambda < \delta$）和任意一组取样点 $(\xi_i, \eta_i)$，都有：
$$\left| \sum_{i=1}^{n} \big[ P(\xi_i,\eta_i)\,\Delta x_i + Q(\xi_i,\eta_i)\,\Delta y_i \big] - I \right| < \varepsilon$$

则称 $P,Q$ 在 $L$ 上 **Riemann 可积**（简称**可积**），$I$ 称为 $P\,dx + Q\,dy$ 在 $L$ 上的**第二型曲线积分**（对坐标的曲线积分），记作：
$$I = \int_L P(x,y)\,dx + Q(x,y)\,dy$$

其中：
- $L$ — **有向积分曲线**（oriented integration curve）
- $P(x,y)$ — **$x$ 方向的被积函数**
- $Q(x,y)$ — **$y$ 方向的被积函数**
- $dx, dy$ — **有向投影微元**（signed projection elements）

**三维情形**：对于空间曲线 $L \subset \mathbb{R}^3$，第二型曲线积分自然推广为：
$$\int_L P\,dx + Q\,dy + R\,dz$$

其中 $R(x,y,z)$ 是 $z$ 方向的分量函数，$dz$ 是 $z$ 方向的有向投影微元。

**可积性条件**：若 $L$ 是光滑或分段光滑的有向曲线，且 $P,Q$ 在 $L$ 上连续，则 $P,Q$ 在 $L$ 上可积。在后续讨论中，我们默认曲线 $L$ 满足这些条件。

### 3.2 方向反转与积分变号

第二型曲线积分最核心的特点——方向依赖性——可以直接从定义中读出。

考虑同一条曲线 $L$ 的两种定向：
- $L_{AB}$：从 $A$ 到 $B$ 的正向
- $L_{BA} = L_{AB}^{-}$：从 $B$ 到 $A$ 的反向（上标 $-$ 表示反向）

**定理（方向反转性质）**：第二型曲线积分在方向反转时变号：
$$\int_{L_{BA}} P\,dx + Q\,dy = -\int_{L_{AB}} P\,dx + Q\,dy$$

**证明**：设分割 $A = M_0, M_1, \dots, M_n = B$ 满足细度 $\lambda < \delta$。对 $L_{BA}$，相同的分点但顺序反转为 $B = M_n, M_{n-1}, \dots, M_0 = A$。第 $i$ 个小弧段 $M_{i}M_{i-1}$（从 $M_i$ 到 $M_{i-1}$）的有向投影为：
$$\Delta x'_i = x_{i-1} - x_i = -(x_i - x_{i-1}) = -\Delta x_i$$
$$\Delta y'_i = y_{i-1} - y_i = -(y_i - y_{i-1}) = -\Delta y_i$$

在 $L_{BA}$ 上的 Riemann 和为：
$$\begin{aligned}
S'_n &= \sum_{i=1}^{n} \big[ P(\xi_i,\eta_i)\,(-\Delta x_i) + Q(\xi_i,\eta_i)\,(-\Delta y_i) \big] \\
&= -\sum_{i=1}^{n} \big[ P(\xi_i,\eta_i)\,\Delta x_i + Q(\xi_i,\eta_i)\,\Delta y_i \big] \\
&= -S_n
\end{aligned}$$

取极限 $\lambda \to 0$，由于极限与分割和取样点无关，$S_n \to I$，因此 $S'_n \to -I$，即：
$$\int_{L_{BA}} P\,dx + Q\,dy = -I = -\int_{L_{AB}} P\,dx + Q\,dy$$

证毕。

**与第一型的本质区别**：对第一型曲线积分，$ds$ 是弧长（标量），方向反转时 $ds$ 不变，因此积分值不变。对第二型曲线积分，$dx, dy$ 是有向投影，方向反转时均变号，因此积分值变号。

### 3.3 有向投影微元 $dx, dy$ 的参数化表达

定义 14.3 给出了第二型曲线积分的严格框架，但按定义直接计算几乎不可能。与第一型曲线积分类似，我们需要通过参数化将曲线积分转化为定积分。

**第 1 步：曲线的有向参数化**

设有向光滑曲线 $L$ 由参数方程给出：
$$\begin{cases}
x = x(t) \\
y = y(t)
\end{cases},\quad a \leq t \leq b$$

其中 $x(t), y(t)$ 在 $[a,b]$ 上连续可微，且 $x'(t), y'(t)$ 不同时为零。参数 $t$ 从 $a$ 到 $b$ 的递增方向**对应**曲线的指定方向（这是我们选择参数化的基本要求）。

**第 2 步：有向投影微元 $dx, dy$**

根据微分学（第四章），函数 $x(t)$ 的微分是：
$$dx = x'(t)\,dt$$

同样地：
$$dy = y'(t)\,dt$$

这是**有向投影微元 $dx, dy$ 的参数化表达式**。

**关键理解**：
- $dx$ 不是"一个无穷小正数"，而是有向投影——当 $x'(t) > 0$ 时 $dx$ 与 $dt$ 同号，当 $x'(t) < 0$ 时 $dx$ 与 $dt$ 异号
- 由于参数 $t$ 从 $a$ 到 $b$ 递增，$dt > 0$，因此 $dx$ 的符号完全由 $x'(t)$ 决定
- 这里 $dx$ 的用法与一维定积分 $\int_a^b f(x)\,dx$ 中的 $dx$ 完全一致——都是"自变量微小变化"的微分

**第 3 步：$dx, dy$ 与 $ds$ 的关系**

回忆第一型曲线积分的弧长微元：
$$ds = \sqrt{x'(t)^2 + y'(t)^2}\;dt \quad (>0)$$

而 $dx = x'(t)dt$，$dy = y'(t)dt$，三者满足勾股关系：
$$ds^2 = dx^2 + dy^2$$

具体验证：
$$dx^2 + dy^2 = [x'(t)]^2 dt^2 + [y'(t)]^2 dt^2 = [x'(t)^2 + y'(t)^2]\,dt^2 = (ds)^2$$

**总结对比**：

| 微元 | 表达式 | 符号 |
|:-----|:-------|:-----|
| $ds$ | $\sqrt{x'(t)^2 + y'(t)^2}\,dt$ | 始终为正 |
| $dx$ | $x'(t)\,dt$ | 可为正、负或零 |
| $dy$ | $y'(t)\,dt$ | 可为正、负或零 |

**关系**：$ds = \sqrt{dx^2 + dy^2}$，$ds$ 是 $dx, dy$ 在几何上的"合成长度"。

### 3.4 第二型曲线积分的向量表示

将第二型曲线积分写成向量形式，既能简化记号，也突出了物理含义。

定义**向量场**（vector field）和**有向线微元**（oriented line element）：
$$\mathbf{F}(x,y) = (P(x,y),\; Q(x,y)),\qquad d\mathbf{r} = (dx,\; dy)$$

则第二型曲线积分可简洁地写为：
$$\int_L P\,dx + Q\,dy = \int_L \mathbf{F} \cdot d\mathbf{r}$$

在参数化下，$d\mathbf{r} = \mathbf{r}'(t)\,dt = (x'(t),\, y'(t))\,dt$，因此：
$$\int_L \mathbf{F} \cdot d\mathbf{r} = \int_a^b \mathbf{F}(x(t),y(t)) \cdot \mathbf{r}'(t)\,dt = \int_a^b \big[ P\,x'(t) + Q\,y'(t) \big]\,dt$$

**物理含义**：$\mathbf{F} \cdot d\mathbf{r}$ 是力 $\mathbf{F}$ 与位移 $d\mathbf{r}$ 的点积，只有在切线方向的分量做功。如果 $\mathbf{F}$ 始终垂直于曲线（例如洛伦兹力始终垂直于速度方向），则 $\mathbf{F} \cdot d\mathbf{r} \equiv 0$，做功为零。

### 3.5 转化公式：第二型曲线积分化为定积分

**定理 14.4（第二型曲线积分化定积分公式）**：设有向光滑曲线 $L$ 由参数方程
$$\begin{cases}
x = x(t) \\
y = y(t)
\end{cases},\quad a \leq t \leq b$$

给出，$x(t), y(t)$ 在 $[a,b]$ 上连续可微且 $x'(t), y'(t)$ 不同时为零。若 $P(x,y), Q(x,y)$ 在 $L$ 上连续，则：
$$\boxed{\displaystyle \int_L P\,dx + Q\,dy = \int_a^b \big[ P(x(t),y(t))\,x'(t) + Q(x(t),y(t))\,y'(t) \big]\,dt}$$

**推导**：从第二型曲线积分的 Riemann 和定义出发。

**第 1 步**：将参数区间 $[a,b]$ 任意分割为 $n$ 份：
$$a = t_0 < t_1 < \cdots < t_n = b,\quad \Delta t_i = t_i - t_{i-1}$$

对应地将曲线 $L$ 分割为 $n$ 个小弧段，分点为 $M_i = (x(t_i), y(t_i))$。

**第 2 步**：第 $i$ 个小弧段 $M_{i-1}M_i$ 的有向投影为：
$$\Delta x_i = x(t_i) - x(t_{i-1}),\quad \Delta y_i = y(t_i) - y(t_{i-1})$$

由微分中值定理，存在 $\tau_i \in (t_{i-1}, t_i)$ 和 $\tau'_i \in (t_{i-1}, t_i)$ 使得：
$$\Delta x_i = x'(\tau_i)\,\Delta t_i,\quad \Delta y_i = y'(\tau'_i)\,\Delta t_i$$

**第 3 步**：在取样点 $(\xi_i, \eta_i) = (x(\tau_i), y(\tau_i))$ 处构造 Riemann 和（为简化推导，$P$ 用 $\tau_i$ 处的值近似，$Q$ 也用 $\tau_i$ 处的值近似——由于连续性，当 $\Delta t_i$ 很小时 $\tau_i$ 和 $\tau'_i$ 趋于同一点）：
$$\begin{aligned}
S_n &= \sum_{i=1}^{n} \big[ P(x(\tau_i), y(\tau_i))\,\Delta x_i + Q(x(\tau_i), y(\tau_i))\,\Delta y_i \big] \\
&\approx \sum_{i=1}^{n} \big[ P(x(\tau_i), y(\tau_i))\,x'(\tau_i)\,\Delta t_i + Q(x(\tau_i), y(\tau_i))\,y'(\tau_i)\,\Delta t_i \big]
\end{aligned}$$

**第 4 步**：当细度 $\lambda \to 0$（等价于 $\max \Delta t_i \to 0$）时，近似误差趋于零，Riemann 和的极限即为定积分：
$$\lim_{\lambda \to 0} S_n = \int_a^b \big[ P(x(t), y(t))\,x'(t) + Q(x(t), y(t))\,y'(t) \big]\,dt$$

根据定义 14.3，极限值就是 $\int_L P\,dx + Q\,dy$，定理得证。

**与第一型转化公式的结构对比**：

| 公式 | 表达式 | 特点 |
|:-----|:-------|:-----|
| 第一型 $\int_L f\,ds$ | $\int_a^b f(x,y)\,\sqrt{x'^2 + y'^2}\,dt$ | 根号因子 $\sqrt{x'^2 + y'^2}$，被积函数为单个标量 |
| 第二型 $\int_L P\,dx + Q\,dy$ | $\int_a^b \big[ P\,x' + Q\,y' \big]\,dt$ | 无根号，被积函数是两个项的和，每项含被积函数乘以对应分量的导数 |

**本质差异**：第一型的微元 $ds$ 是标量长度，参数化后产生根号因子（切向量模长）；第二型的微元 $dx, dy$ 是定向投影，参数化后产生线性因子 $x'(t), y'(t)$，没有根号，且 $x', y'$ 可正可负。

### 3.6 第二型曲线积分的基本性质

与第一型曲线积分类似，第二型曲线积分也满足以下性质（由定义 14.3 可得）：

**性质 1（线性性质）**：
$$\int_L \big[ \alpha P_1 + \beta P_2 \big]\,dx + \big[ \alpha Q_1 + \beta Q_2 \big]\,dy = \alpha \int_L P_1\,dx + Q_1\,dy + \beta \int_L P_2\,dx + Q_2\,dy$$

其中 $\alpha, \beta$ 为任意常数。

**性质 2（曲线可加性）**：若有向曲线 $L$ 由两段有向光滑曲线 $L_1$ 和 $L_2$ 首尾相接而成（$L = L_1 \cup L_2$，方向保持一致），则：
$$\int_L P\,dx + Q\,dy = \int_{L_1} P\,dx + Q\,dy + \int_{L_2} P\,dx + Q\,dy$$

**性质 3（方向反转变号）**：
$$\int_{L^-} P\,dx + Q\,dy = -\int_L P\,dx + Q\,dy$$

这是第二型曲线积分区别于第一型的核心性质。

**性质 4（向量形式的线性）**：第二型曲线积分可以分解为两个独立的线积分的和：
$$\int_L P\,dx + Q\,dy = \int_L P\,dx + \int_L Q\,dy$$

注意：等式右边的两个积分分别是对 $dx$ 和对 $dy$ 的线积分，它们各自的有限存在性已经在定义 14.3 的整体框架中保证。

### 3.7 符号表

| 符号 | 含义 | 说明 |
|:-----|:-----|:-----|
| $\int_L P\,dx + Q\,dy$ | 第二型曲线积分（对坐标的曲线积分） | $L$ 为有向平面曲线 |
| $P(x,y)$ | $x$ 方向的被积函数 | 第一分量（力场中的 $x$ 分量） |
| $Q(x,y)$ | $y$ 方向的被积函数 | 第二分量（力场中的 $y$ 分量） |

## 4. 原理与方法

### 4.1 标准计算流程（三步法）

第二型曲线积分的计算全部遵循定理 14.4 的转化策略。以下是完整的三步流程：

**第 1 步：有向参数化曲线 $L$**
- 根据曲线的几何描述，选择适当的参数 $t$，写出参数方程 $x = x(t),\; y = y(t)$
- **必须保证参数 $t$ 递增方向与曲线的指定方向一致**——这是最关键的一步
- 确定参数范围 $a \leq t \leq b$
- 检查 $x(t), y(t)$ 在 $[a,b]$ 上连续可微且 $x'(t), y'(t)$ 不同时为零

**第 2 步：求有向投影微元**
- 计算 $x'(t)$ 和 $y'(t)$
- 写出 $dx = x'(t)\,dt$ 和 $dy = y'(t)\,dt$
- 注意：$x'(t)$ 可以是负数，此时 $dx$ 与 $dt$ 异号

**第 3 步：代入定理 14.4 并计算定积分**
- 将 $P(x,y)$ 中的 $x,y$ 替换为 $x(t), y(t)$
- 将 $Q(x,y)$ 中的 $x,y$ 替换为 $x(t), y(t)$
- 代入公式 $\displaystyle \int_L P\,dx + Q\,dy = \int_a^b \big[ P(x(t),y(t))\,x'(t) + Q(x(t),y(t))\,y'(t) \big]\,dt$
- 计算定积分

### 4.2 方向依赖性与路径依赖性

**方向依赖性**（已在 3.2 节中严格证明）：
$$\int_{L^-} P\,dx + Q\,dy = -\int_L P\,dx + Q\,dy$$

**实用含义**：在参数化时，如果选择的参数 $t$ 方向与曲线指定方向相反，则需要在定积分前加负号来修正。最安全的做法是：先按参数 $t$ 递增方向计算积分，然后根据方向是否与曲线一致决定是否取负。

更一般地，如果参数化 $\tilde{x} = x(\phi(\tau)),\; \tilde{y} = y(\phi(\tau))$ 中 $\phi(\tau)$ 是**递减**函数（即参数变换反转了方向），则：
$$\int_a^b [P x' + Q y']\,dt = -\int_{\tau(a)}^{\tau(b)} [P \tilde{x}_\tau + Q \tilde{y}_\tau]\,d\tau$$

这一结论可以通过定积分的换元法得到验证。

**路径依赖性（Path Dependence）**：第二型曲线积分的一个本质特征是，从同一点 $A$ 到同一点 $B$ 沿不同路径的积分值通常不同。这一点与第一型曲线积分截然不同（第一型只依赖于曲线本身，不依赖于路径选择）。

**为什么路径相关**？第二型曲线积分 $\int_L \mathbf{F} \cdot d\mathbf{r}$ 累积的是向量场沿有向曲线的"投影和"。不同的路径意味着不同的切线方向和位移分布——即使起点终点相同，力与路径的相互作用也会不同。

**示例直觉**：想象一个人爬山，重力始终竖直向下。如果走一条缓坡上山，重力在沿山路方向的分量较小，克服重力做的负功（重力做负功）较少；如果走一条陡坡上山，重力在沿山路方向的分量较大，克服重力做的负功较多。起点（山脚）和终点（山顶）相同，但功不同。

**重要备注**：路径依赖性并不意味着路径总是相关的。在某些特殊条件（如保守力场）下，第二型曲线积分会变得与路径无关——这将在后续章节（如 Green 定理、势函数）中深入讨论。当前阶段，我们默认第二型曲线积分依赖于路径。

## 5. 例题

### 例 1：路径依赖性——不同路径产生不同结果（对应考试题 Q2）

考虑力场 $\mathbf{F}(x,y) = (0,\; x^2)$，即 $P(x,y) = 0$，$Q(x,y) = x^2$。计算质点沿以下两条不同路径从 $A(0,0)$ 到 $B(1,1)$ 力场所做的功：
- 路径 $L_1$：直线 $y = x$
- 路径 $L_2$：抛物线 $y = x^2$

通过计算验证：第二型曲线积分是否依赖于路径？

---

#### 路径 $L_1$：$y = x$（直线段）

**第 1 步：有向参数化**

直线 $y = x$ 从 $(0,0)$ 到 $(1,1)$，取 $x$ 作为参数：
$$x = t,\quad y = t,\quad 0 \leq t \leq 1$$

参数 $t$ 从 $0$ 到 $1$ 递增时，曲线从 $(0,0)$ 走向 $(1,1)$，与指定方向一致。$\checkmark$

验证光滑性：$x'(t) = 1$，$y'(t) = 1$，在 $[0,1]$ 上连续且 $1^2 + 1^2 = 2 > 0$。

**第 2 步：求有向投影微元**
$$dx = x'(t)\,dt = 1 \cdot dt = dt,\qquad dy = y'(t)\,dt = 1 \cdot dt = dt$$

**第 3 步：代入定理 14.4**

$P(x,y) = 0$，$Q(x,y) = x^2$。用参数表达：
$$P(x(t), y(t)) = 0,\qquad Q(x(t), y(t)) = x(t)^2 = t^2$$

$$\begin{aligned}
W_1 &= \int_{L_1} P\,dx + Q\,dy \\
    &= \int_{0}^{1} \big[ 0 \cdot 1 + t^2 \cdot 1 \big]\,dt \quad (\text{定理 14.4}) \\
    &= \int_{0}^{1} t^2\,dt \\
    &= \left[ \frac{t^3}{3} \right]_{0}^{1} \\
    &= \frac{1}{3}
\end{aligned}$$

因此沿直线 $y = x$ 所做的功 $W_1 = \dfrac{1}{3}$。

---

#### 路径 $L_2$：$y = x^2$（抛物线）

**第 1 步：有向参数化**

抛物线 $y = x^2$ 从 $(0,0)$ 到 $(1,1)$，取 $x$ 作为参数：
$$x = t,\quad y = t^2,\quad 0 \leq t \leq 1$$

参数 $t$ 递增时，曲线从 $(0,0)$ 走向 $(1,1)$，方向一致。$\checkmark$

验证光滑性：$x'(t) = 1$，$y'(t) = 2t$，在 $[0,1]$ 上连续且 $1^2 + (2t)^2 \geq 1 > 0$。

**第 2 步：求有向投影微元**
$$dx = x'(t)\,dt = dt,\qquad dy = y'(t)\,dt = 2t\,dt$$

**第 3 步：代入定理 14.4**

$P(x,y) = 0$，$Q(x,y) = x^2$。用参数表达：
$$P(x(t), y(t)) = 0,\qquad Q(x(t), y(t)) = x(t)^2 = t^2$$

$$\begin{aligned}
W_2 &= \int_{L_2} P\,dx + Q\,dy \\
    &= \int_{0}^{1} \big[ 0 \cdot 1 + t^2 \cdot (2t) \big]\,dt \quad (\text{定理 14.4}) \\
    &= \int_{0}^{1} 2t^3\,dt \\
    &= 2 \cdot \left[ \frac{t^4}{4} \right]_{0}^{1} \\
    &= 2 \cdot \frac{1}{4} = \frac{1}{2}
\end{aligned}$$

因此沿抛物线 $y = x^2$ 所做的功 $W_2 = \dfrac{1}{2}$。

---

#### 结果分析

$$W_1 = \frac{1}{3} \neq \frac{1}{2} = W_2$$

**结论**：同一起点 $(0,0)$ 到同一终点 $(1,1)$，在相同的力场 $\mathbf{F} = (0, x^2)$ 中，沿不同路径所做的功不同。这验证了第二型曲线积分的**路径依赖性**（path dependence）。

**与第一型的对比**：在第一型曲线积分中，$\int_L f\,ds$ 的值与路径的形状有关但**与路径的选择方向无关**，且对同一条曲线的不同参数化方式结果相同。第二型曲线积分则不仅在方向上敏感，而且对不同路径（即使方向相同）通常给出不同的结果。

**思考**：如果力场是 $\mathbf{F} = (y, x)$ 而不是 $(0, x^2)$，沿这两条路径的积分结果是否还不同？这将在学习 Green 定理后得到解答。

### 例 2：上半圆周上的第二型曲线积分（对应考试题 Q3）

计算第二型曲线积分
$$I = \int_L y\,dx - x\,dy$$

其中 $L$ 是上半圆周 $x^2 + y^2 = a^2$（$y \geq 0$），从点 $A(a,0)$ 沿逆时针方向到点 $B(-a,0)$。

**解**：

**第 1 步：有向参数化**

上半圆周采用三角函数参数化（圆心在原点，半径为 $a$）：
$$\begin{cases}
x = a\cos t \\
y = a\sin t
\end{cases},\quad 0 \leq t \leq \pi$$

验证方向：$t=0$ 时 $(x,y) = (a,0) = A$（起点）；$t=\pi$ 时 $(x,y) = (-a,0) = B$（终点）。$t$ 从 $0$ 增加到 $\pi$ 时，点沿上半圆周从右向左运动（逆时针方向？）。

检查方向：参数化 $t: 0 \to \pi$，$\cos t: 1 \to -1$，$\sin t: 0 \to 0$（经过 $\sin t \geq 0$）。从 $(a,0)$ 出发，$t$ 略大于 $0$ 时 $x = a\cos t \approx a$，$y = a\sin t \approx at > 0$，所以点向上移动——这正是我们期望的上半圆周方向（从右端点经上半圆到左端点）。

光滑性验证：$x'(t) = -a\sin t$，$y'(t) = a\cos t$，在 $[0,\pi]$ 上连续且 $x'(t)^2 + y'(t)^2 = a^2 > 0$，满足条件。

**第 2 步：求有向投影微元**
$$dx = x'(t)\,dt = -a\sin t\,dt,\qquad dy = y'(t)\,dt = a\cos t\,dt$$

注意：$dx = -a\sin t\,dt$ 在 $t \in (0,\pi)$ 上 $\sin t > 0$，因此 $dx$ 为负——这与 $x$ 从 $a$ 减小到 $-a$ 一致。$dy$ 在 $t \in (0,\pi/2)$ 为正（$y$ 增加），在 $t \in (\pi/2,\pi)$ 为负（$y$ 减小）。

**第 3 步：代入定理 14.4**

本题中 $P(x,y) = y$，$Q(x,y) = -x$。用参数表达：
$$P(x(t),y(t)) = y(t) = a\sin t,\qquad Q(x(t),y(t)) = -x(t) = -a\cos t$$

被积函数在参数化下：
$$\begin{aligned}
P\,dx + Q\,dy &\xrightarrow{\text{参数化}} P\,x'(t)dt + Q\,y'(t)dt \\
&= (a\sin t)(-a\sin t)\,dt + (-a\cos t)(a\cos t)\,dt \\
&= -a^2\sin^2 t\,dt - a^2\cos^2 t\,dt \\
&= -a^2(\sin^2 t + \cos^2 t)\,dt \\
&= -a^2\,dt
\end{aligned}$$

**第 4 步：计算定积分**

$$\begin{aligned}
I &= \int_L y\,dx - x\,dy \\
  &= \int_0^\pi \big[ (a\sin t)(-a\sin t) + (-a\cos t)(a\cos t) \big]\,dt \quad (\text{定理 14.4})\\
  &= \int_0^\pi (-a^2\sin^2 t - a^2\cos^2 t)\,dt \\
  &= \int_0^\pi (-a^2)(\sin^2 t + \cos^2 t)\,dt \\
  &= \int_0^\pi (-a^2)\,dt \\
  &= -a^2 \cdot [t]_{0}^{\pi} \\
  &= -a^2\pi
\end{aligned}$$

因此 $\displaystyle \int_L y\,dx - x\,dy = -a^2\pi$。

**结果解读**：
- 积分值为负，说明在沿曲线的运动过程中，$y\,dx - x\,dy$ 总体上产生了负的累积
- 这一结果可以利用对称性验证：考虑 $L$ 关于 $y$ 轴对称，但 $y\,dx - x\,dy$ 在对称点上符号相反（因为 $x \to -x$ 使 $-x\,dy$ 变号），净效果为非零

### 例 3：直线段上的第二型曲线积分

计算第二型曲线积分
$$I = \int_L x\,dy$$

其中 $L$ 是平面上从 $A(0,0)$ 到 $B(2,3)$ 的直线段。

**解**：

**第 1 步：有向参数化**

连接 $(0,0)$ 和 $(2,3)$ 的直线段，采用 $t$ 从 $0$ 到 $1$ 的线性参数化：
$$\begin{cases}
x = 2t \\
y = 3t
\end{cases},\quad 0 \leq t \leq 1$$

参数 $t$ 从 $0$ 到 $1$ 递增时，曲线从 $(0,0)$ 走向 $(2,3)$，方向一致。$\checkmark$

光滑性验证：$x'(t) = 2$，$y'(t) = 3$，在 $[0,1]$ 上连续且 $2^2 + 3^2 = 13 > 0$。

**第 2 步：求有向投影微元**
$$dx = 2\,dt,\qquad dy = 3\,dt$$

**第 3 步：代入定理 14.4**

本题中 $P(x,y) = 0$，$Q(x,y) = x$（即 $0 \cdot dx + x \cdot dy$）。用参数表达：
$$Q(x(t),y(t)) = x(t) = 2t$$

$$\begin{aligned}
I &= \int_L x\,dy \\
  &= \int_0^1 \big[ 0 \cdot 2 + (2t) \cdot 3 \big]\,dt \quad (\text{定理 14.4}) \\
  &= \int_0^1 6t\,dt \\
  &= 6 \cdot \left[ \frac{t^2}{2} \right]_{0}^{1} \\
  &= 6 \cdot \frac{1}{2} = 3
\end{aligned}$$

因此 $\displaystyle \int_L x\,dy = 3$。

**验证（几何直觉）**：在直线段上，$y$ 从 $0$ 线性增加到 $3$，$x$ 从 $0$ 线性增加到 $2$。在每一点 $x$ 是位置坐标，$dy$ 是 $y$ 方向的微小位移。$\int_L x\,dy$ 可以理解为"以 $x$ 为杠杆臂、沿 $y$ 方向累积"。直线段上 $x$ 与 $y$ 成正比（$x = 2y/3$），可以验证：$\int_L x\,dy = \int_0^3 \frac{2y}{3}\,dy = \frac{2}{3} \cdot \frac{9}{2} = 3$。$\checkmark$

**另一种解法（直接使用 $y$ 作为参数）**：
将直线段写为 $x = \dfrac{2}{3}y$，$y$ 从 $0$ 到 $3$。此时参数为 $y$：
$$dy = dy,\qquad x = \frac{2}{3}y$$

$$\int_L x\,dy = \int_0^3 \frac{2}{3}y\,dy = \frac{2}{3} \cdot \left[\frac{y^2}{2}\right]_0^3 = \frac{2}{3} \cdot \frac{9}{2} = 3$$

结果一致。注意：用 $y$ 作为参数时，$\int_L x\,dy$ 中的 $dy$ 既是积分微元也是参数微分，所以 $dy = dy$（即 $y'(y) = 1$）。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|:---------|:---------|
| $dx$ 和 $dy$ 是弧长微元，始终为正 | $dx, dy$ 是有向投影微元，可正可负。在参数化下 $dx = x'(t)dt$，符号由 $x'(t)$ 决定。只有 $ds = \sqrt{dx^2 + dy^2} > 0$ 始终为正 |
| 第二型曲线积分与方向无关（像第一型一样） | 第二型曲线积分在方向反转时变号：$\int_{L^-} = -\int_L$。这是与第一型最核心的区别 |
| $\int_L P\,dx + Q\,dy = \int_L \sqrt{P^2 + Q^2}\,ds$，将第二型误认为第一型的变形 | 第二型不是"第一型的另一种写法"。$\int_L P\,dx + Q\,dy = \int_L \mathbf{F} \cdot d\mathbf{r} = \int_L \|\mathbf{F}\|\cos\theta\,ds$，其中 $\cos\theta$ 是 $\mathbf{F}$ 与切向量的夹角余弦，它随方向变化 |
| 参数化时随便选取方向，计算结果都一样 | 参数化必须保证 $t$ 递增方向与曲线的指定方向一致。如果方向反了，结果会差一个负号 |
| 用定理 14.4 时忘记代入 $x'(t)$ 和 $y'(t)$，只把 $P,Q$ 用参数表达 | 完整公式是 $\int_a^b [P\,x' + Q\,y']\,dt$，$x'(t)$ 和 $y'(t')$ 是参数的导数因子，不可省略 |
| 认为从 $A$ 到 $B$ 沿任何路径的积分值都相等（受一维定积分 $\int_a^b f(x)dx$ 的影响） | 一维定积分 $\int_a^b f(x)dx$ 只依赖端点，因为路径被限定在 $x$ 轴上；第二型曲线积分在平面/空间中开放路径选择，通常依赖于路径 |
| 混淆 $\int_L y\,dx$ 与 $\int_L y\,ds$ | $\int_L y\,dx$ 是第二型（$dx$ 是有向投影），$\int_L y\,ds$ 是第一型（$ds$ 是弧长）。两者含义和数值完全不同 |
| 认为第二型曲线积分只能是 $\int_L P\,dx + Q\,dy$ 的形式，不能写 $\int_L Q\,dy$ 或 $\int_L P\,dx$ 单独出现 | $\int_L P\,dx$ 是 $Q \equiv 0$ 的特例，$\int_L Q\,dy$ 是 $P \equiv 0$ 的特例。例 3 中 $\int_L x\,dy$ 即为 $P \equiv 0$ 的情形 |

## 练习题

### 基础巩固

**1.**（定义理解）用 $\varepsilon$-$\delta$ 语言叙述第二型曲线积分 $\int_L P\,dx + Q\,dy$ 可积的确切含义，并说明为什么方向反转会导致积分值变号。

<details><summary>参考答案</summary>

**$\varepsilon$-$\delta$ 定义**：存在实数 $I$，使得对任意 $\varepsilon > 0$，存在 $\delta > 0$，对有向曲线 $L$ 的任意分割（满足细度 $\lambda = \max_i \sqrt{\Delta x_i^2 + \Delta y_i^2} < \delta$）以及任意取样点 $(\xi_i,\eta_i)$ 在第 $i$ 个小弧段上，都有：
$$\left| \sum_{i=1}^{n} \big[ P(\xi_i,\eta_i)\,\Delta x_i + Q(\xi_i,\eta_i)\,\Delta y_i \big] - I \right| < \varepsilon$$

此时 $I$ 称为 $P\,dx + Q\,dy$ 在 $L$ 上的第二型曲线积分，记作 $I = \int_L P\,dx + Q\,dy$。

**方向变号的证明**：当方向反转时，分割顺序反转，第 $i$ 段的投影变为：
$$\Delta x'_i = x_{i-1} - x_i = -\Delta x_i,\qquad \Delta y'_i = y_{i-1} - y_i = -\Delta y_i$$

因此 Riemann 和为：
$$S'_n = \sum [P(-\Delta x_i) + Q(-\Delta y_i)] = -\sum [P\Delta x_i + Q\Delta y_i] = -S_n$$

取极限得 $\int_{L^-} = -\int_L$。

</details>

**2.**（有向微元与参数化）对下列有向曲线，写出一种参数方程，并计算 $dx$ 和 $dy$ 的参数化表达式：

**(1)** 从 $(0,0)$ 到 $(3,4)$ 的直线段

**(2)** 从 $(R,0)$ 到 $(-R,0)$ 的上半圆周（$y \geq 0$）

**(3)** 从 $(1,0)$ 到 $(0,1)$ 的抛物线 $y = (1-x)^2$（$x$ 从 $1$ 到 $0$）

<details><summary>参考答案</summary>

**(1)** 直线段：
$$\begin{cases}
x = 3t \\
y = 4t
\end{cases},\quad 0 \leq t \leq 1$$

$dx = 3\,dt$，$dy = 4\,dt$（均为正，因为 $x$ 和 $y$ 都随 $t$ 增加而增加）。

**(2)** 上半圆周：
$$\begin{cases}
x = R\cos t \\
y = R\sin t
\end{cases},\quad 0 \leq t \leq \pi$$

$dx = -R\sin t\,dt$，$dy = R\cos t\,dt$。

注意：$t \in (0,\pi)$ 时 $\sin t > 0$，所以 $dx$ 为负（$x$ 从 $R$ 递减到 $-R$）；$t \in (0,\pi/2)$ 时 $dy > 0$（$y$ 增加），$t \in (\pi/2,\pi)$ 时 $dy < 0$（$y$ 减少）。

**(3)** 抛物线 $y = (1-x)^2$ 从 $(1,0)$ 到 $(0,1)$，取 $x$ 作为参数，但注意方向——$x$ 从 $1$ 到 $0$：
$$\begin{cases}
x = t \\
y = (1-t)^2
\end{cases},\quad 1 \geq t \geq 0 \quad (\text{即 } t \text{ 从 } 1 \text{ 递减到 } 0)$$

$dx = dt$，$dy = -2(1-t)\,dt$。

注意：参数 $t$ 递减时 $dt < 0$，所以 $dx = dt < 0$（$x$ 确实在减小）。为使用定理 14.4（$t$ 递增形式），可反转参数化方向：
令 $t' = 1 - t$，则 $t'$ 从 $0$ 到 $1$，且 $\begin{cases} x = 1 - t' \\ y = t'^2 \end{cases}$，$dx = -dt'$，$dy = 2t'\,dt'$。选择哪种方式均可，但需注意符号一致性。

</details>

**3.**（第二型曲线积分计算）计算下列曲线积分：

**(1)** $\displaystyle \int_L y\,dx + x\,dy$，其中 $L$ 是连接 $(0,0)$ 到 $(2,1)$ 的直线段，方向从 $(0,0)$ 到 $(2,1)$

**(2)** $\displaystyle \int_L x\,dx + y\,dy$，其中 $L$ 是从 $(0,0)$ 沿直线到 $(1,0)$，再沿直线到 $(1,1)$ 的折线

**(3)** $\displaystyle \int_L x^2\,dy$，其中 $L$ 是 $y = x^2$ 从 $(0,0)$ 到 $(1,1)$ 的一段（方向为 $x$ 增加）

<details><summary>参考答案</summary>

**(1)** 参数化：$x = 2t$，$y = t$，$0 \leq t \leq 1$。

$dx = 2\,dt$，$dy = dt$。

$P(x,y) = y$，$Q(x,y) = x$，代入参数：$P = t$，$Q = 2t$。

$$\begin{aligned}
\int_L y\,dx + x\,dy &= \int_0^1 \big[ t \cdot 2 + 2t \cdot 1 \big]\,dt \\
&= \int_0^1 (2t + 2t)\,dt \\
&= \int_0^1 4t\,dt \\
&= 4 \cdot \left[\frac{t^2}{2}\right]_0^1 = 4 \cdot \frac{1}{2} = 2
\end{aligned}$$

**(2)** 折线 $L = L_1 \cup L_2$。

$L_1$：从 $(0,0)$ 到 $(1,0)$，参数化 $x = t$，$y = 0$，$0 \leq t \leq 1$。$dx = dt$，$dy = 0$。
$$\int_{L_1} x\,dx + y\,dy = \int_0^1 t\cdot 1 + 0\cdot 0\,dt = \int_0^1 t\,dt = \frac{1}{2}$$

$L_2$：从 $(1,0)$ 到 $(1,1)$，参数化 $x = 1$，$y = t$，$0 \leq t \leq 1$。$dx = 0$，$dy = dt$。
$$\int_{L_2} x\,dx + y\,dy = \int_0^1 1\cdot 0 + t\cdot 1\,dt = \int_0^1 t\,dt = \frac{1}{2}$$

求和：
$$\int_L x\,dx + y\,dy = \frac{1}{2} + \frac{1}{2} = 1$$

**(3)** 参数化：$x = t$，$y = t^2$，$0 \leq t \leq 1$。$dx = dt$，$dy = 2t\,dt$。

$P(x,y) = 0$，$Q(x,y) = x^2$，代入参数：$Q = t^2$。

$$\begin{aligned}
\int_L x^2\,dy &= \int_0^1 \big[ 0 \cdot 1 + t^2 \cdot 2t \big]\,dt \\
&= \int_0^1 2t^3\,dt \\
&= 2 \cdot \left[\frac{t^4}{4}\right]_0^1 = 2 \cdot \frac{1}{4} = \frac{1}{2}
\end{aligned}$$

</details>

### 迁移应用

**4.**（方向反转与对比）设 $L$ 是从 $(1,0)$ 沿上半圆周 $x^2 + y^2 = 1$（$y \geq 0$）到 $(-1,0)$ 的弧，计算并比较：

**(1)** $\displaystyle \int_L x\,ds$（第一型曲线积分）

**(2)** $\displaystyle \int_L x\,dx$（第二型曲线积分）

**(3)** 如果 $L$ 的方向反转（从 $(-1,0)$ 到 $(1,0)$），上述两个积分如何变化？

<details><summary>参考答案</summary>

**参数化**：$x = \cos t$，$y = \sin t$，$0 \leq t \leq \pi$。

**第 (1) 问**：第一型曲线积分 $\int_L x\,ds$

$x'(t) = -\sin t$，$y'(t) = \cos t$，$ds = \sqrt{(-\sin t)^2 + (\cos t)^2}\,dt = dt$。

$$\int_L x\,ds = \int_0^\pi \cos t \cdot 1\,dt = \big[\sin t\big]_{0}^{\pi} = \sin\pi - \sin 0 = 0 - 0 = 0$$

**第 (2) 问**：第二型曲线积分 $\int_L x\,dx$

$dx = -\sin t\,dt$。

$$\int_L x\,dx = \int_0^\pi (\cos t)(-\sin t)\,dt = \int_0^\pi (-\cos t\sin t)\,dt$$

利用 $\sin 2t = 2\sin t\cos t$，有 $-\cos t\sin t = -\frac{1}{2}\sin 2t$：
$$\int_0^\pi \left(-\frac{1}{2}\sin 2t\right)dt = -\frac{1}{2} \cdot \left[-\frac{\cos 2t}{2}\right]_0^\pi = \frac{1}{4}\big[\cos 2t\big]_{0}^{\pi} = \frac{1}{4}(\cos 2\pi - \cos 0) = \frac{1}{4}(1 - 1) = 0$$

巧合的是两个积分都等于 $0$，但原因不同：第一型是因为 $\cos t$ 在 $[0,\pi]$ 上对称积分为零（奇对称性），第二型是因为 $x\,dx = -\cos t\sin t\,dt$ 的周期积分为零。

**第 (3) 问**：方向反转的影响

- 第一型 $\int_L x\,ds$：**不变**（仍为 $0$），因为 $ds$ 是标量弧长，与方向无关。
- 第二型 $\int_L x\,dx$：方向反转时，$dx \to -dx$，但 $x$ 不变（曲线上的同一点坐标不变），所以 $x\,dx \to -x\,dx$，积分值**变号**。

验证：若 $L$ 从 $(-1,0)$ 到 $(1,0)$，参数化为 $x = \cos t$，$y = \sin t$，$\pi \geq t \geq 0$（$t$ 递减）。为使用定理 14.4 的标准形式，反转参数令 $t' = \pi - t$，$0 \leq t' \leq \pi$，则 $x = -\cos t'$，$y = \sin t'$。$dx = \sin t'\,dt'$。
$$\int_{L^-} x\,dx = \int_0^\pi (-\cos t')(\sin t')\,dt' = -\int_0^\pi \cos t'\sin t'\,dt' = 0$$

实际上计算结果仍为 $0$，但一般情况应变号：设 $L$ 上的积分为 $I$，则 $L^-$ 上的积分为 $-I$。本例中 $I = 0$，$-I = 0$，恰好相等，这并不与方向变号性质矛盾——$0$ 的相反数仍是 $0$。

**重要结论**：第一型曲线积分方向反转时值不变；第二型曲线积分方向反转时值变号。但当积分值为 $0$ 时，两者数值上相同（$0 = -0$），但从概念上仍然不同。

</details>

**5.**（路径依赖性验证）计算力场 $\mathbf{F} = (y, x)$ 沿以下两条路径从 $(0,0)$ 到 $(1,1)$ 所做的功：

**(1)** 路径 $L_1$：$y = x$（直线）
**(2)** 路径 $L_2$：$y = x^3$（立方曲线）

并判断第二型曲线积分 $\int_L y\,dx + x\,dy$ 是否依赖于路径。

<details><summary>参考答案</summary>

**路径 $L_1$：$y = x$**

参数化：$x = t$，$y = t$，$0 \leq t \leq 1$。$dx = dt$，$dy = dt$。

$P = y = t$，$Q = x = t$。

$$\begin{aligned}
W_1 &= \int_{L_1} y\,dx + x\,dy = \int_0^1 \big[ (t)(1) + (t)(1) \big]\,dt \\
    &= \int_0^1 2t\,dt = 2 \cdot \left[\frac{t^2}{2}\right]_0^1 = 1
\end{aligned}$$

**路径 $L_2$：$y = x^3$**

参数化：$x = t$，$y = t^3$，$0 \leq t \leq 1$。$dx = dt$，$dy = 3t^2\,dt$。

$P = y = t^3$，$Q = x = t$。

$$\begin{aligned}
W_2 &= \int_{L_2} y\,dx + x\,dy = \int_0^1 \big[ (t^3)(1) + (t)(3t^2) \big]\,dt \\
    &= \int_0^1 (t^3 + 3t^3)\,dt = \int_0^1 4t^3\,dt \\
    &= 4 \cdot \left[\frac{t^4}{4}\right]_0^1 = 1
\end{aligned}$$

比较：$W_1 = 1$，$W_2 = 1$，$W_1 = W_2$。

**结论**：力场 $\mathbf{F} = (y, x)$ 沿这两条不同路径所做的功相等。这说明**并非所有第二型曲线积分都是路径依赖的**——在某些力场（保守力场）中，积分值仅依赖于起点和终点，与路径无关。

这与例 1 中的 $\mathbf{F} = (0, x^2)$ 形成对比（例 1 中两条路径结果不同）。关于保守力场的条件（$\partial P/\partial y = \partial Q/\partial x$）将在后续章节中详细讨论。

**当前阶段需要理解**：第二型曲线积分**通常**是路径依赖的（如例 1），但也存在特殊情况下的路径无关性（如本练习）。在计算第二型曲线积分时，必须明确指出积分路径。

</details>
