## 2. 背景与应用场景

**核心问题**：在前序文件中，我们处理的函数都是**显式**给定的——即 $y=f(x)$ 或 $z=f(x,y)$ 这样的形式，因变量直接表达为自变量的函数。但在数学和物理中，大量函数关系是以**方程**的形式隐含地给出的。

**什么是隐函数？**

我们说 $y$ 是 $x$ 的**显函数**，如果 $y$ 直接写成 $x$ 的表达式：$y=f(x)$。但许多曲线由方程 $F(x,y)=0$ 定义，如 $x^3+y^3-6xy=0$——在这个方程中，我们无法（或很难）将 $y$ 解成 $x$ 的显式形式。然而在局部范围内，对于每个 $x$，方程确定了一个对应的 $y$ 值。这样的 $y$ 就是 $x$ 的**隐函数**（implicit function）。

**实际问题举例**：
- **几何曲线**：单位圆 $x^2+y^2=1$ 不是函数图像（一个 $x$ 对应两个 $y$），但在 $(0,1)$ 附近 $y=\sqrt{1-x^2}$ 是确定的函数
- **经济学**：等产量曲线 $Q(K,L)=C$ 隐含地定义了 $L$ 关于 $K$ 的函数关系
- **物理化学**：范德瓦耳斯方程 $(P+a/V^2)(V-b)=RT$ 定义了 $P$ 关于 $V$ 和 $T$ 的隐函数
- **热力学**：状态方程 $f(P,V,T)=0$ 隐含了任意两个变量确定第三个变量的关系

**我们需要回答两个问题**：
1. **存在性问题**：方程 $F(x,y)=0$ 在什么条件下，在某个局部范围内能唯一确定一个可微函数 $y=y(x)$？
2. **计算问题**：如果隐函数存在，如何在不显式解出 $y$ 的情况下计算其导数 $\frac{dy}{dx}$？

隐函数存在定理（implicit function theorem）完整地回答了这两个问题。

## 3. 核心概念与符号约定

### 符号表

| 符号 | 含义 | 说明 |
|------|------|------|
| $F(x,y)=0$ | 确定隐函数的方程 | $F$ 是已知的二元函数 |
| $y=y(x)$ | 方程确定的隐函数 | 满足 $F(x,y(x))\equiv0$ |
| $F_x=\frac{\partial F}{\partial x}$ | $F$ 对 $x$ 的偏导 | 固定 $y$ 对 $x$ 求偏导 |
| $F_y=\frac{\partial F}{\partial y}$ | $F$ 对 $y$ 的偏导 | 固定 $x$ 对 $y$ 求偏导 |
| $\frac{dy}{dx}$ | 隐函数的导数 | $\displaystyle\frac{dy}{dx}=-\frac{F_x}{F_y}$ |
| $z=z(x,y)$ | $F(x,y,z)=0$ 确定的隐函数 | 满足 $F(x,y,z(x,y))\equiv0$ |
| $\frac{\partial z}{\partial x},\frac{\partial z}{\partial y}$ | 多元隐函数的偏导 | $\displaystyle\frac{\partial z}{\partial x}=-\frac{F_x}{F_z}$，$\displaystyle\frac{\partial z}{\partial y}=-\frac{F_y}{F_z}$ |

### 3.1 隐函数与显函数的对比

| 对比项 | 显函数 $y=f(x)$ | 隐函数 $F(x,y)=0$ |
|--------|----------------|-------------------|
| 定义方式 | 直接给出 $y$ 关于 $x$ 的表达式 | 由方程间接定义 $y$ 与 $x$ 的关系 |
| 导数计算 | 直接对表达式求导 | 用隐函数求导公式或链式法则 |
| 存在性 | 显式定义，自然存在 | 需要定理保证存在性 |
| 单值性 | 每个 $x$ 唯一对应一个 $y$ | 可能一个 $x$ 对应多个 $y$（如圆）|
| 定义域 | 明确给出 | 需要从方程中确定 |

### 3.2 隐函数存在定理（二元情形）（A1）

**定理 12.9（隐函数存在定理——二元情形）**：设二元函数 $F(x,y)$ 满足以下三个条件：

1. **零点条件**：$F(x_0,y_0)=0$（即点 $(x_0,y_0)$ 在曲线上）；
2. **光滑性条件**：$F$ 在点 $(x_0,y_0)$ 的某邻域 $U$ 内具有连续的一阶偏导数 $F_x$ 和 $F_y$（即 $F\in C^1(U)$）；
3. **非退化条件**：$F_y(x_0,y_0)\neq 0$（在 $(x_0,y_0)$ 处对 $y$ 的偏导不为零）。

则存在 $\delta>0$ 和 $\eta>0$，以及**唯一**的连续可导函数 $y=y(x)$，定义在区间 $(x_0-\delta,x_0+\delta)$ 上，满足：

- $y(x_0)=y_0$（初始条件）；
- 对任意 $x\in(x_0-\delta,x_0+\delta)$，有 $F(x,y(x))=0$（方程被满足）；
- $y(x)$ 在 $(x_0-\delta,x_0+\delta)$ 上具有连续导数，且

$$\boxed{\frac{dy}{dx} = -\frac{F_x(x,y(x))}{F_y(x,y(x))}}$$

**条件解读（为什么这三个条件缺一不可）：**

1. **零点条件 $F(x_0,y_0)=0$**：这是显然的前提——如果点不在曲线上，不存在什么隐函数。这个条件保证了过点 $(x_0,y_0)$。
2. **光滑性条件 $F\in C^1$**：$F$ 的连续可微性确保了局部范围内 $F$ 的行为是"良好"的，是一阶线性近似有效的保证。此外，$F_x,F_y$ 的连续性还保证了后面使用链式法则时中间函数的可微性。
3. **非退化条件 $F_y(x_0,y_0)\neq 0$**：这是最微妙的条件。它意味着在 $(x_0,y_0)$ 处，$F$ 对 $y$ 不是"不敏感"的。如果 $F_y=0$，方程在 $(x_0,y_0)$ 附近可能无法唯一确定 $y$（如 $F(x,y)=x^2+y^2-1$ 在 $(1,0)$ 处 $F_y=0$，该点附近 $y$ 不是 $x$ 的函数——实际上 $F_y=0$ 对应圆的切线垂直于 $x$ 轴的位置）。

**几何直观**：

方程 $F(x,y)=0$ 定义了一条平面曲线（等值线）。梯度 $\nabla F=(F_x,F_y)$ 垂直于该曲线。条件 $F_y(x_0,y_0)\neq 0$ 意味着梯度有非零的 $y$ 分量，即曲线在 $(x_0,y_0)$ 处的切线**不垂直于 $x$ 轴**（即切线有有限斜率）。此时，在 $(x_0,y_0)$ 附近，曲线可以表示为一个函数 $y=y(x)$ 的图像。

换句话说：$F_y\neq 0$ 保证了曲线在该点"足够横向延伸"，使得对每个 $x$，方程恰有一个 $y$ 解。

### 3.3 隐函数求导公式的推导

隐函数求导公式 $\displaystyle\frac{dy}{dx}=-\frac{F_x}{F_y}$ 可以从链式法则（定理 12.7）直接导出。

**推导**：假设 $y=y(x)$ 是由 $F(x,y)=0$ 确定的隐函数，则对任意 $x$（在定义域内）有恒等式：

$$F(x,\;y(x)) \equiv 0$$

将恒等式两边对 $x$ 求全导数。注意 $F$ 是二元函数，其自变量 $x$ 和 $y$ 中，$y$ 又是 $x$ 的函数。由链式法则（定理 12.7）：

$$\frac{d}{dx}F(x,y(x)) = \frac{\partial F}{\partial x}\cdot\frac{dx}{dx} + \frac{\partial F}{\partial y}\cdot\frac{dy}{dx} = F_x \cdot 1 + F_y \cdot \frac{dy}{dx}$$

由于恒等式 $F(x,y(x))\equiv 0$ 的左边恒为零，其导数也为零，因此：

$$F_x + F_y\cdot\frac{dy}{dx} = 0$$

在 $F_y\neq 0$ 的条件下，解出 $\frac{dy}{dx}$：

$$\boxed{\frac{dy}{dx} = -\frac{F_x}{F_y}}$$

**推导中的关键步骤**：$F_x$ 是 $F$ 对第一个位置变量（即 $x$）的偏导——计算时将 $y$ 视为常数。因为 $x$ 在 $F$ 中出现了两次（一次直接作为第一个位置变量，一次通过 $y(x)$ 间接），链式法则需要将两条路径叠加。路径1：$F$ 对直接 $x$ 的偏导乘 $dx/dx=1$；路径2：$F$ 对 $y$ 的偏导乘 $dy/dx$。两条路径之和为零。

**定理存在性与公式计算性的逻辑分工**：
- **定理 12.9** 保证：在满足三个条件的 $(x_0,y_0)$ 附近，隐函数 $y=y(x)$ **存在**且**唯一**、**可导**
- **公式 $\frac{dy}{dx}=-F_x/F_y$** 提供：一旦存在性得到保证，导数的**计算**手段——无需显式解出 $y$
- 先有存在定理后有计算公式：定理保证了公式的合法使用范围

### 3.4 隐函数存在定理（多元情形）（A2）

将定理 12.9 从二元函数 $F(x,y)$ 推广到三元函数 $F(x,y,z)$，即用一个方程 $F(x,y,z)=0$ 确定两个自变量的隐函数 $z=z(x,y)$。

**定理 12.10（隐函数存在定理——多元情形）**：设三元函数 $F(x,y,z)$ 满足以下三个条件：

1. **零点条件**：$F(x_0,y_0,z_0)=0$；
2. **光滑性条件**：$F$ 在点 $(x_0,y_0,z_0)$ 的某邻域 $U$ 内具有连续的一阶偏导数 $F_x,F_y,F_z$；
3. **非退化条件**：$F_z(x_0,y_0,z_0)\neq 0$。

则存在 $\delta>0$ 和 $\varepsilon>0$，以及**唯一**的连续可微函数 $z=z(x,y)$，定义在 $(x_0,y_0)$ 的某邻域上，满足：

- $z(x_0,y_0)=z_0$；
- 对 $(x,y)$ 在该邻域内，有 $F(x,y,z(x,y))\equiv 0$；
- $z(x,y)$ 具有连续偏导数，且

$$\boxed{\frac{\partial z}{\partial x} = -\frac{F_x}{F_z},\qquad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}}$$

**推导**（与二元情形完全类似的链式法则论证）：

设 $z=z(x,y)$ 是由 $F(x,y,z)=0$ 确定的隐函数。对恒等式 $F(x,y,z(x,y))\equiv 0$ 两边关于 $x$ 求偏导（固定 $y$，使用链式法则，定理 12.7）：

$$\frac{\partial}{\partial x}F(x,y,z(x,y)) = F_x\cdot\frac{\partial x}{\partial x} + F_y\cdot\frac{\partial y}{\partial x} + F_z\cdot\frac{\partial z}{\partial x} = F_x\cdot 1 + F_y\cdot 0 + F_z\cdot\frac{\partial z}{\partial x} = 0$$

因此 $F_x + F_z\cdot\frac{\partial z}{\partial x} = 0$，在 $F_z\neq 0$ 的条件下解得：

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}$$

类似地，对恒等式两边关于 $y$ 求偏导（固定 $x$）：

$$\frac{\partial}{\partial y}F(x,y,z(x,y)) = F_x\cdot 0 + F_y\cdot 1 + F_z\cdot\frac{\partial z}{\partial y} = F_y + F_z\cdot\frac{\partial z}{\partial y} = 0$$

解得：

$$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

**与二元情形的类比**：

| 对比项 | 二元情形 $F(x,y)=0$ | 三元情形 $F(x,y,z)=0$ |
|--------|---------------------|----------------------|
| 方程个数 | 1 个 | 1 个 |
| 自变量个数 | 1 ($x$) | 2 ($x,y$) |
| 隐函数 | $y=y(x)$ | $z=z(x,y)$ |
| 非退化条件 | $F_y\neq 0$ | $F_z\neq 0$ |
| 导数公式 | $dy/dx = -F_x/F_y$ | $\partial z/\partial x = -F_x/F_z$ |
| 几何解释 | 曲线切线非垂直 | 曲面的切平面非"竖直" |

## 4. 原理与方法

### 4.1 隐函数存在定理的直观理解

**为什么 $F_y\neq 0$ 如此重要？**

从线性近似的角度看，在 $(x_0,y_0)$ 附近，$F$ 可以用其全微分近似：

$$F(x,y) \approx F(x_0,y_0) + F_x(x_0,y_0)(x-x_0) + F_y(x_0,y_0)(y-y_0)$$

由于 $F(x_0,y_0)=0$，方程 $F(x,y)=0$ 近似等价于：

$$F_x(x_0,y_0)(x-x_0) + F_y(x_0,y_0)(y-y_0) \approx 0$$

如果 $F_y(x_0,y_0)\neq 0$，我们可以"解出" $y$：

$$y \approx y_0 - \frac{F_x(x_0,y_0)}{F_y(x_0,y_0)}(x-x_0)$$

这就是隐函数 $y=y(x)$ 在 $(x_0,y_0)$ 处的线性近似——其斜率恰好是 $-F_x/F_y$。

**几何图像**：

考虑曲线 $F(x,y)=0$ 和其上的点 $(x_0,y_0)$。如果 $F_y(x_0,y_0)\neq 0$，则曲线的切线斜率 $dy/dx=-F_x/F_y$ 为有限值。也就是说，曲线在该点**不是竖直的**——不会出现"一个 $x$ 对应多个 $y$"的局部现象。反过来，如果 $F_y=0$（且 $F_x\neq 0$），则切线垂直于 $x$ 轴，在 $(x_0,y_0)$ 附近 $y$ 不能表示为 $x$ 的函数。

**用梯度语言表述**：

梯度 $\nabla F=(F_x,F_y)$ 垂直于等值线 $F(x,y)=0$。$F_y\neq 0$ 说明梯度不沿水平方向——换句话说，等值线的切线**不是竖直的**。从方向导数的角度看（定理 12.4），在 $\nabla F$ 具有非零 $y$ 分量的位置，沿 $y$ 方向穿过等值线时 $F$ 的变化率非零，因此可以在局部唯一确定 $y$。

### 4.2 隐函数求导的标准操作流程

**计算隐函数一阶导数的三步法**：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 第 1 步：构建方程 | 将方程写为 $F(x,y)=0$ 的形式，确定函数 $F$ | 将所有项移到等式一边 |
| 第 2 步：计算偏导 | 计算 $F_x$ 和 $F_y$ | $F_x$ 固定 $y$ 对 $x$ 求偏导，$F_y$ 固定 $x$ 对 $y$ 求偏导 |
| 第 3 步：代入公式 | $\displaystyle\frac{dy}{dx} = -\frac{F_x}{F_y}$ | 确保 $F_y\neq 0$（定理保证了存在性，但公式在 $F_y=0$ 的点无定义） |

**计算隐函数二阶导数的四步法**：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 第 1 步：求一阶导 | 用三步法得到 $y' = -F_x/F_y$ | 结果常含 $x,y$ |
| 第 2 步：视为 $x$ 的函数 | $y'$ 对 $x$ 再求导 | $y'$ 表达式中的 $y$ 视为 $y(x)$ |
| 第 3 步：使用链式法则 | 对 $F_x,F_y$ 中的 $y$ 用链式法则展开 | $\frac{d}{dx}F_x = F_{xx}+F_{xy}y'$ |
| 第 4 步：代入 $y'$ 化简 | 将 $y'=-F_x/F_y$ 代入结果 | 得到仅含 $F$ 的偏导及其偏导的表达式 |

### 4.3 隐函数的高阶导数

当我们需要二阶及以上的导数时，对公式 $y' = -F_x/F_y$ 再求一次导即可。由于 $F_x$ 和 $F_y$ 仍然是 $x,y$ 的函数，而 $y$ 又是 $x$ 的隐函数，求导过程需要使用链式法则和商的求导法则。

**二阶导数的通用公式**：

设 $y=y(x)$ 由 $F(x,y)=0$ 确定，且 $F$ 二阶连续可微（$F\in C^2$）。则一阶导数 $y' = -\dfrac{F_x}{F_y}$。对 $y'$ 再求导：

$$y'' = \frac{d}{dx}\!\left(-\frac{F_x}{F_y}\right) = -\frac{\frac{d}{dx}(F_x)\cdot F_y - F_x\cdot\frac{d}{dx}(F_y)}{F_y^2}$$

其中 $\frac{d}{dx}(F_x)$ 和 $\frac{d}{dx}(F_y)$ 需要使用链式法则：

$$\frac{d}{dx}F_x = \frac{\partial F_x}{\partial x} + \frac{\partial F_x}{\partial y}\cdot\frac{dy}{dx} = F_{xx} + F_{xy}\,y'$$

$$\frac{d}{dx}F_y = \frac{\partial F_y}{\partial x} + \frac{\partial F_y}{\partial y}\cdot\frac{dy}{dx} = F_{yx} + F_{yy}\,y'$$

代入 $y' = -F_x/F_y$ 并利用 $F_{xy}=F_{yx}$（Clairaut 定理，定理 12.2，因 $F\in C^2$），得到：

$$y'' = -\frac{(F_{xx} - F_{xy}\frac{F_x}{F_y})F_y - F_x(F_{yx} - F_{yy}\frac{F_x}{F_y})}{F_y^2}$$

整理后通分得：

$$\boxed{y'' = -\frac{F_{xx}F_y^2 - 2F_{xy}F_xF_y + F_{yy}F_x^2}{F_y^3}}$$

这是一个**通用公式**，表达 $y''$ 完全用 $F$ 及其偏导表示。实际计算时，通常不直接套用这个公式，而是按以下方式**逐次求导**更不易出错：

1. 先求出一阶导数 $y'$（通常得到包含 $x,y$ 的表达式）
2. 对 $y'$ 的表达式直接再求导（将 $y$ 视为 $y(x)$，遇到 $y$ 就代 $y'$）
3. 最后将 $y' = -F_x/F_y$ 代入化简

### 4.4 隐函数定理的条件验证方法

在使用隐函数定理之前，必须逐条验证三个条件。以下是标准验证流程：

**条件 1：零点条件**
- 直接将 $(x_0,y_0)$ 代入 $F(x,y)$，计算 $F(x_0,y_0)$
- 确认结果等于 0

**条件 2：光滑性条件**
- 计算一阶偏导 $F_x$ 和 $F_y$
- 判断 $F_x$ 和 $F_y$ 在 $(x_0,y_0)$ 附近是否连续
- 对于多项式、三角函数、指数函数等初等函数的组合，只要定义域合理，一阶偏导自然连续

**条件 3：非退化条件**
- 计算 $F_y(x_0,y_0)$
- 确认 $F_y(x_0,y_0) \neq 0$

三个条件全部满足后，定理保证隐函数存在且可导。然后使用公式 $dy/dx = -F_x/F_y$ 计算导数。

## 5. 例题

### 例题 1：二元隐函数存在定理的条件验证与导数计算

考虑曲线 $x^3 + y^3 - 6xy = 0$（笛卡尔叶形线）。

(1) 验证在点 $(3,3)$ 处，方程满足隐函数存在定理的条件。
(2) 求隐函数 $y=y(x)$ 在 $(3,3)$ 处的一阶导数。
(3) 求隐函数在 $(3,3)$ 处的二阶导数 $y''$。

**解**：

**(1) 条件验证**：

令 $F(x,y) = x^3 + y^3 - 6xy$。

**条件 1（零点条件）**：
$$F(3,3) = 3^3 + 3^3 - 6\cdot3\cdot3 = 27 + 27 - 54 = 0 \quad \checkmark$$

**条件 2（光滑性条件）**：
$$F_x = \frac{\partial}{\partial x}(x^3 + y^3 - 6xy) = 3x^2 - 6y$$
$$F_y = \frac{\partial}{\partial y}(x^3 + y^3 - 6xy) = 3y^2 - 6x$$

$F_x$ 和 $F_y$ 都是多项式，在 $\mathbb{R}^2$ 上处处连续。因此 $F\in C^1$。$\checkmark$

**条件 3（非退化条件）**：
$$F_y(3,3) = 3\cdot 3^2 - 6\cdot 3 = 27 - 18 = 9 \neq 0 \quad \checkmark$$

三个条件全部满足。由定理 12.9，存在 $(3,3)$ 的某邻域，在该邻域内方程唯一确定一个具有连续导数的隐函数 $y=y(x)$，且 $y(3)=3$。

**(2) 一阶导数**：

由隐函数求导公式 $\displaystyle y' = -\frac{F_x}{F_y}$：

$$y'(3) = -\frac{F_x(3,3)}{F_y(3,3)} = -\frac{3\cdot3^2 - 6\cdot 3}{3\cdot3^2 - 6\cdot 3} = -\frac{27 - 18}{27 - 18} = -\frac{9}{9} = -1$$

因此曲线在 $(3,3)$ 处的切线斜率为 $-1$。

**验证**（用直接隐函数求导法）：

对 $x^3 + y^3 - 6xy = 0$ 两边直接对 $x$ 求导（将 $y$ 视为 $y(x)$）：

$$3x^2 + 3y^2\cdot y' - 6y - 6x\cdot y' = 0$$

整理：
$$3x^2 - 6y + (3y^2 - 6x)y' = 0$$

解出 $y'$：
$$y' = \frac{6y - 3x^2}{3y^2 - 6x} = \frac{2y - x^2}{y^2 - 2x}$$

代入 $(3,3)$：
$$y'(3) = \frac{2\cdot3 - 3^2}{3^2 - 2\cdot3} = \frac{6-9}{9-6} = -\frac{3}{3} = -1$$

与公式结果一致。$\checkmark$

**(3) 二阶导数**：

对 $y' = \dfrac{2y - x^2}{y^2 - 2x}$ 再求导。使用商的求导法则，注意 $y$ 是 $x$ 的函数：

$$y'' = \frac{(2y' - 2x)(y^2 - 2x) - (2y - x^2)(2y\cdot y' - 2)}{(y^2 - 2x)^2}$$

代入 $x=3$，$y=3$，$y'=-1$：

分子第一项：$(2(-1) - 6)(3^2 - 2\cdot3) = (-2-6)(9-6) = (-8)\cdot 3 = -24$

分子第二项：$(2\cdot3 - 3^2)(2\cdot3\cdot(-1) - 2) = (6-9)(-6-2) = (-3)(-8) = 24$

分子：$-24 - 24 = -48$

分母：$(3^2 - 2\cdot3)^2 = (9-6)^2 = 3^2 = 9$

因此：
$$y''(3) = -\frac{48}{9} = -\frac{16}{3}$$

曲线在 $(3,3)$ 处二阶导数为 $-\frac{16}{3}$，说明曲线在该点附近是**凸的**（向下弯曲）。

**验证**（使用通用公式）：

$$y'' = -\frac{F_{xx}F_y^2 - 2F_{xy}F_xF_y + F_{yy}F_x^2}{F_y^3}$$

计算二阶偏导：
$$F_{xx} = \frac{\partial}{\partial x}(3x^2 - 6y) = 6x,\quad F_{xy} = \frac{\partial}{\partial y}(3x^2 - 6y) = -6$$
$$F_{yx} = \frac{\partial}{\partial x}(3y^2 - 6x) = -6,\quad F_{yy} = \frac{\partial}{\partial y}(3y^2 - 6x) = 6y$$

在 $(3,3)$ 处：$F_{xx}=18$，$F_{xy}=-6$，$F_{yy}=18$，$F_x=9$，$F_y=9$。

代入公式：
$$y''(3) = -\frac{18\cdot 9^2 - 2(-6)\cdot 9\cdot 9 + 18\cdot 9^2}{9^3} = -\frac{18\cdot 81 + 2\cdot 6\cdot 81 + 18\cdot 81}{729}$$

$$= -\frac{1458 + 972 + 1458}{729} = -\frac{3888}{729} = -\frac{16}{3}$$

与直接求导结果一致。$\checkmark$

### 例题 2：三元隐函数存在定理

设 $F(x,y,z) = x^2 + y^2 + z^2 - 1 = 0$（单位球面）。

(1) 验证在点 $\left(\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}},0\right)$ 处，方程满足定理 12.10 的条件，能确定隐函数 $z=z(x,y)$。
(2) 求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$ 在该点的值。
(3) 验证 $\left(\frac{1}{\sqrt{2}},0,\frac{1}{\sqrt{2}}\right)$ 处不满足非退化条件，解释为何无法确定 $z=z(x,y)$。

**解**：

**(1) 条件验证**：

**条件 1（零点条件）**：
$$F\!\left(\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}},0\right) = \frac12 + \frac12 + 0 - 1 = 0 \quad \checkmark$$

**条件 2（光滑性条件）**：
$$F_x = 2x,\quad F_y = 2y,\quad F_z = 2z$$

三个偏导均为多项式，在 $\mathbb{R}^3$ 上处处连续。$F\in C^1$。$\checkmark$

**条件 3（非退化条件）**：
$$F_z\!\left(\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}},0\right) = 2\cdot 0 = 0$$

不满足 $F_z\neq 0$ 的条件！

因此定理 12.10 的条件**不满足**——在 $\left(\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}},0\right)$ 处，$F_z=0$，无法确定 $z=z(x,y)$。

**几何解释**：点 $\left(\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}},0\right)$ 位于球面的"赤道"上。在该点处，球面的切平面是竖直的（法向量的 $z$ 分量为 0），意味着 $z$ 不是 $x,y$ 的局部函数——事实上，在该点附近，一个 $(x,y)$ 对应两个 $z$ 值（球面的上半和下半）。

**(2) 改换点**：验证 $\left(\frac12,\frac12,\frac{1}{\sqrt{2}}\right)$ 处的情况。

**条件 1**：$F\!\left(\frac12,\frac12,\frac{1}{\sqrt{2}}\right) = \frac14 + \frac14 + \frac12 - 1 = 0 \quad \checkmark$

**条件 2**：同前，$F\in C^1$。$\checkmark$

**条件 3**：$F_z\!\left(\frac12,\frac12,\frac{1}{\sqrt{2}}\right) = 2\cdot\frac{1}{\sqrt{2}} = \sqrt{2} \neq 0 \quad \checkmark$

三个条件满足！由定理 12.10，在该点附近存在唯一隐函数 $z=z(x,y)$。

**求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$**：

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{2x}{2z} = -\frac{x}{z},\qquad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{2y}{2z} = -\frac{y}{z}$$

在 $\left(\frac12,\frac12,\frac{1}{\sqrt{2}}\right)$ 处：

$$\frac{\partial z}{\partial x}\!\left(\frac12,\frac12\right) = -\frac{1/2}{1/\sqrt{2}} = -\frac{1}{\sqrt{2}},\qquad \frac{\partial z}{\partial y}\!\left(\frac12,\frac12\right) = -\frac{1/2}{1/\sqrt{2}} = -\frac{1}{\sqrt{2}}$$

**验证**：在该点附近，取 $z = \sqrt{1-x^2-y^2}$（球面上半部分，$z>0$）。直接计算偏导：

$$\frac{\partial z}{\partial x} = \frac{-x}{\sqrt{1-x^2-y^2}} = -\frac{x}{z}$$

结果一致。$\checkmark$

**(3) 在 $\left(\frac{1}{\sqrt{2}},0,\frac{1}{\sqrt{2}}\right)$ 处**：

**条件 1**：$F\!\left(\frac{1}{\sqrt{2}},0,\frac{1}{\sqrt{2}}\right) = \frac12 + 0 + \frac12 - 1 = 0 \quad \checkmark$

**条件 2**：$F\in C^1$。$\checkmark$

**条件 3**：$F_z\!\left(\frac{1}{\sqrt{2}},0,\frac{1}{\sqrt{2}}\right) = 2\cdot\frac{1}{\sqrt{2}} = \sqrt{2} \neq 0 \quad \checkmark$

三个条件满足！因此在该点附近可以确定 $z=z(x,y)$。

**注意**：$F_z\neq 0$ 是确定 $z$ 关于 $x,y$ 的隐函数的关键。在球面上，只要 $z\neq 0$（即不在赤道上），就能在局部把 $z$ 表示为 $x,y$ 的函数（取上半球或下半球分支）。在 $z=0$ 的赤道上，$F_z=0$，此时虽不能确定 $z=z(x,y)$，但可能确定 $y=y(x,z)$ 或 $x=x(y,z)$——定理选择哪个变量作为"因变量"，取决于哪个偏导不为零。

### 例题 3：隐函数的二阶偏导计算（三元情形）

设方程 $F(x,y,z) = xyz + x^2 + y^2 + z^2 - 4 = 0$ 在点 $(1,1,1)$ 附近确定了隐函数 $z=z(x,y)$。求 $\frac{\partial^2 z}{\partial x^2}$ 在 $(1,1)$ 处的值。

**解**：

**第 1 步：验证条件**。

$$F(1,1,1) = 1+1+1+1-4 = 0 \quad \checkmark$$

$$F_x = yz + 2x,\quad F_y = xz + 2y,\quad F_z = xy + 2z$$

$F_x,F_y,F_z$ 均为多项式，处处连续。$\checkmark$

$$F_z(1,1,1) = 1\cdot1 + 2\cdot1 = 3 \neq 0 \quad \checkmark$$

定理 12.10 条件满足。

**第 2 步：求一阶偏导**。

由公式 $\displaystyle\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}$：

$$\frac{\partial z}{\partial x} = -\frac{yz + 2x}{xy + 2z}$$

在 $(1,1,1)$ 处，$z=1$，代入：

$$\frac{\partial z}{\partial x}\bigg|_{(1,1)} = -\frac{1\cdot1 + 2\cdot1}{1\cdot1 + 2\cdot1} = -\frac{3}{3} = -1$$

类似地：
$$\frac{\partial z}{\partial y} = -\frac{xz + 2y}{xy + 2z},\qquad \frac{\partial z}{\partial y}\bigg|_{(1,1)} = -\frac{1\cdot1 + 2\cdot1}{3} = -1$$

**第 3 步：求二阶偏导 $\frac{\partial^2 z}{\partial x^2}$**。

对 $\displaystyle\frac{\partial z}{\partial x} = -\frac{yz + 2x}{xy + 2z}$ 再关于 $x$ 求偏导。注意 $z=z(x,y)$，且 $y$ 视为常数：

$$\frac{\partial^2 z}{\partial x^2} = -\frac{\partial}{\partial x}\!\left(\frac{yz + 2x}{xy + 2z}\right)$$

使用商的求导法则。记分子 $U = yz + 2x$，分母 $V = xy + 2z$：

先求 $\frac{\partial U}{\partial x}$ 和 $\frac{\partial V}{\partial x}$（注意 $z$ 是 $x,y$ 的函数）：

$$\frac{\partial U}{\partial x} = y\cdot\frac{\partial z}{\partial x} + 2 = y\cdot z_x + 2$$
$$\frac{\partial V}{\partial x} = y + 2\cdot\frac{\partial z}{\partial x} = y + 2\,z_x$$

因此：

$$\frac{\partial^2 z}{\partial x^2} = -\frac{(y\,z_x + 2)(xy + 2z) - (yz + 2x)(y + 2\,z_x)}{(xy + 2z)^2}$$

代入 $(x,y,z) = (1,1,1)$ 和 $z_x = -1$：

分子第一项：$(1\cdot(-1) + 2)(1\cdot1 + 2\cdot1) = (-1+2)(1+2) = 1\cdot 3 = 3$

分子第二项：$(1\cdot1 + 2\cdot1)(1 + 2\cdot(-1)) = (1+2)(1-2) = 3\cdot(-1) = -3$

分子：$3 - (-3) = 6$

分母：$(1\cdot1 + 2\cdot1)^2 = 3^2 = 9$

因此：

$$\frac{\partial^2 z}{\partial x^2}\bigg|_{(1,1)} = -\frac{6}{9} = -\frac{2}{3}$$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 隐函数存在定理保证方程 $F(x,y)=0$ 在整个定义域上都能确定隐函数 | 定理只保证**局部存在性**——仅在 $(x_0,y_0)$ 的某个邻域内存在唯一隐函数。离开该邻域，可能不存在、不唯一或不再可导 |
| $F_y(x_0,y_0)\neq 0$ 是判断隐函数存在性的唯一条件 | 三个条件缺一不可：零点条件（点在曲线上）、光滑性条件（偏导连续）、非退化条件（$F_y\neq 0$）。缺少任何一个定理都不成立 |
| 隐函数求导公式 $dy/dx = -F_x/F_y$ 是"将 $F$ 视为常数后求导" | 推导过程是：对恒等式 $F(x,y(x))\equiv 0$ 使用链式法则，得到 $F_x + F_y\cdot y' = 0$，解出 $y' = -F_x/F_y$。$F_x$ 是 $F$ 对第一个位置变量的**偏导**，计算时将 $y$ 视为常数 |
| 计算 $d^2y/dx^2$ 时，直接对 $-F_x/F_y$ 再求导，但忘记 $F_x$ 和 $F_y$ 中的 $y$ 是 $x$ 的函数 | $F_x$ 和 $F_y$ 中的 $y$ 必须视为 $y(x)$，对它们求 $x$ 的导数时需要链式法则：$\frac{d}{dx}F_x = F_{xx}+F_{xy}y'$ |
| 在多元情形 $F(x,y,z)=0$ 中，只要 $F_z\neq 0$，就能把 $z$ 表示为 $x,y$ 的显函数 | 定理保证的是**局部存在性**和**可微性**，但并不意味着 $z$ 有全局的显式表达式。例如 $x^2+y^2+z^2-1=0$，$z\neq 0$ 时局部可解出 $z=\pm\sqrt{1-x^2-y^2}$，但全局上每个 $(x,y)$ 对应两个 $z$ 值，不存在全局的隐函数 |
| 隐函数定理的"唯一性"意味着全局唯一 | "唯一"是指在 $(x_0,y_0)$ 的邻域内唯一——在该局部邻域中，只有一条曲线通过 $(x_0,y_0)$ 且满足方程。同一方程在其他点的邻域可能有不同的隐函数分支 |

### 检查点

- [ ] 能否完整陈述隐函数存在定理（定理 12.9）的三个条件和结论？
- [ ] 能否从链式法则推导隐函数求导公式 $dy/dx = -F_x/F_y$？
- [ ] 能否解释为什么 $F_y(x_0,y_0)\neq 0$ 条件不可少？
- [ ] 对于给定的方程 $F(x,y)=0$ 和点 $(x_0,y_0)$，能否逐条验证三个条件？
- [ ] 能否将定理 12.9 推广到三元情形 $F(x,y,z)=0$（定理 12.10）？
- [ ] 能否推导并计算隐函数的二阶导数？
- [ ] 能否解释隐函数存在定理中"存在性"与"计算性"的分工？
- [ ] 能否区分局部存在性与全局存在性的区别？

<details><summary>参考答案</summary>

令 $F(x,y) = y - \ln(x^2+y^2)$。

**条件 1（零点条件）**：
$$F(1,0) = 0 - \ln(1+0) = 0 - 0 = 0 \quad \checkmark$$

**条件 2（光滑性条件）**：
$$F_x = -\frac{2x}{x^2+y^2},\qquad F_y = 1 - \frac{2y}{x^2+y^2}$$

在 $(1,0)$ 附近（$x^2+y^2>0$ 的区域），$F_x$ 和 $F_y$ 均为初等函数的组合，因此连续。$F\in C^1$。$\checkmark$

**条件 3（非退化条件）**：
$$F_y(1,0) = 1 - \frac{0}{1+0} = 1 \neq 0 \quad \checkmark$$

三个条件满足。由定理 12.9，存在唯一隐函数 $y=y(x)$，且 $y(1)=0$。

**求导**：

$$y'(1) = -\frac{F_x(1,0)}{F_y(1,0)} = -\frac{-2\cdot1/(1+0)}{1} = 2$$

因此曲线在 $(1,0)$ 处的切线斜率为 $2$。

$$\frac{\partial z}{\partial x} = \frac{x}{\sqrt{x^2+y^2+1}} = \frac{x}{z},\qquad \frac{\partial z}{\partial y} = \frac{y}{\sqrt{x^2+y^2+1}} = \frac{y}{z}$$

结果一致。$\checkmark$

**3.** 设方程 $x^2 + 2xy - y^2 + 4 = 0$ 在点 $(1,3)$ 附近确定了隐函数 $y=y(x)$。
(1) 验证隐函数存在定理的条件。
(2) 求 $y'(1)$ 和 $y''(1)$。

<details><summary>参考答案</summary>

**(1) 条件验证**：

令 $F(x,y) = x^2 + 2xy - y^2 + 4$。

**条件 1**：
$$F(1,3) = 1 + 6 - 9 + 4 = 2 \neq 0$$

不满足零点条件！因此在 $(1,3)$ 处**不能**应用隐函数存在定理。

**正确的点**：解方程 $x^2 + 2xy - y^2 + 4 = 0$ 在 $x=1$ 时的 $y$ 值：
$$1 + 2y - y^2 + 4 = 0 \quad\Rightarrow\quad -y^2 + 2y + 5 = 0$$

解得 $y = 1 \pm \sqrt{6}$。点 $(1,1+\sqrt{6})$ 在曲线上，我们在此点应用定理。

在 $(1,1+\sqrt{6})$ 处验证：

$F_x = 2x + 2y$，$F_y = 2x - 2y$。均为多项式，连续。$\checkmark$

$$F_y(1,1+\sqrt{6}) = 2 - 2(1+\sqrt{6}) = 2 - 2 - 2\sqrt{6} = -2\sqrt{6} \neq 0 \quad \checkmark$$

条件满足。

**(2) 求导**：

$$y' = -\frac{F_x}{F_y} = -\frac{2x+2y}{2x-2y} = -\frac{x+y}{x-y}$$

在 $(1,1+\sqrt{6})$ 处：
$$y' = -\frac{1+1+\sqrt{6}}{1-(1+\sqrt{6})} = -\frac{2+\sqrt{6}}{-\sqrt{6}} = \frac{2+\sqrt{6}}{\sqrt{6}} = 1 + \frac{2}{\sqrt{6}} = 1 + \frac{\sqrt{6}}{3}$$

**求 $y''$**：对 $y' = -\dfrac{x+y}{x-y}$ 再求导。

$$y' = -\frac{x+y}{x-y}$$

使用商的求导法则：

$$
\begin{aligned}
y'' &= -\frac{(1+y')(x-y) - (x+y)(1-y')}{(x-y)^2} \\
&= -\frac{(x-y) + y'(x-y) - (x+y) + y'(x+y)}{(x-y)^2} \\
&= -\frac{(x-y-x-y) + y'(x-y+x+y)}{(x-y)^2} \\
&= -\frac{-2y + y'(2x)}{(x-y)^2} \\
&= -\frac{2(xy' - y)}{(x-y)^2}
\end{aligned}
$$

代入 $x=1$，$y=1+\sqrt{6}$，$y'=1+\frac{\sqrt{6}}{3}$：

先算 $xy' - y = 1\cdot\left(1+\frac{\sqrt{6}}{3}\right) - (1+\sqrt{6}) = 1 + \frac{\sqrt{6}}{3} - 1 - \sqrt{6} = -\frac{2\sqrt{6}}{3}$

再算分母 $(x-y)^2 = (1-1-\sqrt{6})^2 = (-\sqrt{6})^2 = 6$

$$y''(1) = -\frac{2\cdot\left(-\frac{2\sqrt{6}}{3}\right)}{6} = \frac{4\sqrt{6}/3}{6} = \frac{4\sqrt{6}}{18} = \frac{2\sqrt{6}}{9}$$

因此 $y''(1) = \dfrac{2\sqrt{6}}{9}$。

</details>

### 迁移应用

**4.** 设方程 $e^{xy} + yz + xz^2 = 3$ 在点 $(0,1,2)$ 附近确定了隐函数 $z=z(x,y)$。求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$ 在 $(0,1)$ 处的值。

<details><summary>参考答案</summary>

令 $F(x,y,z) = e^{xy} + yz + xz^2 - 3$。

**条件验证**：

$$F(0,1,2) = e^{0} + 1\cdot2 + 0\cdot4 - 3 = 1 + 2 + 0 - 3 = 0 \quad \checkmark$$

$$F_x = ye^{xy} + z^2,\quad F_y = xe^{xy} + z,\quad F_z = y + 2xz$$

均为初等函数组合，在 $(0,1,2)$ 附近连续。$\checkmark$

$$F_z(0,1,2) = 1 + 2\cdot 0 \cdot 2 = 1 \neq 0 \quad \checkmark$$

条件满足。

**求偏导**：

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{ye^{xy} + z^2}{y + 2xz}$$

代入 $(x,y,z) = (0,1,2)$：
$$\frac{\partial z}{\partial x}\bigg|_{(0,1)} = -\frac{1\cdot e^0 + 4}{1 + 0} = -\frac{1+4}{1} = -5$$

$$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{xe^{xy} + z}{y + 2xz}$$

代入 $(0,1,2)$：
$$\frac{\partial z}{\partial y}\bigg|_{(0,1)} = -\frac{0\cdot e^0 + 2}{1 + 0} = -2$$

因此 $\dfrac{\partial z}{\partial x}(0,1) = -5$，$\dfrac{\partial z}{\partial y}(0,1) = -2$。

</details>

**5.** 对于方程 $F(x,y) = y^3 + y - x^2 = 0$，证明：对任意 $x\in\mathbb{R}$，方程唯一确定一个隐函数 $y=y(x)$，并求 $y'(0)$ 和 $y''(0)$。

<details><summary>参考答案</summary>

**全局存在性分析**：

方程 $y^3 + y - x^2 = 0$ 可以写成 $y(y^2+1) = x^2$。

对于任意给定的 $x\in\mathbb{R}$，考虑函数 $g(y) = y^3 + y - x^2$。因为 $g'(y) = 3y^2+1 > 0$ 对一切 $y$ 成立，所以 $g(y)$ 是严格单调递增的连续函数。又因为 $\lim_{y\to-\infty} g(y) = -\infty$，$\lim_{y\to+\infty} g(y) = +\infty$，由介值定理，对每个 $x$ 存在唯一的 $y$ 使得 $g(y)=0$。

因此方程全局唯一确定隐函数 $y=y(x)$。

**在 $x=0$ 处的验证**：

当 $x=0$ 时，$y^3 + y = 0$ 即 $y(y^2+1)=0$，解得 $y=0$。因此 $(x_0,y_0) = (0,0)$。

$F(x,y) = y^3 + y - x^2$。

$F_x = -2x$，$F_y = 3y^2 + 1$。在 $(0,0)$ 处：$F_y = 1 \neq 0$。条件满足。

**求导**：

$$y' = -\frac{F_x}{F_y} = -\frac{-2x}{3y^2+1} = \frac{2x}{3y^2+1}$$

在 $x=0$ 处，$y=0$：
$$y'(0) = \frac{0}{1} = 0$$

**求 $y''$**：

对 $y' = \dfrac{2x}{3y^2+1}$ 再求导：

$$
\begin{aligned}
y'' &= \frac{2(3y^2+1) - 2x\cdot 6y\cdot y'}{(3y^2+1)^2} \\
&= \frac{2(3y^2+1) - 12xy\,y'}{(3y^2+1)^2}
\end{aligned}
$$

代入 $x=0$，$y=0$，$y'=0$：
$$y''(0) = \frac{2(0+1) - 0}{(0+1)^2} = 2$$

因此 $y'(0)=0$，$y''(0)=2$，说明在 $x=0$ 处函数有局部极小值。

</details>
