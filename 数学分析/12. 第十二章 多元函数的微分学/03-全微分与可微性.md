## 2. 背景与应用场景

在文件 01 中，我们学习了全微分的定义（定义 12.3）：

> 若存在常数 $A,B$ 使得 $\Delta f = A\Delta x + B\Delta y + o(\|\Delta P\|)$，则称 $f$ 在 $(x_0,y_0)$ 处**可微**，并称 $A\Delta x + B\Delta y$ 为全微分 $\mathrm{d}f$。

并证明了定理 12.1：**可微 $\Rightarrow$ 偏导存在**，且 $A = f_x(x_0,y_0)$、$B = f_y(x_0,y_0)$。

然而，这一方向的反向并不成立。以下两个关键问题构成了本文件的核心内容：

**问题一（可微的必要条件链的完善）**：在一元函数中，我们有"可导 $\Rightarrow$ 连续"（定理 4.3）。在多元函数中，可微性与连续性之间是什么关系？是否可微的函数必然连续？这是**可微 $\Rightarrow$ 连续**的方向。

**问题二（可微的充分条件）**：偏导存在不能推出可微——这是多元微分学与一元微分学最本质的区别之一。那么，在偏导存在的基础上，需要附加什么条件才能保证可微？这个**充分条件**是什么，如何证明？

**这些问题的实际意义**：
- 在工程应用中，我们经常需要判断一个函数在某点是否可微（即能否用线性模型近似）
- 直接验证可微的定义（计算极限 $\lim_{\|\Delta P\|\to0} \frac{\Delta f - f_x\Delta x - f_y\Delta y}{\|\Delta P\|}=0$）通常计算量很大
- 定理 12.6（偏导连续 $\Rightarrow$ 可微）提供了一个**实用的判别准则**——只需验证偏导函数是否连续，而连续性的验证往往比可微性的直接验证简单得多
- 理解偏导存在与可微之间的差距，能帮助我们避免"认为偏导存在就是可微"的常见错误

**知识链条中的位置**：

$$
\text{偏导连续} \;\xrightarrow{\text{定理 12.6}}\; \text{可微} \;\xrightarrow{\text{定理 12.5}}\; \text{连续} \;\xrightarrow{\text{定理 11.10}}\; \text{极限存在}
$$

而定理 12.1 又给出：**可微 $\Rightarrow$ 偏导存在**。因此可微在这条链中位于"偏导存在"和"连续"之间——它比偏导存在更强，但弱于偏导连续。

## 3. 核心概念与符号约定

### 符号表

| 符号 | 含义 | 说明 |
|------|------|------|
| $\Delta f$ | 函数增量 $f(x_0+\Delta x,y_0+\Delta y)-f(x_0,y_0)$ | 自变量的总变化导致的函数值变化 |
| $\|\Delta P\|$ 或 $\sqrt{(\Delta x)^2+(\Delta y)^2}$ | 自变量的变化距离 | Euclid 范数，即 $\|\Delta P\| = \sqrt{\Delta x^2+\Delta y^2}$ |
| $o(\|\Delta P\|)$ | 比 $\|\Delta P\|$ 高阶的无穷小 | $\displaystyle\lim_{\|\Delta P\|\to0}\frac{o(\|\Delta P\|)}{\|\Delta P\|}=0$ |
| $\varepsilon_1,\varepsilon_2$ | 趋于零的误差项 | 用于定理 12.6 的证明中表示偏导的连续误差 |
| $f\in C^1$ | $f$ 的一阶偏导连续 | 即 $f_x,f_y$ 均连续 |
| $f\in C^0$ | $f$ 连续 | 即 $f$ 本身连续 |

### 3.1 可微定义回顾与 $o(\|\Delta P\|)$ 深度分析

在文件 01 中，我们给出了全微分的定义（定义 12.3）：

**定义 12.3（回顾）**：设二元函数 $z=f(x,y)$ 在点 $(x_0,y_0)$ 的某邻域内有定义。若存在常数 $A,B$（与 $(x_0,y_0)$ 有关但与 $\Delta x,\Delta y$ 无关），使得当 $\Delta x,\Delta y$ 充分小时，

$$\Delta f = f(x_0+\Delta x, y_0+\Delta y) - f(x_0,y_0) = A\Delta x + B\Delta y + o\!\left(\sqrt{(\Delta x)^2+(\Delta y)^2}\right)$$

则称 $f$ 在 $(x_0,y_0)$ 处**可微**，并称 $A\Delta x + B\Delta y$ 为 $f$ 在 $(x_0,y_0)$ 处的**全微分**，记作 $\mathrm{d}f$。

并且定理 12.1 告诉我们，如果 $f$ 可微，则 $A = f_x(x_0,y_0)$、$B = f_y(x_0,y_0)$，因此全微分公式为：

$$\mathrm{d}f = f_x(x_0,y_0)\,\mathrm{d}x + f_y(x_0,y_0)\,\mathrm{d}y$$

**$o(\|\Delta P\|)$ 的精确含义**：

记号 $o(\|\Delta P\|)$ 代表一个比 $\|\Delta P\|$ 更快趋于零的量。更精确地说，若记

$$R(\Delta x,\Delta y) = \Delta f - f_x(x_0,y_0)\Delta x - f_y(x_0,y_0)\Delta y$$

则 $R(\Delta x,\Delta y) = o(\|\Delta P\|)$ 意味着：

$$\boxed{\lim_{(\Delta x,\Delta y)\to(0,0)} \frac{R(\Delta x,\Delta y)}{\sqrt{(\Delta x)^2+(\Delta y)^2}} = 0}$$

这是可微性定义的核心等式。其含义是：当 $(\Delta x,\Delta y)$ 充分接近 $(0,0)$ 时，剩余项 $R$ 与距离 $\|\Delta P\|$ 之比趋于零——也就是说，函数增量与线性近似的差是比自变量的变化距离更高阶的无穷小。

**与一元微分的对比**：

| 对比项 | 一元微分 $\mathrm{d}y$ | 多元全微分 $\mathrm{d}f$ |
|--------|----------------------|------------------------|
| 增量分解 | $\Delta y = f'(x_0)\Delta x + o(\Delta x)$ | $\Delta f = f_x\Delta x + f_y\Delta y + o(\|\Delta P\|)$ |
| 剩余项阶数 | $o(\Delta x)$（与 $\Delta x$ 比） | $o(\|\Delta P\|)$（与 $\|\Delta P\|$ 比） |
| 等价极限形式 | $\displaystyle\lim_{\Delta x\to0}\frac{\Delta y - f'(x_0)\Delta x}{\Delta x}=0$ | $\displaystyle\lim_{\|\Delta P\|\to0}\frac{\Delta f - f_x\Delta x - f_y\Delta y}{\|\Delta P\|}=0$ |
| 几何含义 | 切线近似曲线 | 切平面近似曲面 |

**可微性的另一种等价刻画**（对理解后续证明有帮助）：

$f$ 在 $(x_0,y_0)$ 处可微当且仅当存在 $f_x(x_0,y_0),f_y(x_0,y_0)$ 且

$$\Delta f = f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y + \varepsilon_1\Delta x + \varepsilon_2\Delta y$$

其中 $\varepsilon_1,\varepsilon_2\to0$ 当 $(\Delta x,\Delta y)\to(0,0)$。这种形式将剩余项 $o(\|\Delta P\|)$ 分解为两个"小量乘以增量"的形式，在定理 12.6 的证明中会自然出现。

**两种形式的等价性**：若有 $\Delta f = f_x\Delta x + f_y\Delta y + \varepsilon_1\Delta x + \varepsilon_2\Delta y$ 且 $\varepsilon_1,\varepsilon_2\to0$，则

$$\left|\frac{\varepsilon_1\Delta x + \varepsilon_2\Delta y}{\|\Delta P\|}\right| \leq \frac{|\varepsilon_1||\Delta x| + |\varepsilon_2||\Delta y|}{\|\Delta P\|} \leq |\varepsilon_1| + |\varepsilon_2| \to 0$$

因此 $\varepsilon_1\Delta x + \varepsilon_2\Delta y = o(\|\Delta P\|)$。反过来，若 $R = o(\|\Delta P\|)$，可以构造 $\varepsilon_1 = \frac{R\Delta x}{2\|\Delta P\|^2}$ 等，但一般不作此要求。重要的是记住：**这两种形式在本质上等价**，后面的证明中两种都会用到。

## 4. 原理与方法

### 4.1 可微 $\Rightarrow$ 连续（定理 12.5）

我们首先来完善可微的必要条件链：证明可微的函数必然连续。

**定理 12.5（可微与连续的关系）**：设 $f(x,y)$ 在点 $(x_0,y_0)$ 处可微，则 $f$ 在 $(x_0,y_0)$ 处连续。

**证明**：

由 $f$ 在 $(x_0,y_0)$ 处可微的定义，存在常数 $A = f_x(x_0,y_0)$、$B = f_y(x_0,y_0)$，使得：

$$\Delta f = f(x_0+\Delta x,\ y_0+\Delta y) - f(x_0,y_0) = A\Delta x + B\Delta y + o\!\left(\sqrt{(\Delta x)^2+(\Delta y)^2}\right)$$

我们要证明 $\displaystyle\lim_{(\Delta x,\Delta y)\to(0,0)} \Delta f = 0$，即 $f$ 在 $(x_0,y_0)$ 处连续。

**推导**：

$$\Delta f = A\Delta x + B\Delta y + o(\|\Delta P\|)$$

对等式两边取绝对值，由三角不等式：

$$|\Delta f| \leq |A||\Delta x| + |B||\Delta y| + |o(\|\Delta P\|)|$$

以下逐项分析当 $(\Delta x,\Delta y)\to(0,0)$ 时的极限：

**第一项** $|A||\Delta x|$：由于 $|\Delta x| \leq \sqrt{(\Delta x)^2+(\Delta y)^2} = \|\Delta P\|$，当 $\|\Delta P\|\to0$ 时 $|\Delta x|\to0$，从而 $|A||\Delta x|\to0$。

**第二项** $|B||\Delta y|$：同理，$|\Delta y| \to 0$，从而 $|B||\Delta y|\to0$。

**第三项** $|o(\|\Delta P\|)|$：由 $o(\|\Delta P\|)$ 的定义，$\displaystyle\lim_{\|\Delta P\|\to0}\frac{o(\|\Delta P\|)}{\|\Delta P\|}=0$，这意味着当 $\|\Delta P\|$ 充分小时，$|o(\|\Delta P\|)| \leq \|\Delta P\|$（取 $\varepsilon=1$ 存在 $\delta$ 使得当 $\|\Delta P\|<\delta$ 时 $|o(\|\Delta P\|)| < \|\Delta P\|$）。因此当 $\|\Delta P\|\to0$ 时 $o(\|\Delta P\|)\to0$。

**综合**：三部分均趋于零，因此 $\displaystyle\lim_{(\Delta x,\Delta y)\to(0,0)} \Delta f = 0$。由连续性的极限形式（第十一章第 3.2 节），$\displaystyle\lim_{(x,y)\to(x_0,y_0)} f(x,y) = f(x_0,y_0)$，故 $f$ 在 $(x_0,y_0)$ 处连续。证毕。

**与一元函数"可导 $\Rightarrow$ 连续"（定理 4.3）的逐项类比**：

| 对比项 | 一元函数 | 二元函数 |
|--------|---------|---------|
| 条件 | $f'(x_0)$ 存在 | $f$ 在 $(x_0,y_0)$ 可微 |
| 增量分解 | $\Delta y = f'(x_0)\Delta x + o(\Delta x)$ | $\Delta f = f_x\Delta x + f_y\Delta y + o(\|\Delta P\|)$ |
| 证明关键 | $|\Delta y| \leq |f'(x_0)||\Delta x| + |o(\Delta x)| \to 0$ | $|\Delta f| \leq |f_x||\Delta x| + |f_y||\Delta y| + |o(\|\Delta P\|)| \to 0$ |
| 结论 | $f$ 在 $x_0$ 连续 | $f$ 在 $(x_0,y_0)$ 连续 |

**重要观察**：在一元函数中，"可导 $\Rightarrow$ 连续"的证明依赖于 $\Delta y = f'(x_0)\Delta x + o(\Delta x)$，其中 $|o(\Delta x)|/\Delta x \to 0$，所以 $o(\Delta x) \to 0$。在二元函数中，证明完全平行——只是线性主部变成了两项之和，剩余项的阶也变成了 $o(\|\Delta P\|)$。两者的核心逻辑相同：**增量等于线性主部加上高阶无穷小，线性主部和高阶无穷小都趋于零，因此增量趋于零，函数连续**。

**定理 12.5 的逆否命题**：若 $f$ 在 $(x_0,y_0)$ 处不连续，则 $f$ 在 $(x_0,y_0)$ 处不可微。这一形式在判断不可微时非常有用——只需证明函数不连续，即可立即断定不可微。

**可微的必要条件链的总结**：

$$\boxed{\text{可微} \;\Rightarrow\; \begin{cases}
\text{偏导存在} & \text{（定理 12.1）} \\
\text{连续} & \text{（定理 12.5）}
\end{cases}}$$

注意这两个必要条件**不能逆向**推出可微——偏导存在不保证可微，连续也不保证可微（甚至连续+偏导存在也不保证可微，见下面的反例）。

### 4.2 偏导存在 ≠ 可微——反例分析

这是多元微分学与一元微分学最关键的差异之一。在一元函数中，可导与可微是等价的（定理 4.19）。但在多元函数中，**偏导存在只是可微的必要条件，绝不是充分条件**。

为什么？因为偏导只刻画了沿坐标轴方向的变化率，而可微要求**在所有方向**上函数都能被线性近似一致逼近。这是一个更强得多的要求。

**反例 1（偏导存在但函数不连续，从而不可微）**：

考虑函数

$$f(x,y) = \begin{cases}
\dfrac{xy}{x^2+y^2}, & (x,y) \neq (0,0) \\[6pt]
0, & (x,y) = (0,0)
\end{cases}$$

**第一步：验证偏导存在**。

用定义计算 $f_x(0,0)$：

$$f_x(0,0) = \lim_{h\to0} \frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0} \frac{\frac{h\cdot0}{h^2+0^2} - 0}{h} = \lim_{h\to0} \frac{0}{h} = 0$$

同理，$f_y(0,0) = 0$。因此 $f_x(0,0)$ 和 $f_y(0,0)$ 都存在（均为 $0$）。

**第二步：验证函数不连续**。

沿直线 $y=x$ 趋近 $(0,0)$：

$$f(x,x) = \frac{x\cdot x}{x^2+x^2} = \frac{x^2}{2x^2} = \frac12 \quad (x\neq0)$$

因此 $\displaystyle\lim_{k\to\infty} f\!\left(\frac1k,\frac1k\right) = \frac12 \neq 0 = f(0,0)$。由连续性的 Heine 序列刻画（定理 11.9），$f$ 在 $(0,0)$ 处不连续。

**第三步：由定理 12.5 的逆否命题**，由于 $f$ 在 $(0,0)$ 处不连续，故 $f$ 不可微。

这个反例说明：**偏导存在连连续性都不能保证，因此更不可能推出可微**。

**反例 2（偏导存在且连续，但不可微）**：

考虑函数

$$f(x,y) = \begin{cases}
\dfrac{x^2y}{x^2+y^2}, & (x,y) \neq (0,0) \\[6pt]
0, & (x,y) = (0,0)
\end{cases}$$

这个函数比反例 1 更精细——它在 $(0,0)$ 处连续，偏导也存在，但仍然不可微。

**第一步：验证偏导存在**。

$$f_x(0,0) = \lim_{h\to0} \frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0} \frac{0-0}{h} = 0$$

$$f_y(0,0) = \lim_{k\to0} \frac{f(0,k)-f(0,0)}{k} = \lim_{k\to0} \frac{0-0}{k} = 0$$

偏导存在。

**第二步：验证连续性**。

当 $(x,y)\neq(0,0)$ 时，$|f(x,y)| = \frac{x^2|y|}{x^2+y^2}$。注意 $x^2 \leq x^2+y^2$，因此

$$|f(x,y)| \leq \frac{(x^2+y^2)|y|}{x^2+y^2} = |y|$$

所以 $|f(x,y)-f(0,0)| = |f(x,y)| \leq |y| \leq \sqrt{x^2+y^2}$。对于任意 $\varepsilon>0$，取 $\delta=\varepsilon$，则当 $\|(x,y)-(0,0)\|<\delta$ 时 $|f(x,y)-f(0,0)|<\varepsilon$。故 $f$ 在 $(0,0)$ 处连续。

因此连续性条件和偏导存在性同时满足——这些仍不足以推出可微。

**第三步：检验可微性定义**。

假设 $f$ 在 $(0,0)$ 处可微，则 $A = f_x(0,0)=0$，$B = f_y(0,0)=0$，全微分为 $\mathrm{d}f=0$。可微性要求：

$$\lim_{(\Delta x,\Delta y)\to(0,0)} \frac{f(0+\Delta x,0+\Delta y) - f(0,0) - 0\cdot\Delta x - 0\cdot\Delta y}{\sqrt{(\Delta x)^2+(\Delta y)^2}} = 0$$

代入 $f$ 的表达式（当 $(\Delta x,\Delta y)\neq(0,0)$）：

$$\frac{f(\Delta x,\Delta y) - 0}{\sqrt{(\Delta x)^2+(\Delta y)^2}} = \frac{\frac{(\Delta x)^2\Delta y}{(\Delta x)^2+(\Delta y)^2}}{\sqrt{(\Delta x)^2+(\Delta y)^2}} = \frac{(\Delta x)^2\Delta y}{\big((\Delta x)^2+(\Delta y)^2\big)^{3/2}}$$

现在考虑沿直线 $\Delta y = \Delta x$ 趋近：

$$\frac{(\Delta x)^2\cdot\Delta x}{\big((\Delta x)^2+(\Delta x)^2\big)^{3/2}} = \frac{(\Delta x)^3}{(2(\Delta x)^2)^{3/2}} = \frac{(\Delta x)^3}{2^{3/2}| \Delta x|^3} = \frac{1}{2^{3/2}}\cdot\frac{(\Delta x)^3}{|\Delta x|^3}$$

当 $\Delta x > 0$ 时，$\frac{(\Delta x)^3}{|\Delta x|^3} = 1$，此比值为 $\frac{1}{2^{3/2}} \neq 0$。

因此极限不存在（更精确地说，沿不同路径得到不同的极限值或非零极限），不满足可微定义。故 $f$ 在 $(0,0)$ 处**不可微**。

**反例小结**：

| 反例 | 偏导存在 | 连续 | 可微 |
|:----:|:--------:|:----:|:----:|
| $f(x,y)=\dfrac{xy}{x^2+y^2}$ | **是** | 否 | 否 |
| $f(x,y)=\dfrac{x^2y}{x^2+y^2}$ | **是** | **是** | 否 |

这两个反例揭示了可微性要求比偏导存在+连续更强。直观理解：可微要求用线性函数 $L(\Delta x,\Delta y) = f_x\Delta x + f_y\Delta y$ 来近似 $\Delta f$ 时，误差相对于 $\|\Delta P\|$ 是高阶无穷小——这个要求沿**所有方向**都必须一致成立。仅仅沿坐标轴方向的变化率与线性近似匹配是不够的。

### 4.3 可微的充分条件——偏导连续定理（定理 12.6）

既然偏导存在不能保证可微，那么附加什么条件能保证可微？答案是：**偏导数在一点附近存在且在该点连续**。

**定理 12.6（可微的充分条件）**：设二元函数 $z=f(x,y)$ 在点 $(x_0,y_0)$ 的某邻域 $U$ 上有定义。若 $f$ 的一阶偏导数 $f_x$ 和 $f_y$ 在 $U$ 上存在，且 $f_x$ 和 $f_y$ 在 $(x_0,y_0)$ 处连续，则 $f$ 在 $(x_0,y_0)$ 处可微。

**证明**：

我们需要证明：存在常数 $A,B$（将由 $f_x(x_0,y_0)$ 和 $f_y(x_0,y_0)$ 充当）使得

$$\Delta f = f(x_0+\Delta x,\ y_0+\Delta y) - f(x_0,y_0) = A\Delta x + B\Delta y + o(\|\Delta P\|)$$

**第一步：将增量分解为两个一元增量之和**。

将 $\Delta f$ 写成两个步骤（先变 $x$ 再变 $y$）：

$$
\begin{aligned}
\Delta f &= \big[f(x_0+\Delta x,\ y_0+\Delta y) - f(x_0,\ y_0+\Delta y)\big] + \big[f(x_0,\ y_0+\Delta y) - f(x_0,\ y_0)\big] \\
&\triangleq \Delta_1 f + \Delta_2 f
\end{aligned}
$$

其中：
- $\Delta_1 f = f(x_0+\Delta x,\ y_0+\Delta y) - f(x_0,\ y_0+\Delta y)$：固定 $y=y_0+\Delta y$，仅让 $x$ 从 $x_0$ 变化到 $x_0+\Delta x$
- $\Delta_2 f = f(x_0,\ y_0+\Delta y) - f(x_0,\ y_0)$：固定 $x=x_0$，仅让 $y$ 从 $y_0$ 变化到 $y_0+\Delta y$

**第二步：对每个一元增量应用 Lagrange 中值定理**。

对于 $\Delta_1 f$，将 $y = y_0+\Delta y$ 固定，则 $\varphi(x) = f(x,\ y_0+\Delta y)$ 是 $x$ 的一元函数。由于 $f_x$ 在 $U$ 上存在，$\varphi$ 在 $[x_0,\ x_0+\Delta x]$（或 $[x_0+\Delta x,\ x_0]$，取决于 $\Delta x$ 的符号）上可导。由 Lagrange 中值定理（定理 5.2），存在 $\xi$ 介于 $x_0$ 和 $x_0+\Delta x$ 之间，使得：

$$\Delta_1 f = f(x_0+\Delta x,\ y_0+\Delta y) - f(x_0,\ y_0+\Delta y) = f_x(\xi,\ y_0+\Delta y) \cdot \Delta x$$

对于 $\Delta_2 f$，固定 $x = x_0$，$\psi(y) = f(x_0,\ y)$ 是 $y$ 的一元函数。同理，存在 $\eta$ 介于 $y_0$ 和 $y_0+\Delta y$ 之间，使得：

$$\Delta_2 f = f(x_0,\ y_0+\Delta y) - f(x_0,\ y_0) = f_y(x_0,\ \eta) \cdot \Delta y$$

因此：

$$\Delta f = f_x(\xi,\ y_0+\Delta y)\Delta x + f_y(x_0,\ \eta)\Delta y$$

**第三步：利用偏导的连续性将中值点处的偏导替换为 $(x_0,y_0)$ 处的偏导**。

由于 $f_x$ 在 $(x_0,y_0)$ 处连续，当 $(\Delta x,\Delta y)\to(0,0)$ 时，$(\xi,\ y_0+\Delta y) \to (x_0,y_0)$（因为 $\xi$ 在 $x_0$ 和 $x_0+\Delta x$ 之间，$\Delta x\to0$ 时 $\xi\to x_0$）。因此：

$$f_x(\xi,\ y_0+\Delta y) = f_x(x_0,y_0) + \varepsilon_1$$

其中 $\varepsilon_1 \to 0$ 当 $(\Delta x,\Delta y)\to(0,0)$。

同理，由于 $f_y$ 在 $(x_0,y_0)$ 处连续，且 $\eta$ 在 $y_0$ 和 $y_0+\Delta y$ 之间，当 $\Delta y\to0$ 时 $\eta\to y_0$，因此：

$$f_y(x_0,\ \eta) = f_y(x_0,y_0) + \varepsilon_2$$

其中 $\varepsilon_2 \to 0$ 当 $(\Delta x,\Delta y)\to(0,0)$。（注意 $\varepsilon_2$ 只依赖于 $\Delta y$，但也受到 $\Delta x\to0$ 的约束，总地趋于零。）

**第四步：代入并整理**。

将上述表达式代入 $\Delta f$ 的分解式：

$$
\begin{aligned}
\Delta f &= \big[f_x(x_0,y_0) + \varepsilon_1\big]\Delta x + \big[f_y(x_0,y_0) + \varepsilon_2\big]\Delta y \\
&= f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y + \varepsilon_1\Delta x + \varepsilon_2\Delta y
\end{aligned}
$$

**第五步：证明剩余项 $\varepsilon_1\Delta x + \varepsilon_2\Delta y = o(\|\Delta P\|)$**。

我们需要证明：

$$\lim_{(\Delta x,\Delta y)\to(0,0)} \frac{\varepsilon_1\Delta x + \varepsilon_2\Delta y}{\sqrt{(\Delta x)^2+(\Delta y)^2}} = 0$$

由三角不等式：

$$\left|\frac{\varepsilon_1\Delta x + \varepsilon_2\Delta y}{\|\Delta P\|}\right| \leq \frac{|\varepsilon_1||\Delta x| + |\varepsilon_2||\Delta y|}{\|\Delta P\|}$$

由于 $|\Delta x| \leq \|\Delta P\|$ 且 $|\Delta y| \leq \|\Delta P\|$，得：

$$\left|\frac{\varepsilon_1\Delta x + \varepsilon_2\Delta y}{\|\Delta P\|}\right| \leq |\varepsilon_1| + |\varepsilon_2|$$

由第三步，$\varepsilon_1\to0$ 且 $\varepsilon_2\to0$ 当 $(\Delta x,\Delta y)\to(0,0)$，因此 $|\varepsilon_1|+|\varepsilon_2|\to0$。由夹逼定理：

$$\lim_{(\Delta x,\Delta y)\to(0,0)} \frac{\varepsilon_1\Delta x + \varepsilon_2\Delta y}{\|\Delta P\|} = 0$$

即 $\varepsilon_1\Delta x + \varepsilon_2\Delta y = o(\|\Delta P\|)$。

**第六步：结论**。

我们证明了：

$$\Delta f = f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y + o(\|\Delta P\|)$$

这正是 $f$ 在 $(x_0,y_0)$ 处可微的定义。因此 $f$ 在 $(x_0,y_0)$ 处可微。证毕。

**定理 12.6 的完整逻辑链图示**：

```
f_x 在邻域内存在且 f_x 在 (x_0,y_0) 连续
    └─→ 对 Δ₁f 用 LMVT：Δ₁f = f_x(ξ, y₀+Δy)·Δx
        └─→ 由连续性：f_x(ξ, y₀+Δy) = f_x(x₀,y₀) + ε₁ (ε₁→0)

f_y 在邻域内存在且 f_y 在 (x₀,y₀) 连续
    └─→ 对 Δ₂f 用 LMVT：Δ₂f = f_y(x₀, η)·Δy
        └─→ 由连续性：f_y(x₀, η) = f_y(x₀,y₀) + ε₂ (ε₂→0)

Δf = f_x(x₀,y₀)Δx + f_y(x₀,y₀)Δy + ε₁Δx + ε₂Δy
                                          └─→ |ε₁Δx+ε₂Δy|/||ΔP|| ≤ |ε₁|+|ε₂| → 0
                                              ∴ 可微 ✓
```

**关于定理条件的讨论**：

1. **条件的充分非必要性**：定理 12.6 的条件是充分的但**不是必要的**。存在函数在 $(x_0,y_0)$ 处可微，但 $f_x$ 或 $f_y$ 在 $(x_0,y_0)$ 处不连续。例如 $f(x,y) = (x^2+y^2)\sin\frac{1}{x^2+y^2}$（补充定义 $f(0,0)=0$）在 $(0,0)$ 处可微，但其偏导在 $(0,0)$ 处不连续。因此定理 12.6 是一个实用的**判别工具**而非可微的定义。

2. **条件的几何含义**：偏导连续意味着当 $(x,y)$ 在 $(x_0,y_0)$ 附近变化时，曲面 $z=f(x,y)$ 沿 $x$ 方向和 $y$ 方向的斜率是连续变化的。这种"局部斜率稳定性"保证了切平面的存在。

3. **与一元微分学的类比**：在一元函数中，"导数存在"本身就等价于可微（定理 4.19），不需要连续性假设。在多元函数中，"偏导存在"远弱于"可微"，因此需要更强的条件（偏导连续）来保证可微。

### 4.4 可微性判断的标准流程

综合以上讨论，我们可以给出判断二元函数在某点是否可微的标准流程：

**步骤 1：检查连续性**。
- 若 $f$ 在 $(x_0,y_0)$ 处不连续，则由定理 12.5 的逆否，$f$ 不可微，直接结束。
- 若 $f$ 连续，进入下一步。

**步骤 2：检查偏导存在性**。
- 计算 $f_x(x_0,y_0)$ 和 $f_y(x_0,y_0)$（用定义或用求导公式）。
- 若任一偏导不存在，则由定理 12.1 的逆否，$f$ 不可微，直接结束。
- 若两者都存在，进入下一步。

**步骤 3：检查偏导连续性（充分条件）**。
- 在 $(x_0,y_0)$ 附近求出 $f_x(x,y)$ 和 $f_y(x,y)$ 的表达式。
- 若 $f_x$ 和 $f_y$ 都在 $(x_0,y_0)$ 处连续，则由定理 12.6，$f$ 可微，直接结束。

**步骤 4：若偏导存在但不连续（或不确定连续）**，回到定义直接验证。
- 构造差商 $\displaystyle\frac{\Delta f - f_x(x_0,y_0)\Delta x - f_y(x_0,y_0)\Delta y}{\sqrt{(\Delta x)^2+(\Delta y)^2}}$
- 检查该表达式是否随 $(\Delta x,\Delta y)\to(0,0)$ 趋于零
- 若极限为零，则可微；否则不可微

**流程图**：

```
判断 f 在 P₀ 是否可微
    │
    ├─ f 在 P₀ 不连续？ → 是 → 不可微（定理12.5逆否）
    │
    └─ f 连续？
        │
        ├─ f_x 或 f_y 在 P₀ 不存在？ → 是 → 不可微（定理12.1逆否）
        │
        └─ f_x, f_y 都存在？
            │
            ├─ f_x, f_y 在 P₀ 连续？ → 是 → 可微（定理12.6）
            │
            └─ 不连续或不确定？
                │
                └─ 用定义验证：极限=0? → 是→可微；否→不可微
```

**实例演示**：

**判断 1**：$f(x,y) = x^2 + 3xy - y^2$ 在 $(1,2)$ 处的可微性。

- 步骤 1：多项式函数处处连续 ✓
- 步骤 2：$f_x = 2x+3y$，$f_y = 3x-2y$，在 $(1,2)$ 处均存在 ✓
- 步骤 3：$f_x$ 和 $f_y$ 均为多项式，在 $\mathbb{R}^2$ 上处处连续，特别在 $(1,2)$ 处连续 ✓
- 结论：由定理 12.6，$f$ 在 $(1,2)$ 处可微。全微分为 $\mathrm{d}f = 8\mathrm{d}x - \mathrm{d}y$。

**判断 2**：$f(x,y) = \begin{cases}\dfrac{x^2y}{x^2+y^2}, & (x,y)\neq(0,0) \\ 0, & (x,y)=(0,0)\end{cases}$ 在 $(0,0)$ 处的可微性。

- 步骤 1：已验证连续（见 4.2 节反例 2）✓
- 步骤 2：$f_x(0,0)=0$，$f_y(0,0)=0$，存在 ✓
- 步骤 3：当 $(x,y)\neq(0,0)$ 时，$f_x(x,y) = \frac{2xy^3}{(x^2+y^2)^2}$。沿 $y=x$ 趋近 $(0,0)$ 时 $f_x(x,x) = \frac{2x^4}{(2x^2)^2} = \frac{2x^4}{4x^4} = \frac12$，而 $f_x(0,0)=0$，因此 $f_x$ 在 $(0,0)$ 处不连续。定理 12.6 不适用。
- 步骤 4：从定义验证：$\lim_{(\Delta x,\Delta y)\to(0,0)}\frac{f(\Delta x,\Delta y)}{\sqrt{\Delta x^2+\Delta y^2}} = \lim\frac{\Delta x^2\Delta y}{(\Delta x^2+\Delta y^2)^{3/2}}$。沿 $\Delta y=\Delta x$ 得 $\frac{1}{2^{3/2}}\neq0$，极限不为零。
- 结论：不可微。

## 5. 例题

### 例题 1：利用定理 12.6 判断可微性

判断下列函数在指定点处的可微性，并写出全微分（若可微）。

(1) $f(x,y) = 3x^2y - 2xy^2 + y^3$ 在 $(1,-1)$ 处
(2) $f(x,y) = \ln(1+x^2+y^2)$ 在 $(0,0)$ 处
(3) $f(x,y) = \sqrt{x^2+y^2}$ 在 $(0,0)$ 处

**解 (1)**：

**步骤 1——连续性**：$f$ 是多项式函数，在 $\mathbb{R}^2$ 上处处连续，特别在 $(1,-1)$ 处连续。

**步骤 2——偏导存在性**：
$$f_x = 6xy - 2y^2,\quad f_y = 3x^2 - 4xy + 3y^2$$
$$f_x(1,-1) = 6\cdot1\cdot(-1) - 2\cdot(-1)^2 = -6 - 2 = -8$$
$$f_y(1,-1) = 3\cdot1^2 - 4\cdot1\cdot(-1) + 3\cdot(-1)^2 = 3 + 4 + 3 = 10$$
偏导存在。

**步骤 3——偏导连续性**：$f_x$ 和 $f_y$ 均为多项式函数，在 $\mathbb{R}^2$ 上处处连续，因此在 $(1,-1)$ 处连续。

**结论**：由定理 12.6，$f$ 在 $(1,-1)$ 处可微。全微分为：
$$\mathrm{d}f = -8\,\mathrm{d}x + 10\,\mathrm{d}y$$

---

**解 (2)**：

$f(x,y) = \ln(1+x^2+y^2)$，考察 $(0,0)$ 处。

**步骤 1——连续性**：$\ln(1+x^2+y^2)$ 是初等函数，$(0,0)$ 在定义域内，因此连续。

**步骤 2——偏导存在性**：
$$f_x = \frac{2x}{1+x^2+y^2},\quad f_y = \frac{2y}{1+x^2+y^2}$$
$$f_x(0,0) = \frac{0}{1} = 0,\quad f_y(0,0) = \frac{0}{1} = 0$$
偏导存在。

**步骤 3——偏导连续性**：$f_x = \frac{2x}{1+x^2+y^2}$ 是有理函数，分母 $1+x^2+y^2 \geq 1 > 0$，因此 $f_x$ 在 $\mathbb{R}^2$ 上连续。同理 $f_y$ 处处连续。特别在 $(0,0)$ 处连续。

**结论**：由定理 12.6，$f$ 在 $(0,0)$ 处可微。全微分为：
$$\mathrm{d}f = 0\cdot\mathrm{d}x + 0\cdot\mathrm{d}y = 0$$

几何直观：$f(x,y)\approx x^2+y^2$ 在 $(0,0)$ 附近，其二阶近似为 $x^2+y^2$，但一阶近似在原点处为零（因为原点处梯度为零）。

---

**解 (3)**：

$f(x,y) = \sqrt{x^2+y^2}$，考察 $(0,0)$ 处。

**步骤 1——连续性**：已知 $\displaystyle\lim_{(x,y)\to(0,0)}\sqrt{x^2+y^2}=0=f(0,0)$，连续。

**步骤 2——偏导存在性**：
$$f_x(0,0) = \lim_{h\to0}\frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0}\frac{|h|}{h}$$
此极限不存在（当 $h\to0^+$ 时 $|h|/h = 1$，$h\to0^-$ 时 $|h|/h = -1$）。因此 $f_x(0,0)$ 不存在。

**结论**：由定理 12.1 的逆否（可微 $\Rightarrow$ 偏导存在），$f$ 在 $(0,0)$ 处不可微。

几何理解：$z=\sqrt{x^2+y^2}$ 是圆锥面，在原点处有一个"尖点"，不存在切平面——因此不可微。

### 例题 2：用定义验证可微性

考虑函数

$$f(x,y) = \begin{cases}
\dfrac{x^3 - y^3}{x^2+y^2}, & (x,y)\neq(0,0) \\[6pt]
0, & (x,y)=(0,0)
\end{cases}$$

(1) 验证 $f$ 在 $(0,0)$ 处连续。
(2) 求 $f_x(0,0)$ 和 $f_y(0,0)$。
(3) 判断 $f$ 在 $(0,0)$ 处是否可微。

**解 (1)**：

当 $(x,y)\neq(0,0)$ 时，用极坐标 $x=r\cos\theta$，$y=r\sin\theta$：

$$|f(x,y)| = \frac{|x^3-y^3|}{x^2+y^2} = \frac{r^3|\cos^3\theta - \sin^3\theta|}{r^2} = r|\cos^3\theta - \sin^3\theta| \leq 2r = 2\sqrt{x^2+y^2}$$

因此当 $(x,y)\to(0,0)$ 时，$|f(x,y)| \leq 2\sqrt{x^2+y^2} \to 0$。故 $\displaystyle\lim_{(x,y)\to(0,0)}f(x,y)=0=f(0,0)$，$f$ 在 $(0,0)$ 处连续。

**解 (2)**：

用定义计算偏导：

$$f_x(0,0) = \lim_{h\to0}\frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0}\frac{\frac{h^3-0}{h^2+0^2} - 0}{h} = \lim_{h\to0}\frac{h}{h} = 1$$

$$f_y(0,0) = \lim_{k\to0}\frac{f(0,k)-f(0,0)}{k} = \lim_{k\to0}\frac{\frac{0-k^3}{0+k^2} - 0}{k} = \lim_{k\to0}\frac{-k}{k} = -1$$

因此 $f_x(0,0)=1$，$f_y(0,0)=-1$。

**解 (3)**：

由于 $f_x(0,0)$ 和 $f_y(0,0)$ 都存在，考虑用定义检验可微性。

构造可微性判别极限（取 $\Delta x = h$，$\Delta y = k$）：

$$L = \lim_{(h,k)\to(0,0)} \frac{f(h,k) - f(0,0) - f_x(0,0)h - f_y(0,0)k}{\sqrt{h^2+k^2}}$$

代入 $f(h,k) = \frac{h^3-k^3}{h^2+k^2}$（当 $(h,k)\neq(0,0)$）和 $f_x(0,0)=1$，$f_y(0,0)=-1$：

$$
\begin{aligned}
L &= \lim_{(h,k)\to(0,0)} \frac{\frac{h^3-k^3}{h^2+k^2} - h + k}{\sqrt{h^2+k^2}} \\
&= \lim_{(h,k)\to(0,0)} \frac{h^3 - k^3 - (h-k)(h^2+k^2)}{(h^2+k^2)^{3/2}}
\end{aligned}
$$

展开分子中的 $(h-k)(h^2+k^2) = h^3 + hk^2 - kh^2 - k^3$，因此：

$$
\begin{aligned}
\text{分子} &= h^3 - k^3 - (h^3 + hk^2 - kh^2 - k^3) \\
&= h^3 - k^3 - h^3 - hk^2 + kh^2 + k^3 \\
&= kh^2 - hk^2 \\
&= hk(h - k)
\end{aligned}
$$

因此：

$$L = \lim_{(h,k)\to(0,0)} \frac{hk(h - k)}{(h^2+k^2)^{3/2}}$$

现在用极坐标 $h = r\cos\theta$，$k = r\sin\theta$：

$$L = \lim_{r\to0} \frac{r^3\cos\theta\sin\theta(\cos\theta - \sin\theta)}{r^3} = \cos\theta\sin\theta(\cos\theta - \sin\theta)$$

这个极限依赖于 $\theta$（方向），不是与路径无关的常数。例如：

- 沿 $\theta=0$（$k=0$）：$L = \cos0\cdot\sin0\cdot(\cos0-\sin0) = 0$
- 沿 $\theta=\frac{\pi}{4}$（$h=k$）：$L = \cos\frac{\pi}{4}\sin\frac{\pi}{4}(\cos\frac{\pi}{4}-\sin\frac{\pi}{4}) = \frac{1}{\sqrt2}\cdot\frac{1}{\sqrt2}\cdot0 = 0$
- 沿 $\theta=\frac{\pi}{8}$：$L = \cos\frac{\pi}{8}\sin\frac{\pi}{8}(\cos\frac{\pi}{8}-\sin\frac{\pi}{8}) \approx 0.8536\times0.1464\times(0.9239-0.3827) \approx 0.0678 \neq 0$

因此极限 $L$ 不存在（不是唯一的确定值），$f$ 在 $(0,0)$ 处**不可微**。

**结论对比**：此函数在 $(0,0)$ 处连续、偏导存在，但不可微——再次印证偏导存在不是可微的充分条件。

### 例题 3：用定义验证可微与不可微的综合练习

设

$$f(x,y) = \begin{cases}
\dfrac{x^2y^2}{x^2+y^2}, & (x,y)\neq(0,0) \\[6pt]
0, & (x,y)=(0,0)
\end{cases}$$

(1) 判断 $f$ 在 $(0,0)$ 处是否连续。
(2) 求 $f_x(0,0)$ 和 $f_y(0,0)$。
(3) 判断 $f$ 在 $(0,0)$ 处是否可微。

**解 (1)**：

当 $(x,y)\neq(0,0)$ 时，$0 \leq \frac{x^2y^2}{x^2+y^2} \leq \frac{(x^2+y^2)^2}{2(x^2+y^2)} = \frac{x^2+y^2}{2}$（最后一步用 AM-GM 不等式 $x^2y^2 \leq \frac{(x^2+y^2)^2}{4}$）。

更精确地，由 $|x^2y^2| \leq \frac{(x^2+y^2)^2}{4}$，得：

$$|f(x,y)| \leq \frac{(x^2+y^2)^2}{4(x^2+y^2)} = \frac{x^2+y^2}{4} \to 0 \quad ((x,y)\to(0,0))$$

因此 $\displaystyle\lim_{(x,y)\to(0,0)}f(x,y)=0=f(0,0)$，$f$ 在 $(0,0)$ 处连续。

**解 (2)**：

$$f_x(0,0) = \lim_{h\to0}\frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0}\frac{0-0}{h} = 0$$

$$f_y(0,0) = \lim_{k\to0}\frac{f(0,k)-f(0,0)}{k} = \lim_{k\to0}\frac{0-0}{k} = 0$$

一阶偏导存在，均为 $0$。

**解 (3)**：

检查 $f_x$ 和 $f_y$ 的连续性。当 $(x,y)\neq(0,0)$ 时：

$$f_x(x,y) = \frac{2xy^2(x^2+y^2) - x^2y^2\cdot2x}{(x^2+y^2)^2} = \frac{2xy^4}{(x^2+y^2)^2}$$

沿 $y=x$ 路径：$f_x(x,x) = \frac{2x\cdot x^4}{(2x^2)^2} = \frac{2x^5}{4x^4} = \frac{x}{2} \to 0 = f_x(0,0)$（当 $x\to0$）。但这还不能证明 $f_x$ 在 $(0,0)$ 连续，需要验证任意路径。

实际上，考察 $|f_x(x,y)|$ 的界：

$$|f_x(x,y)| = \frac{2|x|y^4}{(x^2+y^2)^2} \leq \frac{2|x|(x^2+y^2)^2}{(x^2+y^2)^2} = 2|x| \to 0$$

因此 $\displaystyle\lim_{(x,y)\to(0,0)}f_x(x,y)=0=f_x(0,0)$，$f_x$ 在 $(0,0)$ 连续。

同理 $|f_y(x,y)| \leq 2|y| \to 0$，$f_y$ 在 $(0,0)$ 连续。

**由定理 12.6**，$f_x$ 和 $f_y$ 存在且在 $(0,0)$ 连续，因此 $f$ 在 $(0,0)$ 处可微。

**验证**：全微分为 $\mathrm{d}f = 0\cdot\mathrm{d}x + 0\cdot\mathrm{d}y = 0$。事实上

$$\frac{|f(\Delta x,\Delta y) - 0|}{\sqrt{\Delta x^2 + \Delta y^2}} = \frac{\Delta x^2\Delta y^2}{(\Delta x^2+\Delta y^2)^{3/2}} \leq \frac{(\Delta x^2+\Delta y^2)^2/4}{(\Delta x^2+\Delta y^2)^{3/2}} = \frac{1}{4}\sqrt{\Delta x^2+\Delta y^2} \to 0$$

确实满足可微定义。

**本题的意义**：反例 2（$f=x^2y/(x^2+y^2)$）不可微，而本题 $f=x^2y^2/(x^2+y^2)$ 可微。两者看起来相似，但分子从 $x^2y$ 变为 $x^2y^2$ 后，$f_x$ 和 $f_y$ 变得连续，因此可微。这揭示了"分子中 $y$ 的幂次"对结果的关键影响。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 偏导存在 $\Rightarrow$ 可微（受一元函数"可导 $\iff$ 可微"的影响） | 偏导存在是可微的**必要条件**，不是充分条件。反例：$f(x,y)=\frac{x^2y}{x^2+y^2}$（$f(0,0)=0$）在 $(0,0)$ 处偏导存在、连续但不可微 |
| 函数连续 $\Rightarrow$ 可微 | 连续只是可微的必要条件，远非充分。$f(x,y)=\sqrt{x^2+y^2}$ 在 $(0,0)$ 处连续但不可微（甚至偏导都不存在） |
| 函数连续 + 偏导存在 $\Rightarrow$ 可微 | 反例：$f(x,y)=\frac{x^2y}{x^2+y^2}$（$f(0,0)=0$）同时连续且偏导存在，但不可微。需要更强的条件（偏导连续） |
| 混淆 $o(\|\Delta P\|)$ 与 $o(\Delta x)+o(\Delta y)$ | $o(\|\Delta P\|)$ 要求剩余项与 $\sqrt{\Delta x^2+\Delta y^2}$ 之比趋于零，这是比 $o(\Delta x)+o(\Delta y)$ 更强的条件。$o(\Delta x)+o(\Delta y)$ 只说明剩余项相对于每个单独的增量是高阶小，但不能保证相对于"联合距离"是高阶小 |
| 验证可微时，只需检查沿坐标轴方向的差商 | 可微定义要求对 $(\Delta x,\Delta y)\to(0,0)$ 的**所有可能方式**，$\frac{R}{\|\Delta P\|}\to0$ 都成立。仅检查 $\Delta y=0$ 或 $\Delta x=0$ 等特殊情况是不够的——反例中沿 $\Delta y=\Delta x$ 的路径恰好暴露了不可微 |
| 认为 $\mathrm{d}f = f_x\mathrm{d}x + f_y\mathrm{d}y$ 是自动成立的公式 | 这个公式只有在 $f$ **可微**时才成立。如果 $f$ 不可微，虽然 $f_x,f_y$ 存在，但 $\Delta f$ 不能表示为 $f_x\Delta x+f_y\Delta y+o(\|\Delta P\|)$ 的形式，此时 $\mathrm{d}f$ 无意义 |
| 定理 12.6 的逆命题成立：可微 $\Rightarrow$ 偏导连续 | 定理 12.6 是充分条件，不是必要条件。存在可微函数其偏导不连续，例如 $f(x,y)=(x^2+y^2)\sin\frac{1}{x^2+y^2}$（$f(0,0)=0$）在 $(0,0)$ 处可微，但偏导在 $(0,0)$ 不连续 |

### 检查点

- [ ] 能否写出 $f$ 在 $(x_0,y_0)$ 处可微的精确定义（定义 12.3），并解释 $o(\|\Delta P\|)$ 的含义？
- [ ] 能否从可微定义出发，完整证明定理 12.5（可微 $\Rightarrow$ 连续）？
- [ ] 能否清晰阐述可微与一元函数"可导"的类比与本质区别？
- [ ] 能否举出偏导存在但函数不可微的具体反例，并完整验证其不可微？
- [ ] 能否陈述并证明定理 12.6（可微的充分条件）？能否解释为什么偏导连续性条件能保证可微？
- [ ] 能否解释定理 12.6 的证明中 Lagrange 中值定理和连续性各自起到了什么作用？
- [ ] 能否画出可微性判断的流程图（连续检查 → 偏导检查 → 偏导连续性检查 → 定义验证）？
- [ ] 能否解释为什么 $f(x,y)=\sqrt{x^2+y^2}$ 在 $(0,0)$ 处不可微？
- [ ] 对于一个给定的分段函数，能否系统性地判断其在分段点处的可微性？
- [ ] 能否构造一个在 $(0,0)$ 处可微但偏导不连续的函数的例子？（选做）

## 练习题

### 基础巩固

**1.** 判断下列函数在指定点处是否可微。若可微，写出全微分 $\mathrm{d}f$。

(1) $f(x,y)=x^2+2xy-3y^2$ 在 $(2,1)$ 处
(2) $f(x,y)=\frac{1}{x^2+y^2+1}$ 在 $(1,1)$ 处
(3) $f(x,y)=\sqrt{1+x^2+y^2}$ 在 $(0,0)$ 处

<details><summary>参考答案</summary>

**(1)** $f(x,y)=x^2+2xy-3y^2$。

$f$ 是多项式，在 $\mathbb{R}^2$ 上无穷次可微。$f_x=2x+2y$，$f_y=2x-6y$，均为多项式，处处连续。由定理 12.6，$f$ 在 $(2,1)$ 处可微。

计算偏导数值：
$$f_x(2,1)=2\cdot2+2\cdot1=6,\quad f_y(2,1)=2\cdot2-6\cdot1=-2$$

全微分：$\mathrm{d}f = 6\,\mathrm{d}x - 2\,\mathrm{d}y$。

---

**(2)** $f(x,y)=\frac{1}{x^2+y^2+1}$。

$f$ 是初等函数，分母 $x^2+y^2+1\geq1>0$，因此 $f$ 在 $\mathbb{R}^2$ 上连续且无穷次可微。

$$f_x = -\frac{2x}{(x^2+y^2+1)^2},\quad f_y = -\frac{2y}{(x^2+y^2+1)^2}$$

均为有理函数且分母恒正，在 $\mathbb{R}^2$ 上处处连续。由定理 12.6，$f$ 在 $(1,1)$ 处可微。

$$f_x(1,1) = -\frac{2\cdot1}{(1+1+1)^2} = -\frac{2}{9},\quad f_y(1,1) = -\frac{2\cdot1}{9} = -\frac{2}{9}$$

全微分：$\mathrm{d}f = -\frac{2}{9}\,\mathrm{d}x -\frac{2}{9}\,\mathrm{d}y$。

---

**(3)** $f(x,y)=\sqrt{1+x^2+y^2}$。

$f$ 是初等函数，$1+x^2+y^2>0$，因此在 $\mathbb{R}^2$ 上连续。

$$f_x = \frac{x}{\sqrt{1+x^2+y^2}},\quad f_y = \frac{y}{\sqrt{1+x^2+y^2}}$$

均为初等函数，在 $\mathbb{R}^2$ 上连续。

代入 $(0,0)$：$f_x(0,0)=0$，$f_y(0,0)=0$。偏导在 $(0,0)$ 处存在且连续，由定理 12.6，$f$ 在 $(0,0)$ 处可微。

全微分：$\mathrm{d}f = 0\cdot\mathrm{d}x + 0\cdot\mathrm{d}y = 0$。

**验证**：事实上 $f(x,y)=1+\frac12(x^2+y^2)+o(x^2+y^2)$，在 $(0,0)$ 处一阶近似为零，二阶项才开始出现。

</details>

**2.** 考虑函数 $f(x,y)=\begin{cases}
\dfrac{x^3}{x^2+y^2}, & (x,y)\neq(0,0) \\[6pt]
0, & (x,y)=(0,0)
\end{cases}$。

(1) 证明 $f$ 在 $(0,0)$ 处连续。
(2) 求 $f_x(0,0)$ 和 $f_y(0,0)$。
(3) 判断 $f$ 在 $(0,0)$ 处是否可微。

<details><summary>参考答案</summary>

**(1) 连续性**：当 $(x,y)\neq(0,0)$ 时，
$$|f(x,y)| = \frac{|x|^3}{x^2+y^2} \leq \frac{|x|(x^2+y^2)}{x^2+y^2} = |x| \to 0 \quad ((x,y)\to(0,0))$$

因此 $\displaystyle\lim_{(x,y)\to(0,0)} f(x,y)=0=f(0,0)$，$f$ 在 $(0,0)$ 处连续。

**(2) 偏导**：
$$f_x(0,0) = \lim_{h\to0}\frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0}\frac{\frac{h^3}{h^2}-0}{h} = \lim_{h\to0}\frac{h}{h} = 1$$
$$f_y(0,0) = \lim_{k\to0}\frac{f(0,k)-f(0,0)}{k} = \lim_{k\to0}\frac{0-0}{k} = 0$$

**(3) 可微性检验**：

构造可微性判别极限：
$$L = \lim_{(h,k)\to(0,0)} \frac{f(h,k) - 0 - 1\cdot h - 0\cdot k}{\sqrt{h^2+k^2}} = \lim_{(h,k)\to(0,0)} \frac{\frac{h^3}{h^2+k^2} - h}{\sqrt{h^2+k^2}}$$

化简分子：
$$\frac{h^3}{h^2+k^2} - h = \frac{h^3 - h(h^2+k^2)}{h^2+k^2} = \frac{-hk^2}{h^2+k^2}$$

因此：
$$L = \lim_{(h,k)\to(0,0)} \frac{-hk^2}{(h^2+k^2)^{3/2}}$$

沿 $k=0$：$L=0$。
沿 $h=k$：$L = \lim_{h\to0}\frac{-h\cdot h^2}{(2h^2)^{3/2}} = \lim_{h\to0}\frac{-h^3}{2^{3/2}|h|^3}$。

当 $h>0$ 时 $L = -\frac{1}{2^{3/2}} \neq 0$，因此 $L$ 的极限不存在（不同路径得到不同值），$f$ 在 $(0,0)$ 处**不可微**。

**进一步分析**：检查偏导的连续性。当 $(x,y)\neq(0,0)$ 时：
$$f_x(x,y) = \frac{3x^2(x^2+y^2) - x^3\cdot2x}{(x^2+y^2)^2} = \frac{3x^4+3x^2y^2-2x^4}{(x^2+y^2)^2} = \frac{x^4+3x^2y^2}{(x^2+y^2)^2}$$

沿 $y=x$：$f_x(x,x) = \frac{x^4+3x^4}{(2x^2)^2} = \frac{4x^4}{4x^4}=1$，而 $f_x(0,0)=1$，沿这条路径看是连续的。但沿 $y=0$：$f_x(x,0)=\frac{x^4}{x^4}=1$。实际上，用极坐标：$f_x(r\cos\theta,r\sin\theta)=\frac{r^4\cos^4\theta+3r^4\cos^2\theta\sin^2\theta}{r^4} = \cos^4\theta+3\cos^2\theta\sin^2\theta$，这与 $\theta$ 有关，当 $r\to0$ 时的极限取决于 $\theta$。但 $f_x(0,0)=1$，而沿 $\theta=\frac{\pi}{2}$ 时 $\cos^4\theta+3\cos^2\theta\sin^2\theta=0$，与 $1$ 不同，因此 $f_x$ 在 $(0,0)$ 处**不连续**。这正是定理 12.6 的条件不满足的情况。

</details>

---

**3.** 证明：若 $f$ 在 $(x_0,y_0)$ 处可微，则对任意单位方向 $\mathbf{v}=(\cos\alpha,\sin\alpha)$，方向导数 $\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0)$ 存在且等于 $\nabla f(x_0,y_0)\cdot\mathbf{v}$。

（提示：利用全微分的定义和方向导数的极限定义。）

<details><summary>参考答案</summary>

**证明**：由于 $f$ 在 $(x_0,y_0)$ 处可微，由定义 12.3：
$$\Delta f = f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y + o\!\left(\sqrt{(\Delta x)^2+(\Delta y)^2}\right)$$

对任意单位方向 $\mathbf{v}=(\cos\alpha,\sin\alpha)$，取 $\Delta x = t\cos\alpha$，$\Delta y = t\sin\alpha$（$t>0$），则 $\sqrt{(\Delta x)^2+(\Delta y)^2}=t$。

方向导数的极限定义：
$$\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) = \lim_{t\to0^+} \frac{f(x_0+t\cos\alpha,\ y_0+t\sin\alpha) - f(x_0,y_0)}{t}$$

代入可微表达式：
$$
\begin{aligned}
\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) &= \lim_{t\to0^+} \frac{f_x(x_0,y_0)t\cos\alpha + f_y(x_0,y_0)t\sin\alpha + o(t)}{t} \\
&= \lim_{t\to0^+} \left[f_x(x_0,y_0)\cos\alpha + f_y(x_0,y_0)\sin\alpha + \frac{o(t)}{t}\right] \\
&= f_x(x_0,y_0)\cos\alpha + f_y(x_0,y_0)\sin\alpha
\end{aligned}
$$

最后一个等式用到了 $\frac{o(t)}{t}\to0$（$t\to0$），这正是可微定义中 $o(t)$ 的性质。

因此方向导数存在且 $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$。证毕。

**注**：这个结论在文件 02 中作为定理 12.3 给出，其证明依赖于可微性。此处从可微定义出发重新推导，展示了可微性在方向导数存在性中的关键作用。

</details>

### 迁移应用

**4.** 设 $f(x,y)=\begin{cases}
\dfrac{x^ay}{x^2+y^2}, & (x,y)\neq(0,0) \\[6pt]
0, & (x,y)=(0,0)
\end{cases}$，其中 $a>0$ 是参数。

(1) 求 $a$ 的取值范围使得 $f$ 在 $(0,0)$ 处连续。
(2) 求 $a$ 的取值范围使得 $f$ 在 $(0,0)$ 处偏导存在。
(3) 求 $a$ 的取值范围使得 $f$ 在 $(0,0)$ 处可微。

<details><summary>参考答案</summary>

**(1) 连续性条件**：

当 $(x,y)\neq(0,0)$ 时，$|f(x,y)| = \frac{|x|^a|y|}{x^2+y^2}$。使用极坐标 $x=r\cos\theta$，$y=r\sin\theta$：

$$|f(x,y)| = \frac{r^{a+1}|\cos^a\theta\sin\theta|}{r^2} = r^{a-1}|\cos^a\theta\sin\theta|$$

要使 $\displaystyle\lim_{r\to0^+} |f(x,y)| = 0$ 对所有 $\theta$ 一致成立，需要 $a-1 > 0$，即 $a > 1$。

- 若 $a>1$：$|f(x,y)| \leq r^{a-1} \to 0$，连续。
- 若 $a=1$：$|f(x,y)| = |\cos\theta\sin\theta|$，极限依赖于 $\theta$（沿 $y=x$ 得 $|1/\sqrt2\cdot1/\sqrt2| = 1/2$），不连续。
- 若 $0<a<1$：$r^{a-1} \to \infty$，不连续。

因此 **$a>1$ 时 $f$ 在 $(0,0)$ 处连续**。

---

**(2) 偏导存在条件**：

$$f_x(0,0) = \lim_{h\to0} \frac{f(h,0)-f(0,0)}{h} = \lim_{h\to0} \frac{0-0}{h} = 0$$

$f_x(0,0)$ 对任意 $a>0$ 都存在（为 $0$），因为 $f(h,0)=0$。

$$f_y(0,0) = \lim_{k\to0} \frac{f(0,k)-f(0,0)}{k} = \lim_{k\to0} \frac{0-0}{k} = 0$$

$f_y(0,0)$ 对任意 $a>0$ 也存在（分子恒为零）。

因此**对所有 $a>0$**，$f_x(0,0)$ 和 $f_y(0,0)$ 都存在（均为 $0$）。

---

**(3) 可微性条件**：

由 (1) 知，$a>1$ 时 $f$ 连续。$a\leq1$ 时不连续，由定理 12.5 的逆否，$a\leq1$ 时不可微。

现假设 $a>1$。由 (2) 知 $f_x(0,0)=f_y(0,0)=0$。检验可微性判别极限：

$$L = \lim_{(h,k)\to(0,0)} \frac{f(h,k) - 0 - 0\cdot h - 0\cdot k}{\sqrt{h^2+k^2}} = \lim_{(h,k)\to(0,0)} \frac{|h|^a k}{(h^2+k^2)^{3/2}}$$

取 $h=k=t>0$：
$$L = \lim_{t\to0^+} \frac{t^a\cdot t}{(2t^2)^{3/2}} = \lim_{t\to0^+} \frac{t^{a+1}}{2^{3/2}t^3} = \frac{1}{2^{3/2}} \lim_{t\to0^+} t^{a-2}$$

因此当且仅当 $a-2 > 0$，即 $a>2$ 时，沿此路径的极限为 $0$。

当 $1<a\leq2$ 时，沿 $h=k$ 路径的极限不为零（$a=2$ 时极限为 $1/2^{3/2}$，$1<a<2$ 时极限为 $\infty$），因此 $f$ 不可微。

还需验证当 $a>2$ 时极限在所有路径上都为零。用极坐标：
$$|L| = \lim_{r\to0} \frac{r^{a+1}|\cos^a\theta\sin\theta|}{r^3} = \lim_{r\to0} r^{a-2}|\cos^a\theta\sin\theta| \leq r^{a-2} \to 0$$

因为 $a-2>0$ 且 $|\cos^a\theta\sin\theta|$ 有界。因此 $a>2$ 时 $f$ 可微。

**结论**：
- $a>1$：连续
- $a>0$：偏导存在
- $a>2$：可微

**意义**：参数 $a$ 的取值直观地反映了分子中 $x$ 的幂次对函数光滑性的影响——幂次越高，函数在原点附近越"平坦"，越有可能可微。

</details>

**5.** （综合题）设 $f(x,y)=\begin{cases}
\dfrac{x^2y^2}{x^2+y^2}, & (x,y)\neq(0,0) \\[6pt]
0, & (x,y)=(0,0)
\end{cases}$，如例题 3。

(1) 当 $(x,y)\neq(0,0)$ 时，计算 $f_x(x,y)$ 和 $f_y(x,y)$，并证明它们在 $(0,0)$ 处连续。
(2) 用定理 12.6 重新说明 $f$ 在 $(0,0)$ 处可微。
(3) 写出全微分 $\mathrm{d}f$ 在任意点 $(x,y)$ 处的表达式，并验证 $\mathrm{d}f$ 在 $(0,0)$ 处为零。

<details><summary>参考答案</summary>

**(1)** 当 $(x,y)\neq(0,0)$ 时，用商的求导法则：

$$f_x(x,y) = \frac{2xy^2(x^2+y^2) - x^2y^2\cdot 2x}{(x^2+y^2)^2} = \frac{2x^3y^2 + 2xy^4 - 2x^3y^2}{(x^2+y^2)^2} = \frac{2xy^4}{(x^2+y^2)^2}$$

同理：

$$f_y(x,y) = \frac{2x^2y(x^2+y^2) - x^2y^2\cdot 2y}{(x^2+y^2)^2} = \frac{2x^4y}{(x^2+y^2)^2}$$

证明 $f_x$ 在 $(0,0)$ 连续：
$$|f_x(x,y)-f_x(0,0)| = \frac{2|x|y^4}{(x^2+y^2)^2}$$

由于 $y^4 \leq (x^2+y^2)^2$（因为 $|y|^2 \leq x^2+y^2$ 两边平方），因此：

$$|f_x(x,y)-f_x(0,0)| \leq \frac{2|x|(x^2+y^2)^2}{(x^2+y^2)^2} = 2|x| \to 0$$

故 $\displaystyle\lim_{(x,y)\to(0,0)} f_x(x,y) = 0 = f_x(0,0)$，$f_x$ 在 $(0,0)$ 连续。

同理：$|f_y(x,y)-f_y(0,0)| \leq 2|y| \to 0$，$f_y$ 也在 $(0,0)$ 连续。

---

**(2)** $f$ 在 $(0,0)$ 处：
- 连续（例题 3(1) 已证）
- 一阶偏导 $f_x(0,0)=0$，$f_y(0,0)=0$ 存在（例题 3(2) 已证）
- 由 (1)，$f_x$ 和 $f_y$ 在 $(0,0)$ 处连续

由定理 12.6，$f$ 在 $(0,0)$ 处可微。

---

**(3)** 全微分的表达式：

$$\mathrm{d}f = f_x(x,y)\,\mathrm{d}x + f_y(x,y)\,\mathrm{d}y$$

- 当 $(x,y)\neq(0,0)$：$\mathrm{d}f = \dfrac{2xy^4}{(x^2+y^2)^2}\,\mathrm{d}x + \dfrac{2x^4y}{(x^2+y^2)^2}\,\mathrm{d}y$
- 当 $(x,y)=(0,0)$：$\mathrm{d}f = 0\cdot\mathrm{d}x + 0\cdot\mathrm{d}y = 0$

在 $(0,0)$ 处全微分为零，这反映了 $f$ 在原点处的一阶近似为零——即 $f(x,y) \approx 0$ 在原点附近的一阶精度下成立。实际上 $f$ 在原点处的二阶 Taylor 展开才开始出现非零项。

</details>
