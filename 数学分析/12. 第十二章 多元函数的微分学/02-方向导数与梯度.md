# 02. 方向导数与梯度

> 所属章节：第十二章 多元函数的微分学  |  文件序号：02  |  难度：基础
> 常见混淆点：1) 方向导数定义中方向向量 $\mathbf{v}$ 必须是**单位向量**——若 $\mathbf{v}$ 不是单位向量，则 $\lim_{t\to0^+}\frac{f(\mathbf{x}_0+t\mathbf{v})-f(\mathbf{x}_0)}{t}$ 给出的变化率会受向量模长影响，不再是真正的"方向"变化率；2) 梯度 $\nabla f$ 指向函数值**增长最快**的方向（最速上升），而非指向函数的最大值点——在局部范围内梯度指明了变化率最大的走向，但并不意味着沿梯度走一定能到达全局最大值

## 2. 背景与应用场景

**核心问题**：在偏导数中，我们只能测量函数沿坐标轴方向（$x$ 方向或 $y$ 方向）的变化率。但现实中的问题往往要求我们考察沿**任意指定方向**的变化率。

**举例说明**：
- **登山问题**：假设 $z = f(x,y)$ 表示一座山在水平位置 $(x,y)$ 处的高度。如果你站在 $(x_0,y_0)$ 处，朝北偏东 $30^\circ$ 方向行走，你的高度变化率是多少？偏导数 $\frac{\partial f}{\partial x}$ 只告诉你面朝正东方向行走的坡度，$\frac{\partial f}{\partial y}$ 只告诉你面朝正北方向行走的坡度→要得到任意方向上的坡度，需要**方向导数**（directional derivative）。
- **热力学**：温度场 $T(x,y,z)$ 中，某个方向的温度变化率（如沿气流方向）是工程关心的量。
- **气象学**：气压场 $P(x,y)$ 中，沿某个方向的气压梯度决定了风的方向和强度。

**进一步追问**：既然方向导数告诉我们任意方向的变化率，那么是否存在一个方向使得变化率最大？答案是肯定的——这个方向就是**梯度**（gradient）方向。梯度不仅是将偏导数整理成一个向量那么简单，它还编码了函数在一点处最陡峭的上升方向和上升速率。

**从偏导到梯度**：偏导数是沿坐标轴方向的方向导数 $\rightarrow$ 方向导数是沿任意方向的变化率 $\rightarrow$ 梯度是"最佳"方向的方向导数的整合。

## 3. 核心概念与符号约定

### 符号表

| 符号 | 含义 | 说明 |
|------|------|------|
| $\mathbf{v} = (\cos\alpha,\sin\alpha)$ | 单位方向向量 | $\alpha$ 是方向与 $x$ 轴正方向的夹角，$\|\mathbf{v}\|=1$ |
| $\frac{\partial f}{\partial\mathbf{v}}$ 或 $D_{\mathbf{v}}f$ | $f$ 沿方向 $\mathbf{v}$ 的方向导数 | 两种常见记号 |
| $\nabla f$ | $f$ 的梯度 | 读作 "gradient of f" 或 "del f" |
| $\nabla f \cdot \mathbf{v}$ | 梯度与方向向量的点积 | $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$ |
| $\|\nabla f\|$ | 梯度的模长 | 等于最速上升方向上的方向导数值 |
| $\alpha$ | 方向角 | $\mathbf{v} = (\cos\alpha,\sin\alpha)$ |

### 3.1 方向导数的极限定义（A1）

在第十二章文件 01 中，我们定义了偏导数来衡量函数沿坐标轴方向的变化率。现在将其推广到任意方向。

**定义 12.4（方向导数）**：设二元函数 $z=f(x,y)$ 在点 $(x_0,y_0)$ 的某个邻域内有定义。$\mathbf{v} = (\cos\alpha,\sin\alpha)$ 是一个单位向量（其中 $\alpha$ 是 $\mathbf{v}$ 与 $x$ 轴正方向的夹角）。若极限
$$\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) = \lim_{t\to 0^+} \frac{f(x_0+t\cos\alpha,\ y_0+t\sin\alpha) - f(x_0,y_0)}{t}$$
存在（且为有限值），则称此极限值为 $f$ 在 $(x_0,y_0)$ 处沿方向 $\mathbf{v}$ 的**方向导数**（directional derivative）。

**几何解释**：方向导数 $\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0)$ 刻画了从点 $(x_0,y_0)$ 出发、沿方向 $\mathbf{v}$ 移动时，函数值 $f$ 的瞬时变化率。它等于曲面 $z = f(x,y)$ 与过 $(x_0,y_0)$、方向为 $\mathbf{v}$ 的竖直平面相交所得曲线在该点的切线斜率。

**方向导数的其他记号**：
| 记号 | 说明 |
|------|------|
| $\displaystyle\frac{\partial f}{\partial\mathbf{v}}$ | 偏导风格的记号，强调沿方向 $\mathbf{v}$ 求导 |
| $D_{\mathbf{v}}f(x_0,y_0)$ | 算子风格 |
| $\nabla_{\mathbf{v}}f(x_0,y_0)$ | 梯度风格 |
| $f_{\mathbf{v}}(x_0,y_0)$ | 下标风格 |

**方向导数与偏导数的关系**（核心类比）：

偏导数是方向导数的特例——沿坐标轴正方向的方向导数：

- 取 $\alpha = 0$，则 $\mathbf{v} = (1,0)$（沿 $x$ 轴正方向）：
  $$\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) = \lim_{t\to 0^+} \frac{f(x_0+t,\ y_0) - f(x_0,y_0)}{t} = \frac{\partial f}{\partial x}(x_0,y_0)$$
  
- 取 $\alpha = \frac{\pi}{2}$，则 $\mathbf{v} = (0,1)$（沿 $y$ 轴正方向）：
  $$\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) = \lim_{t\to 0^+} \frac{f(x_0,\ y_0+t) - f(x_0,y_0)}{t} = \frac{\partial f}{\partial y}(x_0,y_0)$$

**重要说明**：
- 方向导数的定义要求 $\mathbf{v}$ 是**单位向量**（$\|\mathbf{v}\| = 1$）。若 $\mathbf{v}$ 不是单位向量，则移动 $t$ 距离后实际走过的路程不再是 $t$，导致极限值依赖于 $\mathbf{v}$ 的模长。在使用方向导数公式时务必确认 $\mathbf{v}$ 已单位化。
- 定义中取 $t \to 0^+$（单侧极限），因为方向 $\mathbf{v}$ 本身已指明了朝向。沿相反方向 $-\mathbf{v}$ 的方向导数通常记为 $\frac{\partial f}{\partial(-\mathbf{v})}$。

**例 1（用极限定义计算方向导数）**：设 $f(x,y) = x^2 + 3xy$，求 $f$ 在点 $(1,2)$ 处沿方向 $\mathbf{v} = (\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}})$ 的方向导数。

**解**：$\mathbf{v}$ 已为单位向量（$(\frac{1}{\sqrt{2}})^2 + (\frac{1}{\sqrt{2}})^2 = 1$）。代入定义：
$$
\begin{aligned}
\frac{\partial f}{\partial\mathbf{v}}(1,2) &= \lim_{t\to 0^+} \frac{f(1+\frac{t}{\sqrt{2}},\ 2+\frac{t}{\sqrt{2}}) - f(1,2)}{t} \\
&= \lim_{t\to 0^+} \frac{\left[(1+\frac{t}{\sqrt{2}})^2 + 3(1+\frac{t}{\sqrt{2}})(2+\frac{t}{\sqrt{2}})\right] - [1^2 + 3\cdot1\cdot2]}{t}
\end{aligned}
$$

展开分子：
$$
\begin{aligned}
&\left(1 + \frac{2t}{\sqrt{2}} + \frac{t^2}{2}\right) + 3\left(2 + \frac{3t}{\sqrt{2}} + \frac{t^2}{2}\right) - (1 + 6) \\
&= \left(1 + \sqrt{2}t + \frac{t^2}{2}\right) + \left(6 + \frac{9t}{\sqrt{2}} + \frac{3t^2}{2}\right) - 7 \\
&= \sqrt{2}t + \frac{9t}{\sqrt{2}} + \frac{t^2}{2} + \frac{3t^2}{2} \\
&= \left(\sqrt{2} + \frac{9}{\sqrt{2}}\right)t + 2t^2
\end{aligned}
$$

因此：
$$\frac{\partial f}{\partial\mathbf{v}}(1,2) = \lim_{t\to 0^+} \frac{(\sqrt{2} + 9/\sqrt{2})t + 2t^2}{t} = \sqrt{2} + \frac{9}{\sqrt{2}} = \frac{2+9}{\sqrt{2}} = \frac{11}{\sqrt{2}}$$

### 3.2 可微函数的方向导数公式（A2）

直接用极限定义计算方向导数需要复杂的代数运算。当函数可微时，我们可以利用全微分得到简洁的计算公式。

**定理 12.3（可微函数的方向导数公式）**：若 $f(x,y)$ 在点 $(x_0,y_0)$ 处可微，则 $f$ 在 $(x_0,y_0)$ 处沿任意单位方向 $\mathbf{v}=(\cos\alpha,\sin\alpha)$ 的方向导数存在，且
$$\boxed{\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) = f_x(x_0,y_0)\cos\alpha + f_y(x_0,y_0)\sin\alpha}$$

**证明**：由于 $f$ 在 $(x_0,y_0)$ 处可微，由全微分的定义（定义 12.3），存在常数 $A = f_x(x_0,y_0)$、$B = f_y(x_0,y_0)$，使得当 $\Delta x,\Delta y$ 充分小时有：
$$f(x_0+\Delta x,\ y_0+\Delta y) - f(x_0,y_0) = f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y + o\!\left(\sqrt{(\Delta x)^2 + (\Delta y)^2}\right)$$

**第一步**：令 $\Delta x = t\cos\alpha$，$\Delta y = t\sin\alpha$，其中 $t > 0$ 充分小。则函数增量为：
$$\Delta f = f(x_0+t\cos\alpha,\ y_0+t\sin\alpha) - f(x_0,y_0)$$

**第二步**：代入全微分表达式：
$$\Delta f = f_x(x_0,y_0) \cdot t\cos\alpha + f_y(x_0,y_0) \cdot t\sin\alpha + o\!\left(\sqrt{t^2\cos^2\alpha + t^2\sin^2\alpha}\right)$$

注意 $\sqrt{t^2\cos^2\alpha + t^2\sin^2\alpha} = t\sqrt{\cos^2\alpha + \sin^2\alpha} = t$（因为 $\mathbf{v}$ 是单位向量）。因此：
$$\Delta f = f_x(x_0,y_0) t\cos\alpha + f_y(x_0,y_0) t\sin\alpha + o(t)$$

**第三步**：两边除以 $t$：
$$\frac{\Delta f}{t} = f_x(x_0,y_0)\cos\alpha + f_y(x_0,y_0)\sin\alpha + \frac{o(t)}{t}$$

**第四步**：取极限 $t \to 0^+$。由 $o(t)$ 的定义，$\lim_{t\to 0^+} \frac{o(t)}{t} = 0$。因此：
$$\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0) = \lim_{t\to 0^+} \frac{\Delta f}{t} = f_x(x_0,y_0)\cos\alpha + f_y(x_0,y_0)\sin\alpha$$

证毕。

**用向量点积表示**：若记梯度向量 $\nabla f(x_0,y_0) = (f_x(x_0,y_0), f_y(x_0,y_0))$，则方向导数公式可以简洁地写作点积形式：
$$\boxed{\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}}$$

**验证**：$\nabla f \cdot \mathbf{v} = (f_x, f_y) \cdot (\cos\alpha,\sin\alpha) = f_x\cos\alpha + f_y\sin\alpha$，与定理 12.3 的表达式完全一致。

**定理 12.3 的两个重要推论**：

1. **偏导数是方向导数的特例**：
   - $\mathbf{v} = (1,0)$ 时：$\frac{\partial f}{\partial\mathbf{v}} = f_x\cdot 1 + f_y\cdot 0 = f_x$
   - $\mathbf{v} = (0,1)$ 时：$\frac{\partial f}{\partial\mathbf{v}} = f_x\cdot 0 + f_y\cdot 1 = f_y$

2. **沿相反方向的方向导数互为相反数**：
   $$\frac{\partial f}{\partial(-\mathbf{v})} = \nabla f \cdot (-\mathbf{v}) = -\nabla f \cdot \mathbf{v} = -\frac{\partial f}{\partial\mathbf{v}}$$

**例 2（用公式计算方向导数）**：用定理 12.3 重算例 1。

**解**：$f(x,y) = x^2 + 3xy$，$f_x = 2x + 3y$，$f_y = 3x$。
在 $(1,2)$ 处：$f_x(1,2) = 2\cdot1 + 3\cdot2 = 8$，$f_y(1,2) = 3\cdot1 = 3$。
方向 $\mathbf{v} = (\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}})$，其中 $\cos\alpha = \frac{1}{\sqrt{2}}$，$\sin\alpha = \frac{1}{\sqrt{2}}$。
$$\frac{\partial f}{\partial\mathbf{v}}(1,2) = 8 \cdot \frac{1}{\sqrt{2}} + 3 \cdot \frac{1}{\sqrt{2}} = \frac{11}{\sqrt{2}}$$
与例 1 结果一致，但计算量大幅减少。

### 3.3 梯度（Gradient）的定义（A3）

方向导数公式 $\frac{\partial f}{\partial\mathbf{v}} = f_x\cos\alpha + f_y\sin\alpha$ 提示我们：偏导数 $f_x$ 和 $f_y$ 以特定方式组合起来决定函数在任意方向上的变化率。将它们组织成一个向量，就得到了**梯度**。

**定义 12.5（梯度）**：设二元函数 $z = f(x,y)$ 在点 $(x,y)$ 处的一阶偏导数 $f_x$ 和 $f_y$ 均存在。则称向量
$$\boxed{\nabla f(x,y) = \left(\frac{\partial f}{\partial x}(x,y),\ \frac{\partial f}{\partial y}(x,y)\right) = (f_x(x,y),\ f_y(x,y))}$$
为 $f$ 在 $(x,y)$ 处的**梯度**（gradient），读作 "gradient of f" 或 "del f"。

**梯度与方向导数的关系**：由定理 12.3 可知，当 $f$ 可微时，
$$\boxed{\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}}$$
即：沿单位方向 $\mathbf{v}$ 的方向导数等于梯度与 $\mathbf{v}$ 的点积。

**梯度的记号**：
| 记号 | 说明 |
|------|------|
| $\nabla f$ | Nabla 算子，读作 "del f" |
| $\text{grad}\, f$ | 另一种常见记号 |
| $\frac{\partial f}{\partial \mathbf{x}}$ | 向量形式的偏导记号 |

**推广到 $n$ 维**：对于 $n$ 元函数 $f(x_1,x_2,\ldots,x_n)$，其梯度定义为
$$\nabla f = \left(\frac{\partial f}{\partial x_1},\ \frac{\partial f}{\partial x_2},\ \ldots,\ \frac{\partial f}{\partial x_n}\right)$$
方向导数公式 $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$ 对任意维数均成立。

**例 3（计算梯度）**：求 $f(x,y) = x^2y + 3xy^2$ 在点 $(2,1)$ 处的梯度。

**解**：
$$f_x = 2xy + 3y^2,\quad f_y = x^2 + 6xy$$
在 $(2,1)$ 处：
$$f_x(2,1) = 2\cdot2\cdot1 + 3\cdot1^2 = 4 + 3 = 7$$
$$f_y(2,1) = 2^2 + 6\cdot2\cdot1 = 4 + 12 = 16$$
因此：
$$\nabla f(2,1) = (7,\ 16)$$

### 3.4 梯度的几何性质（A4）

梯度不仅仅是一个方便记法——它有深刻的几何意义。

**定理 12.4（梯度的几何意义）**：设 $f(x,y)$ 在 $(x_0,y_0)$ 处可微且 $\nabla f(x_0,y_0) \neq \mathbf{0}$。则：

(1) **最速上升方向**：在所有单位方向中，$\frac{\partial f}{\partial\mathbf{v}}$ 在 $\mathbf{v}$ 与 $\nabla f$ 同向时取得最大值。即梯度方向是函数值**增长最快**的方向。

(2) **最大变化率**：该最大变化率等于梯度的模长：
$$\max_{\|\mathbf{v}\|=1} \frac{\partial f}{\partial\mathbf{v}} = \|\nabla f\|$$

(3) **最速下降方向**：方向导数在 $\mathbf{v} = -\frac{\nabla f}{\|\nabla f\|}$ 时取得最小值 $-\|\nabla f\|$。

(4) **等值线正交性**：梯度 $\nabla f(x_0,y_0)$ 与过 $(x_0,y_0)$ 的等值线（等高线）$f(x,y) = f(x_0,y_0)$ 在该点的切线方向**正交**（垂直）。

**证明 (1) 和 (2)**：利用方向导数公式和向量点积的几何意义。
由定理 12.3 和定义 12.5：
$$\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$$

对任意单位向量 $\mathbf{v}$（$\|\mathbf{v}\| = 1$），应用 Cauchy-Schwarz 不等式：
$$|\nabla f \cdot \mathbf{v}| \leq \|\nabla f\| \cdot \|\mathbf{v}\| = \|\nabla f\|$$

由于点积 $\nabla f \cdot \mathbf{v} = \|\nabla f\|\|\mathbf{v}\|\cos\theta = \|\nabla f\|\cos\theta$（其中 $\theta$ 是 $\nabla f$ 与 $\mathbf{v}$ 的夹角），当且仅当 $\mathbf{v}$ 与 $\nabla f$ 同向时（$\theta = 0$，$\cos\theta = 1$），方向导数取得最大值：
$$\max_{\|\mathbf{v}\|=1} \frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \frac{\nabla f}{\|\nabla f\|} = \|\nabla f\|$$

**证明 (3)**：与 (1) 类似，当 $\mathbf{v}$ 与 $\nabla f$ 反向时（$\theta = \pi$，$\cos\theta = -1$）：
$$\frac{\partial f}{\partial\mathbf{v}} = \|\nabla f\|\cos\pi = -\|\nabla f\|$$
这是方向导数的最小值，即沿 $-\nabla f$ 方向函数值下降最快。

**证明 (4)**：设曲线 $C$ 是 $f(x,y) = c$ 的等值线（其中 $c = f(x_0,y_0)$），且 $\gamma(t) = (x(t), y(t))$ 是 $C$ 的一条光滑参数化曲线，满足 $\gamma(0) = (x_0,y_0)$。由于 $f$ 在 $C$ 上恒为常数 $c$，有：
$$f(x(t), y(t)) = c,\quad \forall t$$

两边对 $t$ 求导（应用多元链式法则，该法将在后续文件中系统讨论，此处直接使用其结论）：
$$\frac{d}{dt}f(x(t),y(t)) = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt} = \nabla f \cdot \gamma'(t) = 0$$

在 $t=0$ 处，$\nabla f(x_0,y_0) \cdot \gamma'(0) = 0$。由于 $\gamma'(0)$ 正是等值线 $C$ 在 $(x_0,y_0)$ 处的切向量，因此 $\nabla f(x_0,y_0)$ 与等值线的切线方向垂直，即梯度与等值线正交。

**几何直觉**：
- 站在山上某点，梯度方向指向最陡的上坡方向
- 沿等高线行走，海拔高度不变，因此梯度与等高线垂直
- $\|\nabla f\|$ 就是最陡方向的坡度

**图示说明**：在二维平面上，等值线 $f(x,y) = c$ 是一些曲线。梯度 $\nabla f$ 垂直于这些曲线，且指向 $f$ 增大的方向（即朝 $f$ 值更大的等值线方向）。

## 4. 原理与方法

### 4.1 方向导数的完整计算流程

计算方向导数 $\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0)$ 的标准三步法：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 第 1 步：单位化方向向量 | 计算 $\mathbf{v} = \frac{\mathbf{w}}{\|\mathbf{w}\|}$，其中 $\mathbf{w}$ 是指定的方向向量 | 若 $\mathbf{w}$ 本身已是单位向量则跳过 |
| 第 2 步：求梯度 | 计算 $\nabla f(x_0,y_0) = (f_x(x_0,y_0), f_y(x_0,y_0))$ | 使用标准偏导求法 |
| 第 3 步：点积 | $\frac{\partial f}{\partial\mathbf{v}} = \nabla f(x_0,y_0) \cdot \mathbf{v}$ | 直接代入 |

**特别注意**：如果给定的方向向量 $\mathbf{w}$ 不是单位向量，必须在代入公式前**先单位化**。否则直接计算 $\nabla f \cdot \mathbf{w}$ 会得到一个放缩了 $\|\mathbf{w}\|$ 倍的值，而非真正的方向导数。

**例 4（非单位向量的处理）**：设 $f(x,y) = e^{xy}$，求 $f$ 在 $(1,0)$ 处沿方向 $\mathbf{w} = (3,4)$ 的方向导数。

**解**：

**第 1 步：单位化**。
$$\|\mathbf{w}\| = \sqrt{3^2 + 4^2} = 5,\quad \mathbf{v} = \frac{\mathbf{w}}{\|\mathbf{w}\|} = \left(\frac{3}{5},\ \frac{4}{5}\right)$$

**第 2 步：求梯度**。
$$f_x = ye^{xy},\quad f_y = xe^{xy}$$
$$\nabla f(1,0) = (0\cdot e^{0},\ 1\cdot e^{0}) = (0,\ 1)$$

**第 3 步：点积**。
$$\frac{\partial f}{\partial\mathbf{v}}(1,0) = (0,\ 1) \cdot \left(\frac{3}{5},\ \frac{4}{5}\right) = 0\cdot\frac{3}{5} + 1\cdot\frac{4}{5} = \frac{4}{5}$$

**验证**：若错误地使用未单位化的 $\mathbf{w} = (3,4)$，会得到 $(0,1)\cdot(3,4) = 4$，是正确值的 5 倍。这直观地说明了单位化的必要性——$\mathbf{w}$ 的长度是 5，意味着沿 $\mathbf{w}$ 方向走 1 个"单位"实际上走了 5 倍的距离。

### 4.2 梯度与等值线的正交互补

梯度与等值线正交的性质可以反过来帮助我们理解梯度的几何图像：

- **梯度的方向**：垂直于等值线，沿 $f$ 增加的方向
- **梯度的模长**：等值线密集程度的度量——等值线越密集，$\|\nabla f\|$ 越大（代表变化越剧烈）
- **反梯度方向**：垂直于等值线，沿 $f$ 减少的方向

**应用价值**：
- **梯度上升法**：迭代地从当前点沿梯度方向移动一小步，可逼近函数的局部极大值
- **梯度下降法**：沿负梯度方向移动，可逼近函数的局部极小值（机器学习中最常用的优化算法）
- **等值线正交性**：在物理中，电势的梯度（电场强度）与等势面垂直；温度梯度的方向垂直于等温面

## 5. 例题

### 例题 1：方向导数计算的综合练习

设 $f(x,y) = x^3 - 3xy + y^2$。

(1) 求 $\nabla f(x,y)$ 和 $\nabla f(1,2)$。
(2) 求 $f$ 在 $(1,2)$ 处沿方向 $\mathbf{v} = \left(\frac{3}{5},\ \frac{4}{5}\right)$ 的方向导数。
(3) 求 $f$ 在 $(1,2)$ 处方向导数的最大值和最小值，以及取得最值的方向。
(4) 判断等值线 $f(x,y) = f(1,2)$ 在 $(1,2)$ 处的切线方向是否与 $\nabla f(1,2)$ 正交，并验证。

**解**：

**(1)** 计算偏导数和梯度。
$$f_x = 3x^2 - 3y,\quad f_y = -3x + 2y$$
$$\nabla f(x,y) = (3x^2 - 3y,\ -3x + 2y)$$
代入 $(1,2)$：
$$\nabla f(1,2) = (3\cdot1^2 - 3\cdot2,\ -3\cdot1 + 2\cdot2) = (3-6,\ -3+4) = (-3,\ 1)$$

**(2)** $\mathbf{v} = \left(\frac{3}{5},\frac{4}{5}\right)$ 已是单位向量。由定理 12.3：
$$\frac{\partial f}{\partial\mathbf{v}}(1,2) = \nabla f(1,2) \cdot \mathbf{v} = (-3,\ 1) \cdot \left(\frac{3}{5},\ \frac{4}{5}\right) = -3\cdot\frac{3}{5} + 1\cdot\frac{4}{5} = -\frac{9}{5} + \frac{4}{5} = -1$$

方向导数为 $-1$，意味着沿此方向函数值下降，变化率为 $1$。

**(3)** 方向导数的最大值和最小值。
$$\|\nabla f(1,2)\| = \sqrt{(-3)^2 + 1^2} = \sqrt{10}$$
由定理 12.4：
- **最大值**：$\|\nabla f(1,2)\| = \sqrt{10}$，方向为 $\frac{\nabla f}{\|\nabla f\|} = \left(-\frac{3}{\sqrt{10}},\ \frac{1}{\sqrt{10}}\right)$（最速上升）
- **最小值**：$-\|\nabla f(1,2)\| = -\sqrt{10}$，方向为 $-\frac{\nabla f}{\|\nabla f\|} = \left(\frac{3}{\sqrt{10}},\ -\frac{1}{\sqrt{10}}\right)$（最速下降）

**(4)** 等值线 $f(x,y) = f(1,2)$。先计算 $f(1,2) = 1^3 - 3\cdot1\cdot2 + 2^2 = 1 - 6 + 4 = -1$。等值线方程为 $x^3 - 3xy + y^2 = -1$。

由定理 12.4(4)，$\nabla f(1,2) = (-3,1)$ 与该等值线在 $(1,2)$ 处的切线方向正交。

为验证，求切向量。在 $(1,2)$ 附近将 $y$ 视作 $x$ 的函数，对等值线方程两边对 $x$ 求隐函数导数：
$$\frac{d}{dx}(x^3 - 3xy + y^2) = \frac{d}{dx}(-1)$$
$$3x^2 - 3y - 3x\frac{dy}{dx} + 2y\frac{dy}{dx} = 0$$
代入 $(x,y) = (1,2)$：
$$3\cdot1^2 - 3\cdot2 - 3\cdot1\cdot\frac{dy}{dx} + 2\cdot2\cdot\frac{dy}{dx} = 0$$
$$3 - 6 - 3y' + 4y' = 0 \quad\Rightarrow\quad -3 + y' = 0$$
解得 $y' = 3$，即切线方向向量可取为 $\mathbf{T} = (1,3)$。

验证正交性：
$$\nabla f(1,2) \cdot \mathbf{T} = (-3,\ 1) \cdot (1,\ 3) = -3\cdot1 + 1\cdot3 = 0$$
点积为零，确实正交。✓

### 例题 2：物理场景中的方向导数

设二维温度场为 $T(x,y) = 100 - x^2 - 2y^2$（单位：$^\circ$C），一个粒子位于 $(2,1)$ 处。

(1) 粒子沿方向 $\mathbf{v} = (1,1)$ 移动时，温度的变化率是多少？
(2) 粒子应朝哪个方向移动才能使温度下降最快？该方向上的温度变化率是多少？

**解**：

**(1)** 首先将 $\mathbf{v} = (1,1)$ 单位化：$\|\mathbf{v}\| = \sqrt{2}$，$\mathbf{v}_0 = \left(\frac{1}{\sqrt{2}},\ \frac{1}{\sqrt{2}}\right)$。

计算梯度：$T_x = -2x$，$T_y = -4y$。
$$\nabla T(2,1) = (-4,\ -4)$$

方向导数：
$$\frac{\partial T}{\partial\mathbf{v}_0}(2,1) = (-4,\ -4) \cdot \left(\frac{1}{\sqrt{2}},\ \frac{1}{\sqrt{2}}\right) = -\frac{4}{\sqrt{2}} - \frac{4}{\sqrt{2}} = -\frac{8}{\sqrt{2}} = -4\sqrt{2} \approx -5.66\ ^\circ\text{C/单位长度}$$

粒子沿 $(1,1)$ 方向移动时，温度以约 $5.66^\circ$C 每单位长度的速率下降。

**(2)** 温度下降最快的方向是 $-\nabla T(2,1)$（由定理 12.4(3)）：
$$-\nabla T(2,1) = -(-4,\ -4) = (4,\ 4) \quad\text{或单位化为}\quad \left(\frac{1}{\sqrt{2}},\ \frac{1}{\sqrt{2}}\right)$$

注意：这恰好与 (1) 中的方向 $\mathbf{v}$ 相同——在 $(2,1)$ 处，温度沿 $(1,1)$ 方向下降最快，变化率为：
$$-\|\nabla T(2,1)\| = -\sqrt{(-4)^2 + (-4)^2} = -\sqrt{32} = -4\sqrt{2}$$

温度沿最速下降方向的变化率与 (1) 计算所得一致（因为 (1) 中的方向正好是最速下降方向）。

### 例题 3：梯度方向与等值线

设 $f(x,y) = x^2 + y^2$。

(1) 求 $\nabla f(1,0)$ 和 $\nabla f(0,1)$。
(2) 画出等值线 $f(x,y) = 1$，并在 $(1,0)$ 和 $(0,1)$ 处标出梯度方向。验证梯度与等值线正交。
(3) 在 $(1,0)$ 处，沿哪个方向移动 $f$ 保持不变？

**解**：

**(1)** $f_x = 2x$，$f_y = 2y$。
$$\nabla f(1,0) = (2,\ 0),\quad \nabla f(0,1) = (0,\ 2)$$

**(2)** 等值线 $f(x,y) = 1$ 即 $x^2 + y^2 = 1$，是单位圆。

在 $(1,0)$ 处，单位圆的切线方向为 $(0,1)$（圆在 $(1,0)$ 处的竖直切线）。验证正交：
$$\nabla f(1,0) \cdot (0,1) = (2,0)\cdot(0,1) = 0 \quad\checkmark$$

在 $(0,1)$ 处，单位圆的切线方向为 $(1,0)$（圆在 $(0,1)$ 处的水平切线）。验证正交：
$$\nabla f(0,1) \cdot (1,0) = (0,2)\cdot(1,0) = 0 \quad\checkmark$$

梯度方向从圆心指向外——这正是 $x^2 + y^2$ 增加最快的方向。

**(3)** 沿 $f$ 保持不变的方向，方向导数为零。设 $\mathbf{v} = (\cos\theta,\sin\theta)$，则：
$$\frac{\partial f}{\partial\mathbf{v}}(1,0) = \nabla f(1,0) \cdot \mathbf{v} = (2,0)\cdot(\cos\theta,\sin\theta) = 2\cos\theta = 0$$
解得 $\cos\theta = 0$，即 $\theta = \frac{\pi}{2}$ 或 $\theta = \frac{3\pi}{2}$。对应的方向为 $\mathbf{v} = (0,1)$ 或 $(0,-1)$——这正是圆的切线方向。这验证了梯度方向与等值线切线方向正交，意味着沿切线方向移动时 $f$ 的变化率为零。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 方向导数 $\frac{\partial f}{\partial\mathbf{v}}$ 中的 $\mathbf{v}$ 可以是任意向量 | $\mathbf{v}$ 必须是**单位向量**。若给定的方向向量不是单位向量，必须单位化后再代入公式 $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$ |
| 方向导数 $\frac{\partial f}{\partial\mathbf{v}}$ 就是 $\nabla f \cdot \mathbf{v}$，其中 $\mathbf{v}$ 是给定的方向向量 | 公式 $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$ 仅在 $\mathbf{v}$ 为单位向量时成立。若 $\mathbf{v}$ 不是单位向量，该会给出一个放缩了 $\|\mathbf{v}\|$ 倍的值 |
| $\nabla f$ 指向 $f$ 的最大值点 | $\nabla f$ 只给出了在当前点处**瞬时**的最速上升方向，它指向的是"在当前点迈出一步能最快提升函数值的方向"。沿梯度方向迭代移动可以逼近局部极大值（梯度上升法），但 $\nabla f$ 本身并不直接指向全局最大值 |
| 梯度 $\nabla f$ 在任何点都存在 | 梯度存在的前提是 $f$ 在该点的**一阶偏导存在**。即使一阶偏导存在，还要函数可微才能保证方向导数公式 $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$ 对**所有**方向成立 |
| 沿等值线方向的方向导数为零 $\Longleftrightarrow$ 梯度为零 | 沿等值线方向的方向导数为零是因为 $\nabla f \cdot \mathbf{v} = 0$（$\mathbf{v}$ 是切向），这并不要求 $\nabla f = \mathbf{0}$，只说明 $\nabla f$ 与 $\mathbf{v}$ 正交 |
| $\nabla f$ 的各个分量是彼此独立的 | $\nabla f$ 的分量 $f_x$ 和 $f_y$ 通过函数 $f$ 耦合——它们满足可积条件 $f_{xy} = f_{yx}$（Clairaut 定理），并非任意两个函数都能成为某个函数的梯度 |

### 检查点

- [ ] 能否写出方向导数 $\frac{\partial f}{\partial\mathbf{v}}(x_0,y_0)$ 的极限定义，并解释为什么 $\mathbf{v}$ 必须是单位向量？
- [ ] 能否说明偏导数是方向导数的特例（$\mathbf{v}=(1,0)$ 和 $\mathbf{v}=(0,1)$ 时的退化情形）？
- [ ] 能否从全微分的定义出发，完整证明定理 12.3（可微函数的方向导数公式）？
- [ ] 能否写出梯度 $\nabla f$ 的定义，以及它与方向导数的关系 $\frac{\partial f}{\partial\mathbf{v}} = \nabla f \cdot \mathbf{v}$？
- [ ] 能否证明梯度方向是最速上升方向，并计算该方向上的变化率 $\|\nabla f\|$？
- [ ] 能否证明梯度 $\nabla f$ 与等值线（面）正交？
- [ ] 能否准确区分"梯度方向"、"最速上升方向"和"方向导数最大值"三者之间的关系？
- [ ] 给定一个非单位向量的方向 $\mathbf{w}$，能否正确处理单位化后再计算方向导数？
- [ ] 若方向导数为零，能推出什么结论？该方向与梯度是什么关系？

**1.** 设 $f(x,y) = 3x^2 - 2xy + y^2$。

(1) 求 $\nabla f(1,2)$。
(2) 求 $f$ 在 $(1,2)$ 处沿方向 $\mathbf{v} = \left(\frac{1}{2},\ \frac{\sqrt{3}}{2}\right)$ 的方向导数。
(3) 求 $f$ 在 $(1,2)$ 处方向导数的最大值，以及取得最大值的方向。

<details><summary>参考答案</summary>

**(1)** 计算梯度。
$$f_x = 6x - 2y,\quad f_y = -2x + 2y$$
$$\nabla f(1,2) = (6\cdot1 - 2\cdot2,\ -2\cdot1 + 2\cdot2) = (6-4,\ -2+4) = (2,\ 2)$$

**(2)** $\mathbf{v} = \left(\frac{1}{2},\frac{\sqrt{3}}{2}\right)$ 已是单位向量（$(\frac12)^2 + (\frac{\sqrt{3}}{2})^2 = \frac14 + \frac34 = 1$）。
$$\frac{\partial f}{\partial\mathbf{v}}(1,2) = \nabla f(1,2) \cdot \mathbf{v} = (2,2) \cdot \left(\frac12,\ \frac{\sqrt{3}}{2}\right) = 2\cdot\frac12 + 2\cdot\frac{\sqrt{3}}{2} = 1 + \sqrt{3}$$

**(3)** 方向导数的最大值：
$$\|\nabla f(1,2)\| = \sqrt{2^2 + 2^2} = \sqrt{8} = 2\sqrt{2}$$
取得最大值的方向是梯度方向单位化：
$$\frac{\nabla f(1,2)}{\|\nabla f(1,2)\|} = \frac{(2,2)}{2\sqrt{2}} = \left(\frac{1}{\sqrt{2}},\ \frac{1}{\sqrt{2}}\right)$$

即沿 $45^\circ$ 方向（与 $x$ 轴夹角 $45^\circ$）时函数增长最快，变化率为 $2\sqrt{2}$。

</details>

**2.** 设 $f(x,y) = \ln(1 + x^2 + y^2)$，求 $f$ 在点 $(1,1)$ 处沿方向 $\mathbf{w} = (1,-2)$ 的方向导数。

<details><summary>参考答案</summary>

**第 1 步：单位化方向向量**。
$$\|\mathbf{w}\| = \sqrt{1^2 + (-2)^2} = \sqrt{5},\quad \mathbf{v} = \frac{\mathbf{w}}{\|\mathbf{w}\|} = \left(\frac{1}{\sqrt{5}},\ -\frac{2}{\sqrt{5}}\right)$$

**第 2 步：求梯度**。
$$f_x = \frac{2x}{1 + x^2 + y^2},\quad f_y = \frac{2y}{1 + x^2 + y^2}$$
在 $(1,1)$ 处：
$$f_x(1,1) = \frac{2}{1 + 1 + 1} = \frac{2}{3},\quad f_y(1,1) = \frac{2}{1 + 1 + 1} = \frac{2}{3}$$
$$\nabla f(1,1) = \left(\frac{2}{3},\ \frac{2}{3}\right)$$

**第 3 步：点积**。
$$\frac{\partial f}{\partial\mathbf{v}}(1,1) = \nabla f(1,1) \cdot \mathbf{v} = \left(\frac{2}{3},\ \frac{2}{3}\right) \cdot \left(\frac{1}{\sqrt{5}},\ -\frac{2}{\sqrt{5}}\right) = \frac{2}{3}\cdot\frac{1}{\sqrt{5}} + \frac{2}{3}\cdot\left(-\frac{2}{\sqrt{5}}\right) = \frac{2}{3\sqrt{5}} - \frac{4}{3\sqrt{5}} = -\frac{2}{3\sqrt{5}}$$

因此 $f$ 在 $(1,1)$ 处沿方向 $(1,-2)$ 的方向导数为 $-\dfrac{2}{3\sqrt{5}}$（函数值下降）。

</details>

### 迁移应用

**3.** 设温度场 $T(x,y) = 80 - 2x^2 - 3y^2$（单位：$^\circ$C）。

(1) 求 $\nabla T(1,2)$ 并说明其物理意义。
(2) 在点 $(1,2)$ 处，沿哪个方向移动可使温度上升最快？该方向的温度变化率是多少？
(3) 在 $(1,2)$ 处，是否存在使温度不变的方向？如果存在，求出一个这样的方向。
(4) 验证梯度 $\nabla T(1,2)$ 与等温线 $T(x,y) = T(1,2)$ 在 $(1,2)$ 处的切线正交。

<details><summary>参考答案</summary>

**(1)** 计算梯度。
$$T_x = -4x,\quad T_y = -6y$$
$$\nabla T(1,2) = (-4,\ -12)$$

**物理意义**：$\nabla T(1,2)$ 的方向是 $(1,2)$ 处温度上升最快的方向。由于两个分量均为负，温度在 $(1,2)$ 处沿 $x$ 正方向下降（变化率 $-4^\circ$C/单位长度），沿 $y$ 正方向也下降（变化率 $-12^\circ$C/单位长度）。

**(2)** 最速上升方向是 $\nabla T(1,2)$ 方向（定理 12.4），单位化为：
$$\frac{\nabla T(1,2)}{\|\nabla T(1,2)\|} = \frac{(-4,-12)}{\sqrt{16+144}} = \frac{(-4,-12)}{\sqrt{160}} = \frac{(-4,-12)}{4\sqrt{10}} = \left(-\frac{1}{\sqrt{10}},\ -\frac{3}{\sqrt{10}}\right)$$

该方向上的变化率：
$$\|\nabla T(1,2)\| = \sqrt{(-4)^2 + (-12)^2} = \sqrt{160} = 4\sqrt{10} \approx 12.65\ ^\circ\text{C/单位长度}$$

**(3)** 存在使温度不变的方向沿等温线切线方向。设单位方向 $\mathbf{v} = (\cos\theta,\sin\theta)$ 满足方向导数为零：
$$\frac{\partial T}{\partial\mathbf{v}}(1,2) = \nabla T(1,2) \cdot \mathbf{v} = (-4,\ -12) \cdot (\cos\theta,\sin\theta) = -4\cos\theta - 12\sin\theta = 0$$
解得：$-4\cos\theta = 12\sin\theta$，即 $\tan\theta = -\frac{1}{3}$。因此 $\theta = \arctan(-1/3)$ 或 $\theta = \pi + \arctan(-1/3)$。

取其一：$\mathbf{v} = \left(\frac{3}{\sqrt{10}},\ -\frac{1}{\sqrt{10}}\right)$（可验证 $\nabla T \cdot \mathbf{v} = (-4)(\frac{3}{\sqrt{10}}) + (-12)(-\frac{1}{\sqrt{10}}) = -\frac{12}{\sqrt{10}} + \frac{12}{\sqrt{10}} = 0$）。

**(4)** 计算 $T(1,2) = 80 - 2\cdot1^2 - 3\cdot2^2 = 80 - 2 - 12 = 66$。等温线方程为 $80 - 2x^2 - 3y^2 = 66$，即 $2x^2 + 3y^2 = 14$。

在 $(1,2)$ 附近将 $y$ 视作 $x$ 的函数，对等温线方程两边对 $x$ 求导：
$$4x + 6y\frac{dy}{dx} = 0 \quad\Rightarrow\quad \frac{dy}{dx} = -\frac{4x}{6y} = -\frac{2x}{3y}$$
代入 $(1,2)$：$y' = -\frac{2}{6} = -\frac{1}{3}$。切线方向可取为 $\mathbf{T} = (3,-1)$。

验证正交：
$$\nabla T(1,2) \cdot \mathbf{T} = (-4,\ -12)\cdot(3,\ -1) = -12 + 12 = 0 \quad\checkmark$$

梯度与等温线正交得证。

</details>

**4.** 设 $f(x,y) = x^2y$。

(1) 求 $\nabla f(2,1)$。
(2) 求 $f$ 在 $(2,1)$ 处沿与 $x$ 轴正方向夹角 $60^\circ$ 的方向的方向导数。
(3) 求 $f$ 在 $(2,1)$ 处方向导数为零的方向（即 $f$ 的"平坦方向"）。

<details><summary>参考答案</summary>

**(1)** 计算梯度。
$$f_x = 2xy,\quad f_y = x^2$$
$$\nabla f(2,1) = (2\cdot2\cdot1,\ 2^2) = (4,\ 4)$$

**(2)** 与 $x$ 轴正方向夹角 $60^\circ$ 的单位向量为：
$$\mathbf{v} = (\cos60^\circ,\ \sin60^\circ) = \left(\frac12,\ \frac{\sqrt{3}}{2}\right)$$
方向导数：
$$\frac{\partial f}{\partial\mathbf{v}}(2,1) = \nabla f(2,1) \cdot \mathbf{v} = (4,\ 4)\cdot\left(\frac12,\ \frac{\sqrt{3}}{2}\right) = 2 + 2\sqrt{3}$$

**(3)** 设 $\mathbf{v} = (\cos\theta,\sin\theta)$，令方向导数为零：
$$\frac{\partial f}{\partial\mathbf{v}}(2,1) = (4,\ 4)\cdot(\cos\theta,\sin\theta) = 4\cos\theta + 4\sin\theta = 0$$
解得：$\cos\theta + \sin\theta = 0$，即 $\tan\theta = -1$。

因此 $\theta = -\frac{\pi}{4}$ 或 $\theta = \frac{3\pi}{4}$，对应方向：
$$\mathbf{v}_1 = \left(\frac{1}{\sqrt{2}},\ -\frac{1}{\sqrt{2}}\right),\quad \mathbf{v}_2 = \left(-\frac{1}{\sqrt{2}},\ \frac{1}{\sqrt{2}}\right)$$

验证：$\nabla f(2,1) \cdot \mathbf{v}_1 = (4,4)\cdot(\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}}) = \frac{4}{\sqrt{2}} - \frac{4}{\sqrt{2}} = 0$。这两个方向均与 $\nabla f$ 正交，沿它们移动时 $f$ 值不变（一阶近似下），即它们是等值线的切线方向。

</details>
