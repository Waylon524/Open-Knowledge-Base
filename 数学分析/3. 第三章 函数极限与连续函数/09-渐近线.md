## 2. 背景与应用场景

在前序文件中，我们系统学习了函数在有限点 $x_0$ 处的极限理论。但现实世界中的许多函数行为发生在"无穷远处"——当自变量趋于无穷大时，函数值是否会稳定下来？是否会趋近于某条直线？

这一类问题的几何对应就是**渐近线**。

**什么是渐近线？** 直观地说，渐近线是一条直线，当自变量趋于无穷（或趋于某个有限点）时，函数图像无限"贴近"这条直线。就像一条水平线 $y = L$ 如果当 $x\to\infty$ 时 $f(x) \to L$，那么函数图像在向右无限延伸时会越来越靠近这条水平线。

渐近线在科学和工程中有着广泛的应用：
- **物理学中的终端速度**：物体在流体中下落时，速度趋于一个极限值（终端速度），其函数图像有一条水平渐近线
- **化学中的反应平衡**：反应物浓度随时间趋于平衡浓度——函数图像在 $t\to\infty$ 时趋近水平线
- **电路分析中的RC充电**：电容器电压 $V(t) = V_0(1 - e^{-t/RC})$ 在 $t\to\infty$ 时趋近 $V_0$，$y = V_0$ 是一条水平渐近线
- **双曲线函数**：$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 的两条渐近线 $y = \pm\frac{b}{a}x$ 正是斜渐近线的经典范例

渐近线分析是研究函数性质的重要工具——它告诉我们函数在"远方"的走向，帮助我们从整体上把握函数图像的形状。

## 3. 核心概念与定义

### 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\lim_{x\to +\infty} f(x) = L$ | $x$ 趋于正无穷时 $f(x)$ 的极限为 $L$ | $\lim_{x\to\infty} \frac{1}{x} = 0$ |
| $\varepsilon$ | 任意给定的正数，衡量函数值与极限值的接近程度 | $\varepsilon = 0.001$ |
| $X$ | 存在的正数，衡量 $x$ "足够大"的阈值（依赖 $\varepsilon$） | $X = \frac{1}{\varepsilon}$ |
| $y = L$ | 水平渐近线 | $y = 2$ 是 $f(x)=\frac{2x}{x-1}$ 的水平渐近线 |
| $x = x_0$ | 铅直渐近线 | $x = 1$ 是 $f(x)=\frac{1}{x-1}$ 的铅直渐近线 |
| $y = kx + b$ | 斜渐近线（$k \neq 0$） | $y = x$ 是 $f(x)=\sqrt{x^2+1}$ 的斜渐近线 |
| $k$ | 斜渐近线的斜率 | $k = \lim_{x\to\infty} \frac{f(x)}{x}$ |
| $b$ | 斜渐近线的截距 | $b = \lim_{x\to\infty} [f(x) - kx]$ |

### 3.1 函数在无穷远处的极限（$\varepsilon$-$X$ 定义）

在前序文件（文件 01）中，我们只定义了 $x\to x_0$（$x_0$ 有限）时的函数极限。要讨论水平渐近线和斜渐近线，首先需要严格定义"$x$ 趋于无穷大时函数的极限"。

**定义 9.1（$x\to +\infty$ 时函数的极限）**：设函数 $f(x)$ 在 $[a, +\infty)$（$a$ 为某实数）上有定义，$L$ 为常数。若对任意 $\varepsilon > 0$，存在 $X > 0$，使得当 $x > X$ 时，恒有
$$|f(x) - L| < \varepsilon$$
则称当 $x$ 趋于正无穷时 $f(x)$ 的极限为 $L$，记作
$$\lim_{x\to +\infty} f(x) = L$$

用逻辑符号表述：
$$\lim_{x\to +\infty} f(x) = L \iff \forall\varepsilon>0,\ \exists X>0,\ \forall x\,(x > X):\ |f(x)-L|<\varepsilon$$

**定义 9.2（$x\to -\infty$ 时函数的极限）**：设函数 $f(x)$ 在 $(-\infty, b]$（$b$ 为某实数）上有定义，$L$ 为常数。若对任意 $\varepsilon > 0$，存在 $X > 0$，使得当 $x < -X$ 时，恒有
$$|f(x) - L| < \varepsilon$$
则称当 $x$ 趋于负无穷时 $f(x)$ 的极限为 $L$，记作
$$\lim_{x\to -\infty} f(x) = L$$

用逻辑符号表述：
$$\lim_{x\to -\infty} f(x) = L \iff \forall\varepsilon>0,\ \exists X>0,\ \forall x\,(x < -X):\ |f(x)-L|<\varepsilon$$

有时我们简写 $\lim_{x\to\infty} f(x) = L$，此时默认同时涵盖 $x\to +\infty$ 和 $x\to -\infty$ 两个方向，或者只在特定上下文中指代某一方向。但在严格的渐近线分析中，**必须分别处理 $+\infty$ 和 $-\infty$ 两个方向**。

**与 $\varepsilon$-$\delta$ 定义的类比**：

| 对比项 | $\varepsilon$-$\delta$（$x\to x_0$） | $\varepsilon$-$X$（$x\to +\infty$） |
|--------|--------------------------------------|--------------------------------------|
| 条件 | $0 < |x-x_0| < \delta$ | $x > X$ |
| 几何含义 | $x$ 在 $x_0$ 的去心邻域内 | $x$ 在 $X$ 的右侧 |
| 阈值 | $\delta$（依赖 $\varepsilon$） | $X$（依赖 $\varepsilon$） |
| 收敛判定 | $|f(x)-L| < \varepsilon$ | $|f(x)-L| < \varepsilon$ |

**关于四则运算法则在无穷远处的适用性**：文件 02 中证明的四则运算法则（定理 2.5）和夹逼准则（定理 2.6）对 $x\to +\infty$ 和 $x\to -\infty$ 的情形**同样适用**。原因在于这些定理的 $\varepsilon$-$\delta$ 证明只依赖极限定义的逻辑结构（$\forall\varepsilon>0$、$\exists$ 阈值、$\forall x$ 在阈值范围内），而 $\varepsilon$-$X$ 定义具有完全相同的逻辑结构——只需将"$0<|x-x_0|<\delta$"替换为"$x > X$"即可得到完全平行的证明。在后续计算中，我们将直接使用这些运算法则而不重复证明。

### 3.2 渐近线的定义

**定义 9.3（渐近线）**：若曲线 $y = f(x)$ 与某条直线 $L$ 之间的距离在自变量沿某方向趋于无穷（或趋于某有限点）时趋于 $0$，则称直线 $L$ 为该曲线的一条**渐近线**（asymptote）。

渐近线的几何直觉是：函数图像在"远方"无限接近某条直线，但通常不会（也并非必须）与直线相交。

### 3.3 渐近线的三种类型

根据渐近线的几何形态，我们将其分为三类：

| 类型 | 方程形式 | 几何特征 | 判定基础 |
|------|----------|----------|----------|
| 水平渐近线 | $y = L$（$L$ 为有限常数） | 水平方向，$x\to\pm\infty$ 时 $f(x)\to L$ | $\lim_{x\to\pm\infty} f(x) = L$ |
| 铅直渐近线 | $x = x_0$ | 竖直方向，$x\to x_0$ 时 $f(x)\to\infty$ | $\lim_{x\to x_0^\pm} f(x) = \pm\infty$ |
| 斜渐近线 | $y = kx + b$（$k \neq 0$） | 倾斜直线 | $\lim_{x\to\infty}[f(x)-(kx+b)] = 0$ |

## 4. 原理与方法

### 4.1 水平渐近线

#### 4.1.1 定义与判定

**定义 9.4（水平渐近线）**：设函数 $f(x)$ 在 $[a,+\infty)$ 上有定义。若
$$\lim_{x\to +\infty} f(x) = L$$
其中 $L$ 为有限常数，则称直线 $y = L$ 为曲线 $y = f(x)$ 当 $x\to +\infty$ 时的水平渐近线（horizontal asymptote）。

类似地，若 $\displaystyle\lim_{x\to -\infty} f(x) = M$（$M$ 为有限常数），则称 $y = M$ 为曲线当 $x\to -\infty$ 时的水平渐近线。

**关键点**：
- 极限值必须是**有限常数**。若 $\lim_{x\to\infty} f(x) = \pm\infty$，则不存在水平渐近线。
- $x\to +\infty$ 和 $x\to -\infty$ 两个方向需要**分别判定**。它们可能有相同或不同的水平渐近线。
- 一个方向上最多有一条水平渐近线（由极限的唯一性保证）。

#### 4.1.2 判定流程

要判断曲线 $y = f(x)$ 是否存在水平渐近线，只需计算：
$$\lim_{x\to +\infty} f(x),\qquad \lim_{x\to -\infty} f(x)$$

若 $\lim_{x\to +\infty} f(x) = L$（有限），则 $y = L$ 是 $x\to +\infty$ 方向上的水平渐近线。
若 $\lim_{x\to -\infty} f(x) = M$（有限），则 $y = M$ 是 $x\to -\infty$ 方向上的水平渐近线。
若极限为 $\infty$（或 $+\infty$、$-\infty$），则该方向**无水平渐近线**——可能需要进一步检查斜渐近线。

#### 4.1.3 示例

对于 $f(x) = \frac{2x+3}{x-1}$：
$$\lim_{x\to +\infty} \frac{2x+3}{x-1} = \lim_{x\to +\infty} \frac{2 + \frac{3}{x}}{1 - \frac{1}{x}} = \frac{2 + 0}{1 - 0} = 2$$
$$\lim_{x\to -\infty} \frac{2x+3}{x-1} = \lim_{x\to -\infty} \frac{2 + \frac{3}{x}}{1 - \frac{1}{x}} = \frac{2 + 0}{1 - 0} = 2$$

两个方向都有 $\lim f(x) = 2$，因此 $y = 2$ 是曲线在两个方向上的水平渐近线。

---

### 4.2 铅直渐近线

#### 4.2.1 定义与判定

**定义 9.5（铅直渐近线）**：设函数 $f(x)$ 在 $x_0$ 的某个去心邻域内有定义。若
$$\lim_{x\to x_0^+} f(x) = \pm\infty \quad \text{或} \quad \lim_{x\to x_0^-} f(x) = \pm\infty$$
即 $f(x)$ 在 $x_0$ 处的至少一侧极限为无穷大，则称直线 $x = x_0$ 为曲线 $y = f(x)$ 的一条**铅直渐近线**（vertical asymptote）。

**关键点**：
- 铅直渐近线只需要**单侧极限**为无穷大即可判定——不需要双侧同时无穷。
- 铅直渐近线对应**无穷间断点**（第二类间断点中的无穷间断点）。若 $x_0$ 是 $f(x)$ 的可去间断点或跳跃间断点（左右极限均为有限），则 $x = x_0$ 不是铅直渐近线。
- 铅直渐近线通常出现在分母为零但分子非零的位置（对有理函数），或者出现在函数无定义且附近函数趋于无穷的位置。

#### 4.2.2 与无穷大量定义的衔接

铅直渐近线的判定直接依赖于文件 04 中建立的无穷大量 $M$-$\delta$ 定义（定义 3.2）。回顾：

设 $\lim_{x\to x_0^+} f(x) = +\infty$ 意味着：
$$\forall M > 0,\ \exists\delta > 0,\ \forall x\,(x_0 < x < x_0 + \delta):\ f(x) > M$$

几何上：当 $x$ 从右侧足够靠近 $x_0$ 时，函数图像"冲上"到 $y = M$ 水平线以上，且对任意更高的 $M$ 都能做到。此时 $x = x_0$ 就是铅直渐近线。

**例**：对于 $f(x) = \frac{1}{x-1}$：
- $\lim_{x\to 1^+} \frac{1}{x-1} = +\infty$（分子 $1 > 0$，分母 $x-1 \to 0^+$）
- $\lim_{x\to 1^-} \frac{1}{x-1} = -\infty$（分子 $1 > 0$，分母 $x-1 \to 0^-$）

因此 $x = 1$ 是铅直渐近线。

#### 4.2.3 铅直渐近线的寻找方法

铅直渐近线通常（但不限于）出现在以下位置：

1. **有理函数** $f(x) = \frac{P(x)}{Q(x)}$：铅直渐近线出现在分母 $Q(x) = 0$ 且分子 $P(x) \neq 0$ 的 $x$ 值处。若 $P(x_0) = 0$ 且 $Q(x_0) = 0$，则需要先约分消除公共因子后再判断。

2. **对数函数** $f(x) = \ln x$：$x = 0$ 是铅直渐近线（$\lim_{x\to 0^+} \ln x = -\infty$）。

3. **正切函数** $f(x) = \tan x$：$x = \frac{\pi}{2} + k\pi$ 是铅直渐近线。

**重要区分**：并非所有分母为零的点都产生铅直渐近线。若分母为零但分子也为零，且约分后分母不再为零，则该点可能是可去间断点而非无穷间断点。

### 4.3 斜渐近线

#### 4.3.1 定义

**定义 9.6（斜渐近线）**：设函数 $f(x)$ 在 $[a, +\infty)$ 上有定义。若存在非零常数 $k$ 和常数 $b$，使得
$$\lim_{x\to +\infty} \bigl[f(x) - (kx + b)\bigr] = 0$$
则称直线 $y = kx + b$ 为曲线 $y = f(x)$ 当 $x\to +\infty$ 时的**斜渐近线**（oblique asymptote）。

类似地定义 $x\to -\infty$ 方向的斜渐近线。

**几何直观**：斜渐近线是一条倾斜的直线，当 $x$ 趋于无穷时，函数图像与它的竖直距离趋于 $0$。

#### 4.3.2 判定公式的推导

给定 $f(x)$，如何确定斜渐近线的 $k$ 和 $b$？

**第一步：求斜率 $k$。**

由定义，$\displaystyle\lim_{x\to\infty} [f(x) - kx - b] = 0$。

将式子改写为 $f(x) - kx = b + \alpha(x)$，其中 $\alpha(x)$ 是 $x\to\infty$ 时的无穷小量。

两边同除以 $x$（$x \to \infty$ 时 $x \neq 0$）：
$$\frac{f(x)}{x} - k = \frac{b}{x} + \frac{\alpha(x)}{x}$$

当 $x\to\infty$ 时，$\frac{b}{x} \to 0$，$\frac{\alpha(x)}{x} \to 0$（因为 $\alpha(x)$ 是无穷小量，除以 $x$ 后仍是无穷小量）。因此：
$$\lim_{x\to\infty} \left(\frac{f(x)}{x} - k\right) = 0$$

即：
$$k = \lim_{x\to\infty} \frac{f(x)}{x}$$

**关键条件**：$k$ 必须是一个**有限常数**。若 $k = 0$，则斜渐近线退化为水平渐近线；若 $\lim f(x)/x$ 不存在或为 $\infty$，则该方向不存在斜渐近线。

**第二步：求截距 $b$。**

已知 $k$ 后，由定义 $\displaystyle\lim_{x\to\infty} [f(x) - kx - b] = 0$，即 $f(x) - kx \to b$。因此：
$$b = \lim_{x\to\infty} \bigl[f(x) - kx\bigr]$$

**条件**：$b$ 必须是**有限常数**。若 $\lim [f(x) - kx]$ 不存在或为 $\infty$，则不存在斜渐近线。

**第三步：验证**。

验证 $k \neq 0$（但即使 $k=0$ 且 $b$ 有限，此时得到的是水平渐近线），以及 $k$ 和 $b$ 都是有限常数。

#### 4.3.3 水平渐近线作为斜渐近线的特例

当 $k = 0$ 且 $b = L$（有限）时，斜渐近线公式给出 $y = L$，这正是水平渐近线。因此：

**水平渐近线是斜渐近线当 $k=0$ 时的特例**。

在判定流程中，我们习惯先检查水平渐近线（计算 $\lim f(x)$ 是否为有限常数），若不存在再检查斜渐近线。但这两种渐近线不会在同一方向上同时存在——因为若 $\lim f(x) = L$（有限），则 $\lim f(x)/x = 0$，此时斜渐近线的 $k=0$，就退化成了水平渐近线。

#### 4.3.4 有理函数的斜渐近线——多项式除法

对于有理函数 $f(x) = \frac{P(x)}{Q(x)}$，其中 $P$ 和 $Q$ 为多项式，且 $\deg P = \deg Q + 1$（分子的次数比分母高一次），我们可以直接通过**多项式除法**求出斜渐近线。

设 $P(x) \div Q(x)$ 的商为 $kx + b$（一次多项式），余数为 $R(x)$（次数低于 $Q(x)$ 的次数），则：
$$\frac{P(x)}{Q(x)} = kx + b + \frac{R(x)}{Q(x)}$$

当 $x\to\infty$ 时，$\frac{R(x)}{Q(x)} \to 0$（因为分子次数低于分母次数），因此：
$$\lim_{x\to\infty} \left[\frac{P(x)}{Q(x)} - (kx + b)\right] = 0$$

由此直接读出斜渐近线为 $y = kx + b$。

**例**：$f(x) = \frac{x^2 + x + 1}{x - 2}$。做多项式除法：
$$
\begin{aligned}
\frac{x^2 + x + 1}{x - 2} &= x + 3 + \frac{7}{x - 2}
\end{aligned}
$$

商为 $x + 3$，余数为 $7$。当 $x\to\infty$ 时，$\frac{7}{x-2} \to 0$。因此 $y = x + 3$ 是斜渐近线。

多项式除法避免了每次都从极限定义出发计算 $k$ 和 $b$，是处理有理函数斜渐近线的最快捷方法。

**有理函数渐近线总结**：

| 分子次数与分母次数关系 | 渐近线类型 |
|------------------------|------------|
| $\deg P = \deg Q$ | 水平渐近线 $y = \frac{a_n}{b_n}$（首项系数比） |
| $\deg P = \deg Q + 1$ | 斜渐近线（多项式除法的商） |
| $\deg P < \deg Q$ | 水平渐近线 $y = 0$ |
| $\deg P > \deg Q + 1$ | 无水平渐近线，也无斜渐近线（函数趋于无穷的速度高于一次函数） |

#### 4.3.5 含根式函数的斜渐近线——有理化技巧

对于含根式的函数（如 $\sqrt{x^2 + 1}$），计算斜渐近线时需要利用**有理化**技巧来处理 $f(x) - kx$ 中的 $\infty - \infty$ 型不定式。

**例**：$f(x) = \sqrt{x^2 + 1}$ 当 $x\to +\infty$ 时。

**第一步** 计算 $k$：
$$k = \lim_{x\to +\infty} \frac{\sqrt{x^2 + 1}}{x} = \lim_{x\to +\infty} \sqrt{\frac{x^2 + 1}{x^2}} = \lim_{x\to +\infty} \sqrt{1 + \frac{1}{x^2}} = \sqrt{1 + 0} = 1$$

**第二步** 计算 $b$：
$$b = \lim_{x\to +\infty} \bigl(\sqrt{x^2 + 1} - x\bigr)$$

这是 $\infty - \infty$ 型不定式。有理化处理：
$$\sqrt{x^2 + 1} - x = \frac{(\sqrt{x^2 + 1} - x)(\sqrt{x^2 + 1} + x)}{\sqrt{x^2 + 1} + x} = \frac{(x^2 + 1) - x^2}{\sqrt{x^2 + 1} + x} = \frac{1}{\sqrt{x^2 + 1} + x}$$

因此：
$$b = \lim_{x\to +\infty} \frac{1}{\sqrt{x^2 + 1} + x} = \frac{1}{+\infty} = 0$$

**第三步** 结论：当 $x\to +\infty$ 时，斜渐近线为 $y = x$。

**特别注意 $x\to -\infty$ 的方向**：当 $x\to -\infty$ 时，$x < 0$。处理 $\frac{\sqrt{x^2 + 1}}{x}$ 时需要注意 $\sqrt{x^2} = |x|$：
$$\frac{\sqrt{x^2 + 1}}{x} = \frac{|x|\sqrt{1 + 1/x^2}}{x}$$

当 $x < 0$ 时，$|x| = -x$，因此：
$$\frac{\sqrt{x^2 + 1}}{x} = \frac{-x\sqrt{1 + 1/x^2}}{x} = -\sqrt{1 + \frac{1}{x^2}} \to -1$$

所以 $k = -1$（与 $x\to +\infty$ 方向不同！）。类似地可以计算 $b = 0$，得到斜渐近线 $y = -x$。

这个例子说明：**$x\to +\infty$ 和 $x\to -\infty$ 方向必须分别处理，可能得到不同的渐近线**。

### 4.4 渐近线的统一处理与判定流程

在实际分析一条曲线的全部渐近线时，建议遵循以下流程：

```
Step 1: 确定函数的定义域
   ↓
Step 2: 寻找铅直渐近线
        函数的间断点 x₀（尤其是分母为零的点）
    → 检查 lim_{x→x₀⁺} f(x) 和 lim_{x→x₀⁻} f(x) 是否为无穷大
    → 若至少一侧为无穷大，则 x = x₀ 是铅直渐近线
   ↓
Step 3: 检查 x→+∞ 方向
    → 计算 lim_{x→+∞} f(x)
    → 若结果为有限常数 L → 水平渐近线 y = L
    → 若结果为无穷大 → 检查斜渐近线：
       计算 k = lim_{x→+∞} f(x)/x
       若 k 为有限非零常数：
          计算 b = lim_{x→+∞} [f(x) - kx]
          若 b 为有限常数 → 斜渐近线 y = kx + b
       否则该方向无斜渐近线
   ↓
Step 4: 检查 x→-∞ 方向（独立进行，与 Step 3 相同流程）
   ↓
Step 5: 汇总全部渐近线
```

**重要原则**：
- 同一方向上，水平渐近线和斜渐近线**不能共存**（因为水平渐近线是斜渐近线的 $k=0$ 特例）。
- 同一方向上最多有一条水平渐近线或一条斜渐近线（由极限的唯一性保证）。
- 不同方向可以有不同的渐近线（如 $y = \sqrt{x^2+1}$ 在 $+\infty$ 方向有 $y=x$，在 $-\infty$ 方向有 $y=-x$）。
- 铅直渐近线的数量可以有多条（每个无穷间断点对应一条）。

## 5. 例题

### 例 1：有理函数的全部渐近线

求曲线 $f(x) = \dfrac{2x + 3}{x - 1}$ 的全部渐近线。

**解**：

**第一步：确定定义域**。$f(x)$ 在 $x = 1$ 处无定义，定义域为 $(-\infty, 1) \cup (1, +\infty)$。

**第二步：检查铅直渐近线**。

$x = 1$ 是分母为零的点，分子 $2\cdot 1 + 3 = 5 \neq 0$。计算单侧极限：
$$\lim_{x\to 1^+} \frac{2x+3}{x-1} = +\infty \quad (\text{分子}\to 5 > 0,\ \text{分母}\to 0^+)$$
$$\lim_{x\to 1^-} \frac{2x+3}{x-1} = -\infty \quad (\text{分子}\to 5 > 0,\ \text{分母}\to 0^-)$$

一侧为 $+\infty$，一侧为 $-\infty$，满足铅直渐近线的定义。因此 $x = 1$ 是铅直渐近线。

**第三步：检查 $x\to +\infty$ 方向的水平/斜渐近线**。

先检查水平渐近线：
$$\lim_{x\to +\infty} \frac{2x+3}{x-1} = \lim_{x\to +\infty} \frac{2 + \frac{3}{x}}{1 - \frac{1}{x}} = \frac{2 + 0}{1 - 0} = 2$$

极限为有限常数 $2$，因此 $y = 2$ 是 $x\to +\infty$ 方向的水平渐近线。

**第四步：检查 $x\to -\infty$ 方向的水平/斜渐近线**。

与 $x\to +\infty$ 同理：
$$\lim_{x\to -\infty} \frac{2x+3}{x-1} = \lim_{x\to -\infty} \frac{2 + \frac{3}{x}}{1 - \frac{1}{x}} = \frac{2 + 0}{1 - 0} = 2$$

同样得到 $y = 2$。由于两个方向都有水平渐近线（且相同），该方向不再有斜渐近线。

**最终汇总**：
- 铅直渐近线：$x = 1$
- 水平渐近线：$y = 2$（$x\to +\infty$ 和 $x\to -\infty$ 方向共用）

**数值验证**：取 $x = 100$，$f(100) = \frac{200+3}{99} = \frac{203}{99} \approx 2.0505$；取 $x = 1000$，$f(1000) = \frac{2000+3}{999} = \frac{2003}{999} \approx 2.005$。函数值确实越来越接近 $2$。

### 例 2：斜渐近线与多项式除法

求曲线 $f(x) = \dfrac{x^2 + x + 1}{x - 2}$ 的全部渐近线。

**解**：

**第一步：确定定义域**。$f(x)$ 在 $x = 2$ 处无定义，定义域为 $(-\infty, 2) \cup (2, +\infty)$。

**第二步：检查铅直渐近线**。

$x = 2$ 处分母为零，分子 $2^2 + 2 + 1 = 7 \neq 0$。
$$\lim_{x\to 2^+} \frac{x^2 + x + 1}{x - 2} = +\infty \quad (\text{分子}\to 7 > 0,\ \text{分母}\to 0^+)$$
$$\lim_{x\to 2^-} \frac{x^2 + x + 1}{x - 2} = -\infty \quad (\text{分子}\to 7 > 0,\ \text{分母}\to 0^-)$$

因此 $x = 2$ 是铅直渐近线。

**第三步：检查 $x\to +\infty$ 方向的水平/斜渐近线**。

先检查水平渐近线：
$$\lim_{x\to +\infty} \frac{x^2 + x + 1}{x - 2} = \lim_{x\to +\infty} \frac{x + 1 + \frac{1}{x}}{1 - \frac{2}{x}}$$

分子趋于 $+\infty$（二次函数除以一次函数），因此：
$$\lim_{x\to +\infty} f(x) = +\infty \quad (\text{不存在有限极限})$$

所以 $x\to +\infty$ 方向**无水平渐近线**。

进一步检查斜渐近线。方法一：用判定公式计算。

计算斜率 $k$：
$$k = \lim_{x\to +\infty} \frac{f(x)}{x} = \lim_{x\to +\infty} \frac{x^2 + x + 1}{x(x-2)} = \lim_{x\to +\infty} \frac{x^2 + x + 1}{x^2 - 2x}$$

分子分母同除以 $x^2$：
$$k = \lim_{x\to +\infty} \frac{1 + \frac{1}{x} + \frac{1}{x^2}}{1 - \frac{2}{x}} = \frac{1 + 0 + 0}{1 - 0} = 1$$

$k = 1$ 是有限非零常数，继续计算截距 $b$。

计算截距 $b$：
$$b = \lim_{x\to +\infty} \bigl[f(x) - kx\bigr] = \lim_{x\to +\infty} \left(\frac{x^2 + x + 1}{x - 2} - x\right)$$

通分：
$$\frac{x^2 + x + 1}{x - 2} - x = \frac{x^2 + x + 1 - x(x-2)}{x-2} = \frac{x^2 + x + 1 - x^2 + 2x}{x-2} = \frac{3x + 1}{x-2}$$

因此：
$$b = \lim_{x\to +\infty} \frac{3x + 1}{x - 2} = \lim_{x\to +\infty} \frac{3 + \frac{1}{x}}{1 - \frac{2}{x}} = \frac{3 + 0}{1 - 0} = 3$$

$b = 3$ 是有限常数。因此 $x\to +\infty$ 方向存在斜渐近线 $y = x + 3$。

方法二：多项式除法（更快捷）。

直接做除法：
$$
\begin{aligned}
\frac{x^2 + x + 1}{x - 2} &= \frac{(x-2)(x+3) + 7}{x-2} \\
&= x + 3 + \frac{7}{x-2}
\end{aligned}
$$

验证：$(x-2)(x+3) = x^2 + x - 6$，加 $7$ 得 $x^2 + x + 1$，正确。

当 $x\to +\infty$ 时，$\frac{7}{x-2} \to 0$，因此 $y = x + 3$ 是斜渐近线——与公式计算一致。

**第四步：检查 $x\to -\infty$ 方向的水平/斜渐近线**。

$$\lim_{x\to -\infty} \frac{x^2 + x + 1}{x - 2} = -\infty$$

无水平渐近线。

计算 $k$：
$$k = \lim_{x\to -\infty} \frac{x^2 + x + 1}{x(x-2)} = \lim_{x\to -\infty} \frac{1 + \frac{1}{x} + \frac{1}{x^2}}{1 - \frac{2}{x}} = 1$$

计算 $b$：
$$b = \lim_{x\to -\infty} \left(\frac{x^2 + x + 1}{x - 2} - x\right) = \lim_{x\to -\infty} \frac{3x + 1}{x - 2} = \lim_{x\to -\infty} \frac{3 + \frac{1}{x}}{1 - \frac{2}{x}} = 3$$

$x\to -\infty$ 方向也有相同的斜渐近线 $y = x + 3$。

**最终汇总**：
- 铅直渐近线：$x = 2$
- 斜渐近线：$y = x + 3$（$x\to +\infty$ 和 $x\to -\infty$ 方向共用）

**验证（多项式除法）**：$f(x) = x + 3 + \frac{7}{x-2}$。当 $x\to \pm\infty$ 时，$\frac{7}{x-2} \to 0$，$f(x) \to x + 3$；当 $x\to 2$ 时，$\frac{7}{x-2} \to \infty$，产生铅直渐近线。多项式除法一次性揭示了全部渐近线信息。

### 例 3：含根式函数——$x\to +\infty$ 和 $x\to -\infty$ 方向不同

求曲线 $f(x) = \sqrt{x^2 + 1}$ 的全部渐近线。

**解**：

**第一步：确定定义域**。$f(x)$ 在 $\mathbb{R}$ 上有定义且连续，无分母为零的点，因此**无铅直渐近线**。

**第二步：分析 $x\to +\infty$ 方向**。

(1) 检查水平渐近线：
$$\lim_{x\to +\infty} \sqrt{x^2 + 1} = +\infty$$

极限为无穷大，因此无水平渐近线。

(2) 检查斜渐近线：
$$k = \lim_{x\to +\infty} \frac{f(x)}{x} = \lim_{x\to +\infty} \frac{\sqrt{x^2 + 1}}{x}$$

当 $x > 0$ 时，可将 $x$ 移入根号：
$$\frac{\sqrt{x^2 + 1}}{x} = \sqrt{\frac{x^2 + 1}{x^2}} = \sqrt{1 + \frac{1}{x^2}}$$

因此：
$$k = \lim_{x\to +\infty} \sqrt{1 + \frac{1}{x^2}} = \sqrt{1 + 0} = 1$$

$k = 1$ 为有限非零常数，继续计算 $b$：
$$b = \lim_{x\to +\infty} \bigl(\sqrt{x^2 + 1} - x\bigr)$$

这是一个 $\infty - \infty$ 型不定式。有理化处理：

$$\sqrt{x^2 + 1} - x = \frac{(\sqrt{x^2 + 1} - x)(\sqrt{x^2 + 1} + x)}{\sqrt{x^2 + 1} + x} = \frac{(x^2 + 1) - x^2}{\sqrt{x^2 + 1} + x} = \frac{1}{\sqrt{x^2 + 1} + x}$$

因此：
$$b = \lim_{x\to +\infty} \frac{1}{\sqrt{x^2 + 1} + x} = \frac{1}{+\infty} = 0$$

所以当 $x\to +\infty$ 时，斜渐近线为 $y = x$。

**第三步：分析 $x\to -\infty$ 方向**。

(1) 检查水平渐近线：
$$\lim_{x\to -\infty} \sqrt{x^2 + 1} = +\infty$$

无水平渐近线。

(2) 检查斜渐近线。关键区别在于处理 $\sqrt{x^2+1}/x$ 时需要小心 $x$ 的符号。

计算 $k$：
$$k = \lim_{x\to -\infty} \frac{\sqrt{x^2 + 1}}{x}$$

当 $x < 0$ 时，$\sqrt{x^2} = |x| = -x$，因此：
$$\frac{\sqrt{x^2 + 1}}{x} = \frac{|x|\sqrt{1 + 1/x^2}}{x} = \frac{-x\sqrt{1 + 1/x^2}}{x} = -\sqrt{1 + \frac{1}{x^2}}$$

因此：
$$k = \lim_{x\to -\infty} \left(-\sqrt{1 + \frac{1}{x^2}}\right) = -\sqrt{1 + 0} = -1$$

$k = -1$ 是与 $x\to +\infty$ 方向**不同**的斜率！

继续计算 $b$：
$$b = \lim_{x\to -\infty} \bigl(\sqrt{x^2 + 1} - (-1)x\bigr) = \lim_{x\to -\infty} \bigl(\sqrt{x^2 + 1} + x\bigr)$$

这又是一个 $\infty - \infty$ 型不定式（当 $x\to -\infty$ 时，$\sqrt{x^2+1} \to +\infty$，$x \to -\infty$，但 $\sqrt{x^2+1} + x = \infty - \infty$）。同样有理化处理：

$$\sqrt{x^2 + 1} + x = \frac{(\sqrt{x^2 + 1} + x)(\sqrt{x^2 + 1} - x)}{\sqrt{x^2 + 1} - x} = \frac{(x^2 + 1) - x^2}{\sqrt{x^2 + 1} - x} = \frac{1}{\sqrt{x^2 + 1} - x}$$

当 $x\to -\infty$ 时，$x$ 为负，$\sqrt{x^2+1} > 0$，因此 $\sqrt{x^2+1} - x > \sqrt{x^2+1} + |x| \to +\infty$。于是：
$$b = \lim_{x\to -\infty} \frac{1}{\sqrt{x^2 + 1} - x} = \frac{1}{+\infty} = 0$$

所以当 $x\to -\infty$ 时，斜渐近线为 $y = -x$。

**最终汇总**：
曲线 $y = \sqrt{x^2 + 1}$ 无铅直渐近线。
- 当 $x\to +\infty$ 时，斜渐近线为 $y = x$
- 当 $x\to -\infty$ 时，斜渐近线为 $y = -x$

这两条渐近线关于 $y$ 轴对称，这与 $f(x) = \sqrt{x^2 + 1}$ 是偶函数（$f(-x)=f(x)$）的事实一致。当 $x$ 很大时，$\sqrt{x^2+1} \approx |x|$：正方向趋近 $y = x$，负方向趋近 $y = -x$。

**数值验证**：
- $x = 1000$：$\sqrt{1000001} \approx 1000.0005$，与 $y = x$ 的差约为 $0.0005$
- $x = -1000$：$\sqrt{1000001} \approx 1000.0005$，与 $y = -x$ 的差约为 $0.0005$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 函数图像不能穿过渐近线 | 函数可以穿过渐近线。渐近线刻画的是"极限行为"——在足够远处函数无限贴近直线，但中间区域完全可能相交。例如 $f(x)=\frac{\sin x}{x}$ 在 $x\to\infty$ 时有水平渐近线 $y=0$，但其图像会反复穿过 $y=0$ |
| 铅直渐近线要求双侧极限都是无穷大 | 只需要**至少一侧**的极限为无穷大即可。例如 $f(x)=\frac{1}{x}$ 在 $x=0$ 处：$\lim_{x\to 0^+}\frac{1}{x}=+\infty$，$\lim_{x\to 0^-}\frac{1}{x}=-\infty$，$\textstyle x=0$ 仍然是铅直渐近线 |
| 分母为零的点一定是铅直渐近线 | 分母为零但分子也为零的点，在约分后可能变为可去间断点而非无穷间断点。例如 $f(x)=\frac{x^2-1}{x-1}$，$x=1$ 处分母为零但分子也为零，约分后 $f(x)=x+1$，$x=1$ 是可去间断点，不是铅直渐近线 |
| 水平渐近线和斜渐近线可以在同一方向共存 | 同一方向上不能同时存在水平渐近线和斜渐近线。若 $\lim f(x)=L$（有限），则 $\lim f(x)/x = 0$，斜渐近线退化为水平渐近线。两个概念是包含关系（水平是斜的特例），而非并列关系 |
| $x\to +\infty$ 和 $x\to -\infty$ 方向一定有相同的渐近线 | 不一定。如 $f(x)=\sqrt{x^2+1}$：$x\to +\infty$ 方向有 $y=x$，$x\to -\infty$ 方向有 $y=-x$。对于非偶函数，两个方向的渐近线可以完全不同，需要分别分析 |
| 斜渐近线的 $k$ 和 $b$ 只要算出来就行，不用管 $k$ 是否为零或是否为有限 | 判定斜渐近线的条件是：$k$ 和 $b$ 都必须是有限常数。若 $k=0$，曲线在该方向有水平渐近线（$y=b$）；若 $k$ 本身不存在或为无穷，则无斜渐近线；若 $b$ 不存在或为无穷，也无斜渐近线 |
| 铅直渐近线只出现在有理函数中 | 铅直渐近线可以出现在各种函数中：对数函数 $\ln x$ 在 $x=0$ 处、正切函数 $\tan x$ 在 $x=\pi/2$ 处、以及 $\sqrt{x}$ 在 $x\to 0^+$ 时（但注意 $\sqrt{x}$ 在 $x=0$ 处有定义且有限，不是渐近线——铅直渐近线要求极限为无穷大） |
| 检查斜渐近线时，$k = \lim f(x)/x$ 和 $b = \lim [f(x)-kx]$ 两个极限可以任意顺序计算 | $b$ 的计算**依赖** $k$ 的值——必须先算出 $k$，再代入 $b$ 的计算。如果先算 $b$，不知道 $kx$ 项，就无法正确计算 $\lim [f(x)-kx]$ |

### 检查点

- [ ] 能否写出 $\displaystyle\lim_{x\to +\infty} f(x) = L$ 的 $\varepsilon$-$X$ 严格定义？
- [ ] 能否解释为什么 $x\to +\infty$ 和 $x\to -\infty$ 方向的渐近线需要分别检查？
- [ ] 能否准确判断一条直线是否是水平渐近线？什么条件下曲线无水平渐近线？
- [ ] 能否判断铅直渐近线？铅直渐近线的判定条件是什么？
- [ ] 能否从定义推导出斜渐近线的判定公式 $k = \lim f(x)/x$，$b = \lim[f(x)-kx]$？
- [ ] 能否证明水平渐近线是斜渐近线当 $k=0$ 时的特例？

(1) $f(x) = \dfrac{3x - 1}{x + 2}$

(2) $f(x) = \dfrac{4}{x^2 - 1}$

(3) $f(x) = \dfrac{2x^2 + 5x}{x - 3}$

**(1)** $f(x) = \dfrac{3x - 1}{x + 2}$

定义域：$x \neq -2$。

**铅直渐近线**：$x = -2$ 处分母为零，分子 $3(-2)-1 = -7 \neq 0$。
$$\lim_{x\to -2^+} \frac{3x-1}{x+2} = -\infty,\quad \lim_{x\to -2^-} \frac{3x-1}{x+2} = +\infty$$
因此 $x = -2$ 是铅直渐近线。

**水平渐近线**：
$$\lim_{x\to +\infty} \frac{3x-1}{x+2} = \lim_{x\to +\infty} \frac{3 - \frac{1}{x}}{1 + \frac{2}{x}} = 3$$
$$\lim_{x\to -\infty} \frac{3x-1}{x+2} = 3$$
因此 $y = 3$ 是 $x\to +\infty$ 和 $x\to -\infty$ 方向的水平渐近线。

**答案**：铅直渐近线 $x = -2$，水平渐近线 $y = 3$。

**(2)** $f(x) = \dfrac{4}{x^2 - 1}$

定义域：$x \neq \pm 1$。

**铅直渐近线**：$x = 1$ 处：分母 $x^2-1 \to 0$，分子 $4 \neq 0$。
$$\lim_{x\to 1^+} \frac{4}{x^2-1} = +\infty,\quad \lim_{x\to 1^-} \frac{4}{x^2-1} = -\infty$$

$x = -1$ 处：
$$\lim_{x\to -1^+} \frac{4}{x^2-1} = -\infty,\quad \lim_{x\to -1^-} \frac{4}{x^2-1} = +\infty$$

因此 $x = 1$ 和 $x = -1$ 都是铅直渐近线。

**水平渐近线**：
$$\lim_{x\to \pm\infty} \frac{4}{x^2 - 1} = 0$$

因此 $y = 0$ 是水平渐近线（两个方向共用）。

**答案**：铅直渐近线 $x = 1$、$x = -1$，水平渐近线 $y = 0$。

---
**(3)** $f(x) = \dfrac{2x^2 + 5x}{x - 3}$

定义域：$x \neq 3$。

**铅直渐近线**：$x = 3$ 处分母为零，分子 $2\cdot 9 + 15 = 33 \neq 0$。
$$\lim_{x\to 3^+} \frac{2x^2+5x}{x-3} = +\infty,\quad \lim_{x\to 3^-} \frac{2x^2+5x}{x-3} = -\infty$$
因此 $x = 3$ 是铅直渐近线。

**水平/斜渐近线**：
$$\lim_{x\to \pm\infty} \frac{2x^2+5x}{x-3} = \pm\infty$$

无水平渐近线。检查斜渐近线。

多项式除法：
$$\frac{2x^2+5x}{x-3} = 2x + 11 + \frac{33}{x-3}$$

验证：$(x-3)(2x+11) = 2x^2 + 11x - 6x - 33 = 2x^2 + 5x - 33$，加 $33$ 得 $2x^2 + 5x$，正确。

因此斜渐近线为 $y = 2x + 11$（$x\to +\infty$ 和 $x\to -\infty$ 方向共用）。

**答案**：铅直渐近线 $x = 3$，斜渐近线 $y = 2x + 11$。

</details>

**2.** 判断下列说法是否正确，若不正确请举出反例：

(1) 若曲线 $y = f(x)$ 有水平渐近线 $y = L$，则 $f(x)$ 在定义域内恒小于 $L$ 或恒大于 $L$。

(2) 若 $f(x)$ 是有理函数且分子次数比分母次数高一次，则曲线 $y = f(x)$ 一定有斜渐近线。

(3) 若 $\lim_{x\to +\infty} f(x) = +\infty$，则曲线 $y = f(x)$ 在 $x\to +\infty$ 方向一定没有水平渐近线，但可能有斜渐近线。

<details><summary>参考答案</summary>

**(1) 错误**。

反例：$f(x) = \frac{x}{x^2+1}$ 有水平渐近线 $y = 0$（因为 $\lim_{x\to \pm\infty} \frac{x}{x^2+1} = 0$）。但 $f(0) = 0$，函数值可以等于 $L$。且对于 $x > 0$，$f(x) > 0$；对于 $x < 0$，$f(x) < 0$。函数可以在渐近线两侧交替。

另一反例：$f(x) = \frac{\sin x}{x}$（定义 $f(0) = 1$）有水平渐近线 $y = 0$，但函数值在 $y = 0$ 两侧反复振荡并不断穿过 $y = 0$。

渐近线的定义只关心"足够远的极限行为"，不限制中间区域或近处的取值。

---
**(2) 正确**。

设 $f(x) = \frac{P(x)}{Q(x)}$，$\deg P = \deg Q + 1$。多项式除法给出：
$$\frac{P(x)}{Q(x)} = kx + b + \frac{R(x)}{Q(x)}$$

其中 $k \neq 0$（因为分子次数比分母高一次），$\deg R < \deg Q$。当 $x\to \pm\infty$ 时，$\frac{R(x)}{Q(x)} \to 0$，因此 $y = kx + b$ 是斜渐近线。

**注意**：分子次数比分母高一次是必要条件。若分子次数比分母高两次或以上（如 $\frac{x^3}{x-1}$），则 $k = \lim f(x)/x$ 不存在（为 $\infty$），无斜渐近线。

---
**(3) 正确**。

若 $\lim_{x\to +\infty} f(x) = +\infty$，则极限不是有限常数，因此不存在水平渐近线。

但可能存在斜渐近线——此时需要检查 $k = \lim f(x)/x$ 是否为有限非零常数。例如 $f(x) = \frac{x^2+x+1}{x-2}$ 满足 $\lim f(x) = +\infty$ 且存在斜渐近线 $y = x+3$。

但也有可能既无水平也无斜渐近线（如 $f(x) = x^2$，$\lim f(x)/x = \lim x = \infty$，不存在斜渐近线）。

</details>

### 迁移应用

**3.** 求曲线 $f(x) = \dfrac{x^3 - x^2 + 2x - 3}{x^2 + 1}$ 的全部渐近线。

<details><summary>参考答案</summary>

定义域：$\mathbb{R}$（分母 $x^2+1 > 0$ 恒成立），因此**无铅直渐近线**。

**检查水平渐近线**：
$$\lim_{x\to \pm\infty} \frac{x^3 - x^2 + 2x - 3}{x^2 + 1}$$

分子三次，分母二次，趋于无穷，无水平渐近线。

**检查斜渐近线**。做多项式除法：
$$
\begin{aligned}
\frac{x^3 - x^2 + 2x - 3}{x^2 + 1} &= \frac{x(x^2+1) - x^2 + x - 3}{x^2 + 1} \\
&= x + \frac{-x^2 + x - 3}{x^2 + 1} \\
&= x - 1 + \frac{x - 2}{x^2 + 1}
\end{aligned}
$$

验证：
- 第一步：$x^3$ 除以 $x^2$ 得 $x$，乘以 $(x^2+1)$ 得 $x^3 + x$，减去后得 $-x^2 + x - 3$
- 第二步：$-x^2$ 除以 $x^2$ 得 $-1$，乘以 $(x^2+1)$ 得 $-x^2 - 1$，减去后得 $x - 2$

因此 $f(x) = x - 1 + \frac{x - 2}{x^2 + 1}$。

当 $x\to \pm\infty$ 时，$\frac{x-2}{x^2+1} \to 0$，所以斜渐近线为 $y = x - 1$（两个方向共用）。

**验证**：
$$k = \lim_{x\to\infty} \frac{f(x)}{x} = \lim_{x\to\infty} \frac{x^3 - x^2 + 2x - 3}{x^3 + x} = \lim_{x\to\infty} \frac{1 - \frac{1}{x} + \frac{2}{x^2} - \frac{3}{x^3}}{1 + \frac{1}{x^2}} = 1$$
$$b = \lim_{x\to\infty} [f(x) - x] = \lim_{x\to\infty} \frac{-x^2 + 2x - 3}{x^2 + 1} = \lim_{x\to\infty} \frac{-1 + \frac{2}{x} - \frac{3}{x^2}}{1 + \frac{1}{x^2}} = -1$$

与多项式除法结果一致。

**答案**：斜渐近线 $y = x - 1$（无铅直渐近线，无水平渐近线）。

</details>

**4.** 设 $f(x) = \sqrt{4x^2 + x + 1}$。

(1) 求曲线 $y = f(x)$ 当 $x\to +\infty$ 时的斜渐近线。

(2) 求曲线 $y = f(x)$ 当 $x\to -\infty$ 时的斜渐近线。

（提示：利用 $\sqrt{4x^2 + x + 1} = 2|x|\sqrt{1 + \frac{1}{4x} + \frac{1}{4x^2}}$，注意 $x$ 为正负时的区别）

<details><summary>参考答案</summary>

**(1) $x\to +\infty$ 方向**：

先计算 $k$：
$$k = \lim_{x\to +\infty} \frac{\sqrt{4x^2 + x + 1}}{x} = \lim_{x\to +\infty} \sqrt{\frac{4x^2 + x + 1}{x^2}} = \lim_{x\to +\infty} \sqrt{4 + \frac{1}{x} + \frac{1}{x^2}} = \sqrt{4 + 0 + 0} = 2$$

计算 $b$：
$$b = \lim_{x\to +\infty} \bigl(\sqrt{4x^2 + x + 1} - 2x\bigr)$$

有理化：
$$\sqrt{4x^2 + x + 1} - 2x = \frac{(\sqrt{4x^2 + x + 1} - 2x)(\sqrt{4x^2 + x + 1} + 2x)}{\sqrt{4x^2 + x + 1} + 2x}$$
$$= \frac{(4x^2 + x + 1) - 4x^2}{\sqrt{4x^2 + x + 1} + 2x} = \frac{x + 1}{\sqrt{4x^2 + x + 1} + 2x}$$

分子分母同除以 $x$：
$$\frac{x + 1}{\sqrt{4x^2 + x + 1} + 2x} = \frac{1 + \frac{1}{x}}{\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}} + 2}$$

因此：
$$b = \lim_{x\to +\infty} \frac{1 + \frac{1}{x}}{\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}} + 2} = \frac{1 + 0}{\sqrt{4 + 0 + 0} + 2} = \frac{1}{2 + 2} = \frac{1}{4}$$

所以当 $x\to +\infty$ 时，斜渐近线为 $y = 2x + \frac{1}{4}$。

---
**(2) $x\to -\infty$ 方向**：

计算 $k$。当 $x < 0$ 时，$\sqrt{x^2} = |x| = -x$：
$$k = \lim_{x\to -\infty} \frac{\sqrt{4x^2 + x + 1}}{x} = \lim_{x\to -\infty} \frac{(-x)\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}}}{x} = \lim_{x\to -\infty} \left(-\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}}\right) = -2$$

计算 $b$：
$$b = \lim_{x\to -\infty} \bigl(\sqrt{4x^2 + x + 1} - (-2)x\bigr) = \lim_{x\to -\infty} \bigl(\sqrt{4x^2 + x + 1} + 2x\bigr)$$

有理化（注意这里不是减 $2x$ 而是加 $2x$，因此共轭表达式为 $\sqrt{4x^2+x+1} - 2x$）：
$$\sqrt{4x^2 + x + 1} + 2x = \frac{(\sqrt{4x^2 + x + 1} + 2x)(\sqrt{4x^2 + x + 1} - 2x)}{\sqrt{4x^2 + x + 1} - 2x}$$
$$= \frac{(4x^2 + x + 1) - 4x^2}{\sqrt{4x^2 + x + 1} - 2x} = \frac{x + 1}{\sqrt{4x^2 + x + 1} - 2x}$$

当 $x\to -\infty$ 时，分母：
$$\sqrt{4x^2 + x + 1} - 2x = (-x)\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}} - 2x = -x\left(\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}} + 2\right)$$

由于 $x\to -\infty$，$-x \to +\infty$，因此分母 $\to +\infty$。

分子分母同除以 $x$：
$$\frac{x + 1}{\sqrt{4x^2 + x + 1} - 2x} = \frac{1 + \frac{1}{x}}{\frac{\sqrt{4x^2 + x + 1}}{x} - 2}$$

当 $x\to -\infty$ 时：
$$\frac{\sqrt{4x^2 + x + 1}}{x} = -\sqrt{4 + \frac{1}{x} + \frac{1}{x^2}} \to -2$$

因此：
$$\frac{1}{\frac{\sqrt{4x^2 + x + 1}}{x} - 2} \to \frac{1}{-2 - 2} = -\frac{1}{4}$$

但分子还有 $1 + \frac{1}{x} \to 1$，所以：
$$b = \lim_{x\to -\infty} \frac{1 + \frac{1}{x}}{\frac{\sqrt{4x^2 + x + 1}}{x} - 2} = \frac{1}{-2 - 2} = -\frac{1}{4}$$

也可以从有理化后的形式直接判断：分母 $\sqrt{4x^2+x+1} - 2x \to +\infty$（因为两个正项相加），分子 $x+1 \to -\infty$，用抓大头方法：
$\frac{x+1}{\sqrt{4x^2+x+1} - 2x} \sim \frac{x}{(-x)(\sqrt{4}+2)} = \frac{1}{-4} = -\frac{1}{4}$，结果一致。

因此当 $x\to -\infty$ 时，斜渐近线为 $y = -2x - \frac{1}{4}$。

**答案**：
- $x\to +\infty$：斜渐近线 $y = 2x + \dfrac{1}{4}$
- $x\to -\infty$：斜渐近线 $y = -2x - \dfrac{1}{4}$

（无铅直渐近线，无水平渐近线）

**注**：$f(x) = \sqrt{4x^2 + x + 1}$ 既不是偶函数也不是奇函数，因此两个方向的渐近线不对称——不仅斜率互为相反数，截距也不相同。

</details>
