## 2. 背景与应用场景

在前序文件中，我们学习了函数极限的 $\varepsilon$-$\delta$ 定义，以及基于该定义的一系列性质（唯一性、局部有界性、保号性、四则运算法则、夹逼准则）。这些内容都建立在**极限已经存在**的前提之上。

但一个更根本的问题是：**我们如何判断一个函数的极限是否存在？**

这个问题远比"已知极限存在，求极限值"更为深刻。实际上，在两大类重要场景中，我们甚至不知道极限值是什么，但需要证明极限存在：

1. **仅知道函数有某种性质（如单调且有界），但不知道函数值趋于哪个具体数**。例如，在数值分析中，迭代序列 $x_{n+1} = f(x_n)$ 的收敛性判断往往依赖于单调有界性——我们可能不知道极限的具体数值，但能断定极限存在。

2. **仅知道函数值之间的相互接近程度，而不知道极限候选值**。例如，在物理中测量一个趋于稳定的量，我们只能观察到不同时刻的测量值彼此接近，但无法预知最终稳定值——这正是 Cauchy 准则的物理对应。

本文件将建立两个强大的极限存在性判定工具：

- **Cauchy 准则**：函数极限存在的充要条件是"自变量足够接近时，函数值任意接近"——它完全不依赖极限值。
- **单调有界函数极限定理**：单调且有界的函数，其单侧极限必然存在——它只依赖函数的单调性和有界性。

这两个定理构成了函数极限存在性理论的核心，并且与实数完备性（单调有界数列必收敛、Cauchy 数列必收敛）有着深刻的联系。

## 3. 核心概念与符号约定

### 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\lim_{x\to x_0} f(x)$ 存在（有限） | $f(x)$ 在 $x\to x_0$ 时趋于某个有限数 | $\lim_{x\to 2} (3x-1)=5$ |
| $\mathring{U}(x_0,\delta)$ | $x_0$ 的去心 $\delta$ 邻域 $\{x\mid 0<|x-x_0|<\delta\}$ | 用于 Cauchy 条件 |
| $|f(x')-f(x'')|<\varepsilon$ | $x',x''$ 两点函数值之差可任意小 | Cauchy 条件的核心 |
| $\sup S$ | 集合 $S$ 的上确界（最小上界） | $\sup\{x\mid 0<x<1\}=1$ |
| $\inf S$ | 集合 $S$ 的下确界（最大下界） | $\inf\{x\mid 0<x<1\}=0$ |
| $x\to b^-$ | $x$ 从左侧逼近 $b$（左极限） | $\lim_{x\to 1^-} x = 1$ |
| $x\to a^+$ | $x$ 从右侧逼近 $a$（右极限） | $\lim_{x\to 0^+} \frac{1}{x} = +\infty$ |

### 3.1 数列的单调有界定理（回顾）

本定理属于第二章的核心内容，此处显式陈述以便后续引用。

**定理 8.1（数列的单调有界定理）**：

(1) 若数列 $\{a_n\}$ 单调递增（$a_{n+1} \geq a_n$ 对一切 $n$ 成立）且有上界（存在 $M$ 使得 $a_n \leq M$ 对一切 $n$ 成立），则 $\{a_n\}$ 收敛，且
$$\lim_{n\to\infty} a_n = \sup\{a_n \mid n \in \mathbb{N}\}$$

(2) 若数列 $\{a_n\}$ 单调递减（$a_{n+1} \leq a_n$ 对一切 $n$ 成立）且有下界（存在 $m$ 使得 $a_n \geq m$ 对一切 $n$ 成立），则 $\{a_n\}$ 收敛，且
$$\lim_{n\to\infty} a_n = \inf\{a_n \mid n \in \mathbb{N}\}$$

**说明**：该定理是实数完备性的核心表达之一——它断言实数集中单调有界数列必然收敛。在有理数集 $\mathbb{Q}$ 中该结论不成立（例如单调递增的有理数列 $1, 1.4, 1.41, 1.414, \dots$ 趋于 $\sqrt{2} \notin \mathbb{Q}$），因此该定理从根本上刻画了实数与有理数的本质区别。

### 3.2 函数极限的 Cauchy 准则（新概念）

**定义 8.1（Cauchy 条件/函数极限的 Cauchy 准则）**：设函数 $f$ 在点 $x_0$ 的某个去心邻域内有定义。则极限 $\displaystyle\lim_{x\to x_0} f(x)$ 存在（为有限数）的**充要条件**是：
$$\forall\varepsilon>0,\ \exists\delta>0,\ \forall x',x''\ (0<|x'-x_0|<\delta\ \land\ 0<|x''-x_0|<\delta):\ |f(x')-f(x'')|<\varepsilon$$

**直观解释**：
- **必要性方向**（$\Rightarrow$）：如果 $f(x)$ 在 $x\to x_0$ 时趋于某个极限 $L$，那么当 $x'$ 和 $x''$ 都足够接近 $x_0$ 时，$f(x')$ 和 $f(x'')$ 都接近 $L$，因此它们互相也接近（三角形两边之差小于第三边）。
- **充分性方向**（$\Leftarrow$）：如果任意两个足够接近 $x_0$ 的点 $x',x''$ 的函数值之差可以任意小，说明函数值在 $x_0$ 附近"不乱跳"，从而必然趋于某个极限——这本质上是实数完备性的体现。

**与数列 Cauchy 收敛准则的类比**：

| 对比项 | 数列 Cauchy 收敛准则 | 函数极限 Cauchy 准则 |
|--------|---------------------|---------------------|
| 适用对象 | 数列 $\{a_n\}$ | 函数 $f(x)$ 在 $x\to x_0$ |
| 条件 | $|a_m - a_n| < \varepsilon$（$m,n > N$） | $|f(x')-f(x'')|<\varepsilon$（$x',x''$ 在去心 $\delta$ 邻域内） |
| 门槛参数 | $N$（正整数，依赖 $\varepsilon$） | $\delta$（正实数，依赖 $\varepsilon$） |
| 二者关系 | 函数极限的 Cauchy 准则可通过 Heine 定理与数列版本等价转换 | |

### 3.3 单调有界函数的极限定理

**定理 8.2（单调有界函数的极限定理）**：

(1) 设函数 $f$ 在区间 $(a,b)$ 上**单调递增**且**有上界**，则左极限 $\displaystyle\lim_{x\to b^-} f(x)$ 存在（为有限数），且
$$\lim_{x\to b^-} f(x) = \sup\{f(x) \mid x \in (a,b)\}$$

(2) 设函数 $f$ 在区间 $(a,b)$ 上**单调递增**且**有下界**，则右极限 $\displaystyle\lim_{x\to a^+} f(x)$ 存在（为有限数），且
$$\lim_{x\to a^+} f(x) = \inf\{f(x) \mid x \in (a,b)\}$$

**单调递减情形的对称处理**：

若 $f$ 在 $(a,b)$ 上单调递减且有下界，考虑 $-f$（单调递增且有上界），由定理 8.2(1) 得 $\lim_{x\to b^-} (-f(x)) = \sup(-f) = -\inf f$，因此 $\lim_{x\to b^-} f(x) = \inf f$ 存在。同理可得单调递减有上界时右端点左极限存在等其他对称情形。

为方便查阅，列出所有四种情形：

| 单调性 | 有界条件 | 存在结论 | 极限值 |
|--------|---------|---------|-------|
| 单调递增 | 有上界 | $\lim_{x\to b^-} f(x)$ 存在 | $\sup f$ |
| 单调递增 | 有下界 | $\lim_{x\to a^+} f(x)$ 存在 | $\inf f$ |
| 单调递减 | 有下界 | $\lim_{x\to b^-} f(x)$ 存在 | $\inf f$ |
| 单调递减 | 有上界 | $\lim_{x\to a^+} f(x)$ 存在 | $\sup f$ |

**重要推论**：单调函数在任意内点处的左右极限都存在（但可能不相等）。

这是因为对任意 $c \in (a,b)$，将 $f$ 分别限制在 $(a,c)$ 和 $(c,b)$ 上，由定理 8.2 即得 $\lim_{x\to c^-} f(x)$ 和 $\lim_{x\to c^+} f(x)$ 均存在。两者不一定相等——当它们不相等时，$c$ 是 $f$ 的跳跃间断点。因此**单调函数只可能有一类间断点：跳跃间断点**。

## 4. 原理与方法

### 4.1 Cauchy 准则的证明

#### 4.1.1 必要性（$\Rightarrow$）：极限存在 $\Rightarrow$ Cauchy 条件成立

**定理 8.3（Cauchy 准则的必要性）**：若 $\displaystyle\lim_{x\to x_0} f(x)$ 存在（为有限数），则对任意 $\varepsilon > 0$，存在 $\delta > 0$，使得对任意满足 $0<|x'-x_0|<\delta$ 和 $0<|x''-x_0|<\delta$ 的 $x',x''$，都有 $|f(x')-f(x'')|<\varepsilon$。

**证明**：

**第 1 步：利用极限定义获得 $\delta$**。
设 $\displaystyle\lim_{x\to x_0} f(x) = L$（$L$ 为有限数）。对任意给定的 $\varepsilon > 0$，取 $\frac{\varepsilon}{2} > 0$。

由极限的 $\varepsilon$-$\delta$ 定义（定义 1.1），存在 $\delta > 0$，使得当 $0 < |x - x_0| < \delta$ 时：
$$|f(x) - L| < \frac{\varepsilon}{2}$$

**第 2 步：对任意两点应用三角不等式**。
任取 $x', x''$ 满足 $0 < |x'-x_0| < \delta$ 且 $0 < |x''-x_0| < \delta$。则 $f(x')$ 和 $f(x'')$ 都满足与 $L$ 的估值关系。于是：

$$
\begin{aligned}
|f(x') - f(x'')| &= |(f(x') - L) + (L - f(x''))| \\
&\leq |f(x') - L| + |L - f(x'')| \quad \text{（三角不等式）} \\
&< \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon
\end{aligned}
$$

因此 Cauchy 条件成立。证毕。

**直观理解**：必要性方向的直觉非常简单——如果所有人都走向同一个集合点 $L$，那么任意两个人之间的距离必然可以任意小。三角不等式 $|f(x')-f(x'')| \leq |f(x')-L| + |f(x'')-L|$ 将"两点都接近 $L$"转化为"两点互相接近"。

#### 4.1.2 充分性（$\Leftarrow$）：Cauchy 条件成立 $\Rightarrow$ 极限存在

**定理 8.4（Cauchy 准则的充分性）**：设 $f$ 在 $x_0$ 的某个去心邻域内有定义。若 Cauchy 条件成立，则 $\displaystyle\lim_{x\to x_0} f(x)$ 存在（为有限数）。

**证明思路**：利用 Cauchy 条件构造数列极限，再借助 Heine 定理回到函数极限。这是 Heine 定理作为"函数极限与数列极限桥梁"的典型应用。

**第 1 步：任取一个趋近 $x_0$ 的数列，证明对应函数值数列为 Cauchy 数列**。

任取数列 $\{x_n\}$ 满足 $x_n \to x_0$ 且 $x_n \neq x_0$（对所有 $n$ 成立）。

对任意 $\varepsilon > 0$，由 Cauchy 条件，存在 $\delta > 0$，使得当 $0 < |x'-x_0| < \delta$ 且 $0 < |x''-x_0| < \delta$ 时，$|f(x')-f(x'')| < \varepsilon$。

由于 $x_n \to x_0$，存在 $N \in \mathbb{N}$，使得当 $n > N$ 时 $0 < |x_n - x_0| < \delta$。

于是对任意 $m,n > N$，$0 < |x_m - x_0| < \delta$ 且 $0 < |x_n - x_0| < \delta$ 同时成立，代入 Cauchy 条件（取 $x' = x_m$，$x'' = x_n$）得：
$$|f(x_m) - f(x_n)| < \varepsilon$$

因此 $\{f(x_n)\}$ 是 Cauchy 数列。

**第 2 步：Cauchy 数列必收敛（实数完备性）**。

由实数的完备性定理（Cauchy 收敛准则），$\mathbb{R}$ 中的任意 Cauchy 数列必收敛。因此 $\{f(x_n)\}$ 收敛，记
$$\lim_{n\to\infty} f(x_n) = L$$

这个极限 $L$ 目前依赖于数列 $\{x_n\}$ 的选取——我们需要证明 $L$ 与数列的具体选择无关。

**第 3 步：证明极限 $L$ 与数列选取无关**。

假设另有一数列 $\{y_n\}$ 满足 $y_n \to x_0$，$y_n \neq x_0$，且 $\lim_{n\to\infty} f(y_n) = L'$。

考虑混合数列 $\{z_n\}$：$z_{2k-1} = x_k$，$z_{2k} = y_k$（$k = 1,2,3,\dots$）。则 $z_n \to x_0$ 且 $z_n \neq x_0$。

由第 1 步的论证，$\{f(z_n)\}$ 收敛。其奇数项子列 $\{f(z_{2k-1})\} = \{f(x_k)\}$ 收敛到 $L$，偶数项子列 $\{f(z_{2k})\} = \{f(y_k)\}$ 收敛到 $L'$。但一个收敛数列的任何子列必须收敛到同一极限，因此 $L = L'$。

这说明：对任意满足 $x_n \to x_0$（$x_n \neq x_0$）的数列 $\{x_n\}$，$\{f(x_n)\}$ 都收敛到同一个极限 $L$。

**第 4 步：应用 Heine 定理回到函数极限**。

由 Heine 定理（定理 1.1），对任意 $x_n \to x_0$（$x_n \neq x_0$）都有 $\lim f(x_n) = L$ 的充要条件是 $\lim_{x\to x_0} f(x) = L$。因此：
$$\lim_{x\to x_0} f(x) = L$$

极限存在。证毕。

**充分性证明的核心要点**：
1. Cauchy 条件保证了任意"路径"上的函数值数列都是 Cauchy 数列
2. 实数完备性保证了 Cauchy 数列的收敛性
3. 混合数列论证保证了极限与路径无关
4. Heine 定理将数列极限的收敛性"提升"为函数极限的存在性

### 4.2 Cauchy 准则的否定形式及其应用

#### 4.2.1 否定形式

要证明 $\displaystyle\lim_{x\to x_0} f(x)$ 不存在（不以任何有限数为极限），只需证明 Cauchy 条件**不成立**：

$$\exists\varepsilon_0 > 0,\ \forall\delta > 0,\ \exists x',x''\ (0<|x'-x_0|<\delta\ \land\ 0<|x''-x_0|<\delta):\ |f(x')-f(x'')| \geq \varepsilon_0$$

**解读**：存在某个"顽固的" $\varepsilon_0 > 0$，无论 $\delta$ 取得多小，总能在 $x_0$ 的去心 $\delta$ 邻域内找到两个点 $x', x''$，使得它们的函数值之差至少为 $\varepsilon_0$。这意味着函数值在 $x_0$ 附近持续振荡，无法稳定下来。

#### 4.2.2 应用一：证明 $\displaystyle\lim_{x\to 0} \sin\frac{1}{x}$ 不存在

**例 1**：证明 $\displaystyle\lim_{x\to 0} \sin\frac{1}{x}$ 不存在（不以任何有限数为极限）。

**分析**：函数 $f(x) = \sin\frac{1}{x}$ 在 $x=0$ 附近无限振荡——当 $x$ 趋于 $0$ 时，$\frac{1}{x}$ 趋于无穷，$\sin\frac{1}{x}$ 在 $[-1,1]$ 之间来回振荡，不趋近任何固定值。Cauchy 准则的否定形式精确地捕捉了这一现象。

**证明**：

**第 1 步：选取 $\varepsilon_0$**。
取 $\varepsilon_0 = 1$。

**第 2 步：对任意 $\delta > 0$ 构造两点**。
对任意给定的 $\delta > 0$，选择正整数 $n$ 使得 $n > \dfrac{1}{2\pi\delta}$。构造：
$$x' = \frac{1}{2n\pi}, \qquad x'' = \frac{1}{2n\pi + \dfrac{\pi}{2}}$$

**第 3 步：验证两点都在去心 $\delta$ 邻域内**。
$$|x'| = \frac{1}{2n\pi} < \frac{1}{2\pi \cdot \frac{1}{2\pi\delta}} = \delta$$
$$|x''| = \frac{1}{2n\pi + \dfrac{\pi}{2}} < \frac{1}{2n\pi} < \delta$$

因此 $0 < |x'| < \delta$ 且 $0 < |x''| < \delta$。

**第 4 步：验证函数值差**。
$$
\begin{aligned}
\left|f(x') - f(x'')\right| &= \left|\sin\frac{1}{x'} - \sin\frac{1}{x''}\right| \\
&= \left|\sin(2n\pi) - \sin\!\left(2n\pi + \frac{\pi}{2}\right)\right| \\
&= |0 - 1| = 1 \geq \varepsilon_0
\end{aligned}
$$

**第 5 步：结论**。

存在 $\varepsilon_0 = 1$，使得对任意 $\delta > 0$，都能找到两点 $x', x''$ 满足 $|x'| < \delta$ 且 $|x''| < \delta$，但 $|f(x') - f(x'')| \geq \varepsilon_0$。

由 Cauchy 准则的否定形式，$\displaystyle\lim_{x\to 0} \sin\frac{1}{x}$ 不存在。证毕。

**注**：这个证明与 Heine 定理证法（文件 01 例 5 的思路）等价——在 Heine 定理框架下，我们构造两个趋于 $0$ 的不同数列使函数值分别趋于 $0$ 和 $1$；在 Cauchy 准则框架下，我们直接在 $x=0$ 附近找到函数值差为 $1$ 的两个点。两种方法从不同角度刻画了同一振荡现象，但 Cauchy 准则不需要构造数列，操作更为直接。

#### 4.2.3 应用二：用 Cauchy 准则证明局部有界性

我们已经知道如何用 $\varepsilon$-$\delta$ 定义证明极限存在 $\Rightarrow$ 局部有界（文件 02 定理 2.2）。现在我们展示用 Cauchy 准则的另一证法——**不依赖极限值 $L$ 的具体数值**。

**例 2**：若 $\displaystyle\lim_{x\to x_0} f(x)$ 存在（为有限数），证明 $f$ 在 $x_0$ 的某个去心邻域内有界。

**证明（用 Cauchy 准则）**：

**第 1 步：由极限存在得到 Cauchy 条件**。
由 $\displaystyle\lim_{x\to x_0} f(x)$ 存在，由 Cauchy 准则的必要性（定理 8.3），对 $\varepsilon = 1$，存在 $\delta_0 > 0$，使得对任意满足 $0 < |x'-x_0| < \delta_0$ 且 $0 < |x''-x_0| < \delta_0$ 的 $x', x''$，有：
$$|f(x') - f(x'')| < 1$$

**第 2 步：固定一个参考点**。
任取一个满足 $0 < |x_1 - x_0| < \delta_0$ 的点 $x_1$（这样的点存在，因为 $f$ 在 $x_0$ 的去心邻域内有定义）。

**第 3 步：对任意 $x$ 应用 Cauchy 条件并做三角不等式估计**。
对任意满足 $0 < |x - x_0| < \delta_0$ 的 $x$，令 $x' = x$，$x'' = x_1$，则：
$$|f(x) - f(x_1)| < 1$$

由三角不等式：
$$|f(x)| = |(f(x) - f(x_1)) + f(x_1)| \leq |f(x) - f(x_1)| + |f(x_1)| < 1 + |f(x_1)|$$

**第 4 步：得到全局上界**。
取 $M = 1 + |f(x_1)|$。则当 $0 < |x - x_0| < \delta_0$ 时，$|f(x)| \leq M$。即 $f$ 在 $x_0$ 的去心 $\delta_0$ 邻域内有界。证毕。

**两种证明方法的对比**：

| 对比项 | $\varepsilon$-$\delta$ 直接证明（定理 2.2） | Cauchy 准则证明（例 2） |
|--------|------------------------------------------|----------------------|
| 依赖的极限知识 | 已知极限值 $L$ | 只知极限存在，不知 $L$ |
| 有界常数 $M$ | $M = 1 + |L|$ | $M = 1 + |f(x_1)|$ |
| 参考点 | 极限值 $L$ | 某个固定的点 $x_1$ |
| 适用场景 | 极限值已知时更简洁 | 只需存在性时更通用 |

**要点**：Cauchy 准则版本的证明不依赖极限值 $L$ 的具体数值——它只依赖"极限存在"这一事实本身。这在实际应用中非常有用：当我们只知道极限存在但暂未求出极限值时，仍能推导出函数的局部有界性。

### 4.3 单调有界函数极限定理的证明

#### 4.3.1 单调递增有上界的情形

**定理 8.2(1) 证明**：设 $f$ 在 $(a,b)$ 上单调递增且有上界，证明 $\displaystyle\lim_{x\to b^-} f(x)$ 存在。

**第 1 步：引入上确界**。

由于 $f$ 在 $(a,b)$ 上有上界，值域 $f((a,b)) = \{f(x) \mid x \in (a,b)\}$ 是 $\mathbb{R}$ 的非空有上界子集。由实数的**确界原理**，该集合有上确界。令：
$$L = \sup\{f(x) \mid x \in (a,b)\}$$

$L$ 是一个有限实数（因为上界存在，确界原理保证 $\sup$ 存在且有限）。

**第 2 步：利用上确界的定义——存在逼近点**。

由上确界的定义：
- $L$ 是上界：$f(x) \leq L$ 对一切 $x \in (a,b)$ 成立。
- 对任意给定的 $\varepsilon > 0$，$L - \varepsilon$ **不是**上界，因此存在 $x_0 \in (a,b)$，使得：
$$f(x_0) > L - \varepsilon$$

这一步至关重要——它保证了我们可以找到一点 $x_0$，其函数值任意接近上确界 $L$。

**第 3 步：利用单调性将"存在一点"推广为"自此之后所有点"**。

由于 $f$ 在 $(a,b)$ 上单调递增，对任意满足 $x_0 < x < b$ 的 $x$，有：
$$f(x_0) \leq f(x)$$

结合第 2 步的结果：
$$L - \varepsilon < f(x_0) \leq f(x) \leq L$$

因此当 $x_0 < x < b$ 时：
$$L - \varepsilon < f(x) < L + \varepsilon$$

即：
$$|f(x) - L| < \varepsilon$$

**第 4 步：用 $\varepsilon$-$\delta$ 语言表述**。

取 $\delta = b - x_0 > 0$（因为 $x_0 < b$）。则当 $b - \delta < x < b$（即 $0 < b - x < \delta$）时，由第 3 步的推导：
$$|f(x) - L| < \varepsilon$$

这正是左极限 $\displaystyle\lim_{x\to b^-} f(x) = L$ 的 $\varepsilon$-$\delta$ 定义（文件 01 定义 1.3 的对称形式——左极限中 $x$ 从左侧趋近 $b$）。

回忆左极限的定义：$\displaystyle\lim_{x\to b^-} f(x) = L \iff \forall\varepsilon>0,\ \exists\delta>0,\ \forall x\,(b-\delta < x < b):\ |f(x)-L|<\varepsilon$。

因此 $\displaystyle\lim_{x\to b^-} f(x) = L$ 存在。证毕。

**证明的关键要点**：
1. **确界原理**提供了极限候选值 $L$ 的存在性——这是实数完备性的核心体现。
2. **上确界的定义**给出 $L - \varepsilon$ "不是上界"的结论——这意味着存在点 $x_0$ 使 $f(x_0) > L - \varepsilon$。
3. **单调性**将"存在一个点满足"推广为"自此以后所有点都满足"——这是从"存在性"到"全局性"的关键一步。
4. **$\delta = b - x_0$ 的构造**——这个 $\delta$ 就是 $b$ 到那个"阈值点" $x_0$ 的距离。

#### 4.3.2 单调递增有下界的情形（对称版本）

**定理 8.2(2) 证明**：设 $f$ 在 $(a,b)$ 上单调递增且有下界，证明 $\displaystyle\lim_{x\to a^+} f(x)$ 存在。

**第 1 步：引入下确界**。

由 $f$ 有下界，值域有下界。由确界原理，存在下确界：
$$m = \inf\{f(x) \mid x \in (a,b)\}$$

$m$ 是有限实数。

**第 2 步：利用下确界的定义**。

- $m$ 是下界：$f(x) \geq m$ 对一切 $x \in (a,b)$ 成立。
- 对任意 $\varepsilon > 0$，$m + \varepsilon$ 不是下界，因此存在 $x_0 \in (a,b)$，使得 $f(x_0) < m + \varepsilon$。

**第 3 步：利用单调性**。

由单调递增性，对任意 $a < x < x_0$ 有 $f(x) \leq f(x_0)$。结合：
$$m \leq f(x) \leq f(x_0) < m + \varepsilon$$

因此当 $a < x < x_0$ 时：
$$m - \varepsilon < f(x) < m + \varepsilon$$
即 $|f(x) - m| < \varepsilon$。

**第 4 步：$\varepsilon$-$\delta$ 表述**。

取 $\delta = x_0 - a > 0$。则当 $a < x < a + \delta$（即 $0 < x - a < \delta$）时，有 $|f(x) - m| < \varepsilon$。

这正是右极限的定义，故 $\displaystyle\lim_{x\to a^+} f(x) = m$ 存在。证毕。

#### 4.3.3 单调递减情形的处理

对于单调递减函数，有两种等价的处理方式：

**方法一（转化为递增情形）**：若 $f$ 单调递减，则 $-f$ 单调递增。对 $-f$ 应用定理 8.2，再将结论翻译回 $f$。

例如，若 $f$ 在 $(a,b)$ 上单调递减且有下界，则 $-f$ 在 $(a,b)$ 上单调递增且有上界（因为 $f \geq m \Rightarrow -f \leq -m$）。由定理 8.2(1)：
$$\lim_{x\to b^-} (-f(x)) = \sup\{-f(x) \mid x \in (a,b)\} = -\inf\{f(x) \mid x \in (a,b)\}$$

因此 $\displaystyle\lim_{x\to b^-} f(x) = \inf_{x\in(a,b)} f(x)$ 存在。

**方法二（直接对称证明）**：完全仿照定理 8.2(1) 的证明步骤，将"上界"换为"下界"、"上确界"换为"下确界"、"$f(x_0) > L - \varepsilon$"换为"$f(x_0) < m + \varepsilon$"，方向反向即可。

### 4.4 Heine 定理替代证明（为单调有界函数极限定理提供第二种思路）

现在，我们利用 Heine 定理（文件 01 定理 1.1）和数列的单调有界定理（定理 8.1）给出定理 8.2(1) 的另一种证明。这种证法将函数极限问题转化为数列极限问题，展现了 Heine 定理作为桥梁的关键作用。

**定理 8.2(1) 的 Heine 定理替代证明**：

设 $f$ 在 $(a,b)$ 上单调递增且有上界，$L = \sup\{f(x) \mid x \in (a,b)\}$。证明 $\displaystyle\lim_{x\to b^-} f(x) = L$。

**第 1 步：任取一个趋近 $b$ 的数列**。

任取数列 $\{x_n\} \subseteq (a,b)$ 满足 $x_n \to b^-$（即 $x_n$ 趋于 $b$ 且 $x_n < b$ 对所有 $n$ 成立）。不妨设 $\{x_n\}$ 单调递增（因为如果原数列不单调，可以取其子列，或直接利用 $x_n \to b^-$ 可构造递增子列的性质）。

**第 2 步：对应函数值数列的单调性和有界性**。

由于 $f$ 在 $(a,b)$ 上单调递增，且 $x_n$ 递增：
$$x_n \leq x_{n+1} \quad\Rightarrow\quad f(x_n) \leq f(x_{n+1})$$

因此 $\{f(x_n)\}$ 是单调递增数列。

又因为 $f$ 在 $(a,b)$ 上有上界，对所有 $n$ 有：
$$f(x_n) \leq \sup_{x\in(a,b)} f(x) = L < +\infty$$

因此 $\{f(x_n)\}$ 有上界。

**第 3 步：应用数列的单调有界定理**。

由数列的单调有界定理（定理 8.1(1)），单调递增有上界的数列 $\{f(x_n)\}$ 收敛，且：
$$\lim_{n\to\infty} f(x_n) = \sup\{f(x_n) \mid n \in \mathbb{N}\}$$

记该极限为 $\ell$。

**第 4 步：证明 $\ell = L$（极限与序列无关）**。

我们需要证明 $\ell$ 不仅等于 $\{f(x_n)\}$ 的上确界，还等于全局上确界 $L$。

- **一方面**：$f(x_n) \leq L$ 对一切 $n$ 成立。由数列极限的保序性（推论 2.1 的数列版本），$\ell = \lim f(x_n) \leq L$。

- **另一方面**：对任意 $\varepsilon > 0$，由上确界的定义，存在 $x_0 \in (a,b)$ 使得 $f(x_0) > L - \varepsilon$。由于 $x_n \to b^-$，当 $n$ 充分大时 $x_n > x_0$。由单调性，$f(x_n) \geq f(x_0) > L - \varepsilon$。由极限的保序性，$\ell = \lim f(x_n) \geq L - \varepsilon$。由 $\varepsilon$ 的任意性，$\ell \geq L$。

综合得 $\ell = L$。

由于 $\ell = L$ 不依赖于数列 $\{x_n\}$ 的具体选取，这说明：**对任意满足 $x_n \to b^-$ 的数列 $\{x_n\}$，都有 $\lim f(x_n) = L$**。

**第 5 步：应用 Heine 定理**。

由 Heine 定理（定理 1.1），若对任意 $x_n \to b^-$（$x_n \neq b$）都有 $\lim f(x_n) = L$，则：
$$\lim_{x\to b^-} f(x) = L$$

证毕。

**两种证明方法的对比**：

| 对比项 | $\varepsilon$-$\delta$ 直接证明 | Heine 定理替代证明 |
|--------|-------------------------------|-------------------|
| 核心工具 | 确界原理 + 上确界定义 + $\varepsilon$-$\delta$ 语言 | Heine 定理 + 数列单调有界定理 |
| 关键步骤 | 由 $L-\varepsilon$ 不是上界得 $\exists x_0$，单调性推广 | $\{f(x_n)\}$ 单调有界 $\Rightarrow$ 收敛，证极限与序列无关 |
| 依赖的实数完备性表达 | 确界原理 | 数列单调有界定理 |
| 证明长度 | 较短，$\delta$ 直接构造 | 稍长，需混合数列论证 |
| 优点 | 直接、简洁 | 将函数问题化为数列问题，思路更直观 |

## 5. 例题

### 例 3：用 Cauchy 准则证明极限不存在

证明 $\displaystyle\lim_{x\to 0} \cos\frac{1}{x}$ 不存在。

**解**：

**第 1 步：分析**。与 $\sin\frac{1}{x}$ 类似，$\cos\frac{1}{x}$ 在 $x=0$ 附近无限振荡。

**第 2 步：选取 $\varepsilon_0$**。取 $\varepsilon_0 = 1$。

**第 3 步：对任意 $\delta > 0$ 构造两点**。对任意 $\delta > 0$，取正整数 $n$ 使得 $n > \dfrac{1}{2\pi\delta}$。令：
$$x' = \frac{1}{2n\pi}, \qquad x'' = \frac{1}{2n\pi + \pi}$$

**第 4 步：验证距离条件**。
$$|x'| = \frac{1}{2n\pi} < \delta, \qquad |x''| = \frac{1}{2n\pi + \pi} < \frac{1}{2n\pi} < \delta$$

**第 5 步：验证函数值差**。
$$\left|\cos\frac{1}{x'} - \cos\frac{1}{x''}\right| = |\cos(2n\pi) - \cos(2n\pi + \pi)| = |1 - (-1)| = 2 \geq 1 = \varepsilon_0$$

由 Cauchy 准则的否定形式，$\displaystyle\lim_{x\to 0} \cos\frac{1}{x}$ 不存在。证毕。

**注**：实际上我们可以取 $\varepsilon_0 = 1$ 即足够，因为 $|1 - (-1)| = 2 > 1$。这与 $\sin\frac{1}{x}$ 的证明结构完全一致，唯一差别在于选点的相位不同（$\pi$ 而非 $\pi/2$）。

### 例 4：单调有界函数端点的极限存在性

设 $f(x) = 1 - e^{-x}$ 在 $(0, \infty)$ 上。
(1) 判断 $f$ 在 $(0, \infty)$ 上的单调性和有界性。
(2) 求 $\displaystyle\lim_{x\to \infty} f(x)$ 和 $\displaystyle\lim_{x\to 0^+} f(x)$。

**解**：

**(1) 单调性和有界性分析**：

$f(x) = 1 - e^{-x}$。当 $x$ 增大时，$e^{-x}$ 递减，因此 $1 - e^{-x}$ 递增——$f$ 在 $(0, \infty)$ 上严格单调递增。

$e^{-x} > 0$ 对所有 $x$ 成立，因此 $f(x) = 1 - e^{-x} < 1$。又由于 $e^{-x} \to 0$ 当 $x \to \infty$，$\sup f = 1$。同时 $x \to 0^+$ 时 $e^{-x} \to 1$，$f(x) \to 0$，且 $f(x) > 0$ 对 $x > 0$，故 $\inf f = 0$。

$f$ 在 $(0, \infty)$ 上单调递增，有上界 $1$，有下界 $0$。

**(2) 极限计算**：

**求 $\displaystyle\lim_{x\to \infty} f(x)$**：

$f$ 在 $(0, \infty)$ 上单调递增有上界。严格来说，区间 $(0, \infty)$ 不是 $(a,b)$ 的形式，但 $x \to \infty$ 的极限定义与 $x \to b^-$ 的极限定义在逻辑结构上是平行的（把 "趋于无穷" 视为 "端点" 的特例）。由定理 8.2(1) 的思想：
$$\lim_{x\to \infty} f(x) = \sup_{x\in(0,\infty)} f(x) = 1$$

直接验证：$\displaystyle\lim_{x\to \infty} (1 - e^{-x}) = 1 - \lim_{x\to \infty} e^{-x} = 1 - 0 = 1$。

**求 $\displaystyle\lim_{x\to 0^+} f(x)$**：

$f$ 在 $(0, \infty)$ 上单调递增有下界。由定理 8.2(2)：
$$\lim_{x\to 0^+} f(x) = \inf_{x\in(0,\infty)} f(x) = 0$$

直接验证：$\displaystyle\lim_{x\to 0^+} (1 - e^{-x}) = 1 - \lim_{x\to 0^+} e^{-x} = 1 - 1 = 0$。

**验证**：$f$ 可以延拓为 $[0, \infty)$ 上的连续函数：
$$F(x) = \begin{cases}
0, & x = 0,\\
1 - e^{-x}, & x > 0.
\end{cases}$$
在 $x=0$ 处 $F(0) = \lim_{x\to 0^+} f(x) = 0$，因此 $F$ 在 $x=0$ 处右连续。

### 例 5：单调函数的延拓与间断点分析

设 $f(x) = \begin{cases} x, & 0 < x < \dfrac{1}{2}, \\[8pt] 2x, & \dfrac{1}{2} \leq x < 1. \end{cases}$

(1) 验证 $f$ 在 $(0,1)$ 上单调递增且有界。
(2) 求 $\displaystyle\lim_{x\to 0^+} f(x)$ 和 $\displaystyle\lim_{x\to 1^-} f(x)$。
(3) 定义 $F$ 为 $f$ 在 $[0,1]$ 上的延拓，使 $F$ 在端点处单侧连续。$F$ 在 $[0,1]$ 上是否连续？

**解**：

**(1) 单调性和有界性**：

对 $x_1 < x_2$，分情况讨论：
- 若 $x_1, x_2 < 1/2$：$f(x_1) = x_1 < x_2 = f(x_2)$。
- 若 $x_1, x_2 \geq 1/2$：$f(x_1) = 2x_1 < 2x_2 = f(x_2)$。
- 若 $x_1 < 1/2 \leq x_2$：$f(x_1) = x_1 < 1/2 \leq 2x_2 = f(x_2)$（因为 $x_1 < 1/2$ 且 $x_2 \geq 1/2$，$2x_2 \geq 1$，而 $x_1 < 1/2$）。

因此 $f$ 在 $(0,1)$ 上严格单调递增。

$f$ 在 $(0,1)$ 上有界：$0 < f(x) < 2$（因为当 $x < 1/2$ 时 $f(x) = x < 1/2$，当 $x \geq 1/2$ 时 $f(x) = 2x < 2$）。

**(2) 端点极限**：

由定理 8.2(2)（单调递增有下界）：
$$\lim_{x\to 0^+} f(x) = \inf_{x\in(0,1)} f(x) = \inf\{x \mid 0 < x < 1/2\} \cup \{2x \mid 1/2 \leq x < 1\} = 0$$

由定理 8.2(1)（单调递增有上界）：
$$\lim_{x\to 1^-} f(x) = \sup_{x\in(0,1)} f(x) = \max\left\{\sup_{x\in(0,1/2)} x,\ \sup_{x\in[1/2,1)} 2x\right\} = \max\left\{\frac{1}{2},\ 2\right\} = 2$$

注意：由于 $\sup_{x\in[1/2,1)} 2x = 2$，而 $2$ 在 $f$ 的值域内取不到（$x<1$ 时 $2x<2$），所以 $L=2$ 是极限值但不是函数值。

**(3) 延拓与连续性**：

定义 $[0,1]$ 上的函数 $F$：
$$F(x) = \begin{cases}
0, & x = 0,\\
x, & 0 < x < \dfrac{1}{2},\\
2x, & \dfrac{1}{2} \leq x < 1,\\
2, & x = 1.
\end{cases}$$

**端点连续性验证**：
- $x=0$ 处：$F(0)=0=\lim_{x\to 0^+} f(x)$，右连续。
- $x=1$ 处：$F(1)=2=\lim_{x\to 1^-} f(x)$，左连续。

**内部连续性分析**：
$F$ 在 $(0,1/2)$ 和 $(1/2,1)$ 上是多项式函数（$x$ 和 $2x$），因此在这些开区间内连续。

但在 $x = 1/2$ 处：
- 左极限：$\displaystyle\lim_{x\to \frac{1}{2}^-} F(x) = \lim_{x\to \frac{1}{2}^-} x = \frac{1}{2}$
- 右极限：$\displaystyle\lim_{x\to \frac{1}{2}^+} F(x) = \lim_{x\to \frac{1}{2}^+} 2x = 1$
- $F(1/2) = 2 \cdot \frac{1}{2} = 1$（分段定义中的第二种情况）

左右极限不相等（$1/2 \neq 1$），因此 $F$ 在 $x=1/2$ 处不连续——存在跳跃间断点。

**结论**：$F$ 在端点 $0$ 和 $1$ 处单侧连续（由延拓构造保证），但在内部点 $x=1/2$ 处不连续。单调函数的端点延拓只保证端点处的连续性，无法消除内部已有的跳跃间断点。

**扩展思考**：单调函数的间断点全部是跳跃间断点，且最多只有可数多个跳跃——这是因为每个跳跃对应一个"缺口"（左极限与右极限之间的区间），这些缺口互不相交，而实数轴上的互不相交的开区间族至多可数。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| Cauchy 准则的充分性方向是"显然"的——如果两点函数值之差可以任意小，函数当然趋于某个值 | 充分性方向依赖于实数完备性（$\mathbb{R}$ 中 Cauchy 数列必收敛）。在有理数集 $\mathbb{Q}$ 中，Cauchy 条件成立但极限可能不存在（如 $\sqrt{2}$ 的十进制近似数列）。因此充分性方向是实数完备性的深刻结论，需要证明 |
| Cauchy 准则可以写成 $\forall\varepsilon>0\ \exists\delta>0\ \forall x(0<|x-x_0|<\delta):\ |f(x)-f(x_0)|<\varepsilon$ | 这是错误的。Cauchy 条件中比较的是两个**自由变量** $x'$ 和 $x''$ 的函数值，而不是与 $x_0$ 处的函数值比较。$f$ 在 $x_0$ 处可能没有定义，$|f(x')-f(x'')|<\varepsilon$ 中的两个点都取自去心邻域 |
| 单调有界函数极限定理的证明中，取 $\delta = b - x_0$ 是一个"技巧"，需要特殊记忆 | $\delta = b - x_0$ 不是技巧而是自然构造：$x_0$ 是那个"阈值点"（当 $x > x_0$ 时函数值进入 $(L-\varepsilon, L+\varepsilon]$），$\delta$ 就是 $b$ 到 $x_0$ 的距离——从 $x_0$ 到 $b$ 的所有 $x$ 都满足要求 |
| 单调递减情形的结论"自动成立"，不需要单独处理 | 单调递减情形可以通过考虑 $-f$ 转化为单调递增情形来证明，也可以仿照递增情形做对称证明。两种方式都需要明确的论证步骤，不能"想当然"认为成立 |
| Heine 定理的替代证明中，数列 $\{f(x_n)\}$ 的极限 $\ell$ 一定等于 $\sup f$ | 这是需要证明的。$\ell \leq L$ 由保序性直接得到；$\ell \geq L$ 需要利用上确界的定义（对任意 $\varepsilon>0$ 存在 $x_0$ 使 $f(x_0)>L-\varepsilon$）和 $x_n$ 最终超过 $x_0$ 的事实。这两个方向缺一不可 |
| "极限存在"和"Cauchy 条件成立"是两个不同的概念，需要分别记忆 | 两者完全等价（Cauchy 准则正是这个结论）。极限存在 $\Leftrightarrow$ Cauchy 条件成立。Cauchy 条件的优势在于它不涉及极限值 $L$，因此在只需证明存在性而不知极限值时可发挥作用 |
| 单调有界函数在端点处的极限就是端点处的函数值 | 单调有界函数极限定理讨论的是**开区间**上的函数，端点处可能没有定义。即使有定义，极限值也可能不等于函数值（例如 $f$ 在端点不连续时）。定理只保证单侧极限存在，不保证等于端点函数值 |

## 练习题

### 基础巩固

**1.** 用 Cauchy 准则判断下列说法是否正确（正确请简要说明理由，错误请举出反例）：

(1) 若 $\displaystyle\lim_{x\to 0} f(x)$ 存在，则对任意 $\varepsilon > 0$，存在 $\delta > 0$，使得当 $0 < |x| < \delta$ 时 $|f(x) - f(2x)| < \varepsilon$。

(2) 若对任意 $\varepsilon > 0$，存在 $\delta > 0$，使得当 $0 < |x| < \delta$ 时 $|f(x) - f(2x)| < \varepsilon$，则 $\displaystyle\lim_{x\to 0} f(x)$ 存在。

<details><summary>参考答案</summary>

**(1) 正确**。

因为 $\displaystyle\lim_{x\to 0} f(x) = L$ 存在，由 Cauchy 准则的必要性，对任意 $\varepsilon > 0$，存在 $\delta_0 > 0$，使得当 $0 < |x'| < \delta_0$ 且 $0 < |x''| < \delta_0$ 时，$|f(x') - f(x'')| < \varepsilon$。

取 $\delta = \dfrac{\delta_0}{2}$。当 $0 < |x| < \delta$ 时，$0 < |x| < \delta_0/2 < \delta_0$，且 $0 < |2x| = 2|x| < \delta_0$。令 $x' = x$，$x'' = 2x$，代入 Cauchy 条件得：
$$|f(x) - f(2x)| < \varepsilon$$
因此结论成立。

---

**(2) 错误**。

反例：构造 $f(x) = \begin{cases} 1, & x > 0, \\ 0, & x < 0. \end{cases}$（$f(0)$ 可以任意定义）。

对任意 $\varepsilon > 0$，取 $\delta = 1$。当 $0 < |x| < 1$ 时：
- 若 $x > 0$，则 $2x > 0$，$f(x) = f(2x) = 1$，$|f(x) - f(2x)| = 0 < \varepsilon$。
- 若 $x < 0$，则 $2x < 0$，$f(x) = f(2x) = 0$，$|f(x) - f(2x)| = 0 < \varepsilon$。

条件成立，但 $\displaystyle\lim_{x\to 0} f(x)$ 不存在（左极限 $0$，右极限 $1$，不相等）。

**说明**：条件 (2) 只检查了 $x$ 和 $2x$ 这两类特殊点的关系，没有覆盖去心邻域中所有点对。Cauchy 准则要求对**任意** $x', x''$ 都成立，这是一个更强的条件。

</details>

**2.** 利用单调有界函数的极限定理，判断下列极限是否存在（存在则给出极限值，不存在说明理由）：

(1) $\displaystyle\lim_{x\to 0^+} \left(x\sin\frac{1}{x}\right)$

(2) $\displaystyle\lim_{x\to 0^+} \left(\frac{1}{x}\sin\frac{1}{x}\right)$

(3) $\displaystyle\lim_{x\to 1^-} \arctan x$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\lim_{x\to 0^+} \left(x\sin\frac{1}{x}\right)$：

$f(x) = x\sin\frac{1}{x}$ 在 $(0, \delta)$ 上不是单调函数（因为 $\sin\frac{1}{x}$ 在 $x$ 靠近 $0$ 时振荡），不能直接应用定理 8.2。需要用其他方法（如夹逼准则或无穷小量乘以有界函数）。

但实际上该极限存在且为 $0$（文件 04 例 3）：$|x\sin\frac{1}{x}| \leq |x|$，由夹逼准则得极限为 $0$。

本题说明：单调有界函数极限定理只是极限存在的**充分条件**而非必要条件——极限存在不需要单调性。

---

**(2)** $\displaystyle\lim_{x\to 0^+} \left(\frac{1}{x}\sin\frac{1}{x}\right)$：

$f(x) = \frac{1}{x}\sin\frac{1}{x}$ 在 $x\to 0^+$ 时无界且振荡，极限不存在。

可用 Cauchy 准则否定形式证明：取 $x' = \frac{1}{2n\pi}$，$x'' = \frac{1}{2n\pi + \pi/2}$，则 $|f(x') - f(x'')| = 2n\pi(0) - (2n\pi + \pi/2)(1) = 2n\pi + \pi/2 \to \infty$，对任意 $\delta$ 和足够大的 $n$ 该差值可以任意大，因此 Cauchy 条件不成立。

---

**(3)** $\displaystyle\lim_{x\to 1^-} \arctan x$：

$f(x) = \arctan x$ 在 $(0,1)$ 上严格单调递增（因为 $\arctan x$ 的导数 $\frac{1}{1+x^2} > 0$），且有上界 $\arctan 1 = \pi/4$。由定理 8.2(1)：
$$\lim_{x\to 1^-} \arctan x = \sup_{x\in(0,1)} \arctan x = \arctan 1 = \frac{\pi}{4}$$

也可直接由反三角函数连续性得极限等于 $\arctan 1$。

</details>

### 迁移应用

**3.** 设 $f$ 在 $(0,1)$ 上单调递增且有界。定义 $g(x) = f(x) + \frac{1}{x}$。

(1) $g$ 在 $(0,1)$ 上是否单调？是否有界？
(2) 利用定理 8.2 和已知结论判断 $\displaystyle\lim_{x\to 0^+} g(x)$ 是否存在。

<details><summary>参考答案</summary>

**(1) 单调性和有界性分析**：

$f$ 单调递增，$\frac{1}{x}$ 在 $(0,1)$ 上严格单调递减。两个单调性相反的函数之和，单调性不确定。

具体地，取 $f(x) = -x$（单调递减）——实际上题设 $f$ 递增，所以 $g$ 是递增函数加递减函数，整体不一定单调。

例如，$f(x) = 0$（常函数，既是递增也是递减）时，$g(x) = \frac{1}{x}$ 单调递减。
$f(x) = 2x$（递增）时，$g(x) = 2x + \frac{1}{x}$，其导数 $2 - \frac{1}{x^2}$，当 $x$ 很小时为负（递减），当 $x$ 足够大时为正（递增），因此不单调。

$g$ 在 $(0,1)$ 上无界（因为 $\frac{1}{x} \to +\infty$ 当 $x \to 0^+$）。

**(2) 极限存在性**：

$\displaystyle\lim_{x\to 0^+} g(x)$ 不存在（为 $+\infty$ 而非有限数）。

因为 $\frac{1}{x} \to +\infty$ 而 $f(x)$ 有界（$f$ 在 $(0,1)$ 上有界），所以 $g(x) \to +\infty$。

严格地说，对任意 $M > 0$，由于 $\lim_{x\to 0^+} \frac{1}{x} = +\infty$，存在 $\delta > 0$ 使当 $0 < x < \delta$ 时 $\frac{1}{x} > M + \sup|f|$，因此 $|g(x)| \geq \frac{1}{x} - |f(x)| > M$，故 $g(x) \to +\infty$ 发散到无穷，不是有限极限。

**注**：本题说明定理 8.2 要求"有上界"（或"有下界"）的条件是关键的——缺少有界性时，即使单调性存在，极限也可能不存在（或为无穷大）。

</details>

**4.** （综合）设 $f$ 在 $(0,1)$ 上单调递增且有界。定义 $F$ 为 $f$ 在 $[0,1]$ 上的端点延拓：
$$F(x) = \begin{cases}
\lim_{t\to 0^+} f(t), & x = 0,\\
f(x), & 0 < x < 1,\\
\lim_{t\to 1^-} f(t), & x = 1.
\end{cases}$$

(1) 证明 $F$ 在 $[0,1]$ 上单调递增。
(2) 证明 $F$ 在 $x=0$ 和 $x=1$ 处单侧连续。
(3) 构造一个具体的 $f$，使得 $F$ 在 $[0,1]$ 上有且仅有 $2$ 个跳跃间断点。

<details><summary>参考答案</summary>

**(1) 证明 $F$ 在 $[0,1]$ 上单调递增**：

记 $L_0 = \lim_{x\to 0^+} f(x)$，$L_1 = \lim_{x\to 1^-} f(x)$。

需证：对任意 $x_1 < x_2$（$x_1, x_2 \in [0,1]$），有 $F(x_1) \leq F(x_2)$。

分情况讨论：

- 若 $0 = x_1 < x_2 < 1$：$F(x_1) = L_0$，$F(x_2) = f(x_2)$。由 $L_0$ 是 $f$ 在 $x\to 0^+$ 时的极限，由于 $f$ 递增，有 $L_0 = \inf_{t\in(0,1)} f(t) \leq f(x_2)$。
- 若 $0 < x_1 < x_2 < 1$：$F(x_1) = f(x_1)$，$F(x_2) = f(x_2)$，由 $f$ 在 $(0,1)$ 上递增得 $f(x_1) \leq f(x_2)$。
- 若 $0 < x_1 < x_2 = 1$：$F(x_1) = f(x_1)$，$F(x_2) = L_1$。由 $L_1 = \sup_{t\in(0,1)} f(t) \geq f(x_1)$。
- 若 $x_1 = 0$，$x_2 = 1$：$L_0 \leq L_1$（由单调性，$L_0 = \lim_{x\to 0^+} f(x) \leq \lim_{x\to 1^-} f(x) = L_1$）。

因此 $F$ 在 $[0,1]$ 上单调递增。证毕。

**(2) 证明端点单侧连续性**：

在 $x=0$ 处：$F(0) = L_0 = \lim_{x\to 0^+} f(x)$。对任意 $x\in(0,1)$，$F(x) = f(x)$。因此 $\lim_{x\to 0^+} F(x) = \lim_{x\to 0^+} f(x) = L_0 = F(0)$，$F$ 在 $x=0$ 处右连续。

在 $x=1$ 处：$F(1) = L_1 = \lim_{x\to 1^-} f(x)$。因此 $\lim_{x\to 1^-} F(x) = \lim_{x\to 1^-} f(x) = L_1 = F(1)$，$F$ 在 $x=1$ 处左连续。

**(3) 构造恰好有 $2$ 个跳跃间断点的 $f$**：

定义 $f$ 在 $(0,1)$ 上：
$$f(x) = \begin{cases}
1, & 0 < x < \dfrac{1}{3},\\[8pt]
2, & \dfrac{1}{3} \leq x < \dfrac{2}{3},\\[8pt]
3, & \dfrac{2}{3} \leq x < 1.
\end{cases}$$

验证：
- $f$ 在 $(0,1)$ 上单调递增（取值 $1 \to 2 \to 3$，分段常数）。
- $f$ 有界（$1 \leq f(x) \leq 3$）。

端点极限：
$$L_0 = \lim_{x\to 0^+} f(x) = 1,\quad L_1 = \lim_{x\to 1^-} f(x) = 3$$

延拓后的 $F$：
$$F(x) = \begin{cases}
1, & x = 0,\\
1, & 0 < x < \dfrac{1}{3},\\
2, & \dfrac{1}{3} \leq x < \dfrac{2}{3},\\
3, & \dfrac{2}{3} \leq x < 1,\\
3, & x = 1.
\end{cases}$$

$F$ 在 $[0,1]$ 上的间断点分析：
- $x = \frac{1}{3}$ 处：左极限 $= 1$，右极限 $= 2$，$F(\frac{1}{3}) = 2$——跳跃间断点。
- $x = \frac{2}{3}$ 处：左极限 $= 2$，右极限 $= 3$，$F(\frac{2}{3}) = 3$——跳跃间断点。
- 端点 $x=0$ 和 $x=1$ 处单侧连续（由延拓保证）。

因此 $F$ 有且仅有 $2$ 个跳跃间断点（$x = 1/3$ 和 $x = 2/3$）。

**注**：一般结论是单调函数在闭区间上连续当且仅当其值域为区间（无"缺口"）。本例中值域 $\{1,2,3\}$ 不是区间，因此必然有跳跃。

</details>
