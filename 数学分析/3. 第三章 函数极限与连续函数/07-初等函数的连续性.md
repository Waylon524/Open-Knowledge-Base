- 复合函数极限定理和反函数连续性定理作为连接初等函数的桥梁，**从未被正式陈述和证明**

本文件的任务就是填补这些理论空白。我们将按照严格的逻辑依赖顺序，从最基础的三角不等式出发，一路构建到所有基本初等函数（三角函数、指数函数、对数函数）的连续性证明，并建立两个核心定理——**复合函数极限定理**和**反函数连续性定理**——它们是初等函数连续性理论的基石。

**核心链条**：
$$| \sin x | \leq |x| \;\rightarrow\; \lim_{x\to 0} \frac{\sin x}{x} = 1 \;\rightarrow\; \sin x,\ \cos x \text{ 连续} \;\rightarrow\; \text{复合函数定理} \;\rightarrow\; \text{反函数定理} \;\rightarrow\; \text{所有初等函数连续}$$

### 3.2 单位圆与三角函数的分析定义

为进行严格的分析证明，我们需要将 $\sin x$ 和 $\cos x$ 从几何直观转化为可操作的分析对象。考虑单位圆 $x^2 + y^2 = 1$：

- 角 $x$（弧度）对应单位圆上从点 $(1,0)$ 出发逆时针经过的**弧长** $x$
- 该弧的终点坐标为 $(\cos x, \sin x)$
- 当 $0 < x < \pi/2$ 时，$\sin x$、$\cos x$、$\tan x$ 均为正数

这种定义方式下，$\sin x$ 和 $\cos x$ 满足：
$$|\sin x| \leq 1,\quad |\cos x| \leq 1,\quad \forall x \in \mathbb{R}$$
$$\sin^2 x + \cos^2 x = 1$$

以及三角恒等式：
$$\sin(x \pm y) = \sin x \cos y \pm \cos x \sin y$$
$$\cos(x \pm y) = \cos x \cos y \mp \sin x \sin y$$

## 4. 原理与方法

### 4.1 不等式 $|\sin x| \leq |x|$ 的几何证明

**定理 4.1（基本不等式）**：对任意实数 $x$，有
$$|\sin x| \leq |x|$$
且当 $0 < |x| < \dfrac{\pi}{2}$ 时，有
$$|\sin x| < |x| < |\tan x|$$

**证明**：

**第 1 步：考虑 $0 < x < \dfrac{\pi}{2}$ 的情形。**

在单位圆中，设圆心角 $\angle AOB = x$（弧度），其中 $A = (1,0)$，$B = (\cos x, \sin x)$。作 $BD \perp OA$ 于 $D$，则 $BD = \sin x$。过 $A$ 作单位圆的切线交 $OB$ 延长线于 $C$。

现在比较三个几何量：
- 三角形 $\triangle OAB$ 的面积：$S_{\triangle OAB} = \dfrac{1}{2} \cdot OA \cdot BD = \dfrac{1}{2} \cdot 1 \cdot \sin x = \dfrac{1}{2}\sin x$
- 扇形 $OAB$ 的面积：$S_{\text{扇形}} = \dfrac{1}{2} \cdot OA^2 \cdot x = \dfrac{1}{2}x$
- 三角形 $\triangle OAC$ 的面积：$S_{\triangle OAC} = \dfrac{1}{2} \cdot OA \cdot AC = \dfrac{1}{2} \cdot 1 \cdot \tan x = \dfrac{1}{2}\tan x$

从几何图形上可见三个区域有严格的包含关系：
$$\triangle OAB \subsetneq \text{扇形 } OAB \subsetneq \triangle OAC$$

因此它们的面积满足严格不等式：
$$\frac{1}{2}\sin x < \frac{1}{2}x < \frac{1}{2}\tan x$$

消去 $\dfrac{1}{2}$，得：
$$\sin x < x < \tan x,\quad 0 < x < \frac{\pi}{2}$$

**第 2 步：推广到 $0 < |x| < \dfrac{\pi}{2}$ 的情形。**

若 $-\dfrac{\pi}{2} < x < 0$，令 $t = -x > 0$，则 $\sin x = -\sin t$，$|x| = t$，$|\sin x| = \sin t$。由第 1 步，$\sin t < t$，故 $|\sin x| < |x|$，且 $|x| < |\tan x|$（因为 $|\tan x| = \tan t$）。

因此对任意 $0 < |x| < \dfrac{\pi}{2}$，有：
$$|\sin x| < |x| < |\tan x|$$

**第 3 步：推广到所有实数 $x$。**

- 若 $|x| \geq \dfrac{\pi}{2}$，则 $|\sin x| \leq 1$。由于 $\dfrac{\pi}{2} \approx 1.57 > 1$，且 $|x| \geq \dfrac{\pi}{2} > 1 \geq |\sin x|$，所以 $|\sin x| \leq |x|$ 成立（等号仅当 $x=0$ 时成立，其他情况均为严格不等式）。

综上，对一切 $x \in \mathbb{R}$，有 $|\sin x| \leq |x|$。证毕。

---

### 4.2 重要极限 $\displaystyle\lim_{x\to 0} \frac{\sin x}{x} = 1$

**定理 4.2（重要极限）**：
$$\lim_{x\to 0} \frac{\sin x}{x} = 1$$

**证明**：

**第 1 步：建立夹逼不等式。**

由定理 4.1，对 $0 < |x| < \dfrac{\pi}{2}$，有 $|\sin x| < |x| < |\tan x|$。

现分 $x>0$ 和 $x<0$ 讨论。

当 $0 < x < \dfrac{\pi}{2}$ 时，$\sin x > 0$，$\tan x > 0$，不等式 $\sin x < x < \tan x$ 各项同除以正数 $\sin x$：
$$1 < \frac{x}{\sin x} < \frac{1}{\cos x}$$

取倒数（注意各项均为正，倒数不等号反向）：
$$\cos x < \frac{\sin x}{x} < 1$$

当 $-\dfrac{\pi}{2} < x < 0$ 时，令 $t = -x > 0$。由于 $\dfrac{\sin x}{x} = \dfrac{-\sin t}{-t} = \dfrac{\sin t}{t}$，且 $\cos x = \cos(-t) = \cos t$，上述不等式同样成立。

因此对一切 $0 < |x| < \dfrac{\pi}{2}$，有：
$$\cos x < \frac{\sin x}{x} < 1$$

**第 2 步：应用夹逼准则。**

我们需要 $\lim_{x\to 0} \cos x = 1$。由定理 4.1，$|\sin x| \leq |x|$，且由 $\cos x = 1 - 2\sin^2\frac{x}{2}$：
$$0 \leq 1 - \cos x = 2\sin^2\frac{x}{2} \leq 2\left|\frac{x}{2}\right|^2 = \frac{x^2}{2}$$

因此 $\lim_{x\to 0} (1 - \cos x) = 0$，即 $\lim_{x\to 0} \cos x = 1$。

**第 3 步：夹逼准则给出结论。**

由夹逼准则（定理 2.6），$\cos x \to 1$ 且常数 $1 \to 1$，而 $\dfrac{\sin x}{x}$ 夹在两者之间：
$$\lim_{x\to 0} \frac{\sin x}{x} = 1$$

证毕。

**注**：此极限的名称"重要"并非夸张——它是整个三角分析的基石。该极限的几何含义是：当 $x$ 很小时，$\sin x$ 与 $x$ 近似相等（即 $\sin x \sim x$，$x\to 0$），这与定理 4.1 中 $|\sin x| \leq |x|$ 的定性结论相呼应，给出了定量的等价关系。

---

### 4.3 $\sin x$ 与 $\cos x$ 在 $\mathbb{R}$ 上的连续性

**定理 4.3（正弦函数的连续性）**：$f(x) = \sin x$ 在 $\mathbb{R}$ 上处处连续。

**证明**：

**第 1 步：在 $x_0 = 0$ 处连续。**

任取 $\varepsilon > 0$。取 $\delta = \varepsilon$。当 $|x - 0| = |x| < \delta$ 时，由定理 4.1：
$$|\sin x - \sin 0| = |\sin x| \leq |x| < \varepsilon$$

因此 $\sin x$ 在 $x=0$ 处连续。

**第 2 步：推广到任意 $x_0 \in \mathbb{R}$。**

利用三角和差化积公式：
$$\sin x - \sin x_0 = 2\cos\frac{x + x_0}{2} \cdot \sin\frac{x - x_0}{2}$$

于是：
$$|\sin x - \sin x_0| = 2\left|\cos\frac{x + x_0}{2}\right| \cdot \left|\sin\frac{x - x_0}{2}\right|$$

由于 $|\cos\theta| \leq 1$ 对任意 $\theta$ 成立，且由定理 4.1，$\left|\sin\frac{x - x_0}{2}\right| \leq \left|\frac{x - x_0}{2}\right|$，因此：
$$|\sin x - \sin x_0| \leq 2 \cdot 1 \cdot \frac{|x - x_0|}{2} = |x - x_0|$$

**第 3 步：完成 $\varepsilon$-$\delta$ 证明。**

任取 $\varepsilon > 0$。取 $\delta = \varepsilon$。当 $|x - x_0| < \delta$ 时：
$$|\sin x - \sin x_0| \leq |x - x_0| < \varepsilon$$

由连续性的 $\varepsilon$-$\delta$ 定义（定义 3.2），$\sin x$ 在任意 $x_0 \in \mathbb{R}$ 处连续。证毕。

---

**定理 4.4（余弦函数的连续性）**：$f(x) = \cos x$ 在 $\mathbb{R}$ 上处处连续。

**证明**：

利用三角和差化积公式：
$$\cos x - \cos x_0 = -2\sin\frac{x + x_0}{2} \cdot \sin\frac{x - x_0}{2}$$

于是：
$$|\cos x - \cos x_0| = 2\left|\sin\frac{x + x_0}{2}\right| \cdot \left|\sin\frac{x - x_0}{2}\right|$$

由于 $|\sin\theta| \leq 1$，且 $|\sin\frac{x - x_0}{2}| \leq \frac{|x - x_0|}{2}$（定理 4.1），得：
$$|\cos x - \cos x_0| \leq 2 \cdot 1 \cdot \frac{|x - x_0|}{2} = |x - x_0|$$

任取 $\varepsilon > 0$，取 $\delta = \varepsilon$，当 $|x - x_0| < \delta$ 时 $|\cos x - \cos x_0| < \varepsilon$。因此 $\cos x$ 在 $\mathbb{R}$ 上处处连续。证毕。

**注**：$\cos x$ 的连续性也可通过 $\cos x = \sin(\pi/2 - x)$ 和复合函数连续性推出（见 4.5 节），但上述 $\varepsilon$-$\delta$ 直接证明不依赖任何其他定理，更加基础。

---

### 4.4 其他三角函数的连续区间

**定理 4.5（正切函数的连续性）**：$\tan x = \dfrac{\sin x}{\cos x}$ 在其定义域 $\{x \in \mathbb{R} \mid \cos x \neq 0\}$ 上处处连续。间断点 $x = \dfrac{\pi}{2} + k\pi\ (k \in \mathbb{Z})$ 为无穷间断点（第二类）。

**证明**：

由定理 4.3 和 4.4，$\sin x$ 和 $\cos x$ 在 $\mathbb{R}$ 上处处连续。由连续函数的商法则（定理 3.3(4)），$\dfrac{\sin x}{\cos x}$ 在满足 $\cos x \neq 0$ 的所有点处连续。

$\cos x = 0$ 的解为 $x = \dfrac{\pi}{2} + k\pi$，$k \in \mathbb{Z}$。在这些点处，$\sin\left(\dfrac{\pi}{2} + k\pi\right) = (-1)^k \neq 0$，$\cos x \to 0$，因此：
$$\lim_{x \to \frac{\pi}{2} + k\pi} |\tan x| = +\infty$$

即这些点为无穷间断点（第二类间断点）。$\tan x$ 的连续区间为：
$$\bigcup_{k \in \mathbb{Z}} \left(-\frac{\pi}{2} + k\pi,\ \frac{\pi}{2} + k\pi\right)$$

---

**定理 4.6（余切、正割、余割函数的连续性）**：

1. $\cot x = \dfrac{\cos x}{\sin x}$ 在 $\{x \in \mathbb{R} \mid \sin x \neq 0\}$ 上连续。间断点 $x = k\pi\ (k \in \mathbb{Z})$ 为无穷间断点。
2. $\sec x = \dfrac{1}{\cos x}$ 在 $\{x \in \mathbb{R} \mid \cos x \neq 0\}$ 上连续。间断点 $x = \dfrac{\pi}{2} + k\pi$ 为无穷间断点。
3. $\csc x = \dfrac{1}{\sin x}$ 在 $\{x \in \mathbb{R} \mid \sin x \neq 0\}$ 上连续。间断点 $x = k\pi$ 为无穷间断点。

**证明**：由 $\sin x$ 和 $\cos x$ 的连续性，结合商法则（或倒数法则，即 $1/f$ 在 $f(x) \neq 0$ 处连续），直接得到各函数的连续区间。间断点处分母趋于 $0$ 而分子非零（或为 $1$，非零），极限为无穷大，故为无穷间断点。证毕。

**间断点分类汇总**：

| 函数 | 连续区间 | 间断点 | 间断点类型 |
|------|---------|--------|-----------|
| $\sin x$ | $\mathbb{R}$ | 无 | — |
| $\cos x$ | $\mathbb{R}$ | 无 | — |
| $\tan x$ | $(\frac{\pi}{2}+k\pi,\ \frac{\pi}{2}+k\pi+\pi)$ | $\frac{\pi}{2} + k\pi$ | 无穷（第二类） |
| $\cot x$ | $(k\pi,\ k\pi+\pi)$ | $k\pi$ | 无穷（第二类） |
| $\sec x$ | $(\frac{\pi}{2}+k\pi,\ \frac{\pi}{2}+k\pi+\pi)$ | $\frac{\pi}{2} + k\pi$ | 无穷（第二类） |
| $\csc x$ | $(k\pi,\ k\pi+\pi)$ | $k\pi$ | 无穷（第二类） |

### 4.5 复合函数的极限定理与连续性推论

**定理 4.7（复合函数的极限定理）**：设 $\displaystyle\lim_{x \to x_0} g(x) = A$，且函数 $f(u)$ 在 $u = A$ 处连续。则
$$\lim_{x \to x_0} f(g(x)) = f(A)$$

**证明**（$\varepsilon$-$\delta$ 语言）：

任取 $\varepsilon > 0$。

**第 1 步：利用 $f$ 在 $A$ 处的连续性。**
由 $f$ 在 $u=A$ 处连续（定义 3.2），存在 $\eta > 0$，使得当 $|u - A| < \eta$ 时，
$$|f(u) - f(A)| < \varepsilon \tag{1}$$

**第 2 步：利用 $\lim_{x\to x_0} g(x) = A$。**
由极限定义（定义 1.1），对上述 $\eta > 0$，存在 $\delta > 0$，使得当 $0 < |x - x_0| < \delta$ 时，
$$|g(x) - A| < \eta \tag{2}$$

**第 3 步：链式对接。**
当 $0 < |x - x_0| < \delta$ 时，由 (2) 知 $|g(x) - A| < \eta$。令 $u = g(x)$，则 $|u - A| < \eta$，代入 (1) 得：
$$|f(g(x)) - f(A)| = |f(u) - f(A)| < \varepsilon$$

因此对任意 $\varepsilon > 0$，存在 $\delta > 0$（如上所述），使得当 $0 < |x - x_0| < \delta$ 时 $|f(g(x)) - f(A)| < \varepsilon$。由极限定义：
$$\lim_{x \to x_0} f(g(x)) = f(A)$$

证毕。

**证明要点说明**：定理证明的关键在于引入中间变量 $\eta$ 作为"桥梁"——$f$ 的连续性将 $\varepsilon$ 的要求转化为对 $u = g(x)$ 逼近 $A$ 的程度要求 $\eta$，而 $\lim g = A$ 又将 $\eta$ 转化为对 $x$ 逼近 $x_0$ 的程度要求 $\delta$。这种"链式对接"技巧是分析中处理复合结构的标准方法。

---

**推论 4.1（复合函数的连续性）**：若 $g$ 在 $x_0$ 处连续，且 $f$ 在 $g(x_0)$ 处连续，则复合函数 $h(x) = f(g(x))$ 在 $x_0$ 处连续。

**证明**：

由 $g$ 在 $x_0$ 处连续知 $\displaystyle\lim_{x\to x_0} g(x) = g(x_0)$。令 $A = g(x_0)$，则 $\lim_{x\to x_0} g(x) = A$，且 $f$ 在 $A$ 处连续。由定理 4.7：
$$\lim_{x\to x_0} f(g(x)) = f(A) = f(g(x_0)) = h(x_0)$$
由连续的定义（定义 3.1），$h$ 在 $x_0$ 处连续。证毕。

**注**：利用数学归纳法，推论 4.1 可推广到有限次复合的情形：若 $f_1, f_2, \ldots, f_n$ 均在相应点处连续，则 $f_n \circ f_{n-1} \circ \cdots \circ f_1$ 在 $x_0$ 处连续。

**例（复合定理的应用）**：由定理 4.3（$\sin x$ 连续）和推论 4.1（$x^2$ 连续），$h(x) = \sin(x^2)$ 在 $\mathbb{R}$ 上连续。这正是此前文件 03 例 6 和文件 04 例 3 中使用而未证明的结论——现在终于有了严格的理论基础。

### 4.6 反函数连续性定理

**定理 4.8（反函数连续性定理）**：设 $f$ 在区间 $I$ 上严格单调且连续，则其反函数 $f^{-1}$ 在值域 $f(I)$ 上也严格单调且连续。

**证明**：

不妨设 $f$ 在 $I$ 上严格单调递增（递减的情形可类似证明或考虑 $-f$）。

**第 1 步：反函数的严格单调性。**

设 $y_1, y_2 \in f(I)$，$y_1 < y_2$。令 $x_1 = f^{-1}(y_1)$，$x_2 = f^{-1}(y_2)$。

假设 $x_1 \geq x_2$。由于 $f$ 严格递增，若 $x_1 \geq x_2$，则 $f(x_1) \geq f(x_2)$，即 $y_1 \geq y_2$，与 $y_1 < y_2$ 矛盾。因此 $x_1 < x_2$，即 $f^{-1}$ 严格递增。

**第 2 步：反函数的连续性（核心部分）。**

设 $y_0 \in f(I)$，$x_0 = f^{-1}(y_0)$。需证 $f^{-1}$ 在 $y_0$ 处连续。

任取 $\varepsilon > 0$。我们需要找到 $\delta > 0$，使得当 $|y - y_0| < \delta$ 且 $y \in f(I)$ 时，$|f^{-1}(y) - f^{-1}(y_0)| < \varepsilon$。

考虑点 $x_0$ 在 $I$ 中的位置，分三种情况讨论。

**情况 A：$x_0$ 为 $I$ 的内点**（即存在左邻域和右邻域在 $I$ 内）。

取 $a = x_0 - \varepsilon$，$b = x_0 + \varepsilon$（但需保证 $a, b \in I$——若 $\varepsilon$ 太大导致 $a$ 或 $b$ 超出 $I$，则缩小 $\varepsilon$ 或用端点情况处理）。不妨设 $a, b \in I$。

由 $f$ 的严格递增性：
$$f(a) < f(x_0) = y_0 < f(b)$$

令 $\delta = \min\{y_0 - f(a),\ f(b) - y_0\} > 0$。

当 $|y - y_0| < \delta$ 且 $y \in f(I)$ 时：
$$y_0 - \delta < y < y_0 + \delta$$

由于 $\delta \leq y_0 - f(a)$ 且 $\delta \leq f(b) - y_0$，有：
$$f(a) = y_0 - (y_0 - f(a)) \leq y_0 - \delta < y < y_0 + \delta \leq y_0 + (f(b) - y_0) = f(b)$$

即 $f(a) < y < f(b)$。由 $f^{-1}$ 的严格递增性：
$$a = f^{-1}(f(a)) < f^{-1}(y) < f^{-1}(f(b)) = b$$

因此 $|f^{-1}(y) - x_0| < \varepsilon$。

**情况 B：$x_0$ 为 $I$ 的左端点**（即 $x_0 = \min I$，如 $I = [x_0, \beta]$ 或 $I = [x_0, \beta)$）。

此时只需证明右连续。取 $b = x_0 + \varepsilon \in I$（若 $\varepsilon$ 足够小），则 $f(x_0) < f(b)$。令 $\delta = f(b) - y_0 > 0$。当 $0 \leq y - y_0 < \delta$ 时，$y < f(b)$，由单调性 $f^{-1}(y) < b = x_0 + \varepsilon$，即 $|f^{-1}(y) - x_0| < \varepsilon$。

**情况 C：$x_0$ 为 $I$ 的右端点**——对称于情况 B，证左连续。

综上，$f^{-1}$ 在 $f(I)$ 上处处连续。证毕。

**注**：定理 4.8 中"严格单调"和"区间"两个条件缺一不可：
- 缺少严格单调性：$f(x) = x^2$ 在 $[-1, 1]$ 上连续但不严格单调，无反函数
- 缺少区间条件：$f$ 在离散集上严格单调，但 $f^{-1}$ 可能不连续（因为 $f(I)$ 也是离散的，连续性平凡成立，但关键是一般定义域若非区间，反函数的连续性条件需要更细致的分析）

**推论 4.2（常见初等函数的反函数连续性）**：下列反函数在其定义域内连续：
- $\sqrt[n]{x}$（作为 $x^n$ 的反函数）在 $[0, \infty)$ 上连续（$n$ 为偶数）或在 $\mathbb{R}$ 上连续（$n$ 为奇数）
- $\arcsin x$ 在 $[-1, 1]$ 上连续（作为 $\sin x$ 在 $[-\pi/2, \pi/2]$ 上的反函数）
- $\arccos x$ 在 $[-1, 1]$ 上连续
- $\arctan x$ 在 $\mathbb{R}$ 上连续

### 4.7 指数函数 $e^x$ 的定义与连续性

**定义 4.1（指数函数 $e^x$）**：对任意实数 $x$，定义
$$e^x = \lim_{n\to\infty} \left(1 + \frac{x}{n}\right)^n$$
可以证明该极限对每个 $x \in \mathbb{R}$ 均存在（此处利用数列极限的理论），且 $e^0 = \lim_{n\to\infty} 1^n = 1$。特别地，常数 $e = e^1 = \lim_{n\to\infty} \left(1 + \frac{1}{n}\right)^n$。

指数函数满足以下基本性质（其证明依赖于极限运算，此处从略）：
1. $e^{x+y} = e^x \cdot e^y$（指数律）
2. $e^{-x} = \dfrac{1}{e^x}$
3. $e^x > 0$ 对一切 $x \in \mathbb{R}$ 成立
4. $e^x$ 在 $\mathbb{R}$ 上严格单调递增

---

**定理 4.9（指数函数的连续性）**：$f(x) = e^x$ 在 $\mathbb{R}$ 上处处连续。

**证明**：

**第 1 步：证明 $\displaystyle\lim_{x\to 0} e^x = 1$（$e^x$ 在 $x=0$ 处连续）。**

我们需要说明当 $|x|$ 很小时，$e^x$ 可以任意接近 $1$。

由定义 $e^x = \lim_{n\to\infty} \left(1 + \frac{x}{n}\right)^n$，先估计 $e^x - 1$ 的界。

利用 Bernoulli 不等式：对实数 $t > -1$，$(1+t)^n \geq 1 + nt$。

取 $n=1$ 时，有不等式（可直接验证）：
$$1 + x \leq e^x \quad\text{对一切 } x \in \mathbb{R} \text{ 成立}$$
（此处我们接受该基本不等式作为指数函数性质的一个结论，其严格证明需要用到 $e^x$ 定义的极限形式）

更精确地，对 $|x| < 1$，有：
$$1 + x \leq e^x \leq 1 + x + x^2$$

这个双端不等式给出了 $e^x$ 在 $0$ 附近的线性逼近。

由夹逼准则：
$$\lim_{x\to 0} (1 + x) = 1,\quad \lim_{x\to 0} (1 + x + x^2) = 1$$
因此：
$$\lim_{x\to 0} e^x = 1$$

**第 2 步：推广到任意 $x_0 \in \mathbb{R}$。**

对任意 $x_0 \in \mathbb{R}$，由指数律：
$$|e^x - e^{x_0}| = e^{x_0} |e^{x - x_0} - 1|$$

当 $x \to x_0$ 时，$x - x_0 \to 0$。由第 1 步知 $e^{x - x_0} \to 1$，因此 $|e^{x - x_0} - 1| \to 0$。于是：
$$\lim_{x\to x_0} e^x = \lim_{x\to x_0} e^{x_0} \cdot e^{x - x_0} = e^{x_0} \cdot 1 = e^{x_0}$$

由连续定义，$e^x$ 在任意 $x_0 \in \mathbb{R}$ 处连续。证毕。

**注**：第 2 步中 $e^{x_0} |e^{x - x_0} - 1|$ 的因子 $e^{x_0}$ 是有界常数（对固定的 $x_0$），因此当 $|e^{x - x_0} - 1| \to 0$ 时乘积也趋于 $0$。严格来说，需用 $\varepsilon$-$\delta$ 表述：任取 $\varepsilon > 0$，由 $\lim_{t\to 0} e^t = 1$，存在 $\delta > 0$ 使 $|t|<\delta$ 时 $|e^t - 1| < \varepsilon / e^{x_0}$，于是当 $|x - x_0| < \delta$ 时 $|e^x - e^{x_0}| < \varepsilon$。

---

### 4.8 对数函数 $\ln x$ 的定义与连续性

**定义 4.2（自然对数函数）**：将指数函数 $e^x$ 的反函数称为自然对数函数，记作 $\ln x$：
$$\ln x = y \iff e^y = x,\quad x \in (0, \infty)$$

$\ln x$ 的定义域为 $(0, \infty)$，值域为 $\mathbb{R}$。由指数函数的性质可得对数函数的基本性质：
1. $\ln(xy) = \ln x + \ln y$（对 $x, y > 0$）
2. $\ln(1) = 0$，$\ln(e) = 1$
3. $\ln\left(\dfrac{x}{y}\right) = \ln x - \ln y$
4. $\ln(x^r) = r \ln x$（$r$ 为有理数，可推广到实数）

---

**定理 4.10（对数函数的连续性）**：$f(x) = \ln x$ 在 $(0, \infty)$ 上处处连续。

**证明**：

由定义 4.2，$\ln x$ 是指数函数 $e^x$ 的反函数。

$e^x$ 在 $\mathbb{R}$ 上严格单调递增（由指数律 $e^{x+y} = e^x e^y$ 和 $e^x > 0$ 可证，当 $x_1 < x_2$ 时 $e^{x_2} - e^{x_1} = e^{x_1}(e^{x_2 - x_1} - 1) > 0$）。

$e^x$ 在 $\mathbb{R}$ 上连续（定理 4.9）。

因此，$e^x$ 在区间 $(-\infty, \infty)$ 上严格单调递增且连续。由反函数连续性定理（定理 4.8），其反函数 $\ln x$ 在值域 $(0, \infty)$ 上严格单调递增且连续。证毕。

**推论 4.3**：$\displaystyle\lim_{x\to 1} \ln x = \ln 1 = 0$。

**证明**：由 $\ln x$ 在 $x=1$ 处的连续性直接得到。证毕。

---

### 4.9 初等函数连续性总结

至此，我们已经完成了所有基本初等函数连续性的证明。汇总如下：

| 基本初等函数 | 连续性 | 理论依据 |
|-------------|--------|---------|
| 常数函数 $c$ | $\mathbb{R}$ 上连续 | 极限定义直接验证 |
| 幂函数 $x^n$（$n \in \mathbb{N}$） | $\mathbb{R}$ 上连续 | 四则运算法则（推论 3.1） |
| 多项式函数 | $\mathbb{R}$ 上连续 | 推论 3.1 |
| 有理函数 | 定义域内连续 | 推论 3.2 |
| $\sin x$ | $\mathbb{R}$ 上连续 | 定理 4.3 |
| $\cos x$ | $\mathbb{R}$ 上连续 | 定理 4.4 |
| $\tan x$、$\sec x$ | $\cos x \neq 0$ 处连续 | 定理 4.5 |
| $\cot x$、$\csc x$ | $\sin x \neq 0$ 处连续 | 定理 4.6 |
| $e^x$ | $\mathbb{R}$ 上连续 | 定理 4.9 |
| $\ln x$ | $(0, \infty)$ 上连续 | 定理 4.10（反函数定理） |
| $\arcsin x$、$\arccos x$ | $[-1, 1]$ 上连续 | 推论 4.2（反函数定理） |
| $\arctan x$ | $\mathbb{R}$ 上连续 | 推论 4.2（反函数定理） |
| $\sqrt[n]{x}$ | 定义域内连续 | 推论 4.2（反函数定理） |

**定理 4.11（初等函数的连续性）**：任何初等函数在其定义域内连续。即，由基本初等函数经有限次四则运算和复合构成的函数，在其定义域的每一点处连续。

**证明要点**：由连续函数的四则运算性质（定理 3.3）、复合函数连续性（推论 4.1）、反函数连续性（定理 4.8），以及各基本初等函数的连续性，经有限步归纳即得。证毕。

定理 4.11 提供了一个极为实用的结论：在求初等函数的极限时，只要函数在该点有定义，极限值就等于函数值。这正是文件 01 中"代入法"求极限的理论基础——现在终于有了完整的证明。

## 5. 例题

### 例 1：利用重要极限计算含三角函数的极限

求 $\displaystyle\lim_{x\to 0} \frac{\sin 3x}{x}$。

**解**：

**第 1 步：变形**。
$$\frac{\sin 3x}{x} = \frac{3 \cdot \sin 3x}{3x} = 3 \cdot \frac{\sin(3x)}{3x}$$

**第 2 步：变量代换**。
令 $t = 3x$，则当 $x \to 0$ 时 $t \to 0$。注意 $\frac{\sin 3x}{x}$ 可视为 $\frac{\sin t}{t/3} = 3 \cdot \frac{\sin t}{t}$。

由定理 4.2（重要极限）：
$$\lim_{t\to 0} \frac{\sin t}{t} = 1$$

**第 3 步：计算结果**。
由极限的乘法法则（定理 2.5(3)）：
$$\lim_{x\to 0} \frac{\sin 3x}{x} = 3 \cdot \lim_{t\to 0} \frac{\sin t}{t} = 3 \cdot 1 = 3$$

**数值验证**：$x=0.01$ 时 $\frac{\sin 0.03}{0.01} \approx \frac{0.0299955}{0.01} = 2.99955 \approx 3$。

---

### 例 2：利用复合函数极限定理求极限

求 $\displaystyle\lim_{x\to 0} \sin(x^2)$ 和 $\displaystyle\lim_{x\to 0} \cos\!\left(\frac{\sin x}{x}\right)$。

**解**：

**(a)** $\displaystyle\lim_{x\to 0} \sin(x^2)$

令 $g(x) = x^2$，则 $\displaystyle\lim_{x\to 0} g(x) = 0$（由 $x^2$ 的连续性，推论 3.1）。$\sin u$ 在 $u=0$ 处连续（定理 4.3）。由复合函数极限定理（定理 4.7）：
$$\lim_{x\to 0} \sin(x^2) = \sin\left(\lim_{x\to 0} x^2\right) = \sin 0 = 0$$

**(b)** $\displaystyle\lim_{x\to 0} \cos\!\left(\frac{\sin x}{x}\right)$

令 $g(x) = \dfrac{\sin x}{x}$，则 $\displaystyle\lim_{x\to 0} g(x) = 1$（定理 4.2）。$\cos u$ 在 $u=1$ 处连续（定理 4.4）。由复合函数极限定理：
$$\lim_{x\to 0} \cos\!\left(\frac{\sin x}{x}\right) = \cos\left(\lim_{x\to 0} \frac{\sin x}{x}\right) = \cos 1$$

**数值验证**：$\cos 1 \approx 0.5403$。$x=0.1$ 时 $\frac{\sin 0.1}{0.1} \approx 0.9983$，$\cos(0.9983) \approx 0.5403$，吻合。

---

### 例 3：利用初等函数连续性求复合极限

求 $\displaystyle\lim_{x\to 2} \ln(x^2 - 3)$。

**解**：

**第 1 步：分解为复合函数**。
令 $g(x) = x^2 - 3$，$f(u) = \ln u$。则 $\ln(x^2 - 3) = f(g(x))$。

**第 2 步：分别验证条件**。
- $\displaystyle\lim_{x\to 2} g(x) = \lim_{x\to 2} (x^2 - 3) = 4 - 3 = 1$（$g$ 是多项式，连续）
- $f(u) = \ln u$ 在 $u = 1$ 处是否连续？由定理 4.10，$\ln u$ 在 $(0, \infty)$ 上连续，而 $1 \in (0, \infty)$，故 $\ln u$ 在 $u=1$ 处连续。

**第 3 步：应用复合函数极限定理（定理 4.7）**。
$$\lim_{x\to 2} \ln(x^2 - 3) = \ln\!\left(\lim_{x\to 2} (x^2 - 3)\right) = \ln 1 = 0$$

**数值验证**：$x=2.01$ 时 $\ln((2.01)^2 - 3) = \ln(4.0401 - 3) = \ln(1.0401) \approx 0.0393$；$x=1.99$ 时 $\ln((1.99)^2 - 3) = \ln(3.9601 - 3) = \ln(0.9601) \approx -0.0407$。均趋近 $0$。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| $\lim_{x\to 0} \frac{\sin x}{x} = 1$ 可以通过代入 $x=0$（$\frac{\sin 0}{0}$）用 L'Hôpital 法则证明 | 代入 $x=0$ 得到 $\frac{0}{0}$ 不定式，不能直接代入。L'Hôpital 法则依赖于 $\sin x$ 的导数，而导数又依赖于该极限，形成循环论证。必须用几何方法或分析证明 |
| $|\sin x| \leq |x|$ 是"显然"的，不需要证明 | 该不等式在数学分析中需要严格证明（几何面积比较法），不能作为公理接受。它在 $\varepsilon$-$\delta$ 证明 $\sin x$ 连续性时起着关键"桥梁"作用 |
| $\tan x$ 在间断点处的间断是可去间断点（因为 $\tan x$ 在 $\pi/2$ 附近的值很大） | $\tan x$ 在间断点处极限为 $\pm\infty$，属于第二类（无穷）间断点，不可修复。可去间断点要求极限存在有限 |
| $\ln x$ 在 $x=0$ 处不是无穷大，所以 $\ln 0 = -\infty$ 是严格的等式 | $\ln 0$ 无定义（$0$ 不在 $\ln x$ 的定义域内）。$\lim_{x\to 0^+} \ln x = -\infty$ 描述的是极限行为，而非函数值 |
| 复合函数极限定理中，只要 $f$ 在 $A$ 处连续，就自动有 $\lim f(g(x)) = f(\lim g(x))$ | 这要求 $\lim g(x)$ 存在且等于 $A$。如果 $\lim g(x)$ 不存在（或等于 $\infty$），定理不适用。定理的两个前提条件（$\lim g = A$ + $f$ 在 $A$ 连续）缺一不可 |
| 反函数连续性定理说"反函数连续"，所以 $\arcsin x$ 在 $[-1, 1]$ 上连续不需要额外条件 | 定理要求原函数严格单调且连续在区间上。$\sin x$ 在整个 $\mathbb{R}$ 上不严格单调，必须限制在 $[-\pi/2, \pi/2]$ 上才能应用该定理 |

## 练习题

### 基础巩固

**1.** 利用本文件已建立的重要极限和初等函数连续性，求下列极限：

(1) $\displaystyle\lim_{x\to 0} \frac{\sin 5x}{\sin 3x}$

(2) $\displaystyle\lim_{x\to 0} \frac{1 - \cos x}{x^2}$

(3) $\displaystyle\lim_{x\to 0} \tan x \cdot \sin\left(\frac{1}{x}\right)$（提示：有界函数乘以无穷小量）

<details><summary>参考答案</summary>

**(1)** $\displaystyle\lim_{x\to 0} \frac{\sin 5x}{\sin 3x}$

**解**：

$$\frac{\sin 5x}{\sin 3x} = \frac{\frac{\sin 5x}{x}}{\frac{\sin 3x}{x}} = \frac{5 \cdot \frac{\sin 5x}{5x}}{3 \cdot \frac{\sin 3x}{3x}}$$

由定理 4.2，$\lim_{t\to 0} \frac{\sin t}{t} = 1$，因此：
$$\lim_{x\to 0} \frac{\sin 5x}{\sin 3x} = \frac{5 \cdot 1}{3 \cdot 1} = \frac{5}{3}$$

---

**(2)** $\displaystyle\lim_{x\to 0} \frac{1 - \cos x}{x^2}$

**解**：

利用三角恒等式 $1 - \cos x = 2\sin^2\frac{x}{2}$：
$$\frac{1 - \cos x}{x^2} = \frac{2\sin^2\frac{x}{2}}{x^2} = \frac{2 \cdot \sin^2\frac{x}{2}}{4 \cdot \left(\frac{x}{2}\right)^2} = \frac{1}{2} \cdot \left(\frac{\sin\frac{x}{2}}{\frac{x}{2}}\right)^2$$

由定理 4.2，$\lim_{t\to 0} \frac{\sin t}{t} = 1$（令 $t = x/2$），因此：
$$\lim_{x\to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2} \cdot 1^2 = \frac{1}{2}$$

**注**：此极限也是一个重要结论——$\displaystyle\lim_{x\to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2}$，常用作 $1 - \cos x \sim \frac{1}{2}x^2$（$x\to 0$）。

---

**(3)** $\displaystyle\lim_{x\to 0} \tan x \cdot \sin\left(\frac{1}{x}\right)$

**解**：

当 $x\to 0$ 时，$\tan x \to 0$（由 $\tan x = \sin x / \cos x$，$\sin x \to 0$，$\cos x \to 1$，故 $\tan x \to 0$）。因此 $\tan x$ 是 $x\to 0$ 时的无穷小量。

$|\sin(1/x)| \leq 1$ 对一切 $x \neq 0$ 成立，即 $\sin(1/x)$ 在 $x=0$ 附近有界。

由无穷小量的性质（有界函数与无穷小量的乘积仍为无穷小量）：
$$\lim_{x\to 0} \tan x \cdot \sin\left(\frac{1}{x}\right) = 0$$

**数值验证**：$x=0.1$ 时 $\tan 0.1 \approx 0.1003$，$\sin(10) \approx -0.5440$，乘积 $\approx -0.0546$，接近 $0$。

</details>

### 迁移应用

**2.** 已知 $f(x) = \begin{cases} \dfrac{\sin ax}{x}, & x > 0 \\ b, & x = 0 \\ e^{cx} - 1, & x < 0 \end{cases}$。确定常数 $a, b, c$ 的值使 $f$ 在 $x=0$ 处连续。

<details><summary>参考答案</summary>

**解**：

$f(0) = b$，有定义。需要左极限等于右极限等于 $f(0)$。

**第 1 步：右极限**（$x \to 0^+$，$f(x) = \frac{\sin ax}{x}$）。

令 $t = ax$。当 $x \to 0^+$ 时 $t \to 0^+$。由定理 4.2：
$$\lim_{x\to 0^+} \frac{\sin ax}{x} = \lim_{x\to 0^+} a \cdot \frac{\sin ax}{ax} = a \cdot \lim_{t\to 0} \frac{\sin t}{t} = a \cdot 1 = a$$

**第 2 步：左极限**（$x \to 0^-$，$f(x) = e^{cx} - 1$）。

由定理 4.9（$e^x$ 在 $x=0$ 处连续），$\lim_{t\to 0} e^t = 1$。当 $x \to 0^-$ 时 $cx \to 0$（无论 $c$ 为何值），因此：
$$\lim_{x\to 0^-} (e^{cx} - 1) = \lim_{x\to 0^-} e^{cx} - 1 = e^0 - 1 = 1 - 1 = 0$$

因此无论 $c$ 取何值（$c$ 为有限常数），左极限均为 $0$。

**第 3 步：连续条件**。

需要 $f(0) = b$ 等于左极限 $0$ 等于右极限 $a$：
$$b = 0 = a$$

因此 $a = 0$，$b = 0$，$c$ 可取任意实数。

**验证**：当 $a=0$ 时，$x>0$ 时 $f(x) = \frac{\sin 0}{x} = 0$，右极限为 $0$。$f(0)=0$。左极限也为 $0$（与 $c$ 无关）。因此 $f$ 在 $x=0$ 处连续。

</details>

**3.** （综合）设 $f(x) = \sqrt{x} \cdot \sin\left(\frac{1}{x}\right)$（$x>0$），补充定义 $f(0)$ 使 $f$ 在 $x=0$ 处连续。

<details><summary>参考答案</summary>

**解**：

**第 1 步：分析极限行为**。

当 $x \to 0^+$ 时，$\sqrt{x} \to 0$（定理 4.11，$\sqrt{x}$ 连续），而 $|\sin(1/x)| \leq 1$ 有界。

由有界函数乘以无穷小量仍为无穷小量，$\sqrt{x} \cdot \sin(1/x)$ 在 $x\to 0^+$ 时趋于 $0$。

**第 2 步：补充定义**。

补充定义 $f(0) = 0$，则：
$$\lim_{x\to 0^+} f(x) = 0 = f(0)$$

因此 $f$ 在 $x=0$ 处连续（右连续，定义域为 $[0, \infty)$）。

**与例 5 的对比**：回顾文件 03 中 $\sin(1/x)$ 在 $x=0$ 处是振荡间断点（无论怎样定义 $f(0)$ 都无法使其连续）。但这里乘了一个趋于 $0$ 的因子 $\sqrt{x}$，振荡被"压制"了。这是因为 $|\sqrt{x} \cdot \sin(1/x)| \leq \sqrt{x} \to 0$，夹逼准则保证了极限为 $0$。

**进一步思考**：$f(x) = x^\alpha \sin(1/x)$ 在 $x>0$ 时能否连续延拓到 $x=0$？当 $\alpha > 0$ 时，$\lim_{x\to 0^+} x^\alpha \sin(1/x) = 0$（因为 $|x^\alpha \sin(1/x)| \leq x^\alpha \to 0$），因此可补充定义 $f(0)=0$ 使其连续。当 $\alpha \leq 0$ 时（如 $\alpha = 0$ 即 $\sin(1/x)$），极限不存在，无法连续延拓。

</details>
