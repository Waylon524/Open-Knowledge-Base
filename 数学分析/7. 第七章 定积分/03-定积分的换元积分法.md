# 03. 定积分的换元积分法

## 1. 学习目标与先修前置

### 学习目标
- 掌握定积分的第一换元法（凑微分法）公式：$\displaystyle\int_a^b f(\varphi(x))\varphi'(x)\,dx = \int_{\varphi(a)}^{\varphi(b)} f(u)\,du$

- 理解换元法的适用条件：换元函数须连续可导、$f$须连续、第二换元法要求$\psi$严格单调

## 2. 背景与应用场景

### 2.1 定积分计算的核心困难

在文件02中，我们学会了用Newton-Leibniz公式计算定积分：只要找到被积函数的一个原函数，代入上下限即可。但问题在于——**很多被积函数的原函数不是显而易见的**。

例如：
$$\int_0^{\pi/6} \cos 3x\,dx,\quad \int_0^1 2x\,e^{x^2}\,dx,\quad \int_1^e \frac{\ln x}{x}\,dx$$

这些被积函数$\cos 3x$、$2x e^{x^2}$、$\frac{\ln x}{x}$的"原函数"是什么？$\cos 3x$的原函数是$\frac{1}{3}\sin 3x$吗？$2x e^{x^2}$的原函数是$e^{x^2}$吗？$\frac{\ln x}{x}$的原函数是$\frac{1}{2}(\ln x)^2$吗？

这些猜测都是正确的，但我们需要一个系统性的方法——这就是**换元积分法**。

### 2.2 从链式法则到换元法——微分与积分的又一次互逆

回想链式法则（第四章文件03）：
$$\frac{d}{dx}F(\varphi(x)) = F'(\varphi(x)) \cdot \varphi'(x)$$

如果$F' = f$，即$F$是$f$的一个原函数，那么：
$$\frac{d}{dx}F(\varphi(x)) = f(\varphi(x)) \cdot \varphi'(x)$$

对等式两边从$a$到$b$求定积分，左边由N-L公式得$F(\varphi(b)) - F(\varphi(a))$，右边是$\int_a^b f(\varphi(x))\varphi'(x)\,dx$。于是：
$$\int_a^b f(\varphi(x))\varphi'(x)\,dx = F(\varphi(b)) - F(\varphi(a))$$

而另一方面，$\int_{\varphi(a)}^{\varphi(b)} f(u)\,du = F(\varphi(b)) - F(\varphi(a))$。因此：
$$\int_a^b f(\varphi(x))\varphi'(x)\,dx = \int_{\varphi(a)}^{\varphi(b)} f(u)\,du$$

这就是**定积分的换元公式**。它的本质是：**链式法则的逆用**。

### 2.3 换元法的实践意义

换元法的核心思想是：通过引入一个新的变量$u$，将复杂的被积函数转化为一个更简单的函数，同时相应地调整积分限。

在实践中，换元法有两种方向：
- **第一换元法（凑微分法）**：设$u = \varphi(x)$，将复合函数的内层替换为新变量
- **第二换元法（逆向代换）**：设$x = \psi(t)$，将原有的被积变量替换为另一个变量的函数

这两种方法的共同原则是**换元必换限**——引入了新变量后，积分上下限必须根据换元关系重新计算。

## 3. 核心概念与符号约定

### 3.1 定积分换元公式（第一换元法）

**定理 7.7（定积分第一换元法 / First Substitution Method）**：设$f(u)$在区间$[\alpha, \beta]$上连续，$\varphi(x)$在$[a, b]$上可导且$\varphi'(x)$连续，且$\varphi(x)$的值域$\varphi([a,b]) \subseteq [\alpha, \beta]$。则：
$$\int_a^b f(\varphi(x))\varphi'(x)\,dx = \int_{\varphi(a)}^{\varphi(b)} f(u)\,du$$

**证明**：由于$f$在$[\alpha, \beta]$上连续，由定理7.5，$f$存在原函数$F$，即$F'(u)=f(u)$。考虑复合函数$F(\varphi(x))$，由链式法则：
$$\frac{d}{dx}\big[F(\varphi(x))\big] = F'(\varphi(x))\varphi'(x) = f(\varphi(x))\varphi'(x)$$
对等式从$a$到$b$积分，左边由N-L公式（定理7.6）：
$$\int_a^b f(\varphi(x))\varphi'(x)\,dx = \big[F(\varphi(x))\big]_a^b = F(\varphi(b)) - F(\varphi(a))$$
而$F$也是$f$的原函数，由N-L公式：
$$\int_{\varphi(a)}^{\varphi(b)} f(u)\,du = \big[F(u)\big]_{\varphi(a)}^{\varphi(b)} = F(\varphi(b)) - F(\varphi(a))$$
因此两端相等：
$$\int_a^b f(\varphi(x))\varphi'(x)\,dx = \int_{\varphi(a)}^{\varphi(b)} f(u)\,du$$
证毕。$\square$

### 3.2 积分限变换规则——"换元必换限"

**换元后积分限的对应关系**：在做变量替换$u = \varphi(x)$时，积分限按以下规则变换：

$$\boxed{x = a \;\longrightarrow\; u = \varphi(a)}$$
$$\boxed{x = b \;\longrightarrow\; u = \varphi(b)}$$

即：
- **原来的下限$x = a$对应新变量$u$的下限$u = \varphi(a)$**
- **原来的上限$x = b$对应新变量$u$的上限$u = \varphi(b)$**

**关键提醒**：积分限的变换遵循"对应"原则——谁是下限谁是上限，由映射$\varphi$决定，**不要求$\varphi(a) < \varphi(b)$**。若$\varphi$递减则$\varphi(a) > \varphi(b)$，此时新积分的下限大于上限，但仍保持"下限对应下限，上限对应上限"的位置关系。

### 3.3 第二换元法（逆向代换）

**定理 7.8（定积分第二换元法 / Second Substitution Method）**：设$x = \psi(t)$在$[\alpha, \beta]$（或$[\beta, \alpha]$）上可导且$\psi'(t)$连续，$\psi(\alpha) = a$、$\psi(\beta) = b$，且$\psi(t)$的值域$\psi([\alpha,\beta]) \subseteq [a, b]$。若$f$在$[a, b]$上连续，则：
$$\int_a^b f(x)\,dx = \int_\alpha^\beta f(\psi(t))\psi'(t)\,dt$$

**证明**：设$F$是$f$的一个原函数（由定理7.5存在），即$F'(x) = f(x)$。考虑复合函数$F(\psi(t))$，由链式法则：
$$\frac{d}{dt}\big[F(\psi(t))\big] = F'(\psi(t))\psi'(t) = f(\psi(t))\psi'(t)$$
对$t$从$\alpha$到$\beta$积分，由N-L公式：
$$\int_\alpha^\beta f(\psi(t))\psi'(t)\,dt = \big[F(\psi(t))\big]_\alpha^\beta = F(\psi(\beta)) - F(\psi(\alpha)) = F(b) - F(a)$$
而$F(b)-F(a) = \int_a^b f(x)\,dx$，因此：
$$\int_a^b f(x)\,dx = \int_\alpha^\beta f(\psi(t))\psi'(t)\,dt$$
证毕。$\square$

### 3.4 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $u = \varphi(x)$ | 第一换元法的新变量（内层函数） | $u = 3x$、$u = x^2$、$u = \ln x$ |
| $x = \psi(t)$ | 第二换元法的逆向代换 | $x = \sin t$、$x = t^2$ |
| $\varphi(a), \varphi(b)$ | 换元后的新积分下限和上限 | $\varphi(0)=0,\ \varphi(\pi/6)=\pi/2$ |
| $[\cdots]_a^b$ | 代入上下限做差的运算 | $\big[\sin u\big]_0^{\pi/2}=1-0=1$ |
| $du = \varphi'(x)dx$ | 微分关系（换元时的微分变换） | $d(x^2)=2x\,dx$ |

## 4. 原理与方法

### 4.1 第一换元法的标准步骤

第一换元法的基本思路是：观察被积函数，若它形如$f(\varphi(x))\varphi'(x)$，则设$u = \varphi(x)$，将原积分转化为$\int f(u)\,du$再代入新限。

**标准流程**：(1) 识别结构$f(\varphi(x))\varphi'(x)$；(2) 令$u = \varphi(x)$，写出$du = \varphi'(x)dx$；(3) 换限：$x=a \to u=\varphi(a)$，$x=b \to u=\varphi(b)$；(4) 全部用$u$写出新积分；(5) 用N-L公式计算。

### 4.2 三种常见的凑微分模式

**(1) 线性替换**：$f(kx+c)$型，$\varphi(x)=kx+c$，$du=k\,dx$。$$\int_\alpha^\beta f(kx+c)\,dx = \frac{1}{k}\int_{k\alpha+c}^{k\beta+c} f(u)\,du \quad (k \neq 0)$$

**(2) 幂函数替换**：含$x^n$复合与$x^{n-1}$因子，如$\int_a^b x f(x^2)\,dx$，令$u=x^2$，$x\,dx=du/2$。

**(3) 三角函数替换**：含$\sin x$或$\cos x$复合与另一三角函数乘积，如$\int_a^b \sin x \cdot f(\cos x)\,dx$，令$u=\cos x$，$\sin x\,dx=-du$。**注意符号**。

### 4.3 第二换元法的标准步骤

第二换元法的基本思路是：当被积函数中含有根式、分式等复杂结构时，引入一个新的变量$t$，令$x = \psi(t)$，使得关于$x$的复杂表达式转化为关于$t$的简单形式。

**标准流程**：第1步，识别需要换元的结构（根式、分式等）；第2步，设定$x = \psi(t)$；第3步，写出微分关系$dx = \psi'(t)dt$；第4步，由$\psi(\alpha)=a$、$\psi(\beta)=b$确定新积分限；第5步，写出新积分并计算。

**关于单调性**：第二换元法中通常要求$\psi(t)$在$[\alpha,\beta]$上**严格单调**，以保证一一映射和解的唯一性。常见类型包括：
- **根式替换**：含$\sqrt[n]{g(x)}$时令$t = \sqrt[n]{g(x)}$ | **三角替换**：含$\sqrt{a^2-x^2}$时令$x=a\sin t$

### 4.4 换元法的适用条件与边界

**(1) 连续性条件**

换元公式的证明要求$f$在相应的区间上**连续**。若$f$有间断点，换元公式不一定成立。在第一换元法中，要求$f$在$\varphi([a,b])$上连续；在第二换元法中，要求$f$在$[a,b]$上连续。

**(2) 可导性与可积性**

要求$\varphi$（或$\psi$）**可导且导数连续**。这个条件保证了链式法则的适用。

**(3) 值域包含条件**

第一换元法中，$\varphi([a,b])$必须包含在$f$的连续区间内。如果$\varphi(x)$在$[a,b]$上的取值范围超出了$f$的定义域，则换元公式失效。

**(4) 第二换元法的单调性条件**：虽公式证明不严格要求单调性，但实际中为保证积分限唯一性和避免折返，通常要求$\psi$在$[\alpha,\beta]$上严格单调且$\psi'(t)$连续。

**(5) 换元必换限**：无论哪种方法，换元后必须更新积分限，不可保留原限。这是最容易被违反的铁律。

## 5. 例题

### 例 1：第一换元法——线性替换

计算 $\displaystyle \int_0^{\pi/6} \cos 3x\,dx$。

**解**：

**第 1 步：识别结构**。被积函数为$\cos 3x$，是复合函数$f(\varphi(x))$的形式，其中$\varphi(x) = 3x$，$\varphi'(x) = 3$。此结构形如$f(\varphi(x))\varphi'(x)$但缺少因子$\varphi'(x)=3$，因此需要通过系数调整来凑微分。

**第 2 步：换元**。令$u = 3x$，则$du = 3\,dx$，即$dx = \dfrac{du}{3}$。

**第 3 步：换限**。这是最关键的一步——必须同时变换积分限：
- 当$x = 0$时，$u = 3 \times 0 = 0$
- 当$x = \pi/6$时，$u = 3 \times \pi/6 = \pi/2$

**第 4 步：写出新积分**。
$$
\int_0^{\pi/6} \cos 3x\,dx = \int_0^{\pi/2} \cos u \cdot \frac{du}{3} = \frac{1}{3} \int_0^{\pi/2} \cos u\,du
$$

**第 5 步：计算**。由N-L公式，$\cos u$的一个原函数是$\sin u$：
$$
\frac{1}{3} \int_0^{\pi/2} \cos u\,du = \frac{1}{3} \big[\sin u\big]_0^{\pi/2} = \frac{1}{3} \left(\sin\frac{\pi}{2} - \sin 0\right) = \frac{1}{3} (1 - 0) = \frac{1}{3}
$$

因此 $\displaystyle \int_0^{\pi/6} \cos 3x\,dx = \frac{1}{3}$。

### 例 2：第一换元法——复合函数的凑微分

计算下列定积分：

**(a)** $\displaystyle \int_0^1 2x\,e^{x^2}\,dx$

**(b)** $\displaystyle \int_1^e \frac{\ln x}{x}\,dx$

**(c)** $\displaystyle \int_0^{\pi/2} \sin x \cos^2 x\,dx$

**解 (a)**：

**第 1 步：识别结构**。被积函数为$2x \cdot e^{x^2}$。注意到$(x^2)' = 2x$，因此被积函数形如$e^{x^2} \cdot (x^2)'$，这正是$f(\varphi(x))\varphi'(x)$的形式，其中$f(u) = e^u$、$\varphi(x) = x^2$、$\varphi'(x) = 2x$。

**第 2 步：换元**。令$u = x^2$，则$du = 2x\,dx$。

**第 3 步：换限**：
- 当$x = 0$时，$u = 0^2 = 0$
- 当$x = 1$时，$u = 1^2 = 1$

**第 4 步：写出新积分**。
$$
\int_0^1 2x\,e^{x^2}\,dx = \int_0^1 e^u\,du
$$

**第 5 步：计算**。由N-L公式：
$$
\int_0^1 e^u\,du = \big[e^u\big]_0^1 = e^1 - e^0 = e - 1
$$

因此 $\displaystyle \int_0^1 2x\,e^{x^2}\,dx = e - 1$。

---

**解 (b)**：

**第 1 步：识别结构**。被积函数为$\dfrac{\ln x}{x}$。注意到$(\ln x)' = \dfrac{1}{x}$，而被积函数恰好是$\ln x \cdot \dfrac{1}{x} = \ln x \cdot (\ln x)'$。因此这符合$f(\varphi(x))\varphi'(x)$的模式，其中$f(u) = u$、$\varphi(x) = \ln x$、$\varphi'(x) = \frac{1}{x}$。

**第 2 步：换元**。令$u = \ln x$，则$du = \dfrac{1}{x}\,dx$。

**第 3 步：换限**：
- 当$x = 1$时，$u = \ln 1 = 0$
- 当$x = e$时，$u = \ln e = 1$

**第 4 步：写出新积分**。
$$
\int_1^e \frac{\ln x}{x}\,dx = \int_0^1 u\,du
$$

**第 5 步：计算**。由N-L公式：
$$
\int_0^1 u\,du = \left[\frac{u^2}{2}\right]_0^1 = \frac{1^2}{2} - \frac{0^2}{2} = \frac{1}{2}
$$

因此 $\displaystyle \int_1^e \frac{\ln x}{x}\,dx = \frac{1}{2}$。

---

**解 (c)**：

**第 1 步：识别结构**。被积函数为$\sin x \cos^2 x$。注意到$(\cos x)' = -\sin x$，因此$\sin x\,dx = -d(\cos x)$。于是$\sin x \cos^2 x\,dx = \cos^2 x \cdot \sin x\,dx = - \cos^2 x\,d(\cos x)$。这符合$f(\varphi(x))\varphi'(x)$的模式，其中$f(u) = u^2$、$\varphi(x) = \cos x$、$\varphi'(x) = -\sin x$。

**第 2 步：换元**。令$u = \cos x$，则$du = -\sin x\,dx$，即$\sin x\,dx = -du$。

**第 3 步：换限**：
- 当$x = 0$时，$u = \cos 0 = 1$
- 当$x = \pi/2$时，$u = \cos(\pi/2) = 0$

**注意**：这里$u$从$1$到$0$（下限大于上限），这是允许的。

**第 4 步：写出新积分**。
$$
\int_0^{\pi/2} \sin x \cos^2 x\,dx = \int_1^0 u^2 \cdot (-du) = -\int_1^0 u^2\,du
$$

**第 5 步：整理并计算**。交换积分限（区间交换性质：$-\int_1^0 = \int_0^1$）：
$$
-\int_1^0 u^2\,du = \int_0^1 u^2\,du = \left[\frac{u^3}{3}\right]_0^1 = \frac{1}{3}
$$

因此 $\displaystyle \int_0^{\pi/2} \sin x \cos^2 x\,dx = \frac{1}{3}$。

### 例 3：第二换元法（逆向代换）

计算下列定积分：

**(a)** $\displaystyle \int_0^1 \sqrt{1 - x^2}\,dx$

**(b)** $\displaystyle \int_1^4 \frac{1}{1 + \sqrt{x}}\,dx$

**解 (a)**：

**第 1 步：分析结构**。被积函数$\sqrt{1 - x^2}$在文件01中曾通过几何意义得到结果$\pi/4$（四分之一单位圆面积）。现在用第二换元法来严格计算。

**第 2 步：选择代换**。对于含$\sqrt{1 - x^2}$的积分，标准代换是$x = \sin t$（或$x = \cos t$），利用三角恒等式$\sqrt{1 - \sin^2 t} = |\cos t|$消去根号。

令$x = \sin t$，在$t \in [0, \pi/2]$上严格单调递增（$\sin t$在$[0, \pi/2]$上从$0$增加到$1$），可逆性好。

**第 3 步：写出微分关系**。$dx = \cos t\,dt$。

**第 4 步：换限**：
- 当$x = 0$时，$0 = \sin t$，解得$t = 0$
- 当$x = 1$时，$1 = \sin t$，解得$t = \pi/2$

因此在$t$的积分区间为$[0, \pi/2]$。

**第 5 步：写出新积分**。

当$t \in [0, \pi/2]$时，$\cos t \geq 0$，因此$\sqrt{1 - \sin^2 t} = \sqrt{\cos^2 t} = \cos t$。

$$
\begin{aligned}
\int_0^1 \sqrt{1 - x^2}\,dx
&= \int_0^{\pi/2} \sqrt{1 - \sin^2 t} \cdot \cos t\,dt \\
&= \int_0^{\pi/2} |\cos t| \cdot \cos t\,dt \\
&= \int_0^{\pi/2} \cos^2 t\,dt \quad (\because \cos t \geq 0)
\end{aligned}
$$

**第 6 步：计算**。利用倍角公式$\cos^2 t = \frac{1 + \cos 2t}{2}$：
$$
\int_0^{\pi/2} \cos^2 t\,dt = \int_0^{\pi/2} \frac{1 + \cos 2t}{2}\,dt = \frac{1}{2}\int_0^{\pi/2} (1 + \cos 2t)\,dt
$$

**第 7 步：分别积分**。
$$
\frac{1}{2}\left[\int_0^{\pi/2} 1\,dt + \int_0^{\pi/2} \cos 2t\,dt\right]
$$

第一项：$\int_0^{\pi/2} 1\,dt = \frac{\pi}{2}$。

第二项：用第一换元法（线性替换），令$u = 2t$，$du = 2\,dt$：
$$\int_0^{\pi/2} \cos 2t\,dt = \frac{1}{2}\int_0^{\pi} \cos u\,du = \frac{1}{2}[\sin u]_0^{\pi} = \frac{1}{2}(0 - 0) = 0$$

**第 8 步：合并结果**。
$$
\int_0^1 \sqrt{1 - x^2}\,dx = \frac{1}{2} \cdot \frac{\pi}{2} = \frac{\pi}{4}
$$

---

**解 (b)**：

**第 1 步：分析结构**。被积函数为$\frac{1}{1 + \sqrt{x}}$，含有根号$\sqrt{x}$，直接积分困难。通过令$t = \sqrt{x}$（即$x = t^2$）可消去根号。

**第 2 步：选择代换**。令$t = \sqrt{x}$，即$x = t^2$，$t \in [1, 2]$（因为$x$从$1$到$4$，所以$t$从$1$到$2$）。$x = t^2$在$[1, 2]$上严格单调递增。

**第 3 步：写出微分关系**。$dx = 2t\,dt$。

**第 4 步：换限**：
- 当$x = 1$时，$t = \sqrt{1} = 1$
- 当$x = 4$时，$t = \sqrt{4} = 2$

**第 5 步：写出新积分**。
$$
\int_1^4 \frac{1}{1 + \sqrt{x}}\,dx = \int_1^2 \frac{1}{1 + t} \cdot 2t\,dt = 2\int_1^2 \frac{t}{1 + t}\,dt
$$

**第 6 步：化简被积函数**（分子除以分母的技巧）。
$$
\frac{t}{1+t} = \frac{(t+1) - 1}{t+1} = 1 - \frac{1}{t+1}
$$

因此：
$$
2\int_1^2 \frac{t}{1+t}\,dt = 2\int_1^2 \left(1 - \frac{1}{t+1}\right)dt = 2\left[\int_1^2 1\,dt - \int_1^2 \frac{1}{t+1}\,dt\right]
$$

**第 7 步：分别计算**。

第一项：$\int_1^2 1\,dt = \big[t\big]_1^2 = 2 - 1 = 1$。

第二项：用第一换元法，令$u = t+1$，$du = dt$，换限：$t=1 \to u=2$，$t=2 \to u=3$：
$$\int_1^2 \frac{1}{t+1}\,dt = \int_2^3 \frac{1}{u}\,du = \big[\ln |u|\big]_2^3 = \ln 3 - \ln 2 = \ln\frac{3}{2}$$

**第 8 步：合并结果**。
$$
2\left[1 - \ln\frac{3}{2}\right] = 2\left(1 - \ln\frac{3}{2}\right)
$$

因此 $\displaystyle \int_1^4 \frac{1}{1 + \sqrt{x}}\,dx = 2\left(1 - \ln\frac{3}{2}\right)$。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 换元后保持原积分限不变，如$\int_0^{\pi/6}\cos 3x\,dx = \int_0^{\pi/6}\frac{\cos u}{3}\,du$（仍用$x$的限） | **换元必换限**：$u = 3x$，当$x=0$时$u=0$，当$x=\pi/6$时$u=\pi/2$。正确应为$\frac{1}{3}\int_0^{\pi/2}\cos u\,du$ |
| 第一换元法只适用于被积函数恰好有$\varphi'(x)$因子的情况 | 若被积函数只有$f(\varphi(x))$没有$\varphi'(x)$，可以通过乘除系数来构造$\varphi'(x)$。如$\int\cos 3x\,dx$缺少因子$3$，令$u=3x$后调整$dx = du/3$即可 |
| 第二换元法中$x = \psi(t)$可以任意选择，无需考虑单调性 | 实际计算中需要$\psi$严格单调，否则积分限对应不唯一。例如$x = t^2$在$[-1, 1]$上不是单调的，若取$x = t^2$从$x=0$到$x=1$，$t$可对应$\pm1$，会产生歧义 |
| $\int_0^{\pi/2}\sin x \cos^2 x\,dx$换元$u=\cos x$时，$du = -\sin x\,dx$中的负号可以直接忽略 | 负号必须正确处理。$u=\cos x$时$\sin x\,dx = -du$，代入后得到$-\int_1^0 u^2\,du$，再交换限得$\int_0^1 u^2\,du$ |
| 第二换元法只需要换被积函数，$dx$直接写成$dt$不用管 | $dx$也必须通过$\psi'(t)\,dt$进行替换。$x = \psi(t)$时$dx = \psi'(t)\,dt$，不能省略$\psi'(t)$ |
| 换元法证明中不需要$f$的连续性 | 连续性是定理7.5（原函数存在定理）的前提，而换元公式的证明依赖于原函数的存在。若$f$有间断点，原函数可能不存在，换元公式不能保证成立 |

### 检查点

- [ ] 能否默写定理7.7（第一换元法）和定理7.8（第二换元法）并给出证明的完整步骤？
- [ ] 能否用自己的话解释"换元必换限"的含义？能否举出一个忘记换限的错误例子？
- [ ] 能否判断一个定积分应该用第一换元法还是第二换元法？依据是什么？
- [ ] 能否处理换元后积分限"上下颠倒"的情况（如例2(c)中$u$从$1$到$0$）？
- [ ] 能否处理换元函数导数出现负号的情况？如何将负号正确代入积分？
- [ ] 能否说明第二换元法中要求$\psi(t)$严格单调的原因？
- [ ] 能否独立完成$\int_0^{a} \sqrt{a^2 - x^2}\,dx$的第二换元法计算？
- [ ] 能否说明换元法对$f$和$\varphi$（或$\psi$）的连续性要求？

## 练习题

### 基础巩固

**1.** 利用第一换元法（凑微分法）计算下列定积分：

(1) $\displaystyle \int_0^1 e^{2x}\,dx$

(2) $\displaystyle \int_0^1 \frac{x}{1+x^2}\,dx$

(3) $\displaystyle \int_0^{\pi/4} \tan x\,dx$（提示：$\tan x = \dfrac{\sin x}{\cos x}$）

<details><summary>参考答案</summary>

**(1)** $\displaystyle \int_0^1 e^{2x}\,dx$

**第 1 步：换元**。令$u = 2x$，$du = 2\,dx$，$dx = \dfrac{du}{2}$。

**第 2 步：换限**。$x=0 \to u=0$，$x=1 \to u=2$。

**第 3 步：计算**。
$$
\int_0^1 e^{2x}\,dx = \int_0^2 e^u \cdot \frac{du}{2} = \frac{1}{2}\int_0^2 e^u\,du = \frac{1}{2}\big[e^u\big]_0^2 = \frac{1}{2}(e^2 - 1)
$$

**验证**：原函数$\frac{1}{2}e^{2x}$，$\big[\frac{1}{2}e^{2x}\big]_0^1 = \frac{1}{2}(e^2 - 1)$。$\checkmark$

---

**(2)** $\displaystyle \int_0^1 \frac{x}{1+x^2}\,dx$

**第 1 步：识别结构**。$(1+x^2)' = 2x$，分子$x$与分母的导数只差常数因子$2$。

**第 2 步：换元**。令$u = 1+x^2$，$du = 2x\,dx$，$x\,dx = \dfrac{du}{2}$。

**第 3 步：换限**。$x=0 \to u=1$，$x=1 \to u=2$。

**第 4 步：计算**。
$$
\int_0^1 \frac{x}{1+x^2}\,dx = \int_1^2 \frac{1}{u} \cdot \frac{du}{2} = \frac{1}{2}\int_1^2 \frac{1}{u}\,du = \frac{1}{2}\big[\ln u\big]_1^2 = \frac{1}{2}\ln 2
$$

---

**(3)** $\displaystyle \int_0^{\pi/4} \tan x\,dx$

**第 1 步：改写被积函数**。$\tan x = \dfrac{\sin x}{\cos x}$。注意到$(\cos x)' = -\sin x$。

**第 2 步：换元**。令$u = \cos x$，$du = -\sin x\,dx$，$\sin x\,dx = -du$。

**第 3 步：换限**。$x=0 \to u=\cos 0 = 1$，$x=\pi/4 \to u=\cos(\pi/4) = \sqrt{2}/2$。

**第 4 步：计算**。
$$
\int_0^{\pi/4} \tan x\,dx = \int_1^{\sqrt{2}/2} \frac{1}{u} \cdot (-du) = -\big[\ln u\big]_1^{\sqrt{2}/2} = -\ln\frac{\sqrt{2}}{2} = \frac{1}{2}\ln 2
$$

</details>

---

**2.** 利用第二换元法（逆向代换）计算下列定积分：

(1) $\displaystyle \int_0^4 \frac{1}{2 + \sqrt{x}}\,dx$

(2) $\displaystyle \int_0^a \sqrt{a^2 - x^2}\,dx \quad (a > 0)$

<details><summary>参考答案</summary>

**(1)** $\displaystyle \int_0^4 \frac{1}{2 + \sqrt{x}}\,dx$

**第 1 步：选择代换**。令$t = \sqrt{x}$，即$x = t^2$（$t \geq 0$），$dx = 2t\,dt$。

**第 2 步：换限**。$x=0 \to t=0$，$x=4 \to t=2$。

**第 3 步：写出新积分**。
$$
\int_0^4 \frac{1}{2 + \sqrt{x}}\,dx = \int_0^2 \frac{1}{2+t} \cdot 2t\,dt = 2\int_0^2 \frac{t}{2+t}\,dt
$$

**第 4 步：化简被积函数**。
$$
\frac{t}{2+t} = \frac{(t+2)-2}{t+2} = 1 - \frac{2}{t+2}
$$

**第 5 步：计算**。
$$
\begin{aligned}
2\int_0^2 \frac{t}{2+t}\,dt
&= 2\int_0^2 \left(1 - \frac{2}{t+2}\right)dt \\
&= 2\left[\int_0^2 1\,dt - 2\int_0^2 \frac{1}{t+2}\,dt\right]
\end{aligned}
$$

第一项：$\int_0^2 1\,dt = \big[t\big]_0^2 = 2$。

第二项：令$u = t+2$，$du = dt$，$t=0\to u=2$，$t=2\to u=4$：
$$\int_0^2 \frac{1}{t+2}\,dt = \int_2^4 \frac{1}{u}\,du = \big[\ln u\big]_2^4 = \ln 4 - \ln 2 = \ln\frac{4}{2} = \ln 2$$

代入：
$$
2\left[2 - 2\ln 2\right] = 4 - 4\ln 2
$$

因此 $\displaystyle \int_0^4 \frac{1}{2 + \sqrt{x}}\,dx = 4(1 - \ln 2)$。

---

**(2)** $\displaystyle \int_0^a \sqrt{a^2 - x^2}\,dx \quad (a > 0)$

**第 1 步：选择代换**。对于含$\sqrt{a^2 - x^2}$的积分，令$x = a\sin t$，$t \in [0, \pi/2]$。

**第 2 步：微分关系**。$dx = a\cos t\,dt$。

**第 3 步：换限**。$x=0 \to a\sin t = 0 \to t = 0$；$x=a \to a\sin t = a \to \sin t = 1 \to t = \pi/2$。

**第 4 步：写出新积分**。当$t \in [0, \pi/2]$时，$\cos t \geq 0$：
$$
\begin{aligned}
\int_0^a \sqrt{a^2 - x^2}\,dx
&= \int_0^{\pi/2} \sqrt{a^2 - a^2\sin^2 t} \cdot a\cos t\,dt \\
&= \int_0^{\pi/2} a\sqrt{1 - \sin^2 t} \cdot a\cos t\,dt \\
&= a^2 \int_0^{\pi/2} |\cos t| \cdot \cos t\,dt \\
&= a^2 \int_0^{\pi/2} \cos^2 t\,dt
\end{aligned}
$$

**第 5 步：计算**。利用倍角公式$\cos^2 t = \frac{1 + \cos 2t}{2}$：
$$
a^2 \int_0^{\pi/2} \cos^2 t\,dt = a^2 \int_0^{\pi/2} \frac{1 + \cos 2t}{2}\,dt = \frac{a^2}{2}\left[\int_0^{\pi/2} 1\,dt + \int_0^{\pi/2} \cos 2t\,dt\right]
$$

第一项：$\int_0^{\pi/2} 1\,dt = \frac{\pi}{2}$。

第二项：令$u = 2t$，$du = 2\,dt$，$t=0\to u=0$，$t=\pi/2 \to u=\pi$：
$$
\int_0^{\pi/2} \cos 2t\,dt = \frac{1}{2}\int_0^{\pi} \cos u\,du = \frac{1}{2}\big[\sin u\big]_0^{\pi} = \frac{1}{2}(0 - 0) = 0
$$

**第 6 步：合并结果**。
$$
\int_0^a \sqrt{a^2 - x^2}\,dx = \frac{a^2}{2} \cdot \frac{\pi}{2} = \frac{\pi a^2}{4}
$$

</details>

### 迁移应用

**3.** 计算 $\displaystyle \int_0^1 \frac{e^x}{(1+e^x)^2}\,dx$。

<details><summary>参考答案</summary>

**第 1 步：识别结构**。被积函数为$\dfrac{e^x}{(1+e^x)^2}$，而$(1+e^x)' = e^x$，因此形如$\dfrac{\varphi'(x)}{(\varphi(x))^2}$。

**第 2 步：换元**。令$u = 1 + e^x$，$du = e^x\,dx$。

**第 3 步：换限**。$x=0 \to u = 1 + e^0 = 2$，$x=1 \to u = 1 + e^1 = 1 + e$。

**第 4 步：计算**。
$$
\int_0^1 \frac{e^x}{(1+e^x)^2}\,dx = \int_2^{1+e} \frac{1}{u^2}\,du = \int_2^{1+e} u^{-2}\,du
$$

由幂函数积分公式$\int u^{-2}\,du = \frac{u^{-1}}{-1} = -\frac{1}{u}$：
$$
\big[-\frac{1}{u}\big]_2^{1+e} = \left(-\frac{1}{1+e}\right) - \left(-\frac{1}{2}\right) = -\frac{1}{1+e} + \frac{1}{2} = \frac{1}{2} - \frac{1}{1+e}
$$

因此 $\displaystyle \int_0^1 \frac{e^x}{(1+e^x)^2}\,dx = \frac{1}{2} - \frac{1}{1+e}$。

</details>

**4.** 计算 $\displaystyle \int_0^{\pi/2} \sin^2 x \cos x\,dx$。

<details><summary>参考答案</summary>

**第 1 步：识别结构**。被积函数为$\sin^2 x \cos x$。注意到$(\sin x)' = \cos x$，因此形如$(\sin x)^2 \cdot (\sin x)'$。

**第 2 步：换元**。令$u = \sin x$，$du = \cos x\,dx$。

**第 3 步：换限**。$x=0 \to u = \sin 0 = 0$，$x=\pi/2 \to u = \sin(\pi/2) = 1$。

**第 4 步：计算**。
$$
\int_0^{\pi/2} \sin^2 x \cos x\,dx = \int_0^1 u^2\,du = \left[\frac{u^3}{3}\right]_0^1 = \frac{1}{3}
$$

**5.** （综合应用）计算 $\displaystyle \int_1^2 \frac{\sqrt{x-1}}{x}\,dx$。

<details><summary>参考答案</summary>

**第 1 步：选择代换**。含根式$\sqrt{x-1}$，令$t = \sqrt{x-1}$（即$x = t^2 + 1$），则$t \geq 0$。

**第 2 步：微分关系**。$dx = 2t\,dt$。

**第 3 步：换限**。$x=1 \to t = \sqrt{1-1} = 0$，$x=2 \to t = \sqrt{2-1} = 1$。

**第 4 步：写出新积分**。
$$
\int_1^2 \frac{\sqrt{x-1}}{x}\,dx = \int_0^1 \frac{t}{t^2+1} \cdot 2t\,dt = 2\int_0^1 \frac{t^2}{t^2+1}\,dt
$$

**第 5 步：化简被积函数**。
$$
\frac{t^2}{t^2+1} = \frac{(t^2+1)-1}{t^2+1} = 1 - \frac{1}{t^2+1}
$$

**第 6 步：计算**。
$$
\begin{aligned}
2\int_0^1 \frac{t^2}{t^2+1}\,dt
&= 2\int_0^1 \left(1 - \frac{1}{t^2+1}\right)dt \\
&= 2\left[\int_0^1 1\,dt - \int_0^1 \frac{1}{t^2+1}\,dt\right] \\
&= 2\left[\big[t\big]_0^1 - \big[\arctan t\big]_0^1\right] \\
&= 2\left[(1-0) - (\arctan 1 - \arctan 0)\right] \\
&= 2\left(1 - \frac{\pi}{4}\right) = 2 - \frac{\pi}{2}
\end{aligned}
$$

因此 $\displaystyle \int_1^2 \frac{\sqrt{x-1}}{x}\,dx = 2 - \frac{\pi}{2}$。

</details>
