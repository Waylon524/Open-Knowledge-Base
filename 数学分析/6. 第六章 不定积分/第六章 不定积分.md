## 2. 背景与应用场景

在前四章中，我们建立了完整的微分学体系：从导数的极限定义出发，逐步掌握了四则运算法则、链式法则、隐函数求导、高阶导数，并利用导数研究了函数的单调性、极值、凹凸性等整体性质。微分学的核心问题是：**已知函数$f$，求其变化率$f'$**。

然而，数学与科学的许多实际场景中，我们面临的是相反的问题：

- **已知速度求位置**：自由落体运动中，已知瞬时速度$v(t) = gt$（$g$为重力加速度），求物体在时刻$t$的位置$s(t)$。我们已知$s'(t) = v(t) = gt$，那么$s(t)$是什么？显然$s(t) = \frac{1}{2}gt^2$是一个答案。但$s(t) = \frac{1}{2}gt^2 + 5$呢？它的导数也是$gt$！

- **已知加速度求速度**：若已知火箭的加速度$a(t)$，需要反推速度$v(t)$（满足$v'(t) = a(t)$）。

- **经济学中的成本恢复**：已知边际成本函数$C'(q)$（多生产一单位产品增加的成本），需要恢复总成本函数$C(q)$。

- **微分方程的反向求解**：许多自然规律以导数关系的形式给出（如牛顿冷却定律$\frac{dT}{dt} = -k(T - T_0)$），求解这类方程需要从导数反推出原函数。

所有这些问题的共同数学结构是：**已知导函数$f(x)$，求某个函数$F(x)$使得$F'(x) = f(x)$**。这个"逆运算"就是**不定积分**（indefinite integration），它与微分构成了一对互逆的运算。

本章将系统地研究不定积分的概念与计算方法。本文件作为第六章的开篇，建立最基础的概念框架：原函数的定义、不定积分的记号、积分常数的来源与含义、以及最基本的积分公式表。

## 3. 核心概念与符号约定

### 3.1 原函数的概念

**定义 6.1（原函数 / primitive function, antiderivative）**：设函数$f$定义在区间$I$上。若存在可导函数$F$，使得对一切$x \in I$有
$$F'(x) = f(x)$$
则称$F$为$f$在区间$I$上的一个**原函数**（或**反导数**）。

**符号说明**：原函数也称为"反导数"（antiderivative），因为它是求导运算的逆运算结果。

**例 1**：验证$F(x) = x^3$是$f(x) = 3x^2$在$\mathbb{R}$上的一个原函数。

**解**：直接求导：$F'(x) = (x^3)' = 3x^2 = f(x)$，对一切$x \in \mathbb{R}$成立。因此$F(x)=x^3$是$f(x)=3x^2$的一个原函数。

**例 2**：验证$G(x) = \frac{1}{4}x^4 - 2x$是$g(x) = x^3 - 2$的一个原函数。

**解**：$G'(x) = (\frac{1}{4}x^4 - 2x)' = \frac{1}{4} \cdot 4x^3 - 2 = x^3 - 2 = g(x)$。因此$G$是$g$的一个原函数。

**例 3**：验证$F(x) = x\sin x + \cos x$是$f(x) = x\cos x$的一个原函数。

**解**：利用乘积法则求导：
$$
\begin{aligned}
F'(x) &= (x\sin x)' + (\cos x)' \\
&= (1 \cdot \sin x + x \cdot \cos x) + (-\sin x) \\
&= \sin x + x\cos x - \sin x = x\cos x = f(x)
\end{aligned}
$$
因此$F$是$f$的一个原函数。

**关键观察**：原函数的定义是一个**验证性**定义——给定一个$F$，我们可以通过求导来验证它是否是$f$的原函数。但如何从$f$出发"找出"$F$，则是我们需要逐步学习的方法。

### 3.2 不定积分的概念

如果$F$是$f$的一个原函数，那么$F$**不是唯一的**。事实上，对任意常数$C$，$F(x) + C$也是$f$的原函数，因为$(F(x) + C)' = F'(x) + 0 = f(x)$。

这就引出了"全体原函数"的概念。

**定义 6.2（不定积分 / indefinite integral）**：设$F$是$f$在区间$I$上的一个原函数，则$f$的**全体原函数**（即所有形如$F(x) + C$的函数，其中$C$为任意常数）称为$f$的**不定积分**，记作
$$\int f(x)\,dx = F(x) + C$$
其中：
- $\int$称为**积分号**（integral sign）
- $f(x)$称为**被积函数**（integrand）
- $x$称为**积分变量**（integration variable）
- $dx$表示对变量$x$进行积分（与微分中的记号一脉相承）
- $C$称为**积分常数**（constant of integration），$C \in \mathbb{R}$

**读法**：$\int f(x)\,dx$读作"$f(x)$对$x$的不定积分"。

**核心含义**：不定积分不是单个函数，而是一族函数（函数族），这族函数中的每个成员都是$f$的原函数，任意两个成员之间相差一个常数。

**记号小结**：
$$\underbrace{\int}_{\text{积分号}} \underbrace{f(x)}_{\text{被积函数}} \underbrace{dx}_{\text{积分变量}} = \underbrace{F(x)}_{\text{一个原函数}} + \underbrace{C}_{\text{积分常数}}$$

### 3.3 积分常数的含义——原函数族结构定理

为什么不定积分表达式中必须包含$+C$？为什么任意两个原函数之间只相差一个常数？下面的定理回答了这个问题。

**定理 6.1（原函数族结构定理）**：设$F$和$G$都是$f$在区间$I$上的原函数（即对一切$x \in I$有$F'(x) = G'(x) = f(x)$）。则存在常数$C \in \mathbb{R}$，使得对一切$x \in I$有
$$F(x) - G(x) = C$$
或者说
$$F(x) = G(x) + C$$

**证明**：

**第 1 步：构造辅助函数**。

定义$H(x) = F(x) - G(x)$，$x \in I$。由于$F$和$G$都在$I$上可导，$H$也在$I$上可导。

**第 2 步：计算$H'(x)$**。

利用求导的减法法则（第四章文件02）：
$$H'(x) = F'(x) - G'(x) = f(x) - f(x) = 0, \quad \forall x \in I$$

**第 3 步：利用拉格朗日中值定理的推论**。

在第五章文件01（练习题第8题）中，我们利用拉格朗日中值定理证明了如下结论：

> **推论（导数恒为零则函数为常数）**：若函数在区间$I$上可导，且对一切$x \in I$有$H'(x) = 0$，则$H$在$I$上为常数。

简要回顾该推论的证明：任取$x_1, x_2 \in I$（设$x_1 < x_2$），对$H$在$[x_1, x_2]$上应用拉格朗日中值定理，存在$\xi \in (x_1, x_2)$使得
$$H(x_2) - H(x_1) = H'(\xi)(x_2 - x_1) = 0 \cdot (x_2 - x_1) = 0$$
因此$H(x_2) = H(x_1)$。由$x_1, x_2$的任意性，$H$在$I$上为常数。

由这个推论，存在常数$C$使得$H(x) = C$对一切$x \in I$成立。即：
$$F(x) - G(x) = C \quad \text{或等价地} \quad F(x) = G(x) + C$$

证毕。

**几何含义**：定理6.1告诉我们，$f$的所有原函数的图像可以通过沿$y$轴方向平移得到——任意两条原函数曲线之间的垂直距离处处等于同一个常数$C$。

``` 
y
^
|       F(x)+C₂
|      /         (不同的 C 对应不同的原函数)
|     /  F(x)+C₁
|    /  /
|   /  / F(x)
|  /  /
| /  /
+----------------> x
```

这就是不定积分表达式$F(x) + C$中"$+C$"的理论基础——它简洁地表达了全体原函数构成的函数族。

### 3.4 不定积分的几何意义

给定一个原函数$F(x)$，$f$的不定积分$\int f(x)\,dx = F(x) + C$表示一族曲线。这族曲线具有以下几何特征：

1. **相同的切线方向场**：在任意点$x_0$处，所有曲线$y = F(x) + C$（无论$C$取何值）的切线斜率都等于$F'(x_0) = f(x_0)$。因此，给定$x$坐标，族中所有曲线的切线方向相同。
2. **垂直平移关系**：任意两条曲线之间的垂直距离为常数（定理6.1的直接推论）。
3. **已知一点定一条**：如果额外给出函数在某点的函数值（初始条件），就可以从函数族中唯一确定一条特定的曲线（即确定$C$的值）。

### 3.5 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\int f(x)\,dx$ | $f(x)$的不定积分（全体原函数的集合） | $\int 2x\,dx = x^2 + C$ |
| $F(x)$ | $f$的一个具体的原函数 | 若$f(x)=2x$，则$F(x)=x^2$是其一 |
| $C$ | 积分常数（任意实数） | $C = 0, 5, -3, \dots$ |
| $f(x)$ | 被积函数 | $2x$, $\cos x$, $e^x$ |
| $dx$ | 积分变量标识 | 表示对$x$积分 |

**重要约定**：在本章中，所有不定积分均默认在函数有定义的区间上讨论。例如，$\int \frac{1}{x}\,dx$默认在$x>0$或$x<0$的区间上讨论。

## 4. 原理与方法

### 4.1 微分与积分的互逆关系

不定积分与微分构成一对互逆运算：

**从微分到积分**：若$F'(x) = f(x)$，则
$$\int f(x)\,dx = F(x) + C$$

**从积分到微分**：若$\int f(x)\,dx = F(x) + C$，则
$$\frac{d}{dx}\left[\int f(x)\,dx\right] = f(x)$$

第二个等式的证明：$\frac{d}{dx}[F(x) + C] = F'(x) = f(x)$。

**核心关系图**：
$$
\begin{array}{ccc}
f(x) & \xrightarrow{\text{求不定积分}} & F(x) + C \\[6pt]
& \xleftarrow[\text{求导}]{} &
\end{array}
$$

这类似于加法与减法、乘法与除法之间的互逆关系。求导是"正向"运算（已知函数求变化率），不定积分是"反向"运算（已知变化率求原函数）。

### 4.2 基本积分公式表

不定积分最基本的计算方法就是**逆用求导公式**：如果已知某个函数的导数为$f(x)$，那么$f(x)$的原函数就是这个函数（加上积分常数）。

以下基本积分公式全部来自我们已经掌握的求导公式，只需将求导公式"反过来读"。

**表 6.1：基本积分公式**

| 序号 | 求导公式（已学） | 对应积分公式 | 说明 |
|------|-----------------|-------------|------|
| 1 | $(x^{n+1})' = (n+1)x^n$ | $\displaystyle\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \;(n \neq -1)$ | 幂函数积分，$n \neq -1$ |
| 2 | $(\ln |x|)' = \dfrac{1}{x}$ | $\displaystyle\int \frac{1}{x}\,dx = \ln |x| + C \;(x \neq 0)$ | 幂函数$n=-1$的特殊情况 |
| 3 | $(e^x)' = e^x$ | $\displaystyle\int e^x\,dx = e^x + C$ | 指数函数 |
| 4 | $(\sin x)' = \cos x$ | $\displaystyle\int \cos x\,dx = \sin x + C$ | 余弦函数积分 |
| 5 | $(-\cos x)' = \sin x$ 或 $(\cos x)' = -\sin x$ | $\displaystyle\int \sin x\,dx = -\cos x + C$ | 正弦函数积分 |
| 6 | $(\arctan x)' = \dfrac{1}{1+x^2}$ | $\displaystyle\int \frac{1}{1+x^2}\,dx = \arctan x + C$ | 反三角函数 |

**公式1的推导（幂函数积分）**：由求导公式$(x^{n+1})' = (n+1)x^n$，两边同除以$(n+1)$得：
$$\left(\frac{x^{n+1}}{n+1}\right)' = x^n$$
因此$x^n$的一个原函数是$\frac{x^{n+1}}{n+1}$，于是
$$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

**为什么$n = -1$是特殊情况**：当$n = -1$时，公式1的分母$n+1 = 0$，公式失效。此时$x^{-1} = 1/x$的原函数是对数函数$\ln |x|$，因为$(\ln |x|)' = 1/x$（$x \neq 0$）。这就是公式2。

**公式5的推导（正弦函数积分）**：由$(\cos x)' = -\sin x$，两边乘以$-1$得$(-\cos x)' = \sin x$。因此$\sin x$的一个原函数是$-\cos x$，于是
$$\int \sin x\,dx = -\cos x + C$$

**验证每个公式的方法**：对积分结果右端求导，应得到被积函数。这是检验积分是否正确的最直接方法。例如：
$$\frac{d}{dx}\left(\frac{x^{n+1}}{n+1} + C\right) = \frac{(n+1)x^n}{n+1} = x^n \quad \checkmark$$

### 4.3 不定积分的线性性质

不定积分运算具有线性性质，这是积分计算中最重要的工具之一。

**定理 6.2（不定积分的线性性质）**：设$f$和$g$在区间$I$上都有原函数，$\alpha, \beta \in \mathbb{R}$为常数。则
$$\int [\alpha f(x) + \beta g(x)]\,dx = \alpha \int f(x)\,dx + \beta \int g(x)\,dx$$

该性质包含两个方面：

**(1) 齐次性（常数倍性质）**：
$$\int \alpha f(x)\,dx = \alpha \int f(x)\,dx \quad (\alpha \text{为常数})$$
即常数因子可以提到积分号外面。

**(2) 可加性（和差性质）**：
$$\int [f(x) \pm g(x)]\,dx = \int f(x)\,dx \pm \int g(x)\,dx$$
即函数和（差）的不定积分等于各自不定积分的和（差）。

**证明**：

我们只需要验证右端求导后等于左端的被积函数。

设$F$和$G$分别是$f$和$g$的原函数，即$F'(x) = f(x)$，$G'(x) = g(x)$。

令$H(x) = \alpha F(x) + \beta G(x)$。由求导的常数倍法则和和差法则：
$$H'(x) = \alpha F'(x) + \beta G'(x) = \alpha f(x) + \beta g(x)$$

因此$H$是$\alpha f + \beta g$的一个原函数，故
$$\int [\alpha f(x) + \beta g(x)]\,dx = H(x) + C = \alpha F(x) + \beta G(x) + C$$

由于$\int f(x)\,dx = F(x) + C_1$，$\int g(x)\,dx = G(x) + C_2$，适当合并积分常数后：
$$\alpha \int f(x)\,dx + \beta \int g(x)\,dx = \alpha(F(x)+C_1) + \beta(G(x)+C_2) = \alpha F(x) + \beta G(x) + (\alpha C_1 + \beta C_2)$$

由于$\alpha C_1 + \beta C_2$仍然是任意常数，记作$C$，因此等式成立。证毕。

**注**：证明中关键的一点是，两个积分常数$C_1, C_2$经过线性组合后仍然是一个任意常数——这正是$+C$写为一个常数而非多个常数的原因。

**线性性质的意义**：它允许我们将复杂被积函数拆分成简单部分分别积分再求和。几乎所有不定积分计算的第一步都涉及应用线性性质将积分拆分为基本积分公式可以直接处理的形式。

### 4.4 利用基本公式和线性性质求不定积分——标准流程

综合运用基本积分公式和线性性质，我们可以计算许多简单函数的不定积分。标准计算流程为：

**第 1 步：拆分** — 利用线性性质将积分拆分为若干基本积分之和（或差）

**第 2 步：逐个积分** — 对每个拆分后的积分应用基本积分公式

**第 3 步：合并** — 将所有积分结果相加，末尾写一个统一的积分常数$C$

**第 4 步：验证** — 对结果求导，检查是否得到原被积函数

下面通过§5中的例题展示这一流程的完整应用。

## 5. 例题

### 例 1：验证原函数并写出不定积分

**(1)** 验证$F(x) = \frac{1}{3}x^3 + \frac{1}{2}x^2$是$f(x) = x^2 + x$的一个原函数，并写出$\int (x^2 + x)\,dx$。

**(2)** 验证$G(x) = e^x - \cos x$是$g(x) = e^x + \sin x$的一个原函数，并写出$\int (e^x + \sin x)\,dx$。

**解 (1)**：

**第 1 步：求导验证**。
$$F'(x) = \left(\frac{1}{3}x^3 + \frac{1}{2}x^2\right)' = \frac{1}{3} \cdot 3x^2 + \frac{1}{2} \cdot 2x = x^2 + x = f(x)$$
因此$F$是$f$的一个原函数。$\checkmark$

**第 2 步：写出不定积分**。
由定义6.2，$f$的不定积分为一个原函数加上积分常数：
$$\int (x^2 + x)\,dx = F(x) + C = \frac{1}{3}x^3 + \frac{1}{2}x^2 + C$$

**第 3 步：验证**。
对结果求导：$\frac{d}{dx}\left(\frac{1}{3}x^3 + \frac{1}{2}x^2 + C\right) = x^2 + x$，与被积函数一致。$\checkmark$

**解 (2)**：

**第 1 步：求导验证**。
$$G'(x) = (e^x)' - (\cos x)' = e^x - (-\sin x) = e^x + \sin x = g(x)$$
因此$G$是$g$的一个原函数。$\checkmark$

**第 2 步：写出不定积分**。
$$\int (e^x + \sin x)\,dx = G(x) + C = e^x - \cos x + C$$

**第 3 步：验证**。
求导：$(e^x - \cos x + C)' = e^x + \sin x$，与被积函数一致。$\checkmark$

**注意**：验证原函数只需一次求导，积分结果是否正确也只需一次求导。求导是检验积分的黄金标准。

### 例 2：利用基本积分公式和线性性质计算不定积分

计算下列不定积分：

**(1)** $\displaystyle\int (4x^3 - 6x^2 + 2x - 3)\,dx$

**(2)** $\displaystyle\int \left(e^x - 3\cos x + \frac{2}{x}\right)dx \quad (x > 0)$

**(3)** $\displaystyle\int \left(\frac{2}{1+x^2} - \sin x\right)dx$

**解 (1)**：

**第 1 步：拆分**（利用线性性质）。
$$\int (4x^3 - 6x^2 + 2x - 3)\,dx = 4\int x^3\,dx - 6\int x^2\,dx + 2\int x\,dx - 3\int 1\,dx$$

**第 2 步：逐个积分**。
利用基本积分公式$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C$（$n \neq -1$）：
- $\int x^3\,dx = \frac{x^{4}}{4} + C_1$
- $\int x^2\,dx = \frac{x^{3}}{3} + C_2$
- $\int x\,dx = \frac{x^{2}}{2} + C_3$
- $\int 1\,dx = \int x^0\,dx = \frac{x^{1}}{1} + C_4 = x + C_4$

**第 3 步：合并**。
$$
\begin{aligned}
\int (4x^3 - 6x^2 + 2x - 3)\,dx
&= 4 \cdot \frac{x^{4}}{4} - 6 \cdot \frac{x^{3}}{3} + 2 \cdot \frac{x^{2}}{2} - 3x + C \\
&= x^4 - 2x^3 + x^2 - 3x + C
\end{aligned}
$$
其中$C = 4C_1 - 6C_2 + 2C_3 - 3C_4$，合并为一个任意常数。

**第 4 步：验证**。
$$(x^4 - 2x^3 + x^2 - 3x + C)' = 4x^3 - 6x^2 + 2x - 3$$
与被积函数一致。$\checkmark$

**解 (2)**（$x > 0$）：

**第 1 步：拆分**。
$$\int \left(e^x - 3\cos x + \frac{2}{x}\right)dx = \int e^x\,dx - 3\int \cos x\,dx + 2\int \frac{1}{x}\,dx$$

**第 2 步：逐个积分**。
- $\int e^x\,dx = e^x + C_1$（公式3）
- $\int \cos x\,dx = \sin x + C_2$（公式4）
- $\int \frac{1}{x}\,dx = \ln x + C_3$（$x > 0$时$|x| = x$，公式2）

**第 3 步：合并**。
$$\int \left(e^x - 3\cos x + \frac{2}{x}\right)dx = e^x - 3\sin x + 2\ln x + C$$

**第 4 步：验证**。
$$(e^x - 3\sin x + 2\ln x + C)' = e^x - 3\cos x + \frac{2}{x}$$
与被积函数一致。$\checkmark$

**解 (3)**：

**第 1 步：拆分**。
$$\int \left(\frac{2}{1+x^2} - \sin x\right)dx = 2\int \frac{1}{1+x^2}\,dx - \int \sin x\,dx$$

**第 2 步：逐个积分**。
- $\int \frac{1}{1+x^2}\,dx = \arctan x + C_1$（公式6）
- $\int \sin x\,dx = -\cos x + C_2$（公式5）

**第 3 步：合并**。
$$
\begin{aligned}
\int \left(\frac{2}{1+x^2} - \sin x\right)dx
&= 2\arctan x - (-\cos x) + C \\
&= 2\arctan x + \cos x + C
\end{aligned}
$$

**第 4 步：验证**。
$$(2\arctan x + \cos x + C)' = \frac{2}{1+x^2} - \sin x$$
与被积函数一致。$\checkmark$

### 例 3：利用初始条件确定积分常数

已知$F(x) = \frac{1}{4}x^4 + \frac{1}{3}x^3$是$f(x) = x^3 + x^2$的一个原函数。

(1) 写出$\int (x^3 + x^2)\,dx$的一般表达式。

(2) 若要求该不定积分在$x = 1$时的函数值为$2$，求满足条件的特定原函数。

**解 (1)**：

由$F'(x) = x^3 + x^2$，$F$是$f$的一个原函数。因此：
$$\int (x^3 + x^2)\,dx = F(x) + C = \frac{1}{4}x^4 + \frac{1}{3}x^3 + C$$
其中$C$为任意常数。

**解 (2)**：

**第 1 步：设定特定原函数**。
令$H(x) = \frac{1}{4}x^4 + \frac{1}{3}x^3 + C$，其中$C$待定。

**第 2 步：代入初始条件**。
条件为$H(1) = 2$，即：
$$H(1) = \frac{1}{4} \cdot 1^4 + \frac{1}{3} \cdot 1^3 + C = \frac{1}{4} + \frac{1}{3} + C = \frac{3}{12} + \frac{4}{12} + C = \frac{7}{12} + C = 2$$

**第 3 步：求解$C$**。
$$C = 2 - \frac{7}{12} = \frac{24}{12} - \frac{7}{12} = \frac{17}{12}$$

**第 4 步：写出特定原函数**。
$$H(x) = \frac{1}{4}x^4 + \frac{1}{3}x^3 + \frac{17}{12}$$

**验证**：$H'(x) = x^3 + x^2$，且$H(1) = \frac{7}{12} + \frac{17}{12} = \frac{24}{12} = 2$。$\checkmark$

**实际应用**：在物理问题中，初始条件通常对应"初始位置"或"初始速度"。例如，已知$v(t) = gt$（速度），且$s(0) = s_0$（初始位置），则$s(t) = \frac{1}{2}gt^2 + s_0$——积分常数$C = s_0$由初始条件确定。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| $\int f(x)\,dx$ 表示一个具体的函数（即$f$的某个原函数） | $\int f(x)\,dx$ 表示$f$的**全体原函数**构成的函数族，而不是单个函数。因此必须写$+C$，不能省略 |
| $F(x)$是$f(x)$的原函数$\iff F'(x) = f(x)$，所以原函数和$f$是一回事 | 原函数与$f$的关系是**导数关系**：$F'(x) = f(x)$。$F$和$f$是两个不同的函数，$f$是$F$的导函数，$F$是$f$的原函数。例如$f(x)=2x$，$F(x)=x^2$，二者不同但$F'=f$ |
| 积分常数$C$只是一个形式记号，可以省略不写 | $C$有严格的理论基础——定理6.1（拉格朗日中值定理推论）保证了任意两原函数差为常数。没有$+C$，表达式就是不完备的。在后续定积分等应用中，$C$的缺失会导致严重错误 |
| $\int \frac{1}{x}\,dx = \ln x + C$ 对所有$x$成立 | 严格来说，$\int \frac{1}{x}\,dx = \ln |x| + C$（$x \neq 0$）。当$x > 0$时可写为$\ln x + C$，但当$x < 0$时必须写为$\ln(-x) + C$，统一用$\ln |x| + C$更准确 |
| 不定积分的线性性质说$\int f(x)g(x)\,dx = \int f(x)\,dx \cdot \int g(x)\,dx$ | 线性性质只对**线性组合**（常数倍与加减法）成立。乘积的积分不等于积分的乘积——$\int f(x)g(x)\,dx \neq \left(\int f(x)\,dx\right)\left(\int g(x)\,dx\right)$。求乘积的积分需要后续学习分部积分法等技巧 |
| $\int [\alpha f(x) + \beta g(x)]\,dx = \alpha \int f(x)\,dx + \beta \int g(x)\,dx$ 中的$C$可以分别写在每个积分后面 | 合并后的结果应该只写**一个**积分常数$C$。$\alpha F(x) + \beta G(x) + C_1 + C_2$中的$C_1 + C_2$仍是一个任意常数，合并记作$C$即可。写多个常数不仅多余，而且容易引起混淆 |
| 验证原函数不需要求导，直接看形式是否相似即可 | 验证原函数的**唯一方法**是求导：检查$F'(x)$是否等于$f(x)$。例如$F(x)=\frac{1}{2}x^2$和$G(x)=\frac{1}{2}x^2+5$都是$f(x)=x$的原函数——只有通过求导才能确认 |

**1.** 验证下列函数是否为指定函数的原函数：

(1) $F(x) = x^3 - 2x^2 + 5$ 是否为 $f(x) = 3x^2 - 4x$ 的一个原函数？

(2) $G(x) = \sin x + x\cos x$ 是否为 $g(x) = 2\cos x - x\sin x$ 的一个原函数？

(3) $H(x) = \frac{1}{2}e^{2x}$ 是否为 $h(x) = e^{2x}$ 的一个原函数？

<details><summary>参考答案</summary>

**(1)** $F'(x) = (x^3 - 2x^2 + 5)' = 3x^2 - 4x + 0 = 3x^2 - 4x = f(x)$。因此 $F$ 是 $f$ 的一个原函数。$\checkmark$

**(2)** $G'(x) = (\sin x)' + (x\cos x)'$。先对 $x\cos x$ 运用乘积法则：
$$(x\cos x)' = 1 \cdot \cos x + x \cdot (-\sin x) = \cos x - x\sin x$$
因此：
$$G'(x) = \cos x + (\cos x - x\sin x) = 2\cos x - x\sin x$$
而 $g(x) = 2\cos x - x\sin x$，所以 $G'(x) = g(x)$。因此 $G$ 是 $g$ 的一个原函数。$\checkmark$

**(3)** $H'(x) = \frac{1}{2} \cdot e^{2x} \cdot 2 = e^{2x}$（链式法则）。$h(x) = e^{2x}$，因此 $H'(x) = h(x)$。所以 $H$ 是 $h$ 的一个原函数。$\checkmark$

**3.** 利用线性性质和基本积分公式计算下列不定积分：

(1) $\displaystyle\int (x^3 - 2x + 4)\,dx$

(2) $\displaystyle\int \left(3\sin x + \frac{1}{1+x^2}\right)dx$

(3) $\displaystyle\int \left(2e^x + \frac{3}{x}\right)dx \quad (x > 0)$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int (x^3 - 2x + 4)\,dx$

**第 1 步：拆分**。
$$\int (x^3 - 2x + 4)\,dx = \int x^3\,dx - 2\int x\,dx + 4\int 1\,dx$$

**第 2 步：逐个积分**。
$$\int x^3\,dx = \frac{x^4}{4} + C_1,\quad \int x\,dx = \frac{x^2}{2} + C_2,\quad \int 1\,dx = x + C_3$$

**第 3 步：合并**。
$$\frac{x^4}{4} - 2 \cdot \frac{x^2}{2} + 4 \cdot x + C = \frac{x^4}{4} - x^2 + 4x + C$$

**验证**：$\left(\frac{x^4}{4} - x^2 + 4x + C\right)' = x^3 - 2x + 4$。$\checkmark$

---

**(2)** $\displaystyle\int \left(3\sin x + \frac{1}{1+x^2}\right)dx$

**第 1 步：拆分**。
$$\int \left(3\sin x + \frac{1}{1+x^2}\right)dx = 3\int \sin x\,dx + \int \frac{1}{1+x^2}\,dx$$

**第 2 步：逐个积分**。
$$\int \sin x\,dx = -\cos x + C_1,\quad \int \frac{1}{1+x^2}\,dx = \arctan x + C_2$$

**第 3 步：合并**。
$$3(-\cos x) + \arctan x + C = -3\cos x + \arctan x + C$$

**验证**：$(-3\cos x + \arctan x + C)' = 3\sin x + \frac{1}{1+x^2}$。$\checkmark$

---

**(3)** $\displaystyle\int \left(2e^x + \frac{3}{x}\right)dx \quad (x > 0)$

**第 1 步：拆分**。
$$\int \left(2e^x + \frac{3}{x}\right)dx = 2\int e^x\,dx + 3\int \frac{1}{x}\,dx$$

**第 2 步：逐个积分**。
$$\int e^x\,dx = e^x + C_1,\quad \int \frac{1}{x}\,dx = \ln x + C_2 \;(x > 0)$$

**第 3 步：合并**。
$$2e^x + 3\ln x + C$$

**验证**：$(2e^x + 3\ln x + C)' = 2e^x + \frac{3}{x}$。$\checkmark$

(2) 若要求该不定积分在 $x = e$（$e$ 为自然常数）时函数值为 $1$，求满足条件的特定原函数。

**(2)** 设 $H(x) = x\ln x - x + C$。由条件 $H(e) = 1$：

**5.** （物理应用）已知某物体做直线运动的速度函数为$v(t) = 4t^3 - 2t$（$t \geq 0$），且已知$t = 0$时物体的位置$s(0) = 3$。求位置函数$s(t)$。

（提示：$s'(t) = v(t)$，因此$s(t) = \int v(t)\,dt$，再利用初始条件确定积分常数。）

<details><summary>参考答案</summary>

**第 1 步：写出位置函数的积分形式**。

由$s'(t) = v(t) = 4t^3 - 2t$，所以$s(t)$是$v(t)$的一个原函数：
$$s(t) = \int (4t^3 - 2t)\,dt$$

**第 2 步：计算不定积分**。
$$
\begin{aligned}
s(t) &= \int 4t^3\,dt - \int 2t\,dt \\
&= 4\int t^3\,dt - 2\int t\,dt \\
&= 4 \cdot \frac{t^4}{4} - 2 \cdot \frac{t^2}{2} + C \\
&= t^4 - t^2 + C
\end{aligned}
$$

**第 3 步：利用初始条件确定$C$**。

由$s(0) = 3$：
$$s(0) = 0^4 - 0^2 + C = C = 3$$
所以$C = 3$。

**第 4 步：写出位置函数**。
$$s(t) = t^4 - t^2 + 3$$

**验证**：$s'(t) = 4t^3 - 2t = v(t)$，且$s(0) = 3$。$\checkmark$

**物理意义**：$C = s(0) = 3$是物体的初始位置（$t=0$时的位置）。

</details>

**6.** （思考题）判断下列说法是否正确，若不正确请给出反例或解释原因。

(1) 若$F'(x) = f(x)$且$G'(x) = f(x)$，则$F(x) = G(x)$。

(2) $\int 0\,dx = 0$。

(3) $\int [f(x) \cdot g(x)]\,dx = \left(\int f(x)\,dx\right) \cdot \left(\int g(x)\,dx\right)$。

(4) 若$F(x)$是$f(x)$的原函数，则$F(x) + 5$也是$f(x)$的原函数。

<details><summary>参考答案</summary>

**(1)** 不正确。由定理6.1，$F(x)$和$G(x)$相差一个常数：$F(x) = G(x) + C$。除非$C=0$（即有额外条件如$F(a)=G(a)$），否则$F$和$G$不相等。反例：$f(x) = 2x$，$F(x) = x^2$，$G(x) = x^2 + 3$，$F'(x) = G'(x) = 2x$但$F(x) \neq G(x)$。

---

**(2)** 不正确。$\int 0\,dx = C$（常数），而不是$0$。因为常数的导数为$0$，所以任意常数$C$都是$0$的原函数。$0$只是$C=0$时的特例。完整的表达式必须包含$+C$。

**注**：这个陷阱看似简单，但恰恰体现了"全体原函数"的概念——$0$的原函数是全体常数函数$C$，而不是单个函数$0$。

---

**(3)** 不正确。这是最常见的误解之一。积分的线性性质只适用于**线性组合**（常数倍与和差），不适用于乘积。反例：取$f(x) = x$，$g(x) = x$。左边$\int x \cdot x\,dx = \int x^2\,dx = \frac{x^3}{3} + C$。右边$\left(\int x\,dx\right) \cdot \left(\int x\,dx\right) = \left(\frac{x^2}{2} + C_1\right)\left(\frac{x^2}{2} + C_2\right) = \frac{x^4}{4} + \cdots$，显然不等于左边。

---

**(4)** 正确。因为$(F(x) + 5)' = F'(x) + 0 = f(x)$。事实上，对任意常数$C$，$F(x) + C$都是$f(x)$的原函数——这正是"一族原函数"的含义。

</details>

## 2. 背景与应用场景

到目前为止，我们掌握了不定积分的基本概念和线性性质，可以利用基本积分公式直接积分。但许多常见函数形式（如 $x e^{2x}$、$x\cos x$、$\ln x$）并不能直接套用基本积分公式，因为它们表现为**两个不同类型函数的乘积**。

基本积分公式只告诉我们单个函数（如 $x^n$、$e^x$、$\sin x$）的原函数。如何对**乘积形式**的被积函数求积分？这使人联想起求导法则中也有类似的问题——在第四章文件 02 中，我们遇到了同样的困境：如何求两个函数乘积的导数？乘积法则 $(\fg)' = f'g + fg'$ 给出了答案。

分部积分法（Integration by Parts）正是乘积法则在不定积分中的对偶形式。它通过将**一个复杂的积分**转化为**另一形式的积分**，逐步降低难度。标准策略是：将难以直接积分的部分转移到 $du$ 侧，使新的积分 $\int v\,du$ 比原积分 $\int u\,dv$ 更容易处理。

分部积分法在物理、工程和概率统计中有广泛应用：

- **概率论**：计算指数分布、Gamma 分布的矩（原点矩 $E[X^n]$）需要反复使用分部积分
- **信号处理**：傅里叶变换中计算 $\int t e^{-i\omega t}\,dt$ 类的积分
- **物理学**：薛定谔方程中的期望值计算、电动力学中的势函数积分

## 3. 核心概念与符号约定

### 3.1 分部积分公式的推导

分部积分法是乘积求导法则的逆用。

**定理 6.3（分部积分公式 / Integration by Parts）**：设 $u = u(x)$ 和 $v = v(x)$ 均为可导函数，且 $u'(x)v(x)$ 存在原函数，则

$$
\int u(x) v'(x)\,dx = u(x) v(x) - \int u'(x) v(x)\,dx
$$

等价地，用微分形式可简洁地写为：

$$
\boxed{\;\int u\,dv = uv - \int v\,du\;}
$$

其中 $du = u'(x)\,dx$，$dv = v'(x)\,dx$。

**证明**：

**第 1 步：从乘积法则出发**。

由第四章定理 4.6（乘积法则），对可导函数 $u(x)$ 和 $v(x)$ 有：
$$\frac{d}{dx}[u(x)v(x)] = u'(x)v(x) + u(x)v'(x)$$

**第 2 步：两边对 $x$ 求不定积分**。

对等式两边同时求不定积分：
$$\int \frac{d}{dx}[u(x)v(x)]\,dx = \int [u'(x)v(x) + u(x)v'(x)]\,dx$$

**第 3 步：利用不定积分的定义简化左端**。

由不定积分的定义（定义 6.2），$\int \frac{d}{dx}[u(x)v(x)]\,dx$ 就是 $u(x)v(x)$ 的全体原函数：
$$\int \frac{d}{dx}[u(x)v(x)]\,dx = u(x)v(x) + C$$

因此：
$$u(x)v(x) + C = \int u'(x)v(x)\,dx + \int u(x)v'(x)\,dx$$

**第 4 步：移项整理**。

将 $\int u'(x)v(x)\,dx$ 移到等号左侧：
$$\int u(x)v'(x)\,dx = u(x)v(x) + C - \int u'(x)v(x)\,dx$$

由于常数 $C$ 可以与右侧积分中的积分常数合并，我们通常将 $C$ 吸收到最终的积分常数中，写作：
$$\int u(x)v'(x)\,dx = u(x)v(x) - \int u'(x)v(x)\,dx$$

**第 5 步：微分形式**。

令 $u = u(x)$，$v = v(x)$，则 $du = u'(x)dx$，$dv = v'(x)dx$。代入上式得到分部积分公式的简洁形式：
$$\int u\,dv = uv - \int v\,du$$

证毕。

**核心思想**：分部积分法将一个积分 $\int u\,dv$ 转换为另一个积分 $\int v\,du$。技巧在于选择 $u$ 和 $dv$，使得 $\int v\,du$ 比 $\int u\,dv$ 更容易计算。

### 3.2 符号表

| 符号 | 含义 | 说明 |
|------|------|------|
| $\int u\,dv$ | 原积分，需计算 | 选定的 $u$ 和 $dv$ 构成被积函数 |
| $uv$ | $u$ 和 $v$ 的乘积 | 分部积分公式第一项（代数部分） |
| $-\int v\,du$ | 转换后的新积分 | 希望比原积分更简单 |
| $u$ | 选为**求导**的函数 | 通常选择求导后变简单的函数 |
| $dv$ | 选为**积分**的部分 | 需要能直接积分得到 $v$ |
| $du$ | $u$ 的微分，$du = u'\,dx$ | 求导结果 |
| $v$ | $dv$ 的一个原函数 | 对 $dv$ 积分得到，一般取积分常数为 0 |

## 4. 原理与方法

### 4.1 $u$ 与 $dv$ 的选择策略——LIATE 法则

分部积分法成功的关键在于**正确选择 $u$ 和 $dv$**。一般的指导原则是：

1. $u$ 应该选为**求导后形式变简单**的函数
2. $dv$ 应该选为**容易积分**的部分

**LIATE 法则**是一个常用的选择优先级规则。优先级从高到低排列：

| 优先级 | 字母 | 含义 | 示例 | 作为 $u$ 的原因 |
|--------|------|------|------|-----------------|
| 高 | **L** | Logarithmic（对数函数） | $\ln x$, $\log_a x$ | 求导后变为有理函数（$1/x$），形式简化 |
| ↑ | **I** | Inverse trigonometric（反三角函数） | $\arctan x$, $\arcsin x$ | 求导后变为有理函数或代数函数 |
| | **A** | Algebraic（代数函数/多项式） | $x^n$, $x^2+1$ | 求导后次数降低 |
| ↓ | **T** | Trigonometric（三角函数） | $\sin x$, $\cos x$ | 积分后仍在 $\sin$、$\cos$ 间循环 |
| 低 | **E** | Exponential（指数函数） | $e^{ax}$, $a^x$ | 积分后仍为指数函数（形式不变） |

**选择规则**：在 LIATE 中**越靠左的**函数，越应该选为 $u$（求导对象）；越靠右的，越应该选为 $dv$（积分对象）。

**原理分析**：

- 对数函数（L）和反三角函数（I）求导后形式大幅简化（$\ln x \to 1/x$，$\arctan x \to 1/(1+x^2)$）。因此设 $u$ = L 或 I 非常有利。
- 代数函数（A）求导后次数降低，有助于简化。对 $x^n$ 反复使用分部积分可逐步降次至常数，因此多项式适合充当 $u$。
- 三角函数（T）和指数函数（E）无论求导还是积分都保持同类形式（$\sin \leftrightarrow \cos$，$e^{ax} \to e^{ax}$），因此适合放在 $dv$ 侧。

**反例警示**：如果无视 LIATE 法则选反了 $u$ 和 $dv$，新积分 $\int v\,du$ 可能比原积分 $\int u\,dv$ 更复杂。例如对 $\int x e^{2x}\,dx$，若错误地设 $u = e^{2x}$、$dv = x\,dx$，则 $v = x^2/2$，新积分变为 $\int (x^2/2) \cdot 2e^{2x}\,dx = \int x^2 e^{2x}\,dx$ ——次数反而升高了！正确的选择是按 LIATE：A 在 E 之前，所以设 $u = x$（代数函数），$dv = e^{2x}dx$（指数函数）。

### 4.2 标准应用模式

根据被积函数的结构形式，分部积分有以下四种标准应用模式。

---

#### 4.2.1 模式一：多项式 $\times$ 指数/三角函数

**形式**：$\displaystyle\int x^n \cdot e^{kx}\,dx$ 或 $\displaystyle\int x^n \cdot \sin(kx)\,dx$、$\displaystyle\int x^n \cdot \cos(kx)\,dx$

**策略**：按 LIATE，多项式（A）在指数/三角函数（E/T）之前，因此设 $u = x^n$（多项式），$dv = e^{kx}dx$ 或 $\sin(kx)dx$、$\cos(kx)dx$。

**示例框架**（以 $\int x e^{2x}\,dx$ 为例）：

设 $u = x$，$dv = e^{2x}\,dx$，则：
$$du = 1 \cdot dx = dx, \qquad v = \int e^{2x}\,dx = \frac{1}{2}e^{2x}$$

代入公式：
$$\int x e^{2x}\,dx = x \cdot \frac{1}{2}e^{2x} - \int \frac{1}{2}e^{2x} \cdot dx = \frac{1}{2}x e^{2x} - \frac{1}{2}\int e^{2x}\,dx = \frac{1}{2}x e^{2x} - \frac{1}{2} \cdot \frac{1}{2}e^{2x} + C$$
$$= \frac{1}{2}x e^{2x} - \frac{1}{4}e^{2x} + C$$

**关键观察**：多项式的次数在每次分部积分后降低一次。如果 $u = x^n$，应用一次分部积分后，新积分变为 $\int x^{n-1} e^{kx}\,dx$（低一次），重复 $n$ 次后问题归约为 $\int e^{kx}\,dx$。

---

#### 4.2.2 模式二：单函数型

**形式**：$\displaystyle\int \ln x\,dx$、$\displaystyle\int \arctan x\,dx$、$\displaystyle\int \arcsin x\,dx$

**策略**：被积函数表面上只有一个函数，无法直接积分。此时**引入因子 $1$**，将 $1$ 视为 $dv$，将单一函数本身视为 $u$。即：设 $u = \ln x$（或 $\arctan x$ 等），$dv = dx$。

**示例**（$\int \ln x\,dx$）：

设 $u = \ln x$，$dv = dx$，则：
$$du = \frac{1}{x}\,dx, \qquad v = \int 1\,dx = x$$

代入公式：
$$
\begin{aligned}
\int \ln x\,dx &= (\ln x) \cdot x - \int x \cdot \frac{1}{x}\,dx \\
&= x\ln x - \int 1\,dx \\
&= x\ln x - x + C
\end{aligned}
$$

**示例**（$\int \arctan x\,dx$）：

设 $u = \arctan x$，$dv = dx$，则：
$$du = \frac{1}{1+x^2}\,dx, \qquad v = x$$

$$
\begin{aligned}
\int \arctan x\,dx &= x\arctan x - \int x \cdot \frac{1}{1+x^2}\,dx \\
&= x\arctan x - \frac{1}{2}\int \frac{2x}{1+x^2}\,dx \\
&= x\arctan x - \frac{1}{2}\ln(1+x^2) + C
\end{aligned}
$$

其中 $\int \frac{2x}{1+x^2}\,dx = \ln(1+x^2) + C$，因为 $\frac{d}{dx}\ln(1+x^2) = \frac{2x}{1+x^2}$。

**通用策略总结**：对于单一函数 $f(x)$，无法直接积分时，设 $u = f(x)$，$dv = dx$，则：
$$\int f(x)\,dx = x f(x) - \int x f'(x)\,dx$$
这通常将一个无法直接求解的问题转化为一个可能简化的积分。

---

#### 4.2.3 模式三：循环型

**形式**：$\displaystyle\int e^{ax}\sin(bx)\,dx$ 或 $\displaystyle\int e^{ax}\cos(bx)\,dx$

**特点**：指数函数和三角函数（或两个三角函数）的乘积。无论对谁求导或积分，结果仍在指数和三角函数之间循环，不会简单消失。

**策略**：连续使用两次分部积分，且**两次选择 $u$ 的类型保持一致**（例如两次都选三角函数为 $u$），得到原积分 $I$ 的方程，然后解方程求出 $I$。

**示例**（$\int e^x \sin x\,dx$）：

记 $I = \int e^x \sin x\,dx$。

**第一次分部积分**：按 LIATE，三角函数（T）在指数函数（E）之前，设 $u = \sin x$，$dv = e^x\,dx$，则：
$$du = \cos x\,dx, \qquad v = e^x$$

$$
I = e^x \sin x - \int e^x \cos x\,dx \tag{1}
$$

**第二次分部积分**：对 $\int e^x \cos x\,dx$ 再次使用分部积分。保持选择一致——仍然选三角部分为 $u$，即设 $U = \cos x$，$dV = e^x\,dx$，则：
$$dU = -\sin x\,dx, \qquad V = e^x$$

$$
\int e^x \cos x\,dx = e^x \cos x - \int e^x(-\sin x)\,dx = e^x \cos x + \int e^x \sin x\,dx = e^x \cos x + I \tag{2}
$$

**回环求解**：将 (2) 代入 (1)：
$$
\begin{aligned}
I &= e^x \sin x - (e^x \cos x + I) \\
I &= e^x \sin x - e^x \cos x - I \\
2I &= e^x (\sin x - \cos x) \\
I &= \frac{e^x}{2}(\sin x - \cos x) + C
\end{aligned}
$$

**另一种选择**：也可以第一次设 $u = e^x$、$dv = \sin x\,dx$，第二次依然设 $u = e^x$、$dv = \cos x\,dx$（保持指数函数为 $u$）。结果一致。

**警告**：两次分部积分的选择**必须一致**（要么都选三角为 $u$，要么都选指数为 $u$）。如果第一次 $u = \sin x$、第二次 $u = e^x$，会退化为原积分，无法得到方程。

---

#### 4.2.4 模式四：递推/约化公式

当被积函数含有一个参数 $n$（通常是幂次）时，分部积分可以将 $I_n$ 表示为 $I_{n-1}$（或 $I_{n-2}$）的函数，从而建立递推关系。反复使用这个递推关系，最终归约到已知的基本积分。

**约化公式 1：$I_n = \int x^n e^{kx}\,dx$（$n$ 为非负整数）**

设 $u = x^n$，$dv = e^{kx}\,dx$，则 $du = n x^{n-1}\,dx$，$v = \dfrac{e^{kx}}{k}$（$k \neq 0$）。

代入分部积分公式：
$$
\begin{aligned}
I_n &= \int x^n e^{kx}\,dx \\
&= x^n \cdot \frac{e^{kx}}{k} - \int \frac{e^{kx}}{k} \cdot n x^{n-1}\,dx \\
&= \frac{x^n e^{kx}}{k} - \frac{n}{k} \int x^{n-1} e^{kx}\,dx \\
&= \frac{x^n e^{kx}}{k} - \frac{n}{k} I_{n-1}
\end{aligned}
$$

**递推公式**：
$$\boxed{I_n = \frac{x^n e^{kx}}{k} - \frac{n}{k} I_{n-1}}$$

**底情形**：$I_0 = \int e^{kx}\,dx = \dfrac{e^{kx}}{k} + C$。

**应用示例**：计算 $I_2 = \int x^2 e^{x}\,dx$（即 $k=1$）。

$$
\begin{aligned}
I_0 &= e^x + C \\
I_1 &= \frac{x e^{x}}{1} - \frac{1}{1} I_0 = x e^x - e^x + C_1 \\
I_2 &= \frac{x^2 e^{x}}{1} - \frac{2}{1} I_1 = x^2 e^x - 2(x e^x - e^x) + C \\
&= x^2 e^x - 2x e^x + 2e^x + C \\
&= e^x (x^2 - 2x + 2) + C
\end{aligned}
$$

**约化公式 2：$J_n = \int (\ln x)^n\,dx$（$n$ 为非负整数）**

设 $u = (\ln x)^n$，$dv = dx$，则 $du = n (\ln x)^{n-1} \cdot \frac{1}{x}\,dx$，$v = x$。

代入分部积分公式：
$$
\begin{aligned}
J_n &= \int (\ln x)^n\,dx \\
&= (\ln x)^n \cdot x - \int x \cdot n (\ln x)^{n-1} \cdot \frac{1}{x}\,dx \\
&= x (\ln x)^n - n \int (\ln x)^{n-1}\,dx \\
&= x (\ln x)^n - n J_{n-1}
\end{aligned}
$$

**递推公式**：
$$\boxed{J_n = x (\ln x)^n - n J_{n-1}}$$

**底情形**：$J_0 = \int 1\,dx = x + C$。

**应用示例**：计算 $J_2 = \int (\ln x)^2\,dx$。

$$
\begin{aligned}
J_0 &= x + C \\
J_1 &= x (\ln x)^1 - 1 \cdot J_0 = x\ln x - x + C_1 \\
J_2 &= x (\ln x)^2 - 2 J_1 = x (\ln x)^2 - 2(x\ln x - x) + C \\
&= x (\ln x)^2 - 2x\ln x + 2x + C
\end{aligned}
$$

**验证**：对结果求导：
$$\frac{d}{dx}[x (\ln x)^2 - 2x\ln x + 2x + C] = (\ln x)^2 + 2\ln x - 2\ln x - 2 + 2 = (\ln x)^2$$
与被积函数一致。$\checkmark$

**递推/约化公式的价值**：它将 $n$ 次问题转化为 $n-1$ 次问题，反复 $n$ 步后归约到已知积分。这种方法避免了每次重新计算分部积分的重复劳动，在计算 $n$ 较大时（如 $\int x^{10} e^x\,dx$）尤为高效。

## 5. 例题

### 例 1：多项式 $\times$ 指数型（模式一）

计算 $\displaystyle\int x e^{2x}\,dx$。

**解**：

**第 1 步：选择 $u$ 和 $dv$**。

被积函数为 $x$（代数函数，A）乘以 $e^{2x}$（指数函数，E）。按 LIATE，A 在 E 之前，因此设：
$$u = x, \qquad dv = e^{2x}\,dx$$

**第 2 步：计算 $du$ 和 $v$**。

$$du = 1 \cdot dx = dx, \qquad v = \int e^{2x}\,dx = \frac{1}{2}e^{2x}$$

（求 $v$ 时取积分常数为 0——我们需要 $dv$ 的**一个**原函数即可。）

**第 3 步：代入分部积分公式**。

$$
\begin{aligned}
\int x e^{2x}\,dx &= x \cdot \frac{1}{2}e^{2x} - \int \frac{1}{2}e^{2x}\,dx \\
&= \frac{1}{2}x e^{2x} - \frac{1}{2} \int e^{2x}\,dx \\
&= \frac{1}{2}x e^{2x} - \frac{1}{2} \cdot \frac{1}{2}e^{2x} + C \\
&= \frac{1}{2}x e^{2x} - \frac{1}{4}e^{2x} + C
\end{aligned}
$$

**第 4 步：验证**。

对结果求导：
$$
\begin{aligned}
\frac{d}{dx}\left(\frac{1}{2}x e^{2x} - \frac{1}{4}e^{2x} + C\right) &= \frac{1}{2}e^{2x} + \frac{1}{2}x \cdot 2e^{2x} - \frac{1}{4} \cdot 2e^{2x} \\
&= \frac{1}{2}e^{2x} + x e^{2x} - \frac{1}{2}e^{2x} = x e^{2x}
\end{aligned}
$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int x e^{2x}\,dx = \frac{1}{2}x e^{2x} - \frac{1}{4}e^{2x} + C = \frac{e^{2x}}{4}(2x - 1) + C$。

---

### 例 2：多项式 $\times$ 三角型（模式一）

计算 $\displaystyle\int x \cos x\,dx$。

**解**：

**第 1 步：选择 $u$ 和 $dv$**。

被积函数为 $x$（代数函数，A）乘以 $\cos x$（三角函数，T）。LIATE 中 A 在 T 之前，因此设：
$$u = x, \qquad dv = \cos x\,dx$$

**第 2 步：计算 $du$ 和 $v$**。

$$du = dx, \qquad v = \int \cos x\,dx = \sin x$$

**第 3 步：代入分部积分公式**。

$$
\begin{aligned}
\int x \cos x\,dx &= x \cdot \sin x - \int \sin x \cdot dx \\
&= x\sin x - \int \sin x\,dx \\
&= x\sin x - (-\cos x) + C \\
&= x\sin x + \cos x + C
\end{aligned}
$$

**第 4 步：验证**。

$$\frac{d}{dx}(x\sin x + \cos x + C) = \sin x + x\cos x - \sin x = x\cos x$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int x \cos x\,dx = x\sin x + \cos x + C$。

---

### 例 3：单函数型（模式二）

计算 $\displaystyle\int \ln x\,dx \quad (x > 0)$。

**解**：

**第 1 步：识别类型，选择 $u$ 和 $dv$**。

被积函数只有一个 $\ln x$，无法直接积分。引入因子 $1$，将 $\ln x$ 视为 $u$，$1$ 视为 $dv$：
$$u = \ln x, \qquad dv = 1 \cdot dx = dx$$

**第 2 步：计算 $du$ 和 $v$**。

$$du = \frac{1}{x}\,dx, \qquad v = \int 1\,dx = x$$

**第 3 步：代入分部积分公式**。

$$
\begin{aligned}
\int \ln x\,dx &= (\ln x) \cdot x - \int x \cdot \frac{1}{x}\,dx \\
&= x\ln x - \int 1\,dx \\
&= x\ln x - x + C
\end{aligned}
$$

**第 4 步：验证**。

$$\frac{d}{dx}(x\ln x - x + C) = \ln x + x \cdot \frac{1}{x} - 1 = \ln x + 1 - 1 = \ln x$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \ln x\,dx = x\ln x - x + C$。

---

### 例 4：循环型（模式三）

计算 $\displaystyle\int e^{x} \sin x\,dx$。

**解**：

**第 1 步：第一次分部积分**。

设 $u = \sin x$，$dv = e^x\,dx$（按 LIATE，三角函数在指数函数之前）。则：
$$du = \cos x\,dx, \qquad v = e^x$$

记 $I = \int e^x \sin x\,dx$，则：
$$I = e^x \sin x - \int e^x \cos x\,dx \tag{1}$$

**第 2 步：第二次分部积分**。

对 $\int e^x \cos x\,dx$ 再次使用分部积分。保持选择一致——仍然选三角函数为 $u$：
设 $U = \cos x$，$dV = e^x\,dx$，则：
$$dU = -\sin x\,dx, \qquad V = e^x$$

$$
\begin{aligned}
\int e^x \cos x\,dx &= e^x \cos x - \int e^x (-\sin x)\,dx \\
&= e^x \cos x + \int e^x \sin x\,dx \\
&= e^x \cos x + I \tag{2}
\end{aligned}
$$

**第 3 步：建立方程，求解 $I$**。

将 (2) 代入 (1)：
$$
\begin{aligned}
I &= e^x \sin x - (e^x \cos x + I) \\
I &= e^x \sin x - e^x \cos x - I \\
2I &= e^x (\sin x - \cos x) \\
I &= \frac{e^x}{2} (\sin x - \cos x) + C
\end{aligned}
$$

**第 4 步：验证**。

$$
\begin{aligned}
\frac{d}{dx}\left[\frac{e^x}{2}(\sin x - \cos x) + C\right]
&= \frac{e^x}{2}(\sin x - \cos x) + \frac{e^x}{2}(\cos x + \sin x) \\
&= \frac{e^x}{2}\big[(\sin x - \cos x) + (\cos x + \sin x)\big] \\
&= \frac{e^x}{2} \cdot 2\sin x = e^x \sin x
\end{aligned}
$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int e^{x} \sin x\,dx = \frac{e^{x}}{2}(\sin x - \cos x) + C$。

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 分部积分公式为 $\int u\,dv = uv + \int v\,du$（符号错误） | 分部积分公式为 $\int u\,dv = uv - \int v\,du$，是**减号**而非加号 |
| 分部积分公式中的 $uv$ 项不需要加常数 | $uv$ 本身不含积分常数，因为它是乘积法则的直接结果。积分常数统一写在最后 |
| 对 $\int \ln x\,dx$，设 $u = 1$、$dv = \ln x\,dx$ | 设 $u = \ln x$、$dv = dx$。因为 $\ln x$ 无法直接积分，但可以求导（$(\ln x)' = 1/x$），而 $dx$ 可以积分得到 $x$ |
| LIATE 法则中，越靠右越应选为 $u$ | LIATE 中越靠**左**越应选为 $u$（求导对象）。排位靠右的函数应放在 $dv$ 侧（积分对象） |
| 对 $\int x e^{2x}\,dx$，设 $u = e^{2x}$、$dv = x\,dx$ 也可以，因为都是可积的 | 这是错误选择！$u = e^{2x}$ 求导得 $du = 2e^{2x}dx$，$v = x^2/2$，新积分为 $\int (x^2/2) \cdot 2e^{2x}\,dx = \int x^2 e^{2x}\,dx$，次数反而升高，问题更复杂了 |
| 循环型分部积分只需一次分部即可得到结果 | 循环型**必须**做两次分部积分，且两次选择策略一致，才能产生原积分的方程 |
| 递推公式中的 $I_n$ 表达式可以直接使用，不需验证底情形 | 递推公式是**从高次到低次**的递推关系，必须指定底情形（$n=0$ 或 $n=1$）才能终止递推。例如 $I_n = x^n e^{kx}/k - (n/k)I_{n-1}$ 需搭配 $I_0 = e^{kx}/k + C$ |
| 分部积分法一定能让积分变简单 | 分部积分只是一种**转换工具**，它把 $\int u\,dv$ 变为 $\int v\,du$。如果 $u$ 和 $dv$ 的选择不当（违反 LIATE），新积分可能比原积分更复杂 |

### 检查点

- [ ] 能否从乘积法则出发，完整推导分部积分公式 $\int u\,dv = uv - \int v\,du$？
- [ ] 能否解释 LIATE 法则的每个字母含义及其优先级顺序？
- [ ] 能否指出"错误的 $u$/$dv$ 选择"会导致什么后果，并举一个具体例子？
- [ ] 能否计算 $\int x \sin x\,dx$（多项式 $\times$ 三角型）？
- [ ] 能否计算 $\int \arctan x\,dx$（单函数型）？
- [ ] 能否完成循环型积分 $\int e^{ax}\cos(bx)\,dx$ 的"两次分部 + 解方程"过程？
- [ ] 对于 $I_n = \int x^n e^{kx}\,dx$，能否写出递推公式并应用于 $n=3$ 的情况？
- [ ] 对于 $J_n = \int (\ln x)^n\,dx$，能否写出递推公式并应用于 $n=2$ 的情况？
- [ ] 能否解释为什么每次分部积分后多项式的次数会降低 1？
- [ ] 能否对分部积分的结果进行求导验证？

### 基础巩固

**1.** 计算下列不定积分：

(1) $\displaystyle\int x e^{3x}\,dx$

(2) $\displaystyle\int x \sin x\,dx$

(3) $\displaystyle\int \ln(2x)\,dx \quad (x > 0)$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int x e^{3x}\,dx$

设 $u = x$，$dv = e^{3x}\,dx$，则 $du = dx$，$v = \dfrac{1}{3}e^{3x}$。

$$
\begin{aligned}
\int x e^{3x}\,dx &= x \cdot \frac{1}{3}e^{3x} - \int \frac{1}{3}e^{3x}\,dx \\
&= \frac{1}{3}x e^{3x} - \frac{1}{3} \cdot \frac{1}{3}e^{3x} + C \\
&= \frac{1}{3}x e^{3x} - \frac{1}{9}e^{3x} + C \\
&= \frac{e^{3x}}{9}(3x - 1) + C
\end{aligned}
$$

**验证**：
$$\frac{d}{dx}\left(\frac{1}{3}x e^{3x} - \frac{1}{9}e^{3x} + C\right) = \frac{1}{3}e^{3x} + x e^{3x} - \frac{1}{3}e^{3x} = x e^{3x} \quad \checkmark$$

---

**(2)** $\displaystyle\int x \sin x\,dx$

设 $u = x$，$dv = \sin x\,dx$，则 $du = dx$，$v = -\cos x$。

$$
\begin{aligned}
\int x \sin x\,dx &= x \cdot (-\cos x) - \int (-\cos x)\,dx \\
&= -x\cos x + \int \cos x\,dx \\
&= -x\cos x + \sin x + C
\end{aligned}
$$

**验证**：
$$\frac{d}{dx}(-x\cos x + \sin x + C) = -\cos x + x\sin x + \cos x = x\sin x \quad \checkmark$$

---

**(3)** $\displaystyle\int \ln(2x)\,dx \quad (x > 0)$

设 $u = \ln(2x)$，$dv = dx$。注意 $\ln(2x) = \ln 2 + \ln x$，但直接用分部积分也可。

$du = \frac{1}{2x} \cdot 2\,dx = \frac{1}{x}\,dx$，$v = x$。

$$
\begin{aligned}
\int \ln(2x)\,dx &= x\ln(2x) - \int x \cdot \frac{1}{x}\,dx \\
&= x\ln(2x) - \int 1\,dx \\
&= x\ln(2x) - x + C
\end{aligned}
$$

**验证**：
$$\frac{d}{dx}(x\ln(2x) - x + C) = \ln(2x) + x \cdot \frac{1}{x} - 1 = \ln(2x) + 1 - 1 = \ln(2x) \quad \checkmark$$

</details>

**2.** 判断下列 $u$ 和 $dv$ 的选择是否正确。若不正确，请给出正确的选择。

| 积分 | 做出的选择 | 正确/错误？ |
|------|-----------|------------|
| (1) $\int x^2 e^x\,dx$ | $u = e^x$, $dv = x^2\,dx$ | |
| (2) $\int \arcsin x\,dx$ | $u = \arcsin x$, $dv = dx$ | |
| (3) $\int e^x \cos x\,dx$ | $u = \cos x$, $dv = e^x\,dx$ | |
| (4) $\int x^2 \ln x\,dx$ | $u = x^2$, $dv = \ln x\,dx$ | |

<details><summary>参考答案</summary>

**(1)** **错误**。按 LIATE，代数函数（$x^2$）在指数函数（$e^x$）之前，应选 $u = x^2$、$dv = e^x\,dx$。错误选择会导致新积分变为 $\int \frac{x^3}{3} e^x\,dx$，次数反而升高。

**(2)** **正确**。单函数型，$\arcsin x$ 属于反三角函数（I），应设 $u = \arcsin x$、$dv = dx$。求导得 $du = \frac{1}{\sqrt{1-x^2}}\,dx$，新积分变为 $\int \frac{x}{\sqrt{1-x^2}}\,dx$，可通过换元法求解。

**(3)** **正确**（但需要记住保持选择一致）。按 LIATE，三角函数（$\cos x$，T）在指数函数（$e^x$，E）之前，$u = \cos x$ 是合理选择。注意在第二次分部积分时仍应选三角部分为 $u$。

**(4)** **错误**。按 LIATE，对数函数（$\ln x$，L）在代数函数（$x^2$，A）之前，应选 $u = \ln x$、$dv = x^2\,dx$。正确计算：
$$u = \ln x,\; dv = x^2\,dx \;\Rightarrow\; du = \frac{1}{x}\,dx,\; v = \frac{x^3}{3}$$
$$\int x^2 \ln x\,dx = \frac{x^3}{3}\ln x - \int \frac{x^3}{3} \cdot \frac{1}{x}\,dx = \frac{x^3}{3}\ln x - \frac{1}{3}\int x^2\,dx = \frac{x^3}{3}\ln x - \frac{x^3}{9} + C$$

</details>

**3.** 计算 $\displaystyle\int x^2 \cos x\,dx$（提示：需连续使用两次分部积分）。

<details><summary>参考答案</summary>

**第一次分部积分**：设 $u = x^2$，$dv = \cos x\,dx$，则 $du = 2x\,dx$，$v = \sin x$。

$$\int x^2 \cos x\,dx = x^2 \sin x - \int 2x \sin x\,dx = x^2 \sin x - 2\int x \sin x\,dx$$

**第二次分部积分**：对 $\int x \sin x\,dx$ 使用分部积分。设 $u = x$，$dv = \sin x\,dx$，则 $du = dx$，$v = -\cos x$。

$$\int x \sin x\,dx = -x\cos x - \int (-\cos x)\,dx = -x\cos x + \int \cos x\,dx = -x\cos x + \sin x + C_1$$

**合并**：
$$
\begin{aligned}
\int x^2 \cos x\,dx &= x^2 \sin x - 2(-x\cos x + \sin x + C_1) \\
&= x^2 \sin x + 2x\cos x - 2\sin x + C
\end{aligned}
$$

**验证**：
$$
\begin{aligned}
\frac{d}{dx}(x^2 \sin x + 2x\cos x - 2\sin x + C) &= 2x\sin x + x^2\cos x + 2\cos x - 2x\sin x - 2\cos x \\
&= x^2\cos x \quad \checkmark
\end{aligned}
$$

**答案**：$\displaystyle\int x^2 \cos x\,dx = x^2 \sin x + 2x\cos x - 2\sin x + C$

</details>

**4.** 利用递推公式计算 $\displaystyle\int x^3 e^{2x}\,dx$（提示：使用约化公式 $I_n = \frac{x^n e^{kx}}{k} - \frac{n}{k} I_{n-1}$，其中 $k=2$）。

<details><summary>参考答案</summary>

从底情形开始：
$$I_0 = \int e^{2x}\,dx = \frac{e^{2x}}{2} + C_0$$

\begin{aligned}
I_1 &= \frac{x e^{2x}}{2} - \frac{1}{2} I_0 = \frac{x e^{2x}}{2} - \frac{1}{2} \cdot \frac{e^{2x}}{2} + C_1 = \frac{e^{2x}}{2}\left(x - \frac{1}{2}\right) + C_1 \\

\end{aligned}

\begin{aligned}
I_3 &= \frac{e^{2x}}{4}\left(2x^3 - 3x^2 + 3x - \frac{3}{2}\right) + C \\
&= \frac{e^{2x}}{4}\left(2x^3 - 3x^2 + 3x - \frac{3}{2}\right) + C
\end{aligned}
$$

**答案**：$\displaystyle\int x^3 e^{2x}\,dx = \frac{e^{2x}}{4}\left(2x^3 - 3x^2 + 3x - \frac{3}{2}\right) + C$

**5.** 计算 $\displaystyle\int e^{2x} \cos 3x\,dx$（循环型）。

<details><summary>参考答案</summary>

记 $I = \int e^{2x} \cos 3x\,dx$。

**第一次分部积分**：设 $u = \cos 3x$，$dv = e^{2x}\,dx$，则 $du = -3\sin 3x\,dx$，$v = \dfrac{1}{2}e^{2x}$。

$$I = \frac{1}{2}e^{2x}\cos 3x - \int \frac{1}{2}e^{2x} \cdot (-3\sin 3x)\,dx = \frac{1}{2}e^{2x}\cos 3x + \frac{3}{2}\int e^{2x}\sin 3x\,dx \tag{1}$$

**第二次分部积分**：对 $\int e^{2x}\sin 3x\,dx$ 使用分部积分。设 $U = \sin 3x$，$dV = e^{2x}\,dx$，则 $dU = 3\cos 3x\,dx$，$V = \dfrac{1}{2}e^{2x}$。

$$
\begin{aligned}
\int e^{2x}\sin 3x\,dx &= \frac{1}{2}e^{2x}\sin 3x - \int \frac{1}{2}e^{2x} \cdot 3\cos 3x\,dx \\
&= \frac{1}{2}e^{2x}\sin 3x - \frac{3}{2}\int e^{2x}\cos 3x\,dx \\
&= \frac{1}{2}e^{2x}\sin 3x - \frac{3}{2}I \tag{2}
\end{aligned}
$$

**回环求解**：将 (2) 代入 (1)：
$$
\begin{aligned}
I &= \frac{1}{2}e^{2x}\cos 3x + \frac{3}{2}\left(\frac{1}{2}e^{2x}\sin 3x - \frac{3}{2}I\right) \\
I &= \frac{1}{2}e^{2x}\cos 3x + \frac{3}{4}e^{2x}\sin 3x - \frac{9}{4}I \\
I + \frac{9}{4}I &= \frac{e^{2x}}{4}(2\cos 3x + 3\sin 3x) \\
\frac{13}{4}I &= \frac{e^{2x}}{4}(2\cos 3x + 3\sin 3x) \\
I &= \frac{e^{2x}}{13}(2\cos 3x + 3\sin 3x) + C
\end{aligned}
$$

**验证**：
$$\frac{d}{dx}\left[\frac{e^{2x}}{13}(2\cos 3x + 3\sin 3x) + C\right] = \frac{2e^{2x}}{13}(2\cos 3x + 3\sin 3x) + \frac{e^{2x}}{13}(-6\sin 3x + 9\cos 3x)$$
$$= \frac{e^{2x}}{13}(4\cos 3x + 6\sin 3x - 6\sin 3x + 9\cos 3x) = \frac{e^{2x}}{13}(13\cos 3x) = e^{2x}\cos 3x \quad \checkmark$$

**答案**：$\displaystyle\int e^{2x} \cos 3x\,dx = \frac{e^{2x}}{13}(2\cos 3x + 3\sin 3x) + C$

</details>

**6.** （综合题）计算 $\displaystyle\int x \arctan x\,dx$。

（提示：被积函数为代数函数 $\times$ 反三角函数。按 LIATE，反三角函数（I）在代数函数（A）之前，设 $u = \arctan x$，$dv = x\,dx$。）

<details><summary>参考答案</summary>

**第 1 步：选择 $u$ 和 $dv$**。

按 LIATE，$u = \arctan x$（反三角函数），$dv = x\,dx$（代数函数）。

**第 2 步：计算 $du$ 和 $v$**。

$$du = \frac{1}{1+x^2}\,dx, \qquad v = \int x\,dx = \frac{x^2}{2}$$

**第 3 步：代入分部积分公式**。

$$
\begin{aligned}
\int x \arctan x\,dx &= \frac{x^2}{2}\arctan x - \int \frac{x^2}{2} \cdot \frac{1}{1+x^2}\,dx \\
&= \frac{x^2}{2}\arctan x - \frac{1}{2}\int \frac{x^2}{1+x^2}\,dx
\end{aligned}
$$

**第 4 步：处理剩余积分**。

$$\frac{x^2}{1+x^2} = \frac{(1+x^2) - 1}{1+x^2} = 1 - \frac{1}{1+x^2}$$

因此：
$$
\begin{aligned}
\int \frac{x^2}{1+x^2}\,dx &= \int \left(1 - \frac{1}{1+x^2}\right)dx \\
&= \int 1\,dx - \int \frac{1}{1+x^2}\,dx \\
&= x - \arctan x + C_1
\end{aligned}
$$

**第 5 步：合并结果**。

$$
\begin{aligned}
\int x \arctan x\,dx &= \frac{x^2}{2}\arctan x - \frac{1}{2}(x - \arctan x) + C \\
&= \frac{x^2}{2}\arctan x - \frac{x}{2} + \frac{1}{2}\arctan x + C \\
&= \frac{1}{2}(x^2 + 1)\arctan x - \frac{x}{2} + C
\end{aligned}
$$

**验证**：
$$
\begin{aligned}
\frac{d}{dx}\left[\frac{1}{2}(x^2 + 1)\arctan x - \frac{x}{2} + C\right]
&= \frac{1}{2}[2x\arctan x + (x^2+1)\cdot\frac{1}{1+x^2}] - \frac{1}{2} \\
&= \frac{1}{2}[2x\arctan x + 1] - \frac{1}{2} \\
&= x\arctan x + \frac{1}{2} - \frac{1}{2} = x\arctan x \quad \checkmark
\end{aligned}
$$

**答案**：$\displaystyle\int x \arctan x\,dx = \frac{1}{2}(x^2 + 1)\arctan x - \frac{x}{2} + C$

</details>

## 2. 背景与应用场景

在前序文件中，我们掌握了基本积分公式、换元积分法和分部积分法——这些是不定积分计算的三大核心工具。然而，有一大类常见函数——**有理函数**（两个多项式的商）——的积分问题，既不能直接套用基本积分公式，也无法简单地通过换元或分部积分一次性解决。

有理函数的积分在数学和工程中有广泛的应用：

- **控制工程**：系统的传递函数通常为有理函数，求其拉普拉斯逆变换需要部分分式分解
- **微分方程**：求解常系数线性微分方程时，特征方程的有理解对应着有理函数的部分分式分解
- **数值分析**：复杂函数的近似积分中常将有理函数作为中间步骤
- **概率统计**：某些概率分布（如柯西分布）的密度函数为有理函数

有理函数积分的核心思想是：**将一个复杂的有理函数分解为若干个简单分式之和，每个简单分式都可以用基本积分公式或其直接推广来积分**。这个方法体系分为三个层次：预备公式、代数化简（多项式除法）、代数分解（部分分式）。

---

## 3. 核心概念与符号约定

### 3.1 有理函数与真分式、假分式

**有理函数（Rational Function）**：若 $P(x)$ 和 $Q(x)$ 均为多项式且 $Q(x)$ 不是零多项式，则称
$$R(x) = \frac{P(x)}{Q(x)}$$
为一个有理函数。本章中我们默认 $P$ 和 $Q$ 的系数均为实数。

设 $\deg(P)$ 表示多项式 $P$ 的最高次数。根据分子分母的次数关系，有理函数分为两类：

- **真分式（Proper Rational Function）**：$\deg(P) < \deg(Q)$，如 $\displaystyle\frac{3x+5}{x^2+3x+2}$（$\deg=1<2$）
- **假分式（Improper Rational Function）**：$\deg(P) \geq \deg(Q)$，如 $\displaystyle\frac{x^3+2x}{x^2+1}$（$\deg=3>2$）

### 3.2 有理函数积分的总体策略

有理函数积分遵循以下处理流程：

```
有理函数 R(x) = P(x)/Q(x)
        |
        v
  第 1 步：判断是否为真分式？
  |-- 假分式 (deg(P) >= deg(Q)) --> 多项式长除法 --> 多项式 + 真分式
  |-- 真分式 (deg(P) < deg(Q))  --> 直接进入第 2 步
        |
        v
  第 2 步：对分母因式分解，根据结果选择不同处理
  |-- 分母为不同一次因式乘积 --> 部分分式分解 A/(x-a)+B/(x-b)
  |-- 分母为不可约二次因式 --> 分子拆分 + 配方 + arctan
        |
        v
  第 3 步：逐项积分（利用定理 6.4~6.6 和基本公式），合并结果
```

### 3.3 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\deg(P)$ | 多项式 $P$ 的最高次数 | $\deg(x^3+2x)=3$ |
| $\frac{P(x)}{Q(x)}$ | 有理函数 | $\frac{x^3+2x}{x^2+1}$ |
| $A, B$ | 部分分式中的待定常数 | $\frac{1}{(x-2)(x-3)} = \frac{A}{x-2} + \frac{B}{x-3}$ |
| $\Delta = p^2 - 4q$ | 二次多项式 $x^2+px+q$ 的判别式 | $\Delta<0$ 时为不可约二次因式 |
| $u = x + k$ | 配方法中的线性平移 | $(x+1)^2+4$ 中令 $u=x+1$ |

## 4. 原理与方法

### 4.1 预备公式——三个推广的基本积分公式

在有理函数积分中，最常遇到的三种积分形式是 $\int\frac{1}{x-a}dx$、$\int\frac{1}{x^2+a^2}dx$ 和 $\int\frac{f'(x)}{f(x)}dx$。它们都可以从基本积分公式出发，结合链式法则推导得到。

---

**定理 6.4（$\int \frac{1}{x-a}dx$ 公式）**：对任意实数 $a$，在 $x \neq a$ 的区间上有
$$\boxed{\int \frac{1}{x-a}\,dx = \ln|x-a| + C}$$

**证明**：令 $F(x) = \ln|x-a|$，对 $F$ 求导：
- 当 $x > a$ 时，$F(x) = \ln(x-a)$，$F'(x) = \frac{1}{x-a}$。
- 当 $x < a$ 时，$F(x) = \ln(a-x)$，由链式法则 $F'(x) = \frac{1}{a-x} \cdot (-1) = \frac{1}{x-a}$。

因此对一切 $x \neq a$ 有 $F'(x) = \frac{1}{x-a}$。由不定积分的定义，
$$\int \frac{1}{x-a}\,dx = F(x) + C = \ln|x-a| + C$$
证毕。

**推论**：令 $a = 0$ 得到基本积分公式 $\int \frac{1}{x}\,dx = \ln|x| + C$（表 6.1 公式 2），因此定理 6.4 是该公式的直接推广。

---

**定理 6.5（$\int \frac{1}{x^2+a^2}dx$ 公式）**：设 $a > 0$，则
$$\boxed{\int \frac{1}{x^2 + a^2}\,dx = \frac{1}{a}\arctan\frac{x}{a} + C}$$

**证明**：令 $G(x) = \frac{1}{a}\arctan\frac{x}{a}$。由链式法则（$( \arctan x)' = \frac{1}{1+x^2}$）：
$$G'(x) = \frac{1}{a} \cdot \frac{1}{1 + (x/a)^2} \cdot \frac{1}{a} = \frac{1}{a^2} \cdot \frac{1}{1 + x^2/a^2} = \frac{1}{a^2} \cdot \frac{a^2}{a^2 + x^2} = \frac{1}{x^2 + a^2}$$

因此 $G$ 是 $\frac{1}{x^2 + a^2}$ 的一个原函数。证毕。

**推论**：令 $a = 1$ 得到 $\int \frac{1}{1+x^2}\,dx = \arctan x + C$，即表 6.1 公式 6。

**常见错误**：计算 $\int \frac{1}{x^2+4}dx$ 时，忘记乘以 $\frac{1}{a} = \frac{1}{2}$，直接写成 $\arctan\frac{x}{2} + C$。正确结果为 $\frac{1}{2}\arctan\frac{x}{2} + C$。

---

**定理 6.6（对数积分模式 / Logarithmic Integration）**：设 $f(x)$ 可导且 $f(x) \neq 0$，则
$$\boxed{\int \frac{f'(x)}{f(x)}\,dx = \ln|f(x)| + C}$$

**证明**：令 $H(x) = \ln|f(x)|$。由链式法则：
$$H'(x) = \frac{1}{f(x)} \cdot f'(x) = \frac{f'(x)}{f(x)}$$

因此 $H$ 是 $\frac{f'(x)}{f(x)}$ 的一个原函数。证毕。

**应用方法**：遇到形如 $\int \frac{g(x)}{h(x)}dx$ 时，检查分子 $g(x)$ 是否恰好是分母 $h(x)$ 的导数。若是，则积分结果为 $\ln|h(x)| + C$。

**例**：$\int \frac{2x}{x^2+1}\,dx = \ln(x^2+1) + C$，因为 $\frac{d}{dx}(x^2+1) = 2x$ 且 $x^2+1 > 0$ 恒成立。

定理 6.4~6.6 是本节最重要的积分工具。在实际问题中，它们往往需要组合使用。

---

### 4.2 多项式长除法——化假分式为真分式

当有理函数 $R(x) = P(x)/Q(x)$ 为**假分式**时（$\deg(P) \geq \deg(Q)$），可以通过多项式长除法将其写为：
$$\frac{P(x)}{Q(x)} = S(x) + \frac{T(x)}{Q(x)}$$
其中 $S(x)$ 是多项式（商式），$T(x)$ 是余式，且 $\deg(T) < \deg(Q)$（即 $T/Q$ 为真分式）。

**多项式长除法步骤**：以 $\frac{x^3 + 2x}{x^2 + 1}$ 为例。

**第 1 步**：比较次数。被除式 $\deg=3$，除式 $\deg=2$，需要做除法。

**第 2 步**：用被除式的首项 $x^3$ 除以除式的首项 $x^2$，得到第一项商 $x$。

**第 3 步**：将商乘以除式得 $x(x^2+1) = x^3 + x$，从被除式中减去：
$$(x^3 + 2x) - (x^3 + x) = x$$

**第 4 步**：余式 $x$ 的次数为 $1$，小于除式次数 $2$，停止。商为 $x$，余式为 $x$。
$$\frac{x^3 + 2x}{x^2 + 1} = x + \frac{x}{x^2 + 1}$$

**一般形式**的书写方式如下。对 $\frac{P(x)}{Q(x)}$ 执行长除法：

设商式为 $S(x)$，余式为 $T(x)$，则
$$\frac{P(x)}{Q(x)} = S(x) + \frac{T(x)}{Q(x)}, \quad \deg(T) < \deg(Q)$$

**为什么要先做多项式除法？** 因为只有真分式才能直接应用部分分式分解或配方积分法。多项式部分 $S(x)$ 可以直接用基本积分公式积分（如 $\int x\,dx = x^2/2 + C$）。因此假分式的积分被分离为"多项式积分 + 真分式积分"两个独立的部分。

---

### 4.3 部分分式分解——分母为不同一次因式的乘积

当真分式的分母 $Q(x)$ 可以分解为不同一次因式的乘积时：
$$Q(x) = (x-a_1)(x-a_2)\cdots(x-a_n) \quad (a_i \text{互不相同})$$
可以将真分式分解为以下形式：
$$\frac{P(x)}{Q(x)} = \frac{A_1}{x-a_1} + \frac{A_2}{x-a_2} + \cdots + \frac{A_n}{x-a_n}$$
其中 $A_1, A_2, \dots, A_n$ 是待定常数。这种方法称为**部分分式分解**（Partial Fraction Decomposition），$A_i$ 称为**待定系数**。

以两项情形 $\frac{P(x)}{(x-a)(x-b)}$（$a \neq b$）为例，分解形式为：
$$\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$$

确定待定系数 $A, B$ 有两种常用方法。

---

**方法一：代入法（适用于不同一次因式乘积）**

**第 1 步**：设定分解形式。
$$\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$$

**第 2 步**：两边同乘 $(x-a)(x-b)$，消去分母：
$$P(x) = A(x-b) + B(x-a)$$

**第 3 步**：代入使某些项为零的 $x$ 值。令 $x = a$（此时 $x-a=0$，含 $B$ 的项消失）：
$$P(a) = A(a-b) \;\Longrightarrow\; A = \frac{P(a)}{a-b}$$

令 $x = b$（此时 $x-b=0$，含 $A$ 的项消失）：
$$P(b) = B(b-a) \;\Longrightarrow\; B = \frac{P(b)}{b-a}$$

**示例**：分解 $\frac{3x+5}{(x+1)(x+2)}$。

两边同乘 $(x+1)(x+2)$：$3x+5 = A(x+2) + B(x+1)$。

令 $x = -1$：$3(-1)+5 = A(1) \;\Rightarrow\; 2 = A$。
令 $x = -2$：$3(-2)+5 = B(-1) \;\Rightarrow\; -1 = -B \;\Rightarrow\; B = 1$。

因此 $\frac{3x+5}{(x+1)(x+2)} = \frac{2}{x+1} + \frac{1}{x+2}$。

---

**方法二：比较系数法（通用方法）**

从 $P(x) = A(x-b) + B(x-a)$ 出发，展开右端并合并同类项：
$$P(x) = (A+B)x + (-Ab - Ba)$$

比较等式两边 $x$ 的同次幂系数，得到关于 $A, B$ 的线性方程组，解之即得。

**示例**（同上题）：$3x+5 = A(x+2) + B(x+1) = (A+B)x + (2A+B)$

比较系数得：
$$\begin{cases}
A + B = 3 \\
2A + B = 5
\end{cases}$$

两式相减得 $A = 2$，代入得 $B = 1$。

**方法选择**：对于分母为不同一次因式乘积的情形，**代入法**最为快捷；对于重因式等复杂情形，比较系数法更为通用。

---

完成部分分式分解后，每个分式 $\frac{A}{x-a}$ 都可以直接用定理 6.4 积分：
$$\int \frac{A}{x-a}\,dx = A\ln|x-a| + C$$

---

### 4.4 分母为不可约二次因式的积分（配方 + arctan）

当真分式的分母是判别式 $\Delta < 0$ 的二次式时，它不能在实数范围内分解为一次因式的乘积。此时的积分策略取决于分子的形式。

---

**情形一：分子为常数** $\int \frac{1}{x^2+px+q}\,dx$（$\Delta = p^2 - 4q < 0$）

**第 1 步：配方**。
$$x^2 + px + q = \left(x + \frac{p}{2}\right)^2 + \left(q - \frac{p^2}{4}\right)$$

由于 $\Delta < 0$，$q - \frac{p^2}{4} > 0$。记 $a = \sqrt{q - \frac{p^2}{4}}$。

**第 2 步：利用定理 6.5**。作平移 $u = x + \frac{p}{2}$：
$$\int \frac{1}{x^2+px+q}\,dx = \int \frac{1}{u^2 + a^2}\,du = \frac{1}{a}\arctan\frac{u}{a} + C = \frac{1}{a}\arctan\frac{x + p/2}{a} + C$$

**例**：$\int \frac{1}{x^2+2x+5}\,dx$。

配方：$x^2+2x+5 = (x+1)^2 + 4 = (x+1)^2 + 2^2$。

$\int \frac{1}{(x+1)^2+2^2}\,dx = \frac{1}{2}\arctan\frac{x+1}{2} + C$（代入 $a=2$, $u=x+1$）。

---

**情形二：分子为线性函数** $\int \frac{mx+n}{x^2+px+q}\,dx$（$\Delta < 0$）

这是有理函数积分中最常见也最具技巧性的情形。核心策略是**拆分分子**：

**第 1 步**：计算分母的导数：$\frac{d}{dx}(x^2+px+q) = 2x + p$。

**第 2 步**：将分子 $mx+n$ 写为"分母导数的倍数 + 常数"的形式：
$$mx + n = \frac{m}{2}(2x + p) + \left(n - \frac{mp}{2}\right)$$

**第 3 步**：分离积分：
$$\int \frac{mx+n}{x^2+px+q}\,dx = \frac{m}{2}\int \frac{2x+p}{x^2+px+q}\,dx + \left(n - \frac{mp}{2}\right)\int \frac{1}{x^2+px+q}\,dx$$

其中：
- **第一项**：$\int \frac{2x+p}{x^2+px+q}\,dx$ 恰好是 $\int \frac{f'(x)}{f(x)}dx$ 的形式（$f(x)=x^2+px+q$），由定理 6.6 得 $\ln|x^2+px+q| + C$。
- **第二项**：$\int \frac{1}{x^2+px+q}\,dx$ 属于情形一，用配方 + 定理 6.5 处理。

---

## 5. 例题

### 例 1：部分分式分解（分母为不同一次因式乘积）

计算 $\displaystyle\int \frac{3x+5}{x^2+3x+2}\,dx$。

**解**：

**第 1 步：因式分解分母并判断分式类型**。
$$x^2 + 3x + 2 = (x+1)(x+2)$$
分子 $\deg=1$，分母 $\deg=2$，是真分式。

**第 2 步：设定部分分式分解形式**。
设存在常数 $A, B$ 使得：
$$\frac{3x+5}{(x+1)(x+2)} = \frac{A}{x+1} + \frac{B}{x+2}$$

**第 3 步：确定待定常数（代入法）**。
两边同乘 $(x+1)(x+2)$：
$$3x+5 = A(x+2) + B(x+1)$$

令 $x = -1$（此时 $x+1=0$，含 $B$ 的项消失）：
$$3(-1)+5 = A(-1+2) \;\Longrightarrow\; 2 = A \;\Longrightarrow\; A = 2$$

令 $x = -2$（此时 $x+2=0$，含 $A$ 的项消失）：
$$3(-2)+5 = B(-2+1) \;\Longrightarrow\; -1 = -B \;\Longrightarrow\; B = 1$$

因此：
$$\frac{3x+5}{(x+1)(x+2)} = \frac{2}{x+1} + \frac{1}{x+2}$$

**第 4 步：逐项积分**。
利用线性性质（定理 6.2）和定理 6.4（$\int\frac{1}{x-a}dx = \ln|x-a|+C$）：
$$\begin{aligned}
\int \frac{3x+5}{x^2+3x+2}\,dx
&= \int \frac{2}{x+1}\,dx + \int \frac{1}{x+2}\,dx \\
&= 2\ln|x+1| + \ln|x+2| + C
\end{aligned}$$

**第 5 步：验证**。
对结果求导：
$$\frac{d}{dx}\big(2\ln|x+1| + \ln|x+2| + C\big) = \frac{2}{x+1} + \frac{1}{x+2}$$

通分验证：
$$\frac{2}{x+1} + \frac{1}{x+2} = \frac{2(x+2) + (x+1)}{(x+1)(x+2)} = \frac{3x+5}{x^2+3x+2}$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{3x+5}{x^2+3x+2}\,dx = 2\ln|x+1| + \ln|x+2| + C$

---

### 例 2：多项式长除法 + 对数积分模式

计算 $\displaystyle\int \frac{x^3 + 2x}{x^2 + 1}\,dx$。

**解**：

**第 1 步：判断分式类型**。
分子 $\deg=3$，分母 $\deg=2$，是假分式。先做多项式除法。

**第 2 步：多项式长除法**。
$$
\begin{aligned}
& \frac{x^3 + 2x}{x^2 + 1}: \\
& x^3 \div x^2 = x \quad (\text{第一项商}) \\
& x(x^2+1) = x^3 + x \\
& (x^3 + 2x) - (x^3 + x) = x \quad (\text{余式})
\end{aligned}
$$

商为 $x$，余式为 $x$，因此：
$$\frac{x^3 + 2x}{x^2 + 1} = x + \frac{x}{x^2 + 1}$$

**第 3 步：对多项式部分直接积分**。
$$\int x\,dx = \frac{x^2}{2} + C_1$$

**第 4 步：处理真分式部分——识别对数积分模式**。
对于 $\int \frac{x}{x^2+1}\,dx$：
令 $f(x) = x^2 + 1$，则 $f'(x) = 2x$。分子 $x = \frac{1}{2}f'(x)$。由定理 6.6：
$$\int \frac{x}{x^2+1}\,dx = \frac{1}{2}\int \frac{2x}{x^2+1}\,dx = \frac{1}{2}\ln(x^2+1) + C_2$$

（分母 $x^2+1 > 0$ 恒成立，故省略绝对值符号。）

**第 5 步：合并结果**。
$$\begin{aligned}
\int \frac{x^3 + 2x}{x^2 + 1}\,dx
&= \frac{x^2}{2} + \frac{1}{2}\ln(x^2+1) + C
\end{aligned}$$

其中 $C = C_1 + C_2$。

**第 6 步：验证**。
$$\begin{aligned}
\frac{d}{dx}\left[\frac{x^2}{2} + \frac{1}{2}\ln(x^2+1) + C\right]
&= x + \frac{1}{2} \cdot \frac{2x}{x^2+1} \\
&= x + \frac{x}{x^2+1} \\
&= \frac{x(x^2+1) + x}{x^2+1} = \frac{x^3 + 2x}{x^2+1}
\end{aligned}$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{x^3 + 2x}{x^2 + 1}\,dx = \frac{x^2}{2} + \frac{1}{2}\ln(x^2+1) + C$

---

### 例 3：分母为不可约二次因式的积分

计算 $\displaystyle\int \frac{2x + 3}{x^2 + 2x + 5}\,dx$。

**解**：

**第 1 步：检查分母是否可约**。
$\Delta = 2^2 - 4\cdot1\cdot5 = 4 - 20 = -16 < 0$，分母在实数范围内不可分解。
分子 $\deg=1$，分母 $\deg=2$，为真分式。

**第 2 步：拆分分子——写成"分母的导数 + 常数"的形式**。
分母的导数：$\frac{d}{dx}(x^2 + 2x + 5) = 2x + 2$

将分子 $2x+3$ 用 $2x+2$ 和常数表示：
$$2x + 3 = (2x+2) + 1$$

由线性性质（定理 6.2）：
$$\int \frac{2x+3}{x^2+2x+5}\,dx = \int \frac{2x+2}{x^2+2x+5}\,dx + \int \frac{1}{x^2+2x+5}\,dx$$

**第 3 步：处理第一部分——对数积分模式（定理 6.6）**。
令 $f(x) = x^2 + 2x + 5$，则 $f'(x) = 2x+2$。
$$\int \frac{2x+2}{x^2+2x+5}\,dx = \ln(x^2+2x+5) + C_1$$

（分母 $x^2+2x+5 = (x+1)^2 + 4 > 0$ 恒为正，故无需绝对值。）

**第 4 步：处理第二部分——配方 + arctan（定理 6.5）**。

配方：
$$x^2 + 2x + 5 = (x^2 + 2x + 1) + 4 = (x+1)^2 + 2^2$$

因此：
$$\int \frac{1}{x^2+2x+5}\,dx = \int \frac{1}{(x+1)^2 + 2^2}\,dx$$

作平移 $u = x+1$，$du = dx$，由定理 6.5（$a=2$）：
$$\int \frac{1}{u^2 + 2^2}\,du = \frac{1}{2}\arctan\frac{u}{2} + C_2$$

代回 $u = x+1$：
$$\int \frac{1}{x^2+2x+5}\,dx = \frac{1}{2}\arctan\frac{x+1}{2} + C_2$$

**第 5 步：合并结果**。
$$\begin{aligned}
\int \frac{2x+3}{x^2+2x+5}\,dx
&= \ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2} + C
\end{aligned}$$

其中 $C = C_1 + C_2$。

**第 6 步：验证**。

第一部分求导：
$$\frac{d}{dx}\big[\ln(x^2+2x+5)\big] = \frac{2x+2}{x^2+2x+5}$$

第二部分求导（链式法则）：
$$\frac{d}{dx}\left[\frac{1}{2}\arctan\frac{x+1}{2}\right]
= \frac{1}{2} \cdot \frac{1}{1 + \left(\frac{x+1}{2}\right)^2} \cdot \frac{1}{2}
= \frac{1}{4} \cdot \frac{1}{1 + \frac{(x+1)^2}{4}}
= \frac{1}{4} \cdot \frac{4}{4 + (x+1)^2} = \frac{1}{x^2+2x+5}$$

因此：
$$\frac{d}{dx}\left[\ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2} + C\right]
= \frac{2x+2}{x^2+2x+5} + \frac{1}{x^2+2x+5} = \frac{2x+3}{x^2+2x+5}$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{2x+3}{x^2+2x+5}\,dx = \ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2} + C$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 对假分式直接做部分分式分解，例如设 $\frac{x^3}{x^2-1} = \frac{A}{x-1} + \frac{B}{x+1}$ | 假分式必须先做**多项式长除法**化为"多项式 + 真分式"；部分分式分解只适用于真分式。长除法后：$\frac{x^3}{x^2-1} = x + \frac{x}{x^2-1}$，再对真分式 $\frac{x}{x^2-1}$ 做分解 |
| 分解时符号处理错误，如设 $\frac{1}{(x-2)(x-3)} = \frac{A}{x-2} - \frac{B}{x-3}$ | 标准形式用**加号**连接：$\frac{A}{x-2} + \frac{B}{x-3}$。$A$ 和 $B$ 本身可以是负数，无需在形式上写减号 |
| $\int \frac{1}{x^2+4}\,dx = \arctan\frac{x}{2} + C$（遗漏系数 $1/a$） | 由定理 6.5 得 $\int \frac{1}{x^2+2^2}\,dx = \frac{1}{2}\arctan\frac{x}{2} + C$。系数 $1/a$ 是必须的 |
| 对 $\int \frac{2x+3}{x^2+2x+5}\,dx$ 直接对分母配方就试图用 $\arctan$ | 若分子是**线性函数**，必须先**拆分分子**（写成"分母导数 + 常数"形式），否则配方后仍然无法直接积分 |
| 做部分分式代入法时，只得到 $A$ 不回代就结束 | 求出 $A$ 后必须代入原式求出所有系数（$B$ 等），再写回分解形式 | 
| 认为 $\ln|f(x)| + C$ 中的绝对值可以永远省略 | 当 $f(x)$ 恒正时（如 $x^2+1$、$(x+1)^2+4$）可省略；当 $f(x)$ 可能为负时（如 $x-2$）必须保留绝对值 |

## 练习题

### 基础巩固

**1.** 计算下列不定积分：

(1) $\displaystyle\int \frac{1}{x^2 - 6x + 8}\,dx$

(2) $\displaystyle\int \frac{2x + 1}{x^2 + 4x + 3}\,dx$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{1}{x^2 - 6x + 8}\,dx$

因式分解分母：$x^2 - 6x + 8 = (x-2)(x-4)$。

设 $\frac{1}{(x-2)(x-4)} = \frac{A}{x-2} + \frac{B}{x-4}$。两边同乘 $(x-2)(x-4)$：$1 = A(x-4) + B(x-2)$。

令 $x=2$：$1 = A(-2) \Rightarrow A = -\frac12$。
令 $x=4$：$1 = B(2) \Rightarrow B = \frac12$。

因此：
$$\int \frac{1}{x^2-6x+8}\,dx = -\frac12\int \frac{1}{x-2}\,dx + \frac12\int \frac{1}{x-4}\,dx = -\frac12\ln|x-2| + \frac12\ln|x-4| + C = \frac12\ln\left|\frac{x-4}{x-2}\right| + C$$

**验证**：
$$\frac{d}{dx}\left[\frac12\ln\left|\frac{x-4}{x-2}\right| + C\right] = \frac12\left(\frac{1}{x-4} - \frac{1}{x-2}\right) = \frac12\cdot\frac{(x-2)-(x-4)}{(x-2)(x-4)} = \frac12\cdot\frac{2}{(x-2)(x-4)} = \frac{1}{x^2-6x+8} \quad \checkmark$$

---

**(2)** $\displaystyle\int \frac{2x+1}{x^2+4x+3}\,dx$

因式分解分母：$x^2+4x+3 = (x+1)(x+3)$。

设 $\frac{2x+1}{(x+1)(x+3)} = \frac{A}{x+1} + \frac{B}{x+3}$。两边同乘 $(x+1)(x+3)$：$2x+1 = A(x+3) + B(x+1)$。

令 $x=-1$：$-1 = A(2) \Rightarrow A = -\frac12$。
令 $x=-3$：$-5 = B(-2) \Rightarrow B = \frac52$。

因此：
$$\begin{aligned}
\int \frac{2x+1}{x^2+4x+3}\,dx &= -\frac12\int \frac{1}{x+1}\,dx + \frac52\int \frac{1}{x+3}\,dx \\
&= -\frac12\ln|x+1| + \frac52\ln|x+3| + C
\end{aligned}$$

**验证**：
$$\frac{d}{dx}\left[-\frac12\ln|x+1| + \frac52\ln|x+3| + C\right] = -\frac{1}{2(x+1)} + \frac{5}{2(x+3)} = \frac{-(x+3)+5(x+1)}{2(x+1)(x+3)} = \frac{4x+2}{2(x^2+4x+3)} = \frac{2x+1}{x^2+4x+3} \quad \checkmark$$

</details>

---

**2.** 计算下列不定积分：

(1) $\displaystyle\int \frac{x^2}{x-1}\,dx$

(2) $\displaystyle\int \frac{x^4}{x^2+1}\,dx$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{x^2}{x-1}\,dx$

假分式（$\deg=2 \geq \deg=1$），先做多项式除法。

代数方法：$x^2 = (x-1)(x+1) + 1$，因此 $\frac{x^2}{x-1} = x+1 + \frac{1}{x-1}$。

$$\int \frac{x^2}{x-1}\,dx = \int (x+1)\,dx + \int \frac{1}{x-1}\,dx = \frac{x^2}{2} + x + \ln|x-1| + C$$

**验证**：
$$\frac{d}{dx}\left[\frac{x^2}{2} + x + \ln|x-1| + C\right] = x + 1 + \frac{1}{x-1} = \frac{(x+1)(x-1) + 1}{x-1} = \frac{x^2}{x-1} \quad \checkmark$$

---

**(2)** $\displaystyle\int \frac{x^4}{x^2+1}\,dx$

假分式（$\deg=4 \geq \deg=2$），做多项式除法：

$$\frac{x^4}{x^2+1} = \frac{x^4 - 1 + 1}{x^2+1} = \frac{(x^2-1)(x^2+1) + 1}{x^2+1} = x^2 - 1 + \frac{1}{x^2+1}$$

因此：
$$\int \frac{x^4}{x^2+1}\,dx = \int (x^2-1)\,dx + \int \frac{1}{x^2+1}\,dx = \frac{x^3}{3} - x + \arctan x + C$$

**验证**：
$$\frac{d}{dx}\left[\frac{x^3}{3} - x + \arctan x + C\right] = x^2 - 1 + \frac{1}{x^2+1} = \frac{x^4}{x^2+1} \quad \checkmark$$

</details>

---

**3.** 计算下列不定积分：

(1) $\displaystyle\int \frac{1}{x^2 + 4x + 13}\,dx$

(2) $\displaystyle\int \frac{x+1}{x^2 + 2x + 10}\,dx$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{1}{x^2 + 4x + 13}\,dx$

$\Delta = 16 - 52 = -36 < 0$，分母不可约。

配方：$x^2 + 4x + 13 = (x^2 + 4x + 4) + 9 = (x+2)^2 + 3^2$

由定理 6.5（$a=3$，平移 $u=x+2$）：
$$\int \frac{1}{x^2+4x+13}\,dx = \int \frac{1}{(x+2)^2+3^2}\,dx = \frac{1}{3}\arctan\frac{x+2}{3} + C$$

**验证**：
$$\frac{d}{dx}\left[\frac13\arctan\frac{x+2}{3} + C\right] = \frac13\cdot\frac{1}{1+(\frac{x+2}{3})^2}\cdot\frac13 = \frac19\cdot\frac{9}{9+(x+2)^2} = \frac{1}{(x+2)^2+9} = \frac{1}{x^2+4x+13} \quad \checkmark$$

---

**(2)** $\displaystyle\int \frac{x+1}{x^2 + 2x + 10}\,dx$

$\Delta = 4 - 40 = -36 < 0$，分母不可约。

分母的导数：$(x^2+2x+10)' = 2x+2$。分子 $x+1 = \frac12(2x+2)$，恰好是分母导数的一半。

由定理 6.6（对数积分模式）：
$$\int \frac{x+1}{x^2+2x+10}\,dx = \frac12\int \frac{2x+2}{x^2+2x+10}\,dx = \frac12\ln(x^2+2x+10) + C$$

（分母 $(x+1)^2+9 > 0$ 恒为正，省略绝对值。）

**验证**：
$$\frac{d}{dx}\left[\frac12\ln(x^2+2x+10) + C\right] = \frac12\cdot\frac{2x+2}{x^2+2x+10} = \frac{x+1}{x^2+2x+10} \quad \checkmark$$

</details>

### 迁移应用

**4.** 计算 $\displaystyle\int \frac{x^2 + 2x - 1}{x^2 - 1}\,dx$（提示：先做多项式除法，再对余式做部分分式分解）。

<details><summary>参考答案</summary>

分子 $\deg=2$，分母 $\deg=2$，假分式。做多项式除法：
$$\frac{x^2+2x-1}{x^2-1} = \frac{(x^2-1) + 2x}{x^2-1} = 1 + \frac{2x}{x^2-1}$$

对 $\frac{2x}{(x-1)(x+1)}$ 做部分分式分解：

设 $\frac{2x}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}$。两边同乘 $(x-1)(x+1)$：$2x = A(x+1) + B(x-1)$。

令 $x=1$：$2 = A(2) \Rightarrow A = 1$。
令 $x=-1$：$-2 = B(-2) \Rightarrow B = 1$。

因此：
$$\frac{2x}{x^2-1} = \frac{1}{x-1} + \frac{1}{x+1}$$

于是：
$$\begin{aligned}
\int \frac{x^2+2x-1}{x^2-1}\,dx
&= \int 1\,dx + \int \frac{1}{x-1}\,dx + \int \frac{1}{x+1}\,dx \\
&= x + \ln|x-1| + \ln|x+1| + C \\
&= x + \ln|x^2-1| + C
\end{aligned}$$

**验证**：
$$\frac{d}{dx}\left[x + \ln|x^2-1| + C\right] = 1 + \frac{2x}{x^2-1} = \frac{x^2+2x-1}{x^2-1} \quad \checkmark$$

</details>

---

**5.** 计算 $\displaystyle\int \frac{3x + 1}{x^2 + 2x + 2}\,dx$。

<details><summary>参考答案</summary>

$\Delta = 4 - 8 = -4 < 0$，分母不可约。

分母的导数：$(x^2+2x+2)' = 2x+2$。

拆分分子：$3x+1 = \frac32(2x+2) + (1-3) = \frac32(2x+2) - 2$

因此：
$$\int \frac{3x+1}{x^2+2x+2}\,dx = \frac32\int \frac{2x+2}{x^2+2x+2}\,dx - 2\int \frac{1}{x^2+2x+2}\,dx$$

第一部分（对数积分模式，定理 6.6）：
$$\int \frac{2x+2}{x^2+2x+2}\,dx = \ln(x^2+2x+2) + C_1$$

第二部分（配方 + arctan，定理 6.5）：
$$x^2+2x+2 = (x+1)^2 + 1^2$$
$$\int \frac{1}{x^2+2x+2}\,dx = \int \frac{1}{(x+1)^2+1^2}\,dx = \arctan(x+1) + C_2$$

合并：
$$\int \frac{3x+1}{x^2+2x+2}\,dx = \frac32\ln(x^2+2x+2) - 2\arctan(x+1) + C$$

**验证**：
$$\frac{d}{dx}\left[\frac32\ln(x^2+2x+2) - 2\arctan(x+1) + C\right] = \frac32\cdot\frac{2x+2}{x^2+2x+2} - 2\cdot\frac{1}{1+(x+1)^2}$$
$$= \frac{3x+3}{x^2+2x+2} - \frac{2}{x^2+2x+2} = \frac{3x+1}{x^2+2x+2} \quad \checkmark$$

</details>

---

**6.** （综合题）计算 $\displaystyle\int \frac{2x^2 + x + 1}{x^2 - 3x + 2}\,dx$。

<details><summary>参考答案</summary>

分子 $\deg=2$，分母 $\deg=2$，假分式。先做多项式除法。

将分子写为分母的倍数加余项：
$$2x^2 + x + 1 = 2(x^2-3x+2) + (7x - 3)$$

因为 $2(x^2-3x+2) = 2x^2 - 6x + 4$，故 $(2x^2+x+1) - (2x^2-6x+4) = 7x-3$。

所以：
$$\frac{2x^2+x+1}{x^2-3x+2} = 2 + \frac{7x-3}{x^2-3x+2}$$

因式分解分母：$x^2-3x+2 = (x-1)(x-2)$。

对 $\frac{7x-3}{(x-1)(x-2)}$ 做部分分式分解：

设 $\frac{7x-3}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}$。两边同乘 $(x-1)(x-2)$：$7x-3 = A(x-2) + B(x-1)$。

令 $x=1$：$4 = A(-1) \Rightarrow A = -4$。
令 $x=2$：$11 = B(1) \Rightarrow B = 11$。

因此：
$$\frac{2x^2+x+1}{x^2-3x+2} = 2 - \frac{4}{x-1} + \frac{11}{x-2}$$

于是：
$$\begin{aligned}
\int \frac{2x^2+x+1}{x^2-3x+2}\,dx
&= \int 2\,dx - 4\int \frac{1}{x-1}\,dx + 11\int \frac{1}{x-2}\,dx \\
&= 2x - 4\ln|x-1| + 11\ln|x-2| + C
\end{aligned}$$

**验证**：
$$\begin{aligned}
\frac{d}{dx}\big[2x - 4\ln|x-1| + 11\ln|x-2| + C\big]
&= 2 - \frac{4}{x-1} + \frac{11}{x-2} \\
&= \frac{2(x-1)(x-2) - 4(x-2) + 11(x-1)}{(x-1)(x-2)} \\
&= \frac{2(x^2-3x+2) - 4x+8 + 11x-11}{x^2-3x+2} \\
&= \frac{2x^2-6x+4+7x-3}{x^2-3x+2} = \frac{2x^2+x+1}{x^2-3x+2} \quad \checkmark
\end{aligned}$$

</details>

## 2. 背景与应用场景

在前序文件中，我们学习了有理函数的积分方法（部分分式分解、配方法与 $\arctan$ 公式），它使我们可以积分任何有理函数（两个多项式的商）。然而，工程与科学中经常遇到的被积函数并不是有理函数，而是**三角函数的商（或复合形式）**——例如：

- **交流电路分析**：计算含 $\frac{1}{a + b\cos(\omega t)}$ 形式的电压/电流积分
- **机械振动**：含 $\frac{1}{\sin x + \cos x}$ 的积分出现在某些非线性振动模型的求解中
- **天体力学**：开普勒方程中的积分经常涉及 $\int \frac{d\theta}{1 + e\cos\theta}$（$e$ 为离心率）
- **几何学**：计算椭圆弧长等涉及 $\int \frac{dx}{a + b\cos x}$ 类型的椭圆积分（其中一部分可通过万能代换处理）

所有这些被积函数都有一个共同特征：它们是 $\sin x$ 和 $\cos x$ 的**有理函数**。如果我们将 $\sin x$ 和 $\cos x$ 看作"变量"，那么 $R(u, v)$ 就是关于 $u, v$ 的有理函数。

万能代换法的核心思想是：通过代换 $t = \tan\frac{x}{2}$，将 $\sin x$ 和 $\cos x$ 都表示为 $t$ 的有理函数，从而**将三角有理式的积分转化为有理函数的积分**。由于这个代换可以处理**一切**三角有理式的积分，因此被称为"万能"代换。

---

## 3. 核心概念与符号约定

### 3.1 三角有理函数的定义

**定义 6.3（三角有理函数 / Trigonometric Rational Function）**：设 $R(u, v)$ 是 $u, v$ 的二元有理函数，即 $R(u, v) = \frac{P(u, v)}{Q(u, v)}$，其中 $P, Q$ 是 $u, v$ 的多项式。则
$$R(\sin x, \cos x)$$
称为一个**三角有理函数**。关于 $\sin x$ 和 $\cos x$ 的任何四则运算组合均属于此类。

**示例**：
- $\displaystyle\frac{1}{1 + \cos x}$（分子为常数，分母含 $\cos x$）
- $\displaystyle\frac{1}{\sin x + \cos x}$（分母为 $\sin x$ 与 $\cos x$ 的线性组合）
- $\displaystyle\frac{1}{2 + \sin x}$（分母为常数与 $\sin x$ 之和）
- $\displaystyle\frac{\sin^2 x + 3\cos x}{\cos x - \sin x + 1}$（更复杂的有理组合）

### 3.2 万能代换策略概览

万能代换法的基本流程如下：

```
三角有理式 ∫ R(sin x, cos x) dx
        │
        ▼  Step 1: 令 t = tan(x/2)，代入代换公式
        │
∫ R( 2t/(1+t²), (1-t²)/(1+t²) ) · 2/(1+t²) dt
        │
        ▼  Step 2: 化简为有理函数 ∫ S(t) dt
        │
有理函数的积分（见文件04）
   ├── 部分分式分解（分母可分解为一次因式乘积）
   └── 配方 + arctan（分母为不可约二次因式）
        │
        ▼  Step 3: t = tan(x/2) 回代
最终结果（用 x 表示）
```

### 3.3 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $R(u, v)$ | 关于 $u, v$ 的二元有理函数 | $\frac{1}{u+v}$ |
| $t = \tan\frac{x}{2}$ | 万能代换的中间变量 | $x = \pi \;\Rightarrow\; t = \tan\frac{\pi}{2}$ 无定义 |
| $\sin x = \frac{2t}{1+t^2}$ | $t$ 表示的 $\sin x$ | $x = \frac{\pi}{2} \;\Rightarrow\; t=1,\; \sin x = \frac{2}{2}=1$ |
| $\cos x = \frac{1-t^2}{1+t^2}$ | $t$ 表示的 $\cos x$ | $x = \pi \;\Rightarrow\; t \to \infty$（极限情形）|
| $dx = \frac{2}{1+t^2}\,dt$ | $t$ 表示的 $dx$ | 代换的关键组成部分 |

**注**：万能代换要求 $\cos\frac{x}{2} \neq 0$，即 $x \neq (2k+1)\pi$（$k \in \mathbb{Z}$）。在这些点处 $t = \tan\frac{x}{2}$ 无定义。但在实际积分中，这些点通常是可去间断点或积分的收敛点，不影响公式的使用——我们可以在 $\cos\frac{x}{2} \neq 0$ 的区间上积分，再通过连续性延拓到全定义域。

---

## 4. 原理与方法

### 4.1 万能代换公式的推导（定理 6.7）

**定理 6.7（万能代换公式 / Universal Substitution）**：设 $t = \tan\frac{x}{2}$，其中 $x \neq (2k+1)\pi$（$k \in \mathbb{Z}$），则 $\sin x$、$\cos x$ 和 $dx$ 可以表示为 $t$ 的有理函数：
$$\boxed{\sin x = \frac{2t}{1+t^2}, \qquad \cos x = \frac{1-t^2}{1+t^2}, \qquad dx = \frac{2}{1+t^2}\,dt}$$

**证明**：
我们从 $t = \tan\frac{x}{2}$ 出发，分三步推导三个公式。

---

**第一步：用 $t$ 表示 $\sin\frac{x}{2}$ 和 $\cos\frac{x}{2}$。**

由 $t = \tan\frac{x}{2} = \frac{\sin\frac{x}{2}}{\cos\frac{x}{2}}$，得 $\sin\frac{x}{2} = t\cos\frac{x}{2}$。

代入三角恒等式 $\sin^2\frac{x}{2} + \cos^2\frac{x}{2} = 1$：
$$t^2\cos^2\frac{x}{2} + \cos^2\frac{x}{2} = 1$$
$$\cos^2\frac{x}{2}\,(t^2 + 1) = 1$$
$$\cos^2\frac{x}{2} = \frac{1}{1+t^2}$$

由于在 $x \in (-\pi, \pi)$ 的主值区间内 $\cos\frac{x}{2} > 0$，可取正平方根：
$$\cos\frac{x}{2} = \frac{1}{\sqrt{1+t^2}}$$

进而：
$$\sin\frac{x}{2} = t\cos\frac{x}{2} = \frac{t}{\sqrt{1+t^2}}$$

---

**第二步：推导 $\sin x$ 公式。**

利用倍角公式 $\sin x = 2\sin\frac{x}{2}\cos\frac{x}{2}$：
$$\begin{aligned}
\sin x &= 2 \cdot \frac{t}{\sqrt{1+t^2}} \cdot \frac{1}{\sqrt{1+t^2}} \\
&= \frac{2t}{1+t^2}
\end{aligned}$$

这就是 $\sin x$ 的 $t$ 表达式。

---

**第三步：推导 $\cos x$ 公式。**

利用倍角公式 $\cos x = \cos^2\frac{x}{2} - \sin^2\frac{x}{2}$：
$$\begin{aligned}
\cos x &= \frac{1}{1+t^2} - \frac{t^2}{1+t^2} \\
&= \frac{1 - t^2}{1+t^2}
\end{aligned}$$

这就是 $\cos x$ 的 $t$ 表达式。

---

**第四步：推导 $dx = \frac{2}{1+t^2}\,dt$。**

对 $t = \tan\frac{x}{2}$ 两边对 $x$ 求导（利用链式法则和 $\frac{d}{du}\tan u = \sec^2 u = 1 + \tan^2 u$）：
$$\frac{dt}{dx} = \frac{d}{dx}\left(\tan\frac{x}{2}\right) = \sec^2\frac{x}{2} \cdot \frac{1}{2} = \frac{1}{2}\left(1 + \tan^2\frac{x}{2}\right) = \frac{1 + t^2}{2}$$

因此：
$$\frac{dx}{dt} = \frac{2}{1+t^2}$$

即：
$$dx = \frac{2}{1+t^2}\,dt$$

证毕。

---

**记忆技巧**：三个公式有一个统一的二阶分母 $1 + t^2$：
- $\sin x$ 的分子是 $2t$（"正"弦与正数 $t$ 相关）
- $\cos x$ 的分子是 $1 - t^2$（"余"弦与平方差相关）
- $dx$ 的分子是 $2$（常数），分母相同

### 4.2 万能代换法的一般步骤

对于形如 $\int R(\sin x, \cos x)\,dx$ 的积分，万能代换法遵循以下标准流程。

**Step 1：代入万能代换公式。**
令 $t = \tan\frac{x}{2}$，将 $\sin x$、$\cos x$、$dx$ 全部替换为 $t$ 的表达式：
$$\int R(\sin x, \cos x)\,dx = \int R\!\left(\frac{2t}{1+t^2},\; \frac{1-t^2}{1+t^2}\right) \cdot \frac{2}{1+t^2}\,dt$$

**Step 2：化简为有理函数的积分。**
将上一步的被积函数整理、通分、合并，得到关于 $t$ 的有理函数 $S(t)$。化简后的有理函数 $S(t) = \frac{P(t)}{Q(t)}$ 的分母 $Q(t)$ 通常是一个关于 $t$ 的二次多项式 $at^2 + bt + c$ 或可分解的二次因式。

**Step 3：使用有理函数积分法计算 $\int S(t)\,dt$。**
根据 $S(t)$ 的特征选择适当的方法（见文件 04 有理函数的积分）：
- 若分母可分解为不同一次因式的乘积：**部分分式分解** $\to$ $\ln$ 形式
- 若分母为不可约二次因式：**配方法 + $\arctan$ 公式**（定理 6.5）
- 若为假分式：先做**多项式长除法**

**Step 4：代回 $t = \tan\frac{x}{2}$。**
将积分结果中的 $t$ 全部替换为 $\tan\frac{x}{2}$，得到用 $x$ 表示的最终结果。

**Step 5（验证）：对结果求导。**
检查导数是否等于原被积函数。验证过程通常需要运用链式法则和三角恒等式。

### 4.3 关于万能代换的进一步说明

**"万能"的代价**：万能代换对一切三角有理式都有效，但有时会导致复杂的（高阶或有理根的分母）有理函数积分。对于某些特殊形式，使用更简洁的代换可以减少计算量。然而，在本章中，我们以**万能代换**作为三角有理积分的主要方法——它虽不是最快的，但总是可行的。

**常见类型**：通过万能代换，三角有理积分通常归约为以下三类有理函数积分：

| 三角有理式类型 | 代换后有理函数特征 | 处理方法 |
|---------------|-------------------|---------|
| 分母为 $1 \pm \cos x$ 型 | $\int k\,dt$（常数）或 $\int \frac{dt}{t^2}$ | 直接积分 |
| 分母含 $\sin x$ 与常数 | $\int \frac{dt}{at^2 + bt + c}$ | 配方 + $\arctan$（定理 6.5）|
| 分母含 $\sin x$ 与 $\cos x$ 线性组合 | $\int \frac{dt}{at^2 + bt + c}$（可分解）| 部分分式分解（文件 04）|

## 5. 例题

### 例 1：最简情形——分母为 $1 + \cos x$

计算 $\displaystyle\int \frac{dx}{1 + \cos x}$。

**解**：

**第 1 步：代入万能代换公式。**
令 $t = \tan\frac{x}{2}$，由定理 6.7：$\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

代入被积表达式：
$$\begin{aligned}
\int \frac{dx}{1 + \cos x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{1 + \frac{1-t^2}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{(1+t^2) + (1-t^2)}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{2}{1+t^2}} \\
&= \int 1 \cdot dt
\end{aligned}$$

**第 2 步：对有理函数积分。**
$$\int 1 \cdot dt = t + C_0$$

**第 3 步：回代。**
将 $t = \tan\frac{x}{2}$ 代回：
$$\int \frac{dx}{1 + \cos x} = \tan\frac{x}{2} + C$$

**第 4 步：验证。**
对结果求导，由链式法则：
$$\begin{aligned}
\frac{d}{dx}\left(\tan\frac{x}{2} + C\right)
&= \sec^2\frac{x}{2} \cdot \frac{1}{2} \\
&= \frac{1}{2\cos^2\frac{x}{2}}
\end{aligned}$$

利用半角公式 $\cos^2\frac{x}{2} = \frac{1 + \cos x}{2}$（由 $\cos x = 2\cos^2\frac{x}{2} - 1$ 解出）：
$$\frac{d}{dx}\left(\tan\frac{x}{2} + C\right) = \frac{1}{2 \cdot \frac{1 + \cos x}{2}} = \frac{1}{1 + \cos x}$$

与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{dx}{1 + \cos x} = \tan\frac{x}{2} + C$

---

### 例 2：分母含 $\sin x$ 与常数——配方法 + $\arctan$

计算 $\displaystyle\int \frac{dx}{2 + \sin x}$。

**解**：

**第 1 步：代入万能代换公式。**
令 $t = \tan\frac{x}{2}$，由定理 6.7：$\sin x = \frac{2t}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

代入被积表达式：
$$\begin{aligned}
\int \frac{dx}{2 + \sin x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{2 + \frac{2t}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{2(1+t^2) + 2t}{1+t^2}} \\
&= \int \frac{2\,dt}{2(1+t^2) + 2t} \\
&= \int \frac{2\,dt}{2t^2 + 2t + 2}
\end{aligned}$$

分子分母同除以 2，化简得：
$$\int \frac{dx}{2 + \sin x} = \int \frac{dt}{t^2 + t + 1}$$

**第 2 步：确定有理函数类型。**
分母 $t^2 + t + 1$ 的判别式：$\Delta = 1^2 - 4 \cdot 1 \cdot 1 = 1 - 4 = -3 < 0$。

分母为不可约二次因式，需用配方法 + $\arctan$ 公式（定理 6.5）。

**第 3 步：配方并对有理函数积分。**
$$t^2 + t + 1 = \left(t^2 + t + \frac{1}{4}\right) + \frac{3}{4} = \left(t + \frac{1}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}\right)^2$$

因此：
$$\int \frac{dt}{t^2 + t + 1} = \int \frac{dt}{\left(t + \frac{1}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}\right)^2}$$

令 $u = t + \frac{1}{2}$，$a = \frac{\sqrt{3}}{2}$，$du = dt$，由定理 6.5（$\int \frac{du}{u^2 + a^2} = \frac{1}{a}\arctan\frac{u}{a} + C$）：
$$\begin{aligned}
\int \frac{dt}{\left(t + \frac{1}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}\right)^2}
&= \frac{1}{\frac{\sqrt{3}}{2}} \arctan\frac{t + \frac{1}{2}}{\frac{\sqrt{3}}{2}} + C_0 \\
&= \frac{2}{\sqrt{3}} \arctan\frac{2(t + \frac{1}{2})}{\sqrt{3}} + C_0 \\
&= \frac{2}{\sqrt{3}} \arctan\frac{2t + 1}{\sqrt{3}} + C_0
\end{aligned}$$

**第 4 步：回代。**
将 $t = \tan\frac{x}{2}$ 代回：
$$\int \frac{dx}{2 + \sin x} = \frac{2}{\sqrt{3}} \arctan\frac{2\tan\frac{x}{2} + 1}{\sqrt{3}} + C$$

**第 5 步：验证。**
令 $t = \tan\frac{x}{2}$，$F(t) = \frac{2}{\sqrt{3}} \arctan\frac{2t+1}{\sqrt{3}}$，则需验证 $\frac{d}{dx}F(t(x)) = \frac{1}{2+\sin x}$。

先对 $t$ 求导：
$$F'(t) = \frac{2}{\sqrt{3}} \cdot \frac{1}{1 + \left(\frac{2t+1}{\sqrt{3}}\right)^2} \cdot \frac{2}{\sqrt{3}} = \frac{4}{3} \cdot \frac{1}{1 + \frac{(2t+1)^2}{3}} = \frac{4}{3} \cdot \frac{3}{3 + (2t+1)^2}$$

展开分母：$(2t+1)^2 = 4t^2 + 4t + 1$，因此 $3 + (2t+1)^2 = 4t^2 + 4t + 4 = 4(t^2 + t + 1)$。

故：
$$F'(t) = \frac{4}{3} \cdot \frac{3}{4(t^2 + t + 1)} = \frac{1}{t^2 + t + 1}$$

由链式法则，$\frac{dF}{dx} = F'(t) \cdot \frac{dt}{dx}$。而 $\frac{dt}{dx} = \frac{1 + t^2}{2}$（见定理 6.7 证明第四步）：

$$\frac{dF}{dx} = \frac{1}{t^2 + t + 1} \cdot \frac{1 + t^2}{2}$$

现在转化原被积函数 $\frac{1}{2+\sin x}$ 为 $t$ 的表达式。由第 1 步的化简过程：
$$\frac{1}{2 + \sin x} = \frac{1+t^2}{2(t^2 + t + 1)}$$

因此：
$$\frac{dF}{dx} = \frac{1 + t^2}{2(t^2 + t + 1)} = \frac{1}{2 + \sin x}$$

与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{dx}{2 + \sin x} = \frac{2}{\sqrt{3}} \arctan\frac{2\tan\frac{x}{2} + 1}{\sqrt{3}} + C$

---

### 例 3：分母为 $\sin x$ 与 $\cos x$ 的线性组合——部分分式分解

计算 $\displaystyle\int \frac{dx}{\sin x + \cos x}$。

**解**：

**第 1 步：代入万能代换公式。**
令 $t = \tan\frac{x}{2}$，由定理 6.7：$\sin x = \frac{2t}{1+t^2}$，$\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

代入被积表达式：
$$\begin{aligned}
\int \frac{dx}{\sin x + \cos x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{2t}{1+t^2} + \frac{1-t^2}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{2t + 1 - t^2}{1+t^2}} \\
&= \int \frac{2\,dt}{2t + 1 - t^2}
\end{aligned}$$

整理分母符号，将二次项变为正号：
$$\int \frac{2\,dt}{2t + 1 - t^2} = -2\int \frac{dt}{t^2 - 2t - 1}$$

**第 2 步：因式分解分母并做部分分式分解。**
分母 $t^2 - 2t - 1$ 的判别式：$\Delta = (-2)^2 - 4 \cdot 1 \cdot (-1) = 4 + 4 = 8 > 0$。

说明分母可以分解为两个不同一次因式。解方程 $t^2 - 2t - 1 = 0$：
$$t = \frac{2 \pm \sqrt{8}}{2} = 1 \pm \sqrt{2}$$

因此：
$$t^2 - 2t - 1 = (t - 1 - \sqrt{2})(t - 1 + \sqrt{2})$$

设部分分式分解形式：
$$\frac{1}{t^2 - 2t - 1} = \frac{A}{t - 1 - \sqrt{2}} + \frac{B}{t - 1 + \sqrt{2}}$$

**第 3 步：求解待定系数。**
两边同乘 $(t - 1 - \sqrt{2})(t - 1 + \sqrt{2})$：
$$1 = A(t - 1 + \sqrt{2}) + B(t - 1 - \sqrt{2})$$

用代入法求系数。令 $t = 1 + \sqrt{2}$（此时 $t - 1 - \sqrt{2} = 0$，含 $B$ 的项消失）：
$$1 = A[(1+\sqrt{2}) - 1 + \sqrt{2}] = A(2\sqrt{2}) \;\Longrightarrow\; A = \frac{1}{2\sqrt{2}}$$

令 $t = 1 - \sqrt{2}$（此时 $t - 1 + \sqrt{2} = 0$，含 $A$ 的项消失）：
$$1 = B[(1-\sqrt{2}) - 1 - \sqrt{2}] = B(-2\sqrt{2}) \;\Longrightarrow\; B = -\frac{1}{2\sqrt{2}}$$

因此：
$$\frac{1}{t^2 - 2t - 1} = \frac{1}{2\sqrt{2}}\left(\frac{1}{t - 1 - \sqrt{2}} - \frac{1}{t - 1 + \sqrt{2}}\right)$$

**第 4 步：逐项积分。**
注意原积分有系数 $-2$：
$$\begin{aligned}
\int \frac{dx}{\sin x + \cos x}
&= -2 \int \frac{dt}{t^2 - 2t - 1} \\
&= -2 \cdot \frac{1}{2\sqrt{2}} \left(\int \frac{dt}{t - 1 - \sqrt{2}} - \int \frac{dt}{t - 1 + \sqrt{2}}\right) \\
&= -\frac{1}{\sqrt{2}}\Big(\ln|t - 1 - \sqrt{2}| - \ln|t - 1 + \sqrt{2}|\Big) + C_0 \\
&= -\frac{1}{\sqrt{2}}\ln\left|\frac{t - 1 - \sqrt{2}}{t - 1 + \sqrt{2}}\right| + C_0
\end{aligned}$$

其中最后一步利用了定理 6.4（$\int \frac{1}{x-a}\,dx = \ln|x-a| + C$）。

**第 5 步：回代。**
将 $t = \tan\frac{x}{2}$ 代回：
$$\int \frac{dx}{\sin x + \cos x} = -\frac{1}{\sqrt{2}}\ln\left|\frac{\tan\frac{x}{2} - 1 - \sqrt{2}}{\tan\frac{x}{2} - 1 + \sqrt{2}}\right| + C$$

**第 6 步：验证。**
令 $F(x) = -\frac{1}{\sqrt{2}} \ln\left|\frac{\tan\frac{x}{2} - 1 - \sqrt{2}}{\tan\frac{x}{2} - 1 + \sqrt{2}}\right|$。

将对数差展开以便求导：
$$F(x) = -\frac{1}{\sqrt{2}}\Big(\ln|t - 1 - \sqrt{2}| - \ln|t - 1 + \sqrt{2}|\Big)\Big|_{t = \tan\frac{x}{2}}$$

由链式法则，先对内部 $t$ 求导，再乘以 $\frac{dt}{dx} = \frac{1 + t^2}{2}$：
$$\begin{aligned}
F'(x) &= -\frac{1}{\sqrt{2}}\left(\frac{1}{t - 1 - \sqrt{2}} \cdot \frac{dt}{dx} - \frac{1}{t - 1 + \sqrt{2}} \cdot \frac{dt}{dx}\right) \\
&= -\frac{1}{\sqrt{2}} \cdot \frac{dt}{dx} \cdot \frac{(t - 1 + \sqrt{2}) - (t - 1 - \sqrt{2})}{(t - 1 - \sqrt{2})(t - 1 + \sqrt{2})} \\
&= -\frac{1}{\sqrt{2}} \cdot \frac{1 + t^2}{2} \cdot \frac{2\sqrt{2}}{t^2 - 2t - 1} \\
&= -\frac{1 + t^2}{t^2 - 2t - 1}
\end{aligned}$$

现在将原被积函数 $\frac{1}{\sin x + \cos x}$ 用 $t$ 表示。由第 1 步的化简：
$$\frac{1}{\sin x + \cos x} = \frac{1 + t^2}{2t + 1 - t^2}$$

注意 $2t + 1 - t^2 = -(t^2 - 2t - 1)$，所以：
$$\frac{1}{\sin x + \cos x} = \frac{1 + t^2}{-(t^2 - 2t - 1)} = -\frac{1 + t^2}{t^2 - 2t - 1}$$

因此 $F'(x)$ 恰好等于 $\frac{1}{\sin x + \cos x}$。$\checkmark$

**答案**：$\displaystyle\int \frac{dx}{\sin x + \cos x} = -\frac{1}{\sqrt{2}}\ln\left|\frac{\tan\frac{x}{2} - 1 - \sqrt{2}}{\tan\frac{x}{2} - 1 + \sqrt{2}}\right| + C$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| $t = \tan\frac{x}{2}$ 代换后 $dx = \frac{1}{1+t^2}\,dt$（忘记乘以 2）或 $dx = \frac{2t}{1+t^2}\,dt$（与 $\sin x$ 公式混淆） | 由 $\frac{dt}{dx} = \frac{1}{2}(1+t^2)$ 正确推导得 $dx = \frac{2}{1+t^2}\,dt$。注意 $\sin x$ 的分子是 $2t$，而 $dx$ 的分子是 $2$ |
| 代入万能代换公式后不化简就直接开始积分 | 代入后必须通分化简至最简有理函数形式。例如例 1 中化简后得 $\int 1\cdot dt$，若未化简就积分会得到错误结果 |
| 认为 $t = \tan\frac{x}{2}$ 在 $x = \pi$ 处无定义，所以公式对 $x = \pi$ 附近不适用 | 万能代换公式是在 $\cos\frac{x}{2} \neq 0$ 的区间上推导的，但积分结果通过连续性可延拓到全定义域（包括 $x = \pi$）。使用公式时无需担心这些奇点 |
| 回代时忘记将结果中的所有 $t$ 替换回 $\tan\frac{x}{2}$ | 积分结果用 $t$ 表达后，必须执行 $t = \tan\frac{x}{2}$ 的回代，最终答案必须用原变量 $x$ 表示 |
| 对三角有理积分只用万能代换，不会判断何时有更简单的代换 | 万能代换"万能"但未必最高效。例如 $\int \sin^2 x \cos x\,dx$ 用 $u = \sin x$ 更简单。但判断能力需要通过练习逐步积累，初学者先用万能代换保证可解性 |

## 练习题

### 基础巩固

**1.** 使用万能代换 $t = \tan\frac{x}{2}$，计算下列不定积分：

(1) $\displaystyle\int \frac{dx}{1 - \cos x}$

(2) $\displaystyle\int \frac{dx}{1 + \sin x}$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{dx}{1 - \cos x}$

令 $t = \tan\frac{x}{2}$，则 $\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

$$\begin{aligned}
\int \frac{dx}{1 - \cos x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{1 - \frac{1-t^2}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{(1+t^2) - (1-t^2)}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{2t^2}{1+t^2}} \\
&= \int \frac{dt}{t^2} \\
&= -\frac{1}{t} + C \\
&= -\cot\frac{x}{2} + C
\end{aligned}$$

**验证**：
$$\begin{aligned}
\frac{d}{dx}\left(-\cot\frac{x}{2} + C\right)
&= \csc^2\frac{x}{2} \cdot \frac{1}{2} = \frac{1}{2\sin^2\frac{x}{2}} \\
&= \frac{1}{2 \cdot \frac{1-\cos x}{2}} = \frac{1}{1 - \cos x} \quad \checkmark
\end{aligned}$$

**答案**：$\displaystyle\int \frac{dx}{1 - \cos x} = -\cot\frac{x}{2} + C$

---

**(2)** $\displaystyle\int \frac{dx}{1 + \sin x}$

令 $t = \tan\frac{x}{2}$，则 $\sin x = \frac{2t}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

$$\begin{aligned}
\int \frac{dx}{1 + \sin x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{1 + \frac{2t}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{1+t^2+2t}{1+t^2}} \\
&= \int \frac{2\,dt}{t^2 + 2t + 1} \\
&= \int \frac{2\,dt}{(t+1)^2} \\
&= -\frac{2}{t+1} + C
\end{aligned}$$

回代 $t = \tan\frac{x}{2}$：
$$\int \frac{dx}{1 + \sin x} = -\frac{2}{\tan\frac{x}{2} + 1} + C$$

（注意此题也有更简洁的答案 $\tan x - \sec x + C$，但通过万能代换得到的结果也是正确的，二者相差一个常数。）

**验证**（对万能代换结果求导）：
令 $F(x) = -2\left(\tan\frac{x}{2} + 1\right)^{-1}$，由链式法则：
$$F'(x) = -2 \cdot (-1) \left(\tan\frac{x}{2} + 1\right)^{-2} \cdot \frac{1}{2}\sec^2\frac{x}{2} = \frac{\sec^2\frac{x}{2}}{(\tan\frac{x}{2} + 1)^2}$$

而 $\frac{1}{1+\sin x} = \frac{1}{1 + \frac{2t}{1+t^2}} = \frac{1+t^2}{(t+1)^2}$，且 $\sec^2\frac{x}{2} = 1 + t^2$，因此 $F'(x) = \frac{1+t^2}{(t+1)^2} = \frac{1}{1+\sin x}$。$\checkmark$

**答案**：$\displaystyle\int \frac{dx}{1 + \sin x} = -\frac{2}{\tan\frac{x}{2} + 1} + C$

</details>

**2.** 计算 $\displaystyle\int \frac{dx}{3 + 5\cos x}$。

<details><summary>参考答案</summary>

令 $t = \tan\frac{x}{2}$，则 $\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

$$\begin{aligned}
\int \frac{dx}{3 + 5\cos x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{3 + 5\cdot\frac{1-t^2}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{3(1+t^2) + 5(1-t^2)}{1+t^2}} \\
&= \int \frac{2\,dt}{3(1+t^2) + 5(1-t^2)} \\
&= \int \frac{2\,dt}{3 + 3t^2 + 5 - 5t^2} \\
&= \int \frac{2\,dt}{8 - 2t^2} \\
&= \int \frac{dt}{4 - t^2}
\end{aligned}$$

分母 $4 - t^2 = (2-t)(2+t)$，可分解为一次因式乘积。用部分分式分解：

设 $\frac{1}{4-t^2} = \frac{A}{2-t} + \frac{B}{2+t}$。

两边同乘 $(2-t)(2+t)$：$1 = A(2+t) + B(2-t)$。

令 $t = 2$：$1 = A(4) \Rightarrow A = \frac{1}{4}$。
令 $t = -2$：$1 = B(4) \Rightarrow B = \frac{1}{4}$。

因此：
$$\begin{aligned}
\int \frac{dx}{3 + 5\cos x}
&= \frac{1}{4}\int \frac{dt}{2-t} + \frac{1}{4}\int \frac{dt}{2+t} \\
&= -\frac{1}{4}\ln|2-t| + \frac{1}{4}\ln|2+t| + C_0 \\
&= \frac{1}{4}\ln\left|\frac{2+t}{2-t}\right| + C_0
\end{aligned}$$

回代 $t = \tan\frac{x}{2}$：
$$\int \frac{dx}{3 + 5\cos x} = \frac{1}{4}\ln\left|\frac{2 + \tan\frac{x}{2}}{2 - \tan\frac{x}{2}}\right| + C$$

**验证**（简要）：
$$\frac{d}{dx}\left[\frac{1}{4}\ln\left|\frac{2+t}{2-t}\right|\right]_{t=\tan\frac{x}{2}} = \frac{1}{4} \cdot \frac{dt}{dx} \cdot \frac{(2-t)+(2+t)}{(2+t)(2-t)} = \frac{1}{4} \cdot \frac{1+t^2}{2} \cdot \frac{4}{4-t^2} = \frac{1+t^2}{2(4-t^2)}$$

由化简过程知 $\frac{1}{3+5\cos x} = \frac{1+t^2}{2(4-t^2)}$，故两者相等。$\checkmark$

**答案**：$\displaystyle\int \frac{dx}{3 + 5\cos x} = \frac{1}{4}\ln\left|\frac{2 + \tan\frac{x}{2}}{2 - \tan\frac{x}{2}}\right| + C$

</details>

**3.** 计算 $\displaystyle\int \frac{dx}{\sin x - \cos x}$。

<details><summary>参考答案</summary>

令 $t = \tan\frac{x}{2}$，则 $\sin x = \frac{2t}{1+t^2}$，$\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

$$\begin{aligned}
\int \frac{dx}{\sin x - \cos x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{2t}{1+t^2} - \frac{1-t^2}{1+t^2}} \\
&= \int \frac{2\,dt}{2t - (1-t^2)} \\
&= \int \frac{2\,dt}{t^2 + 2t - 1}
\end{aligned}$$

分母 $t^2 + 2t - 1$，判别式 $\Delta = 4 + 4 = 8 > 0$。

解 $t^2 + 2t - 1 = 0$：$t = \frac{-2 \pm \sqrt{8}}{2} = -1 \pm \sqrt{2}$。

因此 $t^2 + 2t - 1 = (t + 1 - \sqrt{2})(t + 1 + \sqrt{2})$。

设部分分式分解：
$$\frac{1}{t^2 + 2t - 1} = \frac{A}{t + 1 - \sqrt{2}} + \frac{B}{t + 1 + \sqrt{2}}$$

去分母：$1 = A(t + 1 + \sqrt{2}) + B(t + 1 - \sqrt{2})$。

令 $t = -1 + \sqrt{2}$：$1 = A(2\sqrt{2}) \Rightarrow A = \frac{1}{2\sqrt{2}}$。
令 $t = -1 - \sqrt{2}$：$1 = B(-2\sqrt{2}) \Rightarrow B = -\frac{1}{2\sqrt{2}}$。

于是：
$$\begin{aligned}
\int \frac{dx}{\sin x - \cos x}
&= 2 \int \frac{dt}{t^2 + 2t - 1} \\
&= 2 \cdot \frac{1}{2\sqrt{2}} \left(\int \frac{dt}{t + 1 - \sqrt{2}} - \int \frac{dt}{t + 1 + \sqrt{2}}\right) \\
&= \frac{1}{\sqrt{2}}\left(\ln|t + 1 - \sqrt{2}| - \ln|t + 1 + \sqrt{2}|\right) + C_0 \\
&= \frac{1}{\sqrt{2}}\ln\left|\frac{t + 1 - \sqrt{2}}{t + 1 + \sqrt{2}}\right| + C_0
\end{aligned}$$

回代 $t = \tan\frac{x}{2}$：
$$\int \frac{dx}{\sin x - \cos x} = \frac{1}{\sqrt{2}}\ln\left|\frac{\tan\frac{x}{2} + 1 - \sqrt{2}}{\tan\frac{x}{2} + 1 + \sqrt{2}}\right| + C$$

**答案**：$\displaystyle\int \frac{dx}{\sin x - \cos x} = \frac{1}{\sqrt{2}}\ln\left|\frac{\tan\frac{x}{2} + 1 - \sqrt{2}}{\tan\frac{x}{2} + 1 + \sqrt{2}}\right| + C$

</details>

**4.** 计算 $\displaystyle\int \frac{dx}{4 + 3\cos x}$。

<details><summary>参考答案</summary>

令 $t = \tan\frac{x}{2}$，则 $\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2}\,dt$。

$$\begin{aligned}
\int \frac{dx}{4 + 3\cos x}
&= \int \frac{\frac{2}{1+t^2}\,dt}{4 + 3\cdot\frac{1-t^2}{1+t^2}} \\
&= \int \frac{\frac{2}{1+t^2}\,dt}{\frac{4(1+t^2) + 3(1-t^2)}{1+t^2}} \\
&= \int \frac{2\,dt}{4 + 4t^2 + 3 - 3t^2} \\
&= \int \frac{2\,dt}{t^2 + 7}
\end{aligned}$$

判别式$\Delta = 0 - 4\cdot1\cdot7 = -28 < 0$，分母为不可约二次因式，用 $\arctan$ 公式。

$$\int \frac{2\,dt}{t^2 + 7} = 2\int \frac{dt}{t^2 + (\sqrt{7})^2} = 2 \cdot \frac{1}{\sqrt{7}}\arctan\frac{t}{\sqrt{7}} + C_0 = \frac{2}{\sqrt{7}}\arctan\frac{t}{\sqrt{7}} + C_0$$

回代 $t = \tan\frac{x}{2}$：
$$\int \frac{dx}{4 + 3\cos x} = \frac{2}{\sqrt{7}}\arctan\frac{\tan\frac{x}{2}}{\sqrt{7}} + C$$

**验证**：由 $t = \tan\frac{x}{2}$，被积函数 $\frac{1}{4+3\cos x}$ 在 $t$ 下为：
$$\frac{1}{4+3\cos x} = \frac{1}{4 + 3\cdot\frac{1-t^2}{1+t^2}} = \frac{1+t^2}{4(1+t^2) + 3(1-t^2)} = \frac{1+t^2}{t^2+7}$$

令 $F(x) = \frac{2}{\sqrt{7}}\arctan\frac{t}{\sqrt{7}}$（$t = \tan\frac{x}{2}$）。由链式法则：
$$F'(x) = \frac{2}{\sqrt{7}} \cdot \frac{1}{1 + \frac{t^2}{7}} \cdot \frac{1}{\sqrt{7}} \cdot \frac{1+t^2}{2} = \frac{2}{7} \cdot \frac{7}{t^2+7} \cdot \frac{1+t^2}{2} = \frac{1+t^2}{t^2+7}$$

因此 $F'(x) = \frac{1+t^2}{t^2+7} = \frac{1}{4+3\cos x}$。验证通过。$\checkmark$

**答案**：$\displaystyle\int \frac{dx}{4 + 3\cos x} = \frac{2}{\sqrt{7}}\arctan\frac{\tan\frac{x}{2}}{\sqrt{7}} + C$

</details>

## 2. 背景与应用场景

在前序文件中，我们学会了有理函数的积分和三角有理式的积分。然而还有一大类常见的被积函数——**含根式的无理函数**——尚未系统处理。例如：

- **几何计算**：求旋转体的表面积和体积时，被积函数常出现 $\sqrt{a^2 - x^2}$ 或 $\sqrt{x+1}$ 等形式
- **物理中的弧长计算**：曲线 $y = f(x)$ 的弧长公式 $L = \int_a^b \sqrt{1 + [f'(x)]^2}\,dx$ 中的被积函数含有根式
处理含根式函数的积分，核心思想是：**通过代换消去根式，化为已经掌握的有理函数积分**。与万能代换将三角有理式化为有理函数类似，根式代换将无理函数化为有理函数。在本章中，我们只处理**最简形式的根式**——被积函数中仅含 $\sqrt{x}$ 或 $\sqrt{x+a}$ 的情形。对于更复杂的根式（如 $\sqrt{ax^2 + bx + c}$），需要后续学习更多技巧才能处理。

---

## 3. 核心概念与符号约定

### 3.1 简单无理函数的定义

**定义 6.4（简单无理函数 / Simple Irrational Function）**：若被积函数是变量 $x$ 与形如 $\sqrt{x}$、$\sqrt{x+a}$（$a$ 为常数）等根式的有理组合（即通过四则运算组合而成），则称为简单无理函数。

**示例**：
- $\displaystyle\frac{1}{1 + \sqrt{x}}$（分母含 $\sqrt{x}$ 与常数 1 的和）
- $\displaystyle\frac{x}{\sqrt{x+1}}$（分子为 $x$，分母为 $\sqrt{x+1}$）
- $\displaystyle\frac{\sqrt{x} - 1}{\sqrt{x} + 1}$（分子和分母都含 $\sqrt{x}$）
- $\displaystyle\frac{1}{x\sqrt{x+1}}$（分母为 $x$ 与 $\sqrt{x+1}$ 的乘积）
- $\sqrt{x} + \frac{2}{\sqrt{x}}$（两项分别含 $\sqrt{x}$ 和 $\frac{1}{\sqrt{x}}$）

### 3.2 根式代换的总体策略

```
∫ f(x, √x) dx 或 ∫ f(x, √(x+a)) dx
  │ Step 1: 选择代换
  ├── 含 √x → t = √x
  └── 含 √(x+a) → t = √(x+a)
  │ Step 2: 写出 x 和 dx
  ├── t = √x: x = t², dx = 2t dt
  └── t = √(x+a): x = t²-a, dx = 2t dt
  │ Step 3: 代入 → 有理函数 ∫ S(t) dt
  │ Step 4: 有理函数积分（长除法/部分分式/配方）
  │ Step 5: 回代 t = √x 或 t = √(x+a) → 结果用 x 表示
```

### 3.3 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $t = \sqrt{x}$ | 针对含 $\sqrt{x}$ 的代换 | $x = t^2$，$dx = 2t\,dt$ |
| $t = \sqrt{x+a}$ | 针对含 $\sqrt{x+a}$ 的代换 | $x = t^2 - a$，$dx = 2t\,dt$ |
| $a$ | 根号内的平移常数 | $\sqrt{x+1}$ 中 $a=1$，$\sqrt{x-1}$ 中 $a=-1$ |
| $S(t)$ | 代换后的有理函数 | $\frac{1}{1+\sqrt{x}} \rightarrow S(t) = \frac{2t}{1+t}$ |

---

## 4. 原理与方法

### 4.1 根式代换法（定理 6.8）

**定理 6.8（根式代换法 / Rationalizing Substitution）**：

**(1)** 若被积函数含有 $\sqrt{x}$，作代换 $t = \sqrt{x}$（$t \ge 0$），则
$$x = t^2,\qquad dx = 2t\,dt$$

**(2)** 若被积函数含有 $\sqrt{x + a}$（$a$ 为常数），作代换 $t = \sqrt{x + a}$（$t \ge 0$），则
$$x = t^2 - a,\qquad dx = 2t\,dt$$

经过上述代换后，原无理函数的积分化为关于 $t$ 的有理函数的积分。

**证明（代换推导）**：

**情形 (1)**：令 $t = \sqrt{x}$。两边平方得 $x = t^2$。在 $x > 0$ 的区间上 $t = \sqrt{x} > 0$，函数 $x = t^2$ 是 $t$ 的可微函数。对 $x = t^2$ 两边关于 $t$ 求导：$\frac{dx}{dt} = 2t$，因此 $dx = 2t\,dt$。代入原积分：
$$\int f(x, \sqrt{x})\,dx = \int f(t^2, t) \cdot 2t\,dt$$

右端的被积函数 $f(t^2, t) \cdot 2t$ 是关于 $t$ 的有理函数（因为 $t^2$ 和 $t$ 都是 $t$ 的多项式，而原被积函数是 $x$ 和 $\sqrt{x}$ 的有理组合）。

**情形 (2)**：令 $t = \sqrt{x + a}$。两边平方得 $t^2 = x + a$，即 $x = t^2 - a$。对 $x$ 关于 $t$ 求导：$\frac{dx}{dt} = 2t$，因此 $dx = 2t\,dt$。代入原积分：
$$\int f(x, \sqrt{x+a})\,dx = \int f(t^2 - a, t) \cdot 2t\,dt$$

右端的被积函数是关于 $t$ 的有理函数。证毕。

**选择代换的规则**：遇到根号内为 $x$ 的一次式（$x$ 或 $x+a$）时，令 $t$ 等于该根式本身。**千万不要令 $t$ 等于 $x$ 或某个其他量**——代换的目标是消去根号，因此被代换的中间变量必须直接等于根式。

---

### 4.2 根式代换的标准流程

对于任意一个含 $\sqrt{x}$ 或 $\sqrt{x+a}$ 的积分，遵循以下五步流程：

**Step 1：识别根式类型，选择代换**。
若被积函数含 $\sqrt{x}$，设 $t = \sqrt{x}$；若含 $\sqrt{x+a}$，设 $t = \sqrt{x+a}$。

**Step 2：写出 $x$ 和 $dx$ 的 $t$ 表达式**。
- 对于 $t = \sqrt{x}$：$x = t^2$，$dx = 2t\,dt$
- 对于 $t = \sqrt{x+a}$：$x = t^2 - a$，$dx = 2t\,dt$

**Step 3：代入化简，得到有理函数积分**。
将 $x$、$\sqrt{x}$（或 $\sqrt{x+a}$）、$dx$ 全部用 $t$ 表示，化简得到关于 $t$ 的有理函数 $\int S(t)\,dt$。

**Step 4：用有理函数积分法计算 $\int S(t)\,dt$**。
运用文件 04 的方法：假分式先做多项式长除法，真分式做部分分式分解或配方 + $\arctan$。

**Step 5：回代**。
将结果中所有 $t$ 替换为 $\sqrt{x}$ 或 $\sqrt{x+a}$，得到用 $x$ 表示的最终结果。然后通过求导验证。

---

### 4.3 代数变形法（替代方案）

对于某些简单情形，可以不进行代换，而是通过代数恒等变形消去根式直接积分。

**常用技巧**：将分子写成与根号内表达式相关的形式。

例如，对于 $\sqrt{x+1}$ 出现在分母中的情形：
$$\frac{x}{\sqrt{x+1}} = \frac{(x+1) - 1}{\sqrt{x+1}} = \sqrt{x+1} - \frac{1}{\sqrt{x+1}} = (x+1)^{1/2} - (x+1)^{-1/2}$$

右端每一项都可以直接用幂函数积分公式计算。

**何时使用代数变形法**：当分子中恰好可以"凑"出根号内的表达式时，代数变形法比代换更简洁。但并非所有情形都能这样做——当无法通过简单恒等变形消去根式时，根式代换总是可靠的通用方法。

---

## 5. 例题

### 例 1：直接积分（无需代换）

计算 $\displaystyle\int \left(\sqrt{x} + \frac{2}{\sqrt{x}}\right)dx \quad (x > 0)$。

**解**：

**第 1 步：化为幂函数形式**。
$$\sqrt{x} = x^{1/2}, \qquad \frac{2}{\sqrt{x}} = 2x^{-1/2}$$

**第 2 步：利用线性性质拆分**（定理 6.2）。
$$\int \left(x^{1/2} + 2x^{-1/2}\right)dx = \int x^{1/2}\,dx + 2\int x^{-1/2}\,dx$$

**第 3 步：套用幂函数积分公式**（表 6.1 公式 1，$n \neq -1$）。

第一项（$n = 1/2$）：
$$\int x^{1/2}\,dx = \frac{x^{3/2}}{3/2} + C_1 = \frac{2}{3}x^{3/2} + C_1$$

第二项（$n = -1/2$）：
$$\int x^{-1/2}\,dx = \frac{x^{1/2}}{1/2} + C_2 = 2x^{1/2} + C_2$$

**第 4 步：合并结果**。
$$\int \left(\sqrt{x} + \frac{2}{\sqrt{x}}\right)dx = \frac{2}{3}x^{3/2} + 4\sqrt{x} + C$$

其中 $C = C_1 + 2C_2$，合并为一个任意常数。

**第 5 步：验证**（对结果求导）。
$$\frac{d}{dx}\left(\frac{2}{3}x^{3/2} + 4x^{1/2} + C\right) = \frac{2}{3} \cdot \frac{3}{2}x^{1/2} + 4 \cdot \frac{1}{2}x^{-1/2} = x^{1/2} + 2x^{-1/2} = \sqrt{x} + \frac{2}{\sqrt{x}} \quad \checkmark$$

**注意**：本例说明并非所有含根式的积分都需要代换——当根式可以直接写成幂函数形式时，直接积分即可。

**答案**：$\displaystyle\int \left(\sqrt{x} + \frac{2}{\sqrt{x}}\right)dx = \frac{2}{3}x^{3/2} + 4\sqrt{x} + C$

---

### 例 2：含 $\sqrt{x}$ 的积分——根式代换 + 多项式长除法

计算 $\displaystyle\int \frac{dx}{1 + \sqrt{x}} \quad (x > 0)$。

**解**：

**第 1 步：选择代换**。被积函数含 $\sqrt{x}$，令 $t = \sqrt{x}$（$t > 0$）。

**第 2 步：写出 $x$ 和 $dx$ 的 $t$ 表达式**。
由 $t = \sqrt{x}$ 得 $x = t^2$，$dx = 2t\,dt$。

**第 3 步：代入化简**。
$$\int \frac{dx}{1 + \sqrt{x}} = \int \frac{2t\,dt}{1 + t} = 2\int \frac{t}{1+t}\,dt$$

右端被积函数 $\frac{t}{1+t}$ 是关于 $t$ 的有理函数。分子分母次数相同（均为 1），是**假分式**。

**第 4 步：化简有理函数——做多项式长除法**。

对 $\frac{t}{1+t}$ 做除法：$t \div (1+t) = 1$ 余 $-1$，即
$$\frac{t}{1+t} = 1 - \frac{1}{1+t}$$

**验算**：$1 - \frac{1}{1+t} = \frac{1+t-1}{1+t} = \frac{t}{1+t}$。$\checkmark$

因此：
$$2\int \frac{t}{1+t}\,dt = 2\int \left(1 - \frac{1}{1+t}\right)dt = 2\int 1\,dt - 2\int \frac{1}{1+t}\,dt$$

**第 5 步：逐项积分**。

第一项：$\displaystyle\int 1\,dt = t + C_1$

第二项：由定理 6.4（$\int \frac{1}{t-a}\,dt = \ln|t-a| + C$），这里 $a = -1$：
$$\int \frac{1}{1+t}\,dt = \ln|1+t| + C_2$$

由于 $t = \sqrt{x} > 0$，$1+t > 0$，可省略绝对值符号。

因此：
$$2\int \frac{t}{1+t}\,dt = 2t - 2\ln(1+t) + C$$

其中 $C = 2C_1 - 2C_2$。

**第 6 步：回代 $t = \sqrt{x}$**。
$$\int \frac{dx}{1 + \sqrt{x}} = 2\sqrt{x} - 2\ln(1 + \sqrt{x}) + C$$

**第 7 步：验证**（对结果求导，利用链式法则）。
$$\begin{aligned}
\frac{d}{dx}\Big[2\sqrt{x} - 2\ln(1+\sqrt{x}) + C\Big]
&= 2 \cdot \frac{1}{2\sqrt{x}} - 2 \cdot \frac{1}{1+\sqrt{x}} \cdot \frac{1}{2\sqrt{x}} \\
&= \frac{1}{\sqrt{x}} - \frac{1}{\sqrt{x}(1+\sqrt{x})} \\
&= \frac{(1+\sqrt{x}) - 1}{\sqrt{x}(1+\sqrt{x})} \\
&= \frac{1}{1+\sqrt{x}} \quad \checkmark
\end{aligned}$$

与被积函数一致。验证通过。

**答案**：$\displaystyle\int \frac{dx}{1 + \sqrt{x}} = 2\sqrt{x} - 2\ln(1 + \sqrt{x}) + C$

### 例 3：含 $\sqrt{x+1}$ 的积分——根式代换 + 部分分式分解

计算 $\displaystyle\int \frac{dx}{x\sqrt{x+1}} \quad (x > 0)$。

**解**：

**第 1 步：选择代换**。被积函数含 $\sqrt{x+1}$，令 $t = \sqrt{x+1}$（$t > 1$，因为 $x > 0$ 时 $t = \sqrt{x+1} > 1$）。

**第 2 步：写出 $x$ 和 $dx$ 的 $t$ 表达式**。
由 $t = \sqrt{x+1}$ 得 $t^2 = x+1$，即 $x = t^2 - 1$。对 $x$ 求导：$\frac{dx}{dt} = 2t$，所以 $dx = 2t\,dt$。

**第 3 步：代入化简**。
$$\int \frac{dx}{x\sqrt{x+1}} = \int \frac{2t\,dt}{(t^2 - 1) \cdot t} = 2\int \frac{dt}{t^2 - 1}$$

右端是关于 $t$ 的有理函数。分母 $t^2 - 1 = (t-1)(t+1)$ 可分解为两个不同一次因式的乘积。分子次数（0）小于分母次数（2），是真分式。

**第 4 步：部分分式分解**。

设 $\frac{1}{t^2 - 1} = \frac{A}{t-1} + \frac{B}{t+1}$。

两边同乘 $(t-1)(t+1)$（去分母）：
$$1 = A(t+1) + B(t-1)$$

用代入法求 $A, B$。

令 $t = 1$（此时 $t-1 = 0$，含 $B$ 的项消失）：
$$1 = A(1+1) = 2A \;\Longrightarrow\; A = \frac{1}{2}$$

令 $t = -1$（此时 $t+1 = 0$，含 $A$ 的项消失）：
$$1 = B(-1-1) = -2B \;\Longrightarrow\; B = -\frac{1}{2}$$

因此：
$$\frac{1}{t^2 - 1} = \frac{1/2}{t-1} + \frac{-1/2}{t+1} = \frac{1}{2}\left(\frac{1}{t-1} - \frac{1}{t+1}\right)$$

**第 5 步：逐项积分**。

注意原积分有系数 $2$：
$$\begin{aligned}
2\int \frac{dt}{t^2 - 1}
&= 2 \cdot \frac{1}{2}\left(\int \frac{dt}{t-1} - \int \frac{dt}{t+1}\right) \\
&= \int \frac{dt}{t-1} - \int \frac{dt}{t+1}
\end{aligned}$$

由定理 6.4（$\int\frac{1}{t-a}dt = \ln|t-a| + C$）：
$$\int \frac{dt}{t-1} = \ln|t-1| + C_1, \qquad \int \frac{dt}{t+1} = \ln|t+1| + C_2$$

因此：
$$2\int \frac{dt}{t^2 - 1} = \ln|t-1| - \ln|t+1| + C = \ln\left|\frac{t-1}{t+1}\right| + C$$

其中 $C = C_1 - C_2$。

**第 6 步：回代 $t = \sqrt{x+1}$**。
$$\int \frac{dx}{x\sqrt{x+1}} = \ln\left|\frac{\sqrt{x+1} - 1}{\sqrt{x+1} + 1}\right| + C$$

**第 7 步：验证**（对结果求导）。

令 $F(x) = \ln\left|\frac{\sqrt{x+1} - 1}{\sqrt{x+1} + 1}\right|$，记 $t = \sqrt{x+1}$。利用对数的性质：
$$F(x) = \ln|t-1| - \ln|t+1|\big|_{t=\sqrt{x+1}}$$

由链式法则，$F'(x) = \left(\frac{1}{t-1} - \frac{1}{t+1}\right) \cdot \frac{dt}{dx}$。

先计算 $\frac{dt}{dx}$：$t = \sqrt{x+1} = (x+1)^{1/2}$，故 $\frac{dt}{dx} = \frac{1}{2}(x+1)^{-1/2} = \frac{1}{2\sqrt{x+1}} = \frac{1}{2t}$。

因此：
$$\begin{aligned}
F'(x) &= \left(\frac{1}{t-1} - \frac{1}{t+1}\right) \cdot \frac{1}{2t} \\
&= \frac{(t+1) - (t-1)}{(t-1)(t+1)} \cdot \frac{1}{2t} \\
&= \frac{2}{t^2-1} \cdot \frac{1}{2t} \\
&= \frac{1}{t(t^2-1)}
\end{aligned}$$

由于 $t^2 - 1 = x$，$t = \sqrt{x+1}$：
$$F'(x) = \frac{1}{\sqrt{x+1} \cdot x} = \frac{1}{x\sqrt{x+1}} \quad \checkmark$$

与被积函数一致。验证通过。

**答案**：$\displaystyle\int \frac{dx}{x\sqrt{x+1}} = \ln\left|\frac{\sqrt{x+1} - 1}{\sqrt{x+1} + 1}\right| + C$

**注**：当 $x > 0$ 时 $\sqrt{x+1} > 1$，因此 $\sqrt{x+1} - 1 > 0$，$\sqrt{x+1} + 1 > 0$，绝对值可省略：
$$\int \frac{dx}{x\sqrt{x+1}} = \ln\frac{\sqrt{x+1} - 1}{\sqrt{x+1} + 1} + C \quad (x > 0)$$

### 例 4：含 $\sqrt{x}$ 的积分——根式代换 + 多项式长除法（分子为二次式）

计算 $\displaystyle\int \frac{\sqrt{x} - 1}{\sqrt{x} + 1}\,dx \quad (x > 0)$。

**解**：

**第 1 步：选择代换**。被积函数含 $\sqrt{x}$，令 $t = \sqrt{x}$（$t > 0$）。

**第 2 步：写出 $x$ 和 $dx$ 的 $t$ 表达式**。
$x = t^2$，$dx = 2t\,dt$。

**第 3 步：代入化简**。
$$\int \frac{\sqrt{x} - 1}{\sqrt{x} + 1}\,dx = \int \frac{t - 1}{t + 1} \cdot 2t\,dt = 2\int \frac{t(t-1)}{t+1}\,dt$$

展开分子：$t(t-1) = t^2 - t$，因此被积函数为 $\frac{t^2 - t}{t+1}$。

**第 4 步：化简有理函数——多项式长除法**。

分子 $\deg=2$，分母 $\deg=1$，是假分式。做多项式长除法：

用分子 $t^2 - t$ 除以分母 $t+1$：
- 首项：$t^2 \div t = t$
- 乘回：$t(t+1) = t^2 + t$
- 相减：$(t^2 - t) - (t^2 + t) = -2t$
- 再除：$-2t \div t = -2$
- 乘回：$-2(t+1) = -2t - 2$
- 相减：$(-2t) - (-2t - 2) = 2$（余数）

因此：
$$\frac{t^2 - t}{t+1} = t - 2 + \frac{2}{t+1}$$

**验算**：$t - 2 + \frac{2}{t+1} = \frac{(t-2)(t+1) + 2}{t+1} = \frac{t^2 - t - 2 + 2}{t+1} = \frac{t^2 - t}{t+1}$。$\checkmark$

**第 5 步：逐项积分**。
$$\begin{aligned}
2\int \frac{t^2 - t}{t+1}\,dt
&= 2\int \left(t - 2 + \frac{2}{t+1}\right)dt \\
&= 2\int t\,dt - 4\int 1\,dt + 4\int \frac{1}{t+1}\,dt
\end{aligned}$$

逐项计算：
- $\displaystyle\int t\,dt = \frac{t^2}{2} + C_1$
- $\displaystyle\int 1\,dt = t + C_2$
- $\displaystyle\int \frac{1}{t+1}\,dt = \ln|t+1| + C_3$（定理 6.4，$a=-1$）

由于 $t = \sqrt{x} \ge 0$，$t+1 > 0$，可省略绝对值。

代入：
$$\begin{aligned}
2\int \frac{t^2 - t}{t+1}\,dt
&= 2 \cdot \frac{t^2}{2} - 4t + 4\ln(t+1) + C \\
&= t^2 - 4t + 4\ln(t+1) + C
\end{aligned}$$

其中 $C = 2C_1 - 4C_2 + 4C_3$。

**第 6 步：回代 $t = \sqrt{x}$**。
$$\int \frac{\sqrt{x} - 1}{\sqrt{x} + 1}\,dx = x - 4\sqrt{x} + 4\ln(1 + \sqrt{x}) + C$$

（注意 $t^2 = x$，$t = \sqrt{x}$，$t+1 = \sqrt{x}+1$）

**第 7 步：验证**（对结果求导）。
$$\begin{aligned}
\frac{d}{dx}\Big[x - 4\sqrt{x} + 4\ln(1+\sqrt{x}) + C\Big]
&= 1 - 4 \cdot \frac{1}{2\sqrt{x}} + 4 \cdot \frac{1}{1+\sqrt{x}} \cdot \frac{1}{2\sqrt{x}} \\
&= 1 - \frac{2}{\sqrt{x}} + \frac{2}{\sqrt{x}(1+\sqrt{x})}
\end{aligned}$$

通分合并：
$$\begin{aligned}
1 - \frac{2}{\sqrt{x}} + \frac{2}{\sqrt{x}(1+\sqrt{x})}
&= 1 - \frac{2(1+\sqrt{x})}{\sqrt{x}(1+\sqrt{x})} + \frac{2}{\sqrt{x}(1+\sqrt{x})} \\
&= 1 - \frac{2 + 2\sqrt{x} - 2}{\sqrt{x}(1+\sqrt{x})} \\
&= 1 - \frac{2\sqrt{x}}{\sqrt{x}(1+\sqrt{x})} \\
&= 1 - \frac{2}{1+\sqrt{x}} \\
&= \frac{(1+\sqrt{x}) - 2}{1+\sqrt{x}} \\
&= \frac{\sqrt{x} - 1}{\sqrt{x} + 1} \quad \checkmark
\end{aligned}$$

与被积函数一致。验证通过。

**答案**：$\displaystyle\int \frac{\sqrt{x} - 1}{\sqrt{x} + 1}\,dx = x - 4\sqrt{x} + 4\ln(1 + \sqrt{x}) + C$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 对 $\int \frac{dx}{1+\sqrt{x}}$ 直接套用基本积分公式，认为结果是 $\ln(1+\sqrt{x}) + C$ | 未被积函数不是 $\frac{1}{x}$ 的形式，不能直接积分。必须先通过 $t = \sqrt{x}$ 代换化为有理函数，再积分 |
| 代换后 $dx$ 转换错误：以为 $t=\sqrt{x}$ 时 $dx = dt$ 或 $dx = \frac{1}{2\sqrt{x}}dt$ | 正确转换：由 $x = t^2$ 得 $\frac{dx}{dt} = 2t$，因此 $dx = 2t\,dt$。口诀："先平方，再微分" |
| 回代时只替换了部分 $t$，忘了将所有 $t$ 换回 $\sqrt{x}$ | 回代必须彻底——结果中所有 $t$ 都应替换为 $\sqrt{x}$ 或 $\sqrt{x+a}$。例如 $t^2$ 换成 $x$，$t$ 换成 $\sqrt{x}$，$t+1$ 换成 $\sqrt{x}+1$ |
| 对 $\int \frac{x}{\sqrt{x+1}}dx$ 直接设 $t = x+1$（而非 $t = \sqrt{x+1}$）导致根式仍在 | 代换的目标是**消去根号**。应设 $t = \sqrt{x+1}$，此时 $x = t^2 - 1$，根号自然消失。若设 $t = x+1$，则 $\sqrt{x+1} = \sqrt{t}$，根号仍然存在 |
| 认为根式代换后得到的一定是真分式，不需要做多项式长除法 | 代换后的有理函数可能是假分式（分子次数 $\ge$ 分母次数），必须先做多项式长除法才能积分。例 2 中的 $\frac{t}{1+t}$ 和例 4 中的 $\frac{t^2 - t}{t+1}$ 都是假分式 |
| 代数变形时符号出错：$\frac{x}{\sqrt{x+1}} = \frac{(x+1)-1}{\sqrt{x+1}} = \sqrt{x+1} - 1$（忘记 $\frac{1}{\sqrt{x+1}}$ 要乘以系数） | 正确变形：$\frac{x}{\sqrt{x+1}} = \frac{(x+1)-1}{\sqrt{x+1}} = \frac{x+1}{\sqrt{x+1}} - \frac{1}{\sqrt{x+1}} = (x+1)^{1/2} - (x+1)^{-1/2}$。第二项是 $(x+1)^{-1/2}$ 而非 $1$ |

### 检查点

- [ ] 能否独立陈述定理 6.8 中两种根式代换的公式（$x$ 和 $dx$ 的 $t$ 表达式）？
- [ ] 对于一个含 $\sqrt{x+a}$ 的积分，能否正确选择 $t$ 并写出 $x$ 和 $dx$ 的表达式？
- [ ] 能否在代换后判断有理函数是否为假分式，并正确执行多项式长除法？
- [ ] 能否将 $\frac{1}{t^2 - a^2}$ 正确分解为部分分式并逐项积分？
- [ ] 能否在最终结果中正确执行回代（$t \to \sqrt{x}$ 或 $t \to \sqrt{x+a}$）？
- [ ] 能否通过求导验证积分结果的正确性？
- [ ] 能否独立判断何时可以用代数变形法代替根式代换？

**1.** 利用幂函数积分公式直接计算下列不定积分：

(1) $\displaystyle\int \left(2\sqrt{x} - \frac{3}{\sqrt{x}}\right)dx \quad (x > 0)$

(2) $\displaystyle\int x\sqrt{x}\,dx \quad (x > 0)$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \left(2\sqrt{x} - \frac{3}{\sqrt{x}}\right)dx$

将根式写为幂函数形式：$\sqrt{x} = x^{1/2}$，$\frac{1}{\sqrt{x}} = x^{-1/2}$。

$$\begin{aligned}
\int \left(2x^{1/2} - 3x^{-1/2}\right)dx
&= 2\int x^{1/2}\,dx - 3\int x^{-1/2}\,dx \\
&= 2 \cdot \frac{x^{3/2}}{3/2} - 3 \cdot \frac{x^{1/2}}{1/2} + C \\
&= \frac{4}{3}x^{3/2} - 6x^{1/2} + C
\end{aligned}$$

**验证**：
$$\frac{d}{dx}\left(\frac{4}{3}x^{3/2} - 6x^{1/2} + C\right) = \frac{4}{3} \cdot \frac{3}{2}x^{1/2} - 6 \cdot \frac{1}{2}x^{-1/2} = 2x^{1/2} - 3x^{-1/2} = 2\sqrt{x} - \frac{3}{\sqrt{x}} \quad \checkmark$$

---

**(2)** $\displaystyle\int x\sqrt{x}\,dx$

$x\sqrt{x} = x \cdot x^{1/2} = x^{3/2}$。

$$\int x^{3/2}\,dx = \frac{x^{5/2}}{5/2} + C = \frac{2}{5}x^{5/2} + C$$

**验证**：$\frac{d}{dx}\left(\frac{2}{5}x^{5/2} + C\right) = \frac{2}{5} \cdot \frac{5}{2}x^{3/2} = x^{3/2} = x\sqrt{x} \quad \checkmark$

**2.** 用根式代换法计算 $\displaystyle\int \frac{dx}{1 - \sqrt{x}} \quad (x > 0, x \neq 1)$。

<details><summary>参考答案</summary>

被积函数含 $\sqrt{x}$，令 $t = \sqrt{x}$，则 $x = t^2$，$dx = 2t\,dt$（$t > 0$）。

$$\int \frac{dx}{1 - \sqrt{x}} = \int \frac{2t\,dt}{1 - t} = 2\int \frac{t}{1 - t}\,dt$$

有理函数 $\frac{t}{1-t}$ 是假分式，做多项式长除法：
$$\frac{t}{1-t} = -\frac{t}{t-1} = -\left(1 + \frac{1}{t-1}\right) = -1 - \frac{1}{t-1}$$

或者直接变形：$\frac{t}{1-t} = -1 + \frac{1}{1-t}$。

代入：
$$2\int \frac{t}{1-t}\,dt = 2\int \left(-1 + \frac{1}{1-t}\right)dt = -2\int 1\,dt + 2\int \frac{1}{1-t}\,dt$$

- $\int 1\,dt = t + C_1$
- $\int \frac{1}{1-t}\,dt = -\ln|1-t| + C_2$（定理 6.4，$a=1$）

因此：
$$2\int \frac{t}{1-t}\,dt = -2t - 2\ln|1-t| + C$$

回代 $t = \sqrt{x}$：
$$\int \frac{dx}{1 - \sqrt{x}} = -2\sqrt{x} - 2\ln|1 - \sqrt{x}| + C$$

**验证**：
$$\begin{aligned}
\frac{d}{dx}\Big[-2\sqrt{x} - 2\ln|1-\sqrt{x}| + C\Big]
&= -2 \cdot \frac{1}{2\sqrt{x}} - 2 \cdot \frac{1}{1-\sqrt{x}} \cdot \left(-\frac{1}{2\sqrt{x}}\right) \\
&= -\frac{1}{\sqrt{x}} + \frac{1}{\sqrt{x}(1-\sqrt{x})} \\
&= \frac{-(1-\sqrt{x}) + 1}{\sqrt{x}(1-\sqrt{x})} \\
&= \frac{\sqrt{x}}{\sqrt{x}(1-\sqrt{x})} = \frac{1}{1-\sqrt{x}} \quad \checkmark
\end{aligned}$$

</details>

**3.** 用根式代换法计算 $\displaystyle\int \frac{\sqrt{x}}{1 + x}\,dx \quad (x > 0)$。

<details><summary>参考答案</summary>

被积函数含 $\sqrt{x}$，令 $t = \sqrt{x}$，则 $x = t^2$，$dx = 2t\,dt$（$t > 0$）。

$$\int \frac{\sqrt{x}}{1 + x}\,dx = \int \frac{t}{1 + t^2} \cdot 2t\,dt = 2\int \frac{t^2}{1 + t^2}\,dt$$

$\frac{t^2}{1+t^2}$ 是假分式，做多项式长除法：
$$\frac{t^2}{1+t^2} = 1 - \frac{1}{1+t^2}$$

代入：
$$2\int \frac{t^2}{1+t^2}\,dt = 2\int 1\,dt - 2\int \frac{1}{1+t^2}\,dt$$

- $\int 1\,dt = t + C_1$
- $\int \frac{1}{1+t^2}\,dt = \arctan t + C_2$（表 6.1 公式 6）

因此：
$$2\int \frac{t^2}{1+t^2}\,dt = 2t - 2\arctan t + C$$

回代 $t = \sqrt{x}$：
$$\int \frac{\sqrt{x}}{1 + x}\,dx = 2\sqrt{x} - 2\arctan\sqrt{x} + C$$

**验证**：
$$\begin{aligned}
\frac{d}{dx}\Big[2\sqrt{x} - 2\arctan\sqrt{x} + C\Big]
&= 2 \cdot \frac{1}{2\sqrt{x}} - 2 \cdot \frac{1}{1 + (\sqrt{x})^2} \cdot \frac{1}{2\sqrt{x}} \\
&= \frac{1}{\sqrt{x}} - \frac{1}{\sqrt{x}(1+x)} \\
&= \frac{(1+x) - 1}{\sqrt{x}(1+x)} = \frac{x}{\sqrt{x}(1+x)} = \frac{\sqrt{x}}{1+x} \quad \checkmark
\end{aligned}$$

</details>

**4.** 用根式代换法计算 $\displaystyle\int \frac{dx}{(x+1)\sqrt{x}} \quad (x > 0)$。

<details><summary>参考答案</summary>

被积函数含 $\sqrt{x}$，令 $t = \sqrt{x}$，则 $x = t^2$，$dx = 2t\,dt$（$t > 0$）。

$$\int \frac{dx}{(x+1)\sqrt{x}} = \int \frac{2t\,dt}{(t^2+1) \cdot t} = 2\int \frac{dt}{t^2 + 1}$$

由表 6.1 公式 6（$\int \frac{1}{1+t^2}\,dt = \arctan t + C$）：
$$2\int \frac{dt}{t^2 + 1} = 2\arctan t + C$$

回代 $t = \sqrt{x}$：
$$\int \frac{dx}{(x+1)\sqrt{x}} = 2\arctan\sqrt{x} + C$$

**验证**：
$$\frac{d}{dx}\big[2\arctan\sqrt{x} + C\big] = 2 \cdot \frac{1}{1 + (\sqrt{x})^2} \cdot \frac{1}{2\sqrt{x}} = \frac{1}{(1+x)\sqrt{x}} \quad \checkmark$$

**5.** 用根式代换法或代数变形法计算 $\displaystyle\int x\sqrt{x-1}\,dx \quad (x > 1)$。

<details><summary>参考答案</summary>

**方法一：根式代换法**。

被积函数含 $\sqrt{x-1}$，令 $t = \sqrt{x-1}$，则 $x = t^2 + 1$，$dx = 2t\,dt$（$t > 0$）。

$$\int x\sqrt{x-1}\,dx = \int (t^2 + 1) \cdot t \cdot 2t\,dt = \int (t^2 + 1) \cdot 2t^2\,dt = 2\int (t^4 + t^2)\,dt$$

逐项积分：
$$2\int (t^4 + t^2)\,dt = 2\left(\frac{t^5}{5} + \frac{t^3}{3}\right) + C = \frac{2}{5}t^5 + \frac{2}{3}t^3 + C$$

回代 $t = \sqrt{x-1}$：
$$\int x\sqrt{x-1}\,dx = \frac{2}{5}(x-1)^{5/2} + \frac{2}{3}(x-1)^{3/2} + C$$

可提取公因子 $\frac{2}{15}(x-1)^{3/2}$ 化简：
$$\int x\sqrt{x-1}\,dx = \frac{2}{15}(x-1)^{3/2}\big[3(x-1) + 5\big] + C = \frac{2}{15}(x-1)^{3/2}(3x + 2) + C$$

**方法二：代数变形法**。

$$x\sqrt{x-1} = [(x-1) + 1]\sqrt{x-1} = (x-1)^{3/2} + (x-1)^{1/2}$$

$$\int \big[(x-1)^{3/2} + (x-1)^{1/2}\big]dx = \frac{2}{5}(x-1)^{5/2} + \frac{2}{3}(x-1)^{3/2} + C$$

与代换法结果一致。

**验证**：
$$\begin{aligned}
\frac{d}{dx}\left[\frac{2}{5}(x-1)^{5/2} + \frac{2}{3}(x-1)^{3/2} + C\right]
&= \frac{2}{5} \cdot \frac{5}{2}(x-1)^{3/2} + \frac{2}{3} \cdot \frac{3}{2}(x-1)^{1/2} \\
&= (x-1)^{3/2} + (x-1)^{1/2} \\
&= \sqrt{x-1}\big[(x-1) + 1\big] = x\sqrt{x-1} \quad \checkmark
\end{aligned}$$

</details>
