## 2. 背景与应用场景

在前序文件中，我们掌握了基本积分公式、换元积分法和分部积分法——这些是不定积分计算的三大核心工具。然而，有一大类常见函数——**有理函数**（两个多项式的商）——的积分问题，既不能直接套用基本积分公式，也无法简单地通过换元或分部积分一次性解决。

有理函数的积分在数学和工程中有广泛的应用：

- **控制工程**：系统的传递函数通常为有理函数，求其拉普拉斯逆变换需要部分分式分解
- **微分方程**：求解常系数线性微分方程时，特征方程的有理解对应着有理函数的部分分式分解
- **数值分析**：复杂函数的近似积分中常将有理函数作为中间步骤
- **概率统计**：某些概率分布（如柯西分布）的密度函数为有理函数

有理函数积分的核心思想是：**将一个复杂的有理函数分解为若干个简单分式之和，每个简单分式都可以用基本积分公式或其直接推广来积分**。这个方法体系分为三个层次：预备公式、代数化简（多项式除法）、代数分解（部分分式）。

## 3. 核心概念与符号约定

### 3.1 有理函数与真分式、假分式

**有理函数（Rational Function）**：若 $P(x)$ 和 $Q(x)$ 均为多项式且 $Q(x)$ 不是零多项式，则称
$$R(x) = \frac{P(x)}{Q(x)}$$
为一个有理函数。本章中我们默认 $P$ 和 $Q$ 的系数均为实数。

设 $\deg(P)$ 表示多项式 $P$ 的最高次数。根据分子分母的次数关系，有理函数分为两类：

- **真分式（Proper Rational Function）**：$\deg(P) < \deg(Q)$，如 $\displaystyle\frac{3x+5}{x^2+3x+2}$（$\deg=1<2$）
- **假分式（Improper Rational Function）**：$\deg(P) \geq \deg(Q)$，如 $\displaystyle\frac{x^3+2x}{x^2+1}$（$\deg=3>2$）

### 3.2 有理函数积分的总体策略

有理函数积分遵循以下处理流程：

```
有理函数 R(x) = P(x)/Q(x)
        |
        v
  第 1 步：判断是否为真分式？
  |-- 假分式 (deg(P) >= deg(Q)) --> 多项式长除法 --> 多项式 + 真分式
  |-- 真分式 (deg(P) < deg(Q))  --> 直接进入第 2 步
        |
        v
  第 2 步：对分母因式分解，根据结果选择不同处理
  |-- 分母为不同一次因式乘积 --> 部分分式分解 A/(x-a)+B/(x-b)
  |-- 分母为不可约二次因式 --> 分子拆分 + 配方 + arctan
        |
        v
  第 3 步：逐项积分（利用定理 6.4~6.6 和基本公式），合并结果
```

### 3.3 符号表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\deg(P)$ | 多项式 $P$ 的最高次数 | $\deg(x^3+2x)=3$ |
| $\frac{P(x)}{Q(x)}$ | 有理函数 | $\frac{x^3+2x}{x^2+1}$ |
| $A, B$ | 部分分式中的待定常数 | $\frac{1}{(x-2)(x-3)} = \frac{A}{x-2} + \frac{B}{x-3}$ |
| $\Delta = p^2 - 4q$ | 二次多项式 $x^2+px+q$ 的判别式 | $\Delta<0$ 时为不可约二次因式 |
| $u = x + k$ | 配方法中的线性平移 | $(x+1)^2+4$ 中令 $u=x+1$ |

## 4. 原理与方法

### 4.1 预备公式——三个推广的基本积分公式

在有理函数积分中，最常遇到的三种积分形式是 $\int\frac{1}{x-a}dx$、$\int\frac{1}{x^2+a^2}dx$ 和 $\int\frac{f'(x)}{f(x)}dx$。它们都可以从基本积分公式出发，结合链式法则推导得到。

---

**定理 6.4（$\int \frac{1}{x-a}dx$ 公式）**：对任意实数 $a$，在 $x \neq a$ 的区间上有
$$\boxed{\int \frac{1}{x-a}\,dx = \ln|x-a| + C}$$

**证明**：令 $F(x) = \ln|x-a|$，对 $F$ 求导：
- 当 $x > a$ 时，$F(x) = \ln(x-a)$，$F'(x) = \frac{1}{x-a}$。
- 当 $x < a$ 时，$F(x) = \ln(a-x)$，由链式法则 $F'(x) = \frac{1}{a-x} \cdot (-1) = \frac{1}{x-a}$。

因此对一切 $x \neq a$ 有 $F'(x) = \frac{1}{x-a}$。由不定积分的定义，
$$\int \frac{1}{x-a}\,dx = F(x) + C = \ln|x-a| + C$$
证毕。

**推论**：令 $a = 0$ 得到基本积分公式 $\int \frac{1}{x}\,dx = \ln|x| + C$（表 6.1 公式 2），因此定理 6.4 是该公式的直接推广。

---

**定理 6.5（$\int \frac{1}{x^2+a^2}dx$ 公式）**：设 $a > 0$，则
$$\boxed{\int \frac{1}{x^2 + a^2}\,dx = \frac{1}{a}\arctan\frac{x}{a} + C}$$

**证明**：令 $G(x) = \frac{1}{a}\arctan\frac{x}{a}$。由链式法则（$( \arctan x)' = \frac{1}{1+x^2}$）：
$$G'(x) = \frac{1}{a} \cdot \frac{1}{1 + (x/a)^2} \cdot \frac{1}{a} = \frac{1}{a^2} \cdot \frac{1}{1 + x^2/a^2} = \frac{1}{a^2} \cdot \frac{a^2}{a^2 + x^2} = \frac{1}{x^2 + a^2}$$

因此 $G$ 是 $\frac{1}{x^2 + a^2}$ 的一个原函数。证毕。

**推论**：令 $a = 1$ 得到 $\int \frac{1}{1+x^2}\,dx = \arctan x + C$，即表 6.1 公式 6。

**常见错误**：计算 $\int \frac{1}{x^2+4}dx$ 时，忘记乘以 $\frac{1}{a} = \frac{1}{2}$，直接写成 $\arctan\frac{x}{2} + C$。正确结果为 $\frac{1}{2}\arctan\frac{x}{2} + C$。

---

**定理 6.6（对数积分模式 / Logarithmic Integration）**：设 $f(x)$ 可导且 $f(x) \neq 0$，则
$$\boxed{\int \frac{f'(x)}{f(x)}\,dx = \ln|f(x)| + C}$$

**证明**：令 $H(x) = \ln|f(x)|$。由链式法则：
$$H'(x) = \frac{1}{f(x)} \cdot f'(x) = \frac{f'(x)}{f(x)}$$

因此 $H$ 是 $\frac{f'(x)}{f(x)}$ 的一个原函数。证毕。

**应用方法**：遇到形如 $\int \frac{g(x)}{h(x)}dx$ 时，检查分子 $g(x)$ 是否恰好是分母 $h(x)$ 的导数。若是，则积分结果为 $\ln|h(x)| + C$。

**例**：$\int \frac{2x}{x^2+1}\,dx = \ln(x^2+1) + C$，因为 $\frac{d}{dx}(x^2+1) = 2x$ 且 $x^2+1 > 0$ 恒成立。

定理 6.4~6.6 是本节最重要的积分工具。在实际问题中，它们往往需要组合使用。

---

### 4.2 多项式长除法——化假分式为真分式

当有理函数 $R(x) = P(x)/Q(x)$ 为**假分式**时（$\deg(P) \geq \deg(Q)$），可以通过多项式长除法将其写为：
$$\frac{P(x)}{Q(x)} = S(x) + \frac{T(x)}{Q(x)}$$
其中 $S(x)$ 是多项式（商式），$T(x)$ 是余式，且 $\deg(T) < \deg(Q)$（即 $T/Q$ 为真分式）。

**多项式长除法步骤**：以 $\frac{x^3 + 2x}{x^2 + 1}$ 为例。

**第 1 步**：比较次数。被除式 $\deg=3$，除式 $\deg=2$，需要做除法。

**第 2 步**：用被除式的首项 $x^3$ 除以除式的首项 $x^2$，得到第一项商 $x$。

**第 3 步**：将商乘以除式得 $x(x^2+1) = x^3 + x$，从被除式中减去：
$$(x^3 + 2x) - (x^3 + x) = x$$

**第 4 步**：余式 $x$ 的次数为 $1$，小于除式次数 $2$，停止。商为 $x$，余式为 $x$。
$$\frac{x^3 + 2x}{x^2 + 1} = x + \frac{x}{x^2 + 1}$$

**一般形式**的书写方式如下。对 $\frac{P(x)}{Q(x)}$ 执行长除法：

设商式为 $S(x)$，余式为 $T(x)$，则
$$\frac{P(x)}{Q(x)} = S(x) + \frac{T(x)}{Q(x)}, \quad \deg(T) < \deg(Q)$$

**为什么要先做多项式除法？** 因为只有真分式才能直接应用部分分式分解或配方积分法。多项式部分 $S(x)$ 可以直接用基本积分公式积分（如 $\int x\,dx = x^2/2 + C$）。因此假分式的积分被分离为"多项式积分 + 真分式积分"两个独立的部分。

### 4.3 部分分式分解——分母为不同一次因式的乘积

当真分式的分母 $Q(x)$ 可以分解为不同一次因式的乘积时：
$$Q(x) = (x-a_1)(x-a_2)\cdots(x-a_n) \quad (a_i \text{互不相同})$$
可以将真分式分解为以下形式：
$$\frac{P(x)}{Q(x)} = \frac{A_1}{x-a_1} + \frac{A_2}{x-a_2} + \cdots + \frac{A_n}{x-a_n}$$
其中 $A_1, A_2, \dots, A_n$ 是待定常数。这种方法称为**部分分式分解**（Partial Fraction Decomposition），$A_i$ 称为**待定系数**。

以两项情形 $\frac{P(x)}{(x-a)(x-b)}$（$a \neq b$）为例，分解形式为：
$$\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$$

确定待定系数 $A, B$ 有两种常用方法。

---

**方法一：代入法（适用于不同一次因式乘积）**

**第 1 步**：设定分解形式。
$$\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$$

**第 2 步**：两边同乘 $(x-a)(x-b)$，消去分母：
$$P(x) = A(x-b) + B(x-a)$$

**第 3 步**：代入使某些项为零的 $x$ 值。令 $x = a$（此时 $x-a=0$，含 $B$ 的项消失）：
$$P(a) = A(a-b) \;\Longrightarrow\; A = \frac{P(a)}{a-b}$$

令 $x = b$（此时 $x-b=0$，含 $A$ 的项消失）：
$$P(b) = B(b-a) \;\Longrightarrow\; B = \frac{P(b)}{b-a}$$

**示例**：分解 $\frac{3x+5}{(x+1)(x+2)}$。

两边同乘 $(x+1)(x+2)$：$3x+5 = A(x+2) + B(x+1)$。

令 $x = -1$：$3(-1)+5 = A(1) \;\Rightarrow\; 2 = A$。
令 $x = -2$：$3(-2)+5 = B(-1) \;\Rightarrow\; -1 = -B \;\Rightarrow\; B = 1$。

因此 $\frac{3x+5}{(x+1)(x+2)} = \frac{2}{x+1} + \frac{1}{x+2}$。

---

**方法二：比较系数法（通用方法）**

从 $P(x) = A(x-b) + B(x-a)$ 出发，展开右端并合并同类项：
$$P(x) = (A+B)x + (-Ab - Ba)$$

比较等式两边 $x$ 的同次幂系数，得到关于 $A, B$ 的线性方程组，解之即得。

**示例**（同上题）：$3x+5 = A(x+2) + B(x+1) = (A+B)x + (2A+B)$

比较系数得：
$$\begin{cases}
A + B = 3 \\
2A + B = 5
\end{cases}$$

两式相减得 $A = 2$，代入得 $B = 1$。

**方法选择**：对于分母为不同一次因式乘积的情形，**代入法**最为快捷；对于重因式等复杂情形，比较系数法更为通用。

---

完成部分分式分解后，每个分式 $\frac{A}{x-a}$ 都可以直接用定理 6.4 积分：
$$\int \frac{A}{x-a}\,dx = A\ln|x-a| + C$$

---

### 4.4 分母为不可约二次因式的积分（配方 + arctan）

当真分式的分母是判别式 $\Delta < 0$ 的二次式时，它不能在实数范围内分解为一次因式的乘积。此时的积分策略取决于分子的形式。

---

**情形一：分子为常数** $\int \frac{1}{x^2+px+q}\,dx$（$\Delta = p^2 - 4q < 0$）

**第 1 步：配方**。
$$x^2 + px + q = \left(x + \frac{p}{2}\right)^2 + \left(q - \frac{p^2}{4}\right)$$

由于 $\Delta < 0$，$q - \frac{p^2}{4} > 0$。记 $a = \sqrt{q - \frac{p^2}{4}}$。

**第 2 步：利用定理 6.5**。作平移 $u = x + \frac{p}{2}$：
$$\int \frac{1}{x^2+px+q}\,dx = \int \frac{1}{u^2 + a^2}\,du = \frac{1}{a}\arctan\frac{u}{a} + C = \frac{1}{a}\arctan\frac{x + p/2}{a} + C$$

**例**：$\int \frac{1}{x^2+2x+5}\,dx$。

配方：$x^2+2x+5 = (x+1)^2 + 4 = (x+1)^2 + 2^2$。

$\int \frac{1}{(x+1)^2+2^2}\,dx = \frac{1}{2}\arctan\frac{x+1}{2} + C$（代入 $a=2$, $u=x+1$）。

---

**情形二：分子为线性函数** $\int \frac{mx+n}{x^2+px+q}\,dx$（$\Delta < 0$）

这是有理函数积分中最常见也最具技巧性的情形。核心策略是**拆分分子**：

**第 1 步**：计算分母的导数：$\frac{d}{dx}(x^2+px+q) = 2x + p$。

**第 2 步**：将分子 $mx+n$ 写为"分母导数的倍数 + 常数"的形式：
$$mx + n = \frac{m}{2}(2x + p) + \left(n - \frac{mp}{2}\right)$$

**第 3 步**：分离积分：
$$\int \frac{mx+n}{x^2+px+q}\,dx = \frac{m}{2}\int \frac{2x+p}{x^2+px+q}\,dx + \left(n - \frac{mp}{2}\right)\int \frac{1}{x^2+px+q}\,dx$$

其中：
- **第一项**：$\int \frac{2x+p}{x^2+px+q}\,dx$ 恰好是 $\int \frac{f'(x)}{f(x)}dx$ 的形式（$f(x)=x^2+px+q$），由定理 6.6 得 $\ln|x^2+px+q| + C$。
- **第二项**：$\int \frac{1}{x^2+px+q}\,dx$ 属于情形一，用配方 + 定理 6.5 处理。

## 5. 例题

### 例 1：部分分式分解（分母为不同一次因式乘积）

计算 $\displaystyle\int \frac{3x+5}{x^2+3x+2}\,dx$。

**解**：

**第 1 步：因式分解分母并判断分式类型**。
$$x^2 + 3x + 2 = (x+1)(x+2)$$
分子 $\deg=1$，分母 $\deg=2$，是真分式。

**第 2 步：设定部分分式分解形式**。
设存在常数 $A, B$ 使得：
$$\frac{3x+5}{(x+1)(x+2)} = \frac{A}{x+1} + \frac{B}{x+2}$$

**第 3 步：确定待定常数（代入法）**。
两边同乘 $(x+1)(x+2)$：
$$3x+5 = A(x+2) + B(x+1)$$

令 $x = -1$（此时 $x+1=0$，含 $B$ 的项消失）：
$$3(-1)+5 = A(-1+2) \;\Longrightarrow\; 2 = A \;\Longrightarrow\; A = 2$$

令 $x = -2$（此时 $x+2=0$，含 $A$ 的项消失）：
$$3(-2)+5 = B(-2+1) \;\Longrightarrow\; -1 = -B \;\Longrightarrow\; B = 1$$

因此：
$$\frac{3x+5}{(x+1)(x+2)} = \frac{2}{x+1} + \frac{1}{x+2}$$

**第 4 步：逐项积分**。
利用线性性质（定理 6.2）和定理 6.4（$\int\frac{1}{x-a}dx = \ln|x-a|+C$）：
$$\begin{aligned}
\int \frac{3x+5}{x^2+3x+2}\,dx
&= \int \frac{2}{x+1}\,dx + \int \frac{1}{x+2}\,dx \\
&= 2\ln|x+1| + \ln|x+2| + C
\end{aligned}$$

**第 5 步：验证**。
对结果求导：
$$\frac{d}{dx}\big(2\ln|x+1| + \ln|x+2| + C\big) = \frac{2}{x+1} + \frac{1}{x+2}$$

通分验证：
$$\frac{2}{x+1} + \frac{1}{x+2} = \frac{2(x+2) + (x+1)}{(x+1)(x+2)} = \frac{3x+5}{x^2+3x+2}$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{3x+5}{x^2+3x+2}\,dx = 2\ln|x+1| + \ln|x+2| + C$

---

### 例 2：多项式长除法 + 对数积分模式

计算 $\displaystyle\int \frac{x^3 + 2x}{x^2 + 1}\,dx$。

**解**：

**第 1 步：判断分式类型**。
分子 $\deg=3$，分母 $\deg=2$，是假分式。先做多项式除法。

**第 2 步：多项式长除法**。
$$
\begin{aligned}
& \frac{x^3 + 2x}{x^2 + 1}: \\
& x^3 \div x^2 = x \quad (\text{第一项商}) \\
& x(x^2+1) = x^3 + x \\
& (x^3 + 2x) - (x^3 + x) = x \quad (\text{余式})
\end{aligned}
$$

商为 $x$，余式为 $x$，因此：
$$\frac{x^3 + 2x}{x^2 + 1} = x + \frac{x}{x^2 + 1}$$

**第 3 步：对多项式部分直接积分**。
$$\int x\,dx = \frac{x^2}{2} + C_1$$

**第 4 步：处理真分式部分——识别对数积分模式**。
对于 $\int \frac{x}{x^2+1}\,dx$：
令 $f(x) = x^2 + 1$，则 $f'(x) = 2x$。分子 $x = \frac{1}{2}f'(x)$。由定理 6.6：
$$\int \frac{x}{x^2+1}\,dx = \frac{1}{2}\int \frac{2x}{x^2+1}\,dx = \frac{1}{2}\ln(x^2+1) + C_2$$

（分母 $x^2+1 > 0$ 恒成立，故省略绝对值符号。）

**第 5 步：合并结果**。
$$\begin{aligned}
\int \frac{x^3 + 2x}{x^2 + 1}\,dx
&= \frac{x^2}{2} + \frac{1}{2}\ln(x^2+1) + C
\end{aligned}$$

其中 $C = C_1 + C_2$。

**第 6 步：验证**。
$$\begin{aligned}
\frac{d}{dx}\left[\frac{x^2}{2} + \frac{1}{2}\ln(x^2+1) + C\right]
&= x + \frac{1}{2} \cdot \frac{2x}{x^2+1} \\
&= x + \frac{x}{x^2+1} \\
&= \frac{x(x^2+1) + x}{x^2+1} = \frac{x^3 + 2x}{x^2+1}
\end{aligned}$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{x^3 + 2x}{x^2 + 1}\,dx = \frac{x^2}{2} + \frac{1}{2}\ln(x^2+1) + C$

### 例 3：分母为不可约二次因式的积分

计算 $\displaystyle\int \frac{2x + 3}{x^2 + 2x + 5}\,dx$。

**解**：

**第 1 步：检查分母是否可约**。
$\Delta = 2^2 - 4\cdot1\cdot5 = 4 - 20 = -16 < 0$，分母在实数范围内不可分解。
分子 $\deg=1$，分母 $\deg=2$，为真分式。

**第 2 步：拆分分子——写成"分母的导数 + 常数"的形式**。
分母的导数：$\frac{d}{dx}(x^2 + 2x + 5) = 2x + 2$

将分子 $2x+3$ 用 $2x+2$ 和常数表示：
$$2x + 3 = (2x+2) + 1$$

由线性性质（定理 6.2）：
$$\int \frac{2x+3}{x^2+2x+5}\,dx = \int \frac{2x+2}{x^2+2x+5}\,dx + \int \frac{1}{x^2+2x+5}\,dx$$

**第 3 步：处理第一部分——对数积分模式（定理 6.6）**。
令 $f(x) = x^2 + 2x + 5$，则 $f'(x) = 2x+2$。
$$\int \frac{2x+2}{x^2+2x+5}\,dx = \ln(x^2+2x+5) + C_1$$

（分母 $x^2+2x+5 = (x+1)^2 + 4 > 0$ 恒为正，故无需绝对值。）

**第 4 步：处理第二部分——配方 + arctan（定理 6.5）**。

配方：
$$x^2 + 2x + 5 = (x^2 + 2x + 1) + 4 = (x+1)^2 + 2^2$$

因此：
$$\int \frac{1}{x^2+2x+5}\,dx = \int \frac{1}{(x+1)^2 + 2^2}\,dx$$

作平移 $u = x+1$，$du = dx$，由定理 6.5（$a=2$）：
$$\int \frac{1}{u^2 + 2^2}\,du = \frac{1}{2}\arctan\frac{u}{2} + C_2$$

代回 $u = x+1$：
$$\int \frac{1}{x^2+2x+5}\,dx = \frac{1}{2}\arctan\frac{x+1}{2} + C_2$$

**第 5 步：合并结果**。
$$\begin{aligned}
\int \frac{2x+3}{x^2+2x+5}\,dx
&= \ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2} + C
\end{aligned}$$

其中 $C = C_1 + C_2$。

**第 6 步：验证**。

第一部分求导：
$$\frac{d}{dx}\big[\ln(x^2+2x+5)\big] = \frac{2x+2}{x^2+2x+5}$$

第二部分求导（链式法则）：
$$\frac{d}{dx}\left[\frac{1}{2}\arctan\frac{x+1}{2}\right]
= \frac{1}{2} \cdot \frac{1}{1 + \left(\frac{x+1}{2}\right)^2} \cdot \frac{1}{2}
= \frac{1}{4} \cdot \frac{1}{1 + \frac{(x+1)^2}{4}}
= \frac{1}{4} \cdot \frac{4}{4 + (x+1)^2} = \frac{1}{x^2+2x+5}$$

因此：
$$\frac{d}{dx}\left[\ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2} + C\right]
= \frac{2x+2}{x^2+2x+5} + \frac{1}{x^2+2x+5} = \frac{2x+3}{x^2+2x+5}$$
与被积函数一致。$\checkmark$

**答案**：$\displaystyle\int \frac{2x+3}{x^2+2x+5}\,dx = \ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2} + C$

## 6. 常见误区与检查点

### 常见误区

| 错误理解 | 正确理解 |
|----------|----------|
| 对假分式直接做部分分式分解，例如设 $\frac{x^3}{x^2-1} = \frac{A}{x-1} + \frac{B}{x+1}$ | 假分式必须先做**多项式长除法**化为"多项式 + 真分式"；部分分式分解只适用于真分式。长除法后：$\frac{x^3}{x^2-1} = x + \frac{x}{x^2-1}$，再对真分式 $\frac{x}{x^2-1}$ 做分解 |
| 分解时符号处理错误，如设 $\frac{1}{(x-2)(x-3)} = \frac{A}{x-2} - \frac{B}{x-3}$ | 标准形式用**加号**连接：$\frac{A}{x-2} + \frac{B}{x-3}$。$A$ 和 $B$ 本身可以是负数，无需在形式上写减号 |
| $\int \frac{1}{x^2+4}\,dx = \arctan\frac{x}{2} + C$（遗漏系数 $1/a$） | 由定理 6.5 得 $\int \frac{1}{x^2+2^2}\,dx = \frac{1}{2}\arctan\frac{x}{2} + C$。系数 $1/a$ 是必须的 |
| 对 $\int \frac{2x+3}{x^2+2x+5}\,dx$ 直接对分母配方就试图用 $\arctan$ | 若分子是**线性函数**，必须先**拆分分子**（写成"分母导数 + 常数"形式），否则配方后仍然无法直接积分 |
| 做部分分式代入法时，只得到 $A$ 不回代就结束 | 求出 $A$ 后必须代入原式求出所有系数（$B$ 等），再写回分解形式 | 
| 认为 $\ln|f(x)| + C$ 中的绝对值可以永远省略 | 当 $f(x)$ 恒正时（如 $x^2+1$、$(x+1)^2+4$）可省略；当 $f(x)$ 可能为负时（如 $x-2$）必须保留绝对值 |

### 检查点

- [ ] 能否独立推导并证明定理 6.4~6.6（通过求导验证）？
- [ ] 能否区分真分式与假分式？对假分式能否正确执行多项式长除法？
- [ ] 能否对分母为 $(x-a)(x-b)$ 的真分式设定分解形式并用代入法求出待定系数？
- [ ] 能否正确使用定理 6.5 计算 $\int \frac{1}{x^2+4x+13}\,dx$（先配方再套公式）？
- [ ] 对 $\int \frac{mx+n}{x^2+px+q}dx$（$\Delta<0$），能否正确拆分分子并分别积分？
- [ ] 能否对有理函数积分的结果进行求导验证？
- [ ] 能否识别 $\int \frac{f'(x)}{f(x)}dx$ 的模式并直接写出 $\ln|f(x)|+C$？

## 练习题

### 基础巩固

**1.** 计算下列不定积分：

(1) $\displaystyle\int \frac{1}{x^2 - 6x + 8}\,dx$

(2) $\displaystyle\int \frac{2x + 1}{x^2 + 4x + 3}\,dx$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{1}{x^2 - 6x + 8}\,dx$

因式分解分母：$x^2 - 6x + 8 = (x-2)(x-4)$。

设 $\frac{1}{(x-2)(x-4)} = \frac{A}{x-2} + \frac{B}{x-4}$。两边同乘 $(x-2)(x-4)$：$1 = A(x-4) + B(x-2)$。

令 $x=2$：$1 = A(-2) \Rightarrow A = -\frac12$。
令 $x=4$：$1 = B(2) \Rightarrow B = \frac12$。

因此：
$$\int \frac{1}{x^2-6x+8}\,dx = -\frac12\int \frac{1}{x-2}\,dx + \frac12\int \frac{1}{x-4}\,dx = -\frac12\ln|x-2| + \frac12\ln|x-4| + C = \frac12\ln\left|\frac{x-4}{x-2}\right| + C$$

**验证**：
$$\frac{d}{dx}\left[\frac12\ln\left|\frac{x-4}{x-2}\right| + C\right] = \frac12\left(\frac{1}{x-4} - \frac{1}{x-2}\right) = \frac12\cdot\frac{(x-2)-(x-4)}{(x-2)(x-4)} = \frac12\cdot\frac{2}{(x-2)(x-4)} = \frac{1}{x^2-6x+8} \quad \checkmark$$

---

**(2)** $\displaystyle\int \frac{2x+1}{x^2+4x+3}\,dx$

因式分解分母：$x^2+4x+3 = (x+1)(x+3)$。

设 $\frac{2x+1}{(x+1)(x+3)} = \frac{A}{x+1} + \frac{B}{x+3}$。两边同乘 $(x+1)(x+3)$：$2x+1 = A(x+3) + B(x+1)$。

令 $x=-1$：$-1 = A(2) \Rightarrow A = -\frac12$。
令 $x=-3$：$-5 = B(-2) \Rightarrow B = \frac52$。

因此：
$$\begin{aligned}
\int \frac{2x+1}{x^2+4x+3}\,dx &= -\frac12\int \frac{1}{x+1}\,dx + \frac52\int \frac{1}{x+3}\,dx \\
&= -\frac12\ln|x+1| + \frac52\ln|x+3| + C
\end{aligned}$$

**验证**：
$$\frac{d}{dx}\left[-\frac12\ln|x+1| + \frac52\ln|x+3| + C\right] = -\frac{1}{2(x+1)} + \frac{5}{2(x+3)} = \frac{-(x+3)+5(x+1)}{2(x+1)(x+3)} = \frac{4x+2}{2(x^2+4x+3)} = \frac{2x+1}{x^2+4x+3} \quad \checkmark$$

</details>

---

**2.** 计算下列不定积分：

(1) $\displaystyle\int \frac{x^2}{x-1}\,dx$

(2) $\displaystyle\int \frac{x^4}{x^2+1}\,dx$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{x^2}{x-1}\,dx$

假分式（$\deg=2 \geq \deg=1$），先做多项式除法。

代数方法：$x^2 = (x-1)(x+1) + 1$，因此 $\frac{x^2}{x-1} = x+1 + \frac{1}{x-1}$。

$$\int \frac{x^2}{x-1}\,dx = \int (x+1)\,dx + \int \frac{1}{x-1}\,dx = \frac{x^2}{2} + x + \ln|x-1| + C$$

**验证**：
$$\frac{d}{dx}\left[\frac{x^2}{2} + x + \ln|x-1| + C\right] = x + 1 + \frac{1}{x-1} = \frac{(x+1)(x-1) + 1}{x-1} = \frac{x^2}{x-1} \quad \checkmark$$

---

**(2)** $\displaystyle\int \frac{x^4}{x^2+1}\,dx$

假分式（$\deg=4 \geq \deg=2$），做多项式除法：

$$\frac{x^4}{x^2+1} = \frac{x^4 - 1 + 1}{x^2+1} = \frac{(x^2-1)(x^2+1) + 1}{x^2+1} = x^2 - 1 + \frac{1}{x^2+1}$$

因此：
$$\int \frac{x^4}{x^2+1}\,dx = \int (x^2-1)\,dx + \int \frac{1}{x^2+1}\,dx = \frac{x^3}{3} - x + \arctan x + C$$

**验证**：
$$\frac{d}{dx}\left[\frac{x^3}{3} - x + \arctan x + C\right] = x^2 - 1 + \frac{1}{x^2+1} = \frac{x^4}{x^2+1} \quad \checkmark$$

</details>

**3.** 计算下列不定积分：

(1) $\displaystyle\int \frac{1}{x^2 + 4x + 13}\,dx$

(2) $\displaystyle\int \frac{x+1}{x^2 + 2x + 10}\,dx$

<details><summary>参考答案</summary>

**(1)** $\displaystyle\int \frac{1}{x^2 + 4x + 13}\,dx$

$\Delta = 16 - 52 = -36 < 0$，分母不可约。

配方：$x^2 + 4x + 13 = (x^2 + 4x + 4) + 9 = (x+2)^2 + 3^2$

由定理 6.5（$a=3$，平移 $u=x+2$）：
$$\int \frac{1}{x^2+4x+13}\,dx = \int \frac{1}{(x+2)^2+3^2}\,dx = \frac{1}{3}\arctan\frac{x+2}{3} + C$$

**验证**：
$$\frac{d}{dx}\left[\frac13\arctan\frac{x+2}{3} + C\right] = \frac13\cdot\frac{1}{1+(\frac{x+2}{3})^2}\cdot\frac13 = \frac19\cdot\frac{9}{9+(x+2)^2} = \frac{1}{(x+2)^2+9} = \frac{1}{x^2+4x+13} \quad \checkmark$$

---

**(2)** $\displaystyle\int \frac{x+1}{x^2 + 2x + 10}\,dx$

$\Delta = 4 - 40 = -36 < 0$，分母不可约。

分母的导数：$(x^2+2x+10)' = 2x+2$。分子 $x+1 = \frac12(2x+2)$，恰好是分母导数的一半。

由定理 6.6（对数积分模式）：
$$\int \frac{x+1}{x^2+2x+10}\,dx = \frac12\int \frac{2x+2}{x^2+2x+10}\,dx = \frac12\ln(x^2+2x+10) + C$$

（分母 $(x+1)^2+9 > 0$ 恒为正，省略绝对值。）

**验证**：
$$\frac{d}{dx}\left[\frac12\ln(x^2+2x+10) + C\right] = \frac12\cdot\frac{2x+2}{x^2+2x+10} = \frac{x+1}{x^2+2x+10} \quad \checkmark$$

</details>

### 迁移应用

**4.** 计算 $\displaystyle\int \frac{x^2 + 2x - 1}{x^2 - 1}\,dx$（提示：先做多项式除法，再对余式做部分分式分解）。

<details><summary>参考答案</summary>

分子 $\deg=2$，分母 $\deg=2$，假分式。做多项式除法：
$$\frac{x^2+2x-1}{x^2-1} = \frac{(x^2-1) + 2x}{x^2-1} = 1 + \frac{2x}{x^2-1}$$

对 $\frac{2x}{(x-1)(x+1)}$ 做部分分式分解：

设 $\frac{2x}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}$。两边同乘 $(x-1)(x+1)$：$2x = A(x+1) + B(x-1)$。

令 $x=1$：$2 = A(2) \Rightarrow A = 1$。
令 $x=-1$：$-2 = B(-2) \Rightarrow B = 1$。

因此：
$$\frac{2x}{x^2-1} = \frac{1}{x-1} + \frac{1}{x+1}$$

于是：
$$\begin{aligned}
\int \frac{x^2+2x-1}{x^2-1}\,dx
&= \int 1\,dx + \int \frac{1}{x-1}\,dx + \int \frac{1}{x+1}\,dx \\
&= x + \ln|x-1| + \ln|x+1| + C \\
&= x + \ln|x^2-1| + C
\end{aligned}$$

**验证**：
$$\frac{d}{dx}\left[x + \ln|x^2-1| + C\right] = 1 + \frac{2x}{x^2-1} = \frac{x^2+2x-1}{x^2-1} \quad \checkmark$$

</details>

**5.** 计算 $\displaystyle\int \frac{3x + 1}{x^2 + 2x + 2}\,dx$。

<details><summary>参考答案</summary>

$\Delta = 4 - 8 = -4 < 0$，分母不可约。

分母的导数：$(x^2+2x+2)' = 2x+2$。

拆分分子：$3x+1 = \frac32(2x+2) + (1-3) = \frac32(2x+2) - 2$

因此：
$$\int \frac{3x+1}{x^2+2x+2}\,dx = \frac32\int \frac{2x+2}{x^2+2x+2}\,dx - 2\int \frac{1}{x^2+2x+2}\,dx$$

第一部分（对数积分模式，定理 6.6）：
$$\int \frac{2x+2}{x^2+2x+2}\,dx = \ln(x^2+2x+2) + C_1$$

第二部分（配方 + arctan，定理 6.5）：
$$x^2+2x+2 = (x+1)^2 + 1^2$$
$$\int \frac{1}{x^2+2x+2}\,dx = \int \frac{1}{(x+1)^2+1^2}\,dx = \arctan(x+1) + C_2$$

合并：
$$\int \frac{3x+1}{x^2+2x+2}\,dx = \frac32\ln(x^2+2x+2) - 2\arctan(x+1) + C$$

**验证**：
$$\frac{d}{dx}\left[\frac32\ln(x^2+2x+2) - 2\arctan(x+1) + C\right] = \frac32\cdot\frac{2x+2}{x^2+2x+2} - 2\cdot\frac{1}{1+(x+1)^2}$$
$$= \frac{3x+3}{x^2+2x+2} - \frac{2}{x^2+2x+2} = \frac{3x+1}{x^2+2x+2} \quad \checkmark$$

</details>

**6.** （综合题）计算 $\displaystyle\int \frac{2x^2 + x + 1}{x^2 - 3x + 2}\,dx$。

<details><summary>参考答案</summary>

分子 $\deg=2$，分母 $\deg=2$，假分式。先做多项式除法。

将分子写为分母的倍数加余项：
$$2x^2 + x + 1 = 2(x^2-3x+2) + (7x - 3)$$

因为 $2(x^2-3x+2) = 2x^2 - 6x + 4$，故 $(2x^2+x+1) - (2x^2-6x+4) = 7x-3$。

所以：
$$\frac{2x^2+x+1}{x^2-3x+2} = 2 + \frac{7x-3}{x^2-3x+2}$$

因式分解分母：$x^2-3x+2 = (x-1)(x-2)$。

对 $\frac{7x-3}{(x-1)(x-2)}$ 做部分分式分解：

设 $\frac{7x-3}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}$。两边同乘 $(x-1)(x-2)$：$7x-3 = A(x-2) + B(x-1)$。

令 $x=1$：$4 = A(-1) \Rightarrow A = -4$。
令 $x=2$：$11 = B(1) \Rightarrow B = 11$。

因此：
$$\frac{2x^2+x+1}{x^2-3x+2} = 2 - \frac{4}{x-1} + \frac{11}{x-2}$$

于是：
$$\begin{aligned}
\int \frac{2x^2+x+1}{x^2-3x+2}\,dx
&= \int 2\,dx - 4\int \frac{1}{x-1}\,dx + 11\int \frac{1}{x-2}\,dx \\
&= 2x - 4\ln|x-1| + 11\ln|x-2| + C
\end{aligned}$$

**验证**：
$$\begin{aligned}
\frac{d}{dx}\big[2x - 4\ln|x-1| + 11\ln|x-2| + C\big]
&= 2 - \frac{4}{x-1} + \frac{11}{x-2} \\
&= \frac{2(x-1)(x-2) - 4(x-2) + 11(x-1)}{(x-1)(x-2)} \\
&= \frac{2(x^2-3x+2) - 4x+8 + 11x-11}{x^2-3x+2} \\
&= \frac{2x^2-6x+4+7x-3}{x^2-3x+2} = \frac{2x^2+x+1}{x^2-3x+2} \quad \checkmark
\end{aligned}$$

</details>
